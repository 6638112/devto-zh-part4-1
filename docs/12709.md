我得了咽喉炎。症状包括头晕、疲劳和大量的新书签文件夹。老实说，我不知道在过去的一周里我启动了多少个编码程序。
这篇文章旨在宣布，今后，我将通过以下方式来引导我的注意力。

1.  主要关注 MITx: 6.00.1x，在 edX 上使用 Python 的计算机科学和编程入门。
2.  使用 Datacamp 和 Codewars 作为免费的学习工具。
3.  至少一周内不要开始任何新课程。

这也是一种求助。还有谁和 startitis 斗争过，谁对如何治疗有建议？
