# 你度假时带笔记本电脑吗？

> 原文:[https://dev . to/dvddpl/do-you-bring-your-laptop-on-vacation-5gpj](https://dev.to/dvddpl/do-you-bring-your-laptop-on-vacation-5gpj)

对我来说，假期是神圣的。我尽可能多地旅行，利用我得到的每一天。

> 假期是我和家人的时间，是无所事事的时间，也是做很多事情的时间。除了编码之外的一切。

事实上，我尽量远离任何类型的屏幕，我尽量避免用智能手机查看我的 dev.to 或 hackernoon feed。

那就是我**重生**的方式。一点点远离编码和技术信息的距离，让我回来时精神饱满，充满新的能量。

我喜欢把它看作是体育训练。**当你做运动**——即使你永远不会因为喜欢而停下来，想快速增长肌肉或变得更好，**你只需*不得不*休息一天**。如果你跳过它，你可能反而会压迫你的肌肉，导致更高的受伤风险。

我相信这对我们的大脑也很重要。或者至少对我有用。
