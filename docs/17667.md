**什么是 PLC？**

PLC 或可编程逻辑控制器是一种常用于监控工业生产操作的计算机。这种设备，你可能会看到被称为可编程控制器，可以在识别工业过程中的错误至关重要。PLC 的设计经久耐用，能够承受工业环境中的物理挑战。它们易于编程，用于控制机器人设备、装配线制造和其他要求可靠性的工业活动。

PLC 是工业自动化实施的基础设施的一部分。它们可以作为独立的计算机来控制单个的工业机器或过程。在许多情况下，多个 PLC 连接在一个网络中，并作为一个 SCADA 系统的组成部分。联网的 PLC 由运行专用 HMI 和 SCADA 软件的服务器控制。通过使用人机界面(HMI ),可以实现与联网 PLCS 的人机交互。PLCS 的配置、监控和控制都通过 HMI 完成。基于 PLC 输入为操作员生成警报是 PLC 的另一个重要用途。HMI 可以简单也可以复杂。一个非常基本的 HMI 可以由灯和按钮组成，为操作员提供执行指定功能的能力。更复杂的 HMI 包括执行编程的计算机和与 PLC 网络交互的监控软件。
工业自动化系统通常遵循相似的控制结构。它们由依靠 PLC 监控和控制工业设备的操作员界面组成。温度、湿度和压力等物理属性由传感器监控。这些传感器产生的电信号被逻辑控制器用来启动各种活动，例如启动设备或过程。访问 HMI 的操作员可以监控自动化过程。在某些情况下，操作员可能有能力超越自动化来处理程序异常。传感器、执行器和逻辑控制器与 HMI 的综合构成了所有自动化控制系统的基本架构。

**远程访问 PLC**

大多数 PLC 中都有通信接口。这些端口通常采用 RS-232、RS-422、RS-485 或以太网协议。PLC 配置通过这些通信端口实现。通过以太网端口将 PLC 连接到网络，这是大多数设备的标准功能。用于通过 PLC 控制和监控过程的服务器通过网络连接获得对它的访问。其他联网设备也可以与 PLC 通信。为了实现对 PLC 的远程访问，需要虚拟连接。
