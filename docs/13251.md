# 用这 9 个很酷的资源爱上 CSS

> 原文：<https://dev.to/ceeoreo/fall-in-love-with-css-with-these-9-cool-resources-djh>

*这篇文章最初刊登在[debugacademy.com](https://debugacademy.com/)上。*

我喜欢漂亮的东西，尤其喜欢有漂亮设计和布局的应用程序和网站。甚至在我开始学习编码之前，我真的很喜欢看到干净多彩的动画和界面。我最终了解到 CSS 是优秀设计的主要组成部分之一。这就是我热爱 CSS 的原因。

但是最近，我注意到许多开发人员甚至不喜欢 CSS，我和 Kelly 的感觉差不多。

We recently covered the basics of CSS in my [Debug Academy](https://debugacademy.com/) course and I started thinking about how I felt when I first started learning CSS. That’s when I realized that I didn’t always love it. In fact, it seemed to confuse me at the beginning. So I asked myself, what made the difference for me? What made me fall in love with CSS?

现在我知道那是我用来补充学习的资源。我使用了很多不同的工具来帮助我开始真正理解 CSS，我希望你也能做同样的事情。这里有 9 个工具帮助我爱上了 CSS。我将这些工具分为三类:游戏、参考网站、灵感网站。

## **游戏**

听到游戏化学习这个词，我通常会想到语言学习或者背欧洲国旗。但是我在这篇文章中展示的 3 个 CSS 游戏让我相信游戏化学习是学习新的 web 开发主题的好方法。

### [1。柔性盒青蛙](https://flexboxfroggy.com/)

一、爱情。Flexbox 青蛙！！！我在推特上发现了这个很酷的游戏。当时，我正在学习 CSS Flexbox，我很难真正掌握关键概念。然后有人发微博说他们是如何使用 Flexbox Froggy 学习 CSS Flexbox 的。所以我决定尝试一下。当 Flexbox 出现在我的 Debug Academy 课程中时，我最近又回到了这个游戏。如果你想学习 Flexbox 或者温习你已经知道的东西，试试这个吧。

### [2。网格花园](https://codepip.com/games/grid-garden/)

网格花园实际上是由制造 Flexbox Froggy 的同一批人创建的。我也是在推特上发现这个的。我用它来补充我当时在 [freeCodeCamp](https://www.freecodecamp.org/) 上学习 CSS Grid 的内容。当我创建网站布局时，我一直在使用 Grid 和 Flexbox。因此，大声喊出来 [Codepip](https://codepip.com/) 创造了这两个超级棒的游戏。没有你，我不会有今天。

### [3。CSS 食客](https://flukeout.github.io)

CSS Diner 是关于 CSS 选择器和特异性的。这是一个让很多人困惑的话题，甚至是一些有经验的 web 开发人员。所以我当然也不例外。有一段时间，我很难理解选择器。我只是忽略了它，并希望我最终能理解它。我不记得我是如何遇到 CSS Diner 的，但它真的帮助我理解了特异性和选择器是如何工作的，以及它们如何影响你的 CSS 结果。作为额外的奖励，这个游戏真的很有趣，设计得很好。

## **参考网站**

当你不确定某个属性是如何工作的，或者当你想用 CSS 创建一个新的效果但不知道怎么做时，参考网站是一个很好的去处。我将讨论一些绝对初学者通常不知道的网站。

### [1。CSS-招数](https://css-tricks.com/)

顾名思义，CSS-Tricks 提供了很多关于 CSS 的信息。这个网站涵盖了 CSS 的一些更微妙的特性，甚至有一些文章和指南涵盖了其他 web 开发和设计主题。我真的很喜欢这篇文章，[Flexbox 完全指南](https://css-tricks.com/snippets/css/a-guide-to-flexbox/)。看看它，领略一下 CSS-Tricks 提供的一些伟大的内容。

### [2。粉碎弹匣](https://www.smashingmagazine.com/category/css)

Smash 杂志是网页开发和网页设计的一个很好的资源。但是我发现他们关于 CSS 的文章特别有用。我真的很喜欢瑞秋·安德鲁的作品。她写了很多关于 CSS 的初级文章，所以我强烈推荐你读一些她的文章。

### [3。时尚设计〔T3〕](https://designmodo.com)

设计 Modo 是一个很棒的网站。它有关于所有设计的文章(很明显)。您可以了解有关 Adobe Photoshop、UX 设计等更多信息。我通常在这个网站上寻找他们的 CSS 教程。这也是跟上设计趋势和新行业标准的重要资源。他们甚至有一个设计灵感的部分！

### [4。调试学院](https://debugacademy.com/)

Debug Academy 是一个专注于 Drupal 8 的 web 开发程序。他们还教授按需 web 开发技术，如 Git、HTML、SASS，当然还有 CSS。我从五月份开始参加 Debug Academy 的课程，我可以诚实地说我的 CSS 技能已经到了一个新的水平。如果你还不愿意接受他们的付费课程，没必要担心。调试学院在其时事通讯中提供了初学者友好的 HTML/CSS 教程。你可以[在这里](https://debugacademy.com/begin)订阅。

## **灵感网站**

有时，你可能有一个项目想法或你必须完成的任务。但是当你想到你希望你的新网站是什么样子时，你的大脑一片空白。这种事经常发生在我身上。这就是灵感网站派上用场的地方。

### [1。密码笔](https://codepen.io)

人们用 CSS 在 Code Pen 上创造的东西总是让我吃惊。人们主要使用 CSS 创建一些非常酷的插图和动画。Code Pen 是一个很好的地方，可以让你了解 CSS 到底能做什么。也可以找一些很酷的登陆页之类的布局创意。

### [2。CSS 获奖者](https://www.csswinner.com/)

这是我最近发现的一个网站。我在为我的作品集网站寻找一些设计灵感🙃)我通过谷歌搜索找到了 CSS 获奖者。我喜欢 CSS 赢家的一点是，他们的很多网站都没有对于初学者来说似乎完全不可企及的布局和设计。当我浏览一些特色网站时，我并没有感到不知所措或力不从心。所以我绝对认为当你缺乏设计想法时，这是一个很好的去处。

## **结论**

CSS 有时确实令人沮丧。但也不是完全不可爱。我喜欢 CSS，我想如果你尝试一下这些资源，你也可以。和往常一样，欢迎在评论中留下你最喜欢的 CSS 资源。

在接下来的几周里，我将创建一些我自己的 CSS 教程。此外，我发了一个帖子，介绍了我参加在线课程的一些技巧。检查一下，留下一些反馈。我真的很感激。
