# 第二天-渲染元素

> 原文:[https://dev . to/miralkumbhani/day-two-rendering-elements-22 C9](https://dev.to/miralkumbhani/day-two-rendering-elements-22c9)

在我以前的文章中，我写过关于 JSX 和它的一些概念。这篇文章将包括 React 中元素的渲染。

元素是 React 应用程序的最小构建块。它是 React 中最小的可渲染单元。这些元素可以使用 *ReactDOM* 来呈现。React 元素是易于呈现的普通对象。反应元素和组件是两个不同的术语。

为了呈现 React 元素，必须用 id="root "或" app "将它们包含在 div 元素中。React 元素*是不可变的*，也就是说，一旦创建了一个元素，就不可能更新它的子元素或属性。因此，要更新元素，需要多次调用其 render 方法。通过使用虚拟 DOM 和高效的差分算法，React 渲染非常高效。该算法检查呈现的数据和更新的数据，并且只重新呈现需要更新的数据，而不是呈现所有数据。
