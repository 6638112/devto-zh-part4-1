# 修复你的记忆！要么写下来，要么丢掉。

> 原文：<https://dev.to/dechamp/fix-your-memory-write-it-down-or-lose-it-12cb>

你说过多少次“我应该知道这个！”，或者“我以为我已经学会了。”。也许你认为你记性不好，就像我告诉自己一百次一样。

我很容易就看了超过一百个小时的关于埃及人的视频，所以你会认为我对他们很了解。如果你问我一个关于它们的简单问题，我无法回答。

或者我已经看了几百个小时的学习材料，但有一半的时间我不得不回去重做，当我真的需要它的时候。

这让我不禁问自己，为什么会这样？！

我已经意识到这不是一个糟糕的记忆，但这是我处理数据的方式。

我们每天都被来自工作、家庭、家人、朋友、新闻、电视、手机和无数其他方式的信息轰炸。

我们的大脑被设计成能够过滤所有这些，并抓住重要的东西。

仅仅因为我们假设我们的大脑应该以某种方式工作，并不意味着它们就是这样工作的。你认为你应该记住你看到的，听到的，感觉到的，闻到的，说的一切...并不能让它成为现实。

当涉及到重要记忆时，你仍然需要帮助你的大脑知道什么是什么，以便它能够成为一台微调机器。

这就是写东西的地方！这告诉你的大脑这很重要，很可能需要长期保存。

你一边写一边读。它不仅给你一个口头输入(在你的脑海中，或者你用声音说出来)，也给你一个视觉输入。

我一直在练习这个，写下我想记住的东西，我立刻看到了进步。

那些年你以为高中老师让你写东西，只是为了惹人讨厌？谁知道他们一直都有这个答案呢？！
