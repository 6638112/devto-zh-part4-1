### [](#step-2-install-nodemon)第二步安装 nodemon

```
$ npm install -g nodemon
$ yarn add global nodemon 
```

### [](#step-4-create-nodemonjson)第四步创建 nodemon.json

我们需要创建 nodemon.json 文件，因为 nodemon 只与`.js`一起工作。

```
{
  "watch": ["src"],
  "ext": "ts",
  "exec": "ts-node ./src/index.ts"
} 
```
