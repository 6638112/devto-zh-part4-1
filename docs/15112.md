# React tldr 反应原生 0.60

> 原文:[https://dev.to/maniekm/react-tldr-react-native-0-60-2jp8](https://dev.to/maniekm/react-tldr-react-native-0-60-2jp8)

*   对可访问性 API 的许多改进

*   React Native 的开始屏幕已经更新

*   React Native 已经迁移到 AndroidX 上

*   CocoaPods 现在是 React Native 的 iOS 项目的一部分

*   WebView 和 NetInfo 不在 React 本地存储库中

*   “升级助手”是一个新的工具，使升级过程更加简单

*   由于“自动链接”,不再需要手动链接库
