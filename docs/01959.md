# 为什么要使用 spread 运算符

在阅读了前面的列表后，您可能会有这样的想法:*“但是 JavaScript 有函数来做所有这些事情…为什么我要使用 spread 操作符呢？”*请允许我向您介绍**永恒**:

> **出自牛津词典:**不变性——不随时间变化或者不能被改变。
