**本文的第一部分将涵盖什么**
对于本系列的第一篇文章，我决定将重点放在两个主题上，动画和排版。这是许多初级开发人员苦苦挣扎的两个领域，希望我能尽可能清楚地解释一切。

在我看来，CSS3 动画是网络行业的一次大革命，因为我们不再局限于使用 Javascript 和 jQuery 来制作网页动画。

现在让我们从基础开始，你可以在三个轴 X，Y 和 Z 上制作你的对象的动画，你可以放大，缩小，模糊，淡入淡出，灰度…但是要做到这一点，你必须学习 CSS 提供的大部分效果。
这是七个动画属性:

*动画名:动画名；
动画——时长:1s；
动画-计时-功能:线性；
动画-延时:0s；
动画-迭代-计数:无限；
动画-方向:交替；
动画-播放-状态:运行；*

**animation-name**:指定@keyframes 动画 duh 的名称。
**动画时长**:指定一个动画完成一个循环需要多少秒或毫秒。
**动画-计时-功能**:指定动画的速度曲线。
**动画延迟**:指定动画开始的时间。
**动画-迭代-计数**:指定动画应该播放的次数。
**动画方向**:指定动画是否在交替循环中反向播放。
**动画播放状态**:指定动画是运行还是暂停。

您可能会认为每次都必须编写所有这些内容会很痛苦，但幸运的是我们有一个替代语法，它是这样的:
*animation:animation-name 1s linear 0s infinite alternate；*

现在让我们看看奇迹真正发生的地方。确定动画及其所有属性后，是时候制作对象的动画了，制作动画的方式取决于动画元素有多少种状态。如果只有两个，可以这样做:

*@ key frames animation-name {
from {/*你的第一个状态*/；}
到{/* 你最后的状态*/；}
}*

但是如果你有更多的州，你可以这样做:

*@ key frames animation-name {
0% {/* animation*/；}
25%{/ *动画*/；}
50%{/ *动画*/；}
75%{/ *动画*/；}
100%{/ *动画*/；}
}*
