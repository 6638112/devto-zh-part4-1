# 为技术创始人销售

> 原文：<https://dev.to/sgallese/sidepact-2019-start-a-company-with-a-full-time-job-50bb>

[为技术创始人销售](https://www.sidepact.com/blog/2019/5/selling-for-technical-founders)。

技术创始人喜欢开发创新产品。然而，工程师出身的创始人将所有精力放在开发上是一个常见的错误。当一个创始人制造了一个没人想要的产品时，这可能会导致灾难性的后果。

在 Jason Mark 在 Sidepact 的演讲摘要中，有技术头脑的创始人将学习如何利用他们的技术能力来推动销售。
