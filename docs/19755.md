# 让我们回到网络广告的基础

> 原文:[https://dev . to/hmhrex/let-s-get-back-to-basics-with-online-advertising-1ol 0](https://dev.to/hmhrex/let-s-get-back-to-basics-with-online-advertising-1ol0)

如今广告业名声不好。许多提供数据作为广告产品的公司最近不仅受到了技术专家的关注，也受到了政府的关注。我们是如何走到这一步的，我们如何摆脱这一困境？随着广告拦截器的出现，在线广告还有未来吗？在没有大量收集数据的情况下，我们能在道德上向人们做广告吗？

### 现代广告如何帮助

现代在线广告服务承诺更好地定位你的广告，让你出现在正确的人面前，同时也防止你浪费金钱展示给错误的观众。每个印象和点击毕竟是要花钱的，你的产品是要改变人们的生活的，只是他们还不知道而已！

这似乎是一个三赢的局面。企业在正确的受众面前获得广告，消费者获得他们可能真正关心的广告，网站可以通过托管针对其受众的广告来保持运行。与过去相比，这是一个巨大的进步，当时你只是在一个网站上砸钱，并希望它与一些观众联系起来。

### [](#how-modern-advertising-hurts)现代广告有多伤人

只不过，这些广告平台开始展示它们丑陋的一面。

用户开始意识到他们喜欢访问的网站越来越慢。他们还注意到，他们达到电话服务数据上限的速度要快得多。然后，他们注意到，他们访问的下三个网站向他们展示了他们半小时前观看过的手表，但实际上并不在乎。

一开始大家只是把它看成是一种必要的恶。还有什么能为这些网站运行的服务器买单呢？因此，几年来，网站变得越来越慢，网站访问者的数据越来越多，人们慢慢对广告变得麻木。政府开始对曾经看似良性的数据感兴趣，并开始利用这些数据为自己谋利。

### [](#the-rise-of-ad-blocking)广告屏蔽的兴起

然后人们决定他们已经受够了，开始反击。他们意识到他们可以屏蔽这些广告。这节省了页面空间，使网站加载速度快得令人难以置信，并防止网站在数据库中收集他们并不真正想要的数据。

### [](#the-wrong-response-to-ad-blocking)对广告拦截的错误反应

网站所有者反击。他们建立了付费墙，除非你关闭广告拦截器或支付月费，否则不允许你欣赏内容。他们决定尝试避开广告拦截器。或者他们只是显示一个弹出窗口，试图向你解释他们的立场，并试图说服你在你的广告拦截器中为他们的网站破例。

[![](../Images/23ab6c1ad7bf3acdc378c50701ae820e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--BRnJZuis--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://magnetolabs.com/wp-content/uploads/2017/09/Business-Insider-Adblock-Example.jpg)

你不能责怪他们。他们不是跟踪你的人。他们所做的只是在网站上放一些额外的代码来赚钱，这样他们就可以继续为你创造高质量的内容。这甚至不是做广告的商家的错。他们只是想卖产品。

然而，网站所有者和企业没有意识到广告拦截器不是在与广告作斗争，而是在与 T2 跟踪和过度的数据收集作斗争。

### [](#the-real-problem)真题

广告服务向客户提供的真实产品是消费者数据。如果他们能获得每个互联网用户的正确数据，他们就能为他们的客户利用这些数据，并产生更多利润。

他们建立了巨大的仓库，里面装满了服务器，这些服务器不断地抓取少量信息，并将其存储在他们接触的每个互联网用户的“匿名”档案中。他们通过添加跟踪代码来利用网站所有者放在他们网站上的代码。他们还开始付钱给公司，让他们提供客户数据。你下载了什么应用程序，你如何使用你的数字设备，你的信用卡购买习惯，以及你与家人和朋友的日常互动，最终都在这些数据仓库中。你的数字自我被一个“匿名”个人资料复制，这一切都是为了定制你的广告体验。

企业知道这一点。他们并不是完全看不到它，但它已经成为道德和伦理上可以接受的规范，因为，嗯，其他人都在这么做。

然而，尽管普遍认为消费者数据对广告来说并不那么必要。

### [](#going-back-to-basics)回归基础

具体信息去哪里找？当你的车发出奇怪的声音时，你会去评论冰淇淋的博客吗？你会去“疲惫的妈妈们”脸书小组寻找电子游戏的最新消息吗？

考虑一下这个问题。

> 网站已经瞄准了市场。

脸书可以很容易地通过特定的群体类别而不是单一的用户数据来定位广告。谷歌只能通过像后起之秀 DuckDuckGo 这样的原始搜索词来定位广告。企业可以联系特定于他们市场的网站，直接与他们合作展示广告，反之亦然。

当我打出最后一段的时候，我意识到“啊哈”的时刻听起来是多么的愚蠢，因为我们已经在做这些事情了。为什么不直接切断追踪呢？

### [](#how-i-envision-it)我是如何设想的

假设我们经营一个关于饲养安哥拉兔的网站。我们需要广告商来维护我们的网站，所以我们需要弄清楚我们的市场是什么。

*   兔子需要吃东西。市场-兔子食物
*   兔子需要庇护所。市场-兔子窝
*   你知道有些人剪安哥拉兔的毛，然后把它们的毛纺成纱线吗？市场-针，纺羊毛的工具等。

现在我们去找那些公司，让他们相信我们的网站是为他们的产品做广告的绝佳场所。我们以合理的价格运行包含文本、小图片和链接的非侵入式广告，钱直接进入我们的口袋。广告加载速度很快，不会被广告拦截器拦截，因为你是自己托管的，除了浏览量和点击量之外不跟踪其他数据。

企业很高兴，因为你为他们的网站带来了流量和销售额，网站访问者很高兴，因为他们为他们的新兔子找到了一个完美的笼子，并且你可以在晚上睡觉，因为你知道你不会将访问者的用户数据发送到某个仓库。

### [](#its-already-begun)已经开始了

道德广告革命已经开始，但悄无声息，缓慢进行。几家公司一直致力于让道德广告变得更加正常。

埃里克·贝里经营着一家道德广告网络[代码基金](https://codefund.io)。他们在托管开源编程项目的网站上投放技术广告。这些广告是轻量级的，目标简单，不跟踪任何不必要的额外数据。

Read Docs 的广告总监 David Fischer 一直致力于在他们的网站上推行他们的[道德广告指导方针](https://docs.readthedocs.io/en/stable/advertising/ethical-advertising.html)。

与 Filezilla 合作的 Roberto Gallopini 也在他们的项目网站上执行了他们的[道德广告指南](https://filezilla-project.org/ethical_ads.php)。

在 RoutineHub，我自己的项目，我主持我自己的道德广告。

如果你是一家企业，找到你所在市场的网站，与他们合作，在那里投放广告。

如果你是一个广告网络，要更加注意如何使用用户数据来定向广告。

如果你经营一个网站，从你自己的服务器上托管广告，并在你的市场上寻找想要在你的网站上做广告的企业。

如果你是网络常客，继续使用广告拦截器。继续反对不必要的跟踪。继续公开反对那些反对创造一个更好、更安全的互联网的公司和网站。
