# 🎉2019 年 CSS 的状态🎉

> 原文:[https://dev.to/paco_ita/the-state-of-css-in-2019-5bn](https://dev.to/paco_ita/the-state-of-css-in-2019-5bn)

Sacha Greif 和[raphal benitet](https://twitter.com/benitteraphael)发起了一项调查，在全球范围内收集从事 CSS 工作的人们的直接经验。

目标是创建一个详细的报告，能够从不同的角度记录 CSS 的状态。不得不承认，这个结果真的让人印象深刻！！

[![congrats](../Images/0fa4b48a47b7f9c5ef5272d0e1952062.png)T2】](https://i.giphy.com/media/g9582DNuQppxC/giphy.gif)

强烈建议在线阅读[整篇报告](https://2019.stateofcss.com/)。

该文件显示了参与者的人口统计数据、工资或经验年限。它报告了哪些是最常用的工具或布局单元，人们对 CSS 学习曲线或最佳实践经验的看法(例如，避免无意义的类名，如。foo”)。毫无疑问，每个人都能从中获得有价值的见解，并能与世界各地的其他同事面对面交流。
