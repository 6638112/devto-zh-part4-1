## 反应-宇宙

Cosmos 扫描您的项目组件，使您能够:

1.  在道具、上下文和状态的任意组合下呈现组件
2.  模仿每一个外部依赖(例如 API 响应、本地存储等)
3.  在与正在运行的实例交互时，查看应用状态的实时变化

[https://github.com/react-cosmos/react-cosmos](https://github.com/react-cosmos/react-cosmos)
