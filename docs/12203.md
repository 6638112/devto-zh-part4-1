# 是什么让你喜欢上了戴夫？

> 原文:[https://dev.to/techhead404/what-got-you-into-dev-1155](https://dev.to/techhead404/what-got-you-into-dev-1155)

是什么让你进入你的领域？是你玩的游戏吗？也许是你在电视上看过的节目？或者高中时的职业工资单。对我来说，这是一个在线游戏。20 世纪 90 年代末 21 世纪初，游戏风靡江湖。那时候它超级简单，图形再简单不过了。我看到了人们的反应，以及在游戏和编程中无限的成长可能性。直到今天，如果我遇到障碍或者感到沮丧，我会登录并玩上几天。它总是提醒我，只要付出足够的努力和强烈的奉献精神，一个简单而基本的想法可以变成什么样。
