## [](#home)首页

大约一年前，我开始创建一个投资组合页面，但我不知道如何让它成为我的投资组合页面。上面有我以前的头像，我很高兴那堆乱七八糟的照片得到了更新。

我使用了我的旧网站的设计(这是在这个网站正在取代的有争议的 Squarespace 网站之前。)我决定我的主页将是一个英雄页面。

这个网站绝不是做了，可能需要我几个星期才能做到这一点。这里有更多的截图和我想让他们去哪里的描述。

## [](#how-i-am-building-it)我是如何建造它的？

现在，我使用简单的 HTML 和 CSS，因为我在处理一年前的文件。这是为了让我有一个好的结构。但是，我计划改用 GatsbyJS，因为我意识到我有一个作为页眉和页脚的组件。我已经创建了 Gatsby 页面，所以我可以将一些代码放在那里。
