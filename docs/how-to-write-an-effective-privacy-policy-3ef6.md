# 如何写一份有效的隐私政策？

> 原文：<https://dev.to/francoscarpa/how-to-write-an-effective-privacy-policy-3ef6>

上网冲浪，喜欢分析个人网站，我发现大多数网站都包含隐私政策。它基本上解释了网站收集、保存和阐述的数据，以及其他内容。每个网站都应该有一个隐私政策，可能还有一个 cookie 政策(如果它在用户的电脑上存储 cookie 的话)，但是我们如何写一个好的呢？我们需要回答哪些问题？我们需要报告哪些信息？让我知道你对它的想法，如果你的网站有隐私政策，请在评论中粘贴它的链接，以便其他开发者和新手可以看看真正的政策！
