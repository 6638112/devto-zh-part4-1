# 如何在 VS 代码中管理多个项目

> 原文：<https://dev.to/raisaugat/how-to-manage-multiple-project-in-vs-code-4be8>

我已经做了一年多的前端工程师，我知道管理多个项目很难。你永远不知道几个月前做的项目什么时候会出现问题。

因此，VScode 有一个很棒的扩展叫做[项目管理器](https://marketplace.visualstudio.com/items?itemName=alefragnani.project-manager)，而不是在你的驱动器上寻找项目文件夹。这对我在项目间快速切换帮助很大。
