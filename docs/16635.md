# 有松露和毛毛雨的 Vue Dapp 教程

> 原文：<https://dev.to/jklepatch/vue-dapp-tutorial-with-truffle-drizzle-2cfm>

[https://www.youtube.com/embed/XaVEZ1ucxac](https://www.youtube.com/embed/XaVEZ1ucxac)

在本[教程](https://youtu.be/XaVEZ1ucxac)中，我将向你展示如何在以太坊区块链上创建 Vue Dapp。

我们将使用毛毛雨，它是构建 Dapp 的前端框架。我们还将使用毛毛雨 vue 插件，它将 Vue 连接到毛毛雨，并提供一些现成的 Vue 组件来与以太坊和智能合约进行交互。
