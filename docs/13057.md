# 辅助功能介绍

> 原文：<https://dev.to/thewhitewulfy/introduction-to-accessibility-35mm>

可及性是一个你不断听到的术语，但可能并不完全理解。这部分是因为这个词本身可能有点令人困惑。词根 access 让这个概念看起来像是和密码联系在一起。如果用户可以“访问”您的站点，则可访问性被取消。要是这么简单就好了。

网页可访问性实际上是关于用户访问你的网站内容的能力，不管是否有身体或精神障碍。互联网的发明者蒂姆·伯纳斯·李爵士说得很好:

> “网络的力量在于它的普遍性。无论是否残疾，每个人都能获得是一个重要方面。”

一个真正无障碍的网站是包容每一个潜在用户的。它涵盖了所有形状、大小和永久的不同能力。失明的用户有自己的需求，断手的用户也一样。可访问性组之间也有重叠。经历脑震荡症状的用户可以受益于旨在使经历癫痫发作的用户更容易访问一段内容的功能。可访问性是一个多方面的话题。

有六个基本原则让网络对每个人都更好！

## 有正确的文字大小

互联网用户的视力水平各不相同(由于受伤、疾病或年龄因素)，拥有合适的字体很重要，这样每个人都可以轻松阅读你的网站。

浏览器的默认字体大小为`16px`。我们至少应该试着不低于这个水平。以`16px`为最小基数，向上构建文本层次。

最起码要用 em 单位设置你的字体大小，这样用户就可以放大你的网站来增加字体大小。

> 除了文本的标题和图像，文本可以在没有辅助技术的情况下调整 200%的大小，而不会丢失内容或功能。
> 
> *   w3.org

## 更好的色彩对比

互联网用户的视力水平各不相同，拥有良好的对比度很重要，这样每个人都能轻松阅读你的网站。我爸爸有痴呆。他不能像我一样看到所有的颜色，这是我在中学时发现的。最近我使用一个插件来检查颜色问题。稍后会详细介绍。

这就是为什么有一个推荐的指南只是为了防止这个问题。它要求，

> 文本和文本图像的视觉呈现具有至少 4.5∶1 的对比度。
> 
> *   w3.org

## 使文字内容易读易懂。

有些句子非常复杂，需要读者投入额外的注意力来理解句子的整体意思。很多人可能还是能理解的。但是，即使在拥有该主题专业知识的高学历用户中，也存在包括阅读障碍在内的残障人士。通过使文本更具可读性来适应这些用户也许是可能的。

以下是必需的指南:

> 当文本在删除专有名称和标题后需要比初中教育水平更高的阅读能力时，补充内容或不需要比初中教育水平更高的阅读能力的版本是可用的。
> 
> *   w3.org

简言之，

*   避免难懂和不常用的单词
*   写短句
*   在一个句子中不要使用两个以上的连词
*   项目符号在任何适用的地方都有帮助

## 标注表单控件

是的，我们知道有时我们会看到一个太小而无法点击的复选框。正常大小的复选框对你来说很容易点击。但这里需要理解的重要一点是，对于某些人来说，即使是正常大小的复选框也很难点击。特别是那些运动技能较弱的人，即能够以最大的确定性做预定运动的人。

这也是为什么有一个推荐的指南将一个`<label>`元素与`<input>`元素关联起来(一般与所有表单控件关联)。这使得标签也成为表单控件的可点击部分。因此您不必精确定位表单控件。

> 显示标识表单中控件的说明或标签，以便用户知道需要什么样的输入数据。
> 
> *   w3.org

请再次注意，这只是表单控件应该被正确标记的原因之一。它有更多的好处，我将在下一部分谈到

## 标签或说明

不幸的是，开发人员决定把它作为输入的占位符。一旦在输入字段中输入任何文本，占位符就会消失。

这里面有什么问题？现在，表单中的许多输入字段都是由浏览器自动填充的。在这种情况下，如果用户希望交叉检查填充的值是否合适，他们没有办法知道在特定的输入字段中真正需要的是什么！同样的问题也适用于记忆力差的用户，他开始填写表单，然后想在中途验证字段。因此，请始终标记您的输入字段，以便用户可以随时访问标签/说明。

> 显示标识表单中控件的说明或标签，以便用户知道需要什么样的输入数据。
> 
> *   w3.org

## 可调计时

> 确保尽可能给予残障用户足够的时间来与 Web 内容进行交互。失明、弱视、灵巧障碍和认知限制等残障人士可能需要更多时间来阅读内容或执行填写在线表单等功能。
> 
> *   w3.org

网站上充斥着警告、通知等内容，它们会显示一小段时间，然后就消失了。你可以做以下几件事来让它们变得更容易理解:

*   让他们待久一点。
*   显示在他们离开之前还剩多少时间。
*   悬停或聚焦时暂停他们的计时器。

在过去的几分钟里，你看了一些网页内容的例子。这些例子的动机有两个:

*   展示网站通常无法访问的几种方式。
*   让人们知道这样一个事实:你觉得对自己合适的东西，对其他人来说可能是不可及的。因此，要让大多数人都能访问我们的网站，我们能做的最好的事情就是——遵循推荐的指南。希望这对你来说是一次学习经历。如果是的话，请把它传给其他所有的开发者，设计师，营销人员，任何你认识的从事网站工作的人。
