# 让我们来谈谈现实生活中的薪水

总工资和净工资是两种截然不同的东西。

我的每一张薪水支票留给我的只是我总工资的 55-60%。加拿大的税收相当高。我也不在最高的税级。

我还必须支付其他费用，如联邦养老金计划、就业保险和其他不可协商的政府项目。即使我从来不用它们。但那是后话了。

让我们从我引用的文章中选取一些指标，这些指标来自 [Stack Overflow 的 2017 年开发者调查](https://insights.stackoverflow.com/survey/2017#work-_-salaries-by-geography)，并看看一些更现实的薪资计算。

# [](#realworld-salary-calculations)真实工资计算

2017 年，机器学习软件开发者在美国平均赚了 108，000 美元。

网络开发者赚了 9 万美元。

那就是一年 18，000 美元的差额。听起来很棒？

就拿我一般的扣款比例来说，在 ***较低的*** 现实端，大概是 40%左右。

18000 美元的 40%是 7200 美元。

因此，我的净收益约为 10，800 美元(假设我没有进入新的税级)。

听起来不怎么样，嗯？

在两周一次的工资支付期内，每次工资增加约 415 美元。所以每个月 830 美元。

这是很大一部分，但没有你可能预先估计的那么多。

取决于你住在哪里，这可能不会抵消生活成本太多，因为位置也在工资金额中发挥作用。

# [](#more-considerations)更多注意事项

现在，让我们包括专门从事机器学习与 web 开发的开销。

## [](#web-development)网页开发

我学习编程作为两年大学课程的一部分。许多 web 开发人员选择加入快速通道训练营，在几个月内完成，然后以合法的 web 开发人员工作结束。总的来说，这些项目每年要花费几千美元。

一旦你找到了一份开发人员的工作，一些训练营甚至会让你“以后”再付钱，从你未来的薪水中拿走一大块。

## [](#machine-learning)机器学习

机器学习呢？

嗯，你需要知道线性代数，应用统计学，大量的数学算法，深度学习和神经网络，自然语言处理等等。

我曾经得到一份工作邀请，在一家软件公司工作，这家公司在医学领域做了一些很棒的事情。我以“理由”拒绝了这个提议。但是我对他们所做的工作印象深刻。

我对他们的核心系统非常感兴趣，并就此问了很多问题。

他们创建了一个自然语言处理系统，让医疗专业人员和组织向其输入包含药物等信息的文本文件。该系统将使用机器学习来提取和分类文本中包含的各种药物和医疗信息，然后将它们与各自的成分联系起来，等等。相当复杂的东西。

网络开发者被允许接触这个系统吗？ ***没有。从来没有。从来没有。从来没有。*T3】**

它是由一位拥有博士学位的科学家建造和维护的。

***这就有麻烦了。*T3】**

真正的机器学习系统将服务于医疗、金融等领域。需要超级高素质的科学家。

换句话说，你需要接受大量的训练。许多...

平均年薪净增 10，800 美元，值得多年的高强度训练和潜在的数万美元学生债务吗？

我同意有些行业没有医疗行业那么重要。这些公司不一定会寻找受过同样训练的开发人员。但是，最低限度也需要一些高强度的数学训练。

# [](#thought-experiment-to-drive-this-home)思想实验开车回家

让我们来做一个小小的思维实验(希望它按照我想要的方式进行)😋).

想象一下，你正在为你在制造业的成长中的新公司雇用一名网络开发人员。贵公司正在为制造工厂引入一种新的管理机器的方法，这种方法每年将为他们节省大量资金。

你需要一个有分布式系统经验的后端开发人员。

你的候选人是:

a.7 年后端开发经验，其中 5 年在同一家公司从事分布式系统专业工作。

b.过去 2 年在分布式系统上工作，总共有 4 年后端 web 开发经验。但是，他已经在知名出版物上撰写了文章，这些出版物以行业专家对该主题的关注而闻名。也在一些知名的播客上发言，采访这一领域的专家。他/她还写了一本小书，帮助开发人员了解分布式系统设计。

你会选谁？

**我会选 b。**

为什么？

就他们的“职业经验”而言，候选人“a”更有经验。他们在分布式系统方面的专业经验是候选人“b”的两倍多。

酪就是这样。他们所有的专业知识都局限于他们工作的公司。

我们如何验证他们实际上做得很好？即使候选人“a”成功地构建了分布式系统，我们如何验证它是以一种有效的方式完成的，是安全的，等等？我们如何真正知道他们正在学习思考这些系统的任何新的更好的方法？

另一方面，候选人“b”将他/她自己置于公众团体的手中。不仅如此，他们还被描述成该领域的专家。并不是说他们在知识上一定是超级先进的，但他们是出类拔萃的。

这个人也写了一本书，针对初学者，可能没有候选人 a 的“纸上”经验。但是，我们可以肯定的是，它们已经过整个社会的“检验”。

也就是说，这个人 2 年的经验可能被认为相当于更多的经验，因为他们已经参与到社区中，并投入了一些认真的努力来掌握他们所写和谈论的领域。

并非所有的经历都是平等的。

# [](#leverage-points)杠杆分

此外，这位候选人有更多(我喜欢称之为)杠杆点。

杠杆点是各种不同但相关的成就、经验等。你可以用它来给潜在的雇主留下深刻印象。

总的来说，你拥有的越多，你留下的印象就越好。

例如，候选人“a”通常有一个优势:在一家私人公司构建分布式系统的专业经验。

然而，候选人 b 有更多:

*   分布式系统的专业经验(大约是其他候选人的一半)
*   在著名出版物 X、Y、Z 中出现
*   出现在著名的播客 X 中，该播客邀请了 A、B、C 等行业专家
*   写了一本关于这个主题的书，受到了社区的好评

这些杠杆点有一种“叠加”,有助于验证这个人需要展示的核心技能。

看，候选人“b”不是“专家”,因为他们和其他候选人有相同的工作经验。

然而，他们被认为是专家，因为他们所知道的已经被社区固化了。

我们相信这个人是可靠的，并且经过了社会调查。

最重要的是，当候选人“b”参加面试时——大多数面试官甚至不具备分布式系统的工作知识。但是，这位候选人会通过指出这些优势给他们留下深刻的印象。

杠杆点在薪资谈判中也很有帮助😊。

在一天结束的时候，这些优势会帮助你给你的潜在雇主留下一个更好的印象，那就是你在你的领域是真正的交易。
