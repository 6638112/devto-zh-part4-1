# 我的编码之旅更新

> 原文:[https://dev.to/jrm1715/my-coding-journey-update-fp1](https://dev.to/jrm1715/my-coding-journey-update-fp1)

这个月富有成效，但也很棘手。我已经完成了柯尔特·斯蒂尔网站开发者训练营 398 节课中的 257 节。就像在以前的博客中所说的。在过去一年左右的时间里，我一直断断续续地学习这门课程。我决定参加这个课程是为了学习如何构建全栈 web 应用，而不仅仅是学习典型的 HTML、CSS 和 Javascript。我当时的目标不是用我所学的知识找到一份工作。更多的是出于好奇。我越深入这门课程，就越喜欢构建完整的 web 应用程序。现在我的目标是找一份全职做这个的工作。我没有学位，我所有的经验都是在 IT 领域，侧重于硬件。我确实需要写剧本，我想这会对我的求职有所帮助。我计划完成这门课程，开发几个网络应用，建立我的作品集，至少每月一次在博客上记录我的进步，润色我的简历，然后开始找工作。

## [](#course-progress)课程进度

这个月，我在这门课程上取得了相当大的进步。这个课程向我介绍了 API 的使用，我认为这真的很有趣，并为我的投资组合中添加项目的新想法打开了许多大门。我构建了一些在课程中教授的小项目，其中一个是专门用于列出露营地的 Yelp 克隆，带有评论、评级和图像。我也开始用同样的概念做一个兼职项目，但是是一个摄影作品集。我用 Node 弄脏了我的手，专注于快递包裹。使用 Express 的 RESTful 路由。本课程将教您使用 MongoDB 进行数据保留。本课程还使用嵌入式 Javascript (EJS)在页面上显示从数据库中检索到的信息。

截至目前，我正在学习数据关联。这将允许我允许评论，并将它们与特定的用户、帖子或其他数据相关联。这些都是用 Mongoose 完成的，它是一个节点包，可以让我们与数据库中的数据进行交互。一旦我完成了这一部分，我将为每个 Yelp Camp 帖子实现完整的功能性评论。我对实现这一目标感到非常兴奋。

## [](#photography-portfolio-project)摄影作品集项目

在课程的某一点上，我们用 Express 结束了 RESTful 路由的学习，我后退了一步，决定进行一个项目，这个项目不是在课堂上构建的，而是使用了所有教授的内容。我决定这样做的原因是为了巩固我对 RESful 路由和使用 Express、MongoDB 和 Mongoose 的理解。完成这一部分后，我觉得自己对所学的一切都了解得很少。我从以前的学习经验中知道，通过实践，而不是按照老师教的去做，我会学得更好。我把摄影作为一种爱好，并认为建立自己的作品集来发布我的照片会很有趣。这个想法将有助于激励我继续努力并完成它。所以我开始研究它。

当我开始构建这个项目时，我注意到的第一件事是我记不住老师展示的大部分语法。我想解决这个问题。我觉得唯一能解决这个问题的方法就是重新看一遍讲座的部分内容，并参考以前的项目。一开始感觉是自己在骗自己。我决定在参考我以前在课程中教授的项目之前，先参考一下官方的 Express 和 Mongoose 文档。我开始一点一点、一点一点地建立网站。最终，我实现了我所学到的一切。我非常兴奋，觉得通过这样做我学到了更多，并且觉得这是巩固你所学知识的好方法。不要只是开发课程和教程中教授的应用程序，开发你自己的应用程序，即使你正在重新创建已经完成的东西。重要的是你学到了什么，而不是你建造了什么。

## [](#my-struggles)我的挣扎

说到学习。肯定会有挣扎。我自学了很多东西。我自学了烹饪，弹吉他，现在又在自学编码。所有这些事情都有特定的斗争。

我目前的工作是 IT 顾问。我的工作是排除故障、实施和改进客户的基础架构。我们的许多客户都面向消费者。这是一份我可能需要长时间工作的工作。如果客户的服务器出现故障。我们有责任让它们重新运转起来。这是我最大的挣扎。我的很多职责都是令人麻木和重复的。其他时候，我在解决需要耗费大量脑力的问题。当我头脑发热时，试图激励自己是很困难的。当我在工作的时候，我所能做的就是想着回家做我的项目或者继续我的课程。一旦我回到家，我就和妻子一起吃晚饭，看一会儿电视，有时我无法强迫自己起床。

当这样的日子发生时，我通常会休息一下。这也让我意识到保持健康的精神状态有多重要。像科技领域的大多数人一样，我也在焦虑中挣扎。如果我休息一下，我会觉得自己落后了，离得到一份开发人员的工作会更远。退一步，专注于放松，不要担心“如果”,这已经成为一个优先事项。致力于你的心理健康是必要的。这也包括身体健康。我的哲学是，如果你身体健康，它会帮助你变得心理健康。当然，如果你身体健康，并不能保证你心理健康，但这是有帮助的。这包括确保你补充水分，健康饮食，做一些体育活动。作为健康饮食的开始，我已经戒掉了咖啡因(主要是苏打水)并降低了糖的摄入量。我也试着每周至少去 3 天健身房。去健身房也很难保持一致。

我经历的其他困难是特定于节点包的。在我学习期间，我很难区分 Express 和 Mongoose 语法。我花了很多时间参考这两个包的文档，以了解我的代码库中什么是什么。我目前的理解是，Express 是为了一个处理路由的包，而 Mongoose 是专门用来与 MongoDB 数据库中的数据进行交互的。Mongoose 还有几个方面我没有完全理解，在项目中输入时需要思考和处理。适应这些包装需要时间。我可能只是学习 Node，而不会回到它的任何包，直到我熟悉 Node。

## [](#current-goals)当前目标

我目前的目标是完成这门课程。我还没有完成我上过的任何课程和辅导课。我知道这门课有很多东西要学，我也知道一旦我完成了，我就不是一个经验丰富的开发人员了。我知道我会更好地理解构建一个功能性的 web 应用程序需要什么。这是主要的焦点。

一旦我完成这门课程。我计划在 CSS 上花更多的时间。我仍然需要学习显示布局，比如 Flex、Grid 和 Float 布局。说到我对 CSS 的理解，我不想缺乏。一旦我觉得我很好地掌握了 CSS，我会继续学习更多关于 Javascript 的知识。当谈到 Javascript 时，我觉得我已经掌握了基本知识，但我认为我还很欠缺。我下载了一个 Javascript 测验应用程序来测试我目前的理解，有很多我觉得我不知道的，这个测验应用程序显示了这一点。一旦我完成课程，这将是我最大的两个关注点。在那之后，我打算投入 React 或另一个前端框架。

总而言之，到目前为止，我很享受我的学习经历。我希望我能用我所学的一切获得一份开发人员的工作。还有很多要学的，还有很多要做的，希望你们继续跟随我的旅程。
