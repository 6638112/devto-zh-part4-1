最后，在 2019 年 1 月，我通过 Ahmad Awais 的课程 [VSCode 超级用户](https://a.paddle.com/v2/click/16413/48895?link=1227)了解了 VS Code Editor

是的。！！现在，您可以通过编辑器播放、暂停和更改您的 Spotify 歌曲。
