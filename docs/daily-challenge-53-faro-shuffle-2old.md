# 每日挑战#53 -法罗洗牌

> 原文：<https://dev.to/thepracticaldev/daily-challenge-53-faro-shuffle-2old>

**挑战**
给法鲁洗牌一副扑克牌，就是把这副牌正好分成两半，然后完美地交织成牌，这样原来的顶牌和底牌就不变了。

写一个函数，接受一个偶数列表，faro 打乱索引。

**举例**
法鲁洗牌列表:`['ace', 'two', 'three', 'four', 'five', 'six']`
会给`['ace', 'four', 'two', 'five', 'three', 'six']`

祝你好运，编码快乐！
