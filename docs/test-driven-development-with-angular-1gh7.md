# 使用 Angular 的测试驱动开发

> 原文：<https://dev.to/learntocodez/test-driven-development-with-angular-1gh7>

在这个视频中，您将学习测试驱动开发的基础知识，同时构建一个简单的 Angular web 应用程序。

它将为您提供信心，将 TDD 用于测试繁重的环境，例如公司 web 应用程序开发。您将确切地知道从哪里开始，并且将有一个使用 TDD 从头构建整个特性的工作示例。

在整个课程中，您将构建一个具有一些简单功能的应用程序，对于每个功能，您将首先编写一个测试，通过添加代码使其通过，然后进行重构以满足标准。这个课程非常适合那些想要开始使用测试驱动开发及其所有好处，但却努力编写第一个失败测试的开发人员。
