# 初学者纠结于 JavaScript 的一个原因是

> 原文:[https://dev . to/Mohammed asker/the-one-reason-why-beginners-fight-in-JavaScript-1p2o](https://dev.to/mohammedasker/the-one-reason-why-beginners-struggle-in-javascript-1p2o)

我开始学习 JavaScript 已经一个月了。是的，我知道第一个月不算什么，我可能要花 6 个月甚至更长时间才能掌握 JavaScript 的概念。

我知道语法。我知道变量、函数、if 语句、循环等概念是什么，以及它们的作用。

好吧，那很好。但是问题来了:

●你知道什么时候应用，什么时候不应用吗？你知道你要用它做什么吗？
●你知道构建你的项目需要哪些工具或特性吗？

没有吗？那你就苦于缺乏语境。

这意味着你不知道如何利用从教程中学到的知识，也不太可能去实践它，因此你会把它们全忘了。

即使你每天都练习，你也很难建立自己的项目，而所谓的简单项目，如待办事项、幻灯片和计算器，都出奇的难。

你可能会认为编程不适合你，会放弃成为软件工程师的梦想。

现在你明白了问题是什么，是时候找到解决方案了。这不是我最初的想法，直到最近，我一直停留在学习循环中——直到我读了我在下面的链接中提供的文章和指南。

终于找到了一条可能是新手突破时刻的路径。换句话说，这是一条让你从初级程序员成为中级程序员的道路。

你所要做的就是改变学习 JavaScript 的方法。我不知道你学 HTML 和 CSS 都用过什么方法，但是对于 JavaScript 不一定管用。

而不是观看或阅读教程，然后马上投入项目；首先，你需要完全理解 JavaScript 的概念。

没有它，你几乎不可能自己建立它，而指导性项目对建立记忆没有帮助，你会觉得你没有学到东西。

有八个概念你应该学习，因为它们非常实用，可以立即用于构建你的小项目。它们是:

1.  变量
2.  功能
3.  InnerHTML
4.  getElementById
5.  样式和类名
6.  查询选择器和查询选择器全部
7.  If 和 else 语句
8.  对象和数组
9.  onclick

一次学一件事。不用急。花点时间去真正理解它是如何工作的，以及为什么它们以某种方式工作。

理解这些特性的最有效的方法是构建一个任务或迷你项目。例如，当您单击按钮时，使用功能更改段落的颜色。

您的主要目标是尽可能多地花时间编写 JavaScript 代码，同时尽可能少地使用 HTML 和 CSS。(我知道，练习它们很重要，但关键是你在拖延自己提高 Javascript 技能的时间)。

一旦你理解了每一个特性是如何工作的，你就会开始有一些想法，知道你可以用哪些小项目来构建。

因为你花了时间去理解 JavaScript 的基本特性，你更有可能完成这个项目。

最后，请不要使用框架或库。你现在不需要它们，作为初学者，你唯一的工作就是提高 Javascript 技能，然后你就可以进入你选择的任何框架。

此外，了解普通的 Javascript 将使使用框架变得更容易，因此花大量精力学习它是完全值得的。

希望这篇文章对你有帮助，祝你编码愉快！
