# 你如何以及为什么使用 GitHub 项目？

> 原文：<https://dev.to/seanballais/how-and-why-do-you-use-github-projects-1d9b>

我习惯使用 Trello 进行项目管理，现在我正在为一个新项目使用它。我昨天刚刚遇到 GitHub 项目，我想知道它的用例是什么，或者相对于 Trello 有什么优势(除了它和你的 repos 在同一个屋檐下)。

对我来说，我看不出它比特雷罗有什么明显的优势。我看到的唯一涉及到 GitHub 项目的场景是当您使用 Trello 进行一般项目管理，将开发任务委托给 GitHub 项目时。然而，我发现这是有问题的，因为非开发人员不会立即知道开发过程中遇到的进展和问题，而开发人员也不能立即知道非开发相关任务的进展和问题。您可以将 GitHub 项目中的项目复制到 Trello，但是这意味着会有项目的重复，并且需要额外的工作，除非两者之间有集成。

我想知道你对 GitHub 项目的想法，以及你如何和为什么使用它。
