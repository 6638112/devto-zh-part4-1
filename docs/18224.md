网页由不同的元素组成，包括文本、图像、媒体文件、脚本等等。因此，当您单击网页时，浏览器会向网站发出一个 HTTP 请求，请求加载所有这些元素。现在，这里要注意的一点是，不是每个元素都是相同的，因此需要不同的时间来加载。图像比文本内容的字节重，因此加载需要时间。通常，每个元素都试图同时加载，这最终会延迟整个网页的加载。按照目前的框架延迟加载网页是一个大禁忌
