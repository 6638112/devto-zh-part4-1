# 狗不是狗工作室

> 原文:[https://dev.to/azure/the-dog-not-a-dog-workshop-2e5f](https://dev.to/azure/the-dog-not-a-dog-workshop-2e5f)

我最近在[hack](https://www.tohacks.ca/)举办了一个研讨会，以开始使用 Azure。我们的目标是尝试不同的 Azure 服务，看看我们如何使用无服务器功能和人工智能来增强现有的网站。

*(法语[)](http://www.cloudenfrancais.com/posts/L-Atelier-workshop-Dog-Not-a-Dog.html)*

[![](../Images/bbbaf7da3da88401c60aa529a74068b2.png)T2】](https://1.bp.blogspot.com/-dyloHAPv0rI/XRKJ4r_m0MI/AAAAAAAA9W8/mmUL6FJKO24leA6eAta1zk_B8BMzFH_-QCLcBGAs/s1600/workshopHeader.png)

在此研讨会期间，GitHub 会自动部署一个网站。然后通过添加一个 Azure 功能，并使用 Azure Cognitive Services 的 Vision API，最终的解决方案能够检测到上传的图片是不是狗，并保持我们的图像文件夹“干净”。我们称这个应用程序为*自动非狗应用程序*。

代码的逐步说明可以在[GitHub-Not-a-Dog-Workshop](https://github.com/FBoucher/Not-a-Dog-Workshop)上找到。研讨会可以在大约 45-60 分钟内完成。
