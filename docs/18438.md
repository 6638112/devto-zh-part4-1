# [](#identifiers-)**标识符:**

*标识符*可以是任何名称，可以是变量名、类名或方法名:

[![Alt text of image](../Images/a06e92af1a61e4e9c696df4cd0e43368.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--GsvpgZCM--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://1.bp.blogspot.com/-PcZXz4J8UiI/XQVA7sGUNKI/AAAAAAAAIQY/nys9wzlEUjQGyw9dqJnd-dl-dJNWdFbLgCLcBGAs/s1600/3.png)

**标识符规则:**

**1-不能使用空格:**

int emp no = 10-> int EMP _ no = 10；

**2-只能使用下划线(_)和美元($):**

int emp _ no = 10

**3-不能使用保留字:**

int public = 10-> int a = 10；

**4-整数值不能用在第一个位置，但可以用在第一个字符之后:**

int 1abc = 10-> int a1bc = 10；int abc1 = 10

# [](#naming-conventions-)**命名约定:**

**变量- >我的，我的 _java，我的 _java_demo**

**方法- > my()，myJava()，my java demo()**

**类/接口- > My，MyJava，MyJavaDemo**
