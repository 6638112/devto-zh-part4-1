# 炮塔射击游戏

> 原文:[https://dev.to/takaneichinose/turret-shooting-game-li7](https://dev.to/takaneichinose/turret-shooting-game-li7)

简单描述:

射击所有的僵尸！(它们可能会停止产卵)

这是一款射击游戏，炮塔人物不动。

只是防止僵尸靠近你。你会失去你的生命点。

描述:

丧尸们越来越饿了，它们甚至想吃掉炮塔里面的大脑(说真的？wt*？).这个炮塔有一个很棒的大脑，因为他是一个人工智能(应该是，但他被你控制了)。

他们需要 3 个僵尸才能消灭 Turry 巨大的大脑。

但遗憾的是，因为他是炮塔，所以根本动不了。但至少，他可以旋转并发射火焰(哦，那可能会烧死僵尸)。

僵尸的数量随着他们的产卵而增加，但是，有一个产卵的最大值。它们的产卵位置是随机的，所以要小心突然袭击！他可能就在你身后产卵。

控制:
按“左”键逆时针转动
按“右”键顺时针转动
按“空格”键开火

这里的所有资源都是我制作的，我使用了“CSS 艺术”或“CSS '盒子-阴影'艺术”来制作它们。

我用 ReactJS 做游戏的逻辑/算法。
