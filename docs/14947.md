# [](#game-development)游戏开发

[![3D multiplayer browser game](../Images/5c3d3ebc80f7ab32aa12b9f0f4a525c2.png "The latest iteration of the commercial 3D multiplayer browser game.")T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--f0kcoZvW--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/pw2q12sf6s9mc0awwspu.gif)

我继续开发在今年上一季度( [Q1 2019](https://webdva.github.io/q1-2019-report/) )启动的商业 3D 多人浏览器游戏。游戏产品创作的一些方面:

*   我为一个 3D 游戏实现了健康栏，这是一个值得注意的成就，因为显示一个二维平面对象的技术细节总是面对三维空间内的观众/玩家。这在一定程度上是由于值得称赞的 Babylon.js 游戏开发框架(用于制作游戏)的性质以及为其使用提供的丰富资源。

*   类似地，我实现了在 3D 多人游戏中，在玩家头上显示可见的玩家名字(为其他玩家识别他们)，这是 3D 多人游戏的常见要求。此外，玩家的名字是在服务器端随机生成的。

*   我实现了将 3D 模型附加到其他 3D 模型上，比如让角色以某种方式拿着武器。

*   我在为玩家角色的攻击使用程序动画方面获得了经验。

*   我原型化并实现了角色扮演游戏(RPG)类，具有基于统计的伤害计算和实时 3D 多人动作 RPG 战斗。

## [](#cancellation)注销

多个问题困扰着 3D 多人浏览器游戏的开发。实际上缺乏企业家和客户的契合:尽管我对玩家和游戏玩家有很大的承诺，但我无法找到一个对我们双方都有利的解决方案，我是企业家，玩家是客户。由于计划中的货币化策略的道德含义与我的正直和我对玩家的同情相冲突，所以提议和萌芽中的免费游戏产品的货币化值得深思。

因此，3D 多人浏览器游戏已经被放弃，取而代之的是更好的解决方案。请注意，出现的任何技术问题都不会对项目的成功构成实际风险。

然而，不要担心。从 3D 多人浏览器游戏的开发中获得的数据、知识、经验教训、经验、文档和技术正在被转换为新努力的成功的实用工具。此外，请注意，我有可能发布甚至开源游戏(其相关文档必须与其共享)，目的是帮助促进免费知识和技术的传播，以便其他人可以受益。但是没有必要这样做。

我最近发现，部分是由于我在上一个季度( [Q1 2019](https://webdva.github.io/q1-2019-report/) )在欧拉角和四元数方面的游戏开发经验，抽象或纯数学的效用，我现在正在从事一项新的倡议，涉及我学习和研究数学，甚至抽象数学，这通常是理论背后的理论，从中找到世界的秘密，神秘和神秘的艺术——一个成功创业并取得收入的想法，尽管收入很少。顺便说一下，这一切必须在不到六个月的时间内完成，在 2019 年底之前，在下一个十年开始之前。

我相信这样一个创新但有风险的过程将会成功，它最终会为我带来好的结果，我永远致力于确保有利的结果发生，对我预想的成功负责并指挥它的到来。

# [](#the-future)未来

随着当前基础设施的建设和建立——因为这是我作为一个企业家一直在做的事情，包括我所有的“成功”和“失败”，“经验教训”等等。我要用我所拥有的这种能力去完成伟大的事业。

我的抱负之一是创造全新的产业和工作岗位，让所有人都富裕起来。这就是我正在成为的那种企业家。

因此，我的繁荣计划必须成功，因为它将使我能够开始执行这样一项重大的行动，正如昔日阿波罗计划的成功证明了美国及其自由和民主制度比其对手更有能力。我也可以收敛到一个空前高度的非支配解，因为改变世界的努力可以被认为本质上是多目标最优化问题。

我只需要做正确的事情，那就是通过勇气、正直和力量获得成功。

我不寻求对任何人或任何事的不体谅的破坏。我只希望为世界和我自己设计一个繁荣的未来，以摆脱被支配的社会经济地位。为了完成这一职责，我准备好扮演这个世界要求我扮演的任何角色，完成任何任务。

看来，我必须扮演一个创新者的角色，在没有榜样可循或有用的文献可参考的情况下，穿越以前未探索的空间，没有任何其他人的帮助，因为没有其他人能够帮助我，因为没有其他人对这样的空间有经验，知道什么是正确的事情。因此，我必须设想-我必须创造-未来的我将作为榜样，我必须创造我自己的成功文学。有了这些出于我自己意愿和设计的义务，我实现了自我克服。这就是我所发现的自己，面对如此前所未有的挑战，同样需要非凡的勇气和力量，因为过去的我在开始这段创业之旅时，无疑没有预料到这样的责任和深刻。

尽管如此，我正在创业，这是对我未来的一项重大投资，所以我会全力以赴让它成功。

为了未来。
