# 介绍云视频编辑 Shotstack

> 原文:[https://dev . to/shot stack/introducing-shot stack-the-cloud-video-editing-API-420m](https://dev.to/shotstack/introducing-shotstack-the-cloud-video-editing-api-420m)

[Shotstack](https://shotstack.io/) 是一个基于云的 API，使用简单的 JSON 格式和渲染引擎创建和编辑视频，可以同时渲染 1000 个视频。

传统上，基于服务器的编程视频编辑的主流技术是开源工具 FFmpeg。FFmpeg 是一个命令行实用程序，执行视频处理任务，如代码转换、调整大小、修剪和拼接剪辑，以及应用一系列滤镜和效果。您可以在您的应用程序中使用它，方法是直接执行它，或者通过包装器库，如 PHP-FFmpeg 或 fluent-ffmpeg for NodeJS。

对于简单的单个视频来说，这是一个很好的方法，但是当您想要进行更复杂的编辑时，命令可能会变得复杂，功能也会受到限制。

当你开始编辑任何比例的视频时，也有一些重要的考虑要考虑；根据输入和输出文件的长度和分辨率，即使是一个简单的编辑也会在几秒钟甚至几分钟内消耗 100%的 CPU。

通常，您可以结合使用队列、负载平衡器、多个服务器、实例或容器以及自动伸缩来解决这个问题。所有这些都需要专业知识、努力和时间来设置和维护。

我创建了 Shotstack 平台来消除所有这些复杂性，让您专注于创建视频。API 允许你使用一种简单易懂的 JSON 格式来指定你的视频编辑是如何安排的。

我试图复制一种类似于 iMovie 或 Adobe Premiere 等非线性视频编辑器的格式，其中包含时间轴、轨道/层、处理图像、标题和视频的剪辑，以及许多内置的效果、过渡和滤镜，可以通过一行代码添加。

该平台的另一个组件是服务器端渲染引擎——只需将您的编辑发布到渲染端点，它就会负责对您的渲染作业进行排队，并按比例增加一个新实例来处理和渲染。您的图像和视频资产只需存储在一个可访问的网址，它们将被下载到编辑。在一小段时间后，它会吐出一个最终渲染的 mp4 视频或 gif 文件供下载。

### 你可能会问，我为什么要做这个？

Shotstack 是多年前作为一个兼职项目开始的，目标是使用机器学习和视觉识别来自动编辑 GoPro 视频。事实证明，人工智能方面的事情比我预期的更复杂，但我开发了后端 API 和渲染引擎，并认为开发人员可以使用它来创建他们自己的自动化、个性化的大规模视频应用程序。

我一直在与越来越多的在线视频服务交流，有些服务每月制作数百万个视频，它们正在建立自己的平台。他们都在孤立地构建类似的东西，花几个月的时间重新发明轮子，我相信这应该更简单——开发人员应该能够在几分钟内而不是几个月内完成设置并创建他们的第一个视频，从而将他们解放出来，专注于他们的应用程序，而不是基础设施和维护。

我想和你分享这个 API，看看你能创造出什么。我的目标是创建一个类似 WordPress 的应用程序、模块和视频模板的生态系统，开发者可以将它们组合在一起，创建新的基于视频的项目、应用程序和服务。
