# Docker:请求在等待连接时被取消(客户端。等待标题时超时)

> 原文:[https://dev . to/Michael brooks/docker-request-canceled-while-waiting-for-connection-client-time out-exceeded-while-waiting-headers-b1i](https://dev.to/michaelbrooks/docker-request-canceled-while-waiting-for-connection-client-timeout-exceeded-while-awaiting-headers-b1i)

这个错误已经让很多人被他们在 GitHub 上的[问题的外观难倒了。我认为错误源于通过他们的主登陆页面下载 Docker，我尝试了很多来解决这个问题，但没有成功。这包括将 DNS 切换到 8.8.8.8，这在过去已经被建议过了，并且奏效了。](https://github.com/docker/kitematic/issues)

我对这个问题的回答是不要使用主登陆页面来安装 Docker，相反，你想从这里下载它，然后一切都应该正常工作。
