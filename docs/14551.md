# 预测编程的未来

> 原文:[https://dev . to/Adam _ cyclones/predict-the-future-of-programming-2 cmd](https://dev.to/adam_cyclones/predict-the-future-of-programming-2cmd)

让我们玩一个小游戏，我将试着预测代码的未来，

我相信在 50 - 100 年后:

1.  代码将不仅仅是手工编写的，大多数语言将与人工智能紧密交织在一起，就像 linter 或 steroids，这种体验将有点像与更聪明的人配对编程，他通过代码前进的方向知道你的意图。

2.  代码会自我修复

3.  量子计算机将和智能手机一样大，人们不会犹豫。

4.  没有光学显示器。从机器到大脑，谁需要屏幕。

5.  编程将主要通过跳舞来实现。(好吧，这是一门艺术，但我做得太过火了)💃🕺

我说的对吗？你对编程的未来有什么看法？
