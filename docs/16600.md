图像为我们的工作增添活力有着悠久的历史。现代网站的媒体越来越丰富，万维网正在迅速成为一个可视化的网络。

我们已经形成了一个普遍的习惯，将图像倾倒到我们的项目中，而这正是问题的根源。在这样做的时候，我们忽略了优化，这是很重要的，并会考虑到我们正在处理的图像的尺寸，大小，格式和质量。

让我们看看优化图片的各种方法。

## [](#1-use-the-right-format)1。使用正确的格式

一个常见的错误是选择一种图像格式并在我们的项目中使用，或者仅仅使用默认格式的图像。这在很多层面上都是非常错误的。图像格式通常有特定的用途和它们的亮点。下面是一些常见的，以及何时使用它们最好；

**JPEG/JPG** 最适用于照片和复杂图像。它们通常有很多颜色，JPEG 格式已经针对这种情况进行了优化。

GIF 非常常用于动画和其他伟大的用途。例如，它最适合用于标志和图标。基本上，低姿态的图像有非常少量的颜色。

当您需要具有透明度的图像时，PNG 必不可少。它们通常比其他格式大。通常用于标志和图标，在这些情况下应该避免使用它们来代替替代物。

SVG 或可缩放矢量图形最近在网络上获得了很大的吸引力，因为它能够在不损失质量的情况下进行缩放。它仍然和设计的一样。SVG 可以用来代替 PNG，尤其是在网络上，并且通常具有较小的文件大小。所以你一定要考虑。

所以，我希望上面的指南能帮助你决定在下一个项目中使用什么样的图像。

## [](#2-select-the-optimum-quality)2。选择最佳质量

你形象最好的品质，不一定是最适合你的品质。你可能只是把许多不必要的像素和细节塞进了不需要的地方。你真的想放松一点图像的质量。这在 web 项目中尤其重要，因为这样做可以显著减小文件大小并缩短加载时间。

优化网页图像质量的一个快速方法是使用 Photoshop 中的“存储为网页格式”选项。如果你没有拷贝，Photoshop 还有其他的替代品。特别是， [GIMP](http://www.gimp.org) 是 Photoshop 的开源替代品，可以在主要的操作系统平台上使用。

## [](#3-resize-to-fit)3。调整大小以适合

你有一张全高清照片，看起来很棒，但你想把它扔进一个 480 像素宽的窗口。在 web 上，你可能会把它强行放入窗口，并使用 CSS 或 JavaScript 来确保它不会溢出。当不在网页上时，你只想拖动边角，直到它适合那个窗口。好吧，如果你有这个习惯，你可能会想要戒掉它。

最佳做法是建议你将图片放回编辑工具，并将其尺寸调整为 480 像素。这在存储大小方面为您节省了成本。像 MS Paint 这样简单易用的应用程序已经可以免费做到这一点了。

## [](#4-do-extra-work-on-thumbnails)4。对缩略图做额外的工作

缩略图比许多人想象的更难处理。这里特别需要许多优化技术。

首先，一定要确保你的缩略图是为适应窗口而创建的，而不是强制的。

然后永远记住，它只是一个缩略图，并不一定是最好的质量。足以让用户明白。

现在对于文件格式，你可能想要 GIF，如上所述。它可以为你节省大量的空间，特别是在目录上，你可以同时展示几十或几百个缩略图。这可以节省你在网上的下载时间。

## [](#5-name-your-images-right)5。正确命名您的图像

是的，我们正在谈论图像优化，但这并不相关。命名你的图片给爬虫提供更多的信息，换句话说，它通过给爬虫提供更多的信息来改善你的 SEO。这个提示在网络上特别有用。

因此，基于上述原因，将你的图片命名为*sunflower-juice.jpg*比命名为【image.jpg】T2 要好，这样可以提高你的 SEO 排名。

## [](#6-plan-ahead)6。提前计划

项目团队犯的一个常见错误是将与图像相关的决策和优化推迟到项目的后期。无论你的项目是什么，最好从一开始就知道。试着理解什么类型的图像最适合这个项目，尤其是当你试图考虑文件的大小、质量和尺寸的时候。这将有助于节省你很多潜在的时间成本。

优化的图像对于媒体丰富的新闻网站和博客、包含用户生成内容的社交网络以及电子商务中使用的动态媒体至关重要。当图像优化做得正确时，图像密集型项目可以看起来很棒并且加载非常快。我们希望本文提供的技巧能够帮助您提高视觉演示的质量，同时更好地服务用户，确保他们保持参与和满意。
