# 这种咨询服务存在吗？它有名字吗？

> 原文：<https://dev.to/peter/does-this-sort-of-consulting-service-exist-does-it-have-a-name-4fhe>

我的一个朋友正在他的公司领导一项重大技术革新的“搜索”过程。基本上，他们是一家大型企业，正在寻求投标，因为他们希望实现网站、内容管理系统、移动应用等的现代化。

我的理解是，他们会去一些专业和更一般的技术咨询公司和开发公司，并通过与每家公司的“范围界定过程”来收集投标。

虽然我很乐意作为一个更有见识的倡导者来指导他们完成这个过程，但是我既没有时间也没有足够的经验来提供最好的建议。

有没有一种服务可以让客户雇佣一个独立的顾问在整个搜索过程中充当他们的联络人和代言人？此人不会被分配执行项目的任务；更确切地说，他们会帮助客户问正确的问题，在重大决策上给他们建议，帮助解构提案，等等。

如果有，这种服务有名字吗？
