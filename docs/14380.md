# 将您的开发人员博客文章发布到 diff.blog

> 原文：<https://dev.to/hackerkid/publishing-your-dev-to-blog-posts-to-diff-blog-4n07>

diff.blog 是一个开发者和工程博客的聚合器。在我很难找到好的开发者博客来关注之后，我建立了它。diff.blog 使得在互联网上寻找和关注令人惊奇的开发者博客变得非常容易。

如果你还没有把你的 dev.to blog 加入到 diff.blog 中，现在是最好的时机。将您的博客添加到 diff.blog 非常简单，几乎不需要一分钟。

*   转到 [https://diff.blog](https://diff.blog) 并创建您的帐户

*   打开[https://diff.blog/account/settings/blog/](https://diff.blog/account/settings/blog/)进入博客设置

*   现在在输入框中输入这个 url。
    `https://dev.to/feed/<username>`
    确保用您的开发者用户名替换您的`<username>`

[![](img/6970f5cd370c1a076932115606a5ee42.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--7pYC6f_v--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/fk7a7q3n2nqvsajaoq7y.png)

*   最后点击保存按钮。

🎉恭喜你。您已将您的博客添加到 [https://diff.blog](https://diff.blog) 。当你在 dev.to 上发布新的博客文章时，它们会出现在 diff.blog 上。如果你有任何问题，请在评论区问我。要了解更多关于 diff.blog 的信息，请访问 https://diff.blog/FAQ/
