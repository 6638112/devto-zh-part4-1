# 7 条决策规则

**斯凯恩公司商业产品主管[丹尼斯·普希金](https://www.facebook.com/densmr)**

# 如何教人们做出正确的决定，毫无压力地委派重要的业务任务。

每位经理都必须承担一项棘手的任务，即教导员工做出正确的决定，做出对企业有利的决定。随着我的团队不断壮大，我也开始意识到我必须委派责任。我不喜欢被微观管理——所以我不会微观管理我的团队。我的工作是领导大家，不是当瓶颈。幸运的是，Skyeng 完全是基于原则的决策；这使得每年 200%的增长成为可能。我只需要将这些公司原则或规则应用到我的商业产品领域。我就在这篇文章里说说我的规则。

# [](#rule-1-estimate-the-monetary-value)规则一。估计货币价值

任何企业经营都是为了赚钱。评估一个决策的货币价值，看它是否有利可图。金钱是一种通用的衡量单位，它有助于比较决策，区分项目的优先次序，并确保您做出了最佳决策。我相信你可以估算任何东西的货币价值——这并不像听起来那么难。你必须经常问自己，“我会获利吗？值得花费的资源吗？”通过使用这条规则，我们不会做出不合理的决定。一个决定越困难，就越需要证明其合理性。最好花额外的时间去思考，带着“为什么不”的想法采取鲁莽的行动

# [](#rule-2-value-the-goal-over-the-process)规则二。重视目标胜于过程

我们的团队如何实现目标对我们来说并不重要。我们在乎的是结果，即使是通过非常规的方式取得的结果。它给予我们的员工思考和创新的自由。通常最大的困难导致最大的突破。我们的团队可以甚至必须重新创建已建立的流程，并使它们变得更好。如果有人知道如何更快更有效地实现目标，他们必须倡导变革，拆除旧流程，建立新流程。然而，保持平衡很重要。假设我们要降低获客成本。我们可以解雇整个销售部门，这样可以降低成本。但也会削减转化率。较低的转换率弊大于利。我们将在技术上实现我们的目标，但公司不会从中受益。这就是为什么规则#2 和规则#1 一起使用，并估计一个变化的货币价值是很重要的。

# [](#rule-3-any-decision-is-better-than-none)规则三。任何决定都比没有好

2017 年，我们对网站首页进行了全面改造。它花了两个月的时间和多达六个利益相关者。除此之外，还有许多感兴趣的人想给些建议。显然，我们不能考虑和使用所有的建议，因为整个过程将永远持续下去。我们没有时间做那件事。这时，我们想出了这条规则以及实施新流程的规则，如果它有助于实现目标的话(见规则 2)。我们起草了一份草案并提交讨论。当人们给出他们的想法时，我们已经在做下一个版本，没有考虑当前的评论。我们用一个版本的滞后实现了修订，或者根本没有实现它们。我们的目标不是考虑所有的评论，我们的目标是做一个好的首页。对于真正重大的决定，我们休息了一下，说，“好吧，伙计们，如果你们想在下一个版本中看到你们的修改，你们有 24 小时的时间来评论。”由于这个原因，只有真正对这个项目感兴趣的人才能做决定。无论参与者的立场如何，我们都删除了所有不必要的交流，并在截止日期前制作了一个转化率明显更高的新首页。

科学告诉我们，搬家就是生活。我求你在工作中生活。不应该有坐在办公桌前不知所措的时候。这就导致了拖延症。外部或内部原因都可能导致停止。如果员工犹豫该做什么决定，最好做出决定，获得反馈并再试一次。迭代的方法让所有的疑问都消失了。如果任何人——我、你的经理、你在另一个团队的同事——需要超过 24 小时来做决定，你可以自己做决定，无需批准。当事人有足够的时间；如果他们不使用它，它对他们来说就没那么重要了。

如果你做了错误的决定怎么办？你一定会的。所以我们想出了下一条规则。

# [](#rule-4-make-more-mistakes-and-learn-from-them)规则四。犯更多的错误并从中吸取教训

这听起来可能很奇怪，但犯错是至关重要的。我总是鼓励我的团队犯错误，因为这是他们获得经验的途径。我就是这么学的。有时我看到我团队中的某人将要犯错误，我没有警告他们，让他们犯这个错误。下一次他们不会等我来指导他们，他们会相信自己的知识和直觉。这样员工变得更加独立，做出更好的决定。但是有时候事实证明员工是对的，我是错的！这是我最喜欢的，意识到我雇佣了一个比我更有能力的人。

然而，只有在对每个错误进行彻底分析，得出结论并修改规则和原则的情况下，犯错误才能导致快速增长和更好的结果。我一直密切关注着那件事。

# [](#rule-5-take-responsibility-for-your-work)规则五。对你的工作负责

自由越大，责任越大。员工可以决定他们是否想要管理、决策和承担风险，或者做他们的工作，让我承担所有风险。两个位置都可以，是喜好问题。每个员工都很重要。但如果有人选择冒险，然后一次又一次地犯同样的错误(这意味着他或她没有吸取教训)，他或她就会被解雇。如果你没有达到目标，你就会被解雇。如果你不遵守原则，你就会被解雇。如果你拖延，你就拿不重要的项目。如果你赚到了利润，你会成长得很快，并从事最有趣和最有创意的项目。职业提升可以带你走任何一条路；你可以很快上升或下降，这取决于你为公司赚取了多少利润。这条规则适用于公司的每个人，从专家到联合创始人。

# [](#rule-6-make-decisions-openly)规则六。公开做决定

任何时候。如果你在私聊中做了决定，那就是一个糟糕的决定。诚实公开地分享信息，尤其是当你的决定有风险时。风险越大，越需要分担。这有助于你在犯错误之前就获得反馈(是的，尽管有第五条规则，还是不要犯错误为好)。你的同事可以看到你计划中被忽略的缺陷。当你独自工作或在一个小团队中工作时，很容易与现实脱节。但是当你分享信息时，你可以从一个更有经验或者更细心的同事那里得到提醒，比你自己发现错误要快得多。

# [](#rule-7-change-rules-if-it-means-better-results)规则七。改变规则，如果这意味着更好的结果

每个新员工都必须学习并遵守我们的规则。但它们并不是一成不变的。规则的存在是为了帮助你做出更好的决定，而不是限制你的自由。如果你遵守规则，得到一个否定的结果，那就太好了！希望你和团队分享一下。它给你一个机会让你周围的世界变得稍微好一点，即使这只是一个微小的规则。不仅人需要不断成长变化；决策规则也需要发展和改变。

# [](#the-rule-of-no-control)没有控制的规则

当你以光速旅行时，很难控制一切。有时我们的测试服务器会失败。根据规则#3，我们不会等待它们被修复，也不会发布没有测试的项目。多亏了规则 2，我们可以实现这样一个有风险的过程。当然，它会导致系统失败，我们根据规则 4 从中学习，我们不害怕这些失败，因为我们已经根据规则 1 估计了它们的货币价值。根据第 5 条规则，我们接受了这种错误的责任，并根据第 6 条规则公开通知了每个人。我确信总有一天我们的规则将不再有效，我们将不得不改变它们。多亏了规则 7，我们可以做到这一点。规则不会给你更多的控制，而是帮助你用更少的努力获得更好的结果。你的空闲时间可以投资到个人成长上。
