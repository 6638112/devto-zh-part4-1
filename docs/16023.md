# 作为专业人士，你应该具备的最重要的技能

> 原文:[https://dev . to/themarcba/the-most-important-skill-you-have-as-a-professional-h34](https://dev.to/themarcba/the-most-important-skill-you-should-have-as-a-professional-h34)

你好人们，请容忍我。我有一件极其重要的事情要告诉你。影响到所有人！(每年 600 多万)。这不是一个技术帖子，如果你们觉得它不适合这个平台，我会把它拿下来。

但是有一件非常重要的事情是你作为一个专业人士必须发展的，同时也是一个非专业人士。但在这之前，让我给你讲个故事...

2015 年，我父亲中风，导致他半边身体瘫痪，并患有表达性失语症(无法表达语言)。他依靠别人来完成大部分日常任务。

从一秒钟到另一秒钟，他完全脱离了他的生活，进入了另一个生活，那里的一切都将不再一样。永远不会。当他想交流什么的时候，他只能用猜测和是或不是来回答。他不得不放弃他热爱的每一个爱好，因为他现在无法做到。

他不得不停止工作，不得不卖掉他建立了 20 多年的公司。

所有这些都是他在工作中给自己施加压力的结果。他试图对公司的每个人进行微观管理，记账，现场处理紧急事件等。他想把一切都做好，而且想做得完美。我很少记得他没有压力。

他得到了一个令人难以置信的可怕的教训，我决定从这个教训中受益，并与你们分享。

# [](#do-not-overwork-yourself)不要过度劳累！

我觉得我不能重复这一点。我认为绝对重要的是，我们都要花很多时间从工作或其他情况给我们的压力中减压。如果有人更长时间地保持一种极端的节奏，我几乎可以向你保证——不会有好结果，或者至少不值得。

如果你在一份工作中，即使你回家后，你仍然对你的工作感到压力，我认为这可能是考虑另一个雇主或与你的主管谈谈的一个理由。

不要误解我的意思——雄心壮志没有错。你必须努力工作，才能得到值得拥有的东西。那是绝对正确的，我同意。但是当你发现自己每天都处于消极的压力之下时，那就是一个问题了！

你现在可能会想——“啊，我还年轻，我能保持下去”，你错了。中风也经常发生在年轻人身上。它每年杀死的人比艾滋病、肺结核和疟疾加起来还要多。

所以如果你压力不断，请减压。去做吧。散散步，喝喝茶，周末旅行，休一天假。此外，密切关注你所爱的人，如果你认为他们压力过大，就告诉他们。

此外，我为此离开了上一份工作(不完全是，但也是最主要的原因之一)，现在失业了。还是自己脱产自学。你觉得怎么样都行。但是我坚信这是一个正确的决定。

我希望我没有吓到你，但是也许害怕是正常的。这是一种保护我们免受坏事伤害的机制。也许我们应该相信这种感觉😉

好吧，我聊了很久。让我知道你是否有亲人遭受这种痛苦，并告诉我这是否影响了你的职业道路。
