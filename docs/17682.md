# 展示你的领导技能，即使你没有领导一个团队

> 原文：<https://dev.to/yashints/show-your-leadership-skills-even-if-you-re-not-leading-a-team-4d0n>

## 背景

几年前，我考虑晋升为高级顾问，这需要我展示我的领导技能和能力。独自完成一项任务，或者只是作为团队的一员，我没有机会领导一个团队，这让我很难展示我的团队领导技能。所以我想，既然这可能发生在许多想要进步并处于类似情况的开发人员身上，那么分享我的经验是值得的。

当你的经理把一个项目的控制权交给你时，这表明你是一个安全可靠的帮手。这是你展示领导技巧和能力的机会。但是，如果你负责领导一个没有正式头衔的团队，或者最糟糕的是你根本不负责，但必须证明你的能力，那该怎么办？

我做了一些关于如何在没有正式职位的情况下展示你的领导技能和能力的研究，并找到了一些关于这个话题的有趣文章。

让我们讨论一些关键领域，当我们处于对其他团队成员有影响力和领导力的位置时，我们都应该考虑这些领域，以便让他们能够在整个旅程中信任和跟随你。

## 以身作则

不论职位高低，每个人都有能力通过做一些实际的事情来展示领导技能和能力。展示领导力可能包括承担一些复杂的事情来支持其他团队成员，化解冲突，做出困难的决定，进行艰难的对话等。

但请记住，这不仅仅是关于*【你做了什么】*，还包括*【你是如何做的】*和*【活动结果】*。有些例子可以由您的经理/团队领导来衡量，而其他例子可能更主观。

在我的一次参与中，我承担了一些实际的例子，包括挑选复杂的待办事项，帮助团队领导复杂的决策，以一种他感觉不到我在干涉的方式做这件事(使用微妙的影响技术让他参与进来)，但同时他可以看到我是帮助他实现目标的人(加强我对客户的价值)。

因此，他开始一点一点地信任我，我们的关系一天比一天牢固。这些天来，基于我对客户、项目及其需求的了解，他很有信心让我来推动项目的方向。后来，在对项目的成功进行反馈时，他特意把我叫了出来。这样做的效果是，其他团队成员意识到了我的能力，比如强大的沟通技巧和不断增强的影响力，这导致团队成员对我和我的决定的尊重程度发生了明显的转变。

团队逐渐开始来问我关于如何做得更好的意见。除了能够展示我的领导技能和能力，这种感觉是如此的惊人和难以置信，以至于我开始意识到这是一个多么伟大的成就。

## 先听，后说

从某种意义上说，这是非常重要的，有时我们认为我们应该控制局面，谈上几个小时，这样其他人就会知道我们有多棒，我们知道多少，或者因为我们认为他们需要听到我们必须说的话。

然而，根据我的经验，在内部软技能培训的帮助下，我意识到最需要的领导技能之一是鼓励对话。

通过这样做，你表明你平等地尊重每个人，你获得了一系列不同的想法和意见，你将了解一些关于人、他们的业务和/或项目的情况，这将帮助你避免假设，你将有时间思考所说的一切。

就像他们说的，如果你忙着说话，你就没有在听。你可以猜测，通过尊重他人，这将有助于你获得他们的尊重。

## 不站队

如果你的角色需要你给别人分配任务，重要的是你不要故意或无意地把有利的任务分配给你的密友或你相处最融洽的人，而把不太有利的任务分配给别人。

重要的是要确保所有团队成员在可行的情况下有一个良好的任务组合，并且任务分配的方式让每个人都感到高兴，或者至少这是一个公平的过程，并且有动力做好工作以交付期望的结果。

通过这样做，你表明你是公平和值得信赖的，这有助于保持环境尽可能和谐。让我举一个例子来详细说明这一点:

我和其他一些当地开发商合作一个项目。我已经建立了信任，所以这是我在我们之间分配任务的机会，以一种每个人都感到高兴的方式。

因为他们中的一个是后端专家，但是没有任何前端开发的经验，我建议给他分配一个 UI 任务，其余的交给其他人，因为我们已经参与了以前类似的任务。

他起初有点犹豫，但在解释了我的目标，即他在 UI 方面的工作会有多有趣，以及他将如何在这一领域积累经验后，他愉快地接受了这个决定。

我不断地支持他，根据需要给予指导和支持，他也享受着冲刺的过程。老实说，我从未见过他在成功完成工作后如此开心。

另一方面，其他人也很高兴，因为他们有机会在不同的领域工作，探索我们正在使用的模式。你可能会说这是一个双赢的局面。

## 拉你的体重甚至更多

我们讨论了你应该如何以一种让每个人都满意或至少保持公平的方式分配任务，然而，你应该始终表明你准备好动手(做困难和不太有利的事情)，并在别人需要时帮助他们。

我总是准备挑最难的任务和没人愿意做的任务。

例如，转换一个遗留应用程序或修复一个存在了很长时间却没人知道为什么会发生的错误。

当别人看到消极的一面时，我看到了一线希望。这帮助我展示了我的优势，也让我知道我关心车队和它的成功。当别人看到你以身作则，比他们更努力工作时，你会发挥最大的影响力。

## 承担责任

这一点和以上任何一点一样重要。在完成一个项目或任何里程碑后，总会有这样的时刻，团队可以得到积极的认可和建设性的反馈，以帮助他们成长。

最重要的是，无论什么时候、以什么方式出了问题，一个真正的领导者都会承担责任，支持他们的团队，而不是推卸责任。

一个真正的领导者应该向每个人展示他们是可以依赖的，他们是一个可靠的人。避免指责，如果你看到一种指责文化，就大声说出来。同样的道理也适用于一切顺利，有表扬的时候。

你应该记得你是如何到达那里的，并且总是承认其他人的贡献。如果某个人工作出色，确保他们的贡献得到适当的认可。

我有机会表达某人的良好努力，也有其他时候我不得不承认我的决定不是最好的，并为我的选择负责。

承认自己错了，本可以做得更好，这是力量、成熟和领导力的重要标志。

## 不断发展你的领导技能

有些人是天生的领导者，但我们大多数人必须通过艰苦的方式来学习。

提高领导技能的最佳方式是通过在职培训，定期反馈和辅导来帮助你不断磨练技能。

在我的案例中，我没有机会成为领导者，但是，我已经主动安排了参观的机会，请求同事和客户的反馈，并寻求指导。

这不仅帮助我提高了技能，还展示了我的技能和能力，我取得的进步，我在同事、领导和客户中获得的信任，以及我建立的声誉，使我能够成为幕后的领导者。

还有其他方法，如研究、阅读相关书籍和文章、课堂或网上学习，也可以从你周围的人那里学习如何成为一名好的领导者。

被委托一个团队项目是一个展示你领导能力的令人兴奋的机会。尽管你的正式头衔没有改变，但你有很多方法可以向你的客户和同事展示你具备成为一名领导者的素质，并赢得他们的尊重。

如果你正面临类似的情况，希望这篇文章能激励你试着站出来展示你在这方面的技能。
