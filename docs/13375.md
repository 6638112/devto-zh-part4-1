# 超学习

> 原文:[https://dev.to/dwayne/ultralearning-3j6n](https://dev.to/dwayne/ultralearning-3j6n)

[Ultralearning](https://www.scotthyoung.com/blog/ultralearning/) 是由 [Scott H. Young](https://www.scotthyoung.com/) 写的一本关于掌握硬技能和加速你职业生涯的书。

> 超学习是一种积极的、自我导向的学习策略。

可用于[预购](https://www.scotthyoung.com/blog/ultralearning/)。

> 四年制学位曾经是伟大事业的保证。现在几乎是门里的一只脚。
> 
> 超学习提供了一种选择。不要回到学校，而是学习进入新职业的硬技能，或者让自己在当前的工作中不可或缺。与其挣扎着跟上，不如学会适应并茁壮成长。不要浪费时间，相反，通过擅长你一直想学的东西，给你的生活新的意义和信心。

我个人已经迫不及待想看了。
