# 使用布局动画创建动画手风琴面板

> 原文:[https://dev . to/skp tricks/react-native-create-animated-accordion-panels-using-layout-animation-4a9o](https://dev.to/skptricks/react-native-create-animated-accordion-panels-using-layout-animation-4a9o)

来源: [React Native 使用布局动画创建动画手风琴面板](https://www.skptricks.com/2019/06/react-native-create-animated-accordion-panels-example.html)

本教程解释了如何在 react 本地应用程序中创建动画折叠面板布局。当您想要在隐藏和显示大量内容之间切换时，手风琴非常有用。可折叠面板是自动控制的，如果用户点击单个面板，它会平滑地向下滑动并自动打开，但是如果你点击了任何关闭的面板，它会首先关闭已经打开的面板，然后使用动画打开被点击的面板。我们使用 React Native 的布局动画来打开和关闭面板。
