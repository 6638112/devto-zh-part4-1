# 快乐的一天-我的 Github 项目在 BlackArch OS 上市

> 原文：<https://dev.to/0xprateek/happy-day-my-github-project-listed-in-blackarch-os-2l6n>

很难相信，至少对我来说。我在 GitHub 上用我的工具“Stardox”查看最近的流量。我注意到 BlackArch 官网的一些流量。
我当时像什么？
这是一个假网站还是一个笑话？打开他们官方网站上的工具页面，发现 Stardox 被列为一个信息收集工具。
这是我编程的工具第一次出现在基于 Linux 的发行版中。
感觉很神奇。相信我，现在没有比这更好的了。❤
