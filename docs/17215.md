# 我偶然发现的有趣的东西#1

> 原文:[https://dev . to/j _ mplourde/interest-stuff-I-buckled-on-1-1jhl](https://dev.to/j_mplourde/interesting-stuff-i-stumbled-on-1-1jhl)

每天，我必须步行 20-30 分钟去上班，像每个人一样，我有日常维护任务，所以这是听播客的最佳时间。我还强制规定睡前一小时不看屏幕，所以我大部分时间都在看书。每当我有空闲时间，我要么阅读我口袋里的文章，打开一本书，听播客或看视频。

我只是想揭露我这一周偶然发现的有趣的东西。

# [](#podcasts)播客

我喜欢听乔·罗根的播客。这家伙才华横溢，表达清晰，还有很多有趣的嘉宾。虽然篇幅很长，但我从未对内容感到失望

**[海军拉维坎特论乔罗根经历](http://podcasts.joerogan.net/podcasts/naval-ravikant)** :拉维坎特是一名企业家，也是科技行业的天使投资人。他也是一位未来学家，对人类将如何参与科技有着有趣的看法。Ravikant 在 twitter 上最活跃，在那里你可以读到他最受欢迎也最不起眼的帖子，比如[如何致富(不走运)](https://threadreaderapp.com/thread/1002103360646823936.html)和他的一些[科技预测](https://threadreaderapp.com/thread/877467629308395521.html)。

**[杰米·梅兹尔谈乔·罗根的经历](http://podcasts.joerogan.net/podcasts/jamie-metzl)** :梅兹尔还是一位科技未来主义者、地缘政治专家、小说家和企业家。他讨论了有一天我们将如何破解我们的基因组密码，从而使我们成为超级人类。他证明在过去的几年里，我们已经改造了植物和动物。我一定会去看看他的书*黑达尔文*。

**[软件工程日报:网飞早期与格雷戈·伯勒尔](https://softwareengineeringdaily.com/2019/05/29/netflix-early-days-with-greg-burrell/)** :你知道网飞已经 14 岁了吗？它曾经提供邮寄 DVD 的服务？我也没有！伯勒尔是网飞雇佣的第一批员工之一，他现在仍在为他们工作。他解释了早期它是如何运作的，以及当他们转向流媒体平台时面临的挑战。对如何在大公司内部与团队和不同服务合作有深刻的见解。我被这个播客吸引是因为在过去的几个月里，我对*弹性工程*非常感兴趣。

# [](#projects)项目

**[任天堂 64 入门手册](http://n64.icequake.net/doc/n64intro/kantan/step2/index1.html)**:N64 毫无疑问已经根植于我们的文化中。小时候，我和朋友一起玩黄金眼玩得很开心。现在我是一名开发人员，我回顾了一项标志着我童年的技术。这本介绍性的手册用详细的细节、清晰的解释和有用的方案解释了它是如何工作的。

**[WTF Python](https://github.com/satwikkansal/wtfpython)** :对我来说，我发现学习东西最好的方法就是打破它们，毫不夸张。Python 是一种令人敬畏的编程语言，像任何技术一样，它也有其怪异和意想不到的行为。阅读这些令人费解的异常现象让我从一个完全不同的角度理解了这种语言。

# [](#books)书籍

**[《太空人:一个宇航员解开宇宙秘密的不太可能的旅程》迈克·马西米诺](https://www.goodreads.com/book/show/34474869-spaceman)** :我就是放不下《小镇》这本书。我是宇航员传记的粉丝，因为他们的旅程充满了艰辛、起伏和冒险。你意识到他们也是人，他们有自己的挣扎，他们一直在与逆境抗争。非常鼓舞人心。

乔治·奥威尔的《动物庄园》 :我已经读过《1984》，当我的朋友把这本书借给我的时候，我非常激动。如果一个农场的动物造反，推翻主人经营农场，会发生什么？我一直喜欢反乌托邦故事。这本书里有一句有力的话特别震撼了我:*所有的动物都是平等的，但有些动物比其他动物更平等*
