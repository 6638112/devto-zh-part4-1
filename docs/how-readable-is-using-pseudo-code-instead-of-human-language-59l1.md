# 用伪代码代替正常的人类语言可读性如何？

> 原文：<https://dev.to/voidjuneau/how-readable-is-using-pseudo-code-instead-of-human-language-59l1>

老实说，人类语言是我的克星。
我的母语不是英语，然而，公平地说，我的母语也很糟糕。

我不擅长口头交流和基于文本的交流。当然，我不擅长编程语言，但我觉得对我来说，写和读都更舒服。我发现用这种方法，我可以写出结构更清晰的文章。

然而，我意识到我通常是出类拔萃的，或者说是一般人，而且常常有点古怪。因此，我想知道别人看到它时的感受。
尽管标准的 pseudo 可能是更好的方法，但我只是在使用 Java 时感觉很舒服，所以我通常用 pseudo-Java 的方式来做这件事。
