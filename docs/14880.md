# 使用 Angular 和 Spring MVC 的全栈 Web 开发

> 原文:[https://dev . to/angular devz/full-stack-web-development-with-angular-and-spring-MVC-51da](https://dev.to/angulardevz/full-stack-web-development-with-angular-and-spring-mvc-51da)

欢迎使用 Angular 和 Spring MVC 进行全栈开发。对于那些不了解 Angular 和 Spring MVC 的人，我想简单介绍一下这两种技术。

Angular 是最流行的构建 web 应用的前端 Javascript 框架之一，另一方面，Spring MVC 也是最流行的构建 Java Web 应用的 Java 框架之一，使用它我们可以构建行业标准的 Java/J2EE 企业应用。

因此，在本课程中，我们将利用这两种技术，构建一个 web 应用程序。我们要构建的应用程序不是一个简单的 Hello World 应用程序，我们要构建一个端到端的数据库应用程序，我们要执行所有的 CRUD 操作，如创建、读取、更新、删除。

我们将在本课程中构建什么？

在本课程中，我们将构建一个图书管理系统，

创建一本书
阅读一本书
更新一本书
删除一本书
我们如何开始本课程？

我们将从创建一个 REST API 开始本课程，这是一个图书 API，我们将使用 Spring MVC 来构建这个 API，我们将使用 Hibernate 作为我们的 ORM。BOOK API 提供了一组服务端点，我们将使用 Angular 对这些服务端点进行 HTTP 调用

基础知识

HTML、CSS 和 Javascript 基础知识
核心 Java 基础知识
Spring MVC 基础知识
Hibernate 基础知识
RESTful web services 基础知识
你会学到什么

因此，在本课程结束时，您将对 Angular 和 Spring MVC 有更好的了解。你会对如何集成 Angular 和 Spring MVC 有更好的理解。您可以创建自己的 API 并使用 Angular 进行 HTTP 调用

你会学到什么

你能够使用 Angular 和 Spring MVC 创建一个 web 应用程序
你能够使用 Angular 进行 HTTP 调用
你能够使用 Spring MVC 创建一个 RESTful API
你能够使用 Spring MVC 和 Hibernate 创建一个完全成熟的数据库应用程序
