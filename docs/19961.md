# 雇佣多样化的员工

> 原文:[https://dev.to/shehackspurple/hiring-a-diverse-workforce-1e](https://dev.to/shehackspurple/hiring-a-diverse-workforce-1e)

很多人问我和其他很多活跃在科技领域的女性，如何雇佣和保持“多元化的员工队伍”。我觉得很奇怪，既然我们是…方面的专家，为什么会被问到这个问题。技术。我们不是人力资源方面的专家。以前我曾告诉人们，我不知道如何吸引其他女性到某个地方工作，但在去年(自 WoSEC 成立以来)，我学到了很多东西，第一次觉得自己可能有一些关于这个话题的有用建议。虽然我要在这篇文章的开头提醒你，我不是这方面的专家(我不代表所有女性！)，我真的希望这些想法能帮助科技公司吸引更多女性和其他来自弱势群体的人加入我们的行业。

*   聘请这方面的专家来帮助你，比如简·弗兰克兰。然后付给她**，因为她的工作有价值**。令人沮丧的是，一些公司要求该行业的女性免费工作，以帮助她们“解决这个问题”，好像她们的时间并不宝贵，好像我们都神奇地理解了这个复杂的话题。当你还不知道任何问题的答案时，聘请专家是一个好办法。
*   和你现在的女性，LGBT，NB，POC，残障员工聊聊。确保他们的任何问题都得到解决(而不是拒绝或忽略)。确保他们得到公平的报酬，确保他们和其他人一样得到同样的晋升，确保他们得到成功所需的培训和其他机会。确保他们快乐。如果他们有问题，他们觉得是由于歧视，他们不会告诉人们来那里工作。正是这些人将成为你最好或最差的广告，吸引人们来为你工作。确保你**实际上**有一个热情、安全和公平的环境，是吸引和留住优秀候选人的最重要的事情。这适用于所有人，而不仅仅是来自代表性不足群体的人。
*   提议举办女性科技聚会(如 [WoSEC](https://twitter.com/WoSECtweets) ！).多年来，Shopify 一直在主办 OWASP 渥太华会议，事实证明，不仅他们的员工能够非常方便地参加我们的活动并从中学习，他们还从我们的会员中招募了不止一名 AppSec 团队成员。双赢！
*   赞助女性科技活动，如 WISP 的“ [DEF CON 奖学金倡议 2019](https://www.wisporg.com/events-calendar/2019/3/15/wisp-def-con-2019) ”年度活动。
*   读一读塔拉·惠勒关于这个话题的书，《科技领域的女性》，如果你有办公室图书馆的话，买一本。
*   如果你在一家初创公司工作，读读艾丽莎·舍文斯基关于这个主题的书，《向外倾斜》。
*   在计划包含酒精的活动之前，要意识到许多女性在涉及酒精的情况下会感到不舒服。
*   听着。当有人向你提出一个问题时，可能需要很大的勇气。说出来很难，尤其是女性被教导“不要捣乱”。要求加薪很难。报告不当事件很难。不要因为不认真对待它们而使事情变得更困难。

公司还可以做很多其他的事情，比如提供生育和变性相关的健康福利，公布你的工资等级，让每个员工都知道他们在哪里，弹性工作时间，育儿假等等。就像我说的，我不是这方面的专家，我只是有一些想法，希望对你有帮助。

感谢您的阅读，非常感谢您有兴趣让信息技术成为一个更加包容、安全和公平的行业。
