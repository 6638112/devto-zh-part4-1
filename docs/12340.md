所以我在几周前发表了新创建的 Babel 播客的第一集！我还写了一篇博文和一个独立的 T2 网站。

但是我想听听这个社区的意见！我很好奇你会对这一集有什么想法，也很好奇你是否有*建议*关于你希望我们谈论的内容:更多关于团队本身，关于开源维护、筹款/金钱、JavaScript 语言等的一般东西。

我想让 Babel 与众不同，但也不想把它变成另一个只采访 JavaScript 库作者的 JavaScript 播客(因为已经有很多了)。我的一些想法将是采访 TC39 成员，解释巴别塔中的一些概念/决策，甚至采访替代工具！我认为我们可以利用这个平台做很多事情。
