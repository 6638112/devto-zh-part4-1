# 你知道 javascript 中的代理吗？

> 原文:[https://dev.to/dg92/do-you-know-proxy-in-javascript-53dc](https://dev.to/dg92/do-you-know-proxy-in-javascript-53dc)

代理似乎是一个不太出名的 javascript 特性，但是如果实现的话，它拥有很大的能力。

我们可以使用它进行验证、值修正、跟踪属性访问、未知属性警告、负数组索引、数据绑定、访问 restful web 服务(方法调用)、可撤销引用、监控异步函数、类型检查等等。
