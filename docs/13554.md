# 让你的爱好项目落地

> 原文：<https://dev.to/kartikjha/getting-your-hobby-project-of-the-ground-3f4g>

我成为一名专业开发人员已经将近两年了，但是我看不出我的编码能力有什么不同。

是的，在面对一个写满错误的屏幕时，我变得更加无畏了，我可以通过自己的方式来处理它，并且学会了一些技术(React、Node、Java ),学会了终端:D 的方式

但是仍然缺少一些东西，我没有任何开源贡献，我的 Github 简介看起来很无聊。几个放弃的项目:(而且这明显影响了我对潜在雇主的第一印象。

所以，我已经决定完成我的第一个业余爱好项目，这是一个用 canvas 和 typescript 制作的游戏(最小可行的产品，因为我打算加入多人动作！).

到目前为止，我已经设法从头开始建立这个项目，并创建了基本的架构。我使用 Typescript、Webpack 和原生 canvas API 来实现 MVP 以及 Eslint 和更漂亮的平滑开发体验。

回购和相应的 GitHub 项目可以在[https://github.com/KartikJha/color-sticks](https://github.com/KartikJha/color-sticks)找到。

我已经通过搜索互联网建立了没有任何样板文件的项目，但很乐意写一篇你需要的综合文章。

谢谢你读到这里。我会确保这个爱好项目得到交付。
