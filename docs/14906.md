### “Ecosia”——种树的搜索引擎🌳️

甚至你可能听说过 Ecosia，我仍然想介绍一下种树的搜索引擎，因为气候变化无疑是非常重要的。与谷歌非常相似，Ecosia 提供了一个搜索引擎，在搜索结果的顶部显示广告。不过，从搜索结果中赚取的广告费不会流向投资者。它流入选定的项目来植树和减少大气中的二氧化碳。如果你能做一件小事对环境产生积极影响，那就换成 Ecosia 吧🌳️

`Green`、`Eco-friendly`、`Search Engine`

### “program CLI”——自托管、基于 CLI 的 Instagram 克隆🤳️

[proGramCLI](https://sarthology.github.io/proGramCLI/) 是由 [Sarthak Sharma](https://sarthak.it/?ref=PeterThaleikis.com) 开发的自托管、基于 CLI 的 Instagram 克隆。隐私很重要，尤其是当涉及到你的亲密朋友和家人时。自托管版本可以让您更好地控制哪些人可以查看您的照片和视频。如果你不想让 Zuck 和朋友看到你分享和输入的所有内容，这就是你的前进方向。自然是开源:[sartology/proGramCLI](https://github.com/sarthology/proGramCLI)。

`Instagram`、`Privacy`、`electron`、`node`

### “curlx”——如果卷毛和邮差生了孩子👶

作为一名开发人员，我经常检查 API 是否像预期的那样工作并为我的目的服务。即使有像 Postman 这样的图形程序，我还是经常会莫名其妙地使用命令行。卷曲是我选择的工具。通过 curlx ， [Shivkanth Bagavathy](https://www.shivkanth.com/?ref=PeterThaleikis.com) 为那些喜欢在命令行上探索 API 的人构建了一个工具。您可以直接在 bash 上看到消息头和格式化的响应。它还提供了一个以前请求的列表(历史)和管理收藏请求的选项。不出所料，是[在 GitHub](https://github.com/shivkanthb/curlx) 上开源的。

`Open source`、`Command-line`、`bash`

### “用于 Chrome 的 Wavenet 将任何标记的文本整合成声音

[Wavenet for Chrome 是一个浏览器扩展](https://chrome.google.com/webstore/detail/wavenet-for-chrome/iefankigbnlnlaolflbcopliocibkffc)，可以将任何标记的文本整合到声音输出中。[迈克尔·普瓦里耶-金特](https://github.com/pgmichael)开发了一个工具，可以帮助有严重阅读障碍的人以及视力受损的人。该扩展是开源的，你可以在 [GitHub](https://github.com/pgmichael/wavenet-for-chrome) 上找到源代码。

虽然扩展是免费的，但谷歌云需要支付少量费用。与大多数云提供商一样，新客户也有一个免费层。有关详细信息，请查看谷歌云产品。

`Open source`、`Browser Extension`、`Chrome extension`

### “懒人”——管理一切的懒惰方式

如果你曾经使用过 docker，你应该已经体验过它众多的命令和选项了。当你是 docker 的新手时，发现你的容器的当前状态是令人讨厌的，也是具有挑战性的。有了[杰西·达菲德](https://github.com/jesseduffield)创作的[懒虫](https://github.com/jesseduffield/lazydocker)，这变得更加容易。它向您显示容器、图像、体积等。一个屏幕上。通过鼠标选择，您可以找到更多关于状态和所用资源的信息。杰西说:“docker 是管理一切的最懒惰的方式”。他还发布了一个简短的[视频](https://www.youtube.com/watch?v=NICqQPxwJWw)展示其特性和功能。

`container`、`serverless`、`Docker`、`open source`

### “Kimai”——自托管、开源的发票管理网络应用

自由职业者和企业家这可能是你感兴趣的。Kimai 是一个跟踪工作时间、管理客户和项目以及生成发票的网络应用程序。这些只是它附带的各种功能中的一部分。在网站上，您可以找到版本 2 中功能的完整列表。您可以在文档中找到关于如何使用 Docker、1-click installers 甚至 FTP 自托管应用程序的信息。如果你正在寻找 Xero 或 InvoiceNinja 的替代品，这看起来是一个不错的选择。由[凯文·帕普斯特](https://www.kevinpapst.de/?ref=PeterThaleikis.com)开发的软件是[在 GitHub](https://github.com/kevinpapst/kimai2) 上的开源软件，他通过捐款支持开发。

`PHP`、`open source`、`self-hosted`、`invoicing system`
