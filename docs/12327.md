# 你真的想要自己的开源项目吗？

> 原文：<https://dev.to/kylemathews/do-you-actually-want-your-own-open-source-project-f43>

开源太棒了！这是作为一名开发人员成长、影响我们的工艺、交朋友、建立您的在线简历以及帮助解决您自己和他人的问题的一种令人惊奇的方式。

开源太可怕了！愤怒的问题、模棱两可的公关、维护你不再使用的项目的压力——所有这些都开始让人觉得是最糟糕的家庭作业。

决定开始并运行开源项目是一个艰难的选择——对任何人、生活环境和项目来说，都没有正确的选择，许多人理所当然地回避它。

以下是一些想法:

## 它的*就是*大量的工作

进去期待这个。一些项目开始时很忙，随着项目接近“完成”，工作逐渐减少。许多其他人继续要求越来越多的工作久而久之。一开始，周六一整天都在做你的项目可能会很有趣，但这不会持续很久。

## 你的动机是什么？

以下是一些强烈且可持续的动机，即使项目要求特别高，它们也会让事情变得有趣且有回报:

*   你的工作需要这个项目
*   你喜欢和其他 OSS 开发者交朋友
*   你想创新并影响我们工艺的方向
*   你想解决一个不断遇到的恼人问题
*   你热爱学习。OSS 项目给你带来了源源不断的新挑战。
