# SQ3R

> 原文:[https://dev.to/vorakl/sq3r-53g8](https://dev.to/vorakl/sq3r-53g8)

[SQ3R](https://en.wikipedia.org/wiki/SQ3R) 是一种[阅读理解](https://en.wikipedia.org/wiki/Reading_comprehension)阅读教材材料的方法，因其五个步骤而得名:*调查*、*提问*、*阅读*、*背诵*、*复习*。它提供了一种将信息存储为文本的有效方法。对于我们的大脑来说，文本形式远非理想或最佳的有意义数据来源，但 SQ3R 方法使记忆、理解和整合新知识与已知知识成为可能。
