# 害怕地跳舞

> 原文:[https://dev.to/zellwk/dancing-with-fear-4k82](https://dev.to/zellwk/dancing-with-fear-4k82)

当我工作的时候，恐惧和怀疑经常会突然出现在我的脑海里。

当我编码时，我想知道:

*   我使用的是最佳实践吗？
*   我是否以最好的方式构建了代码？
*   我做得对吗？

当我写作时，我会想:

*   我凭什么有资格教任何东西？
*   人们会明白我想说什么吗？
*   这件事完全跑题了。这和技术无关！我的天啊。

当我经营我的生意时，我想知道:

*   我应该回复每一封邮件吗？
*   我应该继续关注文章吗？
*   我能看看课程的预订单吗？
*   如果我不能按时完成我的课程怎么办？
*   我的产品和销售邮件是否有说服力，但又不显得太过推销？

我讨厌有这些想法，因为它们让我犹豫不决。我放慢了工作节奏。

但是我喜欢这些想法，因为它们告诉我我在做正确的事情。

## [](#cherish-the-fear)珍惜恐惧

如果你害怕尝试编码、写作、说话，害怕把自己置身于这个世界，这是一件好事。有这种恐惧就好。

当我们试图做我们关心的事情时，这种恐惧就会出现。珍惜它，因为它带你去你想去的地方。

当你在做不重要的事情时，你不会感到这种恐惧。比如，当你决定午餐吃什么的时候，你不会感觉到这种恐惧。

## [](#dance-with-the-fear)与恐惧共舞

塞思·戈丁总是告诉人们“与恐惧共舞”。

对我来说，这意味着让我的恐惧引导我工作。克服恐惧的关键是:

1.  承认它的存在
2.  承认我害怕
3.  向恐惧倾斜
4.  继续工作

恐惧总是爬上我的心头。我总是意识不到，直到为时已晚。当我注意到它的时候，它总是让人残废。我停下来，停止工作，怀疑自己。

有时恐惧变得如此强大，我不得不停止工作喘口气。我通过看漫画、吃饭或其他活动来转移注意力。但是恐惧总是存在。当我开始工作的时候，我仍然需要克服它。

克服这种恐惧的唯一方法是给自己足够的耐心去承认并坚持到底。如果我这样做了，我会得到比我通常所能创造的质量更好的东西。我说我需要说的重要的事情。

例如，这是一篇文章，我写到“我真的不知道我他妈的在做什么”。我犹豫了很久。我给自己几天时间来思考这篇文章。但是没什么好结果。直到我坐下来，努力克服恐惧，我才设法说出正确的话。结果就是你现在读的。

如果你感到恐惧，恭喜你。你正朝着正确的方向前进。接下来是与它共舞。

说起来容易做起来难。

## [](#an-interesting-exercise)一个有趣的练习

闭着眼睛向前走三十步。注意你走路时浮现的情绪。(免责声明:请不要走进车流...).

注意到发生了什么吗？

你开始时充满自信，然后迈出大步。

然后，你开始犹豫。你想知道你是否走在正确的方向上，你的面前是否有障碍，你是否还在直走，等等。你迈出的每一步都比前一步小。

会有一个你停下来的点。恐惧穿过你的脑袋，让你麻痹。你甚至不能不睁开眼睛再往前走一步。

试试看。

当你闭着眼睛向前走的时候，你会经历恐惧，因为你不知道你要去哪里。你面前可能潜伏着危险。

这种恐惧类似于你在做对你来说很重要的事情时的恐惧。性质不同，但感觉几乎一样。

如果你总是被恐惧麻痹，你可以通过这个练习来习惯恐惧的感觉。每当你害怕前进受阻时，试着多走一两步。它训练你的抵抗力。

## [](#an-easier-way-to-dance-with-the-fear)与恐惧共舞的简单方法

当你闭着眼睛向前走的时候，很难克服像你面对的那种可怕的恐惧。更简单的方法是倒着走。

当你向后走(而不是向前)时，会发生一些事情:

1.  你会走得更远
2.  你会感到更少的恐惧
3.  恐惧变得更容易处理
4.  你可以根据你去的地方导航你要去的地方

我想再详细阐述一下第四点。

当你向后走的时候，你的眼睛是睁开的。你可以看到你走过的路。从你的眼角，你可以看到你需要采取的任何潜在的弯曲。

你仍然看不到前方的情况，但这比瞎走容易。

你可以在工作中应用同样的概念。从你过去的成就和经历中汲取力量。承认你在哪里，走了多远。

这并不能帮助你看清你面前有什么障碍。这并不能消除障碍。这并不能消除恐惧。但这让恐惧变得更容易控制。

我希望这能给你一个与恐惧共舞的方法。

现在走吧。
