> Daily 是一个开源的浏览器扩展，它为你的新标签页提供策划开发新闻。

它从数十个独特的来源收集文章并进行排名，以帮助开发人员了解最相关的开发人员新闻。您可以为项目添加书签，跨设备同步数据，并在需要时阅读。它也可以离线工作，并有一个用于移动的渐进式网络应用程序(PWA)。

该代码可在 GitHub 上的组织 [dailynowco](https://github.com/dailynowco) 下获得，并在 [Affero GPL 3.0](https://github.com/dailynowco/daily/blob/master/LICENSE) 下获得许可。我们也写了我们的行为准则，我们希望每个人都遵守❤️

我们要感谢 DEV、Codefund 和其他开源项目，它们为如何开源项目提供了如此丰富的资源！(一篇新博文的创意:P)。这都是关于开发者的经验，我们还有很多要学习的。

当然，这绝不意味着我们开源努力的结束，而仅仅是一个开始。我们仍然有许多文档要写，确保它很容易设置一个本地环境等等。

我们相信，为了获得社区的信任，你必须是透明的、支持的，每天开源会使事情变得容易得多。现在每个人都可以看到引擎盖下是什么，自由提问，甚至看到我们如何获得资金来保持这个产品的运行和增长。

作为这一过程的一部分，我们还发布了一些公告:

1.  日常风格指南和设计也将“开源”，并将很快公之于众。与此同时，我们有了包含所有日常内容的故事书，很快也会在 NPM 出现。
    [https://storybook.dailynow.co](https://storybook.dailynow.co)

2.  没有更多的付费墙内容！我们认为内容应该对所有人免费，因此任何新的付费博客帖子都将被忽略，不会进入我们的数据库。这个机制仍然不完美，因为介质使它很难被发现，但是我们会解决它。

3.  每日博客正在迁移到开发！我们正在寻找一个与我们的核心价值观一致的平台，DEV 看起来非常适合。当然，这并不意味着我们将停止在媒体上发布，因为我们有一个观众，但一切都将写在开发和交叉发布，而不是相反。(这个帖子已经这么写了)。希望我们的观众会随着这个过程一起迁移:)

就是这样！我们有很多事情要做，我们期待着你的贡献🤩
