# 我在脸书 F8 2019 的经历

> 原文:[https://dev . to/copper wall/my-experience-at-Facebook-s-F8-2019-4d 50](https://dev.to/copperwall/my-experience-at-facebook-s-f8-2019-4d50)

我(半)最近去了脸书的开发者大会，F8，加州圣何塞。这是我代表 iFixit 参加 GitHub Universe 2017 以来参加的第一个会议，也是我第一次代表 PayPal 参加会议。或者至少我想我是代表贝宝去的。我问我的经理我是否可以去，因为 React 和 GraphQL 上的一些讲座听起来很酷。React 绝对是 PayPal 前端开发的现在和未来(还有一般的 web？)而且我们公司有一个相当强大的团队支持使用 GraphQL。因此，我认为在会议上了解更多的信息对我来说会很有趣，对我的工作也有好处。

无论如何，我完全打算在我走后写一个总结，但生活发生了，工作有时会变得疯狂。我在美丽的圣路易斯奥比斯波的一家时尚咖啡/甜甜圈店露营，所以我想我应该启动一个番茄计时器，看看在这段时间里我能做些什么。

## [](#going-to-a-conference-alone)独自去开会

这是我第一次在没有朋友或同事的情况下参加会议。我不知道还有谁要去，所以我完全是独自一人。从某种意义上说，这很棒；我可以随心所欲地去参加任何讲座或会议，而不必担心团队中的其他人会做什么。另一方面，去吃午餐或快乐时光却不认识任何人，这是很奇怪的。看起来人们要么已经成群结队地来了，要么已经找到了可以一起跳舞的人，但我觉得这有点吓人。所以我并没有参加太多会议后的欢乐时光或社交活动。我住的地方离会议中心只有 15 个街区，所以如果我不想去的话，可以很容易地走回家。

下次我一个人去参加会议的时候，我肯定需要试着多接触一些人。我真的不知道如何做到这一点，但我已经开始更经常地去城市里的聚会，所以我试图将自己置于我不认识人的情况下，必须找到共同点或有趣的事情来谈论。如果我最终真的要去参加一个会议，那就更是如此。如果我去圣何塞甚至旧金山参加一些活动，结束后回家就很容易了。但是如果我最终去了其他州或湾区以外的 React 或 GraphQL 会议(这是我在去了 F8 之后非常想做的)，那么我真的值得去见见其他去参加那个活动的开发人员。

## F8 不同地区

### [](#the-keynote)基调

基调很好。听起来他们在做一堆很酷的东西。他们几乎每隔一句话就提到隐私。根据他们的历史谁知道那是不是真的。我真的不怎么用脸书，所以我对主题演讲不太感兴趣。看到马克·扎克伯格本人站在舞台上有点意思。

### [](#workshops)作坊

[![hooks workshop](../Images/831694e343abdc2a5d92e29a08525ed8.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--kdrQai0O--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/nbgu276quj9kjcwmje9m.jpg)

他们设置了一个非常酷的工作区，你可以带着笔记本电脑坐在长桌旁，跟着演示脸书工具的人，比如 PyTorch、Docusaurus、React Hooks、GraphQL 等等。Docusaurus 和 React Hooks 研讨会非常值得参加。我不知道 Docusaurus 被这么多不同的地方使用，如 Jest 或 smash . gg。React Hooks intro workshop 介绍了一些钩子基础知识以及如何制作定制钩子，这是我以前从未尝试过的。

### [](#team-meetups)团队聚会

会议区的很大一部分是留给脸书开源工具的，你可以走到 React 或 WebXR 的工作人员面前，问他们问题或与他们交谈。他们也有稍微更有条理的时间专门与团队会面。我参加了多个活动，在这些活动中，React 核心团队的成员会和所有参加活动的人站成一圈，开玩笑，回答人们的问题。令我惊讶的是，我有勇气问他们一些 hooks 问题，并在所有人面前回答 Native Web 问题，他们回答得非常坦率和彻底。

我真的无法表达他们有多好。第一天，我与一些团队成员进行了几分钟的交谈，当我第二天回到开源区域时，他们记住了我，让我问更多的问题！我问了一个核心团队成员关于悬念数据加载的问题，他们谈到了他们内部目前存在的一些问题，以及如果我想使用一些不太稳定的 API，我可以尝试的一些东西。老实说，我留下了比开始时更多的问题，但是能够从那些使用我基本上每天都在使用的工具的人那里获得灵感真是太棒了。他们很友好，也很乐意帮忙。

如果你看不出我对它的进展有多兴奋，这显然是最精彩的部分。

## [](#would-i-go-back)我会回去吗？

我不确定明年是否会回 F8。react meetup 是一次出色的体验，但我想知道参加 react 专用会议是否会更好，如 React Conf 或 React Rally 或其他无数 React 会议。

会议上的演讲确实提供了很多信息，但是我可能通过在 YouTube 上观看也能获得同样的收益。

如果我回去的话，我肯定会带一个同事或朋友来，让他们在与他人交谈时感觉更自在一点。或者，我可能应该优先考虑结识新朋友，看看他们在各自的公司做些什么，建立一点关系网。
