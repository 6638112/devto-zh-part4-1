# 编码成虚无主义

> 原文：<https://dev.to/jonamichahammo/coding-into-nihilism-2im3>

这是我的第一个帖子...想想看，这是一个咆哮！

我第一次对“编码”产生兴趣是在 2017 年。我觉得数据科学看起来很酷，所以我想学习它。事实证明，我并不容易学会它，尤其是当我不得不自学的时候。

不久后，我偶然发现了 Python，混淆了统计学的引用，像机器学习、人工智能等性感的词。我从想成为一名数据科学家变成了被如何循环吓倒。

尽管如此，我至少尝试了。我用 Python 编写了一个程序，它接收输入并在后台进行计算(想象一下一个每日热量摄入计算器)。当 2017 年结束时，我有点亏空，因为我在电脑设备、书籍、教程和大量咖啡馆时间上花了钱。我所做的一切努力几乎没有什么可展示的。

...

2018 年，我从编程语言的低挂果实开始:HTML。

我知道这不是一种编程语言，我只是想骗你！

学习标记相对容易(除了头脑中发生的事情，这部分导致我转向 SEO 和一些不相关的标签)，但我注意到的一件事是，HTML 5 和 CSS 3 的在线资源不是那么清晰，即使它们在一个美丽完美的网站上。

HTML 5 有一些并不是每个人都使用的新元素，这使得我很难了解我将餐饮业比作“错误放置”的最佳实践。你开始用一种方法来做，下一个教程将拆除所有你认为重要的关于标题/导航/主/节等的东西。

CSS 把我拿下了最讨厌的兔子洞。首先，它让我怀疑自己是想成为一名开发人员还是一名设计师。然后它迫使我学习一些设计，尽管我已经决定成为一名开发人员(SVG！做个 logo！让照片看起来完全是你想要的样子！别给那个余量，给那个余量！).

我开始觉得在线教程几乎类似于我们有时听到的 scammy 训练营。但是我并不是通过这些教程变得有能力编码的人之一。

一旦 CSS 把我引向 Sass 和 Less 和 variables，几乎使它看起来像一种编程语言，我就失去了我的$h！这就像想通过跆拳道来减肥和增强腹部肌肉，却花了几个月的时间来观看著名的跆拳道比赛和学习踢球的历史，然后才真正组合起来。嗯，这不是一个好的类比，因为它只是不需要那么长时间。我只是迷失在寻找、发现、好奇、询问、观察、重复、实践和重复的行动中。我比任何事情都更困惑，并不因为这一切而感到更聪明。

...

我只学习了一小部分 JavaScript:Codecademy，YouTube 上的 mmtuts，现在我将在 Scrimba 和一些 YouTube 视频上深入学习。

...

我仍然不明白一些基本的东西，比如 DOM，或者如何像你一样容易地定位我想要的每一个元素，比如说，正方形空间。就好像，一旦我开始理解一个媒体查询，就有另外 100 个属性、值和神秘的选择器需要完美响应的网页设计。

...

这让我压力很大，因为直到几周前丢掉工作(一家 It 公司的实习结束)之前，我一直在下班后和周末写代码。看起来我的耐心和变得不安和沮丧的倾向与教程的长度(比如自由代码营)或我保留信息的能力没有很好的关联。我学到了这么多，但当我看到入门级的工作描述时，我只是感到沮丧和害怕。我还要多久才能学会 Web pack？AJAX？JSON？Vue.js？Angular.js？

我一次又一次地听到这样一个笑话:人力资源部写下招聘启事，并期望实习生具备所有条件，包括厨房洗碗池。

说真的，谁会写这些东西？只有我一个人感到如此绝望吗？

...

反正我知道只有我。我知道任何人都可以在像 MDN 这样的网站上找到文档，但就我个人而言，我经常会迷失在这些行中并感到困惑。除了知道更多我不知道的，我知道的不多。当我找到一个模板时，它看起来很漂亮，但它是别人的模板，不是我的。当我建立自己的网站时，它看起来像 1998 年打电话来，希望它的风格回来。当一个教程巧妙地陷入一个非同步函数时，我的推理能力建议我们申请去麦当劳工作。

...

我当初为什么会卷入这件事？嗯，我希望能够用简单的方式向人们解释复杂的事情。但事实证明我不懂复杂的东西。编程让我觉得自己很蠢。编写任何东西都让我感到焦虑和不舒服，除了少数几次我在 HTML 上得到的乐趣。我真的希望我能为此去上学，因为我开始一直怀疑我自己作为一个 SWE 的能力。

...

最后，我知道为什么我失败了。我对编程的想象与成为一名程序员的条件无关。我喜欢程序员做什么的想法，它可以构建什么，等等。然而，我不喜欢阅读复杂的文档。此外，我讨厌其他人说它容易阅读，因为对他们来说它很容易。在这个领域冒险让我零收入，但更多的债务和对自己聪明的更少信心。更糟糕的是，我发现很难与人联系起来，因为我只见过那些理解它或者自然而然地进入这个领域的人，或者那些不想与语法和概念有任何关系的人。

我很想放弃，但又觉得太投入了。不是我不想懂编程，只是我不懂编程。我不会放弃，但我会挖掘我的报复能量，学会去刁难别人。每一个糟糕的教程，每一个糟糕的实践，每一篇过于复杂的文章，每一份陌生的文档，每一个头脑聪明但人际交往能力糟糕的开发人员，对所有 g0d @ Mn 工作板的描述，他们的期望如此之高，以至于把现实的水搅浑了，对所有那些没有为你的职位提供任何培训方式而没有一些软件开发的高水平诀窍的雇主，弯曲吧。

当我踏进家门，经过实习，一旦我能够日复一日地工作，完成雇主要求的项目，我就要去帮助那些贫困的人。我会制作自己的视频，自己的文章，还会说很多脏话。我将在训练营做志愿者，这些训练营是免费的，而不是那些只迎合中产阶级学士/硕士毕业生的，他们可以享受一年或更长时间的休假。

我出身于贫困家庭。我在创伤中长大，有过心理健康问题。我参加过足球和 MMA 等运动和比赛，滥用过酒精和大麻，很可能有某种程度的脑损伤。

我有我的局限性，和许多缺点，但是我拒绝失败作为我的最终目的地。我相信我能在正确的环境下学习。我将继续在我能得到的任何环境下工作，直到我练习到足够了解足够的语言和软件、概念和实践，以通过面试。我会成功的。我会成功的。我会自己做这件事，直到我有一个真正成长的社区，然后我会成倍地超越并帮助很多像我一样想放弃梦想的人。

对那些未来的人，我要说，坚持住。我马上就到！
