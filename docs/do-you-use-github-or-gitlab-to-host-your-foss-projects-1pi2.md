# 你用 Github 还是 Gitlab 来托管你的 FOSS 项目？

> 原文：<https://dev.to/prahladyeri/do-you-use-github-or-gitlab-to-host-your-foss-projects-1pi2>

虽然 Github 实际上是大多数程序员的代码托管设施，但 Gitlab 只是在微软去年宣布收购 Github 后才获得了一些荣耀。

当时，如果我没有错的话，很多项目都迁移到了 Gitlab。至少，他们中的许多人已经建立了镜像，这样他们的 Github 提交就会自动反映到 Gitlab，他们将此视为一种保险(在极少数情况下，微软会从 Github 中撤出 Skype 或诺基亚！).

作为一个自由/开源软件爱好者，你经常用什么来存放你的代码？是 Github 还是 Gitlab？
