# 简介

随着时间的推移，对于那些对未知事物有风险偏好的开发人员来说，创业正成为一种可能的职业选择。

但这并不意味着可以盲目地做出加入创业公司的决定，并将其视为一场赌博。

当然，有一些初创公司在早年曾经是初创公司，但现在已经变得很大。

我总是建议在加入创业公司之前做好尽职调查。

我希望我的理解是，与一个风险投资者一起促进大量的创业项目，现在就开始创业。

可以帮助你更好地选择加入哪家创业公司，而不是像赌博一样对待它。

# 创业公司应该寻找什么？

选择一家合适的初创公司对你来说可能很难，尤其是当你打算花时间在这家初创公司工作的时候。

因此，以下是我在选择创业公司时会注意的一些事情。

*   他们试图为市场解决的问题是什么？
*   创始团队的背景是什么？
*   他们有最小可行产品(MVP)吗？
*   他们独特的卖点是什么？
*   他们是否拥有知识产权来保护他们的解决方案？
*   他们对自己的创业充满激情吗？
*   他们的文化是什么样的？
*   他们处于 pre-seed、seed 或 A 轮或更远的哪一轮融资？
*   他们的薪水是多少？这是你居住地区的市场价格吗？？
*   他们的薪酬福利是什么？
*   看了上面的信息，你的 **gut** 是什么感觉？

# 技术是锦上添花

我敢肯定，你可能会奇怪，为什么没有提到像 GraphQL，Web Assembly 或我个人的绒毛词，即区块链或人工智能的技术。

我个人认为，技术只是蛋糕上的糖衣，不管初创公司正在实施或使用的技术是什么味道。

我会看看上面的内容，并根据工作描述评估我的技能。

# 关注问题

软件开发总是致力于为用户解决问题。

创业公司和软件开发团队没什么不同。

只是它们作为一个具有多个活动部件的单一实体来实现主要目标。

如果创业公司的解决方案不能让你兴奋，我不会建议你不要加入那家创业公司。

因为你必须准备好至少 3 - 5 年的长期承诺。

做一些让你不兴奋的事情可能是一种痛苦，生命太短暂，你不能虚度。

# 为一家初创公司长时间工作

众所周知，你可能需要付出**额外的努力**去工作**长时间**来帮助初创公司取得成功。

因此，你可能需要做好心理准备，在长期的基础上聪明而努力地工作。

请记住，创业是一场马拉松，而不是 T2 短跑。

专注于聪明地工作**产生**结果**而不是为了短期和长期长时间工作。**

 **因为它可能对你的健康、生产力和精神健康有害。

# 结论

我发现有几个地方你可以研究这家初创公司，比如 [Triplebyte](https://triplebyte.com/) 、 [Keyvalues](https://www.keyvalues.com/) 、 [AngelList](https://angel.co/) 。

最后，为了加快对一家初创公司的研究，与认识这家初创公司任何人的开发人员交谈。

关注他们公司的技术博客或在该创业公司工作的软件开发人员，以获得更多关于该创业公司的信息，这将极大地帮助你进入该创业公司。
