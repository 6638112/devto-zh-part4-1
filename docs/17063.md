# 科技行业的多样性

> 原文：<https://dev.to/jcmc/diversity-in-the-tech-industry-4bjj>

我仍在思考我对这个话题的看法，因此我保留反驳我自己、改变我的想法以及在这篇文章和随后的每篇文章中犯错误的权利。

所以我是技术行业中被认为“多样化”的人。今年，我充分利用了我的特殊身份，申请了许多多元化会议的门票，参加了许多我原本无法参加的活动。但是每次我申请的时候，我都有点内疚，好像我不应该得到特殊待遇和一张神奇的免费机票。

我想起了过去的我，这证明了我自己是对的。如果你现在见到我，你会认为我自信、外向、随和、容易相处。)但 10 年前的我完全不同。由于许多情况，我不是我今天的一半。我非常害羞，不认识人的时候哪儿也不去，不做任何新鲜事。我肯定不会为我最近做的一些蠢事辩护。我为过去的我证明了我的多样性票的合理性，因为如果我真的敢离开家去参加一个会议，看到像现在的我这样的人，在那里变得快乐和正常可能会对我有一点帮助。

对我来说，促进科技领域(以及其他任何领域)的多元化，并不是要出去寻找合适比例的“多元化”人士来雇佣/参加你的活动/成为你的朋友……而是要创造一种氛围，让各种各样的人都感到受欢迎。这并不意味着将我们分成成千上万的小团体，包括科技女性、科技同性恋女性、科技穿鼻环的同性恋女性、科技穿鼻环的同性恋女性和科技中的多只猫……这意味着创造包容性的活动，制定明智的行为准则，在必要时实际执行。我不想去参加*仅仅是*同性恋编码者的聚会，我想去参加编码者的活动，他们中的一些人可能会也可能不会被认为是同性恋。

我不知道。我不是新一代人的一部分，他们的整个身份是建立在“他者”的基础上的。我是一个完整的人。我对许多学科都感兴趣，我有许多特点，其中一些可以归类为“多样化”。如果我想去参加一个技术聚会，我想去那里和任何想去谈论技术的人谈论技术。如果他们碰巧是某种少数民族，那对他们来说是好事。如果我想专门和同性恋交谈，我会加入一个同性恋聚会团体(事实上我已经这样做了，并且交了一些非常可爱的朋友。除了在他们计划婚礼的时候，我们很少谈论我们共同的同性恋。).

这篇文章和这个观点正在进行中。
