## [](#situation)🤔情况

让我们假设你有这样的代码，你想比较实例和会话的性能。

```
def current_role
  return @current_role if @current_role
  # ommit..
  @current_role = session[CURRENT_ROLE].reload
end 
```

## [](#instant-solution)🦄即时解决方案

你可以用这个代码来衡量。

```
def instant_performance(&target)
  start_time = Time.now
  yield
  Time.now - start_time
end 
```
