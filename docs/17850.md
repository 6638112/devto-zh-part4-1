# 如何将在军队中学到的技能，帮助我成为一名更好的网站开发人员

> 原文:[https://dev . to/vetswhocode/how-the-skills-learned in-the-military-will-help me-be-better-a-web-developer-3c1e](https://dev.to/vetswhocode/how-the-skills-learned-in-the-military-will-help-me-to-become-a-better-a-web-developer-3c1e)

当我的编码训练营即将结束时，我开始反思，不仅反思我在编码训练营中学到的 web 开发技能，也反思我在军队服役时学到的技能。正是这些技能将帮助我成为一名更好的网络开发人员。我在美国军队服役了八年，在两个不同的分支机构服役，美国海军陆战队和美国海军。每个分支教会了我不同的技能，这些技能帮助我成为今天的我。

海军陆战队以完成任务为荣。无论情况如何，你都必须不惜一切代价完成任务。有时候在野外你需要完成一些事情，但是天气很糟糕或者你没有合适的设备。等到天气变好或者你有了合适的设备，这本来很容易，但是你有一个必须完成的任务，而且必须按时完成。这给我灌输了完成所有任务并在截止日期前完成它们的动力。作为一名 web 开发人员，这个驱动程序将帮助我完成分配给我的任何任务，并按时完成。

海军为自己的领导地位而自豪。我学到的最重要的领导原则是，了解自己，寻求自我完善。我认为了解自己的优势和劣势很重要。这有助于你专注于你需要加强的领域。在 web 开发的世界中，不断寻求自我完善是非常重要的。你应该总是寻求改进你的技术，以新技术发展的速度，你很容易落后于你的主要编码语言的最新特性或者最新的趋势框架/库。

海军和海军陆战队都教会了我团队合作的重要性。在海军服役时，我是一名航空军械技师。独自举起重型导弹和 500 磅的炸弹是不可能的。需要一个团队将它们提升到飞机的机翼上。作为一名网站开发人员，你必须在一个团队中工作，能够一起完成项目是很重要的。有些项目太大了，无法独自完成。

在军队里，我周游了世界，体验了许多不同的文化。这些文化与我成长的北卡罗来纳州的小农业社区非常不同。但不仅仅是海外的不同文化，还有服役人员来自的不同文化。这教会了我如何与不同的人群合作，如何克服文化障碍。无论是面对面还是远程，这种经历都将有助于我与同事和客户共事。

军队教会了我许多技能，这些技能将帮助我成为一名更好的 web 开发人员。这些技能和我在编码训练营期间学到的技能一样有价值。要成为一名 web 开发人员，不仅仅是知道如何编码。你必须按时完成项目，不断学习新事物，成为一个好的队友，最重要的是，能够与各行各业的人相处和工作。
