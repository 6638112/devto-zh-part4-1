# PHP 中的决策。(模块 3)

> 原文：<https://dev.to/sannilincoln/decision-making-in-php-module-3-ie9>

像大多数编程语言一样，PHP 是用条件语句构建的，用于根据不同的条件做出决策。如果一个条件为真，if 语句将执行代码。语法是
if(条件){
要执行的代码；
}
if 语句适用于只有一个条件需要测试的情况。

如果条件为真，则 if else if 语句执行代码；如果条件为假，则执行另一个代码。语法是
if(条件){
要执行的代码
} else{
要执行的代码；
}
只需测试两种情况时适用。
当要测试的条件多于两个时，使用嵌套 if。语法是
if (condition) {
这个条件为真时要执行的代码；
} elseif (condition) {
第一个条件为假且该条件为真时要执行的代码；
} else{
所有条件为假时要执行的代码；
}
switch case 条件语句用于根据不同的条件执行不同的动作。switch 语句选择要执行的许多代码块中的一个。语法是
switch(a){
case label 1:
a = label 1 时要执行的代码；
破；
case label 2:
a = label 2 时要执行的代码；
突围；
case label 3:
a = label 3 时要执行的代码；
破；
默认:
a 与所有标签不同时要执行的代码；
}
第一个单一表达式(a)通常是一个变量，计算一次。然后，将表达式的值与结构中每个事例的值进行比较。如果表达式的值匹配任何一种情况，则执行与该代码关联的代码块。break 语句用于防止代码自动运行到下一个 case 中。如果没有找到匹配项，则执行 default 语句中的代码。
跳转语句是用于调节条件语句是否中断、继续或退出代码运行的关键字。break 语句通常用于 loop 或 switch 语句中。如果条件为真，代码将停止工作。
continue 语句跳过程序执行，转到下一个程序。exit 语句终止程序的当前执行流。
