# 做事情——直面自我，做点什么

> 原文:[https://dev.to/chantastic/do-the-thing-3n8a](https://dev.to/chantastic/do-the-thing-3n8a)

这是我的工作方式:

*   有主意了
*   坐下来工作
*   做一些不同的事情
*   让这个想法消失吧

当我决定制作开发人员培训视频时，我坐下来，在一个我不知道的狗屁网络框架中构建了一个多租户应用程序，用于销售视频订阅。

我*没有*做视频。

因为那件事，我错过了很多机会。但是我学到了一些关于工作的无价之宝。

创造是一场与自我的战斗。自我讨厌工作的侮辱。
它讨厌简单的东西。
当被迫工作时，自我将努力引向更大、更光明、更大胆的目标。

这就是为什么你的博客看起来像一个梦，但没有帖子。这就是为什么你的经理在向高管团队展示你的工作之前会说“还有一件事”。
这也是为什么在确定完美的基金之前，你不会为你的 401k 基金贡献一毛钱。

自我拒绝显得愚蠢。所以它用一个[高贵的障碍](https://www.goodreads.com/work/quotes/56765588-finish-give-yourself-the-gift-of-done)来对抗你的小想法。

反抗。保持低调。
做事情。
感到它的耻辱。
拥抱渺小。
不要让自我让你脱轨。
