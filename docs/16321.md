# 不要让你的恶魔征服你。

> 原文:[https://dev.to/flrnd/don-t-let-your-demons-conquer-you-1kk](https://dev.to/flrnd/don-t-let-your-demons-conquer-you-1kk)

封面图片:这是我离开广告界之前的最后一部作品。我现在可以展示的一张 Seat Ateca 活动的图片。我分享链接不是为了避免看起来像是自我推销，而这篇文章的情况并非如此。

我不习惯写私人的东西。我喜欢分享经历之类的东西，但我认为这对我和很多人来说很正常，把所有脆弱的东西深埋心底，永远不会向世人展示。

今天的故事不是关于`how to become a millionaire in a few steps`的故事，抱歉。相反，它是关于失败的。嗯，我个人的一些失败，但你知道，失败毕竟不是那么糟糕。

四年前，我可以高兴地说，作为一名设计师，我的职业生涯非常成功。我在一家大型广告公司担任艺术总监，主要从事汽车行业。与真正聪明的团队合作，为本田、西亚特、宝马、奥迪和大众集团的其他品牌以及时尚界效力，比如 Mango 或 Tarik Ediz 等等。

你知道，为这些广告公司工作，我首先应该考虑的是我签署的保密协议。我从来没有太在意过。薪水不错，我也有很好的关系，所以我从来没有超过连续三天没有工作。我为什么要担心？此外，正如我所说，因为我很幸运，我有良好的关系，我为什么要在乎有一个投资组合呢？嗯，我有一个，但它是一个老式的印刷文件夹，在一个很酷的皮革文件夹中，只是在我被要求展示我以前的一些作品的情况下，但这个文件夹充满了非常旧的作品，实际上没有什么有趣的东西。

现在，回到我之前提到的那些保密协议。除非你是一个*摇滚巨星*，否则你将不得不签署一份保密协议(嗯，可能不是所有的机构都有相同的做法，这是根据我与少数合作过的机构的经验)，你应该有一个好的律师在签署前审查它。这是怎么回事？分几点来说:

*   你所做的一切工作都属于他们，并且只属于他们。
*   在一段时间内，您不得谈论您正在处理的细节或产品(以避免泄露，例如，新产品或功能的推出)。
*   无论是在网上、网下、个人网站或社交媒体上，无论是在活动期间(见上一条)，还是在活动结束后，您都不得展示任何与该工作或活动相关的工件。这包括模型，过程拍摄，或最终艺术。我将在稍后谈到这一点。
*   你在向他们出售你的版权或任何你为他们所做的工作的权利，这是一篇很长的文章的素材，所以我会长话短说。对我来说，为换取我作品的版权而支付的金额足以让我答应！我会把我的灵魂卖给你！我太傻了，应该考虑在有限的时间内授予版权许可，但是我可能一开始就不会为他们工作。

回到不被允许展示你来自这些机构的作品，作为一个设计师的问题是你的作品集就是你的展示信。如果没有一个强大的视觉组合，你有多好(或者你认为你有多好)都没用。你属于这样一个行业，在这个行业里`visual`就是一切，而且分量很重。

一些创意总监已经知道我和我的作品，所以不需要作品集。一些以前的同事开始制作个人网站。他们有和我完全一样的保密协议问题。但是他们更聪明。我以前的同事，使用这些组合网站来展示他们的技能，而不是他们的工作，附带项目，罕见的想法。很多优秀的东西。我在另一边，没有太在意。

社交媒体开始成为一种事物。更聪明的人会打理他们的 Instagram 账户、他们的粉丝、脸书等等等等。我从来没有关心和忽视这一切。是的，你可以接受，你不欠社交媒体什么。但这就像第一次工业革命。如果你不适应新的时代，你就会落后。

那么，我怎么了？我母亲四年前病得很重。我真的不想在这里进入细节，因为这仍然是一件非常难以接受的事情，但为了让你了解背景，她因心脏肿瘤引发了两次脑中风，现在她患有失语症，这意味着她需要护理。(对于那些好奇的人来说，谢天谢地，她现在非常健康)。

在离家近 900 公里的地方，保持我的工作和这种状况真的很难。可能是在我职业生涯的巅峰，我花了一些时间来决定该做什么。我知道一个事实，我以前的许多伙伴和同事，在同样的情况下，会把她送到托儿所，继续他们的生活。但你知道吗，我欠她太多了。所以，去他妈的我的事业！

如今，如果没有强大的视觉组合，就别想在设计行业找到一份体面的工作。你想象不到我有多少次后悔没有在那么久以前照顾好。因为我没有几年前那样的精力去开始一个求职的过程，我以为那是自己创业的好时机。会有什么问题呢？我后背有很多经验！

我的天啊。我错了！我开了一家设计室。结果:一年前我关闭了它，或多或少从一开始就有很多错误。现在回想起来:

*   我的价格高得离谱。我像以前在大城市一样给我的服务定价。第一个巨大的错误是，不了解我正在介绍的市场。
*   是一个非常糟糕的推销员。这是一个错误中的两个。我应该雇一个好的。
*   为一些客户做了很多免费的工作，因为我喜欢他们，非常天真。我不后悔，我甚至还交了几个朋友。但从商业角度来看，这是一个糟糕的商业策略。
*   从没解雇过我以前的客户，我应该解雇几个。他们花了我很多钱，宝贵的时间，最重要的是:心理健康。
*   试图模仿大机构如何在简报和演示中舔客户的屁股。这个想法在理论上是好的。嗯，结果是浪费了很多时间。事实上，当他们知道要花多少钱时，他们投入了很多时间去拒绝。我试图把错误的产品卖给错误的顾客，一次又一次。
*   开始沉迷于糟糕的设计。真实的事实:我们被设计糟糕的东西包围着，无处不在，从商标到小玩意，你需要学会适应它。
*   我忘记了一件重要的事情:生活。连续近三年，我没有假期，没有空闲时间或空闲日。

更多的陷阱，你可以想象。做一个企业家是很严肃的事情。难怪只有少数人真正成功。

我现在在哪里？在我关闭我的设计室后，我想要一段属于自己的时间，我真的需要那段时间。

在我居住的一所小型设计学校教书，工资不是来自另一个世界的东西，但有助于支付房租，不会很快花光我的积蓄。这也给我留下了大量的空闲时间，我可以和家人在一起或者重新开始编程。我甚至有时间写作，你自己也看到了，这不是我最擅长的。但是完全没问题。考虑到我的母语不是英语，我意识到我需要大大提高我的写作水平。但是我没有让我的怀疑、恐惧和陷阱决定我下一步要做什么，而是继续前进。不管怎样！我不会让我的恶魔征服我。

生活中没有比自己的错误更好的学校了。
