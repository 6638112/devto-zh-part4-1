**什么是 div joy**

Divjoy 是一个“基于 web 的 react 代码库生成器，帮助您为您的项目创建完美的代码库”。想象一下，你有一个项目，你需要尽快制作一个 MVP，你真的想专注于实现你的业务逻辑和快速设置你的内容。

即使在这一点上，你肯定希望有一个非常稳定的代码库，而不是担心使用什么库进行授权服务、路由和关于 react 东西的辩论…你知道这种做法，这是 divjoy 的精髓——我喜欢这个杜撰的短语“用类固醇创建 react 应用程序”——一个稳定的 react 代码库及其功能，我们很快就会谈到。
