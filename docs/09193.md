# 编码训练营值得吗？

> 原文:[https://dev.to/josylad/are-coding-bootcamps-worth-it-kpk](https://dev.to/josylad/are-coding-bootcamps-worth-it-kpk)

如果你从事技术或计划从事技术或软件开发，我相信你一定听说过“编码训练营”这个词，你可能想知道它是否值得大肆宣传！

好吧，我要帮你澄清你在这个故事中的疑惑！

### [](#what-the-heck-is-a-coding-bootcamp)什么是编码训练营？

对于外行来说:

编码训练营是一个技术培训项目，教授雇主需要的编程技能。编码训练营使不太精通编码的学生能够专注于编码的最重要方面，并立即应用他们的新编码技能来解决现实世界的问题。

换句话说，编码训练营是软件开发的密集项目。这就像学习编码的马拉松，一种在短时间内掌握编码的不间断的学习方法。

在本文中，我将与您分享我刚刚完成的为期 5 周的编码训练营的经历，**这个为期 5 周的训练营专注于让我们成为前端开发人员**，我将根据自己的经历分享参加这样的训练营的利弊。

### [](#my-story)我的故事

嗯，我一直热爱科技、数字营销和所有与之相关的东西，但我从未真正能够完善我的编码技能；我使用像 WordPress 这样的 CMS 建立网站，但是我相信我可以做得更多，所以我决定尝试一个编码训练营。

当然，我有一堆关于 HTML、CSS、JavaScript 等等的书，甚至我曾经为 sitepoint premium 付费(我猜是三次)仍然无法学到任何扎实的东西，所以我发现独自在黑暗的房间里阅读电子书可能不是我学习编码的最佳方式。

我需要在一个学习的环境中，和那些激励和鼓励我尽快学习的人在一起，所以，我决定参加一个编码训练营。

两年前(2017 年)，我在肯尼亚内罗毕听说了一所特殊的编码学校，所以我再次检查了它们，并决定收拾行李离开尼日利亚去内罗毕参加这个编码训练营。

***对，没错！我去了另一个国家学习！*T3】**

也就是说，我想以我的经验为基础，与你分享参加编码训练营的好处。

### [](#pro-1-fast-learning)Pro 1:快速学习

我发现，由于我有 5 周时间来掌握前端开发并完成日常内容，我必须学得相当快；在训练营的每一天，都有需要掌握的主题和概念以及必须完成的**练习练习！**

[![](../Images/b8cfda7886b1975de893e90e46fcd87d.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Y70CBkG_--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/0%2A5SBm8SghalNWVo07)

哈雷戴维森在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片

在训练营开始之前，我对 Git 做什么以及如何有效地使用 Github 版本控制几乎一无所知，但是在训练营的两天之后，我已经在创建 Git Repos，分叉 Repos，创建分支，合并分支，并从 Git 中获得乐趣。

此外，到第二天，我已经能够自如地使用终端(命令行界面)来创建目录、创建文件、打开和安装程序等等。

我正在做一系列一周前做梦都想不到的事情，我就像一个混账忍者，我能在 2-3 天内把这些都想出来。

因此，当谈到速度和快速学习时，我相信编码训练营肯定会帮助你。

### [](#pro-2-less-distractions)Pro 2:杂念少

在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上由 [Chase Clark](https://unsplash.com/@chaseelliottclark?utm_source=medium&utm_medium=referral) 拍摄的照片

编写训练营代码不会让你分心。当你进入一个编码训练营时，你可以感受到气氛中的严肃和“严肃”态度，人们花了很多钱在那里，所以没有人在那里开玩笑(除了少数人)；你在那里是为了一项严肃的工作，在场的其他人也是如此。

在训练营中，你和一些有相似目标的人在一起，这些人实际上对编码和学习新技术非常认真，对分心持零容忍态度；所以集中注意力将是今天的重点！

更少的分心肯定会让你学得更快。

### [](#pro-3-support)Pro 3:支持

对训练营进行编码可以让你获得实际的、有保证的支持。

通常在训练营中，你会有更有经验的开发人员导师，他们随时准备在你遇到困难时帮助你，这让你放心，让你觉得你并不孤单；它还允许你去试验和尝试，因为你知道有人可以帮忙！

因此，无论何时你没有得到一个概念或者你的一些代码不工作，你可以很容易地获得帮助并尽快解决它，而不会减缓你的学习。

### [](#pro-4-atmosphere-and-motivation)Pro 4:氛围和动力

编码训练营的气氛通常是非常不同的，就像你们正处于志同道合的人一起工作来构建下一个大东西。

这种氛围为每个人创造了无与伦比的动力，即使你感到疲惫，这种氛围也会让你坚持下去。

当你想退出的时候，看看房间里其他的新兵训练营成员；各行各业的年轻人和老年人都在努力实现他们成为软件开发人员的梦想，你会立即找到继续前进的力量！

### [](#pro-5-team-work)Pro 5:团队合作

在训练营的时候，我们被鼓励成对学习和编码，我相信这实际上是一件好事。所以，你挑选一个伙伴，你们坐在一起，开始学习一天的内容；问问题，互相帮助。

我发现成对或成组编程更有趣，你可以分享想法，从你的同伴那里得到关于不清楚的概念或主题的不同意见和解释。

你还可以结识各行各业的人，与他们建立关系网，学会如何与来自不同部落、宗教和种族的人交往，这是当今职场的一项重要技能。

### [](#pro-6-practice-projects)Pro 6:练习项目

在编码训练营期间，你会有很多练习项目。

我们有每周的项目，必须在周末完成，并在周日午夜前提交。这确保了每个人通常都可以练习他们在一周内学到的东西，并编写他们自己的个人项目，以提高他们的学习和掌握所学的任何新概念。

相信我，成为一名优秀的软件开发人员或程序员的主要秘诀是**“练习”**，如果你不练习，你就无法提高，而编码训练营将确保你练习。

作为一名软件开发人员，你的技能发展在很大程度上取决于实践，构建素材，你肯定会在编码训练营中做很多这样的事情。

因此，这些是你可以从编码训练营中得到的一些好处。

对于缺点，我将简单谈谈我认为对训练营编码最重要的两个缺点。

### [](#con-1-financial-commitment-cost)缺点 1:财务承诺/成本

我认为参加编码训练营的主要问题是培训费用，编码训练营很贵！

一个便宜的编码训练营的费用在 1000 美元到 2000 美元之间，取决于你所接受的培训的持续时间和质量；一些编码训练营的费用高达 1 万美元。

当你将这一成本(不包括其他个人/杂项成本)与电子书或在线视频课程的成本进行比较时，你会发现成本差距非常大。

但是相信我，你通常从编码训练营中获得的价值应该是值得你投资的。

### [](#con-2-intensiveimmersive-training)缺点 2:强化/沉浸式训练

编码训练营是一种密集的沉浸式训练；事实是，当你在一个编码训练营时，你将没有时间做任何其他事情。

你将把大部分时间花在编码训练营上。通常，你每天至少要花 12 个小时在训练营，这意味着你不能把它和其他任何事情结合起来。

如果你在工作，你可能需要在训练营期间休息一下；你也可能不得不离开酒吧一段时间！

### [](#conclusion)结论

编码训练营值得吗？

***绝对是的！*T3】**

如果你能负担得起，如果你有时间，我会强烈建议你去。

编码训练营将增强你的学习能力，并推动你更接近成为软件开发人员的梦想！
