# 了解支持 macOS 64 位的最新 EntityDAC

> 原文:[https://dev . to/devart software/meet-an-updated-entity DAC-with-support-for-MAC OS-64-bit-55bf](https://dev.to/devartsoftware/meet-an-updated-entitydac-with-support-for-macos-64-bit-55bf)

*Devart 发布了新版本的 [EntityDAC](https://www.devart.com/entitydac/) v.2.2，支持 macOS 64 位并改进了功能。*

[![EntityDAC macOS Support](../Images/4623f1ab56fad052b4f828d94bca2435.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--TMyJJz3K--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i.ibb.co/zbJ5fNp/ENTITYDAC.png)

Devart 是一家公认的数据库连接解决方案供应商，为 Embarcadero Delphi 和 Microsoft 提供服务。NET 开发平台发布了其 EntityDAC 的新版本，带来了几项期待已久的改进。

最值得注意的产品更新是支持 macOS 64 位操作系统。现在，用户可以为 macOS 创建 32 位或 64 位版本的数据库应用程序，以使技术符合业务需求。此外，开发团队还为 GUID 生成器添加了对 Bytes 属性的支持。

考虑到苹果公司计划在未来版本的 macOS 中放弃对 32 位应用程序的支持，这次更新来得正是时候。今年 6 月，苹果公司证实，新的 macOS Catalina 将禁止用户安装 32 位应用程序。

EntityDAC 是一个 Delphi ORM 框架，用于使用 LINQ 将数据库对象快速对象关系映射到 Delphi 类。在 https://blog.devart.com/entitydac-supports-macos-64bit.html 的[了解更多关于最新版本的信息](https://blog.devart.com/entitydac-supports-macos-64bit.html)
