# 社交:这很像约会

> 原文:[https://dev . to/aidiri/networking-it-s-a-lot-like-dating-2d jj](https://dev.to/aidiri/networking-it-s-a-lot-like-dating-2djj)

人际关系网是一条令人望而生畏的建议，经常被抛出。这种现象在科技领域尤为普遍，在这里，人际关系网似乎是找到你想要的工作的唯一途径。网络的问题在于，对大多数人来说，它是社交焦虑、困惑和自我怀疑的怪物。每个人都相信他们是唯一受苦的人，而其他人对他们的人际关系网充满信心。事实是，我们都可以对网络感觉良好。我们只需要换个角度。

但何必呢？难道不应该以你的代码质量和你的代码本身来评价你吗？是的。但只是在招聘人员懒得看一眼之后。在简历上把一个名字和一张脸对应起来有助于招聘人员看到你，而不是一张纸。如果你遇到招聘人员的话。招聘人员只是你将会遇到的人中的一小部分。

人喜欢人。(因此，招聘人员也喜欢人。)简历不是人。这里没有个人联系。人际关系网让其他人了解和联系你，这个人，而不仅仅是你简历上列出的那些非常有效的资格。当人们喜欢别人时，他们想帮助别人。

这就是人际关系网如何让你抓住机遇。人脉可以给你介绍工作机会、导师或学员、内部知识、建议和忠告，或者朋友。

我一直认为人际关系网是那些试图从有恩惠的人那里赢得恩惠的商人的奇怪的闲谈聚会。社交活动中充满了在各自领域出类拔萃、对自己能力充满自信的人。我相信它，因为从来没有人告诉过我相反的情况。毕业多年后，我才意识到自己在求学期间拥有的社交机会。我还意识到，没有一次网络是以它是什么来命名的，更不用说如何去利用它了。

当我参加西雅图的女性科技会议时，我终于揭开了网络的面纱。会议重点关注网络的重要性，到了周末，我是一个真正的皈依者。网络就是与人联系。就是这样。不是闲聊，不是大事，甚至不是找工作。而是寻找关系。那就是你开始约会的时候。

无论你如何看待网络，现实要宽容得多。但是，就像约会一样，用错误的心态和方法，可能会相当糟糕。所以让我们改变这一点。

不要期望找到你的灵魂伴侣——你梦想的工作。更有可能的是，你会遇到一个你喜欢到想要成为朋友的人，仅此而已。这太棒了！你刚交了个朋友。去你的！

你不会为了找工作而参加活动，尽管这种事情可能会发生，而且经常发生。很高兴知道，当你参加和参与活动时，你能做的最糟糕的事情就是交一个朋友。因为这才是重点。你去那里是为了结识新朋友，让你的朋友群体壮大。因为:

你的朋友可能会向你介绍你最重要的关系。在活动中结识新朋友会扩大你的个人关系网。然后通过保持这些联系，你可以通过你已经拥有的联系扩大你的关系网。通过这些关系中的一个，有人会知道你有一个机会。另一方面:

关系是双向的。你的人际网络越大，你与他人分享的机会就越多。你会很自然地做一些事情，因为，你猜怎么着，你是人。我们知道人们喜欢他人，当这种情况发生时，人们想要帮助他人。另外，只有一个人的关系没什么好的。不要成为那样的人。

不要强迫一段关系。不要试图去了解某人，因为他们有你想要的东西，或者认为这是一个很好的联系。即使你努力了，他们也会意识到你不真诚。四处聊天，找一个你喜欢的人，一个你一拍即合的人。如果你点击，然后看看你们是否有任何共同点，任何相似的兴趣。这是一个你能与之保持对话的人吗？有共同经历的人？恭喜你。你正在建立新的联系。不要忘记交流信息，并在活动结束后保持对话。

如果有人不喜欢你，不要往心里去。有些时候，有人不会回报你的兴趣。可能你不是他们现在要找的人。或者可能是时机不对，他们还没有准备好任何严肃的事情。否则，当好事摆在他们面前时，他们可能看不到，他们错过了和你在一起的机会(我的意思是，不管怎样，忘记他们吧，阿米利特？).

事情会有点尴尬。这是社交活动的固有属性。每个人在某些时候都明白这一点。所以不用担心尴尬。每个人都和你在同一条船上。你看到的那些看起来超级自信的人实际上是那些拥抱尴尬可能性的人。

现在向前迈进:参加社交活动，做你自己，不要因为某人的蹩脚笑话而强颜欢笑。玩得开心，交朋友。因为，迟早，你会帮助你的人民，你的人民也会帮助你。
