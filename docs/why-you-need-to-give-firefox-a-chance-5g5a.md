# 为什么你需要给火狐一个机会

> 原文：<https://dev.to/dtroode/why-you-need-to-give-firefox-a-chance-5g5a>

我们生活在浏览器垄断的时代。谷歌凭借 Chrome 占领了整个网络。Chrome 拥有超过 50%的用户:
[![Browsers top](img/d4fd1f2ee3ceab2427348884401d4712.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--0wWS3NWd--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/fz6tvstqupgup3ijw9t5.png) 
Safari 之所以坐上第二把交椅只是因为 macOS 的普及。

现在，大多数开发者在 Chrome 和 Firefox 之间选择。2017 年，Mozilla 发布了他们的 Firefox 更新版本——Firefox Quantum，具有新的定制设置，更好的扩展支持，隐私，更重要的是速度。比老款火狐快 2 倍，比 Chrome 快。
[![Firefox Quantum vs Chrome](img/c328ab0a3a5ed50656bd1fc60518993a.png)T3】](https://res.cloudinary.com/practicaldev/image/fetch/s--d9fF_-rn--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/oj5ozdji2frqxjg7oebs.png)

Firefox 使用他们自己的新引擎，新的 Quantum CSS 和 Quantum DOM 渲染。很多解释[这里](https://hacks.mozilla.org/2017/11/entering-the-quantum-era-how-firefox-got-fast-again-and-where-its-going-to-get-faster/)。Chrome 占用大量内存，而 Firefox 占用较少。

此外，Firefox 有很多有用的功能，比如[显示无用的 CSS 规则](https://twitter.com/nicolaschevobbe/status/1135798960361263104)并支持[偏好颜色方案:深色](https://hacks.mozilla.org/2019/05/firefox-67-dark-mode-css-webrender/)媒体规则，Chrome 现在只在测试版中支持这些。
[![](img/7fd353eac9fecbe9ef4feeaff71e314f.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--nhksd5Nx--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/mpf3dc4ndob5kiwfc4yo.png)

你害怕延期吗？现在，大多数开发者都为 Chrome 和 Firefox 编写扩展。

最后一个原因——停止垄断。给其他规模较小的公司一个机会，你就能看到 frontend 是如何发展的。

将你在 Firefox 中的所有书签移动一周，试试看。
