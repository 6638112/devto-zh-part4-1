## 什么是 Node JS？

Node 是一个跨平台的运行时环境，用于在浏览器之外执行 JavaScript 代码。Node 不是一种编程语言或框架。它是执行 JavaScript 代码的运行时环境。10-15 年前，JavaScript 仅在客户端应用程序上起作用，今天，您可以构建完整的堆栈(前端和后端应用程序),而无需离开 JavaScript 环境。

**NodeJS 是浏览器之外的 Javascript。**

2009 年，Ryan Dahl 采用了谷歌 Chrome 的 V8 引擎，该引擎通常仅限于浏览器，添加了一个事件循环、包管理器和一个低级 I/O API，嵌入在一个 C++程序中，并在他的计算机上运行。Node.js 开启了一个全新的世界，JavaScript 可以做什么:访问文件、监听网络流量和 HTTP 请求、直接访问数据库等等。

基本上，Node.js 允许 JavaScript 做 PHP、Ruby on Rails 或 Python 可以做的任何事情
