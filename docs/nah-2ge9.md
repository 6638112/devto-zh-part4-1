# “不”

> 原文：<https://dev.to/justzeros/nah-2ge9>

我走路时拖着脚，这意味着我有点拖着脚在地上走。正如我父亲所说，这导致了“拖拖拉拉”。

你可以把这种移动想象成一个系统，当我的动作(移动脚步)在一个合适的环境(松散的泥土或沙砾)中遇到一个特定的反应(泥土或沙砾让路)时，这个系统就平衡了。

这也意味着我经常绊倒，因为我被拒绝移动的岩石绊倒。

绊倒使我暂时失去平衡，双臂张开，迈着巨大的步子，试图恢复平衡。

当系统中通常移动的部分说“不”(就像石头拒绝移动)时，系统就会“跳闸”。它失去了平衡。

系统力求平衡(体内平衡)，所以它立即试图重新获得平衡。

这可能意味着无论说“不”，都要更努力地推动

或者，干脆忽略它，下次尽量避免。

或者调整以找到在这种环境下工作的新平衡。

今天，你可能想改变一些系统。

也许第一步就是做那块不动的石头。

说“不”，让系统失去平衡，看看接下来会发生什么。

听起来很冒险？当然，可能是。

但是事情会一直持续下去，直到系统变得不平衡，不得不寻找新的平衡。

改变从不平衡开始。

今天，你想对什么说“不”？

系统会如何应对？

如果你愿意，请给我回信——希望听到你的想法。
