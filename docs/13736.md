# 快速点击:{waffle} 1.0 字体真棒 5 象形图和更多

> 原文：<https://dev.to/hrbrmstr/quick-hit-waffle-1-0-font-awesome-5-pictograms-and-more-4dg6>

{waffle}软件包本周获得了一些 [![💙](img/446bbc1b1aa234d3736c619764cb5876.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--rLSTZr4t--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://s.w.oimg/core/emoji/12.0.0-1/72x72/1f499.png) ，现在有了一个大幅改进的`geom_waffle()`以及一个全新的兄弟函数`geom_pictogram()`，它拥有`geom_waffle()`所有强大的新功能，但允许您使用字体 Awesome 5 品牌和实心字形来制作同型象形图。

[点击观看幻灯片。](https://rud.is/b/2019/07/12/quick-hit-waffle-1-0-font-awesome-5-pictograms-and-more/#gallery-12403-1-slideshow)

一个主要的新特性是`stat_waffle()`(驱动两个 geoms)有一个自动计算比例的选项，这样您就可以使用一个合适的 10×10 网格来显示整体的一部分，而无需做任何额外的工作(也适用于小平面环境)。

你可以预览下面的插图，或者打开 iframes 来看看[华夫饼](https://rud.is/rpubs/building-waffle-charts.html)和[象形图](https://rud.is/rpubs/building-pictograms.html)的使用。
