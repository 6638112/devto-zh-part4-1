# 带有白天或夜晚插图的时间选择器

> 原文:[https://dev . to/takaneichinose/time-picker-with-day-or-night-illustration-4lj](https://dev.to/takaneichinose/time-picker-with-day-or-night-illustration-4lj)

这是一个简单的 VueJS，和一个时间选择器的概念。它可以用于实际的应用程序，或者只是一个简单的时间选择器，用于基于 web 的应用程序，或者网站(或者任何与 html，css，javascript 相关的东西)。

如果时间在早上 6:00 到下午 5:59 之间，图示将变为“日”。否则，插图将变为“夜晚”。

i18n(内化)包括在内，虽然我不会说很多语言，所以我只是在这里写了我说的所有语言。
