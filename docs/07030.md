# 每日挑战#45 -更换机器

> 原文:[https://dev . to/the practical dev/daily-challenge-45-change-machine-414 I](https://dev.to/thepracticaldev/daily-challenge-45-change-machine-414i)

今天的挑战是实现一个`change`函数，它将接受一个表示分的整数参数。该函数应该使用最少数量的硬币返回最佳变化。

该函数还应该为美国货币的每枚硬币(特别是 25、10、5 和 1 枚硬币)返回一个密钥。每枚硬币的价值应该代表零钱中每枚硬币的数量。未包含的每个硬币的值应该返回 0。
