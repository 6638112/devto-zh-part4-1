# 打电话-在拨号屏幕中打开电话号码

> 原文：<https://dev.to/skptricks/react-native-make-a-phone-call-open-phone-number-in-dial-screen-3jak>

来源: [React Native 拨打电话–在拨号屏幕中打开电话号码](https://www.skptricks.com/2019/07/react-native-make-phone-call-open-phone-number-in-dial-screen.html)

本教程解释了如何使用 react 本机应用程序中的链接组件在 react 本机应用程序中拨打电话。链接为您提供了一个通用的界面来与传入和传出的应用程序链接进行交互。在移动应用程序中，您可能多次看到有一个呼叫按钮，当您按下呼叫按钮时，它会在移动拨号键盘屏幕上打开电话号码。这一功能将使您在我们的应用程序中更加容易，并有助于从应用程序中进行调用。
