# 安全冠军

> 原文:[https://dev.to/farmanp/security-champion-1h6l](https://dev.to/farmanp/security-champion-1h6l)

从明天 2019 年 7 月 8 日开始，我将作为安全冠军与安全团队一起工作！这真的让我很兴奋，因为这是我在朋友和家人中宣传了一段时间的一条道路的开始。有些人可能已经知道了，但我无法参加几个月前参加的网络安全训练营。很艰难，但至少这不意味着结束！

那么，什么是安全冠军？为什么这很重要？好吧，让我们简要地介绍一下定义:

*安全冠军是公司中帮助和协助安全团队的个人，是跨职能团队的联络人。*

对我来说，这将是我们的生产支持团队。安全冠军来自 OWASP，它是开放 Web 应用程序安全项目的通用标准。要了解更多信息，这里有一个链接，链接到一名[安全卫士](https://www.owasp.org/index.php/Security_Champions)的职责

现在，我要诚实地告诉你:我喜欢参与任何事情:UX 设计、2D/3D 动画、工程制图、数据科学、安全等。我喜欢认为自己是 T 型的！我希望从成为多面手中获得的是出色的视觉沟通。对我来说，视觉和书面交流是我最好的媒介，但我一直希望我的创造力是有目的的。我四处搜寻，发现了将我的技能和经验投入网络安全的最重要的机会！这是我们这个时代最重要的领域之一。

但是我跑题了。让我们回到安全冠军。以你的身份来:分析师、开发人员、警卫、经理、看门人、消费者等等。这是每个人的责任！如果不是为了安全:网络安全，就当这是看门狗吧！我们需要更多的人作为倡导者。虽然安全是安全团队的责任，但它也是每个人的责任。这就是我认为它最迷人的地方！就像成为网络超级英雄一样。

我希望在接下来的几个月里报道的是:

*   倡导成为公共安全冠军
*   教育大众他们应该知道的安全故事
*   提供进入门槛低的车间
*   提高网络安全素养

当我开始这个旅程时，我必须问那些阅读和关注这些帖子的人:关于网络安全，你想了解什么？
