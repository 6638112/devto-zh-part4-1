> 如何使用这款 Gastby 启动器

```
npm install -g gatsby-cli # install Gatsby CLI

# clone the app
gatsby new gatsby-starter-scss https://github.com/app-generator/gatsby-starter-scss

# start the app
cd gatsby-starter-scss/ && gatsby develop

# Build for production
gatsby build 
```
