# 实用程序员 20 周年纪念版系列

> 原文：<https://dev.to/steadbytes/the-pragmatic-programmer-20th-anniversary-edition-series-1e2l>

> 这篇文章最初出现在[steadbytes.com](https://steadbytes.com/blog/the-pragmatic-programmer-20th/intro/)

> 二十年来，来自实用主义程序员的经验帮助一代程序员检查软件开发的本质，独立于任何特定的语言、框架或方法。
> 
> - [实用主义书架](https://pragprog.com/book/tpp20/the-pragmatic-programmer-20th-anniversary-edition)

戴维·托马斯和安德鲁·亨特所著的《实用程序员》的最初版本对我和(毫无疑问)许多其他人来说是一本极具影响力的书。然而，尽管这本书的大部分*精华*在今天仍然适用，但是大部分的技术参考已经非常过时了。作者认识到这一点:

> 但是对于软件来说，20 年是很长的时间。把一个 1999 年的开发人员放到今天的团队中，他们会在这个陌生的新世界中挣扎。但对今天的开发者来说，20 世纪 90 年代的世界同样陌生。这本书对诸如 CORBA、CASE 工具和索引循环之类的东西的引用充其量是古怪的，更可能是令人困惑的。
> 
> ——《第二版序言》[实用主义程序员 20 周年纪念版](https://pragprog.com/book/tpp20/the-pragmatic-programmer-20th-anniversary-edition)

因此，当我听说 20 周年纪念版将包含全新的主题以及原书的修订/更新主题时，我真的很兴奋！新版遵循与原版相同的格式，每个部分都有**挑战和练习**——非常适合一个博客系列。在这个博客系列中，我将提供我自己对挑战和练习的答案，以及对一些主题的思考/分析。

在这篇介绍的最后，我要感谢戴维·托马斯和安德鲁·亨特对《实用程序员》的专注工作，我鼓励读者从[实用书架](https://pragprog.com/book/tpp20/the-pragmatic-programmer-20th-anniversary-edition)购买这本书。

*   请注意，在撰写本文时，这本书还处于测试阶段，因此在正式发布之前可能会进行一些小的更改和更新(所有这些都将在更新公开后提供给测试版购买者)。
