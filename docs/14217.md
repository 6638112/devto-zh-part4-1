# 母体中的生命

> 原文:[https://dev.to/johnhalsey/life-in-the-matrix-e2f](https://dev.to/johnhalsey/life-in-the-matrix-e2f)

拜托，我们都做过。我们在编码的时候都假装自己在矩阵里。我不是说只是假装在电影里，而是真的在矩阵里，在那里我们可以看到我们周围的世界，以代码的形式，滚动字母和数字，看起来对我们有意义，似乎没有其他人。

我很幸运能从事我喜欢的工作，成为一名网站开发人员，每天花~~所有的~~ ~~大部分的~~时间编写代码，创造新功能是一件非常酷的事情，但这也意味着很难向家人或朋友描述我到底在做什么。有时当我和麻瓜(非魔法/网络人)说话时，我不应该这样，但我确实会稍微强调一下。

“约翰，你是做什么的？”
“我是一名网页开发者”。没什么，我的意思是只是茫然地看着他们，而他们却在想些聪明的话来说。
“那是什么？”游戏开始。
“我教蜘蛛如何制造更好、更有效、更复杂的网，以便捕捉更多的祈祷，从而更有效、更有利可图。”没什么。
“哦对了”。

当然这在一段时间内很有趣，但最终它确实让我看起来有点古怪。这真的不公平，因为它只是部分正确。问题是，即使是知道我做什么的人，也很难理解其中的细节。当我试图向我的妻子(麻瓜)解释一件特定的作品时，我可以看到她在某种程度上很感兴趣，但随后她开始点头并发出“嗯嗯”的声音，她的眼睛闪闪发光，同时她直视着我，进入了无聊和白日梦的黑洞深渊。

所以我尽可能长时间地说话。

她不再问我今天过得怎么样了。

还有一些人知道我做什么，知道它涉及什么，甚至对编程如何工作有一些基本的了解，或者可能做过一些真正简单的编码，或者以前看过一次 HTML。这些人很有趣，因为他们认为他们“明白了”，并会试图挑起关于我可能正在进行的项目的对话，或者他们如何“本来可以成为一名开发人员”，“想过成为一名开发人员”，或者“有一名家庭成员/朋友是开发人员”。(哑炮)。这些人实际上很好相处，可以就基本的编码概念或基本的 HTML 和 Javascript 进行合理的对话，但是问他们关于制表符和空格的问题，他们的游戏就失败了。他们就像一个胖手指的铅笔推进器用冷茶浸泡的湿湿的消化饼干，食物在她的胡子里。

实际上，在《黑客帝国》中，我并不担心我是否能全面解释我的谋生之道，或者是否有人感兴趣。说实话，我不关心您做什么，所以您为什么要关心 XSRF 令牌是什么，或者 418 响应状态真正意味着什么？

与 dev 交谈有点像与 Masons 握手，你周围的每个人都能听到你在说什么，你在说英语，如果他们愿意，他们也可以与我交谈，但对话非常不同，这是一个秘密。

一个开发伙伴可能会说，“我花了一个上午的时间修复了一个错误，这个错误阻止了侧边菜单在移动设备上正常滑动，这意味着导航完全倾斜了”。或者“我已经清理了我的应用程序中的一些控制器，为类似的方法创建了一个特征，现在它要干燥得多了”。合理的开发谈话。

麻瓜可能会说“你和电脑打交道，对吗？你知道我为什么连不上打印机吗？”我在这里说的任何话都不会让我当场被解雇。因此，我不可避免地要浪费三个小时的时间，试图连接一台我看得见、摸得着、闻得见的打印机，但结果是它没有打开。它甚至没有插上电源。一旦它开始运行，他们说“非常感谢”(如果你幸运的话)，我会回答“我很乐意”。但当然不是，我会不惜一切代价用我的余生来躲避他们。

我喜欢尽可能多的呆在矩阵里，在数字世界里，我可以像上帝一样随心所欲地控制和改变元素。是的，现在想想，我基本上是互联网之神。我给予，也拿走。我以前从来没有这样想过，但这很有道理。想想看，起初上帝创造了天地。(框架和数据库迁移)，上帝说，“要有光”，于是就有了光(数据，最初是播种的)，并且看到它是好的(被验证)。我认为这是不言自明的，为什么你认为任何人学习开发的第一件事是 Hello，World 应用程序？

矩阵中的时间也是一件有趣的事情，因为当你在那里时，它比正常情况下走得快得多，这意味着你没有时间做任何事情。这就像《盗梦空间》的反面。说真的，我会在下午 14:00 进去，在你知道之前，已经是第二天晚上五点了！这使得估计任务工期更加困难。做一个小的盒子面板，里面有一些文字，可能有一张图片，还有一个链接，真的要花多长时间？哦，它需要有反应，有动态的内容，只有在一天中有一个' s '出现，并称赞你使用互联网的能力？比你想象的要长。为了确保万无一失，我倾向于在任何估计的基础上增加四个小时。如果是星期五的话，五分钟。如果是星期一，六点。如果是星期二或星期四，七点。

查看我的计算机监视器，特别是我的代码编辑器(PhpStorm，还有什么？)，您会看到不同代码元素的深色背景和冷色调。红色代表操作符和语句，黄色代表文本字符串，蓝色代表类，橙色代表属性，白色代表变量等等。我更喜欢《黑客帝国》的深色主题，因为这样更容易阅读代码。但是现在事情是这样的，麻瓜甚至其他开发者认为因为我喜欢黑暗主题编辑器，所以我喜欢远离事实的黑暗模式。实际上，我更喜欢一个正常的，明亮的，白色的主题。我没有把我的 macOS 设置为黑暗模式，我没有把 Chrome 设置为黑暗模式，或者其他任何东西。一直处于黑暗模式就像关着灯在你的房子里走来走去，或者晚上在没有灯的高速公路上开车，因为这很酷。不，不是，这很愚蠢，每个人都应该马上停止。

黑暗模式就像学校里新来的给你糖果的酷小孩，一开始你可能会喜欢，然后你开始觉得恶心，很快就开始讨厌它，直到你意识到，除非你遵守他的规则，否则实际上是学校的恶霸会把你锁起来，然后把你冲进厕所。

我跑题了。我是上帝，我创造了所有的东西，我看到它是好的。偶尔我会做些显影。
