# Dart 中的数据类型

> 原文:[https://dev.to/jay_tillu/data-types-in-dart-2ag9](https://dev.to/jay_tillu/data-types-in-dart-2ag9)

数据类型是指可以在编程语言中表示和操作的数据类型。让我们看看 dart 是如何表示数据类型的。

Dart 支持三种数据类型。它们以非常简单和简洁的方式被表示。(让我们把列表和地图放在一边，作为收藏部分。)

1.  数字
2.  用线串
3.  布尔代数学体系的

*   **注意:-** 虽然 Dart 是强类型的，但是类型注释是可选的，因为 Dart **可以推断类型**。

当你想明确地说不需要类型时，使用特殊类型 ***动态*** 。

## [](#numbers)数字

* * *

> ***注:****dart 中的数字表示可能因平台而异。这意味着，如果您在 DartVM 上部署代码，那么大小将会不同，如果您在浏览器上部署代码，那么大小将会不同。比如看一下** *int* **。

*   dart 中的数字用于表示数字文字。

**数字可以分为两部分。**

*   整数
*   两倍

### [](#integer)整数

用于表示 64 位非摩擦值(不带小数点的数值)的整数。整数文字用关键字 **int** 表示。

*   例如:10、20、30、400、50 等

**int 的大小**

*   在 DartVM 上——【-2⁶**到** 2⁶ -1】
*   **在浏览器上**—【-2⁵**到**2⁵-1】*(其中数制必须兼容 JavaScript)*

### [](#double)双精度

Double 用于表示小数值(带小数点的数值)。Double 文字使用 **double** 关键字表示。

dart 中的 Double 数据类型表示 64 位(双精度)浮点数。

*   例如:— 10.20、20.40、30.50 等。

> *从 Dart 2.1 开始，整数文字在必要时会自动转换为双精度。在 Dart 2.1 之前，在 double 上下文中使用整数文字是错误的。*

## [](#strings)字符串

* * *

该字符串表示一系列字符。例如，如果您想存储一些数据，如姓名、地址、国家名称等。然后可以使用字符串数据类型。

默认情况下，字符串使用 UTF-16 代码单位，但是你也可以通过使用符文来使用 UTF-32 代码单位。

关键字 **String** 用于表示字符串文字。

嵌入字符串可以使用单引号或双引号。我们将在一篇专门的文章中发现更多关于字符串的内容。

## [](#boolean)布尔

* * *

布尔类型表示一个布尔(标志)值。

**bool** 关键字用来表示布尔值。

布尔值是**真**和**假**。它们必须用小写字母书写。不能用 0 或 1 来表示真或假。
