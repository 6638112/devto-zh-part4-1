# 2019 年最安全的浏览器

> 原文:[https://dev.to/larrynosan/most-secure-browsers-2019-18h9](https://dev.to/larrynosan/most-secure-browsers-2019-18h9)

隐私和匿名在线已经成为许多人的需要。有数以百计的工具、软件和扩展来帮助人们加密流量，保护敏感信息和在线身份。其中一种方法是使用安全浏览器。有许多浏览器，但并不是所有的都承诺真正的匿名搜索。

在这个指南中，你会找到 2019 年最安全的浏览器。我们不会讨论像 Chrome 或 Safari 这样最受欢迎的，我们会讨论一些不太出名的，但更基于隐私的。

【2019 年最佳浏览器

*勇敢*

Brave 是众所周知的基于安全的开源浏览器。我们非常喜欢 Brave browser 的一个功能是，它可以屏蔽第三方广告(默认情况下)，甚至可以显示你通过屏蔽这些广告节省了多少时间。然而，由于广告是一些网站、内容创作者的主要收入来源，Brave browser 为其用户提供了一个安全的选项，可以向他们喜欢的内容制作者捐款(加密货币)。这个功能被称为“勇敢支付”。关于安全性，Brave browser 允许用户自定义他们的安全偏好，browser 有内置的反指纹功能和脚本拦截器，Brave 还允许用户创建私人标签并设置每次会话显示的标签数量。

只要知道 Brave 不允许大量的第三方扩展，因为它们相对较新。

*Epic 隐私浏览器*

Epic 隐私浏览器默认为隐名模式浏览器。这款浏览器据说是唯一一款能够屏蔽那么多追踪器和第三方 cookies 的浏览器。这款浏览器通过内置代理加载搜索引擎，保护用户流量不被保存和跟踪。我们非常喜欢的一个功能是 Epic Privacy Browser 将尽可能在任何地方使用 HTTPS。此外，浏览器提供了一个内置的加密代理，隐藏用户的 IP 地址和加密数据。你只需要知道一件事，在使用浏览器内置代理时，像网飞或 Hulu 这样的流媒体平台将无法工作，而像脸书这样的页面将需要额外的认证步骤。

*Tor 浏览器*

我猜你已经听说了托尔的事。还有 Tor 浏览器这种东西。Tor 浏览器是最安全的浏览器之一，浏览器使用 Tor 网络加密用户数据，甚至多次，这可以确保您的原始 IP 地址永远不会被泄露，网站永远不会知道您的真实位置。Tor 浏览器也将允许用户访问许多被地理屏蔽的网站。Tor browser 将通过加密使您的流量匿名和安全，并通过 DuckDuckGo 搜索引擎路由您的互联网流量，这是最安全的搜索引擎。Tor 浏览器最大的缺点是你不能一直使用它，因为它会大大降低你的速度，如果你一直使用它会很烦。

*HTTPS 无处不在*

HTTPS Everywhere 是一个扩展，通过加密您与网站的通信，使您的在线会话更加安全。不是所有的网站都支持 HTTPS，但这个扩展重写了 HTTP 到 HTTPS，使您的连接尽可能安全。

*DuckDuckGo*

DuckDuckGo 是一个流行的基于隐私的搜索引擎。DuckDuckGo 防止您的网上活动被跟踪，它提供跟踪网络封锁和智能加密。我们真正喜欢的功能是，当用户访问特定网站时，DuckDuckGo 会向用户显示从 A 到 F 的隐私等级。
