# 什么是无限循环？？

> 原文:[https://dev.to/jay_tillu/what-is-infinite-loop-520f](https://dev.to/jay_tillu/what-is-infinite-loop-520f)

无限循环也称为 endless loop，是一种永不结束的循环。循环中的指令将无休止地执行。

### [](#the-reason-behind-the-occurrence-of-infinite-loop)无限循环发生背后的原因

* * *

一般来说，有六个原因会成为无限循环发生的原因。

1.  没有终止条件
2.  有一个永远不能满足的条件
3.  具有导致循环一次又一次开始的条件
4.  错过递增器或递减器(如果你的逻辑需要的话)
5.  在错误的位置继续语句(如果您的逻辑需要的话)
6.  循环条件下的错误比较(小于或大于混淆)

所以每当你得到一个无限循环的时候，只要检查一下这六个条件，你很有可能会出错。

### [](#the-harm-that-infinite-loop-can-do)无限循环的危害

* * *

主要是无限循环会以两种方式危害您的系统:

*   **利用您的处理器时间和功率—** 一般来说，如果您的处理器正在处理任何进程，都需要时间和功率。与任何其他进程一样，一旦启动，无限循环将占用您的处理器时间和功率。在某个时间点，它会降低您的系统速度，使其无法响应。

*   使用你的内存——在电脑中，几乎所有正在运行的进程都驻留在内存中。当无限循环发生时，它开始填满你的 RAM 空间，在某个时间点，你的系统将耗尽内存，这可能导致整个系统的失败。

是的，这也是事实，像 Windows 10、Linux 和 macOS 这样的现代操作系统都有一个预定义的机制来处理这种过程。达到一定级别后，这些操作系统会自动终止这些进程，以避免系统完全崩溃。但是，如果以适当的方式实现，无限循环会造成很大的损害。

### [](#how-to-stop-infinite-loop)如何停止无限循环

* * *

如果你是一名开发人员，并且在终端应用程序中有一个无限循环，那么(ctrl+c)通常会停止它。如果是图形应用程序，那么你可以通过任务管理器终止进程，或者如果你正在使用 IDE，那么找到停止按钮。在大多数情况下，这将终止您的无限循环。
