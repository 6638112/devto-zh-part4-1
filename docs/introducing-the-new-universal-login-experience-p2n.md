# 介绍全新的通用登录体验

> 原文：<https://dev.to/auth0/introducing-the-new-universal-login-experience-p2n>

Auth0 新的通用登录体验——包括重新设计的登录流程、全新的 UX 设计和轻量级页面——现已全面推出。这里是你需要知道的关于这个版本的一切。

一、什么是通用登录？
通用登录通过中央授权服务器认证用户。这与嵌入式登录流形成对比，嵌入式登录流依靠 JavaScript 库通过跨来源请求执行身份验证。(你可以在这里找到通用与嵌入式登录的更详细的解释。)

通用登录的优势包括:

简化的用户体验
简化的应用集成
改进的管理
提高了身份验证安全性，尤其是第三方集成和移动应用
