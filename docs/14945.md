# 编码未来和现在

> 原文：<https://dev.to/javascript_tuto/code-the-future-and-now-3hf3>

我们现在知道如何编写代码来做一些事情。明天或明年做某事的代码呢？有时我们使用批处理作业。但是随着业务的增长，我们的“隔夜”批处理工作开始在午餐时间完成。随着我们向新的地区扩张，我们意识到没有“夜晚”。我们做出改变并中断我们忘记存在的批处理作业只是时间问题。如果明天的代码可以和今天的代码生活在同一个地方会怎么样？如果看起来都一样呢？如果我们能以同样的方式扩展它会怎么样？

加入 Adam，探索如何在我们现在编写的代码中拥抱未来。学习如何将需求捕获为一流的业务逻辑，并避免将它们降级为批处理作业世界的二等公民。深入研究将现成产品与基础计算机科学相结合的技术。

我们要去哪里...我们不需要批处理作业。
