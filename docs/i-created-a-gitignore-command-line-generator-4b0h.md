# 我创建了一个. gitignore 命令行生成器！

> 原文：<https://dev.to/christopherkade/i-created-a-gitignore-command-line-generator-4b0h>

如果你像我一样，每次从头开始一个新项目时，你几乎都会谷歌“忽略模板”。那至少是 15 秒，你永远也回不来了，所以我为此做了一个命令行工具🤷‍♂️

介绍 [gitignore-it](https://github.com/christopherkade/gitignore-it) ，它是这样工作的:

[![gitignoreit](img/6bbf379b204db17253f57b3547f3323b.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--o0XZGdWP--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://user-images.githubusercontent.com/15229355/59565299-e9c0fb00-9051-11e9-8012-04478bf9268c.gif)

请随意在全球范围内安装它，它非常轻便，可以节省您几秒钟的时间。

如果你想投稿，你可以创建新的`.gitignore`文件，马上就可以使用。

如果我在你的项目开始时为你节省了那至关重要的几秒钟，请随意给它一点⭐️
