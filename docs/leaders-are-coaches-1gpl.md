# 领导是教练

> 原文：<https://dev.to/jondouglas/leaders-are-coaches-1gpl>

想象一下，你看着镜子里的自己，告诉自己你有足够的动力领导一群乌合之众成为下一个谷歌。

你听过一句古老的谚语:

> 领导者是天生的，不是后天培养的。

但你不会因此而止步。相反，你会寻找更多的灵感，你会看到像文斯·隆巴迪这样的伟大教练的各种名言。

> 领导者是培养出来的，不是天生的。

事实上，在你内心深处，你知道自己具备领导能力。你拥有瑞士军刀中的一切技能，可以将一群不合群的人塑造成一个社区，你可以与每个人合作，培养并达成一个共同的愿景和目标。

领导力是一条单行道。你的人民会让你成为领袖。他们会欢呼，而不是你。

为了更好地适应领导力，你必须首先了解一些造就伟大领导者的倾向。

## 领导倾向

*   任何领导人的首要任务都是人民的福祉和成功。
*   为了在团队成员之间建立融洽和更好的关系，从更多个人话题开始会议，以了解每个人。
*   为 1:1 建立一个结构，并花时间为它们做准备，因为它们是帮助人们更有效和成长的最佳方式。
*   你的工作是运行一个决策过程，确保所有的观点都被听取和考虑，如果必要的话，打破束缚，做出决定。
*   定义一套原则，这些原则是不变的真理，是帮助指导决策的基础。
*   高绩效但难相处的团队成员应该被容忍，甚至受到保护，只要他们的行为不是不道德的或虐待性的，只要他们的价值超过他们的行为对人们造成的伤害。
*   对员工进行良好的补偿展示了爱和尊重，并将员工与目标和愿景紧密联系在一起。
*   目的是将产品愿景变为现实。所有其他部件都用于该产品。
*   如果你不得不让人离开，慷慨一点，善待他们，庆祝他们的成就。
*   寻找那些让一个人变得易于训练的特质，比如诚实、谦逊、坚持不懈、努力工作的意愿，以及对学习的开放态度。
*   全神贯注地听别人说话。不要提前考虑你接下来要说什么，而是提出问题，找到真正的问题。
*   坚持诚实和坦率。将负面反馈与关心结合起来，并尽快给出反馈。如果反馈是负面的，私下传递。
*   不要对人指手画脚；提供故事，帮助引导他们做出对自己最有利的决定。
*   相信别人多过相信自己，推动他们变得更勇敢。
*   当人们能够做自己并把自己的全部身份带到工作中时，他们是最有效率的。
*   当面临问题或机会时，第一步是确保合适的团队到位并致力于此。
*   寻找智慧和心灵；快速学习的能力，努力工作的意愿，正直，勇气，同理心和团队第一的态度。
*   将人们配对，在问题、项目和决策上建立同伴关系。
*   要想赢，你需要最好的团队，而最好的团队是多元包容的。
*   找出最大的问题，房间里的大象，并把它放在前面和中心，首先解决。
*   说出所有的负面问题，但不要纠结于它们。尽可能快地前进。
*   努力赢得胜利，但始终赢得正确，承诺，团队合作，和完整性。
*   当事情变糟时，人们会从他们的领导那里寻求更多的忠诚、承诺和果断。
*   倾听，观察，填补人与人之间的沟通和理解空白。
*   当你了解并关心人民时，领导人民会变得更加快乐，人民也会更加有效。
*   关心他人，询问他们工作之外的生活，理解他们的家庭，当事情变得困难时，出现。
*   为人们和他们的成功欢呼。
*   在工作内外建立社区。联系在一起的人更强大。
*   慷慨地投入你的时间、关系和其他资源。
*   在你心中为那些用自己的远见和激情战胜困难的人保留一个特殊的位置。
*   在职场中爱别人是有挑战性的，所以要不断练习，直到变得自然而然。
*   无论你卷入了什么，都要承担责任和后果。驾驶它。
*   和有活力的人在一起。与他们接触。
*   找出你唯一擅长的，是什么让你与众不同。了解你内心给你目标感的东西。现在应用它们。
*   人生的大部分转折点是无法预测和控制的。让命运发挥作用。

你能记起在你生命中的某一时刻，你有一个真正关心你的教练、老师或某人吗？你知道，到了他们会推迟那天帮助你学习的承诺的程度？或者他们如何通过询问你的爱好、家庭等各种方式向你表明他们真诚地关心你的生活？

他们表现出许多这样的倾向，你觉得你有归属感。他们向你展示了谦逊、同情和爱。他们向你展示了什么是对生活的影响。他们向你展示了什么是真正的领袖。
