# 职业转变过程:利用你现在的雇主

> 原文：<https://dev.to/conorgaughan/career-change-process-leverage-your-current-employer-56p>

背景:我目前正在学习前端 web 开发，以努力实现最终的职业转变。我开始寻找一个正式的课堂环境来学习 Javascript，但是我负担不起训练营的费用，也没有时间去参加。因此，我必须找到一个最佳点——价格合理，又不需要花费太多时间的东西。我找到了一些由大会组织的编码[研讨会，这些研讨会在时间上完全符合我的要求，但仍然稍微超出了我的预算*。我开始集思广益，想办法报名参加这个班，并想办法全额(或至少部分)支付学费。*](https://generalassemb.ly/education?where=online#catalog-results)

##### 雇主资助

*   我目前在一家软件公司担任客户成功经理，我的日常工作不需要任何编码。我的雇主提供了一份季度培训预算，但我没有从中索取一分钱。过去，我对申请资助犹豫不决，因为我想上的课超出了我的工作范围。但是，不要求它是一个保证*不*所以我最终决定去争取。
*   那么，你如何向你的经理提出这个论点呢？
    *   **说明价值。**确定你目前的职责以及课程如何补充或提高你的表现是关键。在我的例子中，成为产品专家对于有效地与客户沟通是必不可少的。我唯一可以改进的方法就是变得更有技术含量，所以了解我们的产品网站如何运作可以增加你对公司的价值。
    *   “我想确保我为我们的客户群提供了尽可能多的产品知识。为了做到这一点，我想在产品方面变得更有技术含量，我认为这门课将帮助我全面了解我们的客户每天都在经历什么。”
    *   **继续教育。**向管理层展示你对进一步发展的渴望&教育是你想学习的标志&成长。学习特定于你每天所做的事情的技能是不现实的，这样做你的潜力上限是有限的。学习一些你当前角色之外的*技能会极大地影响你的发展，或者在我看来，有希望帮助你找到一个新的角色&兴趣。*
    *   **讲道理。**你现在的雇主可能不会同意为编写训练营代码支付数千美元，所以找一些更实惠的选择吧。例如，我通过[大会](https://generalassemb.ly/education?where=online#catalog-results)参加了两次编码研讨会，每次大约 250 美元。也许有必要问问同事，他们在公司都做了些什么，以此来确定一个基线。
    *   你不得不问。我在现在的雇主那里工作了两年多，他们提供季度培训预算。我对 web 开发感兴趣已经有一年多了，但是总是不愿意要求我的核心工作功能之外的额外培训。我建议在你的经理的日程表上安排时间来讨论你为什么认为你应该参加这次培训，或者在你的 1:1 中安排时间。
