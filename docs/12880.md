# Python 数据科学工具箱，我的评论

> 原文:[https://dev . to/vik Yale/python-data-science-toolbox-my-review-1 bbl](https://dev.to/vikyale/python-data-science-toolbox-my-review-1bbl)

大家好！最近我学到了更多关于如何使用 python 来分析数据。所以，我参加了一些数据营的课程。最后一门课是“Python 数据科学工具箱- Par 1”，学习用 Python 编写函数的艺术和一些关键概念，如作用域和错误处理。

包含三个部分:

*   编写自己的函数。
*   默认参数-范围。
*   Lambda 函数并处理错误。

如果你了解 python 的基础知识，并且想提高自己的技能成为数据科学或数据工程师，那么这个课程很不错。

*   编写你自己的函数:实际上，我们听说过 Pandas、Numpy 和 Matplotlib 库，它们在数据分析中更常用。但是，像开发人员一样，您需要编写自己的函数来解决数据中的问题。

*   默认参数-范围:当您需要多个参数和多个返回值时，这部分对于构建定制函数很重要。

*   Lambda 函数和处理错误:“Lambda”是一个时髦的词，在这一部分中，您可以通过大量的示例逐步学习如何构建一个 Lambda 函数并在自己的函数中处理错误。

每一部分都帮助你编写神奇的函数来分析数据帧，你将有超能力在现实世界中实践大型项目。
