我将会写一系列的帖子，详细介绍我通过[火线强盗战争游戏](https://overthewire.org/wargames/bandit/)的方法。

## 强盗战争游戏的先决条件

1.  ### 能够 ssh 进入匪机

    如果您有一台 Linux/MacOS 机器，默认的 ssh 客户机应该足够了。

    如果你有 Windows 机器，大多数人会推荐安装 [PuTTy](https://putty.org/) 。但我不会。浏览这些 CTF 需要在 Linux 命令行上做大量的工作。因此，我建议安装 VirtualBox 或 VMWare，然后安装 Ubuntu 并熟悉它。

    有很多关于启动和运行 Ubuntu VM 的教程。谷歌一下试试吧。它值得。一旦您启动并运行了一个虚拟机，您就会感到自豪。

    安装虚拟机后，请完成 Linux 命令行速成课程。我保证它会在军事演习中派上用场。

2.  ### 链接微信

    对我来说，CTFs 最有趣的部分是跟踪我的进步，知道我的“分数”。为此我使用了[微信](https://www.wechall.net/about_wechall)

    [不要脸在微信上塞 aka 我的简介](https://www.wechall.net/profile/kkaosninja) :D

    链接微信的步骤都在这里= >[http://overthewire.org/information/wechall.html](http://overthewire.org/information/wechall.html)

    微信会派上用场，因为它们也有很多其他 CTF 网站的链接，让你可以展示你在所有这些网站上的进展。

3.  ### 耐心

    将会有很多挫折，因为在后来的强盗机器上获得旗帜要困难得多。但是请相信我，当我这么说的时候，你会享受这个挑战的每一点，以及你在这个过程中学到的所有东西。

    对于每一次演练，我不会马上公布解决方案。我会试着提供一些提示，鼓励你自己找到解决方案。

    这种风格的灵感来自于 Bresleveloper 写的类似的穿越系列。

    没有人是完美的，所以是的，有时候当你陷入困境后，你可以看看解决方案。但只有在你有信心自己已经尽力了，并且被卡住了之后，才这么做。

    记住=> *目标是学习，而不是和别人比较*

这就是先决条件。现在。让我们开始吧。
