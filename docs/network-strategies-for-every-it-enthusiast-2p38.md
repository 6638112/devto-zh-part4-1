# 每个 IT 爱好者的网络策略

> 原文：<https://dev.to/brettclawson75/network-strategies-for-every-it-enthusiast-2p38>

网络策略是在家庭或办公室中设置网络时，可以配置系统的工作程序和不同方式。网络技术的范围很广，基本上是在网络中设置两台或多台计算机，以便它们可以相互通信并共享打印机、扫描仪等公共设备。每个公司都需要联网，因此所有 it 技术人员至少了解一些基本的联网策略是至关重要的。网络策略可以帮助他们理解计算机在系统中是如何连接的。

不同类型的网络

基本上，网络可以分为三种类型。他们是兰、曼和万。
