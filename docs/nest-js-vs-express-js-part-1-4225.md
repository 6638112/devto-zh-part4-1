# Nest.js vs Express.js

> 原文：<https://dev.to/marius/nest-js-vs-express-js-part-1-4225>

我最近听说了 Nest.js，决定尝试一下，并将其与 Express.js 进行比较(两者都是 Node.js 的 web 应用程序框架)。

Nest 是 Express 的“下一层”,提供了开箱即用的应用程序架构(与 Express 相比，Express 采用了更简单的方法)。

在这个视频中，我试图记录我第一次学习和尝试的过程，重点是如何入门的比较以及如何编写基本控制器。
