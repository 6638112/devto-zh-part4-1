# SwiftUI 中的反应设计模式

> 原文：<https://dev.to/bmcmahen/react-design-patterns-in-swiftui-3nbe>

**TLDR:我已经用 SwiftUI** 中实现的 React 模式创建了一个[存储库](https://github.com/bmcmahen/react-meets-swiftui)

苹果最近发布了 SwiftUI，这是一个声明式 UI 框架，用于在苹果设备上构建原生应用。与 React 的相似之处非常明显，这使得任何 React 开发人员都相对容易掌握。

在接下来的几周里，我将自学 SwiftUI，我将记录在 SwiftUI 中实现的各种常见的 React 设计模式。你可以在[react-meets-swift ui](https://github.com/bmcmahen/react-meets-swiftui)GitHub 知识库上看到它们。希望这些例子能让有经验的 React 开发人员更容易掌握 SwiftUI。

请记住，我有大约 4 年的反应经验，和 3 天的迅捷经验。但是随着我了解的越来越多，我会继续更新这些例子，希望那些有更多经验的人能提供帮助。
