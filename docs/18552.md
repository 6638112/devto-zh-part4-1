# 我以前是铁路工程师，现在是软件工程师，问我什么都行！

> 原文:[https://dev . to/the ringleman/I-used-to-be-a-engineer-on-the-railway-now-I-engineer-software-ask-me-any-3j4b](https://dev.to/theringleman/i-used-to-be-an-engineer-on-the-railroad-now-i-engineer-software-ask-me-anything-3j4b)

嗨！我是萨姆。我从小就有从事科技行业的梦想，但不同的生活事件发生了，我没有追求我的梦想，直到我认为晚了。我多次尝试学习编码，但都失败了，我甚至在注册 CS 的时候辍学了。

我最终去了 BNSF(大橙色列车)当售票员，然后成了一名工程师(开火车的，不是同一类型的工程师)。4 年后，我被解雇了。失业期间，我决定是时候改变一下了。所以我学会了编码，现在三年多过去了，我是 RE/MAX 公司 R&D 团队的一名工程师。

我把这些放在家里书桌上方的窗户里。他们在那里提醒我我从哪里来。提醒我自满是不好的。来提醒我每天都要坚持往前冲。我有一个座右铭，它是陈词滥调，但是如果我能做到，你也能。

我想成为其他试图进入我们行业的人的支持者。我被这一举动祝福了，我只想看到其他人达到同样的目标。所以请问我任何事情，我很想了解你！
