# 作为一名自由网页开发者，需要了解 Wordpress 吗？

> 原文：<https://dev.to/realtoughcandy/do-you-need-to-know-wordpress-as-a-freelance-web-developer-2443>

#### 以下是我的书《自由职业新手 T2》的摘录，该书在 T4 亚马逊有平装本和 Kindle 版本。也可以作为顶级的[视频课程！](https://realtoughcandy.io/p/freelance-newbie-become-a-freelance-web-developer)

## WordPress 现在占据了网络上 1000 万个顶级网站的 1/3 以上。

单从这个统计数据来看，难怪如此多的创意机构和自由职业者将它作为他们的旗舰服务。

此外，WordPress 是:

*   易于部署和维护
*   基于插件:你不必成为一个 PHP 高手来集成伟大的特性
*   最终用户友好
*   开源(免费的啤酒，免费的自由)
*   高度可扩展

### 用户可使用的，人类可读的

所有这些好处都很好，但这意味着你必须提供吗？简而言之，答案是否定的。然而，它是一个极好的赚钱机器。如果你已经熟悉 WordPress，绝对可以考虑将它整合到你的商业计划中。

如果你不熟悉 WordPress，我建议你至少熟悉它的基本用法。创建一个本地副本，尝试各种特性和插件，熟悉文件结构。

虽然 WordPress 的后台是 PHP 和 MySQL，但是你不需要知道 PHP 或 SQL 就可以为客户创建网站。

如果你需要定制它，而你又找不到合适的插件，如果它超出了你目前的技能水平，你总是可以外包这项任务。

随着你找到更多的客户，倾听他们的需求，并制定策略解决他们的业务问题，你会发现 WordPress 通常是一个理想的选择。

#### 我如此热衷于使用 WordPress 的另一个原因是它简单明了的管理面板。

这意味着你可以放心地为你的客户建立一个用户帐户，他们可以处理剩下的事情，无论是更新插件(一次点击)，发布博客文章，添加用户，还是其他任何普通的任务，如果他们让我做的话，会更浪费时间而不是赚钱。

即使作为一个完全的新手，这也是真的:对我来说，无论什么琐碎的任务，10 分钟的计费时间(甚至是固定费用)都不值得。你还面临着客户每小时给你打电话询问另一个小问题的风险，其中许多问题会让你怀疑是否值得麻烦地发一张发票。

让他们参观用户可维护的部分，指导他们阅读常见和简单任务的可读文档，并让他们自由地在自己的时间内完成这些任务，效率会高得多。

WordPress 的管理面板对于非技术终端用户来说很容易使用。

随机但重要的附加说明:永远为你的客户维护你自己的网站。一旦他们访问了代码，你永远不知道他们会做什么来改变你的站点的外观和功能。很多时候情况变得更糟，你将无法在你的作品集或其他宣传材料中展示这个网站。

### 底线，探索 WordPress 作为你客户的解决方案。

即使你从未使用过它，你也能很好地掌握基本知识。如果你决定完全使用 WordPress 模式，你可以学习 PHP 和 MySQL 或者雇佣其他人来制作插件。在我还不知道如何编码的时候，我就已经在为盈利而建立网站了，它变成了我的客户一贯的理想解决方案。
