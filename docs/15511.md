我已经从标签页中删除了关闭按钮，这样当我点击标签页时就不会不小心关闭标签页。颜色是我自己选择的.......我已经修改了颜色主题扩展，并把它放到了我自己的库中。我的 VS 代码完全是键盘驱动的。我在使用 vscode 时从来不碰鼠标。

我支持 vscode CZ，它支持 vim。Vscode vim 扩展可以提供几乎完整的 vim 体验。虽然它不支持加载本地 vimrc 文件，但是通过 vscode 设置，许多 vim 定制是可能的。它支持键映射和引导键。

我映射了所有的键绑定，所以它变得非常快。即使在使用了不必要的 126 扩展之后，我的 VS 代码仍然非常快。我已经删除了所有无聊的菜单，这样它就变得干净了。
