# 如何获得你的第一份技术工作

> 原文：<https://dev.to/codingunicorn/how-to-get-your-first-tech-job-2agc>

获得一份好的第一份工作总是充满挑战。如果你正在寻找一份技术方面的工作，这里有一些你可以做的事情来最大化你的机会。

**个性化**。让公司感到独特和特别。你不只是在找工作，而是想在这家公司工作。没有别的地方。你应该了解公司的一切——他们在做什么，谁在那里工作，公司使用什么技术。玩他们的网络产品，安装他们的移动应用程序。推荐给你的朋友。报告一个错误。

**创建一个演示应用**。看看工作要求，开始了解公司使用的技术。理想情况下，创建一个小的演示应用程序来展示你的技能。app 要和公司业务相关。这也表明你在乎！

**证明自己学得快**。如果你不知道某项技术，承认并告诉他们你准备好学习了。如果你能讲一个真实世界中你很快学到新东西的故事，这对你有好处。伟大的公司不需要无所不知的人。他们需要快速学习者。

**建立公众形象**。例如，科技博客会给你额外的可信度。博客显示你喜欢分享你的知识和紧跟潮流。而且你也足够自律，能够保持更新！

**有一个“知情人”**。如果你有一个内部人士可以把你介绍给招聘人员或面试官，那就太好了。人们会对你更友好，因为他们了解你，或者他们圈子里的人了解你。参加技术聚会和会议，与人们面对面交流，进行愉快的交谈，并结识内部人士。
