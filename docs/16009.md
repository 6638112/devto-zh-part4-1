# Reddit 还是 StackOverflow？

> 原文:[https://dev.to/alephc/reddit-or-stackoverflow-53p0](https://dev.to/alephc/reddit-or-stackoverflow-53p0)

我想在 VS 代码上使用 Python。我喜欢 Jupyter 笔记，但我想尝试新的体验。我在 Reddit 和 StackOverflow 社区上的所有问题后发现，如果我们想在 VS 代码上使用 Python，必须先安装 Python for Windows (Python 3.7.3)。添加到 path，打开 VS 代码，安装 Python 扩展。没有其他 Python 版本被允许，所以今天没有 3.7.3 版本你就不能很好地运行代码。可悲的是，没人回答我那个小把戏。VS 代码打了很久才做到的。在 Reddit VS Code 社区上，我发现比 sleeping stack overflow 高声誉合作伙伴的关注度要高得多。StackOverflow 怎么了？大到不能倒？
