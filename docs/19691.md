    为了开发和运行 PHP 驱动的网页，以下软件是非常必要的:
    I-web 服务器:PHP 需要一个 Web 服务器软件，以便能够执行存储在服务器上的脚本。存在许多不同的 web 服务器，但是使用最多的是 Apache 服务器，它是开源的。阿帕奇服务器可以从 https://httpd.apache.org/download.cgi 的[下载。](https://httpd.apache.org/download.cgi)

ii- A 数据库:PHP 使用一个数据库来存储在给定网站的任何网页上使用的文件。有许多数据库与 PHP 脚本语言非常兼容，但使用最多的是我的 SQL，它还有一个开源版本，可以从 https://www.mysql.com/downloads/下载。

PHP 解析器:PHP 解析器获取已经执行的 web 开发人员的 PHP 脚本，并生成可以在 web 浏览器上发送和显示的 HTML 输出。

在您开始在您的计算机系统上编写和执行 PHP 脚本之前，必须下载并配置上面列出的所有软件，使它们能够相互配合工作。但是近年来，随着技术的进步，已经发布了包含所有三种软件的各种软件包，唯一需要的是下载它们并安装，并且在这些软件包的安装过程中自动完成整个配置。这些软件包的示例包括:

*   XAMPP:跨平台，Apache，MySQL，PHP 和 Pearl。

*   LAMP: Linux，Apache，MySQL，PHP。

1.  PHP 定界符:
    定界符是一组用来区分不同代码段的字符。

    i- PHP 开始和结束标签分隔符(<？服务器端编程语言（Professional Hypertext Preprocessor 的缩写）...？>)，PHP 短标签(<？...？>)，PHP 打印语句(<？=..；？这些 PHP 分隔符支持 PHP 脚本的识别。
    ii- PHP 字母数字分隔符，如正斜杠(/)、散列符号(#)、逗号(，)、句号(。)，波浪符号(~)等等。这些 PHP 字母数字分隔符可以用来分割一个字符串，甚至将一个数组转换成一个字符串，它们还执行许多其他功能。

1.  PHP 数据类型:以下是 PHP 支持的数据类型:
2.  字符串，这是包含在引号中的字符序列，如“Hello world！”
3.  整数，这些都是整数，没有小数点，比如 4195。
4.  Float，这些是浮点数，比如 3.14159 或者 49.1。
5.  布尔型，只有两个可能的值:真或假。
6.  数组，它们是其他值的命名和索引集合。
7.  对象，这些是程序员定义的类的实例，它们可以打包特定于该类的其他类型的值和函数。
8.  NULL，这是一个只有一个值的特殊类型:NULL。
9.  资源，这些是保存对 PHP 外部资源(如数据库连接)的引用的特殊变量。

10.  PHP 常量:
    常量类似于变量，除了它们在被定义后不能被更改或未定义。
    常量名称以字母或下划线开头。
    创建常量，使用 define()函数:
    示例:define(name，value，不区分大小写)
    参数:
    name:指定常量的名称；
    value:指定常量的值；
    不区分大小写:指定常量名是否不区分大小写。默认值为 false

下面的例子创建了一个区分大小写的常量名:
