# 投资自己

> 原文：<https://dev.to/ilvalerione/invest-in-yourself-jgg>

在我的职业道路上，我寻找并阅读了其他开发人员关于他们经历的故事，以了解我的想法是否是疯狂的症状，或者是否有像我一样的其他人。

这就是为什么我想和你分享我对“投资自己”这句话的看法。

如果你正在考虑改变你的职业生涯，也许值得读一读这个故事，看看你是不是唯一一个疯了的人，还是至少有两个人疯了😄。

2018 年成为 Laravel 认证开发人员给了我一个小小的机会来思考自己，以及“投资自己”如何缓慢但不断地改善我的职业生活。

我真的相信，如果我继续我的老方法，我会处于一个非常不同的情况。

当我开始我作为软件开发人员的第一份工作时，似乎一切都在朝着正确的方向发展。

我毕业于软件工程专业，我的职业生涯开始于爱立信，世界上最大的技术公司之一。我离开了西西里岛的家，搬到了那不勒斯，意大利最好的城市之一。

开发其他人使用的软件应用程序总是给我一种非常好的感觉。这就像我小时候开发足球比赛预订 web 应用程序一样，我和我的朋友们想象这个软件会在人们洗澡时通过关闭热水来控制世界…😜。

我很兴奋能在爱立信工作，我想尽可能多的学习…但如果我不得不诚实，什么都没发生。

我在爱立信的日子标志着我人生中唯一一段停止投资自己的时期的开始。背景与我的预期完全不同。我的疑虑很快就来了。

这家公司确实很老了。旧的工作工具、非常高的平均年龄和庞大的中层管理人员金字塔。

在两年半的时间里，我从未听说过“设计模式”这个术语，关于创新的最佳讨论是在一位同事新买的智能手机上。

12 个月后，我问老板:“送我出国，否则我就完了！”

就这样。我搬到巴黎，在布伊格电信公司工作。

巴黎是一次极好的经历。在 Bouygues Telecom 公司，我的经理是我的同事，他们的目标是将整个公司的基础设施提升到一个新的水平。在这里，设计模式、坚实的原则和 UX 成了我的日常面包。

在与我的团队领导 Gregory 喝咖啡休息时，我发现布伊格为我的努力每天付给爱立信 450 欧元。

当然，这一价值来自爱立信品牌，但它让我开始思考…通过出售我自己的技能，我能赚多少钱？

很长时间以来，我第一次想到，“我能给市场什么？我能做些什么来为人民的痛苦找到有价值的解决办法？”

今天，我回到了意大利。我离开了爱立信，从零开始重新开始我的路。现在，我是意大利一家软件开发公司的老板，该公司使用现代工具和方法来建立 SaaS 可持续发展的企业。

当然，这每天都是一个挑战，但我切身体会到，如果你走上了完全不符合个人需求的错误职业道路，后果会有多严重。与其假装一切都好，不如挑战自己。

今天的爱立信怎么样了？他们在欧洲策划了 1000 多人的裁员计划。

这个故事的士气:**投资自己**。
