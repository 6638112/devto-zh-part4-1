# 我是如何从一个酒精聚会中恢复过来并在第二天继续工作的

> 原文：<https://dev.to/zex/how-did-i-recover-from-an-alcohol-party-and-get-back-to-work-the-next-day-32pd>

尽管我知道我不应该喝太多，但有时我会把所有的规则抛在脑后，尽情享受派对。

那现在怎么办？我需要从中恢复过来。

## 汤

通常，我早餐吃松饼和咖啡。派对之夜过后的早晨，这些是我最不想要的。我唯一想到的就是汤。

汤更容易消化和吸收，提供营养、能量和水分。

## 运动&能量饮料

水对于清理体内的废物和残留酒精至关重要，但单纯的水是不够的。我需要一些东西来唤醒我的舌头。

运动饮料供应身体无机盐、维生素以及水，味道也不错。

## 维生素补充剂

我带了一些维生素补充剂，只是像往常一样吃了一些。

## 睡眠

酒精确实有助于睡眠，所以不用担心。找一个身体知道该做什么的舒适的地方。

努力工作，别忘了找乐子。过量饮酒仍然不被鼓励。🐒
