# 将附加网络接口(NIC)和公共 IP(PIP)关联到虚拟机

> 原文:[https://dev . to/patricksameerajayalath/associate-additional-network-interface-NIC-and-public-IP-pip-to-virtual-machine-24](https://dev.to/patricksameerajayalath/associate-additional-network-interface-nic-and-public-ip-pip-to-virtual-machine-24)

在本教程中，我将展示如何:

创建新的资源组
创建虚拟网络并将其与资源组关联
将子网添加到虚拟网络
将虚拟机添加到虚拟网络
将多个 NIC 和 pip 与虚拟机关联
远程连接到虚拟机
在虚拟管理器上安装 IIS
向虚拟机发送 http 流量
