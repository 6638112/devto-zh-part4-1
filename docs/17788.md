# 有没有一个瞬间你知道编程是适合你的职业？

> 原文：<https://dev.to/jrohatiner/was-there-a-moment-when-you-knew-programming-was-the-right-career-for-you-2al1>

有没有一个特殊的例子影响你选择软件工程作为你的职业？是在某个时刻还是随着时间的推移而发生的？

阿波罗 11 号航天发射后，我在佛罗里达州卡纳维拉尔角经历了一个有影响力的时刻。我爸爸带我去看废弃的发射台。他是负责成功发射的团队的一员，设计了龙门支架。很明显，他对那里所取得的成就感到敬畏。我告诉他我也想成为一名工程师。他告诉我，我可以做任何我想做的事情，他肯定我会让他感到骄傲。
