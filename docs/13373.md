# 不，你不傻！

> 原文:[https://dev.to/devdrake0/no-you-re-not-stupid-1k4l](https://dev.to/devdrake0/no-you-re-not-stupid-1k4l)

试图创建一个社区来帮助崭露头角的开发人员让我想起了我当初的感受。

我只懂一种口语(英语)是有原因的。学习语言很难。

这也延伸到学习一门开发语言。刚开始真的很难。你必须学习所有这些新概念，没有一个是天生的。

你甚至可能被那些知道自己在说什么的人包围，或者至少看起来知道，你开始觉得自己很愚蠢，无法在这个行业取得成功。

让我告诉你-你错了。你不傻，你能做到的。

如果有人告诉你，你必须停止听他们的。他们错了。

我不在乎他们是否已经发展了 40+年，是一个“巫师”。他们只是忘记了最初是什么样子。一开始我们都很挣扎。

当你有问题的时候，找一个会帮助你的人，而不是贬低你的人，你会成功的。

如果你需要一些支持，请联系我。我没有链接回我的“社区”,因为这不是关于“宣传”我的东西，这是关于你知道你能做到——不管别人怎么想。
