# 你很棒

> 原文：<https://dev.to/duffn/you-are-great-59l9>

虽然感觉可能不总是这样，但重要的是要记住你是特别的，你是被需要的，你是伟大的。

我做了一件事来帮助一个悲伤的朋友感觉好些。也许它也会给你带来笑容。

[https://youaregreat . FYI](https://youaregreat.fyi)

```
curl https://youaregreat.fyi/api/
{"message":"You are great. 🌟"}
```
