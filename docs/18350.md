# 如何在 Ubuntu 上安装 Python 3.8？

> 原文:[https://dev . to/mortoray/how-to-install-python-3-8-on-Ubuntu-1bp 4](https://dev.to/mortoray/how-to-install-python-3-8-on-ubuntu-1bp4)

我无法在 Ubuntu 18.04 上运行 Python 3.8。我已经安装了主程序，但是缺少 pip。

我做到了:

```
sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt-get update
sudo apt-get install python3.8 
```

我在下面做了进一步，添加了`-venv`和`-dev`来编译更多的包。

```
sudo apt install python3.8-venv python3.8-dev 
```
