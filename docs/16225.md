我个人喜欢使用没有道具的类组件，因为:

*   这是更简单/更短的代码
*   它从 React 继承方法。成分
*   也继承了构造函数，你不需要**编写它(尽管你仍然可以选择)**

*   多写几行代码
*   创建一个带道具的构造器
*   然后将 props 值从构造函数传递给 super
*   调用`super(props)`以便可以使用`this.props`
