# 第一个社区项目

> 原文:[https://dev.to/mlanders/first-community-project-1bhj](https://dev.to/mlanders/first-community-project-1bhj)

在过去的周末，我和一群我从未见过面的开发人员一起参加了我的第一个社区项目。

## [](#saturday)星期六

上午 9:30 左右出现在 [SkySlope](https://skyslope.com/) 。项目的组织者首先讨论了这个项目，以及他们关于我们如何组织大约 20 个项目各个方面的开发人员的想法。我们有一个小型的 DevOps 团队，他们在 AWS 上设置了数据库，并使用 SQL 构建了数据库结构。

我属于后端团队，负责构建路由、逻辑和 sql 查询。整个周末，我们有大约 10 个人在后台工作。很早的时候，我们就决定使用 Node，因为大多数团队成员都知道它。小组中一个更资深的开发人员开始搭建一个环回服务器，但是对于每个人来说，在一个周末学习和使用它变得有点复杂。之所以决定切换到 Express，是因为使用它更容易，并且可以快速启动和运行 0.1.0 项目。

前端团队也是一个由开发人员和 UI/UX 人员组成的更大的团队。他们创造了一些伟大的设计，并开始在 React 中构建应用程序。

## [](#sunday)星期天

周日出现了一个小得多的团队。后端团队由大约 5 人组成。在早期，我们在团队中分配工作量来完成路线，并确保所有的 GET 路线都工作正常。一天快结束时，我们让一些 POST 请求开始工作，并开始缩小 GET SQL 查询的范围，这样它们就不会返回所有数据，而是只返回需要的数据。

学习大公司和高级开发人员使用的术语和结构是一次很棒的经历。在代码结构或如何提交方面没有太多的形式，但我们完成了大量的工作。我计划在接下来的几周内致力于这个项目，希望我们能让这个项目为这个城市做好准备。
