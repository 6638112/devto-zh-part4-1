# 谷歌的“Kubernetes”项目是关于什么的？

> 原文：<https://dev.to/elmomagalona/what-is-google-s-kubernetes-project-all-about-2nbb>

根据谷歌的网站

Kubernetes 是一个开源的容器集群管理器。它跨一组节点实例调度任意数量的容器副本。主实例公开 Kubernetes API，通过它来定义任务。

Kubernetes 在节点上生成容器来处理定义的任务。
容器的数量和类型可以根据需要动态修改。每个节点实例上的代理监控容器，并在必要时重新启动它们。

Kubernetes 针对谷歌云平台进行了优化，但可以在任何物理或虚拟机上运行。
