# 将原生文本复制到剪贴板示例

> 原文：<https://dev.to/skptricks/react-native-copy-text-to-clipboard-example-2h1d>

来源:[将原生文本复制到剪贴板示例](https://www.skptricks.com/2019/07/react-native-copy-text-to-clipboard-example.html)

本教程用一个简单的例子解释了如何在 react 本地应用程序中执行复制文本到剪贴板。通过使用剪贴板 API，在 React Native 中读写剪贴板非常容易。它可以在 Android 和 IOS 平台上运行。剪贴板是电脑和手机中的一个干净的临时存储区域，用于执行剪切、复制和粘贴文本功能。默认情况下，在 react native 的 TextInput 组件中，剪切、复制和粘贴功能会自动工作。有时开发人员需要手动推送和弹出剪贴板的值。我们将使用 react native 的剪贴板组件 API 来手动插入和获取剪贴板的值。
