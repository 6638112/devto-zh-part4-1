# 为网络平台新闻获取资金

> 原文:[https://dev . to/sime vidas/securing-funding-for-web-platform-news-2le 2](https://dev.to/simevidas/securing-funding-for-web-platform-news-2le2)

我在 **web 平台新闻** ( [webplatform.news](https://webplatform.news/) )上为 Web 开发者创建常规内容。以下是我最近写的一些话题:

*   CSS Scroll Snap 即将登陆 Firefox，但在桌面上的可用性并不好
*   第 1014 期 : Safari 将网络共享带到了桌面
*   [问题№1009](https://webplatform.news/issues/2019-04-10):`<datalist>`元素现在得到了广泛的支持，但是使用标签会引起一些问题
*   当 Chrome 的窗口被另一个窗口覆盖时，网页可以检测到
*   [Issue№1005](https://webplatform.news/issues/2019-03-19):CSS`scroll-padding`属性调整滚动容器的最佳查看区域

我还在 **CSS-Tricks** ( [我的文章的完整列表](https://css-tricks.com/author/simevidas/))上发布每周客座博文。这些帖子包含了 web 开发人员的新闻，也包含了我在 Web 平台新闻中的内容。

今天我宣布，我想从公司那里获得网络平台新闻的资金，以换取他们在网站上的推广。宣传横幅将显示在一期的主要主题和新闻项目之间，包含公司的名称和徽标、产品和服务的简短描述以及一个特色链接(公司博客、招聘页面或类似网站上的文章)。

[![](../Images/5226584b693092a9905a91f0631788e9.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--WmcDd1T4--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/gqs90a4h78xegkiv3y90.png)

我的想法是这样的:

1.  一家公司每月资助一期网络平台新闻，该公司的宣传横幅将永远包含在该期**中**(例如，如果访问者在该期发布几个月后查看该期，则关于该期由哪家公司资助的信息仍将包含在该期中)。

2.  我将每 7 天发布一期新的期刊(从 9 月份开始，每 3-4 天)。最新一期总是显示在网站的起始页上。上个月的所有其他问题都列在当前查看的问题下面。

3.  一家公司通过加入我的 Patreon 页面上的“赞助商”层成为出资人。

我在六月只出版了一期(我也在我的 Patreon 页面上发布每日新闻)。根据 Cloudflare 的分析，在 6 月 5 日发行后的 7 天里，我的网站被大约 **5，500 个独立访问者**访问。

[![](../Images/f6a154f36c4764c9c3b0c453bf1e56ff.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--P-gEuGwF--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/3az2ig0t3zk4x9kfgo1o.png)

Patreon 在每个月的第一天处理付款。由于现在是 6 月底，我没有足够的时间为我计划在 7 月出版的所有(四)期杂志筹集资金，但我仍然希望找到一家公司在下个月帮我测试这种新的融资模式。

[![](../Images/a94aabfd7236d17fb3bcfa8e1e98ed7a.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--EBPEreZ5--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/z5adht08mgjaew7t63ob.png)
