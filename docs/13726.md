# 我们可以采取哪些小步骤来拯救环境？🌎

> 原文:[https://dev . to/Abdul isbomb/what-some-small-steps-we-can-take-to-save-the-environment-459g](https://dev.to/abdulisbomb/what-are-some-small-steps-we-can-take-to-save-the-environment-459g)

很多人认为拯救环境意味着你需要完全改变你的生活方式，成为素食主义者，虔诚地花时间去捡垃圾，只吃有机农产品。如果你碰巧做到了所有这些事情，我会为你鼓掌，尽管每个人都可以采取一些小措施来实现更环保的生活方式。我做的一些小改动包括:

*   带着水瓶去工作。
*   当我刷牙的时候关掉水。
*   在工作空闲频道上回收旧迷因。
*   把可重复使用的袋子带到杂货店。
