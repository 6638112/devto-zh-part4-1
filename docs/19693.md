# 如何在不尝试的情况下发展小企业

> 原文：<https://dev.to/therealdarkmage/how-to-grow-a-small-business-without-trying-20op>

我是一个自称“技术领袖”的家伙的粉丝。在我参与编码社区的过程中，他一直是我成长的指导力量。一个周五的深夜，我发现自己在看一个老视频，寻找发展自己业务的灵感:

[不花钱发展小企业的秘诀](https://www.youtube.com/watch?v=TnUO6tdmscI)

1.  如果你试着和每个人说话，你不会和任何人说话
2.  接触社会影响者来宣传你的产品
3.  要真诚
    *   用对话式的、友好的语气写作，不要太正式
4.  优化您的转化漏斗
    *   优化登录页面，最大限度地减少点击购买

我真的只能说我能说的。当然，我想学的东西很多，但我只能满怀信心地谈论我已经巩固在知识库中的话题。每个人不都是这样吗？

我确实需要更努力地与名人交朋友，只是与人们联系。在这方面，我更喜欢 Twitter，而不是脸书。

老实说，有时我似乎在强迫自己，但如果你希望自己的才能得到认可，展示自己的知识是很重要的。不是每个人都注定要做同样的事情，尤其是在 IT 界。当我这样写的时候，我会感到与我的目标紧密相连，这是一种向前迈进的强大感觉。

我自己也有很多优化工作要做...确保我的链接工作，到处都是垃圾邮件...为垃圾邮件生成新内容...确保我发送的内容是高质量的垃圾邮件(按照我的标准)...我希望我的垃圾邮件有用-垃圾邮件...

现在，继续发送这篇文章。
