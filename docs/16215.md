# 学习 JavaScript 的一个问题

> 原文：<https://dev.to/sfwritermdk/an-issue-with-learning-javascript-el6>

我已经学了几个月的 JavaScript 了。我从 Udemy 购买了一些 JavaScript 课程(其中一些是高级的)。尽管这里或那里有一些错误，我总的来说觉得它们是有用的。

Andrew Mead 的[现代 JavaScript Bootcamp](https://www.udemy.com/course/modern-javascript) 可能是我获取 JavaScript 的主要来源。我们正在构建一个 notes 应用程序和一个 todo 应用程序，并提供一些可观的功能。

我大体上很好地理解了这些概念，但是在从头开始编码时，我有一些问题。不仅知道我们为什么放置代码，而且知道我们把它放在哪里。和嵌套函数。这是我最大的挑战。

根据我从网上了解到的情况，构建这些复杂应用程序的最好办法就是开始在更多的项目上练习。我有几个个人项目想做，但是我觉得我还没准备好。

我想我会完成自由代码营的 JavaScript 模块，然后开始与该模块相关的项目。
