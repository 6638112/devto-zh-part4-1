# 生活中的感恩

> 原文:[https://dev.to/ryboflavin42/gratitude-in-your-life-8g1](https://dev.to/ryboflavin42/gratitude-in-your-life-8g1)

作为对@Ryland G 的“早上是什么让你起床？”我想知道你练习感恩吗？如果是的话，你如何练习感恩？

这是我一生中最有影响力的实践之一。我认为这种做法有助于减少焦虑，增加幸福感。正如我们所知，一个快乐的人是一个更有生产力的人。

我每天通过写或思考生活中我感激的三件事来练习。通过这个练习，我发现了一件重要的事情，那就是你真的需要为每一件物品感受情感。

## 我如何划分 3 的列表

*   你生命中最感激的一件事
*   一件你感激的小事(例如，风吹过，草在露水中的样子，等等)
*   关于你自己，有一件事你很感激(比如你的智慧，你的心，因为它从未停止跳动，等等)

你是怎么练习的？你感激什么？
