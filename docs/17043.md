# Bash 历史查看器(每日项目)

> 原文：<https://dev.to/petercour/bash-history-viewer-daily-project-2cdp>

今日项目:制作了一个 bash 历史查看器，可以自动滚动到控制台的底部。

Bash history 是 Linux 和 Apple 系统上的一个文件，它记录了控制台命令。文件是~/。bash _ 历史。

[![](img/18d3d1218480fe11b3dc4fea7b1eab8d.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--4tAQIB9Q--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://raw.githubusercontent.com/petercour/bash-history-viewer-qt/master/program.png)

简单的程序，但比在终端中滚动更容易。如果使用另一个终端模拟器(zsh？鱼？)，您的文件将位于另一个位置。
