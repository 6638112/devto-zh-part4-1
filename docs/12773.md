# 今天您需要了解的关于网络安全事件响应团队的事情

> 原文：<https://dev.to/compliance4all/things-you-need-to-know-about-cyber-security-incident-response-team-today-1hkd>

网络安全是一个组织最重要的组成部分，这一点毋庸置疑。这是不言而喻的，因为对任何组织的计算机系统的攻击可能在任何时间、任何地点、一天中的任何时间发生。这要求他们 24X7X365 全天候保持警惕。

网络安全事件响应团队(CSIRT)对于任何组织(无论是私人组织还是政府组织)来说都是不可或缺的，以确保有价值的数据不被泄露。一个 24X7 全天候准备应对网络攻击的系统应该主要包括一个报告事件和正确传播事件相关信息的机制。

Compliance4All 是一家领先的监管合规领域专业培训提供商，该公司提供的一场有价值的网络研讨会将展示如何建立一个简单可靠的 CSIRT，以帮助避免网络攻击。

本次网络研讨会将于 7 月 23 日举行，时长 60 分钟，主讲人是顾问、演讲者兼作家迈克·雷蒙。

大多数公司确实建立了 CSIRT，但大多数公司的一个主要问题是团队没有经过充分的培训来处理和测试安全问题的所有方面。测试和培训对于在团队中建立意识都是必要的，这将帮助他们更好地应对攻击或潜在的攻击。正确报告事故和传播事故相关信息是必须的。

与网络安全相关的人员，如 CISO、CSIO、首席执行官、风险经理和审计员将从本次会议中获得大量知识，会议将涵盖以下领域:

```
How often to test
Types of testing
Who should be involved
Scenario Development
Organizational process 
```
