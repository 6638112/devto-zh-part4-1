# 2019 年文档生成器入围名单

> 原文：<https://dev.to/sm0ke/best-documentation-generators-and-why-5hfc>

各位编码员好，

在某些时候，为了提高我们的技能，我们阅读下划线技术提供的文档。在这篇文章中，我试图在 2019 年整理一份用于生成文档的最佳工具列表，从我在过去 3/4 年中在各个项目中使用的工具开始。

因为我不是文档工具专家，而且这个列表很短，所以请随意添加您喜欢的文档工具，并解释您为什么选择它。

* * *

以下是我的清单:

*   [Gitbook](https://www.gitbook.com/) - live [样片](https://docs.appseed.us/)
*   一个非常好的工具
*   Sphinx -用 Python 编写，带有原生搜索(不依赖 Algolia)
*   基于 GatsbyJS 的文档生成器，需要 Algolia 插件进行搜索
*   Docusaurus 看起来很有前途，主要缺点是默认主题的丑陋。
*   [Vuepress](https://vuepress.vuejs.org/) -基于 Vue.js 的工具。
*   [KB4IT](https://github.com/t00m/KB4IT) - KB4IT 是基于 Asciidoctor - [demo](https://t00m.github.io/KB4IT/demo/) 的静态网站生成器

**列表将使用评论中提到的工具进行更新**
