# 你应该学习一门新的编程语言

> 原文:[https://dev . to/marcofiset/you-should-learn-a-new-programming-language-37li](https://dev.to/marcofiset/you-should-learn-a-new-programming-language-37li)

我写 C#、PHP 和 Javascript 都很得心应手。这些是我专业使用的语言。虽然我不认为自己是其中任何一个方面的专家，但我至少可以说我相当精通。我不再需要刻意练习这些语言的技能，因为我非常了解它们。我会不时地学习和试验新的库，如果它们能解决我的问题，就把它们集成到我的项目中。

一旦你掌握了一门语言，仅仅学习新的库是不足以让你进一步提高技能的。当然，一些库可能会出现并挑战现状，但是通常你会倾向于坚持你所习惯的模式。你只是通过走过场来完成你的工作，而不去想你在做什么。

那就是你停滞不前的地方。你最不想做的事情就是陷入自满。

过了某一点，如果你真的想把你的技能推进一步，你需要做一些不同的事情。不，只是做“同样的事情”。学习一个新的框架或一门与你已经知道的语言非常相似的语言只是做同样事情的一种新方法。当你已经知道 C#或 Javascript 时，学习 Java 或 Python 是很容易的。几个星期后，你会有宾至如归的感觉。

你想学一门一开始会让你觉得自己很笨的语言。作为开发人员，你需要迷失方向才能成长。这是你将知识的边界推向新领域的地方。
