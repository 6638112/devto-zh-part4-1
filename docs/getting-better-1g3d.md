# 变得更好

> 原文：<https://dev.to/techhead404/getting-better-1g3d>

多年来我发现的小秘密。如果你想学习一门新的编程语言或者一个新的软件。最好的方法是使用它，我不是说按照教程或者随便玩玩。制作一个网站或程序。我想学 c#，所以我一头扎进 unity，做了一个游戏。这样做的一个重要因素是发表你的作品。2019 年 3 月 15 日开始学习 C#和 Unity“还有 Gimp 和 Blender”。我在 2019 年 3 月 28 日发布了我的游戏。通过让人们参与进来并给予反馈，它推动了我，并且仍然在推动我做得更好，学得更多。此外，我还能获得对我工作的即时反馈。网络开发也是如此。我的 HTML 和 CSS 入门是一个网站，当我知道如何公开它时，它就公开了。所以如果你想学习新的东西，就去做吧。那个领域的人都经历过，他们理解。我非常担心人们会盗用我的项目。发布后，我发现检查它们的人除了积极的反馈什么都没有。我可能不会成为下一个脸书或糖果粉碎，但至少我把东西展示给世界看，这比大多数人能说的更多。
