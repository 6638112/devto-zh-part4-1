# 2019 年目标—6 月更新

> 原文：<https://dev.to/jsjoeio/2019-goals-june-update-5fhf>

六月是忙碌的一个月。我对 7 月份很兴奋，因为 a)周末看起来很空闲(目前没有旅行或活动), b)我有一个我很兴奋的项目要做😄

但是在我分享任何更新之前，这里有一点关于六月目标相关的东西的思考。

## 成就

在这里，我将分享目标，以及箭后一个月的相关成就。

### 知性

*   为 [mentored.dev](https://mentored.dev) 启动两个产品搜寻项目->成品 MVP

所以我没有“完成”这个目标，但我达到了一个里程碑。我已经和我的朋友谈论过一段时间要完成辅导。我在 GitHub 上的项目板上充满了我想要构建的所有这些令人敬畏的功能。

但是当我们在菲尼克斯反应堆聚会上聊天时，我们进行了一次现实的对话。他们说，“你还需要添加哪些特性来‘完成’v1？”

我想了一下。好吧，他们是对的。我需要缩小范围把这个弄出去。所以我把所有东西都移到了 backlog 中，只添加了关键的部分——将进度保存到本地存储中，并改进 UI 流程。

我做了所有这些，并在推特上分享了一篇关于这个周末旅程的帖子。感觉真好。这很可怕，但我投入了大量的时间和精力，我很高兴能分享它。

我不确定接下来我会把它带到哪里，但这标志着第一阶段的完成。

### 专业

*   写 6 篇文章->写 2 篇 dev.to 文章！(总数:3/6)

我想我在过去的一个月里处于教学/写作的“高潮”,因为我写了两篇文章。事实上，让它变得“简单”的是写文章，因为我正在学习两件新东西。我想，如果我边学边写，会有助于我巩固所学的知识。

第一个是[“React Native-为 Android 添加应用图标”](https://dev.to/jsjoeio/react-native-adding-app-icons-for-android-47e7)，在这里我分享了为 Android 添加应用图标的基本知识。我开始学习 React Native，并想做最简单的事情:在我的应用程序中添加一个图标。

经过许多误解后，我终于让它工作了😃所以我写这篇文章是为了 a)帮助我记忆，b)分享我学到的东西。

我写的第二篇文章的标题是[“如何在 VSCode 中创建代码概要文件”](https://dev.to/jsjoeio/how-to-create-code-profiles-in-vscode-3ofo)。我妻子让我借用她的个人笔记本电脑，直到我攒够钱买我自己的个人笔记本电脑。因此，在做准备的时候，我决定尝试一种由一位学者导师使用的技术——代码剖析。它们基本上是为 VSCode 保存的`settings.json`状态。例如，在录制需要较大字体的截屏时，这很有帮助。为了更好地理解 GitHub 上的自述文件，我写了这篇文章。

## 更新

处于 2019 年的中间点，我决定重新评估我在 1 月[日](https://medium.com/@jsjoeio/2019-goals-overview-8846eb8cb067)设定的目标。

虽然我已经朝着其中一些目标取得了进展，但我想改变方向，变得更加具体。我的长期目标是实现我所说的“时间自由”——每天按照自己喜欢的方式支配时间的能力。有人称之为财务独立。

我花了一些时间进行思维导图，来思考我如何才能到达那里，所以我希望我的目标与此一致。七月，我将专注于一个项目:一个关于 React 和 npm 包的新的 egghead 课程👀

我不会说更多，但请留意博客帖子和新的知识分子课程。

那都是乡亲们！直到下个月！
