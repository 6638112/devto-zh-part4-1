# 不要以封面来判断书籍

> 原文:[https://dev . to/ghost/never-judge-books-by-the-covers-read-them-first-3c1a](https://dev.to/ghost/never-judge-books-by-their-covers-read-them-first-3c1a)

我们中的许多人(包括我自己)都因对伟大著作的评价而感到内疚，因为它们的外表或苍白或黯淡，或伴随着艺术材料。一些经常阻止我们挖掘伟大的作者或伟大的内容制作者提供给我们的洞察力的东西。

在数字时代，同样的影响是我们对库存照片的感知，这些照片伴随着出版物(如媒体上的出版物)产生的文章。

我们有多少次因为一张糟糕的股票照片而决定放弃阅读有趣内容的机会，或者在对文章内容做出判断时匆忙翻阅一篇中型文章。

但是，当我们对文章、书籍、播客，甚至是人做出价值判断时，我们有没有停下来想一想，想一想我们错过了哪些精彩的内容。我认为这个问题的答案是否定的。

为什么我选择就这个话题写一整篇文章？答案可以在我最近买的一本书里找到，这本书是亨利·艾利森写的，主题是超验美学。

Allison 煞费苦心地(通过使用复杂的形而上学论据)试图向读者证明德国哲学家康德在他的各种互文性评论中使用的推理在这个主题上有相当重要的东西要说，以及关于人类为什么对抽象概念如美和吸引力做出价值判断的推理。

从这个意义上说，美是相对的。我觉得美的东西，你可能觉得不美，而我认为有美学价值的东西，你可能会透过正常的镜头来看。

这可能包括书籍及其封面。不幸的是，情况总是如此，所以我想这篇文章只是一种礼貌的方式，让我们注意到你下一次阅读的页面中的美。

快乐阅读！
