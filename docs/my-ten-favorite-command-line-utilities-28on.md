# 我最喜欢的十个命令行工具

> 原文：<https://dev.to/codingunicorn/my-ten-favorite-command-line-utilities-28on>

根据您使用的工具,“终端”可以提高您的工作效率，也可以降低您的工作效率。下面你会发现**十个让我更有效率的命令行工具和实用程序**。

[![](img/53e5ad0e85008697e1cff9a94c5b7750.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--QtLPtQp_--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/aau0y2pip2ubl488sbv2.jpg)

1️⃣ **[tldr](////tldr.sh)** 是`man pages`更好更简单的版本。当我忘记如何使用某个 unix 实用程序，比如`chmod`，没有比输入`tldr chmod`更好的解决方案了。

2️⃣ **[fx](////github.com/antonmedv/fx)** 是一个非常棒的命令行 JSON 处理器。无需离开命令行就可以显示、过滤和转换 JSON。
