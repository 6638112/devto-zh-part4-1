# 有什么新鲜事？

兰花现在的版本是 0.17.4 。5 月底发布的 0.17.0 是一个非常大的更新，包括了一些新的“集成”项目。这些集成使得从 Github 和 Gitlab wikis 创建 wiki，以及将您的站点发布到 Gitlab Pages 和 Bitbucket Cloud(除了已经支持的 Github 和 Netlify publishers)变得很容易。

# 进行中

你们所有人。一些大的事情正在进行中。非常大。比如，*德州大小的大*。我说的是:

*   一个完全重新设计的文档网站(你可以在这里预览它，我将非常感谢任何关于它的反馈！)
*   多模块代码文档
*   使用 [Kodiak](https://github.com/copper-leaf/kodiak) 将代码文档与 Orchid 完全分离，极大地简化了支持新语言的过程，并确保所有语言使用的一致性
*   一些急需的核心兰花功能的重构
*   一个自动化测试兰花插件和主题的新框架
*   为持续发布迁移到 GitHub 动作

所有这些都将出现在即将到来的 0.18.0 版本中。虽然我不想在这个版本上设定一个日期(我不想在它 100%准备就绪之前发布它)，但是我可以说我几乎完成了我想要添加的修改的代码。从现在开始，主要是给新网站添加收尾工作，记录新功能，并彻底测试一切。
