# 学习是偷偷的

> 原文:[https://dev.to/rpalo/learning-is-sneaky-3ecl](https://dev.to/rpalo/learning-is-sneaky-3ecl)

学编程是偷偷摸摸的。像运动一样偷偷摸摸。就像学骑自行车或滑板一样偷偷摸摸。

像你家的味道一样偷偷摸摸。你知道这味道。嗯，你不知道，直到你离开一个周末，然后回来。然后就像，“哇，那是什么味道？”这本身不是一种不好的气味；这是你的味道，你的东西，你的房子，你的邻居。你已经习惯了，以至于根本不觉得有什么味道，直到你有机会离开。但它就在那里。

学编程就是这样的。这不是一个快速的过程。你不会注意到它的发生。你可能不会觉得和昨天、上周、上个月有什么不同。但是学习就在那里。而且一直在增长。有时候，后退一步，注意到不同的事情，真的很令人鼓舞。

这种情况最近在我身上发生了两次，我想把它们分享给那些感到停滞不前的人，比如还有太多东西要学，他们没有取得任何进展。

## [](#rust-tag)铁锈标记

上周，我整理了一篇关于 rust 的帖子，当我在本地网站上启动服务器进行校对时，我编写的标记页面制作插件通知我它正在为 Rust 创建标记页面。我之前没有在 Rust 上写过任何帖子。

几个月来，我一直在努力学习 Rust 并提高其工作效率，并且一直感觉自己没有取得任何进展。因此，看到我的第一篇博文半聪明地谈论了 Rust，并被我的开发服务器稍微庆祝了一下(紧接着想到我的标签页面生成器实际上非常酷)，这是非常令人鼓舞的。也许所有那些迷你项目、视频和书籍都有点过时了！

## [](#script-magic)脚本魔法

本周我发表了一篇关于写一个脚本来修复困扰我的制表符/空格/Python 的文章。在我写完之后，我想，“伙计，如果我能比`python3 path/to/my/script.py other/path/to/my/file.md`更容易地运行这个就好了。”

因此，通过几个简单的步骤，我将文件重命名为更容易键入的名称:

