# 著名公司<>职业幸福

> 原文：<https://dev.to/alexantra/famous-company-career-happiness-2542>

并不广为人知的是，波兰，尤其是华沙，是一个视频游戏开发的发电站。这一切都要追溯到早期的个人电脑游戏，一些小工作室通过翻译像 Baldurs Gate 这样的主要国际游戏而出名。
快进到现在，我们有了 CD Projekt Red，这是一家 AAA 级波兰视频游戏开发公司，价值超过 10 亿美元。

对于该地区许多想进入视频游戏开发的人来说，CD Projekt Red 是该公司的目标。它很有名，产出高质量的作品，而且价值不菲，所以肯定是个工作的好地方，对吧？

良好的......也许吧。

该公司的 Glassdoor 简介相当暗淡([来源](https://www.gamesindustry.biz/articles/2017-10-16-cd-projekt-red-responds-to-glassdoor-reviews))，有报道称，该公司是一个紧缩文化、指责文化和欺凌的温床。

这两家公司的社交媒体账户和母公司 GOG 游戏(是的，就是他们)的社交媒体在最近几年已经表明，他们可能是 Gamergate 和反 LGBTQIA+文化的传播者。

一名前雇员将华沙独立游戏工作室的繁荣归功于逃离公司的“难民”数量。

现在承认它可能没有那么干净利落。很明显，有些人喜欢在红色裁谈会工作，并且已经工作了很多年。

我们都知道，危机文化、责备文化和职场欺凌可能是一些关键人物造成的，他们曾经改变了一切。

但我还是忍不住拿他们当例子。

当我们谈论职业发展时，不仅仅是钱的问题，公司的声誉也是很重要的一部分。无论是微软或谷歌这样的世界知名公司，还是 CD Projekt Red 这样的本地知名公司，人们都千方百计地想在这些知名公司找到工作。这是我们可以为自己设定的目标，证明自己的价值，如果我们不利用这个目标来赞美他人，那我们就是在撒谎。

但是它能追踪吗？

我认为没有...但是有一些警告。

每个人都是不同的，所以如果你让两个人做同样的工作，不能保证同样的事情/环境/管理风格会让他们开心。

所以说著名公司=每个人的职业幸福是不对的。

在著名公司工作有很多好处。以微软为例，他们有资源让你大规模接触许多不同的技术，这在你的简历上会很好看——仅举几个例子。

然而，这并不意味着微软的其他部门或每个部门都达标，这是一个谬误，我希望每个人都养成挑战的习惯。

对于那些不是微软的公司，一个有几十年经验和成功的公司，这个等式变得更加复杂。

仅仅因为一家公司价值不菲，而且世界闻名，并不意味着他们就能混在一起。

一些公司失败了。

一些公司世界闻名，价值数百万，正在快速扩张，但在幕后，它是一辆着火的垃圾车在倒车。

这并不一定意味着他们会彻底失败，并在一个月后破产。

像这样的公司可以持续很长时间，总是失败到足以保持竞争力，通过人力资本，建立技术债务，失去知识，熬过一次又一次的停电，但永远不会对股票市场造成冲击。

谁知道呢，也许他们的竞争对手也同样失败了，这也是他们没有被击沉的原因。

不要在你推动职业发展的过程中，忽视一家不知名的公司，认为它们不会给你带来快乐。

谁知道你的理想工作可能是在一些小公司，因为实际上小公司最适合你。或者一个公共机构或慈善机构给了你一直以来所缺少的东西。

随意瞄准任何你想去的公司，但是如果它不适合你也不要惊讶。
