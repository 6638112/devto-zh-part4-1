OOP 概念已经成为软件开发的重要组成部分。随着程序复杂性的增加，使用传统结构化程序设计或函数式程序设计方法的自顶向下方法编写程序变得困难。这就是面向对象编程占优势的地方。

OOP 概念允许你把这些复杂的程序分成小块，你可以一个一个地解决。OOP 是一个有效的模型，它允许你的程序增长而不会变得无法维护。除此之外，像 Java 和 OOP 原则这样的语言无处不在，成为了当前的需要。

OOP 是一种全新的解决问题的方式。这里你创建一个类，而不是写一个程序，它有变量和函数。

对象是那个类的自包含的例子，你有一个控件，你可以让它们以多种方式相互交互。这种封装使得面向对象编程及其主要原理的知识成为一种必须，如果你真的想从事大型项目和开发手机软件的话。

什么是 OOP 概念？

面向对象编程是一种基于包含数据和方法的“对象”概念的方法。它的主要目标是在构建小型和大型软件应用程序时带来编程的灵活性。大多数编程语言如 Java、C++、Python、C#等。遵循 OOP 概念。
