# 你用的是什么主题？

> 原文：<https://dev.to/taillogs/what-dev-to-theme-are-you-using-2ben>

你知道 Dev.to 有主题吗？直到最近，我肯定没有。现在我离不开 Dev.to night 主题，它甚至把“写帖子”按钮变成了一个很棒的路牌。

## 如何使用开发到主题

1.  转到`Settings`

    [![](img/9dc1b1f803b10c12fed06c266d3c785a.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--RcuE2Gpw--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ry2uw9yoegbbuj0uomxt.png)

2.  转到`Settings->Misc`

    [![](img/46ad645aa26b7f2629836e690ab45362.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--8y7rNpLI--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/vnlw8mkaob8mqryw12dd.png)

3.  向下滚动并选择你的主题

    [![](img/41242a4dd8ad3eb0e27cfa3b161aa6d5.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--C3k1zzq---/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/0ra4xpzwfxyc5ix8vqos.png)

4.  甚至转移到手机！

    [![](img/51c5daac4e4c686429ff8539841911e6.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--h-uO19MK--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/jv1k9404z7all5m8e9ls.jpeg)
