# 重新设计的东西

这是“响应式全页视差滑块”的一个分支(和翻拍)

我使用了我最近学到的技术:

*   大多数功能中都有 VueJS。
*   GSAP 只是为了滚动。

我还根据我目前的设计风格重新设计了布局。

现在这里有 3 层视差效果。

* * *

概念很相似，但设计不同。

以前，我希望文本被黑色边框包围，但现在，我删除了它。

之前，图像的背景颜色没有被黑色背景掩盖。现在我试图掩盖它。

功能上也有区别。例如，以前，滚动只能通过点击屏幕底部的链接来完成。但是在学了一些技术之后，我可以让它滚动，而你仍然可以点击下面的按钮。

这个东西也把我抛了回来，我以前没有使用过任何库和预处理器。这就是为什么，源代码比以前更复杂。
