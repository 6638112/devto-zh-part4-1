# 编码训练营回顾

> 原文：<https://dev.to/kcarrel/coding-bootcamp-retrospective-1mod>

作为一名刚刚从编码训练营毕业正在找工作的学生，我回答了很多潜在学生关于我在 Flatiron 沉浸式软件工程项目中的经历的问题。因此，这里有一篇博文详细介绍了我的经历以及我对未来学生的建议！

## 我的经历

我有一个非常积极的编码训练营体验！就背景而言，我很幸运地成为了第一批西雅图校园团队的成员(他们都获得了 5000 美元的奖学金)，并能够体验一些被派去开设新校园的“黄金标准”导师。

也就是说——这是令人筋疲力尽和情感枯竭的。最初，15 周看起来并不算长，但很快就感觉像是一场永无止境的冲刺，我的身体状况已经严重不适应了。常用的短语“喝消防水管里的水”是准确的，当生活事件同时发生时，你也试图将新信息塞进你的大脑，这种感觉再真实不过了。以下是我在整个项目中非常倚重的一些学校/生活平衡资源。

*   正念和冥想应用([平静](https://www.calm.com/)和[顶空](https://www.headspace.com/)是个人推荐)
*   Youtube 瑜伽社区非常棒，可以对抗你将要做的所有坐着和编码。我的同伴开始了一个每周一次的瑜伽小组。to_s(tretch)，并跟随 Yoga 推出了 Adriene 的[背痛系列瑜伽](https://www.youtube.com/playlist?list=PLui6Eyny-UzywNCo0zRu9dduvlNXzfDyk)。

我发现在这种学习环境中维持正常的社交生活是不可持续的。我有数不清的预先计划好的活动，因为筋疲力尽或者为了准备项目的下一阶段而不得不跳过。我不知道我的同龄人中有多少人会有不同的看法。对我来说，除了睡觉和偶尔的锻炼，找到平衡(或者我能找到的最接近平衡的东西)并不意味着什么。

总的来说，这个项目非常适合我，我对参加这个项目的决定很有信心。在校园里学习，在我的同学中建立良好的关系/朋友，为我的投资组合和毕业后的职业服务项目带走项目，这些都值得我冒着辞职的风险去追求。我会再做一次，但是我会做些什么不同的事情或者给未来的学生一些建议呢？

# 未来学生建议

## 项目周

我和我的同学一直期待熨斗学校项目期间的项目周。这是你真正巩固前两周所学知识的时候，你可以建立一些切实的东西。考虑到这一点，作为一名毕业生，我强烈建议你带着这是你希望(最终)放入你的文件夹中的心态来对待结对和单独项目周。在研究生的面试和社交过程中，在 Heroku 或 demo 上准备好我可以与之交流的项目会有无穷的帮助。

你是否很难想出项目创意？请浏览以下内容:

*   以我目前的技能，是否存在我可以用新的方式解决或处理的问题？
*   我对什么有热情？(例:我爱化妆！我做了一个[彩妆 API](https://fentybeautyapi.herokuapp.com/) 和[彩妆收集/灵感 app](https://pan-it.herokuapp.com/)
*   是否存在一个公共 API，我可以将它合并到我的项目中以增加功能？(我的 Mod3 合作伙伴项目利用了 [BreweryDB API](https://www.brewerydb.com/) 和[谷歌地图 API](https://developers.google.com/maps/documentation/) 来生成酿酒厂爬行路线)

## 投资组合

早点开始考虑创建作品集网站吧！我的第一个作品集网站在我从熨斗学校毕业后一周才上线，但它在人际交往或获得对工作的最初兴趣方面给了我很大的帮助。

## 数据结构和面试准备

如果我可以完成我的熨斗经历，我会改变的一件事是在毕业后至少一个月开始我的数据结构和面试准备的单独学习。技术面试很难，需要大量的准备。熟悉常见的概念(大 O 符号、动态编程等)和解决问题的方法(滑动窗口、伪代码和测试)对产品化后的程序很有帮助。
