# 人工智能即将出现的突破

> 原文:[https://dev . to/Faber 28 Kevin/coming-breakthrough-in-ai-3147](https://dev.to/faber28kevin/upcoming-breakthroughs-in-ai-3147)

我们都在电影中听说过人工智能，并认为这是科幻小说。但现在，由于人工智能的进步，我们正在目睹许多行业的重大变化。人工智能的[](https://singularityhub.com/2019/04/26/5-ai-breakthroughs-well-likely-see-in-the-next-5-years/)应用于医药、商业、军事、教育、农业和所有其他行业。在即将到来的未来，可以预测人工智能在行业中的一些主要用途。

**建立非人类的模式识别能力**

一个名为 AlphaGo Zero 的人工智能已经连续 100 次在一场名为 Go 的比赛中获胜，而且没有一次失败。人工智能不是通过观察人类的模式来学习，而是通过与自己玩耍来学习，并学习复杂的模式，他后来用这些模式击败了人类玩家。人工智能学习本身是一项巨大的成就，可以成为开发比 AlphaGo Zero 更复杂的人工智能的重大突破。

**人工智能在医疗领域的应用**

[【AI】](https://en.wikipedia.org/wiki/Artificial_intelligence)可用于医学和医院对病人的治疗。人工智能，作为一台机器不会犯人类可能犯的错误。它可用于诊断疾病并根据疾病向患者建议治疗。人工智能可以用于执行复杂的手术，不会出现错误，并且可以挽救许多病人的生命。人工智能可以用来比较数据，并通过分析结果和场景来发现新药。人工智能可用于制定适当的疾病治疗方案。

人工智能将为我们做体力劳动

随着 [人工智能](https://www.theverge.com/2019/1/28/18197520/ai-artificial-intelligence-machine-learning-computational-science) 的发明，我们有可能过上非常轻松的生活。我们可以看到一些推出的无人驾驶汽车。它们是全自动的，依赖人工智能来驾驶。由于人工智能是一台机器，不会出错，事故发生的几率大大降低。 [卡车](http://www.edmunds.com/truck/) 和汽车都可以由人工智能驾驶，在不久的将来，我们可能会看到只有人工智能为我们驾驶汽车。人工智能机器人可以用于餐馆接单和服务顾客。这对老板来说是有利的，因为他不用再每周给员工发工资了。人工智能是一个比人类雇员更好的雇员，因为人类会累和无聊。另一方面，一个人工智能将永远不会疲倦，并不断向顾客提供持续的服务。同样，AI 也可以用于制造业制造机器。在制造业工作是有风险的，因为一个人不得不与有风险的重型机械打交道。人工智能可以用于危险的服务，如消防救援队，刑事辩护和建筑基础设施。

**艾在提供安全保障**

人工智能可以用于军事和保护我们的隐私。近来，网络安全一直是一个大问题。人工智能可能是一件好事，因为它是黑客攻击的一种潜在疗法。人工智能可以被设计成保护我们的网络免受黑客攻击。没有人在运行算法方面比人工智能更好，人工智能可以正确地保护系统免受任何威胁。人工智能也可以用于军事，以保护敏感信息免受黑客攻击。人工智能士兵可以为战斗和危险的行动做好准备，以保护人类的生命。

**人工智能在日常生活中的应用**

除了解决复杂的问题和处理安全威胁，人工智能还可以仅仅用来给我们带来安慰。人工智能可以用来做家务，照看孩子，开车，做饭和做所有我们可能会觉得无聊的事情。在日常生活中使用人工智能可以减少我们每天必须做的令人疲惫的任务，因为人工智能，我们可以过上更舒适的生活。

毫无疑问，如果使用得当，人工智能可以造福人类，人工智能技术的进步必将为我们带来更美好的未来。
