# 什么是计算机视觉？

> 原文:[https://dev.to/petercour/what-is-computer-vision-2899](https://dev.to/petercour/what-is-computer-vision-2899)

计算机视觉是一个研究如何让计算机从图像或视频中获得高层次理解的领域。

这种理解是通过机器学习或深度学习技术获得的。那是因为仅仅通过图像处理，计算机是无法理解的。

### [](#state-of-the-art)最新技术

就像你不能通过看画中的每一个色点来判断画中有什么一样，计算机不了解其中的关系和历史数据也无法判断。

### [](#how-do-i-get-started)我如何入门？

你想学习计算机视觉吗？你可能会被外面的信息淹没。

基本上你需要知道 [Python 编程](https://pythonprogramminglanguage.com/)和[机器学习](https://pythonprogramminglanguage.com/python-machine-learning/)。

因为要掌握的信息太多了，所以从一门课程开始会更容易。

*   [机器学习课程](https://gumroad.com/l/MnRYU)
*   [Coursera 机器学习课程](https://www.coursera.org/learn/machine-learning-with-python)
*   [用 Python 实现计算机视觉](https://gumroad.com/l/GQWGG)

### [](#what-can-i-make)我能做什么？

你可以用计算机视觉制作各种应用程序。它可以应用于任何网络摄像头的饲料，图像或视频。
