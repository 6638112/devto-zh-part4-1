# 星期五爆炸#73

> 原文:[https://dev.to/horia141/friday-blast-73-pb5](https://dev.to/horia141/friday-blast-73-pb5)

[苏联车牌游戏和 Kolmogorov 复杂度(2019)](https://www.johndcook.com/blog/2019/02/02/landau-kolmogorov/) -一把牛逼！不会破坏它，但它也是一个很好的例子，说明了一些事情是可能的，一些事情是*可能的和有效的*。

[执行日落(2019)](https://codeascraft.com/2019/02/01/executing-a-sunset/) -关于 Etsy 围绕移除产品及其数据和代码的流程。一个很少被谈论的话题，在这篇文章中，你会感受到 Etsy sunsat 他们的批发，工作室和制造产品。

[苹果汽车梦(2019)](https://mondaynote.com/apple-car-dreams-b25b82997299) -或者为什么苹果不在短期内制造汽车。有很多很好的理由——因为它与他们的核心业务非常不同(就生产的对象、大小、价格等而言),利润率低，无人驾驶太远等等。OTOH，有一件事没有提到，那就是这将非常符合苹果公司以其产品而闻名的类型——地位的象征。

[呈现给高管(2018)](https://lethain.com//presenting-to-executives/)——一个来自非理性 Exhuberance 的美好。与常规的*演示*相比，这绝对是反直觉的建议。

[PostgreSQL 中的高可用性和可扩展读取(2018)](https://blog.timescale.com/scalable-postgresql-high-availability-read-scalability-streaming-replication-fb95023e2af/)-PostgreSQL 中的流复制机制及其如何影响读取的详细信息。
