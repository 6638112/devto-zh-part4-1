# 在进入 JS(和 Bootstrap)之前，先正确了解 CSS 和 Flexbox

> 原文:[https://dev . to/tarekhassan 410/get-to-know-CSS-and-flexbox-proper-before-get-into-js-3654](https://dev.to/tarekhassan410/get-to-know-css-and-flexbox-properly-before-getting-into-js-3654)

最近才意识到，在一头扎进 JS 之前，CSS 是非常重要的。我使用过不同的框架，比如 Bootstrap 和 Vuetify。但是最近我决定退一步，只上一门关于 CSS 的课程。

我有一个项目，我花了几天时间来创建样式和所有的 CSS 东西。我为此使用了 Bootstrap，但我到处都有问题，无法解决。在完成 60%的课程后，我在 Flexbox 和纯 CSS 的帮助下重新设计了这个项目，这次我只用了几个小时就完成了之前几天完成的工作。

举个例子，我以前大部分时间用 px，不知道什么是 em，什么是 rem。我也没理解好什么是立场。

因此，正如一些人建议初级学生在学习框架之前先学习普通的 JS，我认为这同样适用于 CSS 和框架。在进入框架之前要很好地理解 CSS。
