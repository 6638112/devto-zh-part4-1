# 现代 JavaScript 教程

[![](img/3ded9a7522c8a4fb10b07f2d48db216b.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--5UXZouzm--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/6p0d8dmupzte1e7x3twm.jpg)

我要展示的第一个资源是 Ilya Kantor(以及 Github 上的许多其他贡献者)的在线教程。这是一个简单的教程，涵盖了该语言的许多方面。它由三个主要部分组成:

### 1。JavaScript 语言

在这里，作者从语言的介绍开始，从基本的构建块开始，如数据类型、条件和逻辑操作符、循环、函数、原型、类等，最后是更高级的主题，如承诺、生成器、模块模式等。

### 2。浏览器:文档、事件、界面

第二部分专门讨论浏览器和使用 DOM。元素、事件、事件侦听器、表单是这一部分要处理的一些主题。

### 3。附加文章

在第三部分中，作者展示了一组不同的文章，涵盖了本教程前面部分中没有涉及的主题，例如:

*   框架和窗口
*   二进制数据和文件
*   网络请求
*   动画片
*   正则表达式

更多..

我发现这个教程很有用，也很容易掌握。作者用简单的例子解释每个主题的方式很独特，这使得读者很容易理解重点。此外，教程的结构允许简单快速地搜索任何内容！我经常就各种话题查阅本教程，我鼓励任何人都这样做😀

# 趣味趣味功能

[https://www.youtube.com/embed/2d7s3spWAzo](https://www.youtube.com/embed/2d7s3spWAzo)

我想展示的另一个资源是 YouTube 上的**Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun Fun 在这个频道中，Mattias 每周一 08:00 GMT 发布新视频，带来了非常棒的教学体验！这里涵盖了各种主题，如:**

*   函数式编程

*   TimeDivisionDuplex 时分双工
*   承诺/异步等待/生成器

还有更多的问题，比如更抽象的话题，比如是时候离开你的工作了？、*解题准备*等。

与之前的资源一样，Mattias 非常擅长以清晰易懂的方式解释高级主题，并带有很多幽默！**Fun Fun Fun Fun Function**剧集帮助我理解了承诺、生成器、迭代器和函数式编程原则。

# 埃里克·埃利奥特——中型博客

[![](img/1ac69f9e531e564087978e18291f3d7c.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--vaVoOMZE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/1bow6zr3ab53u9nwmmor.jpg)

埃里克·埃利奥特是《编程 JavaScript 应用程序》和《T2 JavaScript 场景》的作者。他在 Medium 上的博客非常详细地涵盖了广泛的 Javascript 主题。

我读过的最喜欢的一些文章如下:

*   [掌握 JavaScript 面试:什么是纯函数？](https://medium.com/javascript-scene/master-the-javascript-interview-what-is-a-pure-function-d1c076bec976)
*   [转换器:JavaScript 中高效的数据处理管道](https://medium.com/javascript-scene/transducers-efficient-data-processing-pipelines-in-javascript-7985330fe73d)
*   [咖喱和功能组成](https://medium.com/javascript-scene/curry-and-function-composition-2c208d774983)

Eric Elliot 的文章帮助我深入了解了函数范式的一些核心概念(如 currying、纯函数、转换器等)。很明显，Eric 对 Javascript 语言有很深的了解。这反映在他的文章中，每一篇文章都对所研究的主题进行了深刻而有效的分析。我一定会找时间读完他所有的帖子！

# Eric Elliot 编写 Javascript 应用程序

[![](img/98557e75c90955fb723ba05ea49b160c.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--hiUy93yA--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/mwrz14xknkpf3w30775o.jpg)

我将继续阅读 Eric Elliot 的下一篇参考资料，因为这本书全面概述了现代 Javascript 应用程序架构。这本书涵盖了各种主题，如:

*   Javascript 语言的介绍及其优势
*   功能(纯功能、lambdas、寿命、范围、提升等)
*   对象(原型与面向对象)
*   模块(AMD、ES6 模块)
*   客户端与服务器端
*   认证和授权
*   记录
*   REST APIs

还有更多，都有详细的例子。

**编写 Javascript 应用程序**帮助我全面了解了现代 Javascript 应用程序应该如何组织、测试和部署。

# 你不知道凯尔·辛普森的 JS

[![](img/a5d1915cd7030a9c3a8f0b421a807981.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--XF2vaOI8--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/mmreawa43gitts6gkdqi.jpg)

最后但同样重要的是(事实上这是我最喜欢的)，这是凯尔·辛普森的 6 本书系列，名为*你不知道的 JS* 。[凯尔·辛普森](https://me.getify.com/)教 Javascript，他真的很擅长这个！说真的，我相信他的血管里流淌着 Javascript！

该系列包括六本书，建议按以下顺序阅读:

1.  向上发展
2.  范围和关闭
3.  对象原型
4.  类型和语法
5.  异步和性能
6.  ES6 及以上

在这个系列书中，我最喜欢的是作者对这个主题的深刻理解。凯尔·辛普森展示了他对待自己所做的事情的严肃态度，这让我喜欢上了他的写作方式。
