# 我不知道的事情的最终清单

> 原文：<https://dev.to/nikoheikkila/the-definitive-list-of-things-i-don-t-know-5afp>

最近，我一直在思考我的工作，以及我想把它带向何方。因此，理解我需要学习什么才能达到目的是合乎逻辑的。事实证明，我越想我不知道的事情，我就越确信承认这些事情而不是隐藏它们的重要性。

科技行业的很大一部分人都有一种误解，认为自己需要成为无所不知、不会筋疲力尽的超级开发者。多年来，我们已经看到招聘广告荒谬地脱离现实世界。这些广告要求候选人拥有 5 年以上的专业知识，掌握 10 种以上不同的技术。

产品和业务所有者要求用更少的资源获得更高的吞吐量。当招聘人员没有就日常工作中所需的技能对现有的开发人员进行面试时，他们就没有招聘到合适的人。风险资本家建造并生活在他们自己想象的世界里，那里到处都是工程师，他们可以每天开采原钻，同时每周工作 60-80 个小时。

美国开发商也不无辜。不带一丝批评，我们倾向于分享这些大规模的学习路线图，这些路线图是关于我们在获得某个等级之前需要掌握的技能。

甚至不要问所有的冒名顶替综合症都是从哪里来的。

与此同时，我们应该支持人口少数群体在科技领域开始他们的职业生涯。对于一些主要的科技公司来说，在社交媒体上改变你的个人资料照片，加入一条酷彩虹就足够了，一切都是现成的。欢迎你成为我们有毒工作文化的一员，只要你满足这 20 个强加给你的不切实际的条件。

我们有一个严重的问题，作为专业人士，我们有责任通过向世界承认你不需要知道一切来解决这个问题。

这个帖子是由来自**丹·阿布拉莫夫**的[一个类似的列表](https://overreacted.io/things-i-dont-know-as-of-2018/)和在推特上广泛传播的关于所谓的[“10x 工程师”](https://twitter.com/skirani/status/1149302828420067328)的垃圾箱火灾引发的。

当然，这篇文章的目的不是给我自己一个不好的印象。你可以通过关注我的博客和社交媒体账户或者只是和我聊聊，来发现我擅长的很多事情。然而，让我们把它排除在这篇文章的范围之外。

没有进一步的介绍，我们开始吧。我可能会在以后修改这个列表。

## 我不知道的事情的权威列表(2019)

*   记住编程语言语法、算法、模块用法和许多其他东西，而不需要查找互联网备忘单。你以为我是存储和处理能力无限的超级计算机吗？
*   记得一个多星期前我所做的事情当你写的代码在三个星期后被推向生产需要你支持时，这是特别麻烦的。这真的是我写的吗？哦，根据版本控制我做到了。嗯，让我想想…
*   避免 bug。我们都创造了它们，没必要为此而紧张。至少我可以粉碎我生下的缺陷。
*   **命名变量、函数、类**，以及应用程序的其他具体部分。通常，只有在看到应用程序作为一个整体是如何工作得最好的之后，我才能给它们起一个合适的名字。在那之前，请享受阅读我的一般签名，如`def get_item_from_store(n: str) -> Item`。
*   将文档更新为 I 代码。根据定义，软件文档总是过时的，所以这不是一个大问题。
*   **做*测试驱动开发* (TDD)。在很多情况下，我不得不在弄清楚我需要什么样的测试之前，玩弄我的想象力，写一些样板代码。TDD 的许多假设表明，仅仅通过查看验收标准，你就可以立即知道所有的测试用例，这是非常危险的。错误的或者实现糟糕的测试总是比根本没有测试更糟糕。**
*   使用令人难以置信的指令集 *Vim* 宏和**编写代码，实现了每秒 100 行代码的惊人打字速度。**我已经用最重要的快捷键配置了我的编辑器，但我仍然需要不时地用鼠标将光标移动到正确的位置。
*   **估计完成一项功能所需的时间。**按照目前的速度，如果不出意外的话，第三方依赖是我的标准猜测，最多 1-2 周。等一下，我用我的精神 *D & D* 骰子给你掷出一些故事点。把你最好的牌带到会议上来。
*   创造美丽的用户界面。我可以使用`margin`规则将 CSS 中的几个元素居中，并从*谷歌字体*中选择一个令人满意的字体对。除此之外的任何东西，比如复杂的前端框架、网格系统和 flexboxes，都难住我了。不过，我正在慢慢适应*反应*。
*   **写低级代码。**尤其是针对嵌入式系统时。我在学习期间学过一些 C/C++课程，但是在我的职业道路转向 web 开发之后，我很高兴地忘记了大部分课程。当然，我可以通过编程 I/O 信号在 LED 显示屏上创建一个花哨的字幕文本，但我可能会忘记释放所有的内存，而你的设备只会崩溃或着火。
*   **移动平台。**我曾经试图在 Android 上创造一些东西，但注意到我不知道 Java IDEs 的黑魔法，于是放弃了。我听说你现在可以用其他语言代替 Java，但是我还是会用 web 技术来写 PWA。
*   **游戏开发。**我学过游戏设计，写过关于游戏的文章，但我不知道如何实际开发它们。我完成的最先进的东西是一个跟随教程的 *Pong* (1972)克隆体。
*   **图形设计。**在为这个博客寻找快速制作封面图片的方法时，我学会了一些设计工具( *Sketch，Lunacy，Invision* )。所以我知道如何用一些额外的剪贴画在背景上放置阴影文本。不过，不要雇我为你的下一次竞选做广告材料。
*   成为一名 IT 支持人员。驱动程序、外设、消费者操作系统和不同网络堆栈的混乱世界如此之大，以至于我不断地被淹没。如果你打电话给我修理你的打印机或家用路由器，我只能请你关机再开机。我可能会偶然建立一个远程桌面连接来重新安装任何可用的驱动程序。
*   **硬件。**与上述相关，我对我的电脑所做的最先进的事情是在里面更换新的硬盘和固态硬盘，这花了几个小时的时间。我的下一台电脑肯定是笔记本电脑。
*   …还有很多我还没有想到的事情。

我们这些工程师没有一个是“最高开发者”、“摇滚明星程序员”或“敏捷忍者”。甚至在这些愚蠢的风投们最疯狂的梦想中也没有。如果你的人力资源部门认为这份工作的一个重要部分就是不用在网上查资料就能生存，请把我的名字从要找的名单中去掉。我并不擅长所有的事情，但没关系。

但我最擅长的一件事。当我不知道某事时承认并寻求帮助。我希望你和我一样。
