# [](#projects)项目

**[学习合成器——Ableton.com](https://learningsynths.ableton.com/)**:我不是电子音乐的狂热爱好者，但制作电子音乐总是让我困惑。起初，我误以为你只要按几个按钮就能成功。是的，不。它与电子工程和数学有很大关系，但有一个有趣的方法(即使数学对我来说很有趣)来了解合成器是如何工作的。你将会了解到，用小单元的声音修改到这个东西:一个合成器。

# [](#books)书籍

**[受塔拉·韦斯特多弗](https://www.goodreads.com/book/show/35564920-educated)** 教育:塔拉出生在爱达荷州的农村，由非常虔诚的父母在山里抚养长大。生存主义者，整个家庭为一天的结束做准备，并决心远离电网，远离邪恶的政府。当他们因事故或暴力而严重受伤时，他们不会去医院。他们用酊剂或精油治疗一切，把他们的信仰交到上帝手中。从很小的时候起，塔拉的命运就已经被上帝的戒律和她拥有真正家庭权力的父亲所追踪。最终，小女孩对外面的世界产生了好奇，对正常生活的感觉。尽管之前没有高中学历，她还是努力争取获得博士学位。这本书读起来纯粹是一种享受，它让我经历了从愤怒到快乐和解脱的各种情绪。
