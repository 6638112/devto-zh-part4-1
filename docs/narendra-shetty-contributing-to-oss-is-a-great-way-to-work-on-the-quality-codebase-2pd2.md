# 纳伦德拉·谢蒂:为 OSS 做贡献是开发高质量代码库的好方法

> 原文：<https://dev.to/gitnation/narendra-shetty-contributing-to-oss-is-a-great-way-to-work-on-the-quality-codebase-2pd2>

### 我们采访了纳伦德拉·谢蒂, [Twilio](http://twilio.com) 公司的高级前端开发人员，她曾是 Booking.com 和众包的团队负责人

Narendra Shetty 是一名充满热情的高级 web 开发人员，目前在 Twilio 工作，致力于建立一个可编程的联络中心 [Twilio Flex](https://www.twilio.com/flex) 。在这次采访中，纳伦德拉讲述了他从印度搬到荷兰，然后到英国的鼓舞人心的故事，他在转变过程中面临的困难，走出舒适区的重要性，以及他成功克服的对公开演讲的恐惧。纳伦德拉在 2019 年 12 月 6 日柏林的 [React Day 发表演讲。](https://reactday.berlin/?utm_source=devto&utm_medium=blogpost&utm_campaign=interview)

#### 你好，纳兰德，欢迎来到《柏林反应日》的访谈节目！为什么是 web 开发和 JavaScript？

嗨，谢谢你邀请我。我从事 JavaScript 工作已经有 5 年多了。在学习 JavaScript 之前，我迷上了 3D 建模，然后我尝试了自己动手制作 2D 动画。我经常开玩笑说我的电脑运行 3DMax 或 Maya 不够快，而且经常死机。唯一能用的软件是一个记事本，这就是我如何成为一名网络开发人员的。😄

从小到大，我一直想做一些能解决用户问题的东西，我认为技术主要是解决问题的媒介。有了 JavaScript，我觉得我很容易学会并快速构建一些东西。
