# [](#introduction)简介

[![undraw modern professional yt9h](../Images/655e595f544550ce1cba465313617706.png)T2】](//images.ctfassets.net/ly2f59p4unnn/5jPZJAt4pyrpugvxiSU74m/93a0eba599631b858afc1b5ba57735bc/undraw_modern_professional_yt9h.png)

作为一名开发人员，要成为一名更好的开发人员，需要不断更新最新的技术并提高自己的技能。

通过技能积累，它可以帮助你定位自己在一个特定的领域工作。

除了专注于成为 React 开发人员的**前 1%** 之外，只有你能解决这个问题。

相反，你可以专注于融合多种技能或经验。

比如成为 React 前 10%、Django 前 10%、GraphQL 前 10%的开发者。

会让你成为任何需要你专业知识的人的无价之宝。

# [](#what-is-talent-stacking)什么是天赋叠加？

[![undraw percentages 0rur](../Images/3424400c85e6f3ffde6a58da40760902.png)T2】](//images.ctfassets.net/ly2f59p4unnn/5vNOUonU6UBtu0fN0l6YT2/1f31d0b813e4f4300eb2c37cb164375b/undraw_percentages_0rur.png)

[天赋堆积](https://dariusforoux.com/skill-stacking/)是由呆伯特漫画的创作者斯科特·亚当斯(Scott Adams)命名的。

他成功地将自己对商业、技术和漫画的理解融合在一起，创作出了呆伯特连环漫画

我发现这是人力资源专业人士所说的 T 型专业人士的升级形式。

你专注于成为一个专家，并有其他技能来赞美你。

# [](#how-do-you-start-skill-stacking)如何开始技能叠加？

[![undraw start building vqhd](../Images/a28b72a2ddfc3721e7b0edb33c3b7f43.png)T2】](//images.ctfassets.net/ly2f59p4unnn/6ynKoIGNNmVqj9Tba7PNVi/62c84f0ff0490f96c6006a2fbbbca002/undraw_start_building_vqhd.png)

因此，这里有一些技巧，我相信它会对你开始使用技能堆栈有用。

## [](#reviewing-your-skills)回顾你的技能

[![undraw segment analysis bdn4](../Images/6aad08c25b4e19a0e1c22cb8141354ab.png)T2】](//images.ctfassets.net/ly2f59p4unnn/1ZSaFsJE9PK6wETk4dwvTH/4a43d2592c4b8f3ac089fe6bf41093b3/undraw_segment_analysis_bdn4.png)

在我找工作的时候，这是我的导师一直提醒我的一条建议。

他还把这个建议告诉了他的客户，他们是最近被解雇的家庭主妇。

对你所学的知识进行年度技能回顾，了解你所拥有的技能，将它们重新包装成相关的。

## [](#learning-about-interesting-topics-for-yourself)了解自己感兴趣的话题

[![undraw Books l33t](../Images/1b571c3729bbdd14e5e80cb2d74fb6d6.png)T2】](//images.ctfassets.net/ly2f59p4unnn/WIvLs4zXAJNFn1lNV2BOl/7ae7d52cd9278c574e928d1d57d2f6a3/undraw_Books_l33t.png)

这是你在自己的爱好或兴趣中已经做了或学了很长时间的事情。

这会让你进入年轻时从未接触过的新行业或新技能。

就像我在贸易学校的时候，我一直想学习成为一名黑客，因为受到了 Kelvin Mitnick 的《欺骗的艺术》一书的影响。

因此，我开始学习网络安全，并通过学习编程 T4 网络，在学校里学习了 T2 嵌入式系统的开发。

这让我瞬间明白了物联网安全对于物联网设备的重要性。

这让我为一家物联网安全初创公司工作，我目前就在这家公司工作。

## [](#create-an-odyssey-plan)创建奥德赛计划

[![undraw all the data h4ki](../Images/bb8c23cd4b76132e8c7d81ba7523a4d6.png)T2】](//images.ctfassets.net/ly2f59p4unnn/M4ZVPxpwROx3ByGtuPzbO/b7bc53e7ed0aa015ace867602ac6b603/undraw_all_the_data_h4ki.png)

一个[奥德赛计划](https://youtu.be/wnU5DaIPr2Q)是对未来 5 - 10 年 **3** 不同**人生轨迹**的创造。

你想用你现在的技能或将来的技能去做你能做的工作。

避免追赶新技术，并对外面的情况有自知之明。

我是在阅读比尔·伯内特和戴夫·埃文斯的《设计你的生活:如何过上幸福快乐的生活》时了解到奥德赛计划的。

斯坦福生命设计实验室的联合创始人是谁？如果你对自己的职业不确定，我建议你读一读。
