# 从头开始制作四个项目:CSS 和 HTML

> 原文：<https://dev.to/udemyhunt/make-four-projects-from-scratch-css-and-html-1p6l>

如果你是一个有抱负的 web 开发人员，希望对 CSS 有更深的理解，这个课程将把你的技能提升到一个新的水平。

下面所有这些项目将帮助你更好地理解 CSS 中一些更复杂的概念。

例如，你将更加熟悉如何使用变量、函数、框阴影以及如何使用更复杂的 CSS 属性，如动画和变换。

本课程中的项目是使用纯 CSS 制作的:不需要 javascript 知识。
