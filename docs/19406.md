# 你好 Outreachy！

> 原文：<https://dev.to/aarthira/hello-outreachy-nbc>

又见面了，

我是阿尔提·拉贾拉曼。我被选为第 19 轮外展(五月至八月)的 43 名实习生之一。

我通过外展项目在詹金斯社区工作，我的导师是马特·西克。我还有一个来自 Outreachy 的同事 Gayathri，他也在 Jenkins 社区工作。

以下是一份问卷，分享我申请外展计划的经验。

是什么促使你申请 Outreachy？不久前，我偶然知道了外展。外联方案鼓励不同的人成为它的一部分。我可以看到我在鼓励申请的名单上，我试了一下。以前的实习生非常支持申请和贡献。

当你考虑申请时，你最兴奋的是什么？如果我能进入专业领域，我会很兴奋能在家工作。我希望通过与有经验的开发人员一起工作来获得大量的知识。此外，作为一名外展实习生的标签也让我兴奋。

是什么让你对你申请的项目充满热情？我对 Java 有一点了解，这被列为该项目的必备技能之一。此外，我以前听说过 Jenkins，并在我以前工作的一家公司的团队中看到人们使用它。

外展申请过程中有什么令人困惑的地方？填完最初的申请表后，我不知道接下来该做什么。后来，我知道我必须等到外展队评估并确认我的申请。

不然申请工作流程很顺畅。所有需要的信息都可以在外联网站上找到。我们只需要在细节上投入更多的注意力。

你是怎么找到信息的？一个家族朋友的儿子把我介绍给 Outreachy，他以前是谷歌代码之夏的成员，他称之为 GSoC。

当你陷入困境时，你问了谁一些问题？我把我所有的问题都发布到了 Jenkins 社区频道。要么是导师，要么是其他外展申请者帮我解答。我申请的时候，之前的实习生也在做这个项目。他们还帮助我和其他申请人解决问题。

在申请过程中，导师是如何帮助你的？导师帮助我回答了我的大部分问题，还帮我分配了要做的任务。此外，他还通过审查我提交给代码库的 pull 请求来帮助我。除了在项目中提供帮助，他还帮助我理解 Git 和 GitHub 的工作流程。我真诚地感谢他的支持。

你会对担心申请外展的人说些什么？作为申请人，不要太担心，因为你试图联系的人会很帮忙。这些社区非常热情。他们理解你是个新手，并准备好接受你合理的怀疑。如果你今年没有机会，明年再试试。事实上，我今年在詹金斯外展中心的同事有过在不同社区下申请外展的经历。

祝你我编码愉快:)

感谢阅读，
阿尔提
