# 100 个 PHP 面试问题和答案

> 原文：<https://dev.to/yedukondalu319/top-100-php-interview-questions-and-answers-19lh>

PHP 是网络上最流行的脚本语言。没有 PHP 脸书，雅虎，谷歌就不会存在。该课程旨在让你成为一名 PHP 专业人员。一旦你理解了所有的基础知识，本课程将帮助你创建你自己的民意测验应用程序。

1)什么是 PHP？

PHP 是一种基于脚本的 web 语言，允许开发人员动态创建生成的网页。

PHP 的首字母代表什么？

PHP 的意思是 PHP:超文本预处理器。

PHP 像哪种编程语言？

PHP 语法类似于 Perl 和 C

4)梨代表什么？

PEAR 的意思是“PHP 扩展和应用程序库”。它扩展了 PHP，并为 web 开发人员提供了更高级别的编程。

5)实际使用的 PHP 版本是什么？

7.1 或 7.2 版是 PHP 的推荐版本。

6)如何从命令行执行 PHP 脚本？

只需使用 PHP 命令行界面(CLI)并指定要执行的脚本的文件名，如下所示:

PHP script.php

7)如何从命令行界面运行交互式 PHP shell？

只需使用带有选项-a 的 PHP CLI 程序，如下所示:

php -a

8)开始和结束一段 PHP 代码的正确和最常见的两种方法是什么？

启动和结束 PHP 脚本的两种最常见的方法是:

和 [--- PHP code ---] ?>

9)如何将输出直接显示在浏览器上？

为了能够直接在浏览器上显示输出，我们必须使用特殊的标签= and ?>。

10)PHP 4 和 PHP 5 的主要区别是什么？

PHP 5 提供了许多额外的 OOP(面向对象编程)特性。

11)PHP 支持多重继承吗？

PHP 只支持单一继承；这意味着一个类可以使用关键字“extended”从一个单独的类扩展而来。

12)final 类和 final 方法的含义是什么？

PHP5 中引入了“final”。Final 类意味着该类不能被扩展，final 方法不能被重写。

13)PHP 中对象的比较是怎么做的？

我们使用操作符“==”来测试两个对象是否来自同一个类，并且具有相同的属性和相等的值。我们可以通过使用标识运算符“===”来测试两个对象是否引用同一个类的同一个实例。

14)PHP 和 HTML 如何交互？

可以通过 PHP 脚本生成 HTML，也可以将 HTML 中的信息传递给 PHP。

15)通过表单或 URL 传递值时需要什么类型的操作？

如果我们希望通过表单或 URL 传递值，那么我们需要使用 htmlspecialchars()和 urlencode()对它们进行编码和解码。

16)PHP 和 Javascript 如何交互？

PHP 和 Javascript 不能直接交互，因为 PHP 是服务器端语言，而 Javascript 是客户端语言。然而，我们可以交换变量，因为 PHP 可以生成由浏览器执行的 Javascript 代码，并且可以通过 URL 将特定的变量传回 PHP。

17)使用图像功能需要什么？

需要 GD 库来执行图像功能。
