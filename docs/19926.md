# 什么样的书、小说或电影激励你成为更好的程序员？

> 原文:[https://dev . to/prahladyeri/what-kind-of-books-reasoned-or-movie-you-be-a-better-programmer-5f9p](https://dev.to/prahladyeri/what-kind-of-books-novels-or-movies-inspired-you-to-be-a-better-programmer-5f9p)

就我个人而言，我一直最喜欢的电影是《黑客帝国》。如果你仔细想想，它完全基于虚拟现实和编程。黑客帝国本身是一个数字领域，这部电影中经常重复的著名短语*“红色药丸或绿色药丸”*也与编程有很深的联系。

第二好的可能是钢铁侠，仅仅是因为 AI/贾维斯/奥创的因素。

说到小说，有一本叫*数字堡垒*是丹·布朗写的。这是一部普通的悬疑惊悚片，但基于密码学，我建议你如果没有读过的话读一遍！其他的好书还有《阿特米斯奇幻历险》、《LOTR》和《哈利·波特》(尽管它们都不完全是关于编程的！).
