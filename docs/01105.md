## [](#octane-super-rentals-tutorial)[辛烷超级出租教程🚀](https://octane-guides-preview.emberjs.com/release/tutorial/00-part-1/)

Godfrey Chan @chancancode 和[vaidehijoshi @ vaidehijoshi](https://github.com/vaidehijoshi)为 Octane 编写了新的教程内容，做了一些很棒的工作。你现在可以在[辛烷指南](https://octane-guides-preview.emberjs.com/)中预览[修改后的超级租赁教程](https://octane-guides-preview.emberjs.com/release/tutorial/00-part-1/)。

这个[超级租房教程](https://super-rentals-tutorial--ember-super-rentals.netlify.com/)更神奇的是，它是**自动生成的**。它从 markdown 文件中运行命令，并以与阅读器相同的方式编辑文件。它甚至可以捕获已发布教程的输出和截图。这样，如果任何步骤由于 Ember 本身的变化而停止工作，我们将立即知道 CI 故障。

如果你想投稿，或者对内容有任何反馈，请查看 GitHub 上的[超级租赁-教程回购！](https://github.com/ember-learn/super-rentals-tutorial)

## [](#the-inaugural-emberjs-nizhny-novgorod-meetup)[首届 EmberJS 下诺夫哥罗德🥳见面会](https://twitter.com/NataliFel/status/1159389541490417669)

[![Alt Text](img/e8c65338062fb242dea3e703cc2e465b.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--TWZykmPg--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/7bc9uo56adek9nckkxo1.jpeg)

最近，**нижнийновгород("下诺夫哥罗德")**的 Ember 社区聚集在一起，参加[下诺夫哥罗德 EmberJS meetup group](https://emberjs-nn.timepad.ru/events/) 的第一次活动。该活动取得了巨大的成功，围绕 JavaScript、Ember、测试最佳实践等内容进行了深入的讨论。活动组织者 [Natasha Fel aka @NataliFel 在 Twitter 上](https://twitter.com/NataliFel)分享了 meetup 的亮点，包括这些令人惊叹的 [Tomster cookies](https://twitter.com/NataliFel/status/1170245679849099264) ！

期待看到烬事件在你的地区发生，但还不知道如何让这发生？查看 EmberJS meetup 地图,看看你所在的城镇是否已经有一个群组。

想创建自己的 meetup 群吗？好消息——感谢您的支持！运行一个可靠的 meetup 并不复杂，但它可能是令人生畏的:Ember 社区会很乐意帮助你轻松进入它。这里有一些给第一次组织者的提示，你可以在这里为你的活动找到任何 T2 资产。

如有任何其他问题，请务必加入 [Ember Discord chat](https://discordapp.com/invite/emberjs) ，并在 [#discord-server-admin](https://discordapp.com/channels/480462759797063690/480499718330253342) 中请求将正确的角色添加到 meetup 组织者频道。

## [](#i-contribute-to-ember-with-michelle-santarsiero)[《我为余烬做贡献》与米歇尔·桑塔西罗👩‍💻](https://discuss.emberjs.com/t/i-contribute-to-ember-with-michelle-santarsiero/17069)

[![Alt Text](img/9c4ed5b925e3030c2dbab35aff7fbcb7.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--0kAa8F----/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/7q0v28nltp8ue4rcve7q.png)

在我们的贡献者访谈系列的第十二期中，社区成员[Michelle Santarsiero @ lonely ghost](https://github.com/lonelyghost)讲述了她在 **Ember Observer** 上的工作，她的开源之旅，以及她在兼职做开源时取得**增量进展**的技巧。米歇尔，感谢你对烬社区的贡献！

你可以在灰烬论坛上阅读完整的采访。

这些采访背后的想法是为社区成员提供更多关于开源贡献、学习资源和社区活动的信息和见解。社区的贡献者获得了一个平台来分享他们对自己的经验和学习的想法，因为他们对 Ember 做出了贡献。我们希望它能让初次投稿者和高级投稿者更好地认识到这样一个事实，即每个人都是从小处开始的，每个人都可以对社区产生重大影响。
