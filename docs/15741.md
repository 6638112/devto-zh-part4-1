# 您使用的是什么 VSCode 主题？

> 原文：<https://dev.to/matthias/what-vscode-theme-are-you-using-4189>

尤其是 IDE 和 Visual Studio 代码是开发人员使用最多的工具。因为我们大多数人每天都要花很多时间在 Visual Studio 代码上，所以你可以选择自己的主题，这是一个很棒的功能。

现在我使用紫色，在此之前我用了很长时间的[吸血鬼官方](https://marketplace.visualstudio.com/items?itemName=dracula-theme.theme-dracula)，但是我想让我的 IDE 更加丰富多彩🎨

寻找你下一个主题的一个很好的资源是[vscodethemes.com](https://vscodethemes.com)(我与这个网站无关😉).
