# 怎样才能写出一篇优秀的编程/技术文章？

> 原文:[https://dev . to/erxk/what-makes-a-great-programming-technical-article-a65](https://dev.to/erxk/what-makes-a-great-programming-technical-article-a65)

当考虑一篇编程/技术文章是否伟大时，你会寻找什么？你喜欢什么风格的技术文章？提示和技巧、教程、操作指南等...

对我自己来说，我重视文章是否有关联的存储库，这样我就可以下载源代码或示例。如果所提供的信息有实例支持，并且文字清晰简洁，我通常认为这是一篇好文章。
