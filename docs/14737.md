# 真实世界体验

> 原文：<https://dev.to/angelarae63/real-world-experience-4mpk>

我最近读了 Devon Campbell 的帖子:[推进:有抱负的 Web 开发人员的禁欲主义](https://dev.to/raddevon/pushing-through-stoicism-for-aspiring-web-developers-26lg)。和许多其他好建议一样，这一条让我印象深刻:

> 这位哲学家告诉你不要再做教程了。我并不是说你不应该使用它们，而是不要再把它作为你学习的唯一途径。

正如 Devon 解释的那样，开发面临着一个问题，并在软件中构建一个解决方案。因此，成为一名网络开发人员的最好方法，就是用软件解决问题。我从 Dev.to 的其他人那里听到了这个建议，但是我觉得我没有足够的知识来解决任何问题。并非如此。我知道 HTML5 和 CSS，刚刚开始 javascript 的课程。我知道一些事情。:0)如果你和我一样，你也一样！

下面是为初学者准备的 10 个伟大的网络开发学习项目的列表。他们来自德文的网站，拉德德文。我已经将其中的两个添加到我的 GitHub 库中。一是因为我真的很想要，二是因为我知道用 HTML 和 CSS 也能做到。

我希望这将对我的初学者朋友有所帮助，我也喜欢更多的可以用 HTML 和 CSS 或者一点 javascript 构建的学习项目想法。；->有经验的开发者，让这篇文章更有帮助...补充你的建议！此外，一定要在 Dev.to 上阅读 Devon 的文章，给他一个“关注”并查看他的[网站](https://raddevon.com/)。他太棒了。
