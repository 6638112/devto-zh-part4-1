我正在工作的奖励计划登录页面上工作，我很好奇 Dev.to 认为它需要什么或什么可以更好，等等。

## 我的想法

*   而不是有一个“注册”和“登录”按钮具有同等的重量，只是有一个“注册”按钮与“已经有一个帐户？签到”下面的链接。

*   包括一个卡片滑块，其中包含您可以赢取积分的内容以及您可以兑换积分的内容。

*   常见问题部分。
