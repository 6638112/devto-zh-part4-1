如果你点击栈溢出上的`share`链接，你现在会看到三个选项，而以前只有两个。

如果你点击`DEV`图标，你将被带到一个带有预填充 Stack Overflow liquid 标签的 DEV 编辑器，使得在这个社区内分享问题和答案更加容易。

您这样做可能是为了分享一个有用的技巧，丰富一个教程，或者编辑或主持一个关于堆栈溢出内容的“元”讨论。它的格式比堆栈溢出本身要宽松得多，可以自由地进行各种各样的对话。

这个特性之所以成为可能，是因为最近在 DEV 上发布了两个特性:
