使用 destructuring 可以轻松地反转多个值。

假设你有`let a = 1`和`let b = 2`，你很容易就可以
反转数值，使得`a = 2`和`b = 1`只使用:

```
let a = 1;
let b = 2;
[a, b] = [b, a];
// a será 2
// b será 1 
```
