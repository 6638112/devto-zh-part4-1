# 如何在开发人员上启用夜间模式

> 原文:[https://dev.to/max/how-to-enable-dark-mode-on-dev-to-4c3f](https://dev.to/max/how-to-enable-dark-mode-on-dev-to-4c3f)

我看到许多人试图通过扩展、代码技巧等方式让 Dev.to 进入夜间模式。

[![my alt text](../Images/711fd6915c270635b548779de2d5e220.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--zSkUC1sl--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/swctvf6hb9zumn10n4gd.png) 

<figure>

<figcaption>一张 Dev.to 的夜间模式如何的预告图</figcaption>

</figure>

为了在 Dev.to 中启用夜间模式，你唯一需要知道的是:**你的个人资料- >设置- >杂项(左侧菜单)**并在**风格定制- >站点主题**类别中更改你想要的主题。

[![my alt text](../Images/e15b7a9bae95157b04371e02bbb95577.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--Elt4uFNA--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/cqkj4umvs33rqb8gztvf.png) 

<figure>

<figcaption>在这里您可以在几个主题之间进行切换</figcaption>

</figure>

点击**提交**按钮，开始享受吧！🖖

请注意，Dev.to 上的夜间模式是公测。如果你想投稿，可以上 GitHub 回购。🤓

## 什么是 Forem？

Forem 是用于构建社区的开源软件。为你的同行、客户、粉丝、家人、朋友以及任何其他需要聚集在一起成为集体一员的时间和空间的社区[参见我们的公告帖子](https://dev.to/devteam/for-empowering-community-2k6h)以获得 Forem 是什么的高级概述。

[dev.to](https://dev.to) (或者只是 dev)由 Forem 托管。这是一个软件开发人员的社区，他们撰写文章，参与讨论，并建立自己的专业档案。我们重视支持性和建设性的对话，为所有成员追求伟大的准则和职业发展。该生态系统涵盖从初学者到高级开发人员，欢迎所有人找到自己的位置…
