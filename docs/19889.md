在我的 [discord](https://discord.gg/FYjqm99) 频道(我用来召集游戏和技术人员的频道)中，我被问了好几次，有人可以在哪里学习如何编程。这完全取决于你想做什么！

对我来说，我有技术背景，所以对整个堆栈感兴趣。从前端——设计、用户界面、网站的外观和感觉，到后端——网站如何运行，以及所有你看不到的技术细节，但是让一切正常工作。如果你是一个初学者，并且你有一台可以上网的电脑，你可以马上开始使用大量的免费软件和资源。(还有 iOS/Android 移动开发，游戏开发等我会在另一篇帖子里推荐。)

以下是我给刚开始进行 *web* 开发的人列出的清单:

## [freeCodeCamp](https://www.freecodecamp.com/)

FreeCodeCamp 旨在免费教授代码并帮助非营利组织。不需要任何软件，只需要你的电脑、互联网和网络浏览器。

## [由大会破折号](https://dash.generalassemb.ly/)

Dash 是另一个免费的在线课程，它将教你 HTML5 和 CSS 的基础知识，由大会训练营创建。

## [代码学院](https://www.codecademy.com)

Code Academy 为 HTML、CSS 和 React 等更高级的 Javascript 框架以及 Ruby with Ruby on Rails 等其他语言提供了一些免费课程。

## 【Code.org】T2

Code.org 有大量针对 K-12 及以上儿童的教程！你可以做一个小时的辅导或者继续学习计算机科学基础知识。真的没有太早或太晚的年龄范围，所以试试吧！

## [可汗学院](https://www.khanacademy.org/)

Khan Academy 是另一个非营利组织，希望为每个人提供免费教育，并提供如何学习编程的良好选择。

希望你能找到这些网站有用的资源，开始你自己的编码之旅！当我发现更多的免费资源网站时，我会把它们添加到这里。请务必留意我的 [Twitch](https://www.twitch.tv/binarydigit) 频道，我有时会在那里写代码，希望能和你一起聊天学习！🎉
