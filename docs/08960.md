# 每日挑战#37 -姓名交换

> 原文：<https://dev.to/thepracticaldev/daily-challenge-37-name-swap-4hg5>

在今天的[挑战](https://www.codewars.com/kata/559ac78160f0be07c200005a)中，我们要求您编写一个函数，返回一个字符串，其中个人的名字与他们的姓氏进行了交换。

> 比如:nameShuffler(‘艾玛·麦克莱恩’)；= > "麦克莱恩·艾玛"
