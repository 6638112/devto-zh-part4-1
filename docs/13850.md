# 用这 8 个简单的 SEO 技巧让更多的人访问你的网站

> 原文:[https://dev . to/mcapodici/get-more-people-to-your-site with this-8-simple-SEO-tips-40jd](https://dev.to/mcapodici/get-more-people-to-your-site-with-these-8-simple-seo-tips-40jd)

如果你想让人们访问你的网站，那么 SEO(搜索引擎优化)是一个有成本效益和可持续的方式来帮助实现这一目标。SEO 的好处是:

1.  这比付费搜索或社交媒体广告要便宜得多。
2.  你得到的访问者的类型可能是更好的“质量”，即更感兴趣或更有动力专门搜索你提供的东西。
3.  随着时间的推移，对 SEO 的投资会不断增加。

主要缺点是:

1.  这需要耐心。你不能在搜索引擎优化的同一天获得流量，你可能需要等待几天或几周才能获得流量，甚至几个月才能得到满意的结果。
2.  你依赖于谷歌，如果事情出了问题，尽管不是你自己的错，你的流量来源可能会嘎然而止。

如果你想开始搜索引擎优化，这里有一些快速的提示，不会花太多时间，但与什么都不做相比，会有很大的不同:

# [](#1-use-the-title-and-meta-description-tags)1。使用*标题*和*元描述*标签。

这是一个显而易见的问题，但人们仍然会忽略它。它实现了两个目的。一是告诉搜索引擎你的网站是关于什么的，二是配置你希望你的网站在搜索结果中的外观。

因此，有几件事需要考虑:

*   这个页面的目标关键字是什么，我如何在标题和/或描述中插入准确的关键字或类似的内容？
*   当有人在搜索那些关键词的时候，他们的心态是怎样的，他们在寻找什么？
*   他们看到我的标题和描述，会感兴趣并点击吗？
*   一旦他们点击，他们会对在页面上找到的东西感到高兴，还是会感到误导或失望，或者只是“嗯”了一声，然后返回？

这是一个需要考虑的问题，但是如果做对了，你将会得到更多的访客。

下面是一个标签的例子，它需要放在 head 标签里面:

```
<title>This is my title</title>
<meta name="description" content="This is my description"> 
```

# [](#2-use-heading-tags-for-your-keyword)2。为你的关键词使用标题标签。

另一个古老的提示！

你的页面应该有一个主要的目标关键字。通常是“训狗”或者“训狗波士顿”之类的短语。你将在标题和元标签中使用它，你也应该使用它(但不要过度使用它！)在页面的标题标签中。

如果它没有使用一个意思相同的变体。谷歌可以分辨出“训狗”和“训狗”差不多，所以如果第二个更自然，那就用它。

我对关键词的经验法则是把它们放进去，但是如果读者觉得你太频繁地说同一个短语很奇怪，那么你很可能做过头了。为人类读者而写，而不是机器人搜索引擎。

# [](#3-write-great-original-content)3。写出很棒的原创内容。

你不想从其他地方复制和粘贴内容。谷歌会看着那个页面，认为它不如原来的。你也不想产生或获得廉价的作家，你想要的是坚实的内容。人们想要链接的少而精的内容比大量的不良内容要好。

# [](#4-make-sure-your-site-is-fast)4。确保你的网站速度快。

让你的网站更快的秘诀是:

*   理想情况下使用静态托管和 CDN。Netlify 在这里是一个很好的选择，而且是免费的。我现在在我所有的网站上使用它，包括这个网站。
*   如果不是静态托管，那么确保你正在使用一个合适的工具积极地缓存(例如，对于 Wordpress，看看[https://wordpress.org/support/article/optimization-caching/](https://wordpress.org/support/article/optimization-caching/))并且你的托管服务器有足够的能力。如果没有，可能是时候升级到一个更好的主机方案了。
*   通过在 Chrome 中查看您站点的页面，按 F12 键并单击网络选项卡，检查浏览器需要下载多少资源。在不牺牲你想在网站上做的事情的情况下，尽可能地降低成本。总容量小于 250kb 是一个很好的目标。删除不必要的字体、JavaScript 和降低图像质量和/或尺寸会有所帮助。
*   在 https://developers.google.com/speed/pagespeed/insights/用谷歌检查你的网站速度

让你的网站快速有双重效果。一是你会被谷歌排名更好。第二个真正的访问者你的网站不会感到沮丧，点击离开。

# [](#5-build-backlinks-in-an-ethical-way)5。以道德的方式建立反向链接

在 2019 年，你仍然需要反向链接，获得反向链接的最佳方式是编写伟大的内容，让人们了解它。你也可以写客座博文，在论坛/社交媒体签名和个人资料中使用你的链接(如果允许的话)，然后去那些论坛和社交媒体帮助人们。

带有 rel="nofollow "属性的链接对 SEO 没有什么价值。同样，使用 JS 动态创建的链接是没有价值的。所以如果你时间有限，试着优先建立链接，当你在页面上“查看源代码”时，你会得到一个链接，并且没有 nofollow 属性。

避免使用反向链接建设服务，因为他们中的许多垃圾网站，这些链接充其量会很快变成灰尘，因为这些网站的版主删除了垃圾邮件。没有人管理的网站很可能无法维护，最终会消亡，让你一无所有。谷歌也很清楚垃圾链接和权威网站的好链接之间的区别。

# [](#6-google-search-console)6。谷歌搜索控制台

这是必须的，而且非常简单，而且是免费的。注册[谷歌搜索控制台](https://search.google.com/search-console/about)并验证你的网站。现在，只需等待几天并登录，您将开始看到一些见解，包括:

*   你在谷歌中出现过哪些关键词？
*   你得到了多少点击。
*   人们登陆你的网站的地方。
*   典型的搜索结果位置。
*   关于你需要对你的网站做些什么来提高搜索引擎优化的警告。

使用这些信息，你就可以决定在未来的文章中以什么关键词为目标，或者哪些页面可以做一些反向链接，以帮助将它们从谷歌搜索结果的第 2 页推到第 1 页。

# [](#7-make-a-sitemap)7。制作网站地图

XML 站点地图是一种告诉 Google 你的站点上有哪些页面的方式，有助于发现和快速索引。如何在上创建将取决于你的网站的技术。如果你正在使用 Wordpress，那么你可以使用 [sitemap 插件](https://en-au.wordpress.org/plugins/google-sitemap-generator/)。

一旦你有了网站地图，通过登录谷歌搜索控制台提交给谷歌(有用吧！)并提交。
