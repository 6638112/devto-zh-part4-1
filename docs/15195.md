# 在家工作时我如何保持专注:直播我自己

> 原文:[https://dev . to/Andy Li/how-I-stay-focused-when-working-home-live-streaming-my-2d hi](https://dev.to/andyli/how-i-stay-focused-when-working-at-home-live-streaming-myself-2dhi)

我是一个典型的拖延者-90%的时间都没有心情工作，有很多借口做不重要的任务，容易分心。

当我辞去之前的全职工作，开始创业时，我非常担心我会一直玩《英雄联盟》，花光我的积蓄，直到我找到另一份办公室工作。

我试着给自己设定最后期限。不是假的，但是我提交了一份在 Haxe 峰会上介绍 Giffon 的报告，这是一个“真正的”截止日期。这很有效，但主要是在演讲前的两周。我甚至在上台的前一天就开始制作主要的特色。我从峰会回来后，生产力的爆发并没有持续多久。

我遇到了一个关于直播节目的黑客新闻讨论(当然是在我拖延的时候)。这听起来对我来说很有趣，因为我有点喜欢展示东西和与人接触，基本上和我喜欢在会议上发言的原因一样。

一旦我决定试一试，我就拿出我的 Windows 版 Kinect，把它设置成我的相机(只是因为我的电脑没有摄像头)，在推特上发布它，这样我就有了一些观众，并在第二天开始这样做。

[![GIF image banners for the Giffon wish pages](../Images/5f624895357589539072e26b5de22b32.png)T2】](https://gph.is/g/4Lx1M7p)

我的第一个流是我为 Giffon 愿望页面实现 GIF 图像横幅。大约有 5 个观众，其中两个是我在哈克斯团队的朋友。我不知道我的信息流对我的观众有多有趣(可能不太有趣)，但至少我自己感觉很好。我一度陷入困境，我的朋友在聊天中与我讨论了这个问题。我没有尝试过结对编程，但我猜这将是一种类似的体验。在我的第一个视频流结束时，我几乎用了两个多小时就完成了这部电影。这是我能得到的最高编码效率。

我不仅在流期间是有效率的，而且在流之后也是有效率的，因为我已经启动了我的工作，惯性允许我润色特性并在流外做一些重构。

从那以后，我每个工作日都在流。不时有不同的陌生人过来和我聊一会儿，有时聊编程语言，有时聊职业规划，有时聊我正在听的背景音乐。聊天的量正好让我开心，而且我一直都很有效率。有时候，同一位观众在下一个节目中再次出现，这非常激励我继续下去。

所以，是的，到目前为止，直播对我来说是对抗拖延症最成功的武器。如果您还没有尝试过，我鼓励您尝试一下。如果你有抽搐流，请给我留下评论，如果我们的时机合适，我会尝试[袭击](https://help.twitch.tv/s/article/how-to-use-raids)你:)
