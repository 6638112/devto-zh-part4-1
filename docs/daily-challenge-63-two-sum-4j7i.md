# 每日挑战#63-两个夏天

> 原文：<https://dev.to/thepracticaldev/daily-challenge-63-two-sum-4j7i>

编写一个函数，它接受一个数字数组(测试用的整数)和一个目标数字。它应该在数组中找到两个不同的项，当它们加在一起时，给出目标值。这些条目的索引应该以这样的元组返回:`(index1, index2)`。

出于本形的目的，有些测试可能有多个答案；任何有效的解决方案都将被接受。

输入将总是有效的(数字将是长度为 2 或更长的数组，并且所有的项都将是数字；目标将始终是该数组中两个不同项的总和)。

依据:[https://leetcode.com/problems/two-sum/](http://oj.leetcode.com/problems/two-sum/)

```
twoSum [1, 2, 3] 4 === (0, 2) 
```
