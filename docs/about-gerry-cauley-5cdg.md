# 关于格里·考利

> 原文：<https://dev.to/gerrycauley/about-gerry-cauley-5cdg>

Gerry Cauley 是西门子全球能源业务咨询集团的首席顾问。他的工作重点是新兴技术的整合，包括可再生能源、分布式能源和数字技术。他为能源公司提供有关极端风险事件缓解、弹性和网络安全的建议。他还就新出现的监管和合规问题提供专业知识。在加入西门子之前，Cauley 先生曾在南美洲的圭亚那服务，作为曼尼托巴水电国际咨询集团的一员，为国家公用事业圭亚那电力和照明公司的首席执行官和行政团队提供咨询。Cauley 先生重点介绍了系统控制中心的升级、配电馈线断电性能的改善、整合公用事业规模太阳能设施的计划，以及通过购电协议谈判进行的资源扩张。
