# 为什么倾听团队中幸存者的心声至关重要

> 原文:[https://dev . to/desi/why-倾听你团队中幸存者的心声至关重要-26bg](https://dev.to/desi/why-listening-to-the-survivors-on-your-team-is-vitally-important-26bg)

这篇文章的原标题是“为什么在你的团队中有幸存者是至关重要的”，但是在过去的几周里，我意识到这还不够。这不是这里的解决方案。

我最近注意到，有许多应用程序和服务并没有将用户的安全放在心上——不仅仅是数据隐私(我们都知道这是一个神话)，而是防止滥用或剥削，或者(冒着听起来像~ *~千年雪花~* ~)即使在采取建议的步骤以避免进一步滥用后也被触发。

每个人都有他们想在 Twitter 上取消关注或屏蔽的人，无论是他们讨厌的[巨魔还是骚扰者](https://www.huffpost.com/entry/twitter-trolls-almost-ruined-my-life_n_5c70081ce4b03cfdaa546e8d)，他们觉得难以忍受的“思想领袖”，或者他们可能不同意的某些政治人物。在这种情况下，如果有人在 Twitter Moments 或其他地方看到该用户，这可能没有害处。然而，在其他情况下，经历所有这些步骤似乎并不重要。

在这种情况下，我觉得我已经使用了 Twitter 提供给我的工具。“退订”Twitter 邮件(但神秘的是仍然会收到)。阻止用户。去私人的。我已经做了这些事情，但我不仅仍然收到我已经退订的电子邮件，*这个人仍然在他们的主题行和正文内容中非常突出和重要地出现*。

还有什么其他的变通方法？我如何进一步保护自己，避免目睹我所经历的虐待中常见的行为？

现在，我仍然可以在这个人的推文中被标记。事情发生在今天。即使我看不到他对父邮件的回复，这里似乎也应该有某种预防措施。( [@jack](https://dev.to/jack) )，如果你需要一些关于解决这个问题的用户故事或潜在功能，我很乐意成为顾问。)

如果用户被[【验证】](https://www.theatlantic.com/technology/archive/2019/06/instagram-and-twitter-should-eliminate-verification/592351)了，那就更糟了，因为他们经常被“更认真地”对待，或者看起来“更有信誉”，成功的人怎么可能是滥用者呢？(据称，他们还可以获得 Twitter 的优先支持。)

你的队伍里已经有幸存者了。他们可能没有悬挂他们的创伤幸存旗帜，也不应该需要。员工应该得到足够的信任，不需要对他们的每一个建议进行游说或推理——这些建议可以说是生死攸关的。

为了被认真对待并得到他们的建议——通常是他们极其重要的、与安全相关的建议——你需要*倾听他们*。很难变得脆弱，大声说出来，把自己置于一个你觉得自己的安全可能受到威胁的境地。即使在我的创伤过去几年后，我仍然发现很难大声说出来，并对发生的事情超级诚实。我宁愿忘掉它，忘记它的发生，然后继续下一件事。

然而，社交媒体让这变得相当困难。

现在是 2019 年。“不要使用社交媒体”，这是我不止一次听到的话，不是一个选项。

我们日常生活中使用的其他应用和程序也存在安全风险，甚至可能更加危险。

巴塞尔艺术展展示了从某人的推特上截取的图片——没有得到他们的许可——只是在 Instagram 上看到展览才知道的。在这种情况下，这位“艺术家”的 Twitter 账户被停用了，尽管如果他要创建一个新账户，肯定没有什么可以保护海伦免受报复。

我显然不是说你应该在招聘过程中包括这样的资格审查问题(因为剧透一下，你的员工已经有这种多样性了)或者你应该四处询问同事们的创伤。不要催促他们寻找某种行为或用例的理由。他们不是唯一有这种担心的人，他们没有把它作为一个边缘案例提出来。但我要说的是，当员工提出对你的某项功能的安全担忧时，请倾听他们的意见。相信他们。
