# 音乐编程

> 原文：<https://dev.to/lpellis/programming-music-56oi>

你一般更喜欢用什么样的音乐来编码？就我个人而言，我发现自己在寻找:

*   长时间(30 分钟以上)混音，这样我可以保持相同的速度
*   没有人声，我发现听人声让我更难专注于编码(其他人可能不会被它们困扰)

下面是一些我喜欢的混音，通常我会在我的第二台显示器的背景中打开它们(这有助于视觉效果也很好，没有太多的变化)
