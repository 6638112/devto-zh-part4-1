## 什么是 CSS 特异性？

根据 MDN

> **特异性**是浏览器决定哪些 CSS 属性值与某个元素最相关，因此将被应用的方法。特异性基于由不同种类的 CSS 选择器组成的匹配规则

换句话说，它只是一个规则集，web 浏览器使用它来决定将哪个 css 属性以更高的优先级添加到给定的元素或与之匹配的选择器上。

### 如何计算特异性？

有许多方法可以计算特异性，但我将向你展示最简单的方法，它被称为 ABCD 方法。

**A** 代表样式属性或内联样式，优先级最高

**B** 代表 id 选择器，当应用于一个元素时，它具有第二高的优先级

**C** 表示类、伪元素和属性

**D** 代表元素

从上面的描述中你会明白 **A** 的值是 1000，而 **B** 的值是 100，同样 **C** 的值是 10， **D** 的值是 1。

如果一个元素上只应用了一个，看起来会是这样:

```
 A = 1
 B = 0
 C = 0
 D = 0 
```
