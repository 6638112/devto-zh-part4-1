## 凯莉的故事

[![kelly image](img/471d5956e09da71b3f2a687451d8bb01.png)](/kelly)

## [凯莉·沃恩](/kelly)

[Entrepreneur, agency owner, frontend developer](/kelly)

[![twitter logo](img/ecef78ee24c258a213354fc0e60fd71a.png)【kvly】](https://twitter.com/kvlly)[![external link icon](img/7ad9ad23055d49c106b927d92662ca16.png)【http://thetaproom . com】](http://thetaproom.com)

我 11 岁的时候就开始了。我想在尼奥宠物上建立自己的社区。如果你不熟悉尼奥宠物，这个网站可以让你拥有自己的虚拟宠物，你可以喂养它，玩游戏。你可以有自己的社区和各种各样的东西。

我真的想建立自己的社区，并对其进行定制。但是你需要知道如何编码来做到这一点。所以，我向我爸爸要了一本 HTML 书，我的第一本书是 HTML goodies。我觉得这很棒，很有趣。

快进到高中，我修了 AP 计算机科学。我高中的第一门语言是 Java，我学得很差，以至于我需要一个家教来通过这门课。我当时想“我永远不会这么做了。”但是，你知道，很明显，17 年过去了，我仍然在编码，所以最终有了灵感，但不是 Java。

## 林赛的故事

[![lkopacz image](img/cd7d14f0da8b5733c447c96ff9f5a2f6.png)](/lkopacz)

## [林赛·科帕奇](/lkopacz)

[I'm a self-taught Front End & JS Dev and professional learner with accessibility expertise. I'm passionate about breaking down concepts into relatable concepts, making it more approachable.](/lkopacz)

[![twitter logo](img/ecef78ee24c258a213354fc0e60fd71a.png)利特尔科普](https://twitter.com/LittleKope)[![github logo](img/7e90f0f60c25b501324445b96acd3de8.png)lkopacz](https://github.com/lkopacz)[![external link icon](img/7ad9ad23055d49c106b927d92662ca16.png)https://www.a11ywithlindsey.com](https://www.a11ywithlindsey.com)

我是一名行政助理，漫无目的地完成我的工作；我很痛苦。我父亲向我建议编码，就像“最坏的情况是，如果你讨厌它，你可以尝试其他东西。”

所以，他给我买了一本“Drupal 入门”的书——坦率地说，我不建议你在学习编程时开始使用 Drupal，但我就是这么做的。我只是继续学习，继续在早上上班前工作。然后我得到了我的第一份 Drupal 工作。

几年后，我开始了解可访问性。大约一年后，我开始学习 React 和 VueJS，现在我是 React 的开发人员，剩下的就是历史了。

## 阿里的故事

[![aspittel image](img/45a8d86fe0ff981bf7cdb45b471a3c5c.png)](/aspittel)

## [阿里·斯皮特](/aspittel)

[Passionate about education, Python, JavaScript, and code art.](/aspittel)

[![twitter logo](img/ecef78ee24c258a213354fc0e60fd71a.png)阿斯佩特](https://twitter.com/ASpittel) [ ![github logo](img/7e90f0f60c25b501324445b96acd3de8.png)阿斯佩特](https://github.com/aspittel)[![external link icon](img/7ad9ad23055d49c106b927d92662ca16.png)https://www.alispit.tel/links/](https://www.alispit.tel/links/)

我在大学二年级的时候学会了编码。我甚至不知道什么是编码，但我有一个额外的课程模块。我决定报名参加，因为人们一直告诉我，“学习是件好事，无论如何都会对你有帮助。”

我真的以为我将学习如何格式化 Word 文档或类似的东西——我不知道什么是代码。但是，我写的是 Python。这非常有趣:我们在制作游戏。我只是觉得把东西放在一起，然后从中产生一个程序是很神奇的。

我完全重新安排了我的时间表，就好像我要攻读计算机科学的双学位，这是我的未来。所以，我开始了我的第二个 C++课程，这是一个超级密集的“数据结构和算法”课程，也是一个淘汰课程。我在那堂课上通宵达旦地努力学习，就是为了理解它。我确实取得了好成绩。我想“如果我必须这么努力，这意味着我做得不够好。这真的不适合我。”

然后当我在大学三年级时，我有一份实习，主要做 Excel 数据分析类型的工作，我发现我可以使用我所拥有的 Python 技能来自动化我自己的许多工作。因此，从那里，我得到了一份软件工程实习，并过渡到全职从事代码工作。
