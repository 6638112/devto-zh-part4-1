# 又来了！

抱歉，大家发帖晚了。这是忙碌的一周。真是过山车之旅。我是美国空军预备役的一员，参加过一次军事演习。还没有完成大量的编码工作。我将向你们介绍过去一周直到周五的情况，这是我例行的总结。

让我们开始围捕吧！Lol。

*   周六和周日——上周末没有完成任何编码。在等待前往纽约州北部参加军事演习的绿灯时，我专注于与家人在一起的时间。
*   周一——终于得到了绿灯。当我前往机场的时候，我得知我的小弟弟在抱怨胃痛难忍后被送进了医院。他得了阑尾炎，医生说需要切除。我一登机就接到通知说他要接受手术。我的手机收到几条信息，说手术一切顺利，我弟弟正在康复中。
*   星期二-做了些军事上的事情。开始在 JS 上做笔记。开始浏览一些 JS 教程，希望能找到一个合适的。当弗拉维奥开始构建一个从创意到生产的应用程序时，他开始关注[。观察一个人如何经历这些不同的阶段应该很有趣。](https://flaviocopes.com/)
*   星期三-今天没有编码完成。多亏了这个练习，我的睡眠时间表被打乱了。尽管如此，我还是很享受和我的团队一起吃晚饭的时光。他们是很好的伙伴。
*   周四-睡眠时间表还在厕所里。今天也不编码。去了离我们住的地方不远的购物中心的一个非常干净的地方。美味的食物，好喝的饮料和随意的琐事。
*   周五——在 YouTube 上找到了一个针对初学者的 JavaScript 速成班，看起来很受欢迎。在进入韦斯·博斯的 [JavaScript 30](https://javascript30.com/) 课程之前，我会先看完它。

这一周就到此为止。我确实计划继续并完成由 [Jonas Schmedtmann](https://codingheroes.io/) 教授的[2019](https://www.udemy.com/the-complete-javascript-course/)完整 JavaScript 课程。我只是想为基础课找到不同的教学风格。我真的很喜欢 Jonas 带你经历的不同项目的完整 JavaScript 课程 2019。他有一个基本的预算应用程序，我觉得它会为我想为我的投资组合构建的预算应用程序提供一些良好的基础。

我希望每个人都度过了美好的一周。谢谢你跟随我。期待以后给大家更多的内容。
