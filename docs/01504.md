# 命名事物

> 原文：<https://dev.to/skhmt/naming-things-4fca>

所以我在实现一个`forEach`方法，并问我的同事，“向后迭代的 for 循环有名字吗？大概是:“

```
for (let i = myObj.length - 1; i >= 0; i--) { ... } 
```

遇见`rofEach`和`rof loop`。这不仅仅是单词“for”反过来了，它还意味着“颠倒顺序”。
