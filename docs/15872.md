# Google 的 Flutter 移动应用开发的下一个大 UI 框架？

> 原文:[https://dev . to/spiredigitalco/Google-s-flutter-the-next-big-ui-framework-in-mobile-app-development-4ei 7](https://dev.to/spiredigitalco/google-s-flutter-the-next-big-ui-framework-in-mobile-app-development-4ei7)

截至 2018 年，谷歌的品牌价值超过 1200 亿美元。这意味着当谷歌做一些事情，任何事情，很多人都会密切关注。

2017 年，谷歌在颠覆移动应用开发行业并推出名为 Flutter 的新 UI 框架后，引起了世界的关注。该框架允许从单个代码库移植开发本地应用程序，并且已经用于开发许多应用程序。

关于 Flutter 最令人兴奋的部分可能不是框架本身。像 Bootstrap、Foundation、Milligram 这些流行的框架在接口和功能上都不缺乏。Flutter 的大肆宣传很可能来自一个事实:它是由谷歌管理的。

谷歌、DART 和移动应用

谷歌在移动应用开发方面有很强的背景。他们创造了许多成功的移动应用，如谷歌地图和谷歌钱包。但是他们为什么决定制造混乱呢？

自从谷歌的 Dart 语言在七年前首次出现以来，谷歌创建 UI 框架实际上应该不会让任何人感到惊讶。Dart 是 Flutter 开发中使用的语言，也是 Google Ads 和 Google Greentea 中使用的语言。

Flutter Vs Bootstrap、Foundation 和其他 UI 框架

现在你对 Flutter 有了更多的了解，是时候帮你决定是否值得一试了。有什么公司比一个有 20 多年历史的软件开发公司更能帮助你呢？

扑的裁决

在这么短的时间内，颤振已经积累了一个原因的压力。首先，谷歌使用 Dart 和 Flutter 创建自己的移动应用程序。需要注意的是，移动应用程序开发需要时间。可能会有更多的 Flutter 应用程序正在开发中，只是我们还不知道而已。

无论如何，人们应该注意到基金会非常强大的客户名单。基金会有一个很大的客户名单，上面的客户名单没有包括像国家地理，皮克斯和雅虎。

谷歌作为移动应用开发商的未来

很明显，随着谷歌广告、谷歌优化和现在的 Flutter 等工具的推出，谷歌希望从营销人员到开发人员的每个人都依赖它。但是谷歌的下一步是什么？他们所有的软件都会保持免费吗？目前，我们所能做的就是推测他们下一步会发展什么，并对时代做出反应。收购有可能吗？他们会和竞争对手亚马逊合作吗？
