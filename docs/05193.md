我通过恢复这个古老的 Rails 应用程序做出了贡献(最后一次提交是在 2012 年，当时我开始从事这个项目),在长达 7 年的时间里，我学会了所有那些 0 和 1，是的，在这个过程中我学到了很多。

以下是我软件开发生涯的前 10 天。

一点背景知识，这么多年来，它一直运行在 Rails 3.2 上，最后升级到了 v5.2.3。📣 📣

*   用于登录的 twitter API 已经随着时间的推移发生了变化，因此控制器和模型需要相应地修改。

*   所有的生产数据都需要根据当前的用户信息进行更新，例如，当前的个人资料图片和 Twitter 登录数据。

*   UI 需要一些修复，基本上是一些对齐问题。为了实现它，我学习了 CSS flex-box 并应用了所学的知识。

听起来很无聊，现在来点有趣的东西吧！！😆 😉 😃

这 10 天我都做到了。
