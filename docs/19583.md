# 带着成长的心态写博客

> 原文:[https://dev.to/emma/blogging-with-a-growth-mindset-37bf](https://dev.to/emma/blogging-with-a-growth-mindset-37bf)

## [](#is-there-a-point-in-blogging)写博客有意义吗？

我曾经几次想开一个个人编程博客，但是我总是很难找到这样做的理由。我担心是否有人会读它，我写的帖子是否有用，或者万一有人*读了它，他们会认为它很糟糕或者不值得他们花时间。*

即使现在谈论这些担忧，公开我的感受，也是我挣扎的事情。

最近我一直在狂读一些自助书籍，偶然发现了卡罗尔·德韦克关于固定和成长心态的书。总结一下:

> 一个心态**固定的人**认为自己的能力/智力从出生就固定了，无法改变，而一个心态**增长的人**认为自己可以通过努力提高。

当谈到博客(以及我生活中的其他领域！)我在自己身上看到了一种固定的思维模式——我看着其他人发布的所有令人惊叹的内容，却看不到自己能够做同样的事情。而且这种心态很难改变！

所以我想我现在能做的就是认识到我有这种心态，并坚持下去。我会写博客，即使我认为我的帖子很糟糕，我知道我写得越多，我改进得越多，也许有一天有人会发现我写的帖子很有用。
