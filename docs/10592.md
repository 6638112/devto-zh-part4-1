# 关于科普兰

> 原文:[https://dev.to/jo/about-codeland-2oge](https://dev.to/jo/about-codeland-2oge)

我是今年 CodelandConf 的一名演讲者，虽然在众多观众面前发表我的第一次演讲相当令人紧张，但我很高兴我做到了。

听演讲时，我有些害怕。作为一个演讲者，听众会认同我吗？或者他们会认为这个主题很无聊，或者更糟，认为我很无聊？为了准备我的演讲，我把自己录了下来，这样我就可以倾听和评论了。我在自己的演讲中睡着了，然后发疯似的醒来。我无聊地睡着了。我有点惊慌，然后意识到任何人都可能会对自己的生活故事感到厌倦，因为他们经历过。

# 为什么是科德兰？

我决定提交一份在 Codeland 演讲的提议，因为我认为我的故事值得讲述。作为一名受庇护者，这是为无声者发声的关键时刻。作为几个少数群体的一员，我知道我的故事会引起很多人的共鸣，并可能激励行动。我不会重复我的演讲，但是我会在演讲发布后更新这篇文章。

# 应用过程

我提交了我的演讲建议，基本上是我移民和技术之旅的时间表，以及它们如何相互影响。演讲被接受了，然后我和 Saron 一起组织演讲和幻灯片。这是最重要的一步，萨隆的评论非常透彻和直接，我很欣赏这一点。我做了很多笔记，并重读了大约 5 遍幻灯片，直到我对内容和组织感到满意。我会在工作时休息，在公园里散步，读着自己的谈话，可能看起来像个疯子。

# 科普兰的经历

科德兰的经历太棒了。我爱上了纽约。观众们很容易接受我的演讲，反应非常热烈。我肯定鼓励 Codeland 向新的演讲者介绍，因为你不会找到更容易接受和支持的听众。
