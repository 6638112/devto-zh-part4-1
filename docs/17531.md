# 我的第一个开源项目

> 原文:[https://dev.to/elcotu/my-first-open-source-project-24b9](https://dev.to/elcotu/my-first-open-source-project-24b9)

大家好。正如描述所述，我只是想来到这里，告诉各位开发者，我的第一个开源项目。为什么？因为即使我至少从 2005 年就开始开发软件，直到我加入这个网站，我才对开源项目感兴趣。

无论如何，动机很简单。在这一年里，我和我的姐妹们需要执行一系列的财务活动，以便处理一些我英语太差无法解释的事情。我指出，我们需要跟踪每一笔收入和结果，从主要的“现金资源”开始，以及我们中的一些人何时做出了必须退还的支出。
显然，主要的方法是创建一个共享的电子表格，但我想做一个软件，原因如下:

1.  我想用我选择的框架 CakePHP 从头开始做一个新的 web 应用程序，因为今年我主要是维护应用程序，这并不有趣。

2.  我还想根据会计原则记录收入和结果，这在电子表格中很难实现。

3.  然后，我想让这个小软件成为一个开源项目，以防世界上的任何人都可以从这项工作中受益

因此，我创建了“Easyaccounts”，这是一个需要定义账户(从会计学的角度)、现金资源和收入/结果概念的软件。该软件跟踪:

1.  **收入**:记录与现金资源相关的收入，以及现金资源中的货币概念

2.  **结果**:记录与现金资源相关联的结果，以及从现金资源中获得金钱的概念

3.  **转账**:记录资金从一种现金资源转移到另一种现金资源的情况

这三者中的每一个都会生成一个会计分录，其主要动机是保持我们管理的基金的最新余额。
