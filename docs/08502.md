云原生——是什么？粗略定义，它使用容器、微服务和通过 devops 管理的不可变基础设施在云中运行可扩展的应用程序。

[由云本地计算基金会(CNCF)定义](https://github.com/cncf/toc/blob/master/DEFINITION.md#cncf-cloud-native-definition-v10)

> 云原生技术使组织能够在现代动态环境(如公共云、私有云和混合云)中构建和运行可扩展的应用程序。容器、服务网格、微服务、不可变基础设施和声明式 API 就是这种方法的例子。这些技术使得松散耦合的系统具有弹性、可管理性和可观察性。与强大的自动化相结合，它们允许工程师以最少的劳动频繁地、可预测地做出高影响力的改变。
