# 用 python 构建低延迟的地理分布式应用和 API

> 原文：<https://dev.to/durgagokina/building-low-latency-geo-distributed-apps-apis-in-python-23o0>

当今的应用程序需要高度响应且始终在线。为了实现低延迟和高可用性，这些应用程序的实例需要部署在靠近其用户的多个区域。

但是，这些全球分布的应用程序需要一个地理分布的快速数据平台，该平台可以透明地复制世界上任何地方的数据，以使这些应用程序和 API 能够对靠近其用户的数据副本进行操作。同样，应用程序需要地理复制和本地流来处理来自 fast 数据平台的发布-订阅、ETL 和实时更新。
