# 第一份工作是软件工程师，没有学位...

> 原文:[https://dev . to/serializer/first-job-as-a-a-a-software-engineer-not-a-degree-55b 8](https://dev.to/serializator/first-job-as-a-software-engineer-without-a-degree-55b8)

嗨！正如我的个人资料上所说，我的名字是朱利安，19 岁，目前是我们提供的软件工程师。

**“软件工程师”、“全栈”、“Web”、“计算机科学家？!"**
“软件工程师”，老实说，这只是我使用的一个术语，因为我不认为“Web 开发人员”或“全栈软件工程师”(或这些的任何组合)就足够了。

^:我改变主意了，“全栈工程师”听起来更适合我最擅长的工作，web 开发，也许听起来更好听！🙊

我喜欢修补很多东西。我喜欢学习算法和数据结构，这并不会让我成为一个“计算机科学家”或任何东西...我喜欢用 Prometheus 的 Grafana 和集中式日志记录的 Logstash 进行监控，但这并不能让我成为 DevOps 工程师。

*ppssstt，如果你知道更好的职位描述，告诉我！*

我在哪里工作👀
自从我开始在[我们提供](https://www.weprovide.com)工作，到这个月已经一年多了，我学到了很多。绝对不仅仅是在技术方面，在社交和交流方面也是如此。

嗯，你的学位在哪里？！🙄
在我的脑子里，写这个感觉有点，自我中心？我的意思是，我想象你阅读这篇文章的方式是*“哇，看看他，以为他是唯一一个走这条路并取得成功的人！”*。紧接着，另一个想法出现了，如果我写这篇文章是为了更好地了解自己，并真正思考过去两年发生的事情，会怎么样？

*我想这就是了，这篇博文会让我蜷缩在角落里哭泣*😜

【2016 年底的某个地方🎆
我在*开始了我的教育。4*(MBO)，即软件工程。*“Niv。4 (MBO)"* 在维基百科上有比我可能给你的更好的解释[！](https://en.wikipedia.org/wiki/Education_in_the_Netherlands#MBO)

在我上学的整个时间里，我并不觉得我真的学到了什么特别新的东西。我主要是做作业和帮助其他学生。不过，我并不是说在那里没有价值。

事实上，我觉得我属于和我有相同兴趣的人。我不能说这对我有多大的帮助，因为我从高中毕业，不知道一年、两年、三年等等之后我会做什么。

我猜，从 2017 年年中到 2017 年底？🔥💩
实习...我得去找个地方实习...一份独立的实习，喜欢它，在现实世界中学习一些东西，而不是*“让我们用 C#建立一个待办事项列表！”*。令人不愉快的是，我过去和现在都不适应新环境(尤其是社交方面)，也不适应每天要和 8 个小时在一起的新朋友。吓人！

因此...我找到了实习的地方，显然前一天晚上有些紧张，实际上第一天一直非常紧张和焦虑。*“只是要习惯而已”*，我想，没什么奇怪的，每个人第一天都会紧张，不管是有一点还是像我一样。

在连续几天汗流浃背(这也不是一年中最冷的日子)，每天早上发抖和呕吐之后，我决定它不适合我。因为我当时的处境，我开始思考，为什么？我需要这么做吗？是的，如果我想要我的大学学位的话！

长话短说，我在实习的时候和导师还有 HR 坐在一起，聊了聊发生的事情。我们还谈到了我在哪里看到了自己...年，或任何其他合理的时间框架。在我说出我想做的事情和我想去的地方后，HR 说了一些我不太记得的话，但基本上可以翻译成:

我有...，所以我做不到...有时你不得不接受你现在的处境。

之后我们聊了一会儿，得出结论说实习不适合我，然后就离开了。我和学校里想帮助我的几个人谈过，但最终归结为一个事实，我没有想到自己会经历一个整天发抖的实习，只是为了 4 年后的一张纸。

【2018 年年中的某个地方，我决定去求职！📄
我终于摆脱了懒惰，打开了谷歌(显然，没有摆脱我的屁股...)并寻找*“开发者工作”*。那是我第一次尝试，惨败！我很快将范围缩小到*“web developer”*，并找到了几个。

我最终做的是和一个招聘人员建立了(不知不觉的)长期关系。我用这个策略参加了一次工作面试，我想，不...我真的需要自己去寻找一些东西，我知道我想要什么，当我看到它时我就知道了，而不是通过接到一些潜在雇主的电话，他们需要具有“我的技能组合”的人。

我仍然会接到电话:“你的工作怎么样，在找工作吗？!"😢

**我们提供**的工作面试😄
我最终找到了 We Provide，一家正在寻找后端 PHP 开发人员的网络机构，但没有类似于*“HTML，PHP，CSS，MySQL，.”的工作描述。NET，MongoDB，Redis，Docker，AWS，Micro-Services"* (以及所有其他你可以放进去的术语)。它只是看起来像一个有趣的工作场所，一个我可以不断学习和提高自己的地方。

我申请了，很快就得到了回复，并安排了一次工作面试。我记得最清楚的是，他们认为我有学位，因为我在申请表上写了“申请入学资格”。4 (MBO)”在我的工作申请上。我忘了他们大概不会看着年份看着“2016-2017”想“他完成那个了吗？”。

但问题是，我不记得具体是因为它。我记得是因为我们谈这件事的时候，并不是像*“从这里走下坡路，他没有学位，对不起，有门！”他们被激起了兴趣？我想不出更好的词来形容它。*

我现在在哪里...❤️
现在，我不认为*“唉，周末结束了，又是一天在学校/工作！”*，我很高兴。我和一起工作的人在一起很开心，最重要的是，我学到了很多东西。

我想澄清一下，我不认为大家都应该认为“我不需要学位！”，如果你能得到一个，就去做。看情况吧。

我最后想说的是，当有人没有学位时，不要以为他们不想要。不要假设他们懒惰或者没有动力。我认为当你做你喜欢的事情，在业余时间做，并因为它的乐趣而学习，你可以做这么多。
