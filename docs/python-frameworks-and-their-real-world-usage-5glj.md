# Python 框架及其在现实世界中的使用

> 原文：<https://dev.to/shahrozkhan69/python-frameworks-and-their-real-world-usage-5glj>

为什么您应该选择 Python？Python 是一种语言，在 Google 的神奇触摸之后，它在很短的时间内在业界获得了惊人的流行。通过空载燕子项目，这项技术的速度提高了大约两到三倍。因此，对于需要语言核心的高性能和高速度的大型软件应用程序来说，它通常是首选。在 21 世纪，我们不能忽视 web 领域的重要性，幸运的是 Python 在开发企业标准 web 解决方案方面非常灵活。使用 Python 可以实现要求更高速度和能力的 web 应用程序。所有类型的数据驱动的 web 应用程序都可以用 Python 开发，并具有最大的能力和潜力。事实上，Python 在 web 开发人员中非常受欢迎，因为它具有无与伦比的质量和性能，并且很少有高效的 Python 框架。让我们来讨论一下业界的顶级 Python 框架。

DJango 框架:这可能是业界最著名的 Python web 框架。Django 非常强大，由 Jason Sole 和 Jason Mc Laugilin 开发。它首次在一个工作门户中实现，以确保其效率。后来，它面向所有人发布，并在业界获得了巨大的反响。这是业界最大的基于 Python 的 web 框架，因此为 Python 开发者提供了额外的能力和特性。这个框架有一个庞大的支持社区，他们全天候工作，为其他人提供支持。它拥有最强大的管理界面，可以对任何大型 web 应用程序进行顶级控制。实际上，这个框架在开发在线论坛、门户网站和其他社交网站时非常有用。它在以最大的效率和最小的努力开发快速 web 解决方案方面也非常受欢迎。

Flask: Flask 是一个用 Python 开发的微型框架，有着奇怪的背景。这是印度一位聪明的 Python 开发者的愚人节惊喜。帕拉德普·高达先生要求他的同事用 Python 开发一个单文件微框架，他成功了，并作为愚人节惊喜送给了他的朋友。这没有 Django 那么强大，但是它可以用于中等规模的 Python web 应用程序开发项目。最适合想学习 Python，在更短的时间跨度内开始编码的初学者。这在 Python 开发人员中产生了对试验框架的贪得无厌的渴望。它经常被用于低预算和有限时间框架内的小型网站开发。因此，它从一开始就获得了业界的好评。

Pyramid: Pyramid 是另一个很棒的 Python 开发框架，它具有额外的灵活性，可以服务于广泛的 web 应用程序。它也是由 git hub 驱动的，因此不需要担心支持。它是 pylons 1.0 和 restes . bfg 的结合。由于其灵活的特性，它在社区中以更快的速度增长。实际上，它有助于开发企业标准 API Python 项目和创建基于 Python 的 CMS 或 KMS。
