# 作为一名自由网页开发者，你是如何应对学习这么多技术的？

> 原文:[https://dev . to/prahladyeri/as-a-freelance-web-developer-how-do-up-to-learning-so-many-technologies-1562](https://dev.to/prahladyeri/as-a-freelance-web-developer-how-do-you-cope-up-with-learning-so-many-technologies-1562)

众所周知,`webdev`世界目前在所有领域都充满了错综复杂的事物，比如框架、库、标准、编码语言、标记语言、构建系统、文档系统、容器和虚拟化系统、数据库管理、机器学习和统计等等。

现在，一个大公司或智囊团可以在所有这些领域雇佣数百名聪明的技术专家，但是一个独立的自由职业者就没有这种奢侈了，对吗？

作为一名自由职业者，你是如何应对这一切的？为了在今天激烈的自由职业竞争中具有竞争力，你必须至少对所有这些领域有一点点了解。

随着这些复杂性在未来几十年变得更加微妙，你认为自由编码最终会成为一个死亡领域吗？一个人能学到或掌握的东西是有限的，毕竟，你不可能跟上任何地方发生的一切。在这方面，你认为自由编程的未来会怎样？
