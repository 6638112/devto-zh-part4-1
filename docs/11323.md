# 我很幸运，你不是。

> 原文：<https://dev.to/jeromegamez/i-am-lucky-you-are-not-2eco>

*[图片](https://unsplash.com/photos/8UG90AYPDW4)作者[马特·科拉莫](https://unsplash.com/@breakyourboundaries4)，[中文翻译](https://sh.alynx.moe/posts/I-Am-Lucky-You-Are-Not/)作者 [@alynxzhou](https://dev.to/alynxzhou)*

嗨，我叫杰罗姆！我乐于分享我的名字，我住在德国，我并不总是对我的政府感到满意。

我很幸运，我可以公开这样说，而不用担心反响，我也很幸运*甚至不用考虑*我有多幸运(这是有意复制的)。

这是我最近在 [Firebase Admin SDK for PHP](https://github.com/kreait/firebase-php) 的 [Discord 社区](https://discord.gg/nbgVfty)中的一次对话中意识到的。

其中一个成员试图用 SDK 访问 Firebase API，这触发了一个异常，但没有关于错误的详细信息。结果是，他们位于中国，忘记了在代码运行的环境中启用代理。可解决的问题。

当我在 SDK 上实现对连接错误的处理时(这是我以前从未想过要做的事情，因为我很幸运在一个不受限制的国家拥有无处不在的互联网接入)，另一个社区成员插话说:

> [@jeromegamez](https://dev.to/jeromegamez) :看我有多高兴:由于美国贸易控制法的限制，你的 GitHub 账户被限制了。对于个人帐户，您可以有限地访问免费的 GitHub 公共存储库服务，仅用于个人通信。请阅读关于 GitHub 和贸易控制的更多信息。如果您认为您的帐户被错误标记，请提出上诉。

他们曾经在自己的国家没有使用 VPN/代理访问过 GitHub，立即被标记，现在无法访问他们自己的私有库。

> 那太糟糕了...你会对这个决定提出上诉吗？

这是我对这条消息的天真回答，没有意识到他们当然不会上诉成功。

> [@jeromegamez](https://dev.to/jeromegamez) :不幸的是你不能选择在哪里出生！在近 200 个国家中，我是最疯狂的国家之一。我通常使用 vpn，因为有制裁和内部审查，例如 whatsapp，有时我不能上传图片，我不得不使用代理/VPN。通常情况下，你可以通过 VPN 使用国际服务，如 Nvidia.com/Android.developers.com 等，但这一次是不同的，因为一旦你使用特定的 IP 地址访问他们的服务，他们就会阻止你。至少我仍然可以使用公共存储库，并且希望能够备份我的私有存储库！关于上诉的决定，作为这些国家的公民，我不能这样做。

我们在一个私人频道继续我们的谈话，在那里他们告诉我他们的原籍国(这是他们经常感到不舒服的事情)，他们的斗争，他们在公共频道分享这些以提高意识。

像我这样的人(尤其是像我这样的人:白人，男性，欧洲人，有一份高薪工作，可以获得我需要的任何东西和我最想要的东西)在一个相当舒适的地方看着这个世界，对世界上正在发生的事情感到不满或害怕。

但是还有更多不舒服、充满敌意和危险的地方，像你我这样的人就住在那里。

当我在当前的热浪中挣扎时，另一个人正在为生计而挣扎。

虽然我很生气我不能在互联网平台上使用我喜欢的用户名，但其他人必须确保他们的用户名不会透露太多关于他们的信息。

当 A 国政府对 B 国实施制裁时，该国的个人不得不与后果作斗争，而不是他们的领导人。

当我努力加薪的时候，另一个人却在努力生存。

这并不意味着我们的个人问题无关紧要，但我认为重要的是要意识到我们中的许多人很幸运出生在一个好地方，但我们中的许多人并不那么幸运。

我们至少可以做到的是以尊重、体面和同情的态度对待他人，不管他们来自哪里，也不管他们的背景如何。
