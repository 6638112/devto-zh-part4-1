### [](#what-happened-to-domore)多莫尔怎么了？

doMore 是我的小任务清单项目，我将在下周左右返回，然而，我对我如何处理它感觉不好，所以我现在放弃了它。我想以一种更有效的方式来解决它，这样它会回来得更快而不是更晚，但是是以一种更好的方式。

所以我所做的，我很高兴的是创造了 [ezToggle](https://eztoggle.netlify.com/) ！

[![](../Images/428e481223b4cb66bb743f28d42fb927.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--TTvHGKNn--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/wx7dpv36w38bweoqjdn3.jpg) 
*PS 你还是应该点击上面的链接，因为你可以看到一个甜蜜的过渡*

ezToggle 是一个超级简单的主题转换器，你只需要使用 HTML、CSS 和 JavaScript 就可以在网站中实现。这只花了我几个小时来完成，并利用 CSS 变量，所以你可以很容易地注入你选择的颜色的黑暗和光明的主题，没有太多的麻烦。

如果你想查看或使用，这个项目在我的 GitHub 上:)
