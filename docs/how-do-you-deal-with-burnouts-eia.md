# 你如何处理倦怠？

> 原文：<https://dev.to/oyetoket/how-do-you-deal-with-burnouts-eia>

倦怠是真实的，几乎每个人都经历过，不管他们的职业是什么。

这发生在我身上，让我失去了我的兼职，几乎让我失去了我的工作。

由于我的工作地点很远，我可以离开工作近 3 周来处理我的问题，这让我很容易摆脱它。

我看电影，和同事郊游，勉强做单口相声，化妆单口相声，勉强交流，编借口等等。

我感到沮丧和疲惫，好像我应该放弃，面对我的学业，过正常的生活。

第三周，我本来要和我的经理和一个客户开会。然后嘣我忘了，没出现。嗯，问题是，在那两个星期里，我错过了大部分的会议，我没有去查看我的日历，因为我已经把所有关于工作的事情都静音了。

长话短说，在第三周的最后一天，我接到了一个电话，我只有一次机会来调整自己，或者去工作。

嗯，我恢复了理智，去见我的导师，向他解释，他给了我一些建议。

然后，我推倒了所有让我沮丧的事情，集中精力继续工作。

告诉我们你是如何处理的。
