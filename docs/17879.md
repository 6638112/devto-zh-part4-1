# 如何追踪 IP 地址

> 原文:[https://dev . to/authority _ SEO/how-to-trace-a-IP-address-401 I](https://dev.to/authority_seo/how-to-trace-an-ip-address-401i)

IP 地址是互联网上使用的寻址系统。*Soft 博士*提到 IP 地址与物理地址相同，知道一个人的 IP 地址的人可以[始终如一地定位](https://www.drsoft.com/2019/01/15/how-do-ip-addresses-work-ip-addresses-clearly-explained/)该地址来自的地理区域。追踪 IP 地址通常意味着要经过几道关卡才能到达最终的发源点。

然而，有些 IP 地址你无法在线追踪。使用 VPN 的用户(例如像 [Opera VPN](https://vpn-review.com/opera-vpn/) )无论如何都是无法追踪的。类似地，使用 TOR 浏览器的用户也无法追溯到他们的连接来自哪里，因为链接在出现在另一端之前要经过大量的 IP 地址。追踪一个 IP 并不需要世界级的黑客，大多数有基本能力的用户可以尝试追踪网站的 IP，只是为了看看有人如何执行追踪。

为什么要追踪 IP 地址？

互联网使得匿名骚扰某人变得相对容易。在不久前，追踪电子邮件的来源是一件简单的事情，但现代系统让那些想要找到通信另一端的人变得更加困难。此外，如果您在某个特定的地理区域注意到一些应该向警方报告的事情，IP 跟踪可能会很有用。互联网服务提供商通常不会泄露客户的信息，而一个人通常需要传票才能从他们那里获取这些信息。可悲的是，这可能需要很长时间，而仅仅追踪 IP 要容易得多，因为在许多犯罪案件中，时间是至关重要的。

**IP 追踪中的潜在错误**

如前所述，通过 TOR 网络路由连接的用户往往很难被跟踪，因为他们广播的最后一个 IP 地址与他们的物理位置完全无关。虽然很容易找到 IP 的物理位置，但他们的实际站点可能在完全不同的城市甚至洲。任何足够熟练的用户都可以通过代理路由他们的连接，以隐藏他们的地理位置。 *Varonis* 将[代理服务器](https://www.varonis.com/blog/what-is-a-proxy-server/)定义为用户和互联网之间的网关，提供不同级别的安全和隐私。

如果用户使用虚拟专用网(VPN)服务，IP 跟踪也是不可能的。诺顿提到虚拟专用网的 T2 会屏蔽用户电脑的 IP 地址。对于地理跟踪，我们需要始发机器的 IP 地址，否则我们将无法找到信号是从哪里发出的。这两种情况都可能给出错误的报告或者根本没有结果。

**追踪一个 IP**

我们需要定位 IP 的第一件事是 IP 本身。DNS 服务器通常用于将输入的网站转换为 IP 地址，因此如果我们试图跟踪一个网站，我们可以获得网站名称并在该网站上进行查找。在 Windows 设备上，这个过程很简单:

*   通过按 Windows 键和 R 打开命令行提示符，然后键入不带引号的“cmd”
*   在弹出的屏幕中，我们将使用 ping 命令来获取网站的 IP。我们将输入不带引号的“ping < website_name >，必要时我们会替换网站名称
*   得到的读数显示了一个 IP 地址，以及我们发送的数据包到达那里需要多长时间。该 IP 地址就是有问题的站点的地址。

如果我们想再次确认这确实是我们正在处理的网站，我们可以复制 IP 地址并粘贴到浏览器的地址栏中。如果加载的网站是我们正在处理的，我们已经成功地找到了我们的 IP 地址。

现在我们可以更进一步，找到 IP 地址的来源。为此，我们可以利用地理定位服务来追踪 IP 地址的来源。谷歌一个 IP 定位查找器会给出一些结果，我们可以把我们的 IP 插入其中。该网站将会给我们一个大概的地址。

**这对您意味着什么**

如果你能如此容易地追踪到一个网站的 IP 地址，想象一下，对于一个获得你的 IP 地址的人来说，找到你的位置有多容易？隐私和安全要求意识到威胁。当您连接到网站时，您的电脑会通过公共网络将您的不安全 IP 广播到您正在访问的网站。任何人都可以截获这个 IP 地址，然后追踪它，找出你在哪里，并可能利用它来对付你。网上浏览要注意防范。把你的安全放在首位。
