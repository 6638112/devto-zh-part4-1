# Zenika 的开源

> 原文:[https://dev.to/zenika/open-source-at-zenika-2l5i](https://dev.to/zenika/open-source-at-zenika-2l5i)

几周前，我们宣布在 [Zenika](https://www.zenika.com/) 推出关于我们开源文化的全新[网站](https://oss.zenika.com)。它旨在促进我们的项目和我们同事的项目。

以下是其中的一些:

*   immutadot 是一个 JavaScript 库，用于处理嵌套的不可变结构。
*   [bento-starter](https://github.com/kefranabg/bento-starter) 是利用 Vue.js 和 Firebase 快速构建 PWA 应用的全栈解决方案。
*   修订版是一个起草可修订文本的民主平台。

你可以在我们的 [GitHub 的组织](https://github.com/zenika-open-source)或者我们的[网站](https://oss.zenika.com)上找到更多的项目。

我们还写了一篇关于如何开始使用开源软件的文章。这是关于开源世界的六篇文章中的第一篇。

如果你想了解更多关于我们项目的新闻或未来的文章，别忘了关注我们的 twitter [@ZenikaOSS](https://twitter.com/ZenikaOSS) ！
