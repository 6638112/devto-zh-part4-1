# 为什么我们的世界不是更容易接近？

> 原文:[https://dev . to/ashleemboyer/why-not-our-world-more-accessible-4o fl](https://dev.to/ashleemboyer/why-isn-t-our-world-more-accessible-4ofl)

下次你走在城市的街道上，四处看看。

人行道是什么样的？如果它是用砖做的，它看起来是非常光滑还是有一半是松散和不平整的？当你来到一个十字路口时，通往人行横道的匝道是否顺畅地与街道相遇？人行道之间的街道呢？路上有坑坑洼洼还是铺得很好？

如果有商店和餐馆，它们是在一层还是有一两步或更多的台阶才能进去？有启动门的按钮吗？当你在座位区时，有没有需要走几步才能到达的隔间或额外的桌子？如果餐厅有一个露台，在环绕该区域的栏杆外面有多少空间？它是否足够大，可以让轮椅朝一个方向移动，而让一个人朝另一个方向行走？

这是过去几天我脑子里的一些想法。事实上，我们的一些人行道和人行横道的质量令人震惊。这些只是大多数人可以看到的问题，如果他们停下来看看的话。

你能想象存在哪些不可见的可访问性问题吗？
