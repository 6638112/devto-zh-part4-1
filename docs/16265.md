# 我从来没有过分相信 switch 语句比‘if else if else if else if else’更简洁

> 原文:[https://dev . to/Ben/I-ve-never-been-been-been-over-been-confident-than-clean-than-if-else-if-else-if-else-if-else-81l](https://dev.to/ben/i-ve-never-become-overly-convinced-that-switch-statements-are-that-much-cleaner-than-if-else-if-else-if-else-if-else-81l)

我多少知道，当将许多`else if`语句链接在一起时，switch 语句通常被认为是一个好主意。我并不反对这种重构，但我也从未发现它过于重要。

我倾向于发现与 switch 语句相比，`if`和`else`的行为和语法是相当一致的，不管是什么语言，所以达到`if`更容易，但是即使当被链接时，看起来也没有 switch 的*那么*干净。

我绝对同意去掉`if`..`else`..`else`链值得被重构掉，但是我没有发现 switch 语句实际上有意义地提高了质量。

想法？
