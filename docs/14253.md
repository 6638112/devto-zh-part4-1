# 找新工作时，对你来说最重要的因素是什么？

> 原文：<https://dev.to/wools11/job-factors-ai3>

作为一名软件开发人员，一份工作有三个核心因素(这是我能想到的)。

1)我将从事什么产品？它是公众眼中的高调/突破性的东西，是帮助动物/人类福利的东西，还是没人能看到的东西，还是一个只是转移产品/酒店/假期的零售网站？

2)假设它是正确的程序。语言，我将使用哪些其他技术，如何使用？我会使用人工智能、人工智能、无服务器等来构建新颖的解决方案，还是仅仅做 bau 来“让它变得更好”

3)一般工作条件是什么，即工作地点/偏远地区、薪酬、团队、文化、老板、晋升、培训等

当你选择下一份工作时，你会优先考虑这三个因素，你对每个因素的重视程度是什么？第三个因素很关键，假设第一个因素是钱，我不在乎第二个因素...？

产品真的重要吗，或者如果原因是好的，谁会关心技术呢？但是如果你不喜欢这个产品，它就全完了，但是在这种情况下，你的可接受性的界限是什么呢？？？
