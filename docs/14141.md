# 编码的兽医:什么，哪里，如何帮助

> 原文:[https://dev . to/vetswhocode/vets-who-code-what-where-and-how-to-help-4 bco](https://dev.to/vetswhocode/vets-who-code-what-where-and-how-to-help-4bco)

六年前，当我开始这个旅程时，我只是想为退伍军人创造一个负担得起的解决方案，让他们尽可能快地过渡到劳动力市场，而没有平民生活中试图找到一所接受 GI 法案的代码学校的障碍。我不希望我的过渡部队同伴像我一样被遗漏。

这就是我最初进入非营利部门的原因。我想帮助我的兄弟姐妹。我所发现的是一个系统的设计，不是为了帮助他们，而是通过讲述国家如何辜负我们的军人的故事，从捐赠者、赞助者和政府机构那里获得尽可能多的钱；同时创造了极高的需求，然而却有着极低的成功基准。你知道吗，在一些州，除非你真的露宿街头，否则你不会被认为是无家可归者。也就是说，如果你住在车里，住在酒店房间里，或者睡沙发，你就没有资格享受某些支持服务。

我应该知道——我非常不喜欢做这三件事。尽管身处逆境，但毅力让我爬出了黑暗，内心深处有一种让其他老兵不坠入深渊的渴望。回想起来，那是我人生中最难熬的六个月！是开发和设计帮助我度过了那些艰难的时光，而不是现在的非营利部门，那里充满了在做最少工作的同时保持舒适关系的遗迹。

我把醒着的每一分钟都用来学习其他人要么太顽固要么太害怕学习的技能，这给我增加了价值——平民阶层可以理解的价值。然后，当一个阵亡士兵的家人需要帮助时，我新树立的价值观使我有能力响应号召去埋葬一个兄弟。事后，我想“接下来我该做什么？”—产生了把旧的东西(一个怪异的城堡)变旧的愚蠢想法！)转化为新的东西，重新利用它来重新培训退伍军人的真实和数字技能。不幸的是，这不起作用。但是，从中产生了真正的灵感，一些改进过的、辉煌的、实用的东西——我希望甚至能让大卫·海涅迈尔·汉森(我的英雄)感到骄傲。

什么是兽医代码？

Vets Who Code 是一家由退伍军人创建、领导和运营的 501(c)(3)分布式慈善非营利组织，致力于通过软件开发培训和教育来填补过渡退伍军人和现役军人配偶所需和可用的技术专业知识之间的巨大鸿沟。我们利用我们的分布式模型来训练任何地方的退伍军人，但我们不仅仅是这样。我们是一个拥有混合技术技能和精益思想的团队，专注于帮助退伍军人不仅学习如何编程，还在该领域获得工作，并在没有其他非营利组织不必要的臃肿的情况下完成工作。从 UX 和德沃普斯的工程师到小小的我，我投身其中是因为我只是想帮助我的退伍军人同伴，这是这个非营利组织最重要的部分。目前，80%的团队成员都是退伍军人，我们为您提供超越我们的成功和从我们的错误中学习所需的一切。因为如果我们要庆祝这个叫做生活的东西，我们需要一起去。

**为什么编码？**

问得好。现在我们的国家有一个问题:我们有编程工作，但没有足够的程序员。不仅如此，我们没有足够多的优秀程序员。最重要的是，虽然情况确实有所好转，但退伍军人仍然是美国失业率最高的人群。所以我们有了这个疯狂的想法。如果我们创建一个围绕现代技术设计的经过审核的课程会怎么样？不只是任何退伍军人，这些技能可以真正影响。不仅如此，让我们孜孜不倦地工作，在代码、业务和设计的专业标准方面指导他们。你能想象当说话的人在过去的 15 年里每天都在思考这个问题的时候，你在设计思维方面进步了多少吗？然后我们想，如果我们在你和我们打交道 15 周后增加工具来继续学习会怎么样？我们的合作伙伴，如 Frontend Masters 和 Google Cloud，为我们提供了令人惊叹的工具来为退伍军人提供礼物，以便他们可以学习和准备长达一年的时间。不仅如此——我们做介绍，并在面试过程中培训他们，帮助他们在这个行业找到工作。你猜怎么着它工作了。我们的部队得到了高薪职位，我们非常高兴。在这里，我们让了不起的人拥有他们应得的生活方式，而且这一切都是在网上完成的；这在 10 年前是不可能的。

你能提供什么帮助？

这实际上是我们经常遇到的问题。我们的退伍军人和我们的团队一起花了 15 周的时间专注于 JavaScript 语言及其包含的所有内容，并理解软件架构和计算机科学基础。我们专注于工作流程、面试问题、计算机科学和 UX 研究。

你能做的支持我们使命的最重要的事情是 **[捐赠](https://donorbox.org/vetswhocode-donation)** 。作为一个 501(c)(3)我们是免税的，作为一个分布式非营利组织，我们没有任何浪费的运营成本。所有收益直接用于资源和军队的酷事。

如果你是一名至少有两年 ES6 和 Javascript 工作经验的软件工程师，注册成为一名导师。如果你是一个会编码的设计师，我们总是在找人帮助我们改善用户体验。如果你是人力资源部的，请在我们的联系表格中给我们留言，这样我们就可以让我们的队伍进入你的 javascript 工作渠道。
