# 我有钩子...

> 原文：<https://dev.to/oieeaaaa/i-got-hook-1bl5>

变成“钩子”

为什么？

我发现钩子简单易用，在您最终接触 React 并学习它之前，无需预先了解构造函数、绑定和其他 javascript 概念！

有了“钩子”，你只需简单地了解一下*煎饼* (HTML、CSS、JS)的基本知识，就可以一头扎进 React 本身。如果年长的自己问我“哪个是哪个”的问题，这是我更愿意向他学习的。

如果你是 React 和阅读这篇文章的初学者。
记住这一点:

我羡慕你。😈
开个玩笑...

你知道为什么吗？

因为“钩子”让您不必在 javascript 中剥离洋葱概念的层次。

所以祝你好运！如果您刚刚开始使用 React，那么这里的“挂钩”是一个很好的起点。

我还建议您学习更多的 Javascript 概念，而不仅仅是 React。:)
