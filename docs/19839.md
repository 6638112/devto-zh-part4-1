# 有人知道任何*离散*聚类算法吗？

> 原文：<https://dev.to/awwsmm/does-anyone-know-of-any-discrete-clustering-algorithms-1pno>

我正在研究聚类算法，我似乎找到的所有资料都在讨论连续数据的 2D(或更高维)聚类。我找到的最接近我正在寻找的东西是[这篇文章](http://paulovinicius.com/uploads/3/5/3/1/35316805/2015_iros_rob.pdf)，它讨论了*离散-连续*聚类(其中`x`和`y`轴被量化为单元，但是`z`轴允许连续变化)。

有人遇到过对纯离散数据进行聚类分析的算法吗？特别是 2D？
