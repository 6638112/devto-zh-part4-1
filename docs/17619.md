# 绝对初学者的 JavaScript 基础

> 原文:[https://dev . to/learntocodez/JavaScript-fundamentals-for absolute-初学者-267j](https://dev.to/learntocodez/javascript-fundamentals-for-absolute-beginners-267j)

快速简单地动手学习 JavaScript

本 JavaScript 实践培训课程提供了使用 JavaScript 设计和开发动态网页所需的基础知识。它向学生介绍了客户端 JavaScript，以及如何使用该语言将静态 HTML 页面转换为动态的交互式网页。

JavaScript 主题包括向网页添加 JavaScript、使用对话框与用户交互以及 JavaScript 固有对象，如日期、数组和数学。还将向学生介绍文档对象模型(DOM)、函数创建、事件处理、表单验证、对象创建等。

你会学到什么

*   创建 JavaScript 程序
*   用户变量和运算符
*   启动弹出框
*   写条件和循环逻辑
*   创建自定义函数
*   以编程方式处理事件
*   捕捉和处理错误
*   学习调试的基础知识
*   从 JavaScript 操纵 HTML
*   使用浏览器对象模型
*   进行 HTML 表单验证
*   创建自定义 JavaScript 对象
