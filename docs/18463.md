# JavaScript 学习者的 50 个微编码挑战

> 原文:[https://dev . to/codeguppy/50-微编码-面向 javascript 学习者的挑战-5ace](https://dev.to/codeguppy/50-micro-coding-challenges-for-javascript-learners-5ace)

这是一个 50 个微编码挑战的列表，面向刚刚学习 JavaScript 语言基础的初学者。解决它们以提高或测试您的编码技能和语言理解能力。

虽然这些挑战是为 codeguppy.com 的学习者设计的，但是这些挑战是平台不可知的，所以任何 JavaScript 初学者都可以从中获得乐趣。

### [](#what-coding-environment-to-use)使用什么编码环境？

我们建议使用一个在线 JavaScript 编码环境，如果你愿意的话，它允许你与他人共享你的代码。codeguppy.com 是一个选择，但是任何其他在线 JavaScript 平台都可以。

### [](#coding-challenges)编码挑战

1.  打印从 1 到 10 的数字。注意:如果你使用的是 codeguppy.com，可以使用 println()函数直接在屏幕上打印，否则使用 console.log()在浏览器控制台中输出。

2.  打印小于 100 的奇数

3.  用 7 打印乘法表

4.  打印所有从 1 到 10 的乘法表

5.  计算从 1 到 10 的数字之和

6.  算 10！

7.  计算大于 10 小于 30 的偶数之和

8.  计算数字数组中数字的总和

9.  计算数字数组中数字的平均值

10.  创建一个函数，该函数接收一个数字数组作为参数，并返回一个只包含正数的数组

11.  在一组数字中找出最大值

12.  打印前 10 个斐波纳契数，不使用递归

13.  创建一个函数，使用递归找到第 n 个斐波那契数

14.  创建一个函数，该函数将返回一个布尔值，该值指定一个数是否为质数

15.  计算正整数的位数之和

16.  打印前 100 个质数

17.  创建一个函数，返回数组中前 p 个大于 n 的质数

18.  将数组向左旋转 1 个位置

19.  将数组向右旋转 1 个位置

20.  创建一个函数来反转一个数组并将结果作为一个新数组返回

21.  创建一个函数来合并两个数组，并将结果作为一个新数组返回

22.  创建一个函数，该函数将接收两个数字数组作为参数，并返回一个数组，该数组包含第一个数组或第二个数组中的所有数字，但不同时包含这两个数组

23.  创建一个函数，它将接收两个数组，并返回一个数组，其中的元素在第一个数组中，但不在第二个数组中

24.  创建一个函数，该函数将接收一个数字数组作为参数，并将返回一个包含不同元素的新数组

25.  计算前 100 个素数的和，并以数组形式返回它们

26.  打印前 100 个质数之间的距离

27.  创建一个将两个大小不定的正数相加的函数。数字以字符串形式接收，结果也应该以字符串形式提供。

28.  创建一个函数，返回文本中的字数

29.  创建一个将文本中每个单词的首字母大写的函数

30.  计算以逗号分隔的字符串中收到的数字的总和

31.  创建一个函数，返回文本中每个单词出现的次数。返回的将是一个包含{word，count}内对象的数组

32.  创建一个函数将 CSV 文本转换为“二维”数组

33.  创建一个将字符串转换为字符数组的函数

34.  创建一个函数来转换包含每个字符的 ASCII 码的数组中的字符串

35.  创建一个函数来转换一个包含 ASCII 码的字符串数组

36.  将 Caesar 密码实现为一个单独的函数 encryptDecrypt(text，key)；

37.  对数字数组实现冒泡排序算法

38.  创建一个函数来计算由 x，y 坐标定义的两点之间的距离

39.  创建一个函数，该函数将返回一个布尔值，该值指示由中心坐标和半径定义的两个圆是否相交

40.  创建一个函数，该函数将接收一个二维数组作为参数和一个数字，并将提取由该数字指定的列作为一维数组

41.  创建一个将包含二进制数的字符串转换成数字的函数

42.  创建一个函数来计算一个交错数组中所有数字的总和(包含数字或其他数字数组，级数不限)

43.  在一组交错的数字或一组数字中找出最大值

44.  将一个带有数字或其他数组的交错数组深度复制到一个新数组中

45.  创建一个函数来返回字符串中最长的单词

46.  打乱字符串数组

47.  创建一个函数，它将接收 n 作为参数，并返回一个由从 1 到 n 的 n 个随机数组成的数组。这些数字在数组中应该是唯一的。

48.  找出字符串中字母的出现频率。以数组的数组形式返回结果。每个子数组有两个元素:字母和出现次数。

49.  高精度计算斐波那契(500)(全数字)

50.  算 70！高精度(所有数字)
