## [](#enable-the-windows-subsystem-for-linux-wsl)为 Linux 启用 Windows 子系统(WSL)

*   点击“Windows”键，开始输入“Windows 功能”
*   当弹出“打开或关闭 Windows 功能”时，按 enter 键(也可以从控制面板>程序中找到)
*   在“Windows 功能”中搜索“用于 Linux 的 Windows 子系统”设置
*   选中该框。应该会出现另一个弹出窗口，告诉您重新启动计算机

## [](#install-the-ubuntu-terminal)安装 Ubuntu 终端

*   点击“Windows”键，开始输入“Ubuntu”。应该会打开 Microsoft Store 应用程序。
*   按下“安装”按钮。可能会出现一个弹出窗口，要求您登录您的 Microsoft 帐户，您可以直接退出。

[![](../Images/23ffa1e7edd4df5679077aabd66be6bb.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--epXSieRf--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/623s7xuqn9xvhbx8tumz.gif)

下载完成后，你可以退出微软商店，点击“Windows”键并输入“Ubuntu”来启动 Ubuntu 应用程序。

终端应该出现并开始安装。

[![](../Images/9dd9593cb8a3fa40167bf96aaa32e8d7.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--sEkC_FBx--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/pg7629mb6qdbvh7vgl5f.gif)

安装完成后，它会要求您输入正在创建的新用户帐户的用户名和密码。

帐户创建完成后，您应该会看到终端提示您输入命令:`username@computername:~$`

[![](../Images/47c7e918bc880c4223ec97e4c315d6b9.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--enM7yuzP--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/nn09iubqlukdt0jbw3y7.gif)

请注意，您键入的密码不会显示在终端中。

您已经成功设置了 WSL！🎉
