# 内向者充分利用聚会的指南

> 原文:[https://dev . to/piyu Kore 06/an-innert-s-guide-to-making-the-most-the-meetups-2k 84](https://dev.to/piyukore06/an-introvert-s-guide-to-making-the-most-of-meetups-2k84)

聚会是一个认识志同道合的人，分享想法和创意的好地方。但是说实话，在聚会上做一个内向的人并不好玩。你知道你想学习，但你总是在想，你是如何对任何对话都没有贡献的。想想这整件事有多可怕。我来自外国，住在德国。用一种你几乎不懂的语言进行对话就更难了。

我只想澄清一点，我不是主题专家。我参加了很多聚会，亲身经历了一些与人交谈的问题。这是我的一种经历，也是我如何努力克服这种恐惧。

没事的

*   如果你紧张
*   如果你不认识任何人
*   如果你只是想离开，因为你觉得没有人会想念你，如果你走了

没关系，这些事大多数人都经历过。记住，你是来学习的。

*   找到适合自己的 meetup:
    如果你在 meetup，在那里你关心这个话题，你会觉得你知道这个话题或者你不知道，并且对它非常好奇。这两件事都有助于开始对话

*   倾听听起来很简单，但却很有帮助。你会成为一个更积极的倾听者:你肯定会学到更多，并有一个好的理由在活动期间或活动结束后与演讲者展开对话。当你听人们谈论什么的时候。提出反问题通常更容易

*   保持目光接触，保持开放的姿态会让你成为一个自信的人。右对齐表示你对谈话感兴趣。用微笑和握手来问候别人只是为整个谈话定下了基调。

*   微笑:打破僵局有时很难。在这种情况下，微笑大有帮助。如果你在另一个聚会上遇到了某人，只要向你已经见过的人问好。你将能够礼貌地加入对话，结识更多的人。

*   开始谈话的重要性:不要等着，以为有人会注意到你并和你说话。这种情况发生的可能性非常小。相反，找一群你觉得可以交谈的人。走过去，介绍你自己，对这个人和他们的技术感到好奇，很快，对话就会开始。

*   问很多问题:
    如果你不把自己的经历变得有趣，你肯定会感到无聊。和说话的人或者坐在你旁边的人说话，不管是谁。寻求书籍、网站或其他聚会的推荐。如果你在自己的学习中没有成为一个积极的参与者，如果你经历的事情与你无关，不要感到惊讶。

*   不断尝试:
    每个人都是独一无二的。如果一种方法对你不起作用。只要改变方法，再次“清洗并重复”直到你找到适合自己的方法。

所以，现在走出去，从聚会中获得最大的学习体验。这里有一些你可以开始问的问题

*   你觉得这个聚会怎么样？
*   这是你第一次来这里吗？
*   你用什么有趣的方法来解决类似的问题？
*   关于该主题的书籍和网站推荐

保持积极快乐的学习！！！
