# 你在 Linux 上使用 git 的 GUI 吗？哪一个？

> 原文：<https://dev.to/susickypavel/do-you-use-gui-for-git-on-linux-which-one-477c>

**大家好！**

我想问你在你的机器上用的是哪个贵由的 Linux 还是你用的 CLI？

我问是因为我不确定选哪个。我一直在 Windows 上使用 Git 扩展，我爱上了它，因为它是完全免费的，有着令人敬畏的功能和体面的用户界面。

然而对 Linux 的支持很差，它不能正常工作(或者我不能正确配置它)

你最喜欢的是什么？

[更新]:
如果有人正在运行 Linux 并且很难找到合适的 git GUI，我可以推荐 [gitahead](https://gitahead.github.io/gitahead.com/) ，它是开源的，拥有我需要的所有东西:

*   图形化 git 历史
*   简单明了的用户界面
*   用自定义操作系统主题玩得好
