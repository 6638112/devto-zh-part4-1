# [T1】我的工作流程](#my-workflow)

对我来说，我最近不得不在 markdown 写一篇论文。为此，我的工具是:

*   编辑器:Atom，(插件:降价预览，降价作者)
*   乳胶:texmaker
*   协作/注释:谷歌文档

## [](#what-i-like-about-it)我喜欢什么:

Atom 是我最喜欢的文本编辑器之一，所以我真的很高兴继续使用它。有了 markdown 预览插件，我可以实时看到我所做的修改。

## [](#what-i-do-not-like-about-it)我不喜欢的地方:

为了与其他人合作，我目前使用谷歌文档进行实时编辑。如果我想在那里得到文本，我就从我的 markdown 文档中粘贴它。完成实时编辑后，我将从文档中复制并再次粘贴到我的 markdown 文档中。

## [](#other-aspects)其他方面:

我明白了为什么我必须把所有的东西都转换成`.tex`以便查看格式和所有的东西是否工作正常。自动化解决方案可能会更好，但我还没有考虑过。
