# 谈论编程是艰难的！

> 原文:[https://dev . to/tech Jr/talking-about-programming-is-tough-59bh](https://dev.to/techjr/talking-about-programming-is-tough-59bh)

我记得我第一次学习编程的时候。我认为每个变量都应该是 x、y 或 z，试图不用函数来做任何事情，递归的概念让我的脸融化了。(老实说，合并排序还是让我很不爽🤯)

因此，在这一集里，我们试图解释所有我们能想到的概念，从变量开始，逐步增加复杂性。我们不能 100%完成每件事，但我认为我们做得很好！
