# 让我们玩“说出偏见”的游戏

> 原文：<https://dev.to/justzeros/let-s-play-name-that-bias-3nac>

我姐姐有一对 4 岁的双胞胎孩子，一个男孩，一个女孩。两人都患有自闭症，并且在自闭症谱系中处于不同的位置。

周日父亲节那天，我和他们一起吃午饭，我注意到一些有趣的事情。

我们的女服务员碰巧提到她两岁的儿子不喜欢大声喧哗。

我的姐姐立刻大声说，“哦，你应该让他检查一下，我的儿子不喜欢大声的噪音，我们发现他有自闭症。”

后来，又有人提到，她的小女儿晚上难以入睡。我姐姐又一次大声说道:“哦，我们的小女儿也很难接受。你应该让她检查一下自闭症。”

这不是我第一次看到这个，但却是我第一次注意到。

我不知道怎么称呼它，所以我用的是短语**经验偏差**。

经验偏见的一个有效定义可能是，*“一种倾向于**注意到**与我们过去的经验相似的情况，其中包含强烈的情感内容。”*

可观察到的行为不是她注意到了，而是她给别人的“建议”。但这不会发生，除非她发现了什么。例如，我听到了同样的对话，但没有听到同样的东西。

我认为趋势(或强迫！)主动给别人提建议也有个名字，但我不确定是什么。也许*《咨询师的诅咒》*？我自己也遭受过这种痛苦。

当我们有一个包含强烈情感的经历时，我们的注意力似乎会转向我们周围相似的环境。

这可能是有一个患有自闭症的孩子，去度一个期待已久的假期，遭遇一场车祸，甚至是获得一次当之无愧的晋升。

### 这里至少有两节课——可能更多。

首先，注意你在收听什么；这些天你注意到了什么？对我来说，我注意到人们在谈论出版一本书，安装供水系统，以及治疗痛风。

是的，我知道这很奇怪。但是，如果有人提起这三件事，我可能会加以评论(甚至给出建议)。

注意你所注意到的是很重要的，因为它可以让你避免顾问的诅咒。

它允许你停下来问一个问题，比如“你想听听我的经历吗？”或者你会决定保持沉默，而不是贸然给出建议。

第二，当你看到别人提供建议时，你就知道他们身上发生了什么。他们的经验偏见可能是在提醒他们过去情感上重要事件的现状。或者他们正在处理的事情。

无论哪种方式，这都是一种更好地了解你周围人的有趣方式。

你看到这种情况在你身边发生了吗？

你对它有什么看法？

给我回信…

极品，马库斯
