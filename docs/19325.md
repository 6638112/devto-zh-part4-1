# C-mpler 简介

> 原文:[https://dev.to/giladri/c-mpiler-introduction-5g1a](https://dev.to/giladri/c-mpiler-introduction-5g1a)

作为一名 C 语言爱好者，我一直想成为这种语言的编程大师，但我一直没有时间。(我作为开发人员，5 年多的主要语言是 Python)。

在巴尔伊兰大学攻读计算机科学学士学位的最后一年，我开始担任“计算机科学导论”课程的导师，该课程在巴尔伊兰大学是用 c 语言讲授的。

我在教许多学生用 C 语言编程时获得了很多乐趣，并决定写一个笔记本，在里面我总结了我所教的课程。

我把这本笔记本(如果你想要一本- **希伯来文** -请发邮件给我)发给了我的学生和他们的朋友——它在大学里变得相当有名。(我仍会不时收到电子邮件——自从我发表这篇文章以来，已经过去了 5 年多)。

在接下来的帖子中，我将重写这个笔记本(这次用英语！).

在那之后，我将谈论 Python 或 kot Lin——因为我真的相信你必须先学习 C 才能成为最好的程序员，然后才能继续学习另一种编程语言。(在接下来的帖子里你会明白为什么)。
