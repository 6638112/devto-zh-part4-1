# String charAt()方法

了解`charAt()`字符串方法。它将返回一个由基于其索引定位的单个字符组成的新字符串。

* * *

> 注意:如果没有索引被证明到`charAt()`，它将退回到`0`

使用无索引
的`charAt()`

```
const city = 'Miami';
console.log(city.charAt());
// charAt() using default index => 0
/**
 * @return 'M'
 */ 
```

使用`charAt()`索引

```
const country = 'USA';
country.charAt(1);
/**
 * @return 'U'
 */ 
```

的`charAt()`

