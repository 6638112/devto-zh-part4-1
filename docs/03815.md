# 为什么是数据科学？

> 原文:[https://dev.to/patman17/why-data-science-346b](https://dev.to/patman17/why-data-science-346b)

### [](#intro-amp-background)简介&背景

你好，我叫帕特里克·李。我已经在石油和天然气行业工作了 5 年。首先，我担任了一个作业工程师的职位，这是办公室和油田作业之间的一个中间技术职位。我帮助提高了我们钻井和完井项目的有效性和效率。在我职业生涯的后期，我被提升为现场主管。在这个职位上，我负责通过委派和促进工作流程来管理油田服务设备、资产井和现场人员。

最近今年，我的公司被另一家上游石油公司收购，我被解雇了。事实上，这是塞翁失马，焉知非福，因为今年早些时候，我曾考虑辞职转行。我喜欢和我一起工作的人，在对业务产生重大影响中找到满足感，以及金钱上的回报。然而，石油和天然气行业的运营方面并没有激发我的技术兴趣，我也没有看到它与我的长期职业目标相一致。

### [](#the-search)搜索

我开始找工作，想找一个能利用我在石油和天然气行业的商业知识，但更侧重于技术的职位。根据我以前的经验，我发现传统的钻井和完井工程师角色最适合我，但是这个角色的技术部分对我没有吸引力。作为运营工程师，你首先是项目经理，然后是业务分析师。因此，你能做的更高层次的分析是有一些限制的。当务之急是在预算范围内执行和管理运营。

随着我求职的进展，我注意到与分析和数据科学相关的职位越来越多。这是我对这个领域做更多研究的时候。一个主要的信息来源来自我的一个朋友，他刚刚完成了一个数据科学的硕士项目。他谈到这是一个相对较新的领域，以及需求是如何增长的(尤其是在石油和天然气领域)。他还谈到了我感兴趣的数据科学的高级应用。

### [](#the-reason)原因

带着这个兴趣，我开始审视我想从事数据科学的原因。下面总结了要点:

1) **非常适合** -除了具备工程师的技术能力之外，数据科学还适合我的个性，因为我是一个非常有逻辑和分析能力的人，喜欢看手头主题的客观性。
2) **提高技术技能**——数据科学将提高许多技术技能，如编程、统计分析、解决问题等。它将提供一个平台，让我成为一个永远的学习者。我可以想象出数据科学的许多应用，不仅可以用于我的职业，也可以用于我的个人生活。

总的来说，我觉得数据科学是扩展我的技术能力和将我的职业发展到另一个我感兴趣的领域的好方法。
