# 消除通话噪音的 Krisp 应用程序现已正式适用于 Mac 和 Windows

> 原文:[https://dev . to/Anna _ bazo Yan/krisp-app-that-remove-noise-due-in-calls-is-noise-noise-that-noise-is-noise-to-both-MAC-and-windows-8ge 现已正式发布](https://dev.to/anna_bazoyan/krisp-app-that-removes-noise-during-calls-is-now-officially-available-for-both-mac-and-windows-8ge)

我最近写了一篇关于这款新的人工智能应用的帖子，当你与某人通话时，它可以消除背景噪音。它被称为 Krisp，由深度神经网络支持。

因此，它基本上是双向工作的，这意味着它消除了从你到其他通话参与者(Krisp 麦克风)的噪声，以及从其他通话参与者到你(Krisp 扬声器)的噪声。你所要做的就是从你选择的会议应用程序(如 Slack、Zoom、Skype、Webex 或任何你喜欢的应用程序)中选择它作为麦克风和扬声器，然后享受通话。

对于最重要的部分-它可以与任何会议应用程序(允许更改音频设置)，任何耳机或耳机(有线和无线)以及任何麦克风和扬声器配合使用。

现在，该应用程序正式推出，成为 Mac 和 Windows 的完整产品。他们稍微改变了一下界面，看起来很酷。

他们推出了两种新的免费和付费计划——针对个人的 Krisp 和针对团队的 Krisp

使用该应用消除来自他人的噪音(Krisp 扬声器)是*免费的*，而如果你想消除自己的噪音，你必须升级到 Krisp Premium(每月 20 美元或每年 120 美元)。当然，有 14 天的免费试用期，不需要信用卡。

Krisp for teams 的工作方式相同。你用你的公司邮箱注册，用 Krisp 免费清除别人的噪音，升级(20 美元/月/用户)到 Krisp Premium 也能清除你自己的噪音。

我认为 Krisp 现在最好的一点是，他们有一个简洁的推荐计划。

基本上，您可以通过个人链接与朋友分享 Krisp，并获得免费积分。

1.  邀请一个朋友——你可以获得 3 个月的 Krisp 免费积分，而他们可以获得 2 个月。
2.  邀请两个朋友——你可以享受半年的 Krisp 服务，而他们只需 2 年。

这对团队的 Krisp 来说是不同的。在这里，你可以邀请尽可能多的团队成员(使用相同的公司电子邮件)，并且你们都可以免费获得 Krisp 半年。你必须确保在你注册的前 7 天内完成。

就目前而言，我认为这款应用与其他 windows 和 mac 生产力应用一样，有很大的潜力成为生产力应用。这将是对所有远程工作者、自由职业者、团队和企业的额外帮助。
