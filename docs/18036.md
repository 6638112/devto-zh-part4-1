# [](#the-new-way)新的方式！

然后出现了`flat()`。这是一个游戏改变者。

做我们上面做的同样的事情是轻而易举的。

```
var merged = arr.flat(1) 
```

那个论点只是我们想要展平的深度。所以一级深给我们这个，就像以前一样。

```
[1, 2, 3, 4, [5, 6]] 
```

注意，如果你没有传递一个参数，它默认为`1`。这意味着这些语句是等价的。

```
arr.flat(1)
//is the same as
arr.flat() 
```
