# 达到更高水平的新努力

> 原文:[https://dev . to/web dva/a-new-endeavor-to-reach-the-next-level-430 p](https://dev.to/webdva/a-new-endeavor-to-reach-the-next-level-430p)

我发现自己开始了新的努力，因为在我寻求修复世界的过程中，之前的努力只是做了一件错误的事情。

在通往成功的道路上，我不会有朋友，因为我的职责要求我从生活中摒弃这些东西。这是我一个人的责任，不是别人的，因为我的野心是我一个人的责任，不是别人的。在这一努力中，只有我和我研究的这些数学对象，这些抽象的无知觉的生物控制着我试图征服的世界。

这种努力，我现在寻求的答案可以在书中找到，要求我不要从我的物质形态非常熟悉的世界，从你非常热爱并创造了一个舒适住所的现实中找到想法，而是从另一个新世界中找到想法，这个新世界的真理对我们的现实享有巨大的主权。这个世界的土地很少被敢于勘察它的少数男人和女人绘制。

这是先进科学的无意识的立法者和指挥者的世界，他们决定什么是什么以及为什么和如何。

一个充满强大力量的世界，即使是你也有一种天生的本能。它的诱惑很容易诱惑你走向它空虚的怀抱，如果你决定回应它的召唤，把你留在它冰冷的怜悯中，它最终确实诱惑了你。

这个世界需要我们的前辈们同样的勇气。他们同样重要的力量。

一个我必须冒险进入的世界，这样我才能提取它的效用为我造福:繁荣计划在今年结束前取得成功。一个简单的成功和目标:实现一个非主导的社会经济地位，一个当地的最优繁荣，因为有一个主导的社会经济地位是完全不利的。

当你为了实际应用而学习高级科学和数学时，你并不真正与其他参与者直接竞争，尤其是当很少有人敢于冒险进入这个世界的时候。相反，你只是为了未来的自己，以现在的自己为代价——痛苦——与你自己，过去的自己竞争。

这些只是这项努力成功的必要条件。
