# tflookup -开发者日记尾声

> 原文：<https://dev.to/dploeger/tflookup-developer-diary-epilogue-de7>

亲爱的日记，

自从我给你写信讲述我用[创建一个 Terraform 文档查找应用](https://dev.to/t/tflookup)的经历已经有几天了，当我每天使用 tflookup 时，我注意到了一些事情。

当我早上开始工作，并且第一次有兴趣查找 Terraform 的一些提供商资源时，我自然会查看 tflookup。这可不是什么有趣的经历，因为我必须等待，直到 tflookup 最终出现。

这是因为，Heroku 有一个免费订阅的叫做 [dyno sleeping](https://devcenter.heroku.com/articles/free-dyno-hours#dyno-sleeping) 的东西，由于目前没有多少人倾向于使用 tflookup，这个应用程序变得很困，陷入深度睡眠，直到我早上醒来。

我不想为我的这个小项目支付 Heroku，作为一个简单的 Node.js 应用程序，很容易部署，我只是在我的服务器上部署它，我管理了多年(它已经在每天晚上积极重建 tflookup 索引)

所以从现在开始，我将在一个新的地址上检查 [tflookup。](https://tflookup.blueocean-net.de)

你的丹尼斯
