# 是什么阻止你尝试新技术？

> 原文:[https://dev . to/gdotdesign/what-keep-you-out-of-trying-new-tech-2096](https://dev.to/gdotdesign/what-keeps-you-from-trying-out-new-tech-2096)

作为一名开发人员，我的目标是创造人们使用的软件，(特别是 [Mint](https://www.mint-lang.com) )但在我看来，人们真的很难迈出这一步，尝试一些新的东西。

例如，在我真正花时间去理解它并取得进步之前，我已经开始学习 Elm 4-5 次了。

我很好奇，是什么让你不去尝试新的库和技术？最后是什么让你这么做的？
