# 你不知道的 5 个惊人的机器学习框架

> 原文:[https://dev . to/David boo 90/5-amazing-machine-learning-frameworks-you-do-not-know-about-45f 7](https://dev.to/davidboo90/5-amazing-machine-learning-frameworks-you-did-not-know-about-45f7)

如果你像我一样是个科技极客，你可能会密切关注科技新闻。如果是这样的话，那么你可能已经注意到像“机器学习”和“人工智能”这样的术语经常出现在头条新闻中，这是有充分理由的。它们是目前技术领域最热门的两个话题，为改变我们生活的方方面面提供了巨大的潜力。

想想任何一个行业，你可能会发现[人工智能](https://www.techsciencehub.com/future-artificial-intelligence-news-trends/)或机器学习在某种程度上发挥着作用。无论是医疗保健、制造、零售建筑还是交通运输，你都会发现人工智能和机器学习无处不在。一个尚未受到人工智能和机器学习影响的行业是 web & [应用开发](https://www.branex.ca/mobile-app-development/)。谢天谢地，这种情况也即将改变。

机器学习利用其算法赋予机器学习能力。这意味着你不必像编程机器人一样编程。凭借无与伦比的数据分析能力，它使分析模型的创建更加简单和容易。这就是为什么机器学习框架可以改变我们今天所知道的 web 开发。

随着机器学习的进步，我们正在看到面向 web 和应用程序开发人员的机器学习框架和工具的出现，这将使开发变得轻而易举。在本文中，您将了解五个令人惊叹的机器学习框架，您希望早点了解这些框架。

**1.TensorFlow**

TensorFlow 是目前最受欢迎的机器学习框架。主要用于 Java 开发，这是一个开源库，它利用数据流图的能力来进行计算。TensorFlow 从机器学习框架的其余部分中脱颖而出的原因是它能够使用单个应用程序编程接口(API)执行一个或多个 CPU。

Web 开发公司可以利用这一点，并可以使用任何设备，无论是服务器、台式计算机还是移动设备。数学运算用图上的节点表示，而节点之间的多维数据集用图上的边表示。您可以轻松地创建和部署不同的模型，并能够为研究目的进行强大的实验，这使它比竞争对手具有明显的优势。

**2。微软认知工具包**

谷歌、微软和亚马逊等科技巨头正在人工智能和机器学习上下大赌注。这就是为什么他们在这些技术上投入了数百万美元。微软认知工具包是一个开源的深度学习工具包，专注于训练算法和机器像人脑一样学习。这个机器学习工具包允许你充分利用各种机器学习模型，如前馈 DNN，递归神经网络或更多。

不可否认，这个工具包主要是为筛选大量非结构化数据而设计的，但这并不意味着它不能做其他事情。它易于使用的架构和它的能力，以定制你的心内容，使它的头和肩膀以上休息。用户可以根据自己的需要选择自己的算法、网络甚至自己的参数。微软认知工具包还可以支持多 GPU 和多机器架构，这是其他机器学习框架所不支持的。有了微软的支持，你可以放心，你是安全的。

**3。阿帕奇看象人**

虽然不如 TensorFlow 广为人知，但 Apache Mahout 在受欢迎程度方面也不落后。Apache 的这个开源框架是根据数据科学家、统计学家和数学家的需求设计的，它允许您更快地运行算法和计算。Mahout 的主要重点是分类、过滤和分组。

想要创建自己的数学计算？你也可以在一个交互平台上用 Apache Mahout 来做这件事。对于那些对统计和代数计算感兴趣的人来说，Mahout 有一个单独的版本，叫做 Mahout Samsara。通过使用 Apache 自己的 Hadoop 架构来利用大数据的力量，但不限制用户实施。

**4。咖啡**

Caffe 的设计考虑到了速度和模块化，它是一个面向 Java 开发人员的深度学习框架。它的开放式架构有助于开发人员个性化他们的应用程序，并为创新铺平道路。得益于定制选项，用户甚至可以在不同的 CPU 和 GPU 之间切换。他们只需配置一个指示器，就万事大吉了。

代码的可扩展性导致了这个工具的快速发展。它惊人的速度使其成为研究和工业应用的绝佳选择。对于那些不想编码的人来说，他们可以看看模型动物园，在那里他们可以找到预先训练好的准备实现的模型。这是创建应用程序、人工和计算机视觉的最佳选择之一，但在其他行业大规模采用时，它将落后于其他参与者。

**5.Apache Singa**

如果你正在寻找一个可扩展的深度学习解决方案，为你提供增长的灵活性，那么看看 Apache 的 Singa 就知道了。该机器学习框架由新加坡国立大学开发，用 C++、Python 和 Java 编写，提供灵活的架构来满足您的分布式培训需求。它在自然语言处理和图像识别方面表现出色，但它能够在各种硬件上运行，这也使它成为其他应用程序的合适选择。

当谈到不同型号的可用性时，它仍然远远落后于其他同行，因为用户仅限于少数型号。它只支持传统的机器学习模型，如逻辑回归，这是一个很大的缺点，特别是对于那些想要尝试现代机器学习模型并利用其能力的用户来说。
