# 使用 OAuth 和 OpenID Connect 保护 API 和微服务

> 原文:[https://dev . to/JavaScript _ tuto/securing-API-and-micro services-with-oauth-and-OpenID-connect-5452](https://dev.to/javascript_tuto/securing-apis-and-microservices-with-oauth-and-openid-connect-5452)

微服务提供了一种扩展 API 部署的新方法，其中每个组件都是一个孤岛，执行一个小但定义良好的任务。

这些系统开发起来更快，并且允许更灵活的工作方式。与大多数设计一样，安全性不是原始蓝图的一部分，这可能导致昂贵且难以管理的安全解决方案。

在本次演讲中，特拉维斯·斯潘塞将展示如何利用 OAuth 和 OpenID Connect 为微服务创建统一的分布式框架。他将展示如何利用这一点来实现微服务的敏捷性和可扩展性承诺，同时确保安全性。
