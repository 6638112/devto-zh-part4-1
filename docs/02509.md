# 每日挑战#65-伪装序列

> 原文：<https://dev.to/thepracticaldev/daily-challenge-65-a-disguised-sequence-ng>

给定`u0 = 1, u1 = 2`和关系`6unun+1-5unun+2+un+1un+2 = 0`对任意整数`n >= 0`计算 un。

**例如:**
`fcn(n) returns un: fcn(17) -> 131072, fcn(21) -> 2097152`

**注:**
你可以把这看成

-根据 u^n
的定义，纯粹是算法上的-第二个——完全不是强制性的，但作为一种赞美——是让你的脑袋转一转，找到 u^n 背后隐藏的序列
