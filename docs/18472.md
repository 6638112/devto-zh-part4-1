# 框架如何帮助我赚更多的钱

> 原文:[https://dev . to/ilvalerione/how-a-framework-can-help-developers-make-more-money-5d 46](https://dev.to/ilvalerione/how-a-framework-can-help-developers-make-more-money-5d46)

如果你是一名开发人员，你可能已经被你周围的每个人推动去学习一个框架来开发你的应用程序。但是没有人清楚地告诉你这会给你带来什么好处。那么，你为什么要相信一个框架呢？

这个选择从来都不容易。开发人员很难选择一个框架的主要原因是害怕受到不利条件的限制。

但是在每个开发者的旅程中，都有做出这个选择的时刻。几乎总是在同一个时刻，我们终于找到了我们想要认真投入的商业想法。(在这篇文章的最后，我将谈谈我的第一次 SaaS 产品体验。)

采用一个框架可以带来巨大的成果，并使您能够构建为您带来商业成功的资产。

## 建立最重要的资产，从你的努力中获得更多价值

在网络革命爆发之前，资产是这样的东西:

*   要出租的大楼
*   黄金和钻石
*   专利
*   盈利的公司

这个列表仍然不错，但如果你看看当今世界上最大的公司，如谷歌、脸书或苹果，你会很容易地发现，今天通过开发软件产品比找到钻石更容易获得成功。

资产是具有巨大价值的东西。它不需要有形，用砖或铁制成。对于开发人员来说，最有价值的资产是定义一个过程。

试着想象一下，如果你有一个清晰、明确、可重复甚至自动化的方法，你的工作会变得多么容易。

我给合作者的建议是:

**别再找工具或库来少写代码了。相反，找到真正能帮助你定义正确的过程来长期解决具体问题的方法。**

寻找和建立资产。

## 一步一步地，你会看到你的每小时生产率在增长

软件开发人员是这个世界上热爱自己工作的幸运人群之一。许多人学习很多，所以如果他们把注意力放在正确的事情上，他们就能获得巨大的成果。学习一个框架可以带来巨大的成果。

框架是你所拥有的最重要的资源，因为它推动你去创造资产，并使你所有的过程自动化。

例如，如果您花时间研究如何使用一个框架来处理身份验证，而不是从头开始编写代码，这并不意味着您在浪费时间，因为您永远无法以可重复的方式或没有错误地完成它。

第一步是要明白，这种想法并不是承认无知，而是一种尽可能快速、安全地实现我们业务目标的明智方式。

反对一个好的框架就像是对一个为成功企业开发应用程序的经验丰富的国际开发团队说不。他们给你如何达到目标的建议，你说不，你不想要约束。…真的吗？

在说不之前，想一想。也许阻碍你的是你自己。

## [](#my-experience-with-laravel-framework)我对 Laravel 框架的体验

在我的职业生涯中，我接触过所有主要的软件生产环境(PHP、Java、。但是只有使用 PHP 和 Laravel，我才体验到生产力的指数级增长。

Laravel 是一个开发环境，旨在使用基于“服务”概念的最现代、最有效的方法来构建 web 应用程序。

您将能够将一些功能委托给外部服务，并将全部精力放在您的业务和产品上。

无论您选择哪种框架，使用框架来开发您的应用程序都会给您带来许多好处，帮助您从工作中获得价值。

Laravel 是我开发的 SaaS 产品 Inspector 的基础，这是一个 PHP 实时监控工具，可以帮助开发人员快速修复和更改他们的应用程序，保持业务稳定和安全。

这只是用正确的工具和正确的思维方式可以做的数百万个例子中的一个。
