## KoaJS

`yarn add koa`或`npm install koa`
创建一个文件(koa.js)并粘贴如下:

```
const Koa = require('koa');
const app = new Koa();

app.use(async ctx => {
  ctx.body = 'Hello World';
});

app.listen(3000, () => console.log('Running on http://localhost:3000/')); 
```
