# 每日挑战#30 -价格是多少？

> 原文:[https://dev . to/the practical dev/daily-challenge-30-what-the-price-16ee](https://dev.to/thepracticaldev/daily-challenge-30-what-is-the-price-16ee)

你今天面临的挑战是找到打折前产品的原价。

> 例如:
> 
> 给定一个在应用 25%折扣后售价为$75 的商品，该函数应该返回该商品在应用销售百分比之前的原始价格，当然是$100.00，四舍五入到两位小数。
> 
> DiscoverOriginalPrice(75，25) => 100.00M，其中 75 是销售价格(折扣价)，25 是销售百分比，100 是原始价格

*注意*:返回类型必须是 decimal 类型，数字必须四舍五入到两位小数。
