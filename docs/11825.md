# [](#learn-javascript)学习 JavaScript

我们在 Google Play 中发布了一款名为 **Learn JavaScript** 的学习 JavaScript 的 Android 应用。

## [](#features)特性

1.  在线和离线模式下阅读
2.  它将涵盖 JavaScript 的基础到高级主题。
3.  清晰的代码示例和解释。
4.  练习小测验，让你理解题目。
5.  成就，使学习更有趣。
6.  JavaScript 文章，让您了解最新动态。

### [](#download)下载
