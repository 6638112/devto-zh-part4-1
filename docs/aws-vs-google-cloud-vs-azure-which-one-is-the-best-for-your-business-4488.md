# AWS vs Google Cloud vs Azure:哪个最适合你的业务？

> 原文：<https://dev.to/serhiykozlov/aws-vs-google-cloud-vs-azure-which-one-is-the-best-for-your-business-4488>

## 对前三大云平台的详细分析，以及对比图表和构建云迁移有力案例的更多见解。

迁移到云！这是您可能听到的关于优化现有基础架构和降低 TOC 的常见 IT 建议。

事实上，如今 73%的企业已经拥有至少一个应用程序，或者部分云基础设施。最大胆的预测甚至指出，在一年内， [83%的企业](https://www.forbes.com/sites/louiscolumbus/2018/01/07/83-of-enterprise-workloads-will-be-in-the-cloud-by-2020/#1a3362f26261)工作负载将驻留在云中。

风险很高:迁移到云是创新业务和部署新的竞争服务的最简单方式。云服务还通过即时资源扩展提供了更大的灵活性；更高的 SLA 级别和更顺畅的途径[即使在灾难来袭时也能保持业务连续性](https://www.romexsoft.com/blog/disaster-recovery-plan/)。因此，毫不奇怪，超过三分之一的[IT 部门](https://www.idg.com/tools-for-marketers/2018-cloud-computing-survey/)被迫 100%迁移到云平台。

然而，云迁移不应被视为一件“时髦”的事情。要想成功地将你的计划付诸实施并获得收益，需要仔细考虑多个方面。选择符合您业务需求的最佳云服务提供商将是朝着这个方向迈出的重要一步。

### 云迁移:对您的企业有什么好处？

如今，有三大云服务提供商——亚马逊网络服务(AWS)、谷歌云平台(GCP)和微软 Azure。AWS 目前占据 IaaS/PaaS 市场的最大份额(51%)，而谷歌的云平台(该领域的新进入者)的市场份额最低，根据市场观察报告，仅为 6% [。](https://www.marketwatch.com/story/amazon-rules-the-public-cloud-but-google-microsoft-alibaba-are-growing-faster-2017-12-20)

虽然所有供应商都提供相同的通用云服务，但在公司做出云计算决策时，仍有一些差异需要考虑。

在一般服务和优势方面，所有三家云提供商都可以[帮助您的企业获得下一个云化优势](https://www.romexsoft.com/blog/migration-to-aws-cloud-benefits/):

*   随着公司规模的扩大，开发更多的软件、应用程序、基础设施等。他们现有的硬件变得不堪重负。他们有两个选择——1)继续以高昂的成本购买更多的硬件，然后雇用额外的员工来维护额外的硬件和系统，或者 2)使用云存储系统来容纳不断增长的基础架构、软件、应用程序等。
*   迁移到云意味着当前的 IT 员工可以专注于更有成效的工作，如开发，而不是本地基础架构维护。您可以将全天候 IT 支持外包给[托管服务提供商](https://www.romexsoft.com/aws-managed-services/)。
*   敏感信息迁移到基于云的系统(比如健康和财务信息)意味着这些数据被泄露的风险大大降低。[最大的违规原因之一是员工的疏忽](https://www.cnbc.com/2018/06/21/the-biggest-cybersecurity-risk-to-us-businesses-is-employee-negligence-study-says.html)，尤其是个人使用公司电脑。另一个风险是外部供应商，例如几年前对目标数据泄露负有责任的供应商。如今，甚至可以定制公共云配置，以满足顶级安全标准和合规性要求。
*   如果自然灾害或其他事件导致系统崩溃，拥有软件、数据和应用程序等。在云中意味着什么都没有丢失。工作可以不间断地进行。到目前为止，组织不得不寻找远程物理设施来存储数据。

### 亚马逊网络服务(AWS)云比较

亚马逊已经成为市场领导者一段时间了，因为该公司是最早提供云计算解决方案的公司之一，现在几乎拥有业内所有可用的功能。

AWS 最大的好处之一是它可以服务于任何规模的组织。像网飞和 Expedia 这样的大公司依靠 AWS 在全球范围内提供服务。然而，小企业也能找到他们需要的一切。亚马逊的优势如下:

*   定价是基于使用量，而不是固定的费用。小企业觉得这很有吸引力。存储是可定制的，同样，成本是基于数量的
*   AWS 很容易扩展——它可以支持几乎无限数量的用户——这也许是网飞选择的原因。
*   有很多灵活性、定制和对第三方集成的支持，几乎任何组织都可以通过 AWS 获得服务。
*   实时分析和大数据解决方案可通过亚马逊专有的 Kinesis Streams 和 Firehose 获得。你将需要一个小的开发团队来帮助你建立这些。
*   更新和持续的新功能很有吸引力，因为这个平台继续评估客户的需求和痛点。

### 谷歌云平台云对比

谷歌以向各种规模的公司提供各种各样的产品和服务而自豪。最值得注意的是计算、网络、存储、大数据和机器学习。它最大的优势之一是数据管理，尤其是应用程序。

以下是优点:

*   数据分析和存储对用户非常有吸引力，ML 服务相当先进
*   与所有其他谷歌服务轻松集成
*   有一个免费的包，虽然它是相当有限的，和长期合同的高折扣价格
*   特殊安排很容易谈判。

### 微软 Azure 云比较

像 AWS 一样，Azure 提供了一些定制配置的虚拟机。有两个选项可用——Azure SQL 和云 SQL。通过 Cortana 可以进行分析，实时数据通过其专有的流分析进行处理。显然，它可以与任何其他微软产品完美集成。

其优点如下:

*   Azure 几乎是一个“开箱即用”的解决方案——设置非常简单。
*   如果一个组织有微软的产品，与 Azure VM 的集成几乎是无缝的。
*   它根据客户需求提供多种价格。
*   Azure 保证每年停机时间不超过 4.38 小时。
*   它最大的优势是速度。

### 计算云解决方案的成本

当然，这涉及到成本。但是当它与内部解决方案的成本相比较时，真的没有可比性。

所有云服务公司都会收取前期费用来将你的数据迁移到云端。之后，根据您选择的云平台提供商等级，将会有持续的订阅费用。成本也因您现在迁移的数量以及未来添加更多计算而异。你最好的选择是使用一个有按使用付费系统的服务。

成本总是可以预先计算的。在你做决定之前。
