# 杰出的开发者#1:卡斯滕·海兹勒——启蒙运动的创造者

> 原文：<https://dev.to/bengineer/distinguished-devs-1-carsten-haitzler-creator-of-enlightenment-4of0>

卡斯滕·海兹勒(Carsten Haitzler)，又名拉斯特曼(Rasterman)，是 Enlightenment 的创始人，这是一个窗口管理器，用于全球数百万台 Linux 设备——从桌面和智能手机到智能手表和电视。

作为一个大型项目多年的创造者和领导者，在这次采访中，他谈到了开源是如何改变的，以及他是如何成为领导者的。Carsten 也给出了他对学习编码的想法，以及怎样才能成为一名优秀的开发人员。
