近年来，互联网连接速度不断提高，这使得 web 开发人员可以用越来越多的媒体来丰富他们的页面。就在几年前，一个网页的平均大小是 60-70 万。现在，平均水平是 3MB，并且每年都在增加。

图片有助于吸引访问者，降低跳出率，但也可能是导致网站性能不佳的原因。平均来说，图片可以占到网页总大小的 50%以上，从而大大降低了加载网页的速度。

本文将涵盖 web 的[图像优化的所有方面，并将探讨如何在使用高质量图像的同时保持快速加载时间。](https://cloudinary.com/features/optimization_fast_delivery)

## 什么是图像优化？

图像优化是在不降低质量的情况下减少文件大小、尺寸和格式的过程。为了优化网页图片，你可以使用插件或者脚本，这反过来加快了网页的加载速度。有损和无损压缩是两种常用的方法。

## 优化网页图片的好处

为网页优化图片有很多优势，但这些优势有时会被网页开发者所忽视。让我们一个接一个地来看看它们对你的网站的性能有什么贡献:

*   **更少的服务器存储空间和带宽**—优化的图像在 web 服务器上占用更少的存储空间。从服务器检索优化的图像需要较少的带宽。
*   **提高页面加载速度**—图片经过优化的网站加载时间更短。优化的图像遵循各种浏览器和多种屏幕尺寸的最佳实践，从而提高网站的性能。
*   **改善用户体验**—快速加载网站改善了访问者的整体用户体验。访问者最有可能重新访问快速加载的网站
*   改进的搜索引擎优化(SEO)——搜索引擎将会提高快速下载网站的排名

## 如何为网页性能优化图片

图像优化的主要目标是在尽可能小的文件大小和可接受的质量之间找到平衡。执行这些优化的方法不止一种。最受欢迎的方法之一是在上传之前压缩它们。

在优化你的图片之前，先用 [Google PageSpeed Insights](https://developers.google.com/speed/pagespeed/insights/) 在你的网站上运行一个速度测试。该测试将让您确切地知道要优化哪些图像，以及优化它们的最佳方式是什么。

### 选择合适的文件格式

网络上有多种类型的图像格式。为你的网站选择正确的格式至关重要。

*   **JPEG**—一种常用的文件格式，因其文件小且灵活。您可以调整图像质量，以获得质量和文件大小的良好平衡。
*   **PNG**—图像质量更高的文件格式，但文件更大。这种格式的文件是透明的，适用于像图标这样的小图像。
*   **GIF**—仅支持 256 色，只能无损压缩。这种格式的文件是动画图形的好选择。

还有其他几种格式，如 JPEG 200JPEG XR 和 WebP。谷歌的 Lighthouse 开发者工具推荐用这些现代格式编码图像，而不是 JPEG 或 PNG。这些格式具有卓越的压缩和质量特性，加载速度更快，消耗的蜂窝数据更少。

### 调整图像大小

优化网页图片最简单的方法之一就是在你上传图片到你的网站之前调整图片的大小。

在上传之前调整图片大小和裁剪图片会使图片变小，这将有助于网站加载更快，并为更多的图片节省服务器存储空间。

### 压缩图像

为网络压缩图像意味着减小其文件大小。压缩是通过在没有任何质量损失的情况下最小化图像的大小来实现的。有两种主要的压缩类型，有损压缩和无损压缩:

*   **有损压缩**—这意味着以一种大多数用户不会注意到的方式从原始图像中完全删除一些信息。
*   **无损压缩**—这意味着图像文件的大小将会减小，而不会从原始文件中删除任何信息。

### 懒加载图片

延迟加载是一种在页面加载时推迟非关键映像加载的技术。相反，这些非关键映像会在需要时加载。非关键图像是用户看不到的图像。

### SVGs 图像

SVG 是一种可扩展的矢量格式，非常适用于徽标、图标、文本和简单图像。SVG 图像在浏览器和照片编辑工具中都可以自动缩放。SVG 通常比 png 或 jpg 文件小。这可能导致更快的加载时间。

## 用于自动图像优化的 WordPress 插件

如果你的网站是用 WordPress 构建的，那么你可以使用 WordPress 插件优化图片。重要的是，你不能仅仅依赖插件本身。例如，你不应该上传 2MB 的图片到你的 WordPress 媒体库。这可能会导致你的网络主机的磁盘空间消耗得非常快。最好的方法是事先在照片编辑工具中快速调整图像大小，然后上传并使用插件进一步缩小。

## 结论

图像优化既是一门艺术，也是一门科学。它是一门艺术，因为对于如何优化图像没有一个明确的答案；它也是一门科学，因为有许多技术和算法可以减小图像的大小。人工智能(AI)软件在优化图像而不影响质量方面特别有用。但是，为图像找到最佳设置需要在许多方面进行仔细分析，例如格式功能、编码数据内容、质量、像素尺寸等。
