# [](#we-hosted-diversity-chats-and-you-should-too-what-we-learned)我们主持了多元化聊天，你也应该参加——我们学到了什么

[![](../Images/0cf04a46c294c672ce3602c88eb103d5.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--iG2_RIfA--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1600/1%2AQO_Iaxa9wyDXhNcLHFyA-A.jpeg)

多样性让每个人都走到了一起。无论你是对发展一个更具包容性的团队感兴趣，还是你已经生活和呼吸着它，关于平等的讨论可以激发机会和建立社区。

Angular 社区的一些最有才华的演讲者聚集在一起，与铺路基金会合作，在 1 月份的 ngAtlanta 会议期间的 This Dot 炉边聊天上讨论多样性和包容性。

共识很明确:改变科技工作场所的性别比例至关重要。演讲者和与会者能够一起深入探讨和了解个人，同时保持对多样性和包容性的自由判断。

[![](../Images/6f51025bc255a26328b32f78aebdc122.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--CKNdNpdE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/8064/1%2ArPPsUoTXhtFA0yzs7v5ZiA.jpeg)

我们讨论的最活跃的热点话题之一是，“为自己辩护意味着什么？”

演讲者和观众都表达了他们对重视自己、期待尊严和尊重的想法。

最大的外卖？"不管你的水平如何，你仍然有宝贵的投入."

鼓励每个人在各个层面都参与进来，这比以往任何时候都重要。来自各种非传统背景的开发人员正在学习编码，并渴望参与不断增长的技术市场。

但是，如果没有导师和指导，新开发人员将永远无法充分发挥他们的潜力。每个人都可以为科技世界做出贡献。但首先，我们必须通过创造一个公平的竞争环境来树立榜样。

[![](../Images/656f8dfe0920bd9f9d7ce85ee9274c4c.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--W3jVh8Sn--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/8064/1%2AI5q4DsJ_rHDkvSojAUlT_g.jpeg)

我们强烈鼓励所有组织者在他们的会议上主持多样性聊天！围绕包容性和开始进入技术领域的新劳动力展开对话。
