为了根据标准进行计数，d 中有许多选项

```
import std.algorithm;
auto arr = [1, 2, 3, 2];
assert(arr.count(2) == 2); 
```

```
import std.algorithm;
auto arr = [1, 2, 3, 2];
assert(arr
       .filter!(x => x == 2)
       .count() == 2); 
```
