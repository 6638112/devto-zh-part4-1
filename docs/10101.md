# 开发人员投资组合的 3 个强大技巧

> 原文:[https://dev . to/njericooper/3-powerful-tips-for-a-developer-s-portfolio-4g 47](https://dev.to/njericooper/3-powerful-tips-for-a-developer-s-portfolio-4g47)

在过去的 4 个月里，我已经建立和废弃了 3 个版本的个人网站。它丢失了很多东西，如果没有指导，我也无法弄清楚到底是什么。看了几本书，上了几门课，很明显我的网站是基础的。它没有特殊的网络技术。它不会帮助我成为标题背后的开发者。这个过程非常细致，但我明白了拥有个人投资组合的重要性。以下是我旅途中的重要收获:

#### [](#select-all-delete-and-start-from-scratch)全选+删除，从头开始

有时，我们变得依恋，并与我们的愿景和想法结婚。在这 4 个月里，我了解到大多数想法都很糟糕，而且愿景可能不适合预期的最终用户。我的网站没有上线。它没有每月的流量。没有直接指向它的反向链接，所以我没有试图把 Fabreze 放在我网站的便便堆上，而是屏蔽了它。当我重新开始时，每一次我都比上一次拥有更多的知识、技能和技巧。

#### 使用你最好的工具

我喜欢展示相关的工具和技术，但并不过分。拥有一个带有 aria 状态的下拉菜单或者一个动画 CSS 电子邮件表单是没有意义的。😔相反，我做了一个漂亮的按钮，链接到我的 CodePen 项目。这更有意义，因为这是我展示这类项目的地方。我想使用的导航栏有过渡和大量的 Javascript 来展示我的箭头功能技能。可悲的是，我不得不谨慎地使用 Javascript。

[![Saltbae Gif from Giphy](../Images/5d40b0e03700b8f41ec4d36c1f8b2e4f.png)T2】](https://i.giphy.com/media/5zkVwzBPwjBADbvFi6/200w_d.gif)

用你希望被雇佣使用的语言和工具创建你的网站。

#### [](#be-the-product)成为产品

我喜欢创作...对其他人来说。我经历了一段艰难的时间来使我的网站成为产品。我花了一个老派的产品设计会议来概述用户角色和他们的任务。我就是用户想看到的。我的形象、头衔和技能需要在上面，而不是在下面，就像我在创作其他东西一样。

#### [](#bonus-ask-for-help)加成:求助

作为一名初级开发人员，我必须通灵才能知道招聘经理到底在找什么。否则，我可能会参加面试或承担高级开发人员的工作量。我没有这样做过，但是我知道一些我可以评论我的项目的开发者。

你可能不想用像投资组合这样简单的东西来打扰你超级聪明和忙碌的网络。如果这意味着升级到下一个税级，嗯。记住，没有愚蠢的问题。你知道他们说什么，“安全总比不被重视好。”
