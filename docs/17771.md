# 将组合的文字处理、演示文稿、电子表格和电子邮件文档保存为 HTML 格式

> 原文:[https://dev . to/group docs/save-assembled-word-processing-presentation-spread sheet-and-email-documents-as-html-4bf 0](https://dev.to/groupdocs/save-assembled-word-processing-presentation-spreadsheet-and-email-documents-as-html-4bf0)

设想一个场景，您在 MS Word 中创建了一些 Word 文档，并希望在您的 web 应用程序中显示它们。那么，如何查看文件的内容呢？一个合适且简单的解决方案是，如果您可以获得 Word 文档的 HTML 格式，那么就可以在应用程序的 web 浏览器中查看它。不用安装 Office viewer 就可以查看文档，这不是很棒吗？现在让我们看看在制作 GroupDocs 时我们是如何利用 HTML 格式的。汇编更强大，对你更有用。

从 19.5 版本开始，汇编的文字处理文档、电子表格、演示文稿和电子邮件文件可以与外部资源一起保存为 HTML。这意味着现在可以将生成的报告与图像等资源一起保存为 HTML 文件，并且正如我之前提到的，您将能够在 web 应用程序中嵌入和查看生成的报告的内容。阅读更多—[https://bit.ly/2WRtqqc](https://bit.ly/2WRtqqc)
