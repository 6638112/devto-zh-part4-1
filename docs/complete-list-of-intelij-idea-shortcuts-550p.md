# 英特尔创意快捷方式的完整列表。

> 原文：<https://dev.to/thecodersblog/complete-list-of-intelij-idea-shortcuts-550p>

IntelliJ IDEA 是 Java 开发人员最喜欢的 IDE 之一。如果你花了大部分时间在这个想法上，那么这个列表是给你的。

### IntelliJ

*   运行`Shift` + `F10`
*   停止`Ctrl` + `F2`
*   `Alt` + `Enter`
*   `Ctrl` + `Space`
*   `sout` + `Enter` - > `System.out.println();`
*   `Ctrl` + `Click on Variable / Mehod / Class...`
*   设置`Ctrl` + `Alt` + `S`
*   项目结构`Ctrl` + `Alt` + `Shift` + `S`
*   替换`Ctrl` + `F`
*   在路径`Ctrl` + `Shift` + `F`中查找
*   重复行`Ctrl` + `D`
*   切换案例`Ctrl` + `Shift` + `U`
*   项目视图`Alt` + `F1`
*   待办事项视图`Alt` + `6`
*   转到类别`Ctrl` + `N`
*   转到文件`Ctrl` + `Shift` + `N`
*   生成代码`Alt` + `Insert`
*   用… `Ctrl` + `Alt` + `T`包围
*   自动套用格式`Ctrl` + `Alt` + `L`
*   切换工具窗口`Ctrl` + `Tab`
*   行注释`Ctrl` + `/`
*   多行注释`Ctrl` + `Shift` + `/`
*   移动语句`Ctrl` + `Shift` + `Arrowkeys`
*   移动单线`Alt` + `Shift` + `Arrowkeys`
*   *重构*重命名`Shift` + `F6`
*   结构`Alt` + `7`
*   VCS `Alt` + `9`
*   *VCS* 提交`Ctrl` + `K`

## - *吉特*回复`Ctrl` + `Alt` + `Z`

*VCS 标签*显示差异`Ctrl` + `D`
