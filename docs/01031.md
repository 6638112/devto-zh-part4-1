# Xfce:自定义键盘快捷键

> 原文：<https://dev.to/nabbisen/configuring-xfce-keyboard-shortcuts-3lm5>

[Xfce 桌面](https://www.xfce.org)支持自定义键盘快捷键。

打开 Xfce 菜单。
进入【设置】-【键盘】。
然后激活“应用程序快捷方式”:

[![Alt Text](img/e7327bfe9e5e0459d12774e2d8a5734f.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--DGiPesYl--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/mz0mra28ysfra08d7xi6.png)

要添加快捷方式，点击"添加"并输入命令和组合键:

[![Alt Text](img/70a26e5458ef7bf867e2c28b999a9f0d.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--CONSJOfE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/dhr5hpanqi7zdg5rjvt8.png)

[![Alt Text](img/c9ea776db65b75f2476d22ae1218806b.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--EzqqBw58--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ijf3n8exs41sb5y6mbrs.png)

[![Alt Text](img/d63dc23776fca2a23aac4c1751630ab9.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--D5Gq04HO--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/z1a2dar6g1az5frimp6y.png)
