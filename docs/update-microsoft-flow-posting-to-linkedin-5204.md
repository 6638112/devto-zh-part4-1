# 更新:Microsoft Flow:发布到 LinkedIn

> 原文：<https://dev.to/barretblake/update-microsoft-flow-posting-to-linkedin-5204>

在之前的一篇文章中，我展示了如何在 LinkedIn 上发布一篇博文。当时可用的连接器使用 LinkedIn API 的 V1。那个版本不再有效了。令人欣慰的是，Flow 已经增加了对 LinkedIn V2 API 的支持，用 V1 取代 V2 只是简单的步骤交换。

帖子[更新:微软 Flow:发帖到 LinkedIn](https://barretblake.dev/2019/07/update-microsoft-flow-posting-to-linkedin/) 最早出现在 [Barret Codes](https://barretblake.dev) 上。
