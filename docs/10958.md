# 有时道路并不容易，但这使它更值得。

> 原文:[https://dev . to/copperbeardy/有时这种方式并不容易，但这使它更值得 27kf](https://dev.to/copperbeardy/sometimes-the-way-is-not-easy-but-that-makes-it-more-worth-it-27kf)

在我的第一篇文章中，我想分享一下我在成为一名开发者的道路上的最后几年，部分是为了让你更好地了解我。

[![](../Images/6e22add2989d82a47bccb64b33a14c50.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--5CuhI13t--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/m45ckr3r3cdsb1f6wmxl.png)

今年 41 岁，是我攻读计算机科学学位的第八个年头。7 年前，我在[开放大学](https://www.open.ac.uk/)
获得了第一个学位，这是一所很棒的大学，对我来说是一个很好的匹配，不仅因为它给了我很大的支持，而且它允许我以一种可以控制我的焦虑和抑郁的速度学习。

我不会骗你，获得学位并不容易，但如果是这样又有什么意义呢？

有很多次，我觉得自己做不到，只想放弃。但在学校和家人的大力支持下，我坚持了下来。2016 年 12 月，我以 2:1 的成绩获得了计算机与信息技术(荣誉)理学学士学位。你可以想象，我欣喜若狂，为自己感到骄傲。

我休学了一年......算是吧。我变得有点焦躁不安，并且被学习的错误深深地伤害了。
[![](../Images/b1a2a4114efc1803fdc051b72298100d.png)T3】](https://res.cloudinary.com/practicaldev/image/fetch/s--L8u6BKZy--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/b5fs4m0fa4yhippepxqh.jpg)

当我在读本科学位时，我在业余时间学习在线课程。但在间隔年，我超负荷地学习了相当多的在线课程，从哲学到 ASP.NET 核心身份。我仍然在攻读下一个学位时学习这些课程，迄今为止已经完成了 146 门课程。同时，在此期间，我创建了一个 Xamarin。表单应用程序，并将其放在完全无广告和无成本的  上。作为对自己的证明，我能做到。

完成本科学位一年后，我决定再次冒险，继续在赫特福德郡大学攻读计算机科学硕士学位。我目前仍在攻读这个学位，我最近开始了我在 MSC 的顶点项目，该项目旨在调查和演示 Azure maps 服务在开发位置感知系统方面的准确性和可用性。我应该在 2020 年中期完成这个学位。

我觉得所有这些研究。这可能有点过火，但让我对软件开发的其他方面以及其他人的角色和对他们的要求有了更好的理解。它还帮助我更好地关注和处理我的心理健康问题。
[![](../Images/6954f81a8447f6cf27f62a0484ffa2cf.png)T3】](https://res.cloudinary.com/practicaldev/image/fetch/s--rohVRGNS--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/03a2kgh6v4epu5fx2z6b.png)

在学习期间，我发现了一些有用的资源，并把它们放在了 Github 的页面上。

不幸的是，尽管进行了所有这些研究，我仍然没有工作，但我的情况很困难，我的焦虑仍然留在家里。尽管如此，我还是会继续学习新的东西，我相信我最终会找到一种方法来运用我的技能。

你应该从中吸取什么？

*   多学习可以让你有更广阔的理解。
*   学习更多永远不晚
*   心理健康没什么好羞愧的。
