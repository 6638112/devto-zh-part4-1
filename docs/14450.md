# 全新的 IDE v2.7.0 已经发布

> 原文:[https://dev . to/John found/fresh-ide-v2-7-0-已发布-5h5n](https://dev.to/johnfound/fresh-ide-v2-7-0-has-been-released-5h5n)

[全新 IDE v2.7.0](https://fresh.flatassembler.net/index.cgi?page=content/2_download.txt) 已经发布。

Fresh IDE 是用于汇编语言编程的高级 IDE(FASM)。

它完全是用汇编语言编写的，自托管，特别是为非常大的汇编语言项目的舒适编程而设计的。

当前版本重点介绍:

*   此版本的主要变化是 FreshLib 最终过渡到新的 GUI 架构。没有完成，但现在走上了正确的道路。

*   FlatAssembler 编译器已更新至版本 1.73.12。

*   编辑器的语法突出显示得到了改进，以支持用户定义的用于着色的关键字。

*   编辑器主题中增加了两种内置字体，特别适合汇编编程。

*   Windows 10 上提供了更多扩展测试。因此，编辑器的渲染和整体稳定性得到了改善。

*   包含的示例已经过修改和修复，可以使用最新的 FreshLib 库进行编译。

*   当然，通常的 bug 修复和新的 bug 引入也已经完成。
