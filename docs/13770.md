# 科特林中的关键词

> 原文:[https://dev.to/jay_tillu/keywords-in-kotlin-4116](https://dev.to/jay_tillu/keywords-in-kotlin-4116)

*   关键字是预定义的保留词，在语言中有特殊的含义。

*   通常，建议不要将其用作标识符。

*   但是在 kotlin 中，有一些关键字可以在某些情况下用作标识符。

*   kotlin 中基本上有四种类型的关键字:

*   硬关键字

*   软关键字

*   修饰符关键字

*   特殊标识符
