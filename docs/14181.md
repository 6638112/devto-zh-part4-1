# 超越薪酬(我的观点)

> 原文：<https://dev.to/ballen2713/looking-beyond-the-pay-my-opinion-2g2f>

在当今时代，软件工程的兴起已经成为高需求，比以往任何时候都更具竞争力。计算机科学领域的每个人都在为工作而奋斗，其他人则努力在任何地方工作以获取经验。

我在自己和周围的人身上注意到的一件事是，计算机科学领域的人有这样一种观念，即寻找支付巨额起薪的顶级公司，然后向他人吹嘘，好像这是一场获得最高工资工作的竞争。我不知道为什么，但这让我有几次感觉，因为大多数人看不起其他人，因为他们进不了顶级公司或拿不到高薪。这改变了我的想法，让我开始只关注薪酬，而不是其他。这种想法并不理想，也不应该适用于其他人。

最近，我和一位主管谈到，在一家公司，寻找最高工资并不总是一切。看看公司给世界带来的影响，看看你的软件或解决方案可能如何影响他人，这是评估你在公司的时间是否有价值的更有意义的方式。

显然，有软件公司做低球软件开发人员，这是不行的，但有许多公司，我见过的薪酬很可观，但气氛就像有第二个家庭。对我来说，这比一开始就有 6 位数的收入更有价值。我是一个努力工作的人，从最底层开始，然后一路奋斗到最高层！

话虽如此，但对于那些不愿接受起薪较低的工作机会的人来说，你应该给他们第二次机会，这样你就能更好地评估公司的价值，并有更好的机会随着时间和经验的积累而成长。(仅供参考，显然不是公司低球你，但开始相当不错的 lol)
