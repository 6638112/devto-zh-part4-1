# 从 C++到 ASP.NET 核心的学习之路

> 原文:[https://dev . to/alexo educative/a-learning-path-from-c-to-ASP-net-core-1e 62](https://dev.to/alexoeducative/a-learning-path-from-c-to-asp-net-core-1e62)

我的一个联系人最近问了我一个问题。他们想知道，如果他们知道一些 C++但还不熟悉，如何学习使用 ASP.NET 核心 2.1 进行 web 开发。NET 框架。他们问我是否可以为他们建议一条可能的学习路径。因此，我联系了我团队中的一些开发人员:*如果有人懂 C++，那么他们会做些什么呢？网*？

我的一个队友建议如下。

## [](#c-gt-c)C++ - > C

虽然。NET 框架支持多种语言，C#是使用最广泛的语言。例如，ASP.NET 是一个专门用于构建 web 应用和服务的框架。NET 和 C# 。因此，学习 C#编程的基础知识以及 C#与 C++的不同之处是必不可少的第一步。

## [](#web-development)网页开发

然后，学习者还应该花时间学习 web 开发:客户端框架、服务器端框架、超文本标记语言(HTML)、级联样式表(CSS)、JavaScript 如何与 HTML 和 CSS 一起用于构建 web 页面，以及相关的 web 开发概念共同形成了一个知识层。特定于网络的学习。

## [T1。ASP.NET->->ASP.NET 核心网](#net-gt-aspnet-gt-aspnet-core)

奠定了 C#和 web 开发基础的初步基础之后，花时间了解一下。NET 框架，以及 ASP.NET 的使用细节。后者是一个服务器端框架，它本身是。并且是/曾经是 ASP.NET 核心的祖先[。这里要记住的关键是有许多*框架和库可供使用。如果你开始了解所有这些，它会变得令人不知所措。一个人刚开始。希望用这个框架构建 web 应用的人应该关注如何使用 ASP.NET 核心，但不要太深入到。现在是 NET vs ASP.NET。*](https://en.wikipedia.org/wiki/ASP.NET_Core)

在熟悉了如何使用 ASP.NET 核心构建 web 应用程序的基础知识，并学习了教程之后，即使是简单的教程，如微软的[“ASP。NET 教程-10 分钟后的 Hello World "](https://dotnet.microsoft.com/learn/web/aspnet-hello-world-tutorial/intro)，ASP.NET 核心的新来者应该知道足够创建一个非常早期的地图，他们可以用它来确定下一步转向哪里。

所有伟大的旅程都是从某个小地方开始的。我希望这能帮助我的联系人，也能帮助你，读者！
