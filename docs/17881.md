# 在环城工作，第二天:善待来往车辆

> 原文:[https://dev . to/ashleemboyer/work-in-circle-city-day-2-3 PCC](https://dev.to/ashleemboyer/working-in-circle-city-day-2-3pcc)

第二天相当平静和顺利，但仍然令人兴奋。我参加了两次董事会议，下周每天都有一两次。这是更多了解公司的起步、运作方式和公司价值观的机会。这也是一个很好的机会去认识来自不同团队的人，这很酷！每个人都非常友好。

我尝试了另一个新的(对我来说)食物的地方！是 Qdoba 和 Chipotle 的杂交品种。我真的很喜欢在一个大城市的中心工作所需要的步行。说到这里:我买了一张停车月票！这次步行的距离是今天和昨天的两倍，但这正是我的目标。由于我的背不好，走路是我重新开始锻炼的唯一方式之一。😊

我想说这一天唯一不好的事情是回家路上的交通堵塞。我听说虽然这个州的交通不像人口多的城市那么糟糕，但人们非常粗鲁，充满了路怒。我早就知道会这样，但还是很恶心。我希望更多的人能善良一点，意识到每个在路上的人都有一个故事。

敬请关注第三天！我将开始更仔细地研究代码，所以事情会变得更加有趣。😀
