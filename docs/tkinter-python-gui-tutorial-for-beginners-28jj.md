# Tkinter Python GUI 初学者教程

> 原文：<https://dev.to/learntocodez/tkinter-python-gui-tutorial-for-beginners-28jj>

本视频将详细演示如何在 Tkinter 中创建 GUI 应用程序。
那么什么是 Tkinter 呢？Tkinter 是 Python 附带的 Tk GUI 工具包的 Python 接口。在 Python 中，你可以使用 Tkinter 模块来创建简单的 GUI 程序。Tkinter 并不总是在 IDLE 下可靠地运行。使用 IDLE 的编辑器编写 GUI 程序，但是为了获得最佳结果，可以从 OS 命令行运行程序。
通过简单易行的步骤学习 Tkinter。
