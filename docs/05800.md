[@healeycodes](https://dev.to/healeycodes) 回复分享一个 regex 巫师的心路历程:

[![healeycodes profile image](../Images/095869ca524b6d8703efab6994f7987b.png) ](/healeycodes) [ Andrew Healey ](/healeycodes) • [<time datetime="2019-08-26T18:46:43Z" class="date-short-year"> Aug 26 '19 </time> • Edited on <time datetime="2019-08-27T07:33:13Z" class="hidden m:inline-block date-no-year">Aug 27</time>](https://dev.to/healeycodes/comment/eh26) 

在以前的工作中，有一个正则表达式向导。一切都很好。他很聪明。我们可以去找他，向他提出一个问题，他会在一些纱线上写下解决方案。

在他离开后不久，我发现自己遇到了一个 regex 可以解决的问题。我被留在了谷歌。所以我黑了。我复制、粘贴并使用正则表达式助手网站，在中国的字符串解析商店里到处跑，一路上粉碎一切。

我当时并不知道，但当我一起搜索和破解一个解决方案时，我学到了一些东西。一些零散的建议让我难以释怀。偶尔，我会不知不觉地发现自己在一个科技博客上解释搜索模式的复杂性。

后来有一天，有人看到了我的一些不同之处，他找到我，要求一个正则表达式的解决方案。不知不觉中，我成了巫师。一个几乎一无所知的专家。

从一开始就学习理论会更快！😊

作为一名自由职业者，我如何与 Wix、SquareSpace 和 WordPress 竞争？ thread， [@jamesthomson](https://dev.to/jamesthomson) 提供了一些关于开发者在所有这些自助建站平台中的机会的额外想法:

[![jamesthomson profile image](../Images/1a3fbce492a025d8e19221f7c9040cef.png) ](/jamesthomson) [ James Thomson ](/jamesthomson) • [<time datetime="2019-08-26T06:48:16Z" class="date-short-year"> Aug 26 '19 </time>](https://dev.to/jamesthomson/comment/egc7) 

实际上，我一直把这看作是积极的一面。开发人员可以专注于为真实的企业解决真实的问题，而不是盲目地创建模板化的 Wordpress 网站。作为开发者，我们可以专注于真正驱动我们的东西，并进一步推动 web。提到的产品可能会帮助小企业获得网络存在(这是伟大的！)，但任何需要定制工作的东西(中型+企业，许多创业公司，政府等)。)千篇一律的产品根本不可能与真正的开发者竞争。

在过去 15 年左右的时间里，我一直以这样或那样的方式在这个行业工作，但我从未见过像现在这样对前端和后端 web 开发人员的需求。我认为只要你能把自己的技能发展到那些需要定制工作的行业，你就会有很多工作等着你。

最后，在 **[如何克服工作中的不良时期。来分享一下我们的经验](https://dev.to/gabriela/how-to-overcome-bad-periods-at-work-let-s-share-from-our-experiences-45mo)** ， [@andrewjensen42](https://dev.to/andrewjensen42) 分享一些从业 20 多年的圣人智慧:

[![andrewjensen42 profile image](../Images/5eec9cace6c241dcacc7089096b82360.png) ](/andrewjensen42) [ andrewjensen42 ](/andrewjensen42) • [<time datetime="2019-08-23T10:41:25Z" class="date-short-year"> Aug 23 '19 </time>](https://dev.to/andrewjensen42/comment/eedg) 

自 1998 年以来，我一直从事编码工作，不用说，在我的职业生涯中，有几次我觉得我的环境已经变得有害或者简直无法忍受。在这一过程中，我学到了一些简单的事情来预防和缓解这种情况。

首先，要明白你是一个个体。你不可能什么都知道。寻求帮助并不可耻。

其次，不管问题有多大，大多数事情都可以通过对话解决。如果你有不明白的地方，就去问。

最后，我不能过分强调这一点，你创造你自己的环境。如果你一直脾气暴躁，那是你的环境。互动决定了你的职场成功。
