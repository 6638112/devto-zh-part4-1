# 提升您的终端/命令行导航

> 原文:[https://dev . to/gander ZZ/leveling-up-your-terminal-command-line-navigation-5e 69](https://dev.to/ganderzz/leveling-up-your-terminal-command-line-navigation-5e69)

使用终端的一个激励因素是生产率的提高...也就是说，一旦你能像 80 年代的金属吉他手一样扫键盘。获得一些基本的键盘快捷键/命令可以极大地提高终端的可用性。让我们来看看我最喜欢的几个。

*(在本文中，我将展示 Git Bash，但是这些命令在大多数终端中都有效)*

### [](#movement)运动

**移动到终点，输入**
`ctrl + e`

[![Move to the end of the input](../Images/8991b6953ce74310e4dfa884b5cc23d1.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--bd70PHf1--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/eqz7oivtmwvm2lecozxv.gif)

**移动到输入**
`ctrl + a`的开头

[![Move to the beginning of the input](../Images/4daf081b7e30fbcee331d3fbc9aadf48.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--OAgUbb1T--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/pcviwamsu92rsbvv1rl6.gif)

**在每个‘字’之间移动**
`ctrl + [arrow left/right]`

[![Move between each 'word'](../Images/624908662d3881b76cec0ed28a0b5ee5.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--mpYj9wNW--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/5piz8q8un1bk2ui1fw3r.gif)

### [](#deleting)删除

**向后删除一整‘字’**
`ctrl + w`

[![Delete backwards a whole 'word'](../Images/a9b05d56ab697df0b6f23519647909b0.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--kkwY97Rj--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/5fhignw1fjziw7x72jjh.gif)

**删除一切**
`ctrl + u`

[![Delete everything](../Images/c78abe124083310e1b3ea42870d828e2.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--imMmWsAi--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ijiermx0qon7zuea799t.gif)

### [](#open-file-explorer)打开文件浏览器

在当前位置打开文件浏览器会非常有用。该命令因操作系统而异。

(在当前目录下打开文件资源管理器。更改“.”去任何地方！)

**Windows:**`explorer .`
**Unix/MAC OS:**`open .`

[![Open File Explorer](../Images/c283847ac0f1ebc15ec0d84fe3bf6208.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Ju6-K07I--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/0xcgy3c1453zwizjrhaq.gif)

### [](#history)历史

**重新运行最后一个命令**
`!!`

[![Rerun the last command](../Images/9dc895ca154371e20152d1dcfb7879ae.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--zhwpre7---/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/sqofemwp8trpnsf4pwss.gif)

**获取命令的全部历史**
`history`

**从历史中重新运行命令**
`![#number]`
