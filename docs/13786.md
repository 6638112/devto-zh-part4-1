# 科特林的 Hello World

> 原文:[https://dev.to/jay_tillu/hello-world-in-kotlin-3kjf](https://dev.to/jay_tillu/hello-world-in-kotlin-3kjf)

***《你好世界》*** 是每种语言的介绍性节目。通常，这是介绍这门语言的最佳方式。它展示了语言的基础，如语法、基本工作原理和功能等。在这里，我向你展示科特林的 Hello World。

```
fun main(args: Array<String>) {
    print("Hello World")
} 
```

*   **`fun main (args: Array<String>) {…}`** —这是一个`main`函数，在每个 kotlin 应用程序中都是必需的。编译器从`main`函数开始执行。你也可以称之为每个 Kotlin 程序的**入口点。这个函数接受一个字符串数组作为参数。**

*   **`fun`**——`fun`是一个关键字，用来表明这是一个函数。在这种情况下，`main`是一个函数。

*   **`print ()`**—`print()`函数是一个内置的函数，它打印其中的值。您可以打印数值、计算结果等。

*   **`println ()`**—`println()`函数的工作方式与 print()相同，但它也在输出后打印一个换行符。所以如果你有下一个输出，它会转到下一行。如果什么都不需要打印，它将打印一个空行。许多传统语言如 C 和 C++都缺少这个功能，在这些语言中我们仍然使用`\n`，尽管它非常方便和有用。

*   您可以猜到，分号是可选的。编译器足够聪明，能够理解换行符是语句的结尾。

所以，伙计们，这就是你好世界。如有任何疑问，请随时询问。
