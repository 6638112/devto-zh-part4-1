# 学习 JavaScript 的 5 个秘密

> 原文:[https://dev . to/realtoughcandy/5-secrets-to-learning-JavaScript-2lk 8](https://dev.to/realtoughcandy/5-secrets-to-learning-javascript-2lk8)

## [](#learning-javascript-is-tough)学习 JavaScript 很艰难。

大师们让它看起来很容易，但是花五分钟研究继承和原型链，你会很容易发现自己深陷在一堆热气腾腾的二进制文件中。

幸运的是，在我学习 JavaScript 的这些年里，我学到了一些生活窍门，它们真的对我的知识很有帮助。我今天在这里分享它们是为了帮助那些可能被这种疯狂的语言淹没的人。

### 这里是我学习 JavaScript 的 5 个秘密。

1.有香草味的 JS…还有其他所有的。

编程的世界实际上是无限的。所以你在 JavaScript 变量和对象的孩子池中闲逛，想着，“好的变量和原型继承是好的，但是我什么时候能得到酷的东西反应和节点和 HTTP 请求，就像我听说 MONGODB 很酷一样……”

哇哇哇哇哇！我知道渴望得到真正有趣的东西，我在那里。但是理解所有 JavaScript“分支”的唯一方法是理解 JavaScript。

无框架、不支持 jQuery 的老式普通无麸质 JavaScript。

这种诱惑似乎不可抗拒，但是尽量把你的目光集中在 JavaScript 生态系统的核心。这不仅是学习语言的一条坚实的道路，而且在学习从 NoSQL 数据库到运行时环境(如 Node)的其他东西时，也将为您的成功打下基础。

2.马克·迈尔斯。认识他。爱他。

我真的相信 Mark Myers 的《学习 JavaScript 的更聪明的方法》是唯一一本适合 JavaScript 初学者的书。

大多数其他作者在教授这种语言时完全失控，在 10 分钟内抛出各种疯狂的概念。

用更聪明的方法，马克一次给你一个题目。最好的。书。永远不会。每章通常有两三页，字体很大。

[![A Smarter Way to learn JavaScript](../Images/ecd04c9ba46034e73e57126418686abe.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--tRV_szRe--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://images-na.ssl-images-amazon.com/images/I/512KPmZIG7L._SX348_BO1%2C204%2C203%2C200_.jpg)

所以你读了这个概念，然后你去他的网站，做与主题相关的练习和编码挑战。这本书获得疯狂的五星评论是有原因的，因为他的方法很有效。

如果学习 JavaScript 对你来说简直是地狱之旅，那么试试更聪明的学习 JavaScript 的方法。说真的。

3.十分钟就够了。

我们都有自己的生活和责任，我们中的许多人没有一个晚上花四个小时来研究 JavaScript。没关系！你能抽出十分钟吗？如果是这样，那么你已经开始学习 JavaScript 了。

我喜欢用一个缩写词——ABC 或 Always Be code。试着每天找一点点空闲时间拿出你的代码编辑器来练习。即使只有十分钟也会很有帮助。即使你没有键盘，也要在心里练习 JavaScript。

想想所有你有机会这样做的地方，无论是堵车，在杂货店排队，还是其他任何地方。社交活动可以等等！只要有机会，就练习你的基础知识，学习 JavaScript 会变得更容易。

4.普通的 JavaScript 项目是黄金。

查看大多数 Udemy web developer 项目，它们大多数都涉及 JS 库或框架。甚至许多像 TODO apps 这样的初学者项目也包含了一个像 jQuery 这样的库。尝试使用普通 JavaScript 构建几个项目。

这将给你语言的实践经验，并向你展示事物是如何结合在一起的。

5.**人有力量。**

你参加过 JavaScript 或其他 web 开发者会议吗？没有吗？为什么不呢？如果你住在城市，很可能会有迎合我们职业领域的聚会。这些地方到处都是忠诚于党的人才。尽管有软件坏脾气的刻板印象(感谢 Stack Overflow 确认了每一个负面的编程个性刻板印象)，还是有很多友好的人想要帮忙。

聚会是结识 JavaScript 人的好地方。无论你是第一次学习 JavaScript，还是已经尝试了很多次，Meetups 都是认识懂 JavaScript 的有用的人的好地方。

去参加聚会。我很害羞。你很害羞。大家一起害羞吧。但至少我们可以相见，在世界上最被误解的语言上变得更好，不是吗？

### [](#to-summarize-learning-javascript-is-hard-it-may-be-the-most-challenging-thing-youve-ever-done-and-the-thing-is-you-never-actually-stop-learning-javascript-so-dont-beat-yourself-up-if-the-concepts-arent-sticking-right-away-this-is-a-tough-language)总结一下，学习 JavaScript 很难。这可能是你做过的最有挑战性的事情！事实上，你永远不会停止学习 JavaScript。所以，如果概念没有马上被记住，不要责备自己——这是一门难学的语言！

你可以做的最有效的事情是开始并坚持使用普通的 JavaScript，如果基础知识对你来说很难，向马克·迈尔斯介绍自己，ABC(总是编码，即使每次只有 10 分钟)，出去见见 JavaScript 人，你会及时看到结果。
