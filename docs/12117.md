# 软件腐烂

> 原文：<https://dev.to/beardedpayton/software-rot-43d9>

你已经建立了最新最棒的网络应用程序，用户正在到处注册。现在是时候坐下来看着钱进来了，对吗？

也许吧...

## 什么是软件腐烂？

软件腐烂是指你的软件随着时间慢慢“死亡”,变得有缺陷、不可用并且难以维护。

我们的软件实际上并没有腐烂，也没有在每次打开代码编辑器时开始发臭。但是久而久之如果不好好对待它，它就会慢慢变得不能用了。

## 防止软件腐烂

软件腐烂可能有多种方式。但我想谈一个我们都能从中受益的特定话题。

### 留下比你发现的更好的代码

破窗理论
研究表明，家中的一扇破窗会导致一种被遗弃感。不知不觉中，所有的窗户都被打破了，到处都是垃圾。

你的代码就是那个家，不要在里面留下任何破损的窗户。不管是不是你写的，让代码保持比你发现时更好的状态。不断改进你的代码库，即使这感觉像是浪费时间。不要偷懒，只是让它工作，像写最后一行代码一样写每一行代码。

我可以谈论很多其他的话题，但是我们现在就可以开始讨论这个话题！向前迈进，像对待自己的家一样对待代码:)
