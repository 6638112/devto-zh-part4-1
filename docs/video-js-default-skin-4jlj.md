# Video.js 默认皮肤

> 原文：<https://dev.to/devjoo/video-js-default-skin-4jlj>

这是 Video.js 的基础皮肤，可以修改它来制作自定义皮肤。

关于 Video.js 皮肤的伟大之处在于它们可以在 HTML5 视频和 Flash 中工作！
