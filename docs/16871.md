# 个人开发者&生活项目管理

> 原文：<https://dev.to/tomasforsman/solo-developer-life-project-management-11bd>

TL；你会为独立开发者推荐什么样的项目管理工具？
我所说的工具是指软件、硬件和系统。

* * *

*   我目前正在重建我的开发网页和我的更一般的网页。
*   我(慢慢地)独自开发一款游戏，同时尝试一些不同的游戏理念/机制。
*   我正在和一个朋友一起开发一个组件库。
*   我把所有东西都放在本地服务器上，包括 git 存储库和我自己的网站，以此来提高我的开发运营技能。
*   我还学习了更多关于 React、Web 组件、ES6、C#和 C 开发的知识。
*   我在用树莓酱和时间追踪装置为我儿子做一台收音机。
*   我也在慢慢扩展我的智能家居，寻找新工作，在一家开发公司工作，制造家具，计划去苏格兰度假，写一本书，成为一个更健康的人，正如你所想象的，我不确定我忘记了什么。

所以，你知道，我在生活。

我相信单任务而不是多任务，但是，特别是当涉及到开发项目时，我越来越难以跟踪我面前的所有单个任务。我经常需要处理许多不同的想法，当我专注于一个项目时，我经常记得我应该在另一个项目中完成一项任务，但我忘记了，当我完成当前任务时，我又忘记了。

你会为独立开发者推荐什么样的项目管理工具？

我所说的工具是指软件、硬件和系统。
