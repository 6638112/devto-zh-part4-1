# 具有加载特性的 Opencart 和 Magento 多供应商市场扩展

在任何电子商务网站上，客户都会从网站所有者列出的网站上购买产品。在这种类型的结构中，业务仅是 B2C(仅业务 2 客户)。其中网站的管理是允许网站上的卖家产品和客户购买它的所有者。
市场上有许多被称为 CMS(内容管理系统)的开源电子商务平台，这使得管理员可以轻松管理整个管理流程，即管理员不必编写代码来销售产品。管理员必须有一些技术知识，但编码在这里并不重要。他只需要使用 CMS 的预定义功能，这使得管理员很容易销售和管理产品。

电子商务行业中有两种最广泛使用的 CMS，即 Magento 和 Opencart。使用这两个 CMS 管理员有更多的控制销售，订单，付款等。只需在您的服务器上安装 CMS，并开始在网站上列出产品。在这里管理员必须购买域名，托管，CMS 的安装等。

然后是另一部分，管理员只是想成为卖家和客户之间的中介，即管理员创建了整个平台，然后是卖家问管理员，他们想在他们的网站上销售他们的产品。卖方不想沉迷于任何网站制作和安装 Magento，服务器上的 Opencart 和所有。卖家有一些有限的产品，或者也可以有一千或十万的产品，但只想在著名的电子商务网站上销售他们的产品。Purpletree 多供应商的需求来了，它使大量的特性成为可能。

purple tree multi vendor for open cart 和 Magento 提供了大量的功能，使任何网站所有者都可以在网站上注册为卖家，并立即开始销售产品，无需创建任何网站，购买托管服务器，以及大量其他技术事情。所有这些都是管理员的责任，他必须使用如上所述的 Opencart 或 Magento 创建一个网站，然后添加多供应商市场的功能。只需在您的网站中安装用于 Opencart 或 Magento 的 Purpletree Marketplace。
