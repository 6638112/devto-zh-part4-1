# 美式足球如何让我成为更好的开发者

> 原文:[https://dev . to/PEM tajo/how-American-football-make-me-a-better-developer-47he](https://dev.to/pemtajo/how-american-football-makes-me-a-better-developer-47he)

在我开始攻读计算机科学本科学位之前，我从未听说过美式足球。

我第一次独自生活在另一个城市，在一个新的地方做一个新人，所以当一个朋友问我是否想开始在新的足球队训练时，我想“好吧，为什么不呢？”这改变了我的一生。

## [](#why-changed)为什么变了？

我在 Maringá Pyros 培训了五年。在队里，我是一名 RB / WR，之后，我做了一年的足球裁判，几年后，我成为了巴西一家当地电台的体育评论员，在那里我学到了很多东西，这些东西对我的职业生涯有所帮助。

# [](#workgroup)工作组

足球是一项集体运动，团队合作对工作至关重要。总的来说，训练、协调和调动整个团队是至关重要的，这样才能使比赛成功，从而赢得比赛。我发现这很神奇，就像在足球中一样，在软件团队中，每个人都必须团结一致以确保操作的成功。

# [](#motivation)动机

这很简单，如果你是一个 80 公斤的人，球在场地中央，而赌比赛的唯一方法是击中一个 150 公斤的人，你会怎么做？你停下来了吗？你哭了？你是放弃还是回来？没有这样的选择，唯一的方法就是前进，整个团队信任你，整个团队都在尽自己最大的努力，所以你可以做到最好，你所要做的就是全力以赴。

最后一种情况如何应用于软件团队？原理是一样的。如果你遇到问题，你不能就此停止并放弃。你不能丢下一切，和你的团队一起失败。你需要信任站在你这边的团队，你必须尽最大努力共同完成任务，让事情顺利进行。

# [](#competition)竞争

最大的竞争不是和其他同龄人的竞争，最大的竞争是你和你自己的竞争。你应该每天都寻求自己的最好，每天都寻找自己的弱点并让它们一天比一天好，每天都寻求自己的优势并训练自己的技能。你必须付出你所拥有的一切，你生命中的每一秒，每一刻。

但这里有一个重要的提示，我不是说耗尽你的大脑和超越你的极限，我是在告诉你要照顾好你的大脑和你的肌肉能力，它们需要休息，需要良好的营养，也需要良好的训练。拥有一个运动员的心态是很重要的，因为要学会尊重自己的极限并提高自己。

# [](#leadership)领导

我不是领导者，但我总是希望鼓励和激励我所在的团队。有时我们需要给我们的团队注入动力，我们需要每个人都继续战斗，有时你可以成为你的团队不放弃的决定性因素。即使不是领袖，我也总是愿意拥抱每个人，帮助每个人。在办公室和在副业上并没有太大的不同。边线更加情绪化，我们可以尖叫，鼓励和帮助尖叫底部的每个人。在办公室里，你不能大喊大叫来激励别人。但重要的是，当你的同事不得不处理一个持续两周的 bug 时，你也要做同样的工作，鼓励他，鼓舞他的士气。
我擅长？没有，但我一直在努力。

# [](#communication)沟通

这是我在生活中必须提高的一项新技能，我从来都不是一个好的沟通者，我已经接受了提高它的挑战。当我开始从事体育评论员的工作时，我的演讲能力有了很大的提高。在我做的第一个音频中，我需要写和读，这样我才能录音。然而，它花了几个音频征服或多或少我认为可以接受的。我不能同时思考和说话。需要大量的训练才能变得更好。

现在，我正在显著地改善我说话的方式和语调，我并不完美，但现在我相信我正在一点一点地比昨天更好。这也对我的工作产生了影响，说话清晰，语调清晰，并且能够将我清楚地表达的内容传递给其他人，这是一项在工作中产生巨大影响的技能。对我来说，这是一个伟大的胜利。

# [](#responsibility)责任

我和我的团队失败了几次，三次我有球赢得比赛，但我失败了。我已经让所有信任我的人失望了，我也从来没有把我的错误归咎于别人。

没有借口，不管是什么原因，如果我失败了，我会为此负全责。

这是另一个美式足球的教学，对我的生活产生了很大的影响，因为原则是一样的，你永远不应该让别人为你的错误负责。我更喜欢晚上睡个好觉，知道自己做了正确的选择，知道自己可以改进，让这些错误不再发生。

# [](#the-game)游戏

这项运动令人难以置信，你学得越多，你就越需要学习。在几毫秒的压力下做出正确的决定，保持冷静是优秀球员与不可思议球员的区别。

与发展平行，这项技能使你即使在天空着火的时候也能做好工作。

所以我写这些话是想告诉你，其他与工作本身无关的活动可以让你成为更好的开发者。如果你有其他的爱好，提高了你的开发者工作，在下面放入评论，我们来谈谈。

干杯。
