恢复 Apache NiFi 注册表流文件。

**创建流程**

邮政

/buckets/{ bucket id }/flow

创建一个流

使用 bucket id，通过 POST 上传一个流，并获得一个新的流 id

创建流程的版本

用桶 id 和流 id 发布

/buckets/{ bucket id }/flows/{ flowId }/versions
