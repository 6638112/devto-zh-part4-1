# What is Kubernetes?

> 原文:[https://dev.to/educostadev/what-is-kubernetes-4h81](https://dev.to/educostadev/what-is-kubernetes-4h81)

Kubernetes(简称 k8s)是一款用于部署、管理和扩展容器的开源编排软件。当应用程序(微服务)及其依赖项和配置打包到一个容器中时，需要一个工具来根据应用程序访问需求管理和自动扩展容器。Kubernetes 是实现这一目标的合适工具。

随着应用程序的增长和部署在多个服务器上，Kubernetes 可以控制这些容器的运行方式和位置。它可以轻松地水平扩展您的应用程序，同时监控和维护容器的健康状况。

使用 Kubernetes 还为您带来了可移植性，您可以将容器化的应用程序从本地开发、QA 或 stage 机器部署到生产中，并确保运行相同的测试版本。该工具还拥有大量的扩展和插件，带来了更强大的功能。
