# 无头网页设计的战略优势

> 原文:[https://dev . to/stolleydotdev/the-strategic-advantages-of-headless-web-design-jkn](https://dev.to/stolleydotdev/the-strategic-advantages-of-headless-web-design-jkn)

Gal Shachar 正在提供关于无头内容管理系统的硬销售:

> 一个真正的无头 CMS 完全去掉了头部，只留下了后端。换句话说，没有专门的前端展示系统。虽然你最初可能想知道这种类型的结构是否会对你不利，但它实际上是在所有设备和界面上向你的网站访问者显示内容的最佳方式，使你的机构处于最有利的位置。

一个很好的面向 CMS 的替代网络策略的概述。但是 headless 为与网络完全分离的内容带来了新的问题。HTML 有工作要做，而且做得很好。
