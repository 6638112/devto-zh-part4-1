这里有一个窍门:如果你让人们容易接受采访，他们会非常愿意接受采访。

渐渐地，我添加了更多的“网络”类功能，但只是因为它对日常用户的数量有意义。除了我自己的博客之外，早期有太多的功能是没有意义的，但我只是一次添加一件东西。即使在最好的情况下，这也需要很长的时间，所以这只能是一个单独的项目，直到它看起来有希望。

然后我找到了 [@jess](https://dev.to/jess) 和 [@peter](https://dev.to/peter) 共同创立了这家公司，当时看起来这确实是一个很好的想法，但这距离它成立已经有两年多了。这也是进化的一个要素。这两个人给公司带来了非常不同的、互补的技能。每当我们有机会扩展业务时，我们都会寻找能带来新观点和新技能的人。

从那时起，我们凭直觉寻找有趣的方法，让这个平台对更多人更有吸引力。但这一切都是因为你觉得，让你的朋友和家人真正使用你自己制作的东西已经够难了。很难改变人们的习惯，所以你必须真正致力于观察人们对事物的反应，对不同的用例有同理心，并在项目上投入大量时间。

无论如何，这是对我们有用的。我认为每种情况都有不同的机会，但是从某种程度上“单人模式”开始并根据需要发展的想法似乎是一种可重复的模式。DEV 是一个多方面的想法市场，最初是单向流动的。任何市场都可能有类似的动态。
