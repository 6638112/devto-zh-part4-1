***“不要第一个说一个数字。”*T3】**

自从我成为软件工程师以来，这一直是薪资谈判的主流建议。避免在潜在雇主告诉你他们想付多少钱之前告诉他们你想要多少钱。潜在雇主首先想要你的号码，以压低你的身价。你想让他们的数量最大化你的补偿。如果公司神奇地给你提供了比你要求的更多的东西，你就“赢了”

但你不是可以以最低价格出售的可替代商品，潜在雇主不应该被骗去支付你合理的工资。这是一场零和游戏，只有自信的人才能生存。这是对许多已经被边缘化的候选人的非人化和不利。为什么我们不能诚实点*和*期望得到公平的补偿呢？

要求加薪会带来巨大的财务影响。根据拉米特·塞西的《加薪终极指南》&*中的计算，在你的职业生涯中，一次性加薪 5000 美元可以累积成几十万美元(甚至更多)。*一个被忽视的推论是，一年少拿 5000 美元*会让你付出同样的代价*。接受低工资是昂贵的。

雇主引诱应聘者“先说一个数字”的策略之一是询问薪水历史。如果一个潜在的雇主根据你的薪资历史给你一份工作，那 5000 美元的赤字会成倍增加，并且随着每次薪酬的增加而滚雪球。在你目前和未来的每一个工作机会中，你都应该得到纠正薪酬赤字的机会。T3】

这就是为什么像纽约州和加利福尼亚州这样的州禁止询问薪资历史和/或禁止雇主在考虑薪酬方案时使用薪资历史。你的工资应该基于当地同类工作的工资、你在该职位中的级别以及雇主对你的需求程度。法律或道德上不要求你向当前或潜在的雇主披露你的薪资历史，你也不应该这样做。

### [](#dont-answer-salary-history-questions)不要回答薪资历史问题。

就是不做。如果你住在禁止薪资历史问题的州或城市，那你就准备好了。然而，如果你生活的地方还没有禁止薪资历史问题，或者法律还没有生效，你仍然不必/不应该回答这个问题。以下是一些你可以做的事情:

1.  如果工作申请需要薪资历史记录，请填写 0 或 1 美元。不要说谎。输入一些明显意味着你拒绝回答的内容。如果你告诉这位潜在雇主，为什么他们应该把这个问题从他们的申请中去掉，这是加分的。
2.  如果有人问你，直接将话题转向市场利率。面试压力很大，如果你没有准备好，直接回答可能会更容易。提前练习一下你的回答，这样它就会自然而然地出现，你就不用去想它了。试试*“我不喜欢分享这些信息。这个职位在这个领域的工资是 x，这个职位的工资范围是多少？”*

当你去面试的时候，你应该知道在你的城市里人们做同样的工作能挣多少钱。在这篇文章的底部有几个地方你可以找到工资信息。

### [](#say-a-number-first%E2%80%8A%E2%80%8Aif-you-want)先说一个数字——如果你愿意。

如果你知道当地的工资数字，并且你想从一开始就更积极地参与薪酬讨论，你可以先说一个数字。不管你现在的工资是多少，你都可以说出你想要的数字。你可以通过招聘人员或者**来完成，你可以使用**[](https://hired.com/x/dk3GKA)**(充分披露:这是我的推荐链接)。**

 **Hired 是一个招聘平台，在这里“公司申请提前面试你的薪资细节。”因为你在前面列出了你想要的薪水，它首先会过滤掉那些不愿意给你足够薪水的公司，它会把你的期望和公司提供的东西放在前面。它不会束缚你——你以后还有机会协商更多。我在之前的两次求职中使用了成功录用。我有过两次不同的薪资谈判经历。

2018 年，Hired 帮我联系了 DC 一家旅游初创公司的人才总监 Hilliary Turnipseed(她现在经营自己的节目，并正在积极招聘，[告诉她是我让你来的](https://www.linkedin.com/in/hilliaryturnipseed/))。她没有问我以前的工资(虽然这是申请表上的必填字段，我输入了 0 美元)。她问我什么数字会让*让我兴奋*，不管我的雇佣档案上有什么。我的人才代表告诉我，我在个人资料中低估了自己。他们认为这会给我更多的选择。这比我目前的工资要低，我知道我必须谈判。问什么能让我兴奋让我很容易做到这一点(注意，招聘人员)。我解释了这一点，并要求更多。

上个月，我重返工作岗位，希望能重复我的经历。这一次，我在个人资料上写了一份令我兴奋的薪水。这是我一生中最短的一次求职，也是我从第一天开始的首选。到了讨论薪酬的时候，招聘经理承认这些对话让人不舒服(另一个给招聘人员的建议是，薪酬讨论不必是对抗性的)。我确认了我个人资料上的数字正是我想要的。完成了。

我喜欢我的新工作。在 Mix.com 找到我们。

无论你的舒适程度如何，你都应该得到公平的报酬。即使招聘经理/招聘人员不这样做，你也可以为自己的成功做好准备。了解这些数字，练习你的回答，记住你应该得到公平的报酬*和*来纠正薪酬赤字。你能行的。祝你好运！

不确定在哪里可以找到你所在地区的薪资信息？Glassdoor.com【Salary.com】和[雇佣状态工资报告](https://hired.com/page/state-of-salaries)是开始的几个地方。当你在做这件事的时候，读一读拉米特的整个 [*加薪终极指南&提高你的薪水*](https://www.iwillteachyoutoberich.com/guides/ultimate-guide-to-asking-for-a-raise-and-negotiating-salary/) 。

你是招聘经理还是招聘人员？检查你的工作申请，删除任何薪资历史问题。确保所有面试官都没有询问薪水历史。让薪酬讨论对候选人来说尽可能简单。对他们来说，感到完全舒适是对你最有利的，这样你们都可以得到你们想要的。**
