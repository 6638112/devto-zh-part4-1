# 如何修复 MacBook Pro 在获得 800%的 kernel_task CPU 使用率后速度变慢的问题

> 原文：<https://dev.to/katzueno/when-your-macbook-pro-got-800-kerneltask-cpu-usage-and-slowed-down-1lhn>

6 月的一天，我的 2015 年中期 MacBook Pro 开始变慢。当我查看活动监视器时，kernel_task 正在最大限度地使用它的 CPU。

我试着:

*   清空我的硬盘。
*   在安全模式下启动
*   重置 nvram

Ref: [如何修复你 Mac 的“kernel_task”高 CPU 使用率 Bug](https://www.makeuseof.com/tag/fix-mac-kernel-task-high-cpu-usage/)

什么都不管用，也没有提高我的 Mac 电脑的性能。

我的是 2016 年 7 月份买的。我已经用了 3 年了。或许，它的时代已经到来。

这不是真的。

我找到了最佳解决方案。

*   打开空调。
*   拿一个风扇，放在我的 MacBook Pro 旁边，打开它，指向 MBP。

就是这样。

夏季开始了。温度上升。我只是需要给我的 MBP 降温。
