# macOS Catalina 将 zsh 作为默认 shell

> 原文：<https://dev.to/siddb/macos-catalina-to-have-zsh-as-default-shell-2014>

苹果的#WWDC2019(全球开发者大会)于 6 月 3-7 日举行。macOS 的下一个重大升级将被称为 Catalina，随之而来的是开发者的巨大变化。

默认情况下，到目前为止的 macOS 版本都使用 bash 作为默认 shell。macOS Catalina 中的新用户帐户将 zsh 作为默认 shell。Zsh 是一个非常受欢迎的开源 shell，具有大量的特性。

Zsh 在开发社区中非常受欢迎，也可以安装在 Catalina 之前的 macOS 中。Zsh 和 ITER m2([https://www.iterm2.com](https://www.iterm2.com))配合得非常好，提高了开发人员的生产力。
