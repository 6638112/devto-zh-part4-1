# 是时候成为开源项目的一部分了

> 原文：<https://dev.to/sandorturanszky/it-s-high-time-to-become-part-of-an-open-source-project-25ho>

JavaScript 开发人员，我正在从事一个激动人心的开源项目，追求两个目标:

1.  学习 JavaScript/NodeJS 的最佳实践
2.  帮助开发人员和我自己开发和发布 MVP 来快速验证想法。

作为开发人员，我们有大量的想法，如果有一个简单的工具来快速搭建一个安全的项目，添加一些表单，一些项目特定的逻辑，那么就可以部署和测试您的想法了。

目前的栈是 JavaScript/Nodejs，MongoDB，Redis，Graphql。计划添加 React、Vue、Angular、TypeScript。

这不是一个新的想法，但是将学习和构建你的项目与你正在积极开发的开源软件结合起来是很棒的。作为一名开发者，活跃在开源世界对你是有益的。

如果有人想从事开源项目，请联系我，让我们一起构建一个了不起的项目。
