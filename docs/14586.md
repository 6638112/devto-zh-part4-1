# 我的桌面设置——第 3 部分:VS 代码

> 原文:[https://dev . to/manoeltlobo/my-desktop-setup-part-3-the-vs-code-5 ffc](https://dev.to/manoeltlobo/my-desktop-setup-part-3-the-vs-code-5ffc)

我已经详细讨论了我的操作系统外观和终端配置。现在该说说我的主要日常伴侣了，VS 代码编辑器。

* * *

当我刚开始学习如何做网页时，我使用 Windows 的记事本编辑 HTML 代码(美好的旧时光！).当我第一次看到 Macromedia Dreamweaver(现在是 Adobe 产品)并有机会使用语法高亮显示、自动完成、文件管理和许多代码片段时，我当时 15 岁的头脑充满敬畏。

快进快二十年 <sup id="fnref1">[1](#fn1)</sup> 。我花了这段时间在开发世界进进出出。尝试了成熟的 ide(Eclipse，NetBeans)，轻量级编辑器(Sublime)，纯粹的极简主义(Vim)。但我相信微软的 Visual Studio 代码对我来说已经达到了最佳状态。

## [](#themes)主题

引起我注意的第一个主题是韦斯·博斯的《钴 2》。这是蓝色和黄色色调之间的一个很好的平衡，连续使用非常舒适。最近也在用莎拉·德拉斯纳的夜猫子。除了绝对华丽之外，它还提供了一个轻便的版本，当房间极度日光浴时相当有用。

## [](#fonts)Fonts

我喜欢绷带。当我看一个符号而不是一系列必须顺序阅读的有特殊含义的字符时，它们允许我减少认知负荷 <sup id="fnref2">[2](#fn2)</sup> 。作为一个在经济上相当节俭的人，我寻找好的免费替代品。

我的主要竞争者是 Space Mono 和 Fira Code。选择 Nerd 字体补丁版本很有意思，它包括字形来装饰命令行，并在您的日常任务中使用字体、字形和图标。这些字体可以在[书呆子字体网站](//nerdfonts.com)或 [GitHub repo](https://github.com/ryanoasis/nerd-fonts/tree/master/patched-fonts/) 获得。

另一个选择，实际上是相当新的，是维克多单声道字体。它提供了一组类似草书的斜体字，如果您喜欢的话，可以为您的代码添加更多的风格。它可以免费下载，但是如果你打算使用它，向它的创造者捐赠是对那些负担得起的人的一种友好的表示。

## [](#extensions)分机

如果您使用特定的工具或语言，一些扩展是有用的，但是这些是一些非常好的扩展，它们是准内容不可知的:

*   [括号对着色 2](https://marketplace.visualstudio.com/items?itemName=CoenraadS.bracket-pair-colorizer-2) :当你不知道什么里面是什么的时候；
*   [颜色高亮](https://marketplace.visualstudio.com/items?itemName=naumovs.color-highlight):看看#068910 是不是你想的那种绿色——直接在你的代码里，当你输入的时候；
*   [Git 历史](https://marketplace.visualstudio.com/items?itemName=donjayamanne.githistory)、 [Git 指标](https://marketplace.visualstudio.com/items?itemName=lamartire.git-indicators) & [GitLens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens) :如果你用 Git，可以用这些掩护你的基地；
*   [Polacode](https://marketplace.visualstudio.com/items?itemName=pnp.polacode) :当代码美到必须用一个完美排列的框架与世界分享的时候；
*   设置同步:让你的每一台不同的电脑选择相同的设置，让你远离疑虑；
*   记录你在项目中的时间去了哪里。
