# 永远做一个初学者！培养初学者的思维

> “保持一颗初学的心并不难。在一个初学者的头脑中有许多可能性，
> 但是在专家的头脑中很少
> —苏斯基·老紫

正信(初心)，日语一个初学者心态的词，来源于禅宗。初学者的心态是一种心理上的心态，在这种心态下，你可以放开所有先入为主的观念，接受新的想法。当初学者的头脑看到一个主题时，就好像是第一次经历这个主题。

放下以自我为中心的思想。放下限制性的信念。
放下成就。
放开专长。

开始带着渴望理解的心情看待事物。开始用同情的眼光看待事物。
开始带着好奇心看事物。
开始惊奇地看待事物。

你如何将初学者的思维应用到一项任务中？

*   活在当下。不要去想你过去的经历，而是专注于你在那一刻看到的。
*   假设你是第一次学习的学生。
*   对每个小细节都要好奇。花点时间去完成任务，只是观察。
*   别评判了！寻找积极的一面，而不是指向错误的一面。保持探索模式，考虑所有选项，无论它们有多简单或困难。
