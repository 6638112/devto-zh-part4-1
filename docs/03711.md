# 振作起来，金凤花:痛苦、疾病和工作场所

我已经 38 岁了，当我上班去洗手间的时候，我仍然会把我的卫生棉条藏起来。当我去洗手间的时候，我开始在脑海中哼唱《不可能完成的任务》( T1 ),经历了从包里把卫生棉条拿在手里，然后在同事们不注意的情况下去洗手间的痛苦经历。尽管我确信，如果我的同事们注意的话，我脑海中的《T2:碟中谍 3》更接近于《T4》中的《憨豆先生 5》。

嗨，我叫克里斯托。我有社交焦虑症，我来月经了。

还和我在一起吗？

在这里，我要警告大家，我会在科学层面上稍微有点 TMI。

我们继续。

我的月经糟透了。我对它在学术上的意义没什么意见，但在情感层面上，它可能会被严重搞砸。从某种意义上说，我确实认为自己是幸运的。我的周期像时钟一样运转。我知道它会到来的那一天，并且多次告诉我它会在 5 小时内开始。对许多人来说，这不是他们所经历的。相反，这是一个猜谜游戏，他们的周期将在何时何地开始，这种不确定性会导致重大的不便和灾难。

但我和其他人分享的一件事是大姨妈来时的疼痛。

每个月我都会经历比我几个小时后死于肺炎时更严重的疼痛。我的肺塞满了粘液，慢慢地失去了吸入空气的能力，这比我一个月有 2-3 天因为我不打算要孩子而大发脾气要少得多。

我很少因为经期疼痛而请假。在我职业生涯的早期，这不是一个选择。休假意味着没钱，有时甚至没有工作，这是因为雇主的态度，如果你敢请病假，除了濒临死亡。

在我职业生涯的后半段，我有机会为一些公司工作，在这些公司，完全或部分远程工作并不是什么大问题。在最糟糕的日子里，我尽量调整我的时间表，在家工作。在那里，我仍然可以工作，而且还可以把我的 MacBook 放在肚子上，充当 3000 美元的加热垫。

每个月我都告诉自己，我会请一天病假。当我希望每 5 秒钟就能踢踢自己的子宫时，我真的有那么多产写测试用例吗？我经常拒绝请病假，并告诉自己要坚持下去。我目前工作的公司请病假并不是不可原谅的罪过，但美国职业道德的幽灵耍了一个赖克尔花招，张开一只白爪子，让我的大脑呼吸沉重。

事实上，忍受疼痛会适得其反。出勤主义，一个带病工作的花哨名字，在美国劳动力中很普遍。在公司可以展示大量关于工作场所缺勤成本的白皮书的地方，出勤主义很少被讨论。许多关于这一主题的研究发现，员工生病期间工作给企业带来的成本可能远远高于员工休假。据《福布斯》报道，“一项美国生产率审计计算出美国的出勤主义成本超过每年 2250 亿美元；其中 71%是由于生产力低下。”

出勤主义也是传播传染病的绝佳媒介。流感季节，办公室成了雷区。通常不是是否会生病的问题，而是你什么时候会生病。流感在员工之间传播的频率，比被雇佣时被吹捧为“酷福利”的桌上乒乓球还要高。

公司有责任阻止出勤主义的传播。一个伟大的公司文化是尊重员工，让他们健康。一个健康的员工是快乐的、高效的。这可能听起来很老套，但这是事实。正如我上面简要提到的，科技公司的一个趋势是提供“厨师津贴”，如乒乓球桌、无限量小吃和 6 种现成的 IPAs。这些福利还不错(随时可用的 IPAs 对我来说似乎是惩罚性的)，但如果没有有意义的病假、远程和弹性工作时间政策，这些福利最终都是空洞的姿态。
