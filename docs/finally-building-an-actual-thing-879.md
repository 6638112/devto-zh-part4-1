# 终于造出了一个真实的东西

> 原文：<https://dev.to/thecodepixi/finally-building-an-actual-thing-879>

我非常激动地宣布，我终于造出了一个我引以为豪的东西，它来自我自己的大脑，没有遵循任何教程，也没有抄袭任何人的作品(据我所知)。我已经学习编码一年多了，直到现在我还没有勇气或知识去构建任何我有想法的东西，除了一些纯粹的 HTML/CSS 响应网站和组件。

今年夏天，我一直在做#100DaysOfCode(第二次)，我已经完成了大量的 freeCodeCamp JavaScript 练习，学习了一些#JavaScript30 教程，学习了 Flatiron School bootcamp 预备课程 JavaScript labs，我对下一步该做什么感到有点困惑。

我在 Flatiron 的“Rock Dodger”项目(lol)中进退两难，我决定需要更多的 JavaScript 实践。我需要访问 JS 中的 DOM，遍历数组或对象，并操作 DOM 元素。

所以，我决定建立一个 [Pantone 颜色样本随机发生器](https://codepen.io/TheCodePixi/full/WVNNPQ)。(强烈推荐访问链接。我个人觉得玩起来真的很有趣。)

我也有一个想法在我的大脑后面浮动，另一个涉及随机化的项目，所以我认为这将是一个实践的好机会。我想挑选一些相对容易随机化的东西(东西的颜色),并在对象中使用关系数据，不仅返回颜色，还返回它的名称。

我也迫切需要学习 Git 和 GitHub，于是一举两得(rock？)!如果你想看这个项目的 GitHub 回购，欢迎你来看一看！
