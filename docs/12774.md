关键特征

1.  创建 11 种不同类型拼贴画。
2.  开始时预装或不预装图库中的图片(仅适用于 Android)。
3.  允许用户在学院中添加或删除照片。
4.  用户可以从图库/相机更新图像。
5.  别担心权限，我们处理好了。
