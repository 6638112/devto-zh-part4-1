# 我从银行家到软件工程师的旅程

> 原文：<https://dev.to/tremainebuchanan/my-journey-from-banker-to-software-engineer-epc>

[![Alt Text](img/a1338f67d29a1a4ee8cc961cfdacd21c.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--MnsKfgLF--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/9oeiz23l61qcxslq9yas.jpeg)

他们说重要的不是目的地，而是旅程。就我而言，成为一名软件工程师的旅程花了三年多时间。然而，我不认为我对旅程的任何方面感到遗憾。所以让我们开始吧！

### 后高中

19 岁高中毕业后，我不知道下一步该做什么。理想的情况是，在 2000 年至 2005 年期间，我在沃尔默男子高中参加了田径比赛，我在田径方面的天赋和成就本应使我获得体育奖学金，在海外攻读本科学位，但似乎宇宙另有安排，因为我离开 13 年级时没有任何奖学金。

我必须做出决定，是去理工大学(UTECH)、西印度大学(U.W.I)还是去找工作。幸运的是，我被华盛顿大学录取，攻读电子工程学士学位，然而，由于缺乏资金，我的大学梦被搁置了，这促使我积极寻找全职工作。

### 我的第一份工作

我的求职相当成功，我面试并在国家商业银行(NCB)圣贾戈分行获得了一个客户服务的职位(出纳员)。这只持续了一年，因为这不是我认为我可以再做 10 年的事情——我想建立或创造一些东西。任何事！

我相信天生的建筑天性是遗传自我父亲的。在我的记忆中，他总是将钢/金属弯曲成型为格栅(牙买加家庭中常见的保护性金属网格装置)。此外，我的大哥摆弄小玩意和小发明，所以我天生就会跟随他们的脚步。

[![Alt Text](img/394d39cd2f7733af5faf4bf2061e79b7.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--lWwCLqtU--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/0wtiy8qv51qcwb2as29q.jpeg)

意识到我的创造性遗传倾向，我开始在 VB.Net 写小代码片段来做简单的事情。我惊讶于只用几行代码所取得的成就。它激发了我对软件开发的兴趣，以至于我当时认为我的角色仅仅是一份工作。所以我一时兴起决定辞职，没有一个合适的策略来获得一个软件开发的角色。

失去了工作和金钱，我很快意识到自己的错误，并在技术领域之外寻找另一份工作。猜猜我最后又去了哪里？另一家金融机构。是啊！你没看错。从一家金融机构到另一家。老实说，这绝对是一种倒退。

尽管最初我对自己的工作失误感到失望，但我在获得一个不面对客户的职位(后台)时找到了快乐。我很高兴自己从排队服务顾客的单调工作中解脱出来。虽然我离我的理想工作还很远，但我还没有准备好从不同的角度融入银行业——商业银行在我的整体职业发展中至关重要。

### 商业银行与夜校

我可以把当时的商业银行业务描述为包括牙买加境内的大公司。我了解到的这个业务部门代表了 CIBC 第一加勒比国际银行的一个重要收入来源。在我的日常工作中，我主要专注于为一批小企业银行家提供行政支持，他们是关系经理，负责满足年收入低于 2.5 万美元的客户的金融需求。

在打一封简单的推荐信或分析财务报表之间，作为团队的一员，我完全投入并沉浸其中，并迅速获得了在商业银行团队成员缺席的情况下担任代理所需的必要技能。我获得了从投资角度看待企业的宝贵经验。

与此同时，我进入了怡东社区学院，通过学院的晚间学位课程攻读管理信息系统的理学学士学位。在我任职期间，当我走在成为一名软件开发人员的道路上时，我终于看到了隧道尽头的亮光。但遗憾的是，我很快发现，要想在这个领域找到工作，我需要的不仅仅是一个学位。

### 无数次的拒绝

在成功完成学位课程的所有课程后，我于 2011 年以二等荣誉毕业。我教育之旅的顶点。我不知道真正的工作是在毕业后开始的。虽然我现在在金融和技术方面很全面，但我严重缺乏实际的编程技能和开发软件的知识。作为一个新手，我以为会有开放的初级软件开发角色。但在那个时候，还没有初级开发人员的广告。尽管如此，我还是回应了其他高级开发人员的工作，远远超出了我的能力范围，我非常清楚自己缺乏基本的广告要求。多么浪费时间啊！

[![Alt Text](img/d8bc75f630982e899a117e7a24eac61a.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--jdn-s8bw--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/lh0enpbuxd5bgp6km4id.jpeg)

我没有被收到的许多拒绝信吓倒，我的坚持最终得到了回报。足够幸运的是，我被当时的雇主 CIBC 第一加勒比国际银行选中参加 Java 开发员的面试。我高兴极了！五彩纸屑！我相信我的雇主欣赏我的努力学习，并分享了我得到一份软件开发工作的梦想。我为面试做了不懈的准备，因为这是我努力在我的研究领域内寻找工作的高潮。向上流动和自我实现相距甚远，然而，在收到一封详述面试结果的电子邮件后，这些愿望很快破灭了。电子邮件通知我，他们选择了另一个候选人。真扫兴。至少可以说，我崩溃了。我保存了拒绝邮件，继续前进。

[![Alt Text](img/e8152a647695c4d57b99f2809e5f887c.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--Mr5I9opV--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/nxvfcb8qqtus2cx0hk6n.jpeg)

你可能会问，我是如何施压的？我只是决定向我的雇主展示我是一个有能力的开发人员，他们最终会失去一个伟大的人才。令人惊讶的是，在 2011 年左右，牙买加的技术领域兴奋不已，黑客马拉松变得司空见惯。我决定参加我能参加的任何黑客马拉松比赛。我相信这些黑客马拉松将提供一个平台来发展我的软件开发技能，并与其他开发人员建立联系，以获得如何进入软件开发职业生涯的脉搏。

### 终于成功了！

在参加我的第一次黑客马拉松后，我们团队赢得了四部黑莓手机(注意，这是在 2012 年，黑莓很火！).你可以想象，我欣喜若狂。但并没有就此止步。我参加的每一次黑客马拉松比赛，我都是第一、第二或第三名。这种情况持续了接下来的三年，我的简历记录了我的旅程，并巩固了在短时间内构建软件(无论是应用程序还是网站)所必需的技能。

我现在可以自信地申请与软件开发相关的职位了。2014 年 12 月，大学毕业三年后，我递交了辞职信，打算从事软件开发工作。从那以后，我换了三次角色。我曾为一家主要为政府机构开发软件的非盈利组织工作过。具有讽刺意味的是，我的下一个软件开发职位实际上是在政府部门，并且持续了大约两年。你觉得我接下来会去哪里？对，我开始了。在银行业——开发软件。

这是一次多么曲折的旅程啊！我希望这能激励那些目前没有从事软件开发的人，并能从我的旅程中得到一些启示。
