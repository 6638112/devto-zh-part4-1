听到“Python”这个词，你首先想到的是什么？是凶猛的蛇还是有趣的编程语言？

虽然这两个术语听起来很相似，但它们最终会产生截然不同的效果！！

Python(蛇)是危险的敌对生物，而 Python(编程语言)则是更友好的一种。所以让我们更多地了解更友好的一个。

Guido Van Rossum: 荷兰程序员，以 Python 编程语言的作者而闻名。他在 1989 年就已经创造了 Python，并在 Google 和 Dropbox 工作过！

# [](#ever-wondered-why-is-it-named-python)有没有想过为什么取名 Python？

[![Alt text of image](../Images/4ecf58e0fbad552e63f65d1918f61649.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--P895KHG4--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://d1jnx9ba8s6j9r.cloudfront.net/blog/wp-content/uploads/2018/07/monty-pythons-flying-circus-1.jpg)

关于它有一个有趣的故事。在实现 Python 的同时，Van Rossum 还阅读了 BBC 世纪 70 年代的喜剧系列“蒙蒂·Python 的飞行马戏团”的出版脚本。因为他想为自己的发明取一个简短、独特且略带神秘的名字，所以他从这个系列中获得灵感，将其命名为 Python！
