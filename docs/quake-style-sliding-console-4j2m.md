# 地震式滑动控制台

> 原文：<https://dev.to/jeansberg/quake-style-sliding-console-4j2m>

厌倦了你的简单，旧的 cmder 控制台？本教程将告诉你如何添加一些复古的雷神之锤风格的味道！

[![After](img/43818f4f6ec0372be719650cac8d8ea1.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--AIFaunTb--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/1xxrts7ltk5oiyvv7hvo.gif)

## 先决条件

使用默认设置的 cmder 安装。

*   [https://cmder.net/](https://cmder.net/)

## 背景图片

让我们从添加 Quake 控制台背景图像开始。从[这里](https://vignette.wikia.nocookie.net/quaimg/f/f5/Conback.png/revision/latest?cb=20150212215701)下载图片。打开设置对话框并转到`General->Background`。检查背景图片，用*拉伸*设置你刚刚下载的图片的路径。

## 字体

下一步是下载雷神之锤字体，以增加更多的视觉效果。你可以从[这里](http://www.thealmightyguru.com/GameFonts/Fonts/Font-Quake1.zip)得到它。安装字体并转到`General->Fonts`并将主控制台字体设置为*地震 1* 。

## 滑下

现在我们将添加来自雷神之锤和类似游戏的下滑行为。转到`General->Quake style`并检查地震式下滑和总是在顶部。将帧宽度设置为 0 可以隐藏控制台窗口周围的边框。

转到`Keys & Macro`并选择最小化/恢复(也是地震式热键)。单击“选择热键”字段，然后按下波形符键(或您喜欢的任何键)。确保在选择热键(如 Ctrl 或 Shift)后没有在下拉菜单中选择任何内容。

## 点睛之笔

转到`Features->Status bar`并取消选中显示状态栏。转到`General->Tab bar`并勾选不显示。“设置”按钮现在将被隐藏，但您可以通过 Win+Alt+P 访问设置。

不要忘记保存您的新设置！
