# 我的 3 美分给找工作的初级开发人员/数据科学家

> 原文：<https://dev.to/lidderupk/my-3-cents-for-junior-developers-data-scientists-looking-for-a-job-cbj>

[![](img/00589d1246d5e5036931463c8e044aff.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--XWRuQqnY--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/667/1%2AJ40PyNdDSXEjQIzeOfK1PA%402x.jpeg)

我们每年都会在 IBM 得到一些优秀的实习生。这些聪明而有才华的年轻人从高中到大学都有。我给他们的建议有三点…

1.  尝试新事物，尽可能多的学习。并不是说学习永远不会停止。无论你是软件开发人员还是数据科学家，在这个职业生涯中，你总是在学习。不管你是一岁还是十年。然而，现在是无畏地学习的时候了，毫不犹豫。你拥有的时间比任何时候都多。早点犯错！不尝试就不会犯错！哦，当你在做的时候，玩得开心点！！
2.  **建立投资组合**。将你所学的一切变成 Github 项目(同时尊重公司知识产权)。不管怎样，你都在学习，好好利用它。在你转向其他事物多年后，你会惊讶于你所依赖的项目。公司对你有什么学位和证书不太感兴趣，更感兴趣的是你到目前为止建立了什么。你在 app store 上有应用吗？你有可以展示的故障点吗？freecodecamp 投资组合怎么样？
3.  **求助交友**。很难找到良师益友，但是如果你足够幸运找到了，那就利用他们。我在 IBM 指导过两个人，通常我从他们身上学到的比他们从我身上学到的更多！所以相信我，你是在帮你导师的忙！你知道你还在做什么吗？你在建立一个网络。我怎么强调这个网络对你的重要性都不为过。

好吧，你实习结束了。你回到学校，并以优异的成绩通过了考试！你现在正在找工作。如果你曾经向我寻求建议，并且真的照做了，你有一些项目，你从你的导师那里学到了很多东西，你有一个惊人的网络，你可以寻求帮助。接下来呢？你如何获得帮助？我看到许多应届毕业生犯的一个错误是写一封电子邮件或信息，大意是…

> 亲爱的导师你好！我刚毕业。这是我的 Github 简介。我正在找工作。你有什么吗？

作为一名导师，我将把这放在我的待办事项的底部。尽管我认为你是一名出色的开发人员，但为你寻找公开的机会似乎还有很多工作要做。除非你的导师是你的家人，否则这不会激励你的导师提供帮助。家人几乎总是会帮忙！这里的技巧可以帮助你的关系网帮助你。让他们轻松点。以下是我的建议…

#### I .当面或通过电话见面，但要非常尊重他们的时间

我可能很守旧，认为面对面喝咖啡或打电话比通过电子邮件交流更有效。一封电子邮件丢失了，或者被压到了最底下。一张脸和一段对话留在记忆里。我知道并不总是有可能面对面，但即使是 facetime 或 zoom 会议也是值得的！

只要求你需要的时间。我从一个 15 分钟的会议开始，如果有时间限制，你能完成的事情是惊人的。尊重他们的时间！我以前试过下面的方法，它对我很有效。

> 你好，我从 xyz 得到了你的联系方式。我在你的团队中寻找机会。我在这封邮件中附上了我的简历。我希望你能抽出时间和我聊聊。下周一我已经给你发了 15 分钟的邀请。我很乐意根据您的方便重新安排时间。

一旦安排好会议，就要有一个议程。是的，15 分钟的会议也需要提前计划和思考。最后，确保你的问题在会议结束时是清楚的。一个请求可以是澄清某个角色，获得一个团队领导的面试机会，或者为你建立一些联系，把你介绍给合适的人。

#### 二世。带着招聘启事去找他们

更好的办法是，如果你在你的社交网络中看到一个公开的招聘信息，拿着招聘信息去找他们。记住，这样做的目的是让你的联系人尽可能容易地让你得到他们公司/团队/部门的面试机会。

> 你好，我从 xyz 得到了你的联系方式。我在你的求职网站上找到了一个空缺职位(id-xyz ),我相信自己非常适合这个职位。我在这封邮件中附上了我的简历。我希望你能抽出时间和我聊聊。下周一我已经给你发了 15 分钟的邀请。我很乐意根据您的方便重新安排时间。

#### 三世。做家庭作业

这也是一个大问题。与你想加入的职位上的人交谈。索要组织结构图。求痛点和策略。尽可能深入地了解你未来的角色。更好的办法是，制作一些笔记或幻灯片，说明你为什么非常适合这个职位，并可能以一个演示开始你的 15 分钟会议。我说有可能，因为每次会议都不一样。你也可以浏览你的笔记，而不用把它们显示成幻灯片。它不一定是一个官方的演示来表达你的观点。这将为你做两件事

1.  这将会给你的联系人和/或导师留下深刻的印象。
2.  他们不再拉着你的手了。你再次让他们更容易帮助你。你让他们为你而战。

最后，即使不找工作，也要和你的人际网络和导师保持联系！
