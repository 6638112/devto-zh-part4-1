# 再说招聘人员

> 原文:[https://dev.to/citizen428/let-s-talk-about-recruiters-3p83](https://dev.to/citizen428/let-s-talk-about-recruiters-3p83)

大多数软件开发人员与招聘人员都有过不愉快的经历:垃圾邮件和 LinkedIn 信息，完全不适合的工作邀请，很少的技术知识，等等。

从招聘的角度来看，情况也几乎一样糟糕:由于佣金可能相当高，招聘人员会敞开渠道，因为这增加了他们获得报酬的机会。此外，他们通常缺乏正确评估候选人的技术知识(我遇到过一些明显的例外，但他们似乎很少)。

这让我想到一个问题:人们想要/期望招聘人员做什么？你理想的招聘人员具备哪些技能？他们做什么/不做什么？我对候选人和招聘人员的观点都感兴趣。
