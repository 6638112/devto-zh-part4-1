# Kotlin 中的函数

> 原文：<https://dev.to/jay_tillu/functions-in-kotlin-4p64>

> ***函数是执行特定任务的一组语句。*T3】**

*   函数主要用于自动化重复的任务。它们基本上是用来把大程序分成小块的。

*   将我们的程序分成更小的块使它更有组织性和可管理性。此外，它避免了重复，使我们的代码可重用。

*   关于函数的不好的一面是它们以几个名字而出名。不同的编程语言对它们的命名不同。例如函数、方法、子程序、过程等。当你听到他们中的任何一个时，想象一下同样的概念。

编程中基本上有两种类型的函数。

1.  内置函数
2.  用户定义的函数

*   **内置函数—** 内置函数是语言自带的函数，可以随时使用。例如在 kotlin 中`print()`、`readLine()`、`sqrt()`都是内置函数。在 kotlin 中，有一个 [kotlin 标准库](https://kotlinlang.org/api/latest/jvm/stdlib/index.html)，里面存储了所有的内置函数。你可以查看更多信息。

*   **用户自定义函数—** 用户自定义函数是程序员根据需要创建的函数。您也可以根据需要创建函数，这些函数将被称为用户定义函数。
