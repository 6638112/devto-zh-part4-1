# 自学成才的开发人员的问题...

> 原文:[https://dev . to/amberchisholm/question-for-自学成才开发者-gdp](https://dev.to/amberchisholm/question-for-self-taught-developers-gdp)

我很好奇...对于那些自学编码的人来说，导师在你的旅程中发挥了多大的作用？

很多吗？一点点？
一点都没有？

当我回顾我的 JavaScript 研究并为某些事情而挣扎时，我不禁想知道是否大多数其他自学成才的开发人员也在独自完成这一切？或者大多数人在某个时候找到了一个导师来帮助指导他们？

虽然我对自己成为一名全栈开发人员的能力充满信心，但我确实认为一名以前做过这些工作的有经验的开发人员在编码过程中会非常有帮助。

各位自学成才的程序员朋友们，你们怎么看？现在有导师不现实吗？
