# 加入我的不和谐社区

> 原文:[https://dev . to/therealdanvega/join-my-discord-community-1jcf](https://dev.to/therealdanvega/join-my-discord-community-1jcf)

我决定创建一个[不和谐](https://discordapp.com)社区。在过去的几个月里，我一直在思考这个问题，对此我感到非常兴奋。在这篇文章中，我将向你简要介绍什么是 Discord，为什么我选择它作为一个平台，以及它如何与 Slack 或脸书集团相比较。

## [](#what-is-discord)什么是不和？

如果你不熟悉 Discord，这是一款为游戏玩家提供的语音和文本聊天一体机，免费、安全，可以在你的桌面和手机上使用。很长一段时间以来,《不和》主要面向游戏玩家，现在依然如此。也就是说，它已经建立了一些非常棒的功能，吸引了很多社区。

[![Discord Home](../Images/4b696e98e763e162d951e30ab6cf01bb.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--3IGUTPdo--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ez8n7gaozngjbmeyy8bo.png)

今年我已经加入了 5 个左右的 Discord 社区，我真的爱上了它的易用性。我也喜欢我所在的社区，那里的每个人都非常乐于助人，对我很好。

人们希望加入社区，在那里他们可以进行有益的讨论而不会被嘲笑。这就是为什么 Discord 非常注重防止垃圾邮件，并为服务器管理员提供管理垃圾邮件所需的工具。

## [](#why-do-i-need-a-community)我为什么需要社群？

我一直纠结的真正问题是 ***我真的需要开一个社区*** 吗？我是说我是谁？我只是一个喜欢帮助人们学习如何编码的人。我开始越来越多地思考这个问题，尽管它对我来说越来越有意义。

### [](#course-student-discussions)课程学生讨论

如果你是新来的，我在网上教授课程，目前在多个平台上有超过 85，000 名学生。首先，在这些平台上管理学生的问题变得非常困难。我所教的平台为学生提供了提问的途径，但它们不太适合让学生相互交流。

我认为通过让这些学生不和，我可以更好地处理被问到的问题。一旦我能更好地处理被问到的问题，我就能尝试在我的课程中纠正这些问题，并减少我必须回答的问题的数量。我还认为这可能会变成学生可以帮助其他学生解决问题的事情。现实是，有了全职工作和家庭，我需要充分利用我仅有的一点空闲时间。

### [](#not-just-for-students)不仅仅针对学生

如果你不是我的学生，别担心，这里也有适合你的东西。我喜欢通过文章和视频来教学。如果您对这些内容有任何疑问，这里会有一个地方供您提问。

我也时不时地尝试在 Twitter 上回答问题。这些可能是我看到其他人问的问题，或者有人问过我的问题。虽然我喜欢 Twitter，但这并不是回答技术或编码问题的最佳场所。所以这些是我希望在我的社区中开始的其他类型的讨论。

### [](#why-not-slack)为什么不懈怠？

这听起来正是 Slack 的用途，对吗？我在工作中使用 Slack，实际上我很喜欢它，但它对社区来说不是很好。我已经在某种程度上理解了这一点，因为如果你不为 Slack 消息付费，它们就会开始消失，这在你想对内容进行归档时可不是什么好事。

我可以继续谈论使用 Slack 建立社区的问题，但我不会。相反，你应该看看扎克·沙勒贝特的这篇令人惊叹的文章，标题是 ***[【懈怠正在摧毁社区】](https://medium.com/@zachshallbetter/slack-is-destroying-communities-1ce49dad96ca)*** 。

### [](#why-not-facebook-groups)为什么不是脸书集团？

我目前为 Spring Boot 的开发者们管理一个脸书小组。我已经把这个团队扩大到了 11，000 多人，但老实说，这真的变得很难管理。批准新成员很容易，但我每天必须筛选大约 50 个帖子，其中大部分是有人试图发布他们的文章或视频。

想要分享你创造的东西并没有错，但在脸书，我觉得这不是一个好地方。facebook 上没有频道，所以这是一个很大的反馈，问题很容易丢失，所以我试着把小组限制在讨论上。除了自我推广链接之外，我还收到了一堆垃圾邮件，脸书集团的工具也不太好用。

### [](#join-my-discord)加入我的不和
