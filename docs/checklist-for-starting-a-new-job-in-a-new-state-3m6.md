# 在新的州开始新工作的清单

> 原文：<https://dev.to/johnphamous/checklist-for-starting-a-new-job-in-a-new-state-3m6>

开始一份新工作是令人兴奋的。搬到另一个州也令人兴奋，但也可能令人畏惧。我希望当你发现自己处于这种情况时，这个清单对你有用。

*这是在假设你要飞往新的地方并轻装上阵的情况下写的*

## 未动之前

[![moving](img/dff09dd23989c2f85b51828eac6fde9d.png)](https://i.giphy.com/media/mwWSwdrfjzO7e/giphy.gif)

1.  找个地方住
    *   在你搜索的时候，去朋友家、AirBnb 或者公司的房子里找找。这需要大约一周的时间。(挑选最佳地点在你看完 37%你想去的地方后，这是[最佳选择](https://en.wikipedia.org/wiki/Optimal_stopping)
2.  获取你的医疗记录的副本
    *   拜访你的医生、牙医和任何你过去见过的人，获取物理或数字记录。这些记录将对你的新保健提供者有用。
3.  取消您不会使用的设施/服务/会员资格
4.  使用 [USPS](https://moversguide.usps.com/mgo/disclaimer) 请求邮件转发
    *   如果你希望旧地址有邮件，你可以把它转寄到新地址
5.  向您的银行更改帐单地址
6.  使用 8822 表格[向 IRS 更改邮寄地址](https://www.irs.gov/forms-pubs/about-form-8822)
    *   确保你尽快收到所有的信件和退税

## 一到那里

[![arriving](img/762f1497e7ab5e0195d4516c2107551c.png)](https://i.giphy.com/media/3oxRmwOLK6OLi9IFI4/giphy.gif)

1.  登记投票
    *   有些州允许你在网上或申请驾照时申请
2.  获得驾驶执照或[增强型驾驶执照](https://www.dhs.gov/enhanced-drivers-licenses-what-are-they)
    *   后者可用作公民身份的证明，并作为有限的护照
3.  注册公用事业
    *   包括:电话，互联网，煤气，水，电
    *   向你住的地方、你的雇主和你所属的任何组织核实一下，如果他们有交易的话
4.  在新州建立合法居留权([合法住所](https://en.wikipedia.org/wiki/Domicile_(law)#United_States))允许您:
    *   避免支付两个州的所得税
    *   加州有所得税，华盛顿没有，所以你不用缴纳所得税
    *   有资格申请州内大学学费
    *   选举地方和州政府的权力
5.  尽可能买二手的
    *   你可以在二手市场上以零头的价格买到很棒的家具
    *   如果你住在大学或大型工作场所附近，查一查他们是否有车库拍卖
    *   购买二手物品不仅经济实惠，而且对环境也有好处
6.  在你通勤的时间内计划好你的通勤。谷歌地图和城市地图是你可以使用的工具。

## 工作第一周后

[![excited](img/8305b83f6b460abe15fb876f2ea4bfda.png)](https://i.giphy.com/media/5GoVLqeAOo6PK/giphy.gif)

1.  注册雇主的 401(k)和健康储蓄账户
    *   如果你的员工有 401(k)匹配，这基本上是免费的钱
2.  查看雇主提供的津贴/折扣
3.  寻找新的医疗保健提供者

## 一旦你安顿下来

[![cozy](img/4367000c5046e49bbee645de3cd23643.png)](https://i.giphy.com/media/Zw97g5eB1mBji/giphy.gif)

1.  报名参加健身房/健身俱乐部
2.  参与你公司的[员工股票购买计划](https://www.investopedia.com/terms/e/espp.asp)
3.  向您的个人退休账户缴款(将缴款设置为自动进行)
4.  探索你的新地方！
    *   远足，课程，美食，和新的爱好！
