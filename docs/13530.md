创建新的模型用户

*   rails g 模型用户:姓名:城市

```
class CreateUsers < ActiveRecord::Migration[5.2]
  def change
    create_table :users do |t|
      t.string :name
      t.string :city

      t.timestamps
    end
  end
end 
```

*   rails g 迁移重命名用户帐户

```

  def change
    rename_table :users, :accounts
  end
end
```

```
    add_column :accounts, :email, :string
```
