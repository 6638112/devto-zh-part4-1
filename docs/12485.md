# 有趣的莫尼解释道

> 原文：<https://dev.to/johnedwards/funnymoney-explained-3ac7>

我的朋友奥利建议把这个贴在这里。最近的[系绳肥手指失态](https://twitter.com/whale_alert/status/1150161710587764737)促使他在一个松弛的频道说“我只是不明白假钞”。我写了以下未经编辑的版本作为一个咆哮的解释。

难怪比特币等被称为“密码”，因为这些有趣的东西很像他们的同名加密术:理论上很棒，但实际上通常很糟糕。

我知道我会因此受到你们中一些人的严厉批评，但请听我说完。

围绕密码学的理论都经得起逻辑/数学/统计的检验。我在考虑像克霍夫原理这样的想法，它归结为这样一个想法:如果系统是公开的，那么密码系统应该是安全的，但密钥是保密的。不要把它与控制电路的基尔霍夫定律混淆。奥古斯特·克霍夫和古斯塔夫·基尔霍夫(显然)是不同的人。类似地，依赖分解大素数的困难作为公钥加密基础的一般理论是合理的。

在这里我不得不引用莱纳斯·托沃兹的话:[“理论和实践有时会发生冲突。当这种情况发生时，理论就失败了。每一次。”](https://en.wikiquote.org/wiki/Linus_Torvalds)

当人们实现加密时，理论就都见鬼去了。你已经得到了国家安全局推荐的质数曲线，他们已经走后门了。某些加密协议[允许你欺骗接收者接受“出口级加密”](https://www.symantec.com/connect/blogs/freak-vulnerability-can-leave-encrypted-communications-open-attack)几十年后那不再是一件事。然后，有可怕的“我是郊区办公园区最聪明的人”开发人员推出自己的加密。

回到有趣的话题，我并不认为它们有真正有趣的理论属性:分布式账本，代表去中心化；隐私保护；核查机制，这将导致更大的信任；低成本或零成本交易，与渴求费用的“大坏银行”相对。

然而，在实践中，很多有趣的特性都不尽人意:“消除大银行的高额交易费”，比特币之所以失败，是因为价格高；“安全性”失败了，因为每个钱包或平台都写在小本启动预算上，可能是由一些“我是我的计算机科学课上最聪明的人”的新手开发者写的；等等。等等。

如果你从事技术工作，你总是要处理一些这样的问题。总有一个自以为聪明的戴夫。(是的，我写的是何，因为干这种事的总是公子哥儿。)总有一些古老的代码或系统没人想碰或更新。我打个比方:技术问题就像一堆木头。他们总是在那里。一堆木头也没什么坏处。

与 funnymoney 的不同之处在于，不是只有我们这些书呆子在发明新的 javascript 框架——当我在做的时候，请停止这样做——而是你将一大群华尔街交易员加入其中。(请允许我在这里享受一下亨特·S·汤普森时刻。)这些人都是骗子&骗子&无赖，除了在音乐节前采购上好的可卡因之外，不要相信他们。

别误会，我的一些好朋友是华尔街人士。但我了解他们。他们缺乏顾忌。他们看到一些不受监管的、完美的抽水和倾倒计划，他们开始垂涎三尺。他们的欲望开始蠢蠢欲动。你轻声说“破坏性的”和“绿色领域”——哦，宝贝——然后你接着说 ICO 和“超增长”——哦，是的，就像这样——当你点击“证券化”时，他们的脸变得又红又紧，他们尖叫着“BITCONNEEEEEEEEEEEEEECT！”

从交易的角度来看，他们的所有活动都是有意义的。想想《大空头》里的那个扑克场景。你得到的东西像系绳和 Monero，这可能是不同的。但是你还有加密期权和衍生品。很快就会出现二阶导数(如果还没有的话)。所有这些垃圾的存在是为了让商人可以把一个柠檬卖给那些他们认为有价值的笨蛋。但事实可能并非如此。不久之后，将会出现抵押加密债券。

延伸一下我的比喻，华尔街人士是技术这一堆干柴上的液体。

在这一点上，你所需要的是一群 Icarusian 风投(他们没有其他投资)说“我们在扰乱金钱”的火花。但他们谈论的不是货币，而是“法定货币”，这听起来很聪明，很欧洲化，所以人们会买它的账。

我通常认为事情很简单。股票是公司的股份。债券是债务。房地产是你可以居住/工作的东西。加密货币是一个位串，描述了一个保存“货币”的“钱包”，这些货币是由可能存在也可能不存在的人发明的。没有人读过他们所投资的硬币的白皮书。如果他们有，他们还没有理解它。但是他们肯定告诉人们他们理解它。

你可能会在评论里烤我。你会告诉我“我就是不明白。”你会说“没错，但是 XYZcoin 就是为了解决这个问题而建立的。”我觉得这一切都很可疑。我只是一个简单的人，试图告诉每个人不要在离他们的房子太近的地方生火。在加州。在干旱期间。
