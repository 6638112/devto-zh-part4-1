## [](#what-is-an-operating-system)什么是操作系统？

重要的事情先来。

操作系统是在计算机上运行的软件。它允许用户与计算机的硬件进行交流以完成任务。

例如，如果您正在处理一个 Word 文档，操作系统会从您的键盘接收您按下了哪些按钮的信息，并与 Word 应用程序进行交互，从而显示正确的字符。

操作系统分为两个部分，用户模式和内核模式。用户模式是指，例如，您使用的应用程序、网络浏览器、游戏、Word 等。内核模式就是内核所在的地方。内核的目的是解释你正在使用的应用程序和计算机硬件之间的消息。当一个操作系统被称为 Linux 时，这意味着该操作系统上的内核是 Linux 内核。而且，如果你用的是一些最新的 Windows 版本，那里的内核是 Windows NT。

最常见的操作系统有:

*   微软视窗系统(例如:XP，10)
*   苹果操作系统(例如:iOS，OS X)
*   Linux(例如:Android、Debian、CentOS)

## Linux 从何而来？

Linux 最初是一个名叫 Linus Torvalds 的人在赫尔辛基大学学习时的爱好。在他使用了由大学教师 Andrew Tanenbaum 发明的 MINIX 之后，他产生了创建自己的类 UNIX 系统的愿望。MINIX 的目的是作为一种教学工具，帮助学生理解和使用 UNIX。

1991 年 10 月 5 日，托瓦尔兹决定与全世界分享他的这个小爱好。这使得其他人可以帮助他增加他的项目。这条消息引起了滚雪球效应，成千上万的人参与了这个项目，并成就了今天的 Linux 内核。

## [](#why-is-it-free)为什么免费？

托沃兹知道一个正在进行的名为 GNU (GNU 不是 UNIX)项目的运动，它是由理查德·斯托尔曼在 1983 年发起的。GNU 哲学的一部分是，软件的所有源代码都应该免费供每个人审查和修改。托瓦尔兹喜欢这个想法，因此分享了他的项目，如上所述。

## [](#why-so-many-distros)为什么这么多发行版？

发行版是发行的缩写。所以一个 Linux 发行版是一个操作系统，它由 Linux 内核(这是 Torvalds 创建的)、GNU 工具和库、包管理系统和基于它的附加软件组成。所以发行版是完整操作系统的基础。

原因是因为 GNU 运动。因为每个人都可以下载、修改和重新发布 Linux 内核。因为人们有不同的需求，他们创造了满足他们目标的发行版并重新发行。每个发行版都有不同的实用程序、软件和不同版本的 Linux 内核。每个发行版都试图达到一个特定的需求。

最常见的 Linux 发行版有:

*   一种自由操作系统
*   Ubuntu(基于 Debian)
*   Linux Mint(基于 Ubuntu)
*   红帽企业版 Linux
*   CentOS(免费版红帽)
*   Fedora(更像桌面版的红帽——不太稳定)
*   苏塞
*   OpenSUSE(不太稳定的 SUSE 版本)

## [](#where-is-it-used)用在哪里？

就像我们上面提到的，有许多发行版可以满足不同的需求，任何人都可以创建他们的发行版。正因为如此，Linux 有很多用途。从智能电视、网络设备到航天器软件和医疗设备，无所不包。还有一个有趣的事实，所有排名前 500 的超级计算机都运行在 Linux 上。
