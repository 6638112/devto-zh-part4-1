# PHP 初学者指南

> 原文:[https://dev.to/adewalek/beginners-guide-to-php-epl](https://dev.to/adewalek/beginners-guide-to-php-epl)

PHP 可能是网络上最流行的脚本语言。它用于增强网页。使用 PHP，你可以创建用户名和密码登录页面，检查表单细节，创建论坛，图片库，调查，等等。如果你遇到了一个以 PHP 结尾的网页，那么作者已经写了一些编程代码来使普通的、陈旧的 HTML 变得生动起来。

PHP 被认为是一种服务器端语言。这是因为 PHP 不会在你的电脑上执行，而是在你请求页面的电脑上执行。然后结果会交给您，并显示在您的浏览器中。你可能听说过的其他脚本语言有 ASP、Python 和 Perl。(你不需要知道这些来开始学习 PHP。其实这些教程都是假设你完全没有编程经验的。)

关于 PHP 代表什么，最流行的解释是“超文本预处理器”。但那会使它成为 HPP，对吗？另一种解释是，首字母来自该程序的最早版本，即所谓的个人主页工具。至少你得到了字母“PHP”的正确顺序！

但是 PHP 是如此受欢迎，如果你想在网页设计/网页脚本行业找一份职业，那么你必须了解它！在这些教程中，我们将帮助您启动和运行。希望这比你想象的要容易得多。

```
 What you need to get started with PHP 
```

在编写和测试 PHP 脚本之前，您需要一样东西——服务器！幸运的是，你不需要出去买一个。事实上，你不会花任何额外的钱。这就是 PHP 如此受欢迎的原因！但是因为 PHP 是一种服务器端的脚本语言，你要么从一个支持 PHP 的托管公司那里获得一些网络空间，要么让你的计算机假装它安装了一个服务器。这是因为 PHP 不是在你的 PC 上运行的——它是在服务器上执行的。然后，结果被发送回客户端 PC(您的计算机)。

如果这一切听起来有点令人生畏，请不要担心-我们已经找到了一种更简单的方法来帮助您启动和运行。我们将使用一种叫做 Wampserver 的软件。这允许您在自己的计算机上测试 PHP 脚本。(Wampserver 的 WAMP 代表 Windows、Apache、MySQL、PHP。)如果你有 Windows PC，它会安装你需要的一切。我们稍后将解释如何安装它，以及从哪里获得它。但对非 windows 用户来说只是一句话。
苹果用户

对于苹果的使用，MAMP 是相当不错的。像 Wampserver 一样，它安装了本课程所需的一切。您可以从这里下载:

[https://www.mamp.info/en/](https://www.mamp.info/en/)

您现在要做的是启动并运行 apache 服务器，这样您就可以离线运行 PHP 脚本。请特别注意文件存储的位置，以及“localhost”地址。

Linux 用户

有很多网站可以帮助 Linux 用户使用 Apache 服务器和 PHP。不幸的是，没有像 Windows 和 Mac 用户那样简单的方法。你最好在谷歌上搜索你的发行版:

Lamp +Ubuntu

或者

Lamp +Debian

YouTube 上也有很多视频可以帮助你安装你需要的所有东西。
