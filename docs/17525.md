# 在环城工作，第四天:社区

> 原文:[https://dev . to/ashleemboyer/work-in-circle-city-day-4-community-53 E3](https://dev.to/ashleemboyer/working-in-circle-city-day-4-community-53e3)

# [](#the-am)上午

首席执行官今天早上花了一些时间谈论公司的愿景，但在此之前我们进行了一次龙卷风演习！😉这是一个有趣的事件。谢天谢地，我们只住在四楼，没有走太远就到了地下室(顺便说一下，那正好是一个雪茄吧😂).

我们现在所在的这座建筑相当古老(1923 年完工)，所以当我不得不走上人生中最陡峭、最狭窄的台阶时，我不禁想知道，这座建筑建成时，人们是否没有微退。今天我了解到一个关于这座建筑的有趣事实:当它建成时，建筑物的高度限制是 150 英尺，这样它们就不会遮挡士兵和水手纪念碑。我一辈子都住在这里，却从来不知道！

# [](#the-pm)下午

我尝试了另一家新餐馆:珍珠街披萨店和酒吧。在市区工作的一个主要好处是有知道好吃的地方的队友。在市中心工作有一点不太愉快，那就是有太多诱人的餐馆，而我的钱包又很紧。既然我已经安顿下来了，我应该开始带午餐了。😁

在(我的)工作日结束时，我的招聘班被带到 Pearings 咖啡馆和冷冻酸奶店，庆祝我们几乎完成了第一周的工作！让我们花点时间注意一下他们的七叶树味冷冻酸奶是多么的棒。😍这个地方也有很棒的咖啡和茶，所以我会尽快回来品尝这些美味。

# post-PM

我作为“印第安纳波利斯女性编码”的志愿者参加了 Powderkeg 的推介之夜，我正在努力在未来几周内更深入地参与这个组织。我在冷冻酸奶店发现，我公司的某人已经无法参加活动，我们需要有人在公司名称宣布成为 2019 年印第安纳突破技术文化奖 50 强之一后走上前来！

所以在我的第四天，我可以和来自其他 49 家优秀公司的 folx 合影。这次活动中的网络也很棒。我很快就明白了这个活动是关于什么的，以及为什么会有很多参展商出席。每个人都是为了建立一个相互支持的社区。这是一个我可以 100%支持的运动，我首先跳进去。
