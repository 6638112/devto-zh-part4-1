# 想象的种子是他人

> 原文：<https://dev.to/stephencweiss/the-seed-of-imagination-is-others-371l>

我的思绪万千。没办法。每当我与人交谈，阅读一些东西，观察一个互动，等等。我立刻开始思考其他事情。这不是心不在焉。我在场。我只是忙着把我正在学的和已经学的联系起来。如果我幸运的话，当我以一种新的方式结合两个想法时，我会想出一些主意。

我开始看到积累这些想法的好处。我接触到的想法越多，我经历的经历就越丰富，飞轮转得越快，我找到的联系就越多。

几周前，我感到筋疲力尽。这是一种身体上的疲惫，我对阅读和学习不像平常那样感兴趣。幸运的是，我的朋友萨姆打电话给我。挤出时间来交谈，没有议程，探索想法让我恢复了活力。

回想一下我从那次谈话中获得的突然注入的能量，我得出了两个教训:

1.  对我来说，总是装满我的灵感之杯是很重要的——倾听、阅读、观察为我的实现提供了原材料。它们是必要的，但还不够。
2.  将这些材料提炼为成品是通过交谈来完成的最好方式，我的心情也反映了这一点。

这创造了一个良好的良性循环。我学习新的东西。我分享它们。我变得更加富有，甚至有了更多的新想法。重复一遍。

重要的是，我需要分享。这就是我写作的原因-开始对话。这就是为什么我打电话给朋友，即使发短信可能更容易。这不是关于广播，因为这不是关于我。这是关于思想的交流。

当我和别人交谈时，我会有新的想法。这就是为什么我开始相信想象力的种子不是我自己，而是其他人。
