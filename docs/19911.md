我想更新一下我的进展。到目前为止，我已经做了很多学习。我加入了一家初创公司，试图制作一个独特的职业网站。我不能列出太多的细节，但希望这个想法能够成功。为此，我一直在学习大量的 web 开发知识，这样我就能对团队有所帮助。我学习了 React 和 Webpack，也提高了我的 HTML、CSS 和 JavaScript 技能。

另外，我也一直在尝试使用 Android Studio 学习 Android 开发。我在创业公司的队友不认为一个原生 app 是必要的，但我希望这些知识可以用于未来的就业机会。大约一个月前，我参加了一个 iOS 开发会议，我与其中一名成员进行了广泛的交谈，他告诉我应用程序开发非常受欢迎。不仅如此，Java 是我最喜欢的语言，比起 web 开发，我更喜欢这种开发。我希望我有足够的实践来获得这些工作，但如果没有，所有这些 web 开发肯定是有用的，特别是因为 JavaScript 和 Java 非常相似，这是我喜欢的。

在我的下一篇文章中，我希望更多地谈论我在初创公司和我启动的任何 Android 项目中取得的成就。
