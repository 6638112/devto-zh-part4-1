# 你应该开始一个激情项目的 4 个实际理由

> 原文:[https://dev . to/skill _ pathway/4-practical-reasons-you-should-start-a-passion-project-16 H3](https://dev.to/skill_pathway/4-practical-reasons-you-should-start-a-passion-project-16h3)

我们最近在 DM 中发现了一个有趣的模式。开发人员似乎发现很难长期坚持一个项目，而不感到疲惫、厌倦或怀疑它是否能有所成就。

模式不是信息的主题，而是他们对这个问题的回答:“你为什么开始这个项目？”

最常见的答案是这样的。

> 我在寻找一些被动收入

虽然开发人员是允许我们构建任何东西的奇怪职业之一，但是进入一个项目只是为了赚取一些额外收入是非常不切实际的。所涉及的工作量通常比我们预期的要多，包括营销、打磨、支持和补丁。

像这样的项目，我们甚至还没花几周时间，就很容易被压垮。被无聊压倒，害怕没人买，害怕没人需要，不想把我们的名字放在我们不在乎的东西上。

# [](#enter-the-passion-project)进入激情项目

虽然激情项目不会解决任何资金问题，但如果你发现很难长期坚持项目，你可能更有可能以这种方式赚钱——以赞助、合作和货币化机会的形式，因为事实上你可以长期坚持。

以下是你应该开始一个激情项目的 4 个实际原因:

# 1)你不需要激励自己，它会激励你

如果你做你真正热爱的事情，它会给你动力，而不是你需要不断鞭策自己。

对于那些与我们真正感兴趣的事情无关的“副业”，我们不得不每天督促自己继续努力，而这对我们大多数人来说是不可持续的。

# 你会经历更少的倦怠，因为激情给我们能量

这与最后一点相似——但是理解一个真正的激情项目是如何证明倦怠的真的很重要。

想想那些你感到筋疲力尽的时候——除了回复到基本的人类需求，通常还有一些你热衷于做的事情，而不是你需要做的事情。

在这段精疲力尽的时期，也许你是那种浏览口袋妖怪(Pokemon subreddit)的人——也许你正在浏览脸书上搜救犬的照片——不管是什么，这都是你精疲力尽时会做的事情。顾名思义，它不会让你筋疲力尽。

将这种激情(有些人可能称之为拖延症)嵌入到一个实际的开发项目中，你会发现倦怠已经大大减少或者几乎不存在了。

# 你可以开始考虑长远，因为激情是终生的

当我们的焦点是金钱、名誉或类似的东西时，我们往往希望尽可能快地到达那里。通常是因为我们选择的道路是我们不热衷的，我们不想再为之努力，我们不顾一切地想尽快解决这个问题。

这有利于短期思维，并且(毫不奇怪地)与你的金钱或名誉目标背道而驰。(实际上)没有“一夜成功”——这种类型的人通常会努力工作多年，直到事情发生连锁反应，给他们所需要的东西。

如果你想赚到真正的钱，你需要有几年甚至几十年的心态，而不是几个月。最实际的方法是把它和一个激情项目联系起来。谁在乎要花多长时间-你喜欢做它！如果第一年失败了呢？重建它！将业务转向完全不同的模式。

你不再不顾一切地去完成它，所以享受这个过程——这就是赢家的产生。

# 你将会建造一些让你引以为豪的东西

当你创造了一些你真正感到自豪并想告诉所有人的东西时，这真的很酷——当它不是一个赚钱的计划或你出于其他原因而做的事情时，你会对你创造的产品感到更加舒适和自豪。

这也给了你一个机会和其他有相似热情的人合作，他们也认同你的想法——加倍的棒。
