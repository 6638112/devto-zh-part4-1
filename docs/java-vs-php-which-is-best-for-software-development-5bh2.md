**什么是 PHP？**
PHP 是一种广泛有用的脚本语言，在 1995 年发布后，迅速成为 web 开发人员公认的服务器端决策语言。它有一点灵活性，因为它是为 web 而构建和制作的，而不是为 web 而调整的语言(如 Ruby 或 Python)。今天，网站的主要部分仍然在 PHP 上运行，PHP 程序员仍然对其工作非常感兴趣，因为它是 WordPress、Drupal 和 Joomla 等高管框架(CMS)以及 Laravel、Symfony 和 CakePHP 等各种现代系统的基础，这些系统加速了这种成熟语言的开发。

**JAVA 是什么？**
Java 被结构化为一种广泛有用的编程语言，用于结构化独立的应用程序。当 Java 在 1991 年被 Sun 公司发布时，它首先被用来为像录像机这样的用户设备编程。

Java 是一种集成的语言，所以当你积累代码时，对于运行你产品的特定工作框架来说，它已经完成了一半。它的应用程序被累积成字节码，可以在使用 Java 虚拟机(JVM)的情况下继续运行。JVM 有助于克服源代码和 PC 获得它的 0 之间的任何问题。任何引入了 JVM 的机器都可以运行 Java。

在开发中，Java 基本上是 web 的服务器端语言，也是 Android 平台上通用开发的决策编程语言。此外，作为一个 Java applet，它仍然有一个不太差的接近前端的特性，尽管出于安全考虑，它已经不再被支持。
