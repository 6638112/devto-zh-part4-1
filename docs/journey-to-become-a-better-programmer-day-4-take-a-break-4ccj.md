# 成为更好的程序员的旅程(第 4 天，休息)

> 原文：<https://dev.to/montanacoder/journey-to-become-a-better-programmer-day-4-take-a-break-4ccj>

休息一下...直到最近，我都没有休息，而是坐下来/站着，埋头苦干。我从一些编码爱好者那里听说了一种叫做[番茄工作法](https://francescocirillo.com/pages/pomodoro-technique)的技术，在此之前，我从未听说过这种技术，并对它产生了浓厚的兴趣，这是大约 7 个月前的事了。我真的认为不值得花那么多时间来专注于我的工作。大约一周前，我开始尝试这种技术，我必须说它太神奇了！我强烈建议每个人至少试试这个。

我所学到的:
如果你和我一样，我几乎对休息感到内疚。我努力工作才得到这份工作，我想让他们开心。这对我来说是一件很难克服的事情，在做了 Pomodoro 一周后，我的工作质量有所提高，我能够真正专注于某件事情，然后放松大脑，然后再次投入工作。这让我有了今天的发现，因为我没有因为休息而紧张，我让我的大脑思考一些我们已经计划好的家庭项目。休息过后，我花了一秒钟看了一下这个问题，答案就像是从屏幕里出来的，然后给了我一巴掌。我在 chrome 上添加了这个[扩展](https://chrome.google.com/webstore/detail/marinara-pomodoro%C2%AE-assist/lojgmehidjdhhbmpjfamhpkpodfcodef?hl=en)，它真的有助于保持任务/休息。记得休息一下，没事的。
