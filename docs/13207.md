# Typescene 框架:快速入门

> 原文：<https://dev.to/jcormont/typescene-framework-quick-start-43jf>

我花了 5 年时间自己写了一个 JavaScript 框架。

Typescene 是一个用 TypeScript 编写的健壮的前端库:强类型、无依赖、无废话。这对桌面类(或移动)应用程序来说非常好，但对博客和其他内容来说就不那么好了。它不是由一些大公司支持的，甚至不是一家初创公司，但它是由我创建的:一个致力于创建一个严肃的无依赖性框架的开发人员。
