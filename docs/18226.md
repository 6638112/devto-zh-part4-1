# VS 代码键盘快捷键

> 原文:[https://dev.to/fishj123/vs-code-keyboard-shortcuts-5feh](https://dev.to/fishj123/vs-code-keyboard-shortcuts-5feh)

如果你和我一样，那么你可能会发现自己一有机会就伸手去抓鼠标。但是，也许你已经在 YouTube 上看了你最喜欢的开发人员，看到他们如何只用键盘浏览他们的代码。这就是我发现自己所处的情况，我意识到我想掌握键盘——所以我花了一些时间来学习快捷键。除了看起来很酷的明显好处之外，少用鼠标多用键盘可能会提高你的编码速度。如你所知，时间就是金钱，所以让我们开始学习 VS 代码的一些最好的键盘快捷键。

我将只介绍我最常用的快捷方式，我在本文的底部提供了一个备忘单，总结了我在本文中讨论的快捷方式。如果你想查看 VS 代码快捷键的完整列表，你可以在 VS 代码中按 CTRL + K 再按 CTRL + S 来显示所有快捷键的列表。如果你只是想看看我最喜欢的，那就继续读吧。

**注释掉代码块**
*窗口:突出显示文本+ctrl+/*
T6】MAC:突出显示文本+ ⌘ + /
我最喜欢的快捷方式就是快速注释掉大块代码。这是我学到的第一个捷径之一，因为它确实是最有用的。注释掉代码是我调试过程中必不可少的一部分，所以我真的不能没有这个。

**从右向左缩进代码**
*windows:ctrl+[*
*MAC:⌘+[*
好的代码需要适当的缩进，就我个人而言我喜欢用 TAB 来缩进我的代码，但是当你缩进太多并且你想把代码块向另一个方向(从右向左)移动时会发生什么呢？这就是携程的作用。现在我有了更漂亮的扩展，可以自动格式化我的代码，所以我更少使用这个扩展，但它仍然是一个需要了解的有用扩展。

**突出显示文件**
中的所有代码*windows:ctrl+a*
t30】MAC:⌘+a
这是一个好记也非常有用的认识。如果文件相当大，一次选择文件中的所有文本可以节省大量时间。

**突出显示该行中的所有代码**
*windows:ctrl+l*
*MAC:⌘+l*
这是一种快速突出显示光标当前所在代码行的方法。

**将一行代码向上或向下移动**
*WINDOWS: Alt +向上/向下键*
MAC:⌥+向上/向下键
我大量使用这一个来重新排列我的代码行的顺序。我主要用它来一次移动一行，但如果你首先突出显示文本，它也可以移动整个代码块。

**在 VS 代码中搜索所有文件**
*windows:ctrl+shift+f*
MAC:⇧+⌘+f
这是我最近刚刚学到的一个很棒的东西。如果你有一个加载的文件和文件夹在 VS 代码和你正在寻找一些具体的你可以使用这个快捷方式来快速找到它。我希望我能早点知道这件事。

**多光标在匹配代码**
*窗口:突出显示文本+ CTRL + D*
*MAC:突出显示文本+ ⌘ + D*
此选项非常适合在整个代码块或文件中更新相同的变量名或类名。该命令将在与突出显示的代码匹配的代码上放置多个游标，然后您可以一次性编辑它们。

**多光标**
*windows:alt+click*
*MAC:⌥+click*
这个有点作弊，因为它涉及到鼠标，但它是一个有用的快捷方式，所以我无论如何都要包括它。它允许您在文件中的任意位置创建多个光标。

**转到新文件**
*windows:ctrl+p*
*MAC:⌘+ p*
此快捷方式将弹出包含所有文件的下拉菜单，以便您可以快速导航到新文件。绝对是了解

**的好方法浏览你打开的文件**
*窗口:ctrl+tab*
*MAC:⌃+tab*
这是了解你是否会减少对鼠标依赖的重要方法。按住 CTRL 并按 Tab 键浏览每个打开的文件。按 SHFIT + Tab 会往反方向走。
