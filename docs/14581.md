# 是什么给了你一种错误的确定感/安全感...？

> 原文：<https://dev.to/_bigblind/what-gives-you-a-false-sense-of-certainty-security-1epi>

这个周日，在一次小型的家庭旅行中，我父亲评论道，建议自行车道(见上图)给了人们一种错误的安全感。咨询自行车道类似于普通的自行车道，但它们被画在一条通常太窄而无法拥有它们的道路上。这个想法是，两个方向的汽车使用车道之间的空间，但可能会进入自行车道，以超越相反方向的汽车。

任何看起来像自行车道的东西都会让骑自行车的人觉得这是路上的一条单独的车道，所以他们是安全的。他们不考虑汽车驶入他们车道的可能性。

这让我想到，它们是不是其他东西，尤其是在整个科技行业，让人感觉它们对安全、生产力、公司文化、..但是**不要**？我认为这些东西甚至经常会造成损害，因为我们只是假设它们的好处就在那里。
