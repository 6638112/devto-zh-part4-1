# 我如何才能满足我对网络世界无所不知的渴望？

> 原文:[https://dev . to/francos carpa/how-can-I-live-with-my-贪得无厌的求知欲望-39m3](https://dev.to/francoscarpa/how-can-i-live-with-my-insatiable-desire-of-knowing-everything-about-the-web-world-39m3)

我是一名 web 开发人员，我每天都在学习 web 世界的新事物:一天学习新的 HTML 元素，一天学习新的 CSS 属性，一天学习新的 UX 模式，一天学习新的 a11y 规则，等等。我每天都在努力学习如何让网络成为一个对每个人都更好的地方。但是这种努力正导致挫折。不是因为我意识到我不喜欢为网络开发，恰恰相反！我如此热爱这个世界，以至于我想知道关于它的一切，但我知道这是一个不可能的愿望。你不可能对一个话题了如指掌。所以我在这里问这个可爱的 dev.to 社区:我怎么能忍受这种挫折？我该怎么管理？你以前有过这种感觉吗？
