# 万人迷-视觉:让网络更容易访问

> 原文:[https://dev . to/vt Norton/万人迷-vision-making-the-web-more-accessible-3hgh](https://dev.to/vtnorton/heartthrob-vision-making-the-web-more-accessible-3hgh)

通过简单的配置，您可以通过 Azure Cognitive Services 自动为您的图像添加描述，从而使您的网站更易于访问。

在加载你的页面后，万人迷视觉将把你页面上每张照片的 url 发送到微软 Azure 认知服务，以创建图像上显示的内容的描述。

注意:我们建议您为网站上的每个图像手动创建所有替代文本。这将比使用万人迷视觉更能提高你网站的可访问性。

看看文档中的例子([http://heartthrob.vtnorton.com/vision.html](http://heartthrob.vtnorton.com/vision.html))和 GitHub 库([https://github.com/vtnorton/heartthrob-vision](https://github.com/vtnorton/heartthrob-vision))。
