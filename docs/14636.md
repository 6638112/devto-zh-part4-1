# 偶尔睡一会儿觉，有三种方法可以让你的职业发展充满活力

> 原文:[https://dev . to/EGF/three-way to-搞活你的职业发展——偶尔睡一觉 4dbd](https://dev.to/egf/three-ways-to-invigorate-your-development-career-while-getting-some-sleep-once-in-a-while-4dbd)

你刚刚工作了 8.5 个小时，可能还得做些家务，你已经让一堆孩子穿上了睡衣，去了 3 次洗手间，喝了 4 杯水。现在你要熬夜到凌晨 1 点，处理个人项目，提升你的开发技能，对吗？

如果你和我一样，当孩子们上床睡觉的时候，你会拖着自己上床睡觉。刷牙洗脸:纯可选。你不可能醒着坐在电脑前。不想把所有的空闲时间都花在电脑后面？

如果你有一个家庭和家务要照顾，你很难跟上这个行业。同时你也在全职工作。也许你不是刚出校门的大学生，没有时间编码到深夜，生活和呼吸代码。那么你什么时候有时间学习呢？你如何对自己的职业发展保持激情？

播客:我最喜欢的提升方式之一就是听播客。无论是在工作的时候，还是在我的情况下，我都喜欢在开车的时候听播客。我每隔一周去一次办公室，车程为 90 分钟，所以我可以在往返办公室的路上听几个关于发展的播客。我可以在割草的时候再听一个小时左右。即使我没有时间完全沉浸在某项技术中，听到关于它的有趣对话也会让我记忆犹新。

人际关系网:尽管我主要是一名远程员工，但我确实每隔一周去一次办公室。能够见到和你一起工作的人并享受一些共同工作的时间是很好的。与你习惯于在网上聊天和发电子邮件的人进行面对面的午餐会议有助于激发活力。

当你不能亲自见面时，和你公司的其他开发人员保持联系，甚至是以前的同事和其他开发人员。保持渠道畅通，讨论即将到来的新技术，分享链接和想法，并解决问题。

**主题列表**:保存一个关键词列表，下次你有空的时候可以在手机或桌面上搜索。拿我来说，我的记性很差，所以我保留了一个流行词和其他想法的列表，我想在有时间的时候研究一下。当我的手不空闲时，使用语音备忘录或 Siri/Alexa/任何方便的东西。

我发现跟上一些较新的技术，甚至仅仅是熟悉最新和最棒的技术，并与其他志同道合的开发人员交流，是让我对自己的开发职业保持兴奋的良好开端。我希望这些想法可以帮助其他开发者，不管是传统的还是非传统的。
