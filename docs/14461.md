## [](#whats-in-the-box)盒子里是什么

组件按照视觉可用性标准分组(页脚、英雄、导航条..)，并作为 [pug](https://pugjs.org/api/getting-started.html) 模板直接在布尔玛(不依赖 Javascript)之上设计，这使得它们非常容易扩展和定制。

* * *

## [](#how-to-use-the-project)如何使用项目

要使用 [bulma components](https://github.com/bulma-css-templates/bulma-components) 项目和组件，只需输入:

```
# Clone the repo
$ git clone https://github.com/bulma-css-templates/bulma-components.git
$ cd bulma-components

# Install dependencies
$ yarn

# Run dev server with live preview (Browsersync)
$ yarn start 

# Or make a production build
yarn build 
```
