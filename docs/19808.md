## 什么是波洛涅克斯？

Poloniex 是一个纯粹的加密-加密交换平台。因为有许多交易所存在，所以 Poloniex 总是被证明是第一个也是最好的交易所(所有交易所的领导者)。Poloniex 为交易者提供了最安全的高级交易选项，因此它是大多数交易者寻找的对象。

发射年份:2014 年
创始人:特里斯坦·达戈斯塔
被收购方:Circle
总部:美国马萨诸塞州波斯顿。
服务可用国家:全球(包括印度)

只有拥有一些加密货币的交易者才能在波兰交易所进行交易。与其他交易所一样，Poloniex 不支持用法定货币购买加密货币。它只支持加密到加密的交换。

## 波洛涅克斯的特殊功能

1.  它几乎是美国最好的加密货币交易所，支持世界各地各种加密硬币的交易。
2.  Poloniex 接受全球交易所，任何人都可以使用它进行交易，只要他们的国家没有针对加密货币及其交易的法律。
3.  Poloniex 是一个开源的交易所，交易者可以查看交易接口代码的执行情况。
4.  波洛涅克斯交易所为交易者提供了一个学习和检查交易执行的简单方法。
5.  Poloniex exchange 包括许多可以有效使用的数据分析工具。
6.  Poloniex API 允许交易者创建管理账户和分析数据的工具。
7.  Poloniex 支持许多功能，如交易、借贷、保证金交易等等。
8.  Poloniex 遵循“做市商-接受者模型”，这意味着用户在过去 30 天的交易量越高，用户支付的费用就越低。
9.  Poloniex 交易所支持超过 56 种不同的加密货币和超过 117 个交易对(在撰写本文时)。
