# 处理机器学习项目时要采取的步骤

> 原文：<https://dev.to/flippedcoding/steps-to-take-when-working-on-a-machine-learning-project-448d>

从贝叶斯统计到产品经理，一个机器学习项目有很多部分，它们都必须一起工作才能成功。你的团队将由不同专业的人组成，这是机器学习如此酷的原因之一。每当有很酷的事情发生，人们都会为之疯狂，机器学习团队也不例外。一些企业认为，如果他们有足够的数据，他们可以投入一个团队来处理这些数据，并获得难以置信的有价值的结果。

有些公司甚至在不知道要用这些数据做什么之前就购买了大量的数据。机器学习不是所有问题的答案，尤其是如果你没有对你的数据提出正确的问题。为了确保你对机器学习项目的流程有一个实际的理解，我们将涵盖你需要包括的五个基本步骤。

## 决定解决什么问题

拥有大量数据并希望从中发现有用的东西并不是一个好策略。你应该有一个具体的问题，你试图了解更多或预测。如果不涵盖这第一步，你可能会花费大量的时间和金钱来转动每个人的轮子。机器学习，目前来说，不是笼统地“思考”。你可以整天把信息扔给一个人，他们会开始把它做成模式，不管它们是不是真的。

你还不能用程序做到这一点。你还是要根据你给它的信息，告诉它你到底在找什么。记住，数据不是解决方案。它是一种工具。在没有特定目的的情况下向机器学习团队扔数据就像用香蕉给电路布线一样。你肯定会有所收获，但可能不会是你所期望的。

## 决定你需要什么参数

一旦你确定了问题，你的团队就可以开始工作了。现在，他们需要弄清楚他们需要什么数据，以及如何从他们现有的数据中获取这些数据。比如汽车的属性。你可能为一家汽车经销商工作，他们想知道如何最大限度地提高某一型号汽车的利润。所以你可以看看以前买过车的顾客的人口统计数据，一天中的时间，天气，也许还有最流行的颜色。

这些只是一些简单的例子，你可以用在你的算法中。这一步对于确定你需要多少数据和什么样的数据，以及训练你的机器学习模型需要多长时间至关重要。参数越多，结果就越精确。虽然太多的参数会使你的模型超级慢，从而导致其他问题。

## 选择合适的训练数据

你知道你要解决什么问题，你也知道你需要什么样的参数来着手解决它。现在你需要得到正确的训练数据。如果你在阿拉斯加销售汽车，使用田纳西州的数据对你没有任何好处。您的数据必须与您试图解决的问题保持一致。这是一家公司考虑购买数据或从用户那里收集数据的时候了。

你的机器学习模型只会和你给它的数据一样好。大多数人关注算法是因为它们是多么前沿，他们忘记或忽略了好的、干净的数据的重要性。在项目全面展开之前，这是容易被忽视的事情之一，如果你等待纠正的时间太长，它真的会咬你一口。

## 决定使用哪些算法

这就是你的智能机器学习人员的用武之地。他们应该能够获取你正在使用的参数，并找出使用什么算法，或者如何调整它们来满足你的需求。他们会比较训练时间、错误率和预测值。根据他们拥有的资源和时间，他们将通过小型测试选择什么算法能做得最好。

在更多的统计分析之后，他们会开始计算出代表你的算法的数学公式。然后他们会做更多的测试和分析。一旦他们将误差限制在可接受的范围内，并与团队的其他成员讨论了细节，开发人员就可以接手了。也有可能你的机器学习人员会为算法编写代码。

## 编写代码

这是事情变得更熟悉的时候。您仍然会和您的开发团队一起经历类似的过程。将会有冲刺和代码审查和部署。这个小组将导致这一时刻的所有理论联系起来。他们将编写实际训练机器学习模型的代码，并将它与用于训练的数据联系起来。在这一点上，事情真的在发展，很难对项目做出根本性的改变。

当开发团队完成时，你应该有一些软件，它会根据你给它的输入参数给你预测或优化。这就是整个过程的本质。你决定如何处理软件提供的信息取决于你要解决的问题。

## 检查结果

你成功了！这个项目已经完成，你可以看到一些有趣的结果。首先，你可能想检查你的结果是否有意义。使用全新的数据来看看你的模型是否成立。随着时间的推移，这可能会慢慢发生，因为您获得了更多的用户数据，或者您可以通过使用与您训练时不同的数据集来加快速度。

在你做任何大的演示之前，确保你的结果对你和团队的其他成员有意义。保持你的测试与最初的问题相关。我们在机器学习方面还不够先进，不足以让一个软件进行批判性思考，所以要由你来做一次理智检查。

这一过程背后有更多的细节，它们都变得非常有趣。此外，有了 TensorFlow 和 Brainjs 这样的库，web 开发人员可以更容易地开始测试机器学习。这个领域对任何足够勇敢(或足够疯狂)的人都是敞开的。

你认为机器学习值得大肆宣传吗？我认为它给我们一些老问题的新解决方案，随着时间的推移，它会变得更好。但这往往会引发哲学问题，比如，机器能思考吗？我们应该在机器学习上走多远？
