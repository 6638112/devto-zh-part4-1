# [](#context)上下文

我觉得每次我开始写作，我都被迫说“有一段时间了”，通常是因为它已经过去了。在我的上一份工作中，我几乎没有剩余的精力和时间来写作。我喜欢表达自己，并与他人分享我所知道的；然而，当工作/生活平衡时，只有工作，没有生活，很难有动力去随意写作。

失业是一次令人羞愧的经历。我当时在一家初创公司工作，虽然#startuplife 令人兴奋，但当你有一群乌合之众的唯利是图的开发人员跟着同一个鼓点前进时，这一切都很有趣，直到他们不再能够支付你工资。

人们听你的话。换挡毫不费力，富有挑战性，令人兴奋。这能行吗？谁知道呢？我们明天会破产吗？谁在乎。让我们今天把这件事做完，然后抱最好的希望。

# 你为什么要重新开始？

我觉得有必要。我对 JellyFin 有一个具体的想法，最近经历了过多的技术面试，但最终得到了一份工作，并接受了它，我渴望回去工作。

我申请的一份工作有一个技术挑战，就是创建一个连接到 Giphy API 的应用程序，你需要能够搜索任何 gif，将其保存到你的个人资料中，并能够对其进行分类。对于这个挑战，我选择了 Firebase 和 React。我花了很多时间在上面，因为我想让它好看，但是我在上面花的时间可能比我需要的要多(考虑到我没有得到这个角色)。不管怎样，我从中获得了知识。

嗯，我从与 Firebase 互动和使用谷歌认证中学到的东西，我正在应用于 JellyFin。在接下来的几个月里，我会一点一点地去做。如果你看到一些不合适的东西或更好的方法，请随时给我发一份公关。
