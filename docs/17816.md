# Libra -脸书的加密货币将于 2020 年投入使用

> 原文：<https://dev.to/harryzaara/libra-facebook-s-cryptocurrency-will-be-in-use-by-2020-4ffi>

## 1。天秤座-是什么？

Libra 是一种由脸书创建的加密货币，由脸书创建的新版区块链提供支持。根据脸书的说法，Libra 是一个“全球货币和金融基础设施”。Libra 将于 2020 年投入使用，并允许脸书用户在世界各地进行金融交易。天秤座是天秤座加密货币的一个单位，用$表示，其值意味着保持稳定，就像 stablecoins 一样。

## 2。为什么脸书推出天秤座？

脸书推出新的加密货币 Libra 背后的两大看点如下:

1.  以克服监管障碍和对脸书的反垄断担忧。
2.  帮助 17 亿没有银行账户的人使用他们的脸书账户进行全球交易。

## 3。天秤座的执行权力

数字货币 Libra 将由“Libra 协会”提供服务，这是一个位于瑞士的独立非营利组织。它的两个主要功能是

1.  验证利博瑞区块链网络上的交易。
2.  管理储备金，为社会事业分配资金。天秤座协会理事会将是天秤座协会内部的管理机构，负责投票决定政策和运营决策。一旦该货币在 2020 年推出，Libra 协会将退出其领导角色，该协会的每个成员都将在 Libra 的治理中拥有平等的投票权。

## 4。天秤座——我们该如何购买和使用？

脸书没有详细提到该货币将如何发行，但看起来它遵循一种混合的方法。
Calibra 是 facebook 加密货币 libra 的数字钱包，用于存储 libra 货币。这将使用户可以把它发送给任何有智能手机的人。这种 calibra 钱包的用法将在 Messenger、WhatsApp 中提供，并作为一个独立的应用程序。Calibra 将提供一种通过扫描二维码向商家付款的快捷方式。
