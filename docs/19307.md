# [](#the-web-application-hackers-handbook)[《网络应用黑客手册》](https://amzn.to/2IBFkuD)

我通常不看科技书籍。在大学，我会从大学书店买书，但如果我能避免阅读它们，我会这样做。我主要需要他们做课堂作业。大多数时候，教师会在他们的网站上以 PowerPoint/幻灯片或 PDF 格式提供课堂材料。

那么，为什么[这本书](https://amzn.to/2IBFkuD)？

我在奎托斯 Secureinfo 工作期间，雇佣我的人特别推荐了这本书。他们说“如果你对这种工作很认真，读读这本书吧”。

我照做了。我一直在重读它的一部分。

这是一本有点老的书，但包含许多有用的章节，有助于让你的大脑寻找一般的漏洞。如果你想学习如何像黑客一样思考，这是给你的书。

作者开发了 burp suite，这本书鼓励使用它，而且我想不出任何好的替代方法，以至于我考虑编写自己的实现。形势如此严峻！

在它的封面中，您可以找到有关映射攻击面、情报收集、漏洞发现、利用、攻击 javascript、mysql、php、perl、web 服务器、数据库、WAFs (web 应用程序防火墙)、本机应用程序、源代码、随机令牌、cookies 等信息！

从一个程序员的背景来看，仅仅用这本书就可以探索一个世界。和我一起踏上通往黑客帝国的旅程吧。
