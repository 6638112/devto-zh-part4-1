# [判断合同](https://github.com/loaiabdalslam/Justice/)

选举和民意调查的分散化应用
正义是一种分散化的应用，旨在举行公平的选举，在所有人的视觉和听觉上没有欺诈。

我遇到了很多问题，因为我是区块链领域的负责人，尤其是以太坊。因此，我决定让这个项目开源，但根据我自己的条款和规则，在不久的将来与这个项目的每一个贡献者进行商业交易。

整个想法是，政府为选举支付了数十亿美元，由于一些国家和政府缺乏司法独立，每个人都在质疑他们的诚信，所以我决定将我的热情转向这一点:区块链网络上的应用司法

从那时起，我准备了一个一起散步的计划:

*   准备选举室，每个选举室包含若干选民号码(身份证号码)
*   将投票人的房间连接到投票人，并确保他们附属于那个房间
*   使用 angular 或 react 等前端框架之一来完成项目
*   设置以启动 ICO，从而将两个系统集成在一起

### 安装

正义要求 [Node.js](https://nodejs.org/) v4+运行。
[Meta Mask](https://github.com/MetaMask/metamask-extension/releases/tag/v5.3.1) 客户端提供商
[松露](https://www.trufflesuite.com/)区块链网络

安装依赖项和 devDependencies 并启动服务器。

```
$ cd Justice
$ npm install
$ truffle migrate
$ npm run dev 
```
