我打算买一台新的笔记本电脑，我倾向于 Linux PC，因为我已经用 Linux 代替 MacOS 工作了几个月，我看不出 MacOS 有什么值得 MacBook 价格的大优势。

还有一台 Windows PC，但我甚至不考虑它，因为我喜欢在一个类似的环境中工作，我的所有服务器都是(UNIX*)的，我不使用它。网上的东西。

所以，我很好奇:**你为什么决定选择现在的工作站？为什么是 MacBook？或者为什么是 Linux/Windows PC？**
