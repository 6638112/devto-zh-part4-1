# 47 岁的大学生

> 原文：<https://dev.to/lilleonz/microverse-student-at-47-2gj2>

47 岁的我在干什么？

47 岁的我在做什么？在相隔十三年的两场战争之后。在过去的 14 年里，帮助我的爱人度过了最艰难的战斗。当她为她的三个儿子，或者说两个年轻人和两个孙子而战的时候。脑癌是一种野兽，它不仅仅来自一个人。而是来自一个成长中的家庭。

现在有男生来指导。我今年 47 岁，开始了编程生涯。作为 X 世代的人，我应该处于计算机革命的开端。所以现在，当千禧一代似乎应该锁定它时。我在摸索着回去。

在过去的几年里。我一直在努力自学。Udemy.com，YouTube.com 和 FreeCodeCamp.org。我又开始了。

这一次是一家名为 Microverse 的新公司。他们通过给你更多的结构并让你和一个编码伙伴一起工作来帮助你。然后，当你完成一个项目时，你前面的学生会对它进行同行评议。他们给你问题的指点和解决问题的更好方法。到目前为止，我已经完成了 Microverse.org 课程的 HTML/CSS 部分。加上我目前学到的东西。但现在它变得有意义，也更有粘性。

从对 HTML 和 CSS 的基本理解开始。现在对使用 git、HTML、CSS、linters、SASS 和媒体查询有了更好的掌握。它们有时仍会让我不安，但我不再害怕了。47 只是一个数字。没多大关系但是给你更多的生活经验。

有些概念可能很难，但生活也是如此，所以要战胜它。这些概念会被记住。就像生命的其他部分一样储存在我们的大脑里。

所以在 47 岁，我是一个编码婴儿。学习走路和说话。我的页面是用 HTML 创建的，但是用 CSS 我可以改变它的外观。我正在学习更好地沟通。自从失去母亲后，学会更好地引导男孩们。我们可能看起来像 90 年代末的网页。但是随着时间的推移，它将会充满 Javascript、Ruby 和算法。

是时候坐下来学习并享受旅程了。
