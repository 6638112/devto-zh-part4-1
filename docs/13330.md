# DIY Cheesy 搭讪线和 Twilio

> 原文:[https://dev . to/stefi 0227/DIY-cheesy-pick-up-line-and-twilio-2p9b](https://dev.to/stefi0227/diy-cheesy-pickup-line-and-twilio-2p9b)

嘿伙计们，欢迎回到我的频道！开个玩笑，不是这样的，我只是不知道怎么开博客。我甚至不是优步。

无论如何，几个月前，我决定参加我们公司的健康保险激励计划。我预约了一次常规体检——多拿一点钱，确保一切都像一台运转良好的机器一样运转，这有什么不好？

在我体检的前一天，我收到一条短信，确认我和我的医生的预约。选择一个进行确认。这让我开始思考这些短信是如何获得和发送的。当我在网上浏览大量信息时，我发现了 Twilio。

Twilio 是一种简单而有趣的方式，让企业以对他们有意义的方式与他们的成员交流。Twilio 有短信、语音和视频选项。Twilio 会教你如何使用他们的产品。我选择了一个电话号码，我希望我的信息来自，我去了比赛。我开始了我的第一个探索:语音功能。(让我先说一个事实，我的脑海中不断闪过随机的歌词。)多种语言选项让拥有国内和国际用户的公司轻松应对。我选择了英语，并带有英国口音，只是为了让我的选择更优雅一点。不管出于什么原因，我脑海中首先闪现的是伊吉·阿德利亚的话，“首先，我是最真实的。”我拨打我选择的电话号码，然后我听到一个自动语音在说歌词。我很兴奋；超级无痛教程，我在探索中获得了经验。

[![](../Images/e7ae90a718f4ab9eda1743fed6eb5ad5.png)T2】](https://i.giphy.com/media/d1JforDQzGxd7nG0/giphy.gif)

Twilio 教程中的第二个任务是实现一个 dog facts API，并在你发送你选择的电话号码时接收一个随机的 dog facts。在完成了随机狗事实教程后，我决定我准备展开我的翅膀飞起来，写我自己的函数。我在想一些你可以在短信中得到的东西，然后选择搭讪语。谁不喜欢俗气的搭讪台词？但我知道我将需要建立一个像在狗的事实教程这样的 API。

我决定使用 Sinatra，这是一个用 Ruby 构建的微框架，有助于使编写 API 变得简单和可实现。我第一次使用 Sinatra，我按照另一个教程做了一个快速 API。我被困住了，很多次。我们能不能暂时停下来讨论一下关于放弃的说法有多正确？托马斯·爱迪生说过:“生活中的许多失败都是人们在放弃的时候没有意识到他们离成功有多近。”我坚持下来，完成了 API 的编写。我选择复制并粘贴一些我在网上找到的搭讪台词，因为我没有那么多锦囊妙计。我将它们粘贴到一个文本文件中，并添加了一条路线。一切都很好，直到它不工作。

[![](../Images/8c0787fb4b2d00795d00b10ebd2706d8.png)T2】](https://i.giphy.com/media/UZo0oqyGoy6C4/giphy.gif)

由于这是一个轻松的项目，我选择继续使用 Heroku，它免费且简单，就像推送到 Github 一样。当然，我会得到一个错误，因为我丢失了一个文件，感谢日志让我确切地知道我丢失了什么。添加 Configure.ru 文件并执行快速捆绑命令。它工作了。

现在我把它放在 Heroku 上，我感觉好极了；让我们完成这个，把那些搭讪的话写进短信里。我很快将路线添加到我全新的 API 中，感觉棒极了。我忘了 Heroku，因为它是免费的，当它不运行时会有一些停机时间。不管用，我还没说完。因为我忘记了，我变得很恼火，并且正在改变一些东西，因为很明显，我认为正确的东西并不正确。我决定后退一步，因为我需要休息一下，而且已经很晚了，超过了我的就寝时间。

[![](../Images/cc2a8302e7b6ba8eb21b8178a20afaac.png)T2】](https://i.giphy.com/media/xT8qBvH1pAhtfSx52U/giphy.gif)

我梦想着我需要解决的问题，我看到自己提出了一个有意义的解决方案。我整天在工作中思考这个问题，恨不得马上回家解决它。当我回到家，打开我的电脑，我开始做更多的研究，寻找其他的选择，因为我不能是唯一一个处理这个问题的人，对吗？我在 YouTube 上看到一个 Twilio 教程，我说，“这是他们的产品，他们当然知道该怎么做。”我改变了我的原始代码中的一切，但它仍然不工作。奇怪。我不得不打电话给一个朋友。我花了一段时间才意识到这一点，但我喜欢大声处理。我陷入了困境，向其他人解释事情的运作方式以及我对结果的预期有助于我驱散迷雾。

我打电话给我的妹夫，他是我所有编码问题(以及吊扇安装，但是我们下次再讨论这个问题)的专家。我们最终回到原始代码，讨论它为什么以及如何不能工作，因为它是有意义的。我们再试了一次，成功了。再测试一次，以防万一。当我给 Twilio 的电话号码发短信 1 时，我可以确认我的手机何时收到蹩脚的接机线。

我为自己没有放弃并完成了为自己设定的目标而感到自豪，这篇博文也包括在内。我喜欢向人们展示我所做的事情，这不仅是为了证明我可以做一些很酷的事情，比如建立一个随机接电话短信项目，还因为我希望人们意识到，只要他们不放弃，他们也可以这样做。谁知道你最终会不会对自己有所了解，比如我是如何了解到我认为俗气的搭讪语有多好笑的。
