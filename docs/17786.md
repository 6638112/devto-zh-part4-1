# 3 面向学生开发人员的资源

> 原文：<https://dev.to/jasterix/3-resources-for-the-student-developer-3k45>

学习一门新语言很容易，但使用它却是另一回事。作为一名全日制工科学生，由于 Flatiron 提供的结构，让自己沉浸在 Ruby(以及现在的 Javascript)中是理所当然的。

但是使用这些语言来编程仍然充满了随机错误、错误和困惑，包括我们为什么做某些事情以及以什么顺序做，以及可视化幕后发生的事情的总体困难。

幸运的是，我能够诊断我的挫折并开出相关的处方。但是人们经常默认最大的资源

*   谷歌
*   堆栈溢出
*   MDN/W3

没有意识到作为一个初学者，作者使用的语言会让你进一步陷入混乱和混乱的黑洞。

幸运的是，一生的问题解决教会了我如何学习该学的东西。这意味着:

1.  隔离问题
2.  理解障碍
3.  标记关系

因此，与大多数编程入门文章不同，我不会给你人们通常提供的相同资源，并期望你找出如何最好地利用它们。

### 1。视频——特别是 YouTube

有很多优秀的 Youtubers 创建编程视频。有些很棒，有些很可怕，有些很适合 ***你*** 。

找一个会说你的学习语言的人是一个反复试验的过程。但是下面我列出了我最喜欢的教练名单:

*   [Rails 简介](https://www.youtube.com/playlist?list=PLmfGSTDbvdnaPdr5e2ihINxl7Ee9QoVkc)

*   [JavaScript 简介](https://www.youtube.com/playlist?list=PLmfGSTDbvdnacO0_lWne912GWSV8HYNyv)

### 2。代码库

有些人只是代码忍者。他们的代码干净、有效，最重要的是，易于遵循。我不会在这里链接任何内容(因为我还没有得到许可，但是每当我遇到代码符合这些标准的开发人员时，我会确保将他们标记为书签以供将来参考。

### 3。沙盒(沙盒)

我自称是应用程序管理员。因此，我为自己拥有一套能够实现最佳应用的应用工具包而自豪。就沙盒而言，并非所有的沙盒都是平等的，所有的沙盒都提供了最佳的功能。

我最喜欢的三个测试代码的应用程序是:

*   非常适合用大多数语言编写独立代码。我在 Javascript、Python 和 Ruby 中使用过，效果很好
*   Codepen -非常适合创建或测试前端代码片段。如果 HTML 很重要，默认为 Codepen
*   Python Tutor 的[可视化工具](http://pythontutor.com/visualize.html#mode=edit)——您的代码是否中断或给了您一个意外的结果，通过使用该工具可视化函数中的每一步来隔离问题。Python Tutor 在网站上也有导师，他们可以看一看你的代码和/或解释为什么它不工作

这些是我在熨斗大厦工作的 7 个星期里观察到的事情。在接下来的 7 周里，我无疑会有新的或额外的资源来满足我最迫切的需求。与此同时，我希望这会让你的旅程有所不同。
