我是 Erika Heidi，Dev 教育团队的技术撰稿人。今天的综述包括 Kubernetes 上的 Istio、Ansible、MySQL Cache 等等！

事不宜迟，以下是本周的特色帖子:

### [如何在 Kubernetes 上安装和使用 Istio】](https://www.digitalocean.com/community/tutorials/how-to-install-and-use-istio-with-kubernetes)

在本教程中， [Katy Juell](https://twitter.com/katjuell) 向我们展示了如何使用 [Istio](https://istio.io/) ，这是一个[服务网格](https://www.digitalocean.com/community/tutorials/an-introduction-to-service-meshes)，通常与 Kubernetes 一起使用来管理应用的微服务之间的通信。

### [如何在 Ubuntu 18.04 上使用 Ansible 安装设置 Docker](https://www.digitalocean.com/community/tutorials/how-to-use-ansible-to-install-and-set-up-docker-on-ubuntu-18-04)

这是我在 DigitalOcean 社区最新发布的教程之一，它展示了如何使用 Ansible 在 Ubuntu 18.04 服务器上自动安装 Docker。自动化所有的事情！

### [如何在 Ubuntu 18.04 上优化 MySQL 查询缓存](https://www.digitalocean.com/community/tutorials/how-to-optimize-mysql-with-query-cache-on-ubuntu-18-04)

加速你的 MySQL 服务器，试试查询缓存！在由 [Francis Ndungu](https://twitter.com/francisndungu83) 撰写的本教程中，测试使用查询缓存时查询的性能差异，Francis Ndungu 通过我们的[为捐赠而写](https://www.digitalocean.com/write-for-donations/)项目发布了本教程。
