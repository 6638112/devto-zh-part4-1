# 我是如何平衡工作和生活的

> 原文：<https://dev.to/pwadhwani2592/how-i-m-managing-work-life-balance-i>

去年，我开始变得更加活跃，在日常工作的不同领域挑战自己，发表演讲，组织和参加聚会和会议，旅行，以及其他一些事情。最初，我非常兴奋，我一直在做不同的事情——只要我有时间，我就让自己忙着做上面的事情。我想做所有的事情，在某种程度上，我做到了。然而，到了年底，我注意到生活不一样了。我注意到我感觉更累了，没有得到足够的休息，吃了太多我吃不完的东西——这些都不是好事。我是一个喜欢阅读他人及其经历的人，我发现(在 Twitter/Medium/Devto/Google 上)有很多人有类似的经历。我意识到我想做我喜欢的事情，同时关心我自己，我的身体健康和心理健康。所以我开始着手做这件事，下面是我为此做的几件事:

1.  制定几个有意义的目标:当我意识到一次做所有事情对我来说并不好时，我列出了一个更有针对性的个人和职业目标清单，这是我今年要关注的目标。我也拒绝了很多机会，因为我知道我没有时间和精力做出有意义的贡献。

2.  经过反复试验，我找到了一个适合我的好时间表。我确保周末至少有一天我得到适当的休息，花时间和家人朋友在一起，不使用手机或笔记本电脑超过一个小时，除非真的需要。

3.  列出优先事项清单:每当我开始做一件事，我都会列出优先事项清单。这有助于我专注于最重要的事情，并更加关注它们。我也试着一次只写两到三个任务，其余的被添加到我以后要看的待办事项中。减少重要任务确实帮助我提高了注意力。

4.  改善时间管理:我把我正在做的每件事都定时，包括办公室工作和个人目标。例如，我决定每周花 4 个小时在我的个人目标上，无论如何我都要坚持——不多也不少。

5.  关注身体和精神健康:我确保每天有七八个小时的睡眠，并努力改善我的睡眠模式。我非常喜欢帮助我练习正念的[顶空](https://www.headspace.com/)应用程序。我也开始经常慢跑，并努力养成习惯。

6.  保持一致:我努力与我的时间表和我想完成的任务保持一致，以实现我的目标。

这些事情一起做，极大地帮助了我做得更好，同时，让我对生活感到更加快乐和满足。更重要的是，**这不是一天之内发生的** -我已经为自己努力了几个月，直到现在我才看到进步。**我失败了很多次**，比我承认的次数还要多，但正如他们所说的“如果你想让事情成功，你需要不断尝试。”我教会自己接受失败，然后**带着新的洞察力继续前进，重新开始。**

如果你对此有任何建议，请告诉我，因为我很想知道你是如何做到的。如果你发现以上有任何帮助，试一试，让我知道它是如何为你工作的。:)
