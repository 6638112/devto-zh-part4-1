Vue slots 是一种告诉 Vue 在组件主体中放置任何内容的方式。今天，我们将建造房屋来了解 Vue 插槽！

在我们开始之前，让我们把 Vue 组件想象成房子。为了能够进入一所房子，我们需要使用一扇门；在我们的组件中，我们的`<slot>`标签将作为其他组件进入的大门。让我们做我们的第一栋房子吧！
