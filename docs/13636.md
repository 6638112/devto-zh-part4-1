# [](#laker-sparks)湖人火花

[![Laker Sparks](../Images/f9afa38a92cfc2c709cbf3ecca0a8910.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--a2_t4Xzh--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://2018.cascadiajs.com/speakers/laker.png)

当您听说开发工作室时，您通常会想到一群拥有多年经验和教育的行业专家。但是新一代的开发人员并不满足于等着上大学，甚至是高中才开始游戏。作为 Roblox 上的信息体验主管，Laker Sparks 目睹了年轻的业余程序员转变为价值数百万美元的开发团队。在这个演讲中，他将分享他对这些年轻开发者的经历和挑战的学习和观察，然后讨论你可以用来帮助培养下一代开发者和创造者的资源和技术。
