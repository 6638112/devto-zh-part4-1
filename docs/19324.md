# 你是如何找到你的第一份开发工作的？

> 原文：<https://dev.to/torianne02/how-did-you-find-your-first-dev-job-c24>

对于所有被雇佣和工作的开发人员，我有一个问题要问你们:**你是如何找到你作为开发人员的第一份工作的？**我的意思是，你是如何找到第一份工作的？你是在 LinkedIn 上随机看到的吗？公司招聘板？一个朋友？老同事？Meetup？招聘会？等等。**你有计算机科学学位或者参加过训练营吗？你是自学的吗？**

我是作为一个寻找我的第一个角色的年轻开发人员来问的。我已经寻找了两个多月了，感觉我正在问自己这是否会发生在我身上。我知道两个月不算长，不要误会我的意思，但在找工作的过程中似乎总是有起有落，我目前正处于低谷。我认为这将有助于激励我(和其他年轻的开发者)去看看有经验的开发者是如何遇到他们的第一个机会的。

如果你还想加上**你找工作找了多长时间**，那也很好。
