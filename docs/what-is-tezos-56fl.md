# Tezos 是什么？

> 原文：<https://dev.to/harryzaara/what-is-tezos-56fl>

Tezos 是一个类似以太坊的区块链网络，用于开发和部署智能合约和分散应用程序。泰佐斯是全球第一个自我进化的区块链。

Tezos 是一个 P2P、分布式、无许可的区块链网络，旨在对区块链进行一些显著的改进和创新。

## Tezos 和以太坊的区别

Tezos 区块链的智能合约语言是迈克尔逊，而以太坊的智能合约语言是以太坊虚拟机(EVM)。EVM 和迈克尔逊都是基于堆栈的编程语言，它们通过将一系列数据元素转换成一些指令序列来工作。

迈克尔逊和 EVM 之间的主要区别是:迈克尔逊是以人类易读的文本格式编写的，而 EVM 是以字节来操作的。
迈克尔逊数据元素是打字的，但在 EVM 不一样。

## 烘焙——是什么意思？

Tezos baking 是签署并向 Tezos 区块链发布一个新的交易块的行为。向 tezos 网络添加新块的过程称为烘焙，也称为认可。

要获得代表资格，面包师需要至少 10，000 XTZ，如果他们有额外的代表股份，增加被选为面包师或代言人的机会。烘焙可以获得 16 XTZ 的奖励。
