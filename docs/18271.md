# 我们不是看事物的本来面目，而是看事物的本来面目。

> 原文:[https://dev . to/just zeros/we-not-see-things-as-them-as-we-see-them-as-we-are-2n 0](https://dev.to/justzeros/we-don-t-see-things-as-they-are-we-see-them-as-we-are-2n0)

领导者需要明白，心智模型是我们间接看到现实的透镜。

否则，我们会陷入天真现实主义的偏见，相信我们客观地看待事物，因为它们“真的”是这样。

这是一个问题，因为它让我们确信我们所感知的是真实和正确的。

确信某事可以激励我们采取“明显的”行动或反应。

这也让任何不同意我们观点的人显得无知、疯狂或愚蠢。

我们最好停下来问问自己:

1.  我现在使用的视角是什么，这个视角的子部分是什么？
2.  我可以从多个角度考虑情况吗？
3.  我的分析中是否遗漏了他们的观点？

任何特定的情况都有多个视角。

让我们试一个简单的例子:

你是 BIGCORP 的开发经理，你的公司宣布将在 2020 年实行“无限假期”政策。

1.  你对这一政策变化的本能反应是什么？
2.  你的直觉用的是什么视角？
3.  你能看到透视图的哪些子部分？
4.  你能从其他角度考虑一下情况吗？它们提供了不同的反应吗？
5.  你的分析可能遗漏了哪些观点？
6.  不同视角之间可能会产生哪些冲突/惊喜？

在你的团队中传播这一点可能会很有趣，看看他们有什么不同的观点和反应。

或者，花点时间给我回信，告诉我你的答案。

极品，马库斯
