# 像素天气

> 原文:[https://dev.to/aveeksaha/pixel-weather-2li7](https://dev.to/aveeksaha/pixel-weather-2li7)

### 像素天气是一个小巧紧凑的 8 位风格的天气小工具，为你的 Linux 桌面设计。

# [](#features)特性

*   它的透明背景允许它融入你当前的设置
*   始终放在您的桌面上，不会妨碍您。这意味着当你用`alt+tab`切换窗口时它是隐藏的
*   在暗模式和亮模式之间切换，分别用于较亮和较暗的背景。
*   干净的界面，没有杂乱，所有的选项都可以通过右键访问。

# [](#how-to-use)如何使用

*   安装软件包。
*   注册并从 [`Dark Sky`](https://darksky.net/dev) 控制台获取您的 API 密钥。
*   当你第一次打开应用程序时，会弹出一个设置菜单，把你的 API 密匙粘贴在那里。
*   您还可以更改天气更新的频率。
*   提交设置后，右键，刷新，等待天气更新。
*   您可以通过从“位置”菜单中选择来选取放置 widget 的位置。
*   右键单击小部件上的任意位置都可以访问该菜单。
