# 通过学习提高

当采取客观的方法来提高自己时，你应该努力以你目前无法理解的方式来提高。这可以理解为“催款克鲁格效应”的一个子集，这并不意味着让你觉得自己很愚蠢，这只是我们都面临的现实。

正确饮食、充足的睡眠和水以及锻炼都是自我改善养生法的关键部分。但是还有另一个被大多数人忽视的重要因素，学习新事物。

这里有一些关于学习新事物和认知健康之间联系的很好的参考资料

*   [哈佛健康研究——“学习新技能可以减缓认知老化”](https://www.health.harvard.edu/blog/learning-new-skill-can-slow-cognitive-aging-201604279502)
*   [爱尔兰政府学习新事物](https://www.mentalhealthireland.ie/keep-learning/)
*   [英国政府-《为自尊而学习》](https://www.nhs.uk/conditions/stress-anxiety-depression/learn-for-mental-wellbeing/)

学习新事物的好处包括

*   提高自尊
*   减缓认知衰退
*   更加乐观
*   在某些情况下，甚至是大脑物质的物理增长

所以是的，学习真的能让你脑袋变大。我可以列出更多，但引用英国政府主页上的这段话比我更恰当

> 研究表明，终生学习会带来更大的满足感和乐观情绪，并提高充分利用生活的能力。那些在童年后继续学习的人报告说他们更幸福，应对压力的能力也更强。他们还报告了更多的自尊、希望和目标感

# 如何保持学习？

显然，学习的方法是无限的。如果你已经在日常生活中融入了学习新事物，那就继续做对你有用的事情。事实上，学习新奇的东西对大多数人来说是一项不切实际的任务，仅仅因为它需要时间。

不久前，我决定每天坚持学习新奇的东西。我花了一段时间思考一个有效的无偏见的方法来完成这个，然后我想起了[随机维基页面](https://en.wikipedia.org/wiki/Wikipedia:Random)。

有了 random Wiki，每次你点击[这个链接](https://en.wikipedia.org/wiki/Special:Random)就会得到一篇独特的文章。每篇文章都是完全随机的，与你之前搜索或浏览过的内容完全无关。这提供了一种每天学习 *n* 新事物的方法，只需点击链接 *n* 次并阅读打开的网页。

你将会被那些你从来不知道的事情所震撼。你也会经常很困惑，比如当[我得知德克萨斯 A & M 有一个卡塔尔校区](https://en.wikipedia.org/wiki/Texas_A%26M_University_at_Qatar)的时候。

# 结论

如果你还没有把学习新奇的东西纳入你的日常生活，我建议你每天打开几次随机维基页面。它只需要很少的时间，真正拓宽了你所做和不知道的界限。

如果你已经将学习新奇事物作为日常生活的一部分，我很乐意在评论中听到它们！
