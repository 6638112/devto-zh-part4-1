Django 是一个网络框架，可以帮助你使用 Python 建立交互式网站。使用 Django，可以定义站点需要处理的数据类型，并定义用户处理这些数据的方式。

通常最好将 Django 安装到虚拟环境中，在虚拟环境中，project 可以与其他 Python 项目隔离开来。

创建虚拟环境
$ python–m venv ll _ env
激活环境(Linux 和 OS X)
$ source ll _ env/bin/Activate
激活环境(Windows)

> ll_env\Scripts\activate
> 将 Django 安装到活动环境中
> (ll_env)$ pip install Django
