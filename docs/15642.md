# 宣布 Auth0 SPA JS SDK 正式上市

> 原文:[https://dev . to/auth 0/announding-the-general-avail-of-the-auth 0-spa-js-SDK-4c PD](https://dev.to/auth0/announcing-the-general-availability-of-the-auth0-spa-js-sdk-4cpd)

我们很高兴地宣布发布我们新的用于单页应用程序(spa)的 JavaScript SDK，名为 auth0-spa-js。

为什么是新的 SDK？过去几年，auth0.js 库一直是我们默认的基于浏览器的 SDK，也是我们产品和成功的关键部分。它的每周下载量超过 10 万次，几乎每个 Auth0 用户都以这样或那样的方式使用它。

在这段时间里，我们在核心产品中开发了大量的功能。随着平台的发展，auth0.js 在规模和功能上都有所增长。虽然 auth0.js 是一个非常强大的 SDK，可以处理大量不同的场景，但这种多功能性是有代价的。

我们回到绘图板，重新设想 web 平台 SDK 应该如何为开发人员将要实现的最常见场景工作。
