# 作为一名初级开发人员，你需要知道的 5 件事

> 原文:[https://dev . to/vetswhocode/the-top-5-things-you-need-to-know-to-be-successful-as-junior-developer-394 c](https://dev.to/vetswhocode/the-top-5-things-you-need-to-know-to-be-successful-as-a-junior-developer-394c)

无数的空缺职位和蓬勃发展的科技行业，是成为一名网络开发人员的最佳时机。然而，要在第一份工作中取得成功，需要的不仅仅是 JavaScript 知识——你还需要练习和强化各种非技术技能。

无论你是刚刚获得初级开发人员的第一个职位，还是正在找工作，进入这个领域都可能会有点令人生畏。以下是我们整理的五条建议，可以让初级 JavaScript 开发人员的生活变得轻松一些。

1) **寻求帮助**

作为一名开发人员，你工作的时间越长，你就越会意识到你所知甚少。不要害怕查找你不熟悉的术语或概念，也不要害怕向你更资深的同事寻求帮助。得到你的问题的答案对每个人都有好处，因为这会让你成为一名更好的员工。

除了提问，作为初级开发人员，您还可以通过多种方式获得帮助。与更有经验的开发人员结对编程是一种极好的方式，可以了解组织内部的情况，并找到有用的工具和技术。你也可以让你的同事对你的工作做一次代码审查，以便得到一些建设性的批评。

2) **显示主动权**

你的雇主将雇佣和培训你担任某个特定的角色，这可能与你目前所学的不完全相符。作为一名初级开发人员，您应该接受这样的事实:您可能并不总是能够从事您所选择的项目。

然而，这并不意味着你不应该继续寻找自己的利益。在你的职业生涯中，你可能会遇到许多不同的雇主，每一个雇主都需要不同的技能。如果你目前不在理想的职位上，通过学习你想要的工作的才能和技术来提升自己。一个很好的方法是从副业开始，你可以把它推向生产，并在你的简历中展示出来。

3) **不断学习**

尽管你可能被聘为初级开发人员，但学习新概念会给你更好的晋升机会。从如何编写更好的代码到如何有效地交流你的想法，总有另一种技能可以让你在工作中受益。

在网上，你会找到大量的资源、文章和视频，可以帮助你学习高级 JavaScript 技术和库。你甚至可以在 Twitch 这样的网站上观看实时显示人们编写代码的截屏。为开源软件项目做贡献是保持你的技能和帮助编程社区的另一个好方法。

4) **找导师**

即使当你作为团队的一员工作时，作为一名初级开发人员可能是一种令人困惑，有时甚至是孤立的经历。为了应对这些问题，找一个好的技术导师，他可以为你的工作提供建议，并帮助指导你的职业轨迹。

导师可以以多种形式出现，来自组织内部或外部。如果是你的一个同事或关系，随着你们更好地了解对方，关系通常会自然发展。你还可以在编程会议上找到外部导师，他们可以就工作面试和行业状况等话题提供建议。

5) **不要放弃**

耐心和决心是成功的 web 开发人员的两大美德。等待测试完成，学习一个新的框架，修复一个令人讨厌的 bug，这些都需要相当大的毅力和对挫折的高度容忍。

然而，重要的是要认识到，只要每天坚持锻炼，你就会变得更好。无论是学习骑自行车还是学习编码，没有大量的汗水和努力，任何有价值的事情都无法实现——所以要准备好抓住机会，犯错误并坚持下去。
