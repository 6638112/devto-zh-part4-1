# 项目 twist-开发日志#1(序言)

> 原文：<https://dev.to/jamelh/project-twiist-dev-log-1-prologue-4j8i>

**TL；我的第一个网络项目已经开始取得进展:一个在线视频平台，以“Twiist 项目”的名义托管用户生成的内容。**

我是编程新手..嗯，算是吧。从我 11 岁开始，我就对开放世界视频游戏产生了浓厚的兴趣，尤其是侠盗猎车手系列。开发人员能够创造一个巨大的世界，完全可以探索(在一定范围内)并且充满了可以做的事情，这让我感到惊讶。问题？嗯，总会有一些问题，对我来说，那就是我 11 岁，侠盗猎车手 GTA 系列的最新成员)被评为 M+ (18+)。这意味着我的父母不会让我玩它。我明白他们的想法，但是我反复向他们保证我不是为了获得 M 等级的东西而玩，我只是想玩，这样我就可以探索 GTA 的成就，以及在整个游戏中产生许多车辆。

仍然是:不。我的父母，负责任的和友好的，告诉我找些别的事情做。所以，我做了。我下载 Unity3d，开始到处玩。现在，我是一个乐观主义者，也是一个勤奋和执着的孩子。所以当我第一次推出 unity.exe 时，我心想“这能有多难？”，好家伙被我惊到了。

显然，我没有坐下来，11 岁的我，并开始编码 GTA V 的副本，但我肯定尝试过...一段时间。我不断遇到问题，但慢慢地我拼凑出了一个游戏。现在，这里是我需要具体说明的地方:我正在把拼凑成一个游戏。你必须记住，我不一定对“编码”游戏感兴趣，但更感兴趣的是玩 GTA V(或我试图创造的任何版本)，因此我允许自己走很多很多捷径。Unity 资产商店的发现真正加快了事情的进展。

有了资产，我很快开始拼凑一个真实的原型，有一个动画角色，工作的武器，一个小地图，一个世界(不是我建模的！)，可驾驶车辆，甚至多人。我为我的创作感到自豪，但真正属于“我”的创作。

虽然在我买 GTA V (15)的时候，我已经有了我的 GTA 的工作原型，但我对真正的游戏太感兴趣了，不再关心了。但是，我真正关心的是“构建”一个游戏到底有多有趣，以及我有多享受这个过程。

这给了我编码的冲动，让我真正感到自豪。这就是我产生创建 Twiist 的想法的地方，Twiist 是一个托管用户生成内容的在线视频平台。一项大工程？是的。我能胜任吗？双重肯定。
