# 你是否设立了 Patreon 或其他服务来接受捐赠？💰

> 原文:[https://dev . to/Andy Li/do-you-set-up-patreon-or-other-services-to-accept-捐赠-2m6e](https://dev.to/andyli/do-you-set-up-patreon-or-other-services-to-accept-donation-2m6e)

一方面，接受项目捐赠听起来很棒，人们可以梦想有足够的捐赠来辞掉他们的日常工作。另一方面，即使是维护相当受欢迎的项目的人也没有收到多少钱。例如， [Babel](https://babeljs.io/) 的维护者目前每月通过 [Patreon](https://www.patreon.com/henryzhu) 获得 2580 美元，但我很确定一个人可以很容易地在全职技术工作中赚更多的钱。

你觉得 Patreon 或者类似的服务(GitHub 赞助商)对你有帮助吗？收到捐款会激励你参与项目吗？
