# 编码的招聘人员:我讨厌它

> 原文：<https://dev.to/eaustin88/a-recruiter-who-codes-i-get-it-4m7k>

大约一年前，出于对所看内容的好奇，我开始自学编程。我像世界上其他人一样开了一个博客，偶然在键盘上按了一个组合键，弹出了开发者工具。我利用谷歌，并开始意识到我可以用这个工具来改变我博客上我讨厌的一切，真的变得沉迷于学习，改变和设计我博客的外观。不用说，我的博客毫无进展，因为我太专注于它的设计了！最近，我开始通过 SoloLearn、Flatiron 和移动应用程序学习在线课程，这些课程帮助我进一步了解自己在做什么。作为招聘人员，我们寻找获得计算机科学或软件工程硕士学位的候选人。我遇到过成千上万的候选人，他们是客户的完美搭档，但由于没有学位/硕士学位而无法提交。这让我痛不欲生，因为我不得不承认，我自己学到的东西比在课程中学到的要多得多。
现在，我确信上大学并支付教育费用已经深入学习，然而，我们中的一些人没有能力支付我们的教育费用。作为 5 个孩子的母亲，一个妻子和一份全职工作，我不可能在我已经拥有的基础上再增加更多的债务。那么，为什么招聘者和客户界如此反对那些自学成才的人呢？有没有公司不关心你的经验来自哪里，谁会对你的 GitHub、GitLab、portfolio 等印象更深刻？我来到这个科技世界是因为我真正的爱和自由的创造力，我加入招聘是为了学习不同的道路，结识不同的设计师/工程师，并帮助其他人找到他们的方向。同时...我开始对一些公司感到失望，这些公司只想看到你上过成千上万人上过的现场结构化课程，而不是那些花更多时间自学的人...
