# 成为一名 18 岁的软件开发者

> 原文:[https://dev . to/MK RUP/成为一名 18 岁的软件开发者-14f0](https://dev.to/mkrup/becoming-an-18-year-old-software-developer-14f0)

大家好，我是 Mykolas，来自立陶宛维尔纽斯的一名热情的开发人员。我主要使用 JavaScript 和 React👩‍💻⚛️.虽然我并不局限于任何一种技术，但总体来说，我对软件开发非常感兴趣。

我想分享我迄今为止在软件开发世界中的旅程，以及我是如何在高中学习时开始作为一名开发人员工作的。

## [](#how-the-journey-started)旅程是如何开始的

我一直对任何技术性的东西都感兴趣。从很小的时候起，计算机、逻辑谜题和游戏就吸引我开始编程。

当我大约 14 岁的时候，我开始接触编码和 JavaScript 的基础知识。我开始学习的资源是吸引人的、有趣的和有帮助的。我从 code.org 开始，后来我去了 khanacademy.org T2 的 JavaScript 课程。

初学的课程很棒，但是过了一段时间，它们不够有挑战性，我失去了一点兴趣。我必须找到一些更真实、更有用的东西。

在短暂的休息后回到编码的工作中，想起它实际上是多么有趣和有回报，我突然有了将软件开发作为职业的冲动。许多工作和学习摆在我面前👨‍🎓💻。

[![peer programming](../Images/0cf08e80bd543f7fa71bdb42fcf7ef77.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--VAVQpWFr--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/zn07y05b6y2gr11kbsrp.jpg)

对我来说，编程一直是一项社交活动。我在左边，和朋友一起参加黑客马拉松。

## [](#starting-to-take-it-more-seriously)开始认真对待

回顾过去，我发现我做了几件非常正确的事情，为成为一名软件开发人员做好了准备。

### [](#online-courses)在线课程

我大约 15 岁的时候，收到了一个名为[lynda.com](https://www.lynda.com/)(现在的 LinkedIn Learning)的在线学习平台的会员资格作为礼物。在我的学习中投入的大约 40 美元已经带来了惊人的回报。📖🦄可能是我一生中最好的投资。

我学到了如何构建网络软件的最基本的基础知识。开始接触 HTML、CSS，并更深入地研究 JavaScript，它很快成为我最喜欢的编程工具。

不仅课程质量更高，而且花钱上这些课程的事实激励我去完成它们。这就是为什么我会推荐购买你的课程。有大量的免费资源，但是如果你真的完成了，你更有可能完成课程💸**已投资**💸在金钱上。

在 web 开发方面，我很想推荐一下 [@wesbos](https://twitter.com/wesbos) 的课程。它们真的很有趣，信息量也非常大。在我的职业生涯和编程之旅中，它们具有很大的变革性。与 JavaScript 和那个生态系统相关的[课程](https://wesbos.com/courses/)对我帮助特别大。从小技巧和窍门到基于项目的实践经验，这些课程给了我很多。

在线课程一直是我学习过程中相当大的一部分。即使在今天，我也尽量把时间分配给课程，它们让我以一种浓缩而有效的方式学习新的东西。作为一名软件开发人员，改进、学习和掌握新技能是必不可少的。

### [](#attending-events)出席活动

聚会、黑客马拉松和任何与发展相关的活动过去是，现在仍然是我学习过程和职业发展中至关重要的事情。

#### [](#meetups)聚会

一般来说，非常非正式和友好的活动，在那里你不仅可以学到东西，更重要的是你可以认识业内的其他人。

在你的城市有很多平台可以找到聚会，meetup.com 是一个很好的平台。活动通常是针对某项技术或编程语言的，所以选择你最感兴趣的。

主要是 JavaScript，面向 React 的聚会帮助我跟上了外面的新情况，认识新的志同道合的人也很棒。

但是最重要的是，大多数聚会都有免费的小吃和饮料🌮🍩🥤。

参加这些活动没有坏处，所以你应该参加。

#### [](#hackathons)黑客马拉松

黑客马拉松太棒了！🌞

他们*强迫*你(以一种非常积极的方式)坐下来，从零开始建造一些东西。构建东西最终是你在软件开发中学习的方式。

无论你是任何技术的专家还是初学者都没关系。当一个项目充满激情和责任感时，你会想办法让它运转起来🐱‍💻。这只是人类的本性，我们适应，学习和调整。

在乞讨、新的人、新的任务、完成工作的严格期限中，这可能不是很舒服，因为大多数黑客马拉松都在一个周末举行。但是走出你的舒适区是一个提高和学习的好方法。

黑客马拉松教会了我很多技术技能，我也通过他们认识了一些好朋友。我甚至有幸赢得了其中的几个奖项。虽然这项活动的本质不是为了获胜，而是为了创造大胆的项目和结识新朋友。

[![hackathon win](../Images/01796d9868dc78b9c1ee682fd1d1dbd8.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--uazAZ4uj--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/l5hqtlq2fqiloorqosmh.jpg)

我在左边，睡眠不足，但在和一个了不起的团队一起赢得胜利后，最终还是很开心。

这些各种各样的活动、黑客马拉松和聚会对我的生活产生了巨大的改变。如果我一直坐在家里，不参加任何社会活动，我就不会得到我所得到的机会，我就不会认识我所认识的那些伟大的人😎🤜🤛🤓。

甚至我的职业生涯也是在我参加黑客马拉松后开始的，因为我在参加了一家科技公司组织的黑客马拉松后获得了我在该公司的第一份实习。

### [](#internship-and-work)实习和工作

我非常感谢那些接受我作为实习生加入他们团队的人。在一个支持性的环境中成长真是太棒了。当我对开发一无所知的时候，人们相信我对我这个年轻的开发人员来说意味着一切🐣🐣。

毫无疑问，如果我周围的人不像他们那样支持我，我会放弃这整件事，不再继续下去。

作为一名实习生，我不做咖啡或类似的事情，我做的事情和其他开发人员一样。而且不得不说，在职场软件开发方面，我学的最多。

在我的编程工作之旅中，我非常幸运，能够向伟大的人和专业人士学习。向更资深的开发人员和导师学习对于年轻的程序员来说是非常有用的👨💻 😄💻。

提出问题、分享想法、讨论代码和不同的解决方案非常重要。这就是你学习和提高的方式，没有比做、创造和发展更好的方式了。

在你的编程之旅中，有已经经历过你正在经历的事情的人在你身边是非常有帮助的。如果你在这个广阔的世界中感到孤独或迷失，这些人肯定能帮助你。

我从实习生到初级开发人员，再到中级开发人员，在同一个公司，同样的支持环境。作为一名软件开发人员，我在很多方面都成长了很多。

## [](#next-steps-in-my-journey)我旅程的下一步

我可以看到在过去的几年里我已经进步了很多，但是我希望我永远不会停止扩大我的知识库。尽管我已经开始了软件开发的职业生涯，但我的下一步是获得计算机科学的传统学位🎓📚。

我觉得传统学术界将会给我巨大的帮助，不仅仅是在知识方面，还包括从更深入的理论层面更深入地了解技术世界。

最后，我希望我已经获得的实践经验和传统学术界更注重理论的方法将帮助我创建更好、更有用的软件。
