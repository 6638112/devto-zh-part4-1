# Java 应用程序的 4 个主要类别

> 原文：<https://dev.to/yedukondalu319/4-main-categories-of-java-applications-jlc>

Java 是一种通用编程语言，是一种基于类的、面向对象的编程语言，是 IT 公司开发软件应用程序的首选编程语言。学习 Java 编程语言有自己的好处。

Java 既是一种编程语言，也是一个遵循特定语法和风格的平台。Java 平台是运行 Java 编程语言应用程序的特定环境。

Java 有几个平台，如下面讨论的，由几个 Java 虚拟机(JVM)和应用编程接口(API)组成。

今天我们将讨论各种类型的 Java 应用程序、Java 应用程序示例以及许多其他重要方面。

Java 编程语言平台

首先，让我们专注于理解不同的 Java 平台或 Java 版本。

Java 为开发特定类型的应用程序提供了不同的平台。以下是 Java 平台或 Java 版本列表。每个 Java 平台都由 Java 虚拟机和应用编程接口(API)组成。

Java 编程语言有 4 个平台，如下所述:

```
Java platform, Standard Edition (Java SE)
Java platform, Enterprise Edition (Java EE)
Java platform, Micro Edition (Java ME)
Java FX (Java FE) 
```
