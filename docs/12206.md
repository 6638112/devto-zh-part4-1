# 新程序员的资源

> 原文:[https://dev.to/kcarrel/resources-for-a-new-programmer-2lpf](https://dev.to/kcarrel/resources-for-a-new-programmer-2lpf)

你正在寻找编程，但你不知道从哪里开始？虽然那句开场白读起来像是一个糟糕的电视广告的开头，但我唯一想向你推销的是查看下面的资源！我开始了我的编码之旅，把教程挤到我的空闲时间，然后参加了 15 周的全职软件工程强化课程。在我的整个旅程中，以下资源帮助我导航不熟悉的技术领域，我希望它也能为你做同样的事情。

如果你完全是编程新手，我建议你先阅读一下阿里·斯皮特尔的入门指南。
