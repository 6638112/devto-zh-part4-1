# 脸书的天秤座货币

> 原文：<https://dev.to/vuild/facebook-s-libra-cryptocurrency-1e78>

在层出不穷的新加密货币中，似乎只有少数几种真正脱颖而出。脸书的天秤座会是其中之一吗？经过一年多的开发，Libra Coin(内部称为 GlobalCoin)是脸书的新密码宝贝。旧帽子霍德勒可能会抱怨另一种“去中心化”公司硬币的出现，但这种硬币可能主要面向没有银行账户的人&那些不熟悉加密场景的人。

### 天秤座的内幕

虽然预计要到明年才会发布(与 Calibra wallet 一起发布),但其发布的宣传机器已经准备就绪。此外，脸书已经在金融行业获得了不少重量级人物的支持(大部分都是 1000 万英镑)，包括 Visa、万事达、PayPal &优步(合作伙伴名单如下)。为了应对加密市场的潜在波动性，Libra 将与政府发行的一篮子货币挂钩。

### 卡利布拉公司&钱包

Calibra 是一家脸书子公司，将提供金融服务和 libra 网络接入。他们计划推出的第一款产品是 Libra 新加密货币的数字钱包(预计将于 2020 年推出)。他们的钱包将可用于 Messenger、WhatsApp 或作为一个独立的应用程序&将允许用户保存、发送、存储(最有可能购买)Libra 硬币。我们假设这种货币也将在其他加密交易所提供(特别是比特币基地，因为他们是投资者)。
