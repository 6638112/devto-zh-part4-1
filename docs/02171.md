# Dart 中的函数

> 原文:[https://dev.to/jay_tillu/functions-in-dart-4795](https://dev.to/jay_tillu/functions-in-dart-4795)

*   函数是执行特定任务的一组语句。

*   函数主要用于自动化重复的任务。它们基本上是用来把大程序分成小块的。

*   将我们的程序分成更小的块使它更有组织性和可管理性。此外，它避免了重复，使我们的代码可重用。

*   关于函数的不好的一面是它们以几个名字而出名。不同的编程语言对它们的命名不同。例如函数、方法、子程序、过程等。当你听到他们中的任何一个时，想象一下同样的概念。

### [](#advantages-of-functions)功能优势

* * *

函数有三大优势:

1.  它们使我们的代码易于管理。
2.  他们减少了样板代码
3.  他们使我们的代码可重用。

## [](#types-of-functions)功能类型

* * *

编程中基本上有两种类型的函数。

1.  内置函数
2.  用户定义的函数

### [](#builtin-functions)内置函数

*   内置函数是语言自带的函数，可以随时使用。例如在 dart print()中，readLineSync()、tryParse()都是内置函数。

*   在 dart 中，有许多库，所有的内置函数都存储在其中，随时可以为您提供帮助。Dart SDK 附带了许多预构建的库，如 dart: core、dart: async、dart: collection 等。

**Dart 库有三种主要风格:**

*   **核心库—** 核心库直接来自 Dart SDK。你可以在 [Dart API](https://api.dartlang.org/stable/2.1.1/index.html) 上找到详细的文档。

*   **社区共享库—** 一些库与 Dart 社区共享，它们作为库包分发并在[酒吧网站](https://pub.dev/)发布。 [pub](https://dart.dev/tools/pub/cmd) 工具允许我们创建、发布和管理库包。

*   您也可以从 GitHub、URL 或本地路径使用库。请访问[本](https://dart.dev/tools/pub/dependencies#dependency-sources)了解更多信息。

### [](#user-define-functions)用户定义功能

*   用户定义函数是由程序员创建的函数。您也可以根据需要创建函数，这些函数将被称为用户自定义函数。
