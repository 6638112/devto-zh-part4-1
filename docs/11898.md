# 每日挑战#22 -简单的拉丁猪

> 原文:[https://dev . to/the practical dev/daily-challenge-22-simple-pig-Latin-3d dl](https://dev.to/thepracticaldev/daily-challenge-22-simple-pig-latin-3ddl)

今天挑战的目标是把英语翻译成猪拉丁语。这个[挑战](https://www.codewars.com/kata/520b9d2ad5c005041100000f)来自 CodeWars 上的 user2505876。

> 写一个函数，把每个单词的第一个字母移动到单词的末尾，然后加上`"ay"`。保持标点不变。
> 
> 例子:
> `pig_it('Pig latin is cool') # igPay atinlay siay oolcay`
> `pig_it('Hello world !') # elloHay orldway !`

这个有点棘手。我打赌把猪拉丁语变回英语会更难，我很想看到有人尝试一下~！

Oodgay ucklay，appyhay odingcay！
