# 转行很难

> 原文：<https://dev.to/rygelxvi/career-changes-are-hard-1e4i>

### 转行难，

尤其是在一个不相关的领域全职工作的时候。

我认为完成我的课程在时间管理上是一个挑战。我错了。在时间管理中，寻找第一份工作是真正的挑战。

我不仅需要通过从事个人项目和解决代码问题来保持我的编码技能，我还必须筛选数量惊人的招聘信息、写求职信和社交网络。

除此之外，我决定学习 Java。我的类是用 Ruby 和 Javascript 编写的，所以学习一门强类型语言并内置树类之类的会很好。更重要的是，许多角色需要 C、Java 或 Python 的迭代，所以一个基础(无论多小)就很好了。

#### **下面是名单**

1.  求职

2.  网络

3.  实践准则挑战

    *   在电脑上
    *   在白板上(也就是我家的滑动玻璃门)
4.  继续从事个人项目

5.  学习 Java

我需要每周抽出几天时间来确保当我有效率的时候，我专注于我需要关注的事情，而不是试图找出我每天应该做什么。

#### 以下是计划:

当工作缓慢时——写关于编码问题和 CS 主题的博客文章

一周两天(可能是周二/周五)——一周两天处理工作事务
——一周一天用每种语言做一次编码挑战
——一周一天练习白板
——什么都不做/睡觉/过度网飞*耶*

我需要休息，我做我真正想做的事情，学习 Java 和做我的个人项目。

让我们看看这是怎么回事。
