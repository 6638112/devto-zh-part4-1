# 在终端上获取天气信息

> 原文:[https://dev . to/anuragrana/get-weather-information-on-terminal-55ee](https://dev.to/anuragrana/get-weather-information-on-terminal-55ee)

你可以在你的终端上获得你所在地区的天气信息。

您需要在 URL `wttr.in/$1?m1`
上发送 curl 请求

```
curl -s "wttr.in/$1?m1" 
```

您可以将此添加到您的。bashrc 文件作为命令。

```
weather(){ curl -s "wttr.in/$1?m1"} 
```

获取 3 天的信息:`curl -s "wttr.in"`
