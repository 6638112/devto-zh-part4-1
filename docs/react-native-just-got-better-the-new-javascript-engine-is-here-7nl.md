# 反应原生刚刚变得更好！新的 Javascript 引擎在这里！

> 原文：<https://dev.to/rishikc/react-native-just-got-better-the-new-javascript-engine-is-here-7nl>

[爱马仕引擎](http://hermesengine.dev) for React Native 来了！

所以从这里开始:

> ![React Native profile image](img/1d41a22454650f7b1b6de59d54559237.png)React Native@ React Native![twitter logo](img/4c8a2313941dda016bf4d78d103264aa.png)如今脸书正在开源 Hermes，这是一个为在 Android 上运行 React Native 而优化的小型轻量级 JavaScript 引擎。我们目前正在为 React Native 发布一个新的补丁，其中将包含 Hermes 作为选择加入。[github.com/facebook/hermes](https://t.co/YUFCqvTrAV)2019 年 7 月 11 日下午 16:01[![Twitter reply action](img/44d8b042100e231770330321e5b63d65.png)](https://twitter.com/intent/tweet?in_reply_to=1149347916877901824)[![Twitter retweet action](img/93d9c70ccc54851d2e8e881b53c21dae.png)](https://twitter.com/intent/retweet?tweet_id=1149347916877901824)[![Twitter like action](img/2e93f7775eadefab8bcd34a4542cc5a7.png)](https://twitter.com/intent/like?tweet_id=1149347916877901824)

大约两小时前脸书正式发布了 React Native v0.60.2，这使得该引擎成为一个可选功能。这为 react native 的移动开发带来了巨大的进步，因为它将使 React Native 构建的 Android 应用程序在性能上有显著的提升，构建尺寸更小，内存使用量更少！💜❤️

集成是如此简单！只需在您的`build.gradle`文件中启用它即可。完成的🙆‍♂️字面上，只有一行代码，就是这样！✨
