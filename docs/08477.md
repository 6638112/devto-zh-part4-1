# 我是如何有机会在国际 Ruby 大会上发言的

> 原文:[https://dev . to/where _ I _ stuck/how-I-get-a-chance-on-the-international-ruby-conference-4hh](https://dev.to/where_i_stuck/how-i-got-a-chance-to-speak-at-the-international-ruby-conference-4hh)

我刚刚从计算机科学专业毕业，在 OnceHub 担任软件工程师。我在 2017 年初开始开发小型 web 应用程序。

我也有一个专业的建议，你可以在博客的最后找到。:)

我在这里和那里做了一些事情，帮助我被选为 3 个国际 Ruby 会议的发言人，为了回报社区，我在这里做了这些事情:

我想让它保持简短，这样你就可以阅读一小段，并可以有实际的想法，而不会浪费很多时间。

[![My journey](../Images/26d895c855a3a21d88c0f04bd6ceab59.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--HuwA_tIW--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/cwi3vl21epkdtg4xjio3.jpeg)

根据我的经验，我可以说参加会议的人是来听开发者经验的。当他们遇到问题时，他们是如何克服的。他们不太喜欢那些可以从博客中学到的东西，比如:

*   如何使用软件
*   下一个功能中有什么新内容
*   如何制作 React App
*   开发 Angular 应用程序的最佳实践

你可能会在一些地方发现这些演讲，大部分是在当地的聚会上，但我认为它们没什么意思。相反，以下主题非常有趣:

*   我遇到了一个问题，我需要在一夜之间迁移一个大数据库
*   我做了一个 CLI 应用程序来帮助一个大产品。这是我的演讲:)
*   VS 会谈很有名。Rails VS Django，Angular VS React 等等。

好了，现在让我们看看我是如何做到的:

* * *

# [](#1-doing-online-courses-and-simultaneously-applying-for-internships)1。一边做网络课程，一边申请实习

2017 年初，我做了很多在线课程，主要是 Udemy 上的 Angular 和 Rails。如果你想知道我的课程，请在下面评论，我会提供链接。与此同时，我申请了许多实习机会。你可以由此有个想法:我从 2017 年到 2018 年年中申请了 77 份实习，5 份被选中。

我不认为他们会选择我，很多人都没有选择我，但没关系，我最终会找到我可以做出贡献的创业公司，一旦我得到了第一份实习，那么下一份就变得容易了。

关键是要不断申请，不断学习。

# [](#2-while-doing-internship-keep-looking-for-content)2。在实习期间，不断寻找内容

这是你满足的时候。工作的时候，我在很多地方停留。我过去常常在一个问题上花 3-4 天来解决它。但在那段时间里，我学会了事情是如何运作的。这就是内容。谈话的内容。

写一份草稿。向当地的聚会申请一次谈话。不管你会被拒绝多少次，只要坚持申请就好。如果他们提供了反馈，那么在每次被拒绝后，一定要努力改进内容。了解内容质量的一个方法是坚持参加聚会，如果你负担得起，也参加会议。

# [](#3-apply-for-the-first-talk)3。申请第一次谈话

当我在浦那 Coupa 实习时，我制作了一个 CLI 应用程序来帮助一个大型 Rails 应用程序。这对我来说是一件大事，所以我用它做了一篇论文，并申请了 2018 年的 DeccanRubyConf ，我首先作为一名替补发言者，一名发言者选择退出。所以我有机会在我的第一次大型国际 Ruby 会议上发言。这里是我演讲的[环节](https://www.youtube.com/watch?v=DMg17iak2lU&t=1s)。

之后，我在 RubyFuza 2019 和[ruby conf 2019](https://twitter.com/NairubyKE)上发言

## [](#conclusion)结论

我的旅程是我做大量的实习，从他们那里获得内容，然后根据这些内容申请许多会议和集会。最终，我得到了一个机会，并成功了。

如果你们中的任何人在这个过程中挣扎，那么请在 [twitter](https://twitter.com/Raounaksharma) 上与我联系，让我们看看我能如何帮助你，我们能如何共同成长。

快乐成长:)👋
