# 结论🔎

看着这些结果很有趣，我觉得我可以从我使用的键绑定中猜出我写的是什么样的代码。此外，它强调了我可以通过更多地关注一些可以做更多使用的按键来提高我的生产率的潜在方法，例如`Surround with`快捷方式，我确信我可以使用它> 30 次。

当然，你的收获会因你的偏好、你正在处理的任务甚至只是你编码的方式而不同。

这些也绝不是 IntelliJ 中所有的键盘快捷键(有几百个),它们只是我多年从事 it 工作中获得的有用的快捷键。

我希望这对你们中的一些人有用，我知道如果我很长时间不使用 IDE(可能不会😆).
