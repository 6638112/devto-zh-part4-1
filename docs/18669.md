# React Native ScrollView 动画标题示例

> 原文:[https://dev . to/skp tricks/react-native-scroll view-animated-header-example-1 DLI](https://dev.to/skptricks/react-native-scrollview-animated-header-example-1dli)

来源: [React Native ScrollView 动画头示例](https://www.skptricks.com/2019/06/react-native-scrollview-animated-header-example.html)

本教程解释了如何在 react 本地应用程序的 scrollview 中创建动画头。你已经注意到，在许多应用程序中，固定高度的标题位于屏幕顶部，滚动列表位于标题下方。当你开始滚动列表时，标题会根据滚动列表的滚动值从当前高度到固定的最小高度平滑地折叠起来，并停留在屏幕的顶部。在这里，您将看到 react 本机应用程序中类似的动画效果。
