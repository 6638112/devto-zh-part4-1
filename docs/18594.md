# 加入开发团队！

> 原文：<https://dev.to/devencourt/joining-the-dev-team-pec>

大家好。我叫 Brian，我很荣幸能够作为两位新的业务发展和增长实习生之一加入开发团队。

在申请这个实习机会之前，我已经在 DEV.to 上潜伏了一段时间，以便对这个社区有所了解。我很快意识到，我经常上这个网站和应用程序，因为它的社区开放友好，所有成员都能提供大量的知识。

我在软件开发方面的知识有限，但我热衷于扩展我的知识。我有使用 Python 3、HTML/CSS 和 Lua 的经验。我也涉猎了 2D 和 3D 游戏设计。当我进入大学时，我选择离开软件开发，去追求我的创造性和语言努力，但是我仍然在业余时间对技术和开发充满热情。

我非常高兴能与团队及其社区成员合作。作为实习生，我的职责将主要集中在非技术性工作上，尤其是吸引和支持当前和潜在的合作伙伴和赞助商。这次实习对我来说是一个惊人的机会，我的目标是尽我所能支持开发团队。

我会说英语、西班牙语，目前正在加强日语。请用对你来说最简单的语言联系我，我会尽最大努力及时回复。
