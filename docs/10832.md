我们的社区很棒，因为任何人都可以发表意见，帮助有编码问题的人。我们在 DEV 上提升了许多开发人员。

但我们也是一个现有行业领导者已经开始分享并成为其中一部分的地方，我们希望吸引更多像这样的人(而不是变得过于精英化)。行业“领导力”并不容易定义，但它通常意味着要么你发明了一些起飞的东西，要么你因教学而出名并发展了大量的 Twitter 追随者。😄

这里有一些最近开始发帖的著名 JavaScript 人，你可能想关注一下。这是一个不完整的列表，但是这里有一些人最近发了*。*
