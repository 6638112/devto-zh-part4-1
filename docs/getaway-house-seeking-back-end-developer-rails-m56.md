## 关于我们

如果你正在阅读这份工作描述，你可能是一个在电脑和互联网上花费大量时间的人。虽然我们像其他企业一样利用这些工具，但我们的产品试图让我们的客户远离这些东西，努力找到他们一直拼命寻找的魔力和放松。我们把小房子放在偏远地区的营地，这样我们的“用户”就可以离开，并与他们的人性建立高速接口。我们创造和维护技术，在某些情况下，无形地为我们的客人带来神秘的体验，并消除他们生活中的压力。

escape 是一个规模虽小但发展迅速的团队，刚刚筹集了 2250 万美元的 B 轮融资。我们的办公室位于布鲁克林市中心，但我们对远程候选人开放，他们每月至少可以访问我们的办公室两次。

## 关于机遇

后端开发人员将关注以下方面:

*   在半敏捷的工作环境中，用 React/SASS 前端和 Ruby on Rails 后端为我们的网站构建新功能并维护其当前功能
*   一些次要的开发人员操作任务，确保网站很好地为我们的客户服务，并按照我们的期望执行-我们目前托管在 Heroku 上
*   支持机舱自动化工作，开发软件使客人体验无缝和神奇——目前用 Groovy 和 Ruby 编写，正在转向 Python 和 Ruby
*   编写内部工具，以支持我们的游客体验团队，并让游客非常容易地获得他们需要的东西，度过一个美好的假期
*   继续寻找效率和利用软件的方式来改善我们团队的工作流程和我们开展业务的方式
*   偶尔在火前写代码，如果你愿意的话(真的)

## 技能&资历

我们正在寻找有大约 2 年经验的人，但不希望任何人回避申请，如果他们认为自己是一个真正的“摇滚明星/忍者/等”。这不是一个初级发展角色，那些直接从学校或训练营出来的人不需要申请，不幸的是，我们现在没有资源来支持这样的候选人。

## 如何申请

感兴趣的开发人员应发送一封简短的求职信和简历至[jobs @ escape . house](mailto:jobs@getaway.house)，主题行“后端开发人员角色”。
