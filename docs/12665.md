# 嘿，让我们分享我们的职业生涯开始的故事。

> 原文:[https://dev . to/ypedroo/hey-let-s-share-our-career-begin-stories-31oc](https://dev.to/ypedroo/hey-let-s-share-our-career-begin-stories-31oc)

这是我的。一切都要追溯到 2015 年我上大学时，那时我唯一知道的是我热爱科技，并想从事科技工作。2015 年相当艰难，因为年底我妈妈离开了家，我和我失业的老奶奶在一起，不得不养活我们两个人。幸运的是，我在一个停车场找到了一份白天的工作，这份工作
拯救了我们，我从中得到了很多乐趣，缺点是我没有像在大学时那样写很多代码，也没有学到很多东西。2016 年，我在一家科技公司实习，但为了得到一个支持角色，我开始希望得到一份开发工作，但我没有准备好，整个实习令人沮丧，因此又失去了一年。但也有积极的一面，我遇到了我的朋友卡伊奥，他向我推荐了一家需要系统分析师的销售公司，这时候事情开始对我有利了。在这家公司，我会用 SQL 做几乎任何事情。所有功能都基于过程或查询，这不是我梦想的理想或工作，但我付出了 100%的努力，这样可以给我带来有用的经验，这就是 2018 年的结局。由于我在编写 SQL，接触了更多的编程，我也赚了更多的钱，有更多的时间学习和参与 GitHub 项目，这让我有信心申请 jr 开发职位，最终在 2019 年，我有机会通过在一家大型巴西公司获得 jr 职位来真正证明自己是一名开发人员。

对我来说，这份新工作的前 7 个月是一个疯狂而紧张的学习期，紧张但令人惊讶的是，我有这么多很酷的项目要做，并有机会通过与人交谈和分享我所学到的东西来学习、实践和展示我所学到的东西。

仅此而已。感谢花时间阅读这篇文章的人，我希望这篇讨论能帮助一些人进入编码领域:)。
