# 你的软件利基是什么？

> 原文：<https://dev.to/makiten/what-s-your-software-niche-3jd0>

背景是这样的:我一直在看很多建议文章，他们都建议一件事，不管你是想自由职业，创业，还是找工作，那就是找到一个合适的位置。这个想法是，有很多竞争，如果你让自己太“一般”，你会在价格竞争中挣扎

然而，我反思了我的职业生涯，我的职业就是这样。我一直在为专业化而挣扎，因为最常见的情况是我会专业化，雇主会有一个我能做的独特要求，我做得很好，并且，至少从理论上讲，无论我本可以在一个领域花费多少时间，我都浪费在做不同的事情上了。老实说，这是因为我的角色通常与营销有关。例如，在营销机构或营销/通信部门工作，但严格来说是在开发部门。即使是我现在做的，尽管是一家提供*支付*服务的公司，我也在做与营销相关的项目。所以，当然，当我想到我的定位是什么时，我认为它一定与营销有关。

但这并不是说我在寻求帮助来找到我的定位。其实我想知道的是*你把自己的利基看做什么*。如果你要找工作(如果你是自由职业者/个体户，无论是工作还是项目合约，或者如果你有一个产品初创公司)，你会关注什么？
