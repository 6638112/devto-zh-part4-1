# 手和手腕疼痛？

> 原文：<https://dev.to/cecilelebleu/hand-and-wrist-pain-4mmg>

几天来，我的右手、手指、手腕和前臂都疼得厉害。我知道罪魁祸首:在 MacBook 平板键盘上写代码，使用触控板，在空闲时间使用手机和玩视频游戏。

我用我的左手拇指在手机上写下这些，我的右手紧紧地裹着以减轻疼痛。我简直不能用我的右手。因为疼痛，我甚至在过去的几个晚上醒了好几次。但是如果我不会写代码，我将如何实现我的目标呢？“要变得更好，就多写代码”，这是最好的建议，但如果做不到呢？...

因此，我正在向正在遭受或已经遭受类似问题的人寻求建议。你是怎么坚持下去的？冰？热度？花式人体工学键盘？语音转文字代码？
