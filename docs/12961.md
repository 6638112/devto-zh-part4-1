1.  前端-使用 VanillaJs 和其他(相当令人沮丧)
2.  后端-使用节点

4.  桌面应用-使用电子。射流研究…
5.  绘图/创意应用程序——使用 HTML 内置画布和其他框架
6.  智能手表应用-使用 Pebble。射流研究…
7.  移动应用程序-使用 React Native

以及其他等等
