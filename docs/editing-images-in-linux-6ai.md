# 在 Linux 中编辑图像

> 原文：<https://dev.to/k4ml/editing-images-in-linux-6ai>

tldr 它仍然很痛。

我刚刚换到 Thinkpad X1 Carbon，运行 Fedora 30。真的很怀念 MacOS 中简单的预览应用。

基本上，我只是想做通常的事情-裁剪/调整照片的大小，以便贴在博客上。然后是一些更“高级”的东西，比如在另一张照片中嵌入照片。

对于裁剪和调整大小，gthumb 似乎可以胜任。

但是为了嵌入图像，最后我不得不求助于 Gimp。试试 Krita，但是在 UX 方面，我没有看到 Gimp 有任何改进。

所以要在 Gimp 中嵌入图像，首先打开你想要嵌入的图像。选择全部，然后选择复制。如果您只需要选择的一部分，请使用选择。然后打开目标图像，选择选择工具并绘制要嵌入图像的区域。进入编辑-粘贴到选区(这是关键的事情)。嵌入的图像可能不适合目标图像，因此您可能需要调整它的大小。转到图层，嵌入的图像将显示为新图层，选择它，右键单击并选择缩放图层。

仅此而已。
