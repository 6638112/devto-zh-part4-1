# [](#shortcuts)快捷键:

将光标放在一个角上，然后按住 **Shift+Alt** ，同时拖动到另一个角上。您可以使用键盘上的向上/向下键将更多光标放在您想要编辑的行上。按**【ESC】**键退出多用户模式。
