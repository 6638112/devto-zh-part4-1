# SLI、SLO 和 SLA

> 原文：<https://dev.to/zex/sli-slo-and-sla-12bp>

### SLI -服务水平指示器

定义服务提供商提供的服务水平的衡量标准。
例子

*   服务的正常运行时间
*   交易数量

### SLO -服务水平目标

定义服务水平的目标值或目标范围，由 SLI 衡量。
例子

*   服务的可用性应该是 99.999%

### SLA -服务水平协议

定义服务提供商和客户之间的承诺。
例子

*   退还服务费
*   提供一段时间的免费服务
