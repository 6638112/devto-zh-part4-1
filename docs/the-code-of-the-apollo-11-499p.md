指令舱(Comanche055)和登月舱(Luminary099)的原始阿波罗 11 号制导计算机(AGC)源代码。由虚拟 AGC 和麻省理工学院博物馆的人们数字化。目标是成为阿波罗 11 号原始源代码的回购。因此，欢迎 PRs 对这个库中的转录和 [Luminary 099](http://www.ibiblio.org/apollo/ScansForConversion/Luminary099/) 和 [Comanche 055](http://www.ibiblio.org/apollo/ScansForConversion/Comanche055/) 的原始源扫描之间发现的任何问题，以及我可能错过的任何文件。
