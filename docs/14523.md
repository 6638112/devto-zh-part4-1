# 使用 web 组件构建下一代 web 应用

> 原文：<https://dev.to/javascript_tuto/build-next-gen-web-apps-using-web-components-dki>

如今，从一个框架转移到另一个框架或者更新到最新版本真的很难。Web 组件是一个相对较新的标准，它使前端开发人员能够创建跨框架兼容的组件。它让您可以使用本机浏览器 Api 编写可重用的组件，而无需外部依赖。

在本次研讨会中，我们将学习使用 web 组件(自定义元素、阴影 DOM、Html 模板、模块)构建一个具有可重用组件的 web 应用程序。我们还将熟悉 helper library lit-element，以及如何将这些组件与现有框架集成。
