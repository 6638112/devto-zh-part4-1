# 使用样式表对原生下划线文本做出反应

> 原文：<https://dev.to/skptricks/react-native-underline-text-using-stylesheet-4fhg>

来源:[使用样式表](https://www.skptricks.com/2019/07/react-native-underline-text-using-css-stylesheet.html)对原生下划线文本做出反应
