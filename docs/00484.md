弧(241，237，111，122，0，180)；

241-> x 位置
237- > y 位置
111- >宽度
122- >高度
0- >圆弧起点
180- >圆弧终点

与前四个参数一样，它接受了大量的参数，但实际上只给出了最后两个参数的起始角度值和结束角度值。也就是说，我们接受 0 到 180 之间的角度值。如果是 30 度，140 度，则角度值为 110 度。
