## [](#less-tester)少测试仪

这个工具非常有用，因为我学得很少，现在我对 CSS 预处理程序有了更多的经验，它仍然很有用。考虑这样的情况，你的代码看起来非常好，但是由于某种原因，你的风格没有像预期的那样呈现。将您的代码复制并粘贴到 LESS Tester 中，看看呈现的 CSS 是否如您所愿。

</figure>

# [](#conversion-tools)转换工具

在线转换工具的集合。

## [](#base64-encoder-and-decoder)Base64 编码器和解码器

能够对文件进行 base64 编码非常有用。我们将它用于图标，然后使用 CSS 中的 base64 数据来渲染它。

**网站**:【https://www.mobilefish.com/services/base64/base64.php T2】

## [](#pxtoem)PXtoEM

我们大部分的字体大小是用 em 定义的，然而，当我们的设计师在 Photoshop 中工作时，他们使用的字体大小规格是以像素为单位的。使用这个方便的网络工具将 px 转换为 em，反之亦然。

[![](../Images/98735f043f210e2d7370d394a426a2d8.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--pBkEbfSO--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/976/1%2Aj3KFv6ksMCcarBQfddmv0Q.png)

**网站**:【http://pxtoem.com/ T2】
