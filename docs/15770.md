# 我的软件工程之旅

> 原文：<https://dev.to/ashleemboyer/my-journey-into-software-engineering-4hen>

我于 2013 年秋季开始上大学，在印第安纳州的一所小型私立工程学校主修化学工程。第一季度对我来说结束得非常糟糕，让我怀疑自己是否能在学校里成功。在我看来，其他人都很聪明，做得很好！(事实上，我认为我们中的很多人都在默默地独自挣扎。)

接下来的两个季度要好得多。我喜欢我的化学课程，并且必须非常努力地学习数学和物理课程。那个春季学期，我上了一门 Excel 编程课程。那是我第一次写代码，我喜欢它！这可能是化学课之外我第一次真正感觉到我知道自己在做什么。所以，我决定明年秋天去上编程入门课。

他们教我们如何用 Python 编程。(学习编码我强烈推荐 Python！)对我来说太神奇了。能够控制我的电脑并告诉它我想做什么给了我一种成就感。对于学期项目，我们必须设计一个机器人。我学会了如何给一个*机器人*编程！太匆忙了。我觉得自己很聪明，直到...

我拿到了所有其他课程的期末成绩。我的化学工程课成绩很差。我可以分别处理化学和物理，但我似乎不能把两者结合起来。我有机化学学得不错，但我真的不再喜欢化学了。我要做一个决定。

1.  根据课程的开设时间，在夏天或秋天上课
2.  换一个专业
3.  彻底退学

第三个对我来说不是一个真正的选择。在这一点上，我已经搬出了父母家，完全独立了。如果我没有拿到那个学位，我就没有办法还清已经欠下的贷款。第一个选择很可怕:不能保证我第二次会通过。这些材料对我来说没有意义。鉴于我在编程入门课程中的出色表现，以及我对编写代码的热爱，我选择了第二个选项。

从那时起，直到我大学毕业，这段旅程感觉像真的地狱，但这不是因为课程。我喜欢我所学的几乎每一门软件工程课程。有些我甚至拍了两次。😉独自一人真的很难，有很多事情阻碍了我的教育。

当我回头看的时候，我希望我已经知道了我所有的选择，而不是感到压力很大，要赶紧毕业。我不确定我会向每个人推荐大学路线。尤其是在过去的几个月里，我不明白为什么一个自学成才的人不能像一张昂贵的纸一样成功。都是努力工作，表面看不出来。GPA 也是如此，对我来说，它还不如不存在的好。

这几天听了大家的故事，真的让我大开眼界。这个行业有很多来自各种背景的人才。让我们继续在技术领域建立包容性文化！如此多的可能性在等待着。
