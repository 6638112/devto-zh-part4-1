# [](#introduction)简介

很多在另一个行业的朋友经常问我:**成为一名 Web 开发人员需要学习什么**，**我应该从哪里开始？**

# [](#1-patience)1。耐心

当你想进入这个行业时，第一步也是最重要的一步是要有耐心。你必须每天努力提高你的技能。尤其是在开始的时候，你可能不会意识到开发一个应用程序所花费的时间的回报。

所以，现在和永远，记住:**耐心**。

* * *

# [](#2-html)2。超文本标记语言

第一个必须学习的 Web 开发技术是 HTML。HTML 对于 Web 应用程序就像地基对于建筑一样。HTML 诞生于大学，用于将文档相互关联。这使得查找内容相似的文档变得更加容易。然而，web 的发展使得 HTML 被用作创建 Web 应用程序的基础文档。从版本 5 开始，这种标记语言为标签增加了语义。这允许创建由其他信息系统解释的文档。

学习 HTML 的最大缺点是你的应用程序会有可怕的视觉效果。这可能会令人沮丧，因为你可能会觉得你的努力和结果没有达到林宁。但是每当你有这种感觉的时候，你必须记住第一步:**耐心！**。

通常我们和 CSS 同时学习 HTML。使用 CSS 为我们的 web 应用程序添加样式可以帮助减少前面提到的挫折，因为视觉方面得到了很大的改善。

有两个很棒的短期课程，涵盖了 HTML 最重要的方面。

### [](#interneting-is-hard-but-it-doesnt-have-to-be)[上网很难(但也不尽然)](https://internetingishard.com/)

一个友好的网页开发教程，适合完全的初学者。

> 学习编码应该不难。我们把一套全面的网络开发教程放在一起，帮助完全的初学者转变为有才华的互联网专业人士，让这变得更容易。我们已经有了课程，你需要的只是开始阅读的动力。

# [](#3-css)3。半铸钢ˌ钢性铸铁(Cast Semi-Steel)

你应该采取的下一步是学习 CSS。如果 HTML 是基础，CSS 就像房子的绘画、装饰或个性化。

乍一看，它似乎没什么用，但这个特性征服了客户，因此也是赚钱的特性。那些不理解 CSS 重要性的人可能有兴趣知道，大多数时候，我们的 Web 应用程序背后的技术并不重要。通常情况下，顾客不会为他不觉得有视觉吸引力的产品付钱。

但是，CSS 会在学习其他技术的同时学习。在你旅程的这一部分，你应该集中学习足够的知识来设计一个网页。显然，在这一步中不需要对 CSS 有深刻的了解，但是你必须能够保护自己。随着你旅程的进行，你的 CSS 技能会逐渐提高。

这里有三个可用的基本资源:一门包含 20 个互动教程的课程，一份由导师准备的练习清单和一份参考资料。

### [](#tutorial-css)[教程 CSS](https://scrimba.com/g/gintrotocss)

本课程包含 20 个互动教程，将在一个小时内教会你 CSS 的基础知识。对于想要简单学习 CSS 的有抱负的 web 开发人员和设计人员来说，这是一个完美的起点。

### [](#css-exercises)[CSS 练习](https://github.com/dangodev/css-exercises)

作为导师项目设计的 CSS 练习。

### [](#codrops-css-reference)[Codrops CSS 参考](https://tympanus.net/codrops/css_reference/)

一个广泛的 CSS 参考，包含所有重要的属性和信息，可以从基础开始学习 CSS

# [](#4-javascript)4。Java Script 语言

今天的革命是: **JavaScript** 。

你必须掌握 JavaScript/TypeScript，因为它是网络的现在和未来的语言。请记住，没有人有水晶球，我们不知道未来 20 年会发生什么。JavaScript 是目前存在于前端的语言，也是你需要能够完美处理的语言。

[![javascript](../Images/39a99a4955978075a88aa7d99bbe9493.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--6dEYP4YD--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1000/0%2AErTkGkRoeaVAoM8N.jpg)

JavaScript 是一种诞生时就有成为脚本语言意图的语言，然而，现在它是一种试图覆盖多种范例的语言。这种语言的真正发展是在过去的 5 年里，因为它的扩张是巨大的。

这就是为什么要深入研究 JavaScript:你可以构建桌面、移动、后端和前端应用。资源列表将会很广泛，这就是为什么我建议从下面列出的书籍开始。相信我，我告诉你，你每天都会学到新的东西。在过去的 7 年里，我一直是一名 JavaScript 教师，并且几乎每年都会修改课程大纲。不过不要惊慌，编程的基础将保持不变。

所以为这一步做好准备:

### [](#interactive-learning)互动学习

*   [code academy](https://www.codecademy.com/learn/javascript)-JavaScript 基础。
*   [函数式编程](http://reactivex.io/learnrx/) -通过使用 map、filter、concatAll、reduce 和 zip，以互动的方式学习函数式编程的基本原理。

### [](#free-ebooks)免费电子书

*   [雄辩的 JavaScript](http://eloquentjavascript.net)——涵盖语言和运行时细节。
*   [说 JavaScript](http://speakingjs.com) -从基础开始的深入指导。
*   [JavaScript 设计模式](http://addyosmani.com/resources/essentialjsdesignpatterns/book/)——经典的和 JavaScript 特定的设计模式。
*   函数式编程的基本指南——使用 JavaScript 进行函数式编程的优秀入门。

### [](#es6-and-above)ES6 及以上

*   [探索 ES6](http://exploringjs.com/es6.html)——有深度章节的好介绍。
*   [探索 ES2016 和 ES2017](http://exploringjs.com/es2016-es2017.html) -探索 ES6 的后续。
*   [ES6 Katas](http://es6katas.org)——在线解决单元测试学习 ES6。
*   [实用 ES6](https://github.com/mjavascript/practical-es6) -包含实用示例和建议的电子书。

# [](#5-stop-and-start-practicing)5。停下来开始练习！

现在是休息的时候，不要试图向你的头脑中引入更多的概念。没有更多的技术。

到目前为止，你已经有足够的知识来申请 IT 行业的工作。在我看来，学习和巩固知识最好的方法就是进行项目。现在你能够执行项目，是的，把它们作为练习来加强你已经获得的知识。然而，要有足够的堆栈来生成真正的项目，还有很长的路要走。

我有一篇关于这个话题的帖子，推荐你阅读。

# [](#6-databases-mysqlpostgresql)6。数据库- MySQL/PostgreSQL

一旦你掌握了客户端 web 开发(前端)的基础技术，是时候开始掌握服务器端开发(后端)的基础了。Web 开发中的这个新学科，必须从理解数据是如何存储的开始。信息是软件中最有价值的资产,因此，我们必须对数据库的工作原理有广泛的了解。

人们围绕数据库管理建立了他们的整个事业。所以这是一个大而重要的领域。要开始这方面的工作，我建议您学习关系数据库:MySQL 或 PostgreSQL。

### [](#database-design)[数据库设计](https://www.udemy.com/database-design/)

通过实用的概念和示例学习关系数据库设计。本系列适用于任何数据库系统！

### [](#sql-exercises)[SQL 练习](https://github.com/XD-DENG/SQL-exercise)

这些模式和问题来自维基百科 [**SQL 练习**](https://en.wikibooks.org/wiki/SQL_Exercises) 。解决方案来自这本维基百科和回购的作者。

# [](#7-backend-phpnodejsjava)7。后端- PHP/NodeJS/JAVA

后端语言允许您充分利用数据库的潜力，因为您可以生成动态的 web 应用程序。在这些 Web 应用程序中，前端将从后端接收数据，这就是 Web 开发的神奇之处，因为数据是动态和持久的。

[![backend](../Images/09368fa96732dddbd48730a0923b6dbb.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--gGtK0EaV--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1000/0%2AhCrPRSdeOdcXB7uy.png)

最简单的开始方式是构建一个 APIrestful 第一次会非常非常复杂，但是很正常。我们每个人都遇到过这种情况，但是一旦完成，你会发现你只是揭开了冰山一角，这是后端全部潜力的第一瞥。我建议开始学习后端是从广泛使用的语言开始，如 [node.js](https://nodejs.org/en) 或 [PHP](https://www.php.net) 。

### [](#php-tutorial-for-beginners-learn-in-7-days)[PHP 初学者教程:7 天学会](https://www.guru99.com/php-tutorials.html)

PHP 是网络上最流行的脚本语言。没有 PHP 脸书，雅虎，谷歌就不会存在。该课程旨在让你成为一名 PHP 专业人员。一旦你理解了所有的基础知识，本课程将帮助你创建你自己的民意测验应用程序。

Node.js 也基于 JavaScript 框架，但它用于开发基于服务器的应用程序。在浏览整个教程时，我们将详细研究 Node.js，以及如何使用它来开发基于服务器的应用程序。

### [](#pokemon-api)[口袋妖怪 API](https://github.com/NyaGarcia/pokeAPI)

一个 Pokemon RESTful API，内置于 node.js、Express、MongoDB、Docker，尤其是 TypeScript。

### [](#node-tutorial-for-frontend-devs)[前端开发节点教程](https://github.com/cwbuecheler/node-tutorial-for-frontend-devs)

带有 Node.js、Express 和 MongoDB 的员工数据库 CRUD 应用程序示例。

网络上有很多过时的信息，将 PHP 新用户引入歧途，传播不良实践和不安全的代码。PHP:《正确的方法》是 PHP 流行编码标准的易读、快速的参考资料，链接到网上的权威教程，以及贡献者认为目前的最佳实践。

# [](#bonus-frameworks)加成:框架

在这一点上，你将有知识开始你的职业生涯作为一个网页开发人员。然而，从第三步开始，你已经可以在这个行业中找到一份工作，并从那里开始，继续发展成为一名开发人员。

下一步是加深你对框架的了解，因为它们允许你轻松地配置你在这篇文章中学到的基本概念。框架是帮助你更快开发的工具，并且允许你创建更大的项目。

现在，我将为您提供一些最有用和最流行的框架的参考，这些框架的特性将帮助您生成优秀的代码:

*   [循序渐进 Laravel 后端教程](https://github.com/0x3fc/laravel-backend-tutorial)。

*   [Angular 教程:从零开始一步一步地学习 Angular](https://github.com/AngularTemplates/learn-angular-from-scratch-step-by-step)-这个项目是从零开始一步一步地学习 Angular 教程的一部分，在这个教程中，我们从 Angular 框架的基本概念和原因开始探索，到使用 Angular 材质组件构建一个完整的 Angular 7 应用程序。

*   Spring Boot 有很多吸引人的地方。用它开发服务很酷也很有趣。Spring Boot 使得创建独立的、生产级的基于 Spring 的应用程序变得容易，你可以“直接运行”。大多数 Spring Boot 应用需要很少的弹簧配置。
