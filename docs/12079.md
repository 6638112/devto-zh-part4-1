# 你如何回应“你最大的优点/缺点是什么？”在采访中？

> 原文：<https://dev.to/spacerockmedia/how-do-you-respond-to-what-is-tour-greatest-strength-weakness-in-interviews-3b68>

就像标题所说的那样。在一次采访中，我从来没有被问到 queariins:

旅游最大的优势是什么？
你最大的弱点是什么？

最近，我试图回应他们，告诉他们我必须先告诉我的朋友。我的弱点是冒名顶替综合症。当我看到一个 12 岁的程序设计时，我觉得自己不够格，或者有时会觉得自己不够格，因为 5 岁和我处于同一水平。描绘出我知道他们不是，但感觉是那样的。

我的优势在于，我知道我的坚持、对细节的关注和团队合作将永远推动我做出可靠的软件。

他们似乎对此反应很好。我没有得到这份工作，但这是一个要求很高的职位，我也没想到会得到。但是我喜欢他们对我的回答的回应。
