## 挑战规则

1.  我将使用#DisneyCodesChallenge 标签在推特上发布我对迪士尼编码挑战的承诺。如果我觉得大胆和大胆，我可以说你想用什么字符编码。这个角色可以是迪士尼拥有的任何角色，比如迪士尼、星球大战、漫威、皮克斯等等。
2.  在挑战期间，我将每天编码 23 分钟。
3.  我会更新日志页面，这样其他人就可以看到我的进展，看到我在建设什么。
4.  我会每天用#DisneyCodesChallenge 标签发布我的进展。
5.  我将编码项目。这意味着没有教程，在线课程和任何类似的资源。
6.  我会编码项目是迪士尼主题或由迪士尼拥有的任何东西的启发。你可以在下面的常见问题部分了解更多迪士尼主题。

## 常见问题

### 问:你说的迪士尼主题项目是什么意思？

迪士尼主题项目是任何受华特·迪士尼公司拥有的任何东西启发的项目。因此，简而言之，这意味着你可以制作受漫威、星球大战和皮克斯启发的项目。你甚至可以制作受华特·迪士尼公司部门启发的项目，如迪士尼公园、迪士尼消费品，甚至他们管理的电视频道，如 ESPN、国家地理和美国广播公司。你可以在华特·迪士尼公司页面上了解更多关于迪士尼的信息。

[== >点击这里访问华特·迪士尼公司页面！](https://www.thewaltdisneycompany.com/about/#our-businesses)

### 问:我可以用其他编码挑战来完成这个挑战吗(例如，100 天的代码，301 天的代码)？

是啊！编码挑战是养成项目编码习惯和突破教程的好方法。这个挑战是为了做同样的事情，这些编码挑战只有你有机会与迪士尼人物编码。

### 问:对于这个挑战，我需要使用特定的编程语言/框架吗？

没有。在这个挑战中，你可以使用任何你喜欢的语言/框架。我建议选择有助于你成为更好的开发人员的语言和框架。所以选择那些你可能在教程中学到的或者你觉得需要更多练习的。

### 问:我刚刚开始学习如何编码。我能做这个挑战吗？

是啊！这个挑战是为任何人设计的，无论他们是迪士尼的新手还是软件工程师，都可以做这个挑战。编码类似于学习任何技能，所以你练习得越多，你就会变得越好。

### 问:我想不出任何要构建的项目。我该怎么办？

两个很好的起点是自由代码营或 Odin 项目。这是两个受欢迎的开发资源，包括学生可以用来练习他们正在学习的新技能的项目。你可以将这些项目创意作为基础，然后根据迪士尼主题进行定制。

[== >点击这里了解更多免费代码营！](https://www.freecodecamp.org/)
[== >点击这里了解更多奥丁计划！](https://www.theodinproject.com/)

帮助你开始的另一个很好的选择是其他的编码挑战。像 30 天、30 个网站或每日 CSS 图像这样的挑战是开始构建项目的好方法，因为这些网站通常会提供每日提示，告诉你每天可以构建什么。

[== >点击这里了解更多 30 天，30 个站点！](https://www.subscribepage.com/30days30sites)
[== >点击这里了解更多日常 CSS 图片！](http://dailycssimages.com/)

要有创造力，运用你的想象力。是否尝试重新创建迪士尼商店网站？你想为蒂安娜宫建立一个网站吗？也许你想为复仇者联盟设计一个网站。也许你想为艾尔的玩具谷仓创建一个应用程序。这是你的机会，以建立你的版本的一些这些项目。

### 问:我有一个博客。我可以在那里张贴我的日志吗？

是啊！只需使用日志模板来帮助你在你最喜欢的博客平台上写博文。

### 问:我喜欢这个挑战！我如何才能看到其他人在为挑战做些什么？

在推特上搜索#DisneyCodesChallenge。在这里，您可以看到其他人发布的内容，并支持其他人进行挑战。在这个帖子的时候，你可以看到我所有的帖子和挑战过程中的进展。您将了解我在构建什么，以及我编码时发生了什么。

## 我该如何加入挑战？

1.  阅读规则和常见问题页面。
2.  使用#DisneyCodesChallenge 标签发布你对挑战的承诺。如果你愿意，你也可以在推特上说你想和哪个迪士尼角色一起写代码。
3.  叉挑战回购。如果你喜欢迪士尼代码挑战，一定要点击页面右上角的★按钮。
4.  每天 23 分钟开始编码。每天结束时，在日志页上写下你所取得进展的简要总结。
5.  使用#DisneyCodesChallenge 标签在推特上发布你每天的进展。
