# 推进:有抱负的 Web 开发人员的坚忍

> 原文：<https://dev.to/raddevon/pushing-through-stoicism-for-aspiring-web-developers-26lg>

你决定成为一名网络开发人员。厉害！在几周或几个月的时间里，保持对这个目标的承诺会很容易。你会很兴奋，不断学习新东西，锻炼创造性肌肉你现在的工作可能一点也不工作。你会觉得比很久以前更有活力。

然而，这种感觉并不总是会有的，你是否能实现这个目标取决于当事情*不*那么美好时，你是否能坚持下去。您无法让 Webpack 构建您的项目，并且您不确定原因。你已经申请了几个月的工作，甚至还没有得到回复。你刚刚在 StackOverflow 上发布了一个问题，而不是一个答案，它被遗忘了。

古希腊人来救援了！

## 什么是斯多葛主义(以及为什么你应该关心)

你可能不会想到从古代哲学中寻找这个问题的答案，但是斯多葛派并不仅仅是古代哲学。许多哲学流派研究的是社会应该如何构建，或者宇宙的本质和意义，而斯多葛派研究的是个人如何处理自己的问题。

我将引用一些著名的斯多葛派哲学家的话，并解释这将如何帮助你成为一名网站开发者。

> 抱怨不是一种策略。我们都有有限的时间和精力。我们花在抱怨上的任何时间都不可能帮助我们实现目标。也不会让我们更幸福。
> 
> —兰迪·波许

作为 web 开发人员——作为人类，真的——我们有太多的抱怨，但是这些都没有用。找出前进的方向，并抓住它。

这是兰迪·波许的最后一堂课。他是斯多葛派的现代英雄。如果你还没有，请注意。

> 命运引领着心甘情愿的人，拖着不情愿的人。
> 
> ——塞涅卡

人们问我对 SquareSpace、WordPress.com 和 Wix 等简单的网站构建者有何看法。我听到许多开发人员和设计人员感叹工具的存在使他们传统的工作变得更容易和更容易。

我对此有不同的看法。网站建设者是一个伟大的选择，有人与基本需求和低预算，以获得一些伟大的东西，只有几美元一个月和一些汗水公平！与其对这些工具生气，**我会主动推荐客户使用它们**。当我这样做的时候，我通过给他们一个选择来建立信任，当我太贵或者只是不适合工作的时候。

后来，当他们取得一些成功，需要比他们的网站更多的东西时，他们会回来找我。这些工具让我的客户的事情变得更容易，并清理了我的时间表，让我可以从事**更有趣的项目**。厉害！

Web 开发变化很快，所以在你的职业生涯中，你会经历许多类似的转变，你过去做的事情不再有价值。你可以生气，因为你投入时间学习的框架不再相关，或者你可以找到一种新的方式为你的客户和雇主增加价值。

> 紧紧抓住下面的规则:不要向逆境屈服，永远不要相信顺境，永远要充分注意到命运的习惯，她想做什么就做什么，对待她就好像她真的要做她力所能及的一切。无论你期待了多长时间，这都不会带来太大的冲击。
> 
> ——塞涅卡

早在我听说斯多葛主义之前，我就有了自己的哲学版本:做最坏的打算，抱最好的希望。(就问我家；他们完全听腻了！😜)

当你进入最后一轮面试或者一份自由职业合同即将签署时，你会需要这个。你会直觉地开始庆祝，就像你做的那样，**一切都会分崩离析**。

相反，**继续下去，就好像你甚至*没有*这个机会**。继续安排下一次面试，下一次演出…如果第一次成功了，那太好了！如果没有，你仍然有一些光明的替代工作。

> 人生的主要任务很简单:识别和区分事物，这样我就能清楚地告诉自己，哪些是不受我控制的外部事物，哪些与我实际控制的选择有关。
> 
> —爱比克泰德

你*不能*根据你的简历控制别人是否会雇用你，但是你*可以*控制**把简历放在多少张桌子上**。

当你的银行账户几乎空空如也时，你*不能*控制你是否会得到一份自由职业，但是你*可以*控制**你在**投了多少份建议书。

你*不能*控制客户是否会从你提供给他们的三个选项中选择你最喜欢的解决方案，但是你*可以*控制**你说什么来教育他们**为什么他们应该这样做。

花很多时间在你能控制的事情上，不要花时间在你不能控制的事情上。

> 这就是为什么哲学家警告我们不要满足于单纯的学习，而是要加上实践和训练。
> 
> —爱比克泰德

*这位*哲学家在告诉你 **[停止做教程](https://raddevon.com/articles/stop-doing-coding-tutorials/)** 。我并不是说你应该*永远不要*使用它们，而是不要再依赖这种拐杖作为你学习的唯一途径。

教程很有趣，因为你遵循一个很酷的食谱，最终你会得到一个让你感觉良好的工作。不幸的是，发展不是这样。开发就是除了问题之外一无所有，并且必须在软件中构建解决方案。这是在 **4 个糟糕的选项**中做出选择，每个选项都有你认为无法接受的警告……但你别无选择。发展正在迎接这些挑战，尽管有这些挑战,**还是拿出了解决方案。教程不会教这些，所以，如果你想成为一名开发人员，请少用它们。**

> 不要追求每件事都如你所愿，而是希望每件事都如它实际发生的那样——这样你的生活会过得很好。
> 
> —爱比克泰德

和

> 首先对自己说，你会成为什么样的人；然后做你该做的。
> 
> —爱比克泰德

你想成为一名网络开发人员，否则你不会在这里。你越严格地定义你的目标，你就越不可能实现它。如果你在大学读了 4 年，获得了计算机科学学位，获得了谷歌的实习机会，然后被 YouTube 的分析公司聘为工程师，那么祝你好运。你只定义了一种成功的方式，却留下了无数种失败的方式。

设定一个宽松的目标，选择一条你认为可能让你达到目标的道路，然后开始沿着它走下去。睁大你的眼睛，看看你可能实现目标的其他方式。我转行从事网络开发，但我从 16 岁起就知道我想开发软件。我以为我会上 4 年大学，然后他们会把我送到一家很棒的公司，在那里我会一直工作到退休。

相反，我在去社区大学的时候在沃尔玛工作。之后，我开始在田纳西大学攻读学士学位，直到我女儿出生。我真的没有时间继续上学，所以那学期之后我就没有入学了。我换了其他工作，但都不是软件方面的，直到我设定了搬到西雅图的目标(是的，甚至不是成为一名开发人员的目标；[下面是为什么](https://raddevon.com/stick-with-web-development/))30 多岁。我能够实现这个目标，部分原因是因为**我没有明确定义必须如何实现**。

> 如果一个人不知道他驶向哪个港口，没有风是有利的。
> 
> ——塞涅卡

很多人问我[应该学哪门语言](https://raddevon.com/articles/which-web-development-language/)，[应该买哪台笔记本电脑](https://raddevon.com/articles/the-best-laptop-for-web-development-its-complicated/)，或者[应该用哪个编辑器](https://raddevon.com/articles/best-ide-beginning-web-developer/)。我用一个简单的建议和最少的解释来回答这些问题，尽管我获得这些建议的途径更加微妙。为什么？因为这些问题会让人分心。

甚至看起来更大的问题也不是真正的问题。如果我是自由职业者，我应该成立公司还是独资经营？我应该努力为哪种客户服务？我应该支付 500 美元加入这个 BNI 小组吗？

决策最少的道路会让你朝着正确的方向前进。选择一个方向，然后朝着它前进。稍后更正。几乎在一瞬间，这些决定中的任何一个都可以被改变**。当你回顾你因为犹豫不决而失去的所有机会时，那将是你*无法改变的决定。***

> 对别人宽容，对自己严格。
> 
> —马库斯·奥勒留

当你递交简历，与招聘人员交谈，或者尝试接自由职业者的工作时，你会发现**人们背弃了对你的承诺**。人们经常这样做**。你*不想这么做，因为这对你来说很糟糕，不管你在追求什么，但同时，当别人这么做时，你希望得到理解。***

 **当我第一次开始工作时，我在安排和左右两边的人开会。每当有人在我们约定的时间放我鸽子，我都会怒气冲冲地回家。我会对自己说，“真不敢相信他们让我坐在那里。人怎么了？他们都只是混蛋吗？”

这种情况持续了几个星期，直到我变成了那个混蛋。我已经安排了一个会议，全神贯注于一个项目，完全忘记了去。因为我知道*我*不是一个怪人，那次经历让我**为其他犯错误的人腾出空间**。今天，我可以容忍和原谅任何错误，因为这些错误都是我自己犯的。

这并不意味着你应该继续依赖那些经常出问题的人。给他们一些机会，不要附加任何条件，然后继续前进。我的目标是成为 100%可靠的自己，同时给别人留有余地，我希望有一天他们会允许我。

> 你可以现在就离开生活。让它决定你做什么，说什么，想什么。
> 
> —马库斯·奥勒留

当然，这不是转行的合适时机，因为你刚刚有了一个新的孩子，或者你将要有一个孩子，或者你刚刚搬家，或者你即将获得晋升。生活总是在变化，而你总是在做*某事*。如果你等到生活节奏慢下来才换职业，**你永远也不会成功。**

既然你的工作生活在你的生活中占了这么大的一部分，你应该对它感到满意。不要等待合适的时机。**放手一搏**。

当你觉得自己永远也不会成为一名 web 开发人员时，使用这些工具来让自己坚持下去。如果你想要更多的工具来帮助，参加[我的大目标课程](https://raddevon.com/stick-with-web-development/)作为另一个重要工具**来补充你的意志力并让你朝着正确的方向前进**。你会成功的，但这需要努力、坚持和坚忍不拔的精神来打造你的未来。**
