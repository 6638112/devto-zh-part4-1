# 面向初学者的 Python 书籍

> 原文:[https://dev.to/bodiam/python-books-for-beginners-3il4](https://dev.to/bodiam/python-books-for-beginners-3il4)

我很快就要开始新的工作了，为此，是时候再次重温一下我的 Python 技能了。对我来说，重新学习一门语言的最好方法之一就是读一本书，尝试一些例子，所以我搜索了一下，列出了这个列表。不幸的是，有相当多的书有虚假评论，所以我想我会与你分享我的过滤结果。我将分享几本适合初学者的最好的 Python 书籍，还有一本稍微高级一点的。

如果你对 Python 完全陌生；Python 是一种解释的、面向对象的高级编程语言，相对容易学习。那么，想学 Python 从哪里开始呢？享受这些书籍，它们将引导你了解你需要知道的一切:

*   **[学习 Python，第 5 版](https://amzn.to/2XpXsRv)** 学习 Python，第 5 版，今年即将满 6 岁，但它仍然为您提供了一个全面的 Python 学习工具。它涵盖了 Python 编程的广泛主题，比如类型&操作、语句和语法、函数和生成器、模块和包等等。第一章是一个很好的开始，这是一个关于 Python 和为什么人们使用 Python 的问答环节。如果你是初学者，这可以帮助你很多。这本书的一个很好的特点是每章都以一个小测验结束，所以你可以验证自己是否正确理解了主题。
*   [**Python 速成班，第二版:基于项目的编程实践入门**](https://amzn.to/2RkGChA) 这是一本非常适合初学者的有趣的书，刚刚更新到新版本！这本书得到了很好的评论，但它假设这是你的第一本编码书。如果不是，那么也许《学习 Python》是一本更好的书，但如果这是你的第一本编码书，作者将带你通过几个有趣的项目来动手实践学习编码！
*   **[以为蟒。如何像计算机科学家一样思考](https://amzn.to/2XhsGKH)** [](https://amzn.to/2XhsGKH)如果你是编程新手或者来自另一种语言，这是一本很棒的书。每章结尾都有调试技巧、术语表和一些练习来帮助你继续。这真的是一个实践指南，它将一步一步地教你 Python。
*   **[Python 入门:从新手到专业，第二版](https://amzn.to/2Rkvz80)** 这本书给了你一个很好的概述，并深入涵盖了 Python 的基础知识。非常喜欢，这本书的第一部分指导你完成一系列的任务，包括处理文件、数据库、网络编程和测试。最后十章由十个编程项目组成:“为了学习编程，你必须编程”。这些项目包括:自动文本标记、远程文本编辑器、XML-RPC 文件共享、一个“自己动手”的街机游戏等等。
*   **[学 Python 3 敬酒不吃吃罚酒](https://amzn.to/2Knxubf)** 有人喜欢，有人不喜欢，似乎没有中间地带。《艰难地学习代码》以免费的 pdf 格式提供。该系列中的其他书籍涵盖了 Python、Ruby、C、SQL 或 Regex“the hard way”。这些书是 Zed Shaw 写的，会给你更多关于如何开始上述话题的见解。《艰难之路》是为对编程一无所知的初学者编写的，它将教会你编程的基础。您将学习如何使用终端和文本编辑器。通过尝试这些例子，这本书关注的是在实践中学习。
*   **[《Python 袖珍参考》(袖珍参考(O'Reilly))](https://amzn.to/2RnA1mD)** 如果你经常旅行，这本书很适合你！一本方便的袖珍指南，值得随身携带。本参考涵盖了 Python 3.4 和 2.7。虽然有点老套，但它准确、易懂、写得非常好。如果你还在路上，并且不能在网上浏览 Python 的文档，这本袖珍书会非常方便。
*   **[自然语言处理在行动:用 Python 理解、分析和生成文本](https://amzn.to/2XhpFtO)** 如果你是 Python 的初学者，这可能不是最好的入门书籍，但 NLP 是一个很好的主题，这本书得到了很多疯狂的评论，这一次还没有一个机器人写的。如果你对自然语言处理感兴趣，这绝对是一本值得一看的书。

当然还有很多关于 Python 的书，所以如果你有关于哪些书真正值得一提的建议，请告诉我！
