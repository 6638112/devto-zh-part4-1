## 打造你的 [first-pwa](https://pwabuzz.web.app/)

在[这个初学者代码实验室](https://pwafire.org/developer/codelabs/pwafire/)学习如何用[pwafire.org](https://pwafire.org)构建你的第一个渐进式网络应用

*   在这个 codelab 中，你将能够使用大部分静态资源构建一个渐进式 web 应用程序，或者最适合静态 web 应用程序或网站。

## 使用[工具箱](https://developers.google.com/web/tools/workbox/)代替

我们将添加一个更高级的 codelab，帮助您学习如何构建动态的或用于大规模生产场景的渐进式 Web 应用程序，它具有以下功能:

*   选择要缓存的内容

*   规定如何缓存

*   决定缓存多长时间

*   决定何时缓存

*   决定缓存哪些资源

*   决定要缓存多少资源
