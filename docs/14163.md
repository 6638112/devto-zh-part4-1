## [](#gitmoji)吉泰米

这是由卡洛斯·单面山创建的提交消息惯例😎。它在 GitHub 上的 gitmoji 的[库](https://github.com/carloscuesta/gitmoji/)中描述了这个约定的目的:

> Gitmoji 是一个标准化和解释 GitHub 提交消息上表情符号使用的倡议。
> 
> 在提交消息上使用表情符号提供了一种识别提交目的或意图的简单方法，只需查看使用的表情符号。由于有很多不同的表情符号，我发现有必要创建一个指南，可以帮助使用表情符号更容易。

## [](#gitmojichangelog)gitmoji-changelog

与[本杰明·皮特托](https://twitter.com/bpetetot)和[杨奇煜·朱伊夫](https://twitter.com/fabienjuif)一起，我们开发了一个命令行工具，按照 gitmoji 惯例为 javascript 项目生成变更日志文件。我们简单地把它命名为`gitmoji-changelog`😀。

## [](#20-is-in-alpha)2.0 是在α🎉

`gitmoji-changelog`仍在不断发展。我们目前正在开发版本 2.0.0💪它已经可以在 alpha 中使用了🎉

你可以在我们的[更新日志](https://github.com/frinyvonnick/gitmoji-changelog/blob/master/CHANGELOG.md#200-alpha1-2019-06-12)中找到新的特性😉！

为了测试它，你可以安装 alpha 版本:

```
npm install -g gitmoji-changelog@alpha 
```
