# 仅 CSS:3D 动画主页图标概念

> 原文：<https://dev.to/takaneichinose/css-only-3d-home-icon-concept-2ljo>

最近，我对制作一个有过多图标和动画的网站很感兴趣。可能吧，因为我看了一篇关于用野蛮主义设计的文章。

也许，如果我有机会做一个网站设计，我想尝试一下野蛮主义的概念。虽然我想尽量减少一点。此外，一个过度的动画，这将实际上使网站愉快，而不是恼人的。

我还没有使用任何 Javascript 代码来制作动画，不过，有一个特定的库我想学习。

这只是一个简单的“主页图标”的概念(悬停时会有动画)，用于基于 3D 的用户界面，然后，会有一个左或右图标，将转到其他导航，如“关于”，“联系我”等。所以也许，它对菜单项有好处。
