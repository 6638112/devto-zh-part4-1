##### [](#daily-news-)每日新闻📰：

*   [dev.to](https://dev.to)
    本网站。不需要更多信息😍

*   我用推特来了解最新消息。我真的很喜欢 Twitter 的新设计，它还为我提供了[探索功能](https://twitter.com/explore)，你可能已经从这个应用程序中知道了。如果你愿意，也可以在 Twitter 上关注我( [@fullstack_to](https://twitter.com/fullstack_to) )😉。

*   css-tricks.com
    在我的工作中，我的重点不是写 CSS 或做网页设计。但是我对这个话题很感兴趣。CSS-Tricks 有一些很棒的文章和解决我大部分(CSS)问题的方法。

##### [](#cheat-sheets-)备忘单📋：

*   数组浏览器
    在 JavaScript 中你可以用数组做很多事情。太多了，记不全了。Array explorer 允许您选择想要实现的功能，并为您选择的功能提供代码示例和文档。

*   [对象浏览器](https://objectexplorer.netlify.com)
    类似于*数组浏览器*，但用于 JavaScript 对象。

*   [CSS-Grid generator](https://cssgrid-generator.netlify.com)
    如上所述，我并不经常做 CSS。要创建完美的 CSS 网格，CSS-Grid 生成器非常方便。

*   [cheat.sh](https://cheat.sh)

*   [Git 和 Git 流备忘单](https://github.com/arslanbilal/git-cheat-sheet#readme)
    基本的 Git 命令(像`git add`、`git commit`、`git checkout`或者`git merge`)已经在我的大脑里显现了。但是我记不住更多的命令，因此我使用 Git 和 Git Flow 备忘单。

##### [](#tools-)工具🧰:

*   [Regex 101](https://regex101.com/)
    对于*我*个人来说，正则表达式很难记住，我欢迎任何帮助我编写正则表达式的工具。Regex101 是我的首选网站，因为它有快速提示，匹配解释，并向我解释为什么我的 Regex 匹配(或不匹配😅).

*   [Base64 编码器/解码器](https://base64decode.org)
    您也可以在您的终端中对 Base64 数据进行编码和解码。但是大多数时候我用 base64decode.org。

*   [去除背景](https://www.remove.bg/)
    有时候我需要从图像中去除背景。remove.bg 是完成这项任务的完美工具。

*   [JSON 查询操场](https://jqplay.org/)
    我经常需要处理 JSON 数据(例如在 API 响应中)。我喜欢用 [jq](https://dev.to/fullstack_to/tools-services-i-use-je9#jq) 来查询 JSON 数据。JSON Query Playground 允许我在浏览器中构建 jq 查询。

*   你可能已经注意到，我的母语不是英语🤗。Deepl 帮助我检查我的措辞是否正确(如果你能推荐一个好的语法检查工具，请放在评论里🙏).

*   语言工具帮助我校对我的文章。还有 Chrome、Firefox、谷歌文档、微软 Word、LibreOffice 和[等更多](http://wiki.languagetool.org/software-that-supports-languagetool-as-a-plug-in-or-add-on)的浏览器扩展。

#### [](#hosting-)主持☁️:

*   Netlify
    Netlify 是静态网站的绝佳去处。我在那里有几个网站，部署在几秒钟内完成。

*   由于 DigitalOcean 提供托管 Kubernetes，我建立了一个集群进行测试。

*   来自谷歌的托管数据库、认证服务和静态文件托管。非常适合兼职项目👏

##### [](#docs-)文档📗：

*   [码头工人](https://docs.docker.com/)
    码头工人🐳文档。我还能说什么？

*   Kubernetes
    Kubernetes 是一只强大的野兽，但它让容器化应用的部署、扩展和管理变得如此简单(这取决于你将它与什么相比较😉).Kubernetes 文档是解决我的问题的一站式商店。

*   我最近开始使用 *GatsbyJS* ，对于大多数任务，我需要查阅他们的文档。

*   T2:我用 Angular 很多年了，但几个月前换成了 React。我经常需要查看 React 文档(即使我非常努力，我也记不住*条件渲染*语法🤫).

*   [DigitalOcean 教程](https://www.digitalocean.com/community/tutorials)
    有很多 DigitalOcean 社区成员写的好教程。在撰写本文时，有超过 2200 个教程。主题范围从设置 Linux 服务器到高度专业化的主题，比如安装 Galera 集群或使用 Istio 和 Kubernetes 进行 canary 部署。
