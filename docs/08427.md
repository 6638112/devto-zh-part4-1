# [](#my-discovery)我的发现

在大学期间，我发现听有歌词的音乐完全让我分心，让我很难集中注意力。

*是的，我经历了惨痛的教训*

当我需要集中注意力的时候，我会听爵士乐、Lofi、古典音乐、原声音乐，或者基本上任何乐器。
