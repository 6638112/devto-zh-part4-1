# 我怎么知道苹果讨厌开发者？

> 原文:[https://dev . to/iglizer/how-I-know-apple-hates-developers-kn8](https://dev.to/iglizer/how-i-know-apple-hates-developers-kn8)

因此，我一直在尝试将 iMessage 的免费贴纸包上传到 AppStore。为什么是“尝试”？因为它总是因为一些愚蠢的原因而被拒绝。没什么复杂的:只是一堆我们公司关于 dev life 的贴纸。“开发者的贴纸，开发者的。”

第一次苹果因为整体设计已经婉拒了 app，第二次是因为我在标题里用了 iMessage。但这次完全没有意义。这是我从苹果公司得到的信息:

“您的应用包含对测试、试用、演示、测试版、预发布或其他不完整内容的引用。
具体来说，你的应用的一些贴纸有“测试版”参考。

啊？查看图片-我们的一个开发相关的贴纸上写着“仍在测试中”。我想苹果把这种方式看得太重了。还好他们没有告诉我我的应用程序有太多错误，因为它有一个错误标签，上面写着“这不是一个错误，这是一个功能”...

仔细想想，如果 AppStore 是一个现代的精灵呢？也许我应该加一张贴纸，上面写着“苹果欠我 2000 万美元股票”？或者只是一个写着“苹果批准这个贴纸包”的。
