# AWS 和 Azure 上完全托管的 PostgreSQL 托管及时发布，以支持传统迁移

> 原文：<https://dev.to/scalegrid/fully-managed-postgresql-hosting-on-aws-and-azure-launches-in-time-for-legacy-migrations-la2>

[![Fully Managed PostgreSQL Hosting on AWS and Azure Launches in Time For Legacy Migrations](img/f83b234342f54e54888665a5298830ab.png)](https://scalegrid.io/blog/fully-managed-postgresql-hosting-on-aws-and-azure-launches-in-time-for-legacy-migrations/)

加利福尼亚州 PALO 阿尔托，2019 年 7 月 17 日——数据库即服务(DBaaS)领域的领导者 ScaleGrid 刚刚[宣布支持](https://scalegrid.io/blog/fully-managed-postgresql-hosting-on-aws-and-azure-launches-in-time-for-legacy-migrations/)在云中托管[PostgreSQL](https://scalegrid.io/postgresql.html)。这一完全托管的平台允许组织自动化其耗时的 PostgreSQL 操作，专注于数据库开发，并通过在 [AWS](https://scalegrid.io/postgresql/aws.html) 和 [Azure](https://scalegrid.io/postgresql/azure.html) 上的高级监控、高可用性和灾难恢复来优化性能。

随着组织继续对其遗留部署进行现代化改造，并从 Oracle、DB2 和 SQL Server 等商业数据库管理系统转向开源，PostgreSQL 已迅速成为云和内部部署的首选数据库。ScaleGrid 平台允许 PostgreSQL 用户自动化其日常操作，而不必牺牲对其部署的控制。ScaleGrid 是唯一一个允许您保留对您的机器的完全 PostgreSQL 超级用户访问和 SSH 访问的多云 DBaaS，使公司可以轻松采用托管数据库服务。

ScaleGrid 创始人兼首席执行官 Dharshan Rangegowda 说:“在过去的一年里，对 PostgreSQL 的需求绝对激增。“组织一直在寻找一种价格合理的 Oracle 替代产品，以避免高昂的服务成本，PostgreSQL 不仅在成本上符合要求，在功能和性能上也符合要求。”

借助 ScaleGrid 的 DBaaS 解决方案，PostgreSQL DBA、开发人员和 DevOps 工程师能够自动执行备份、升级、扩展、复制、加密、操作系统修补和警报，从而降低人为错误的风险，并将更多时间用于开发和性能优化。他们最受欢迎的工具之一是 PostgreSQL 慢速查询分析器，它允许用户快速地对减慢应用程序速度的查询进行故障诊断。作为[最耗时的 PostgreSQL 管理任务](https://scalegrid.io/blog/latest-postgresql-trends-most-time-consuming-tasks-important-metrics-to-track/)，查询分析对于保持健康的数据库性能极其重要，ScaleGrid 的热图可视化工具允许您轻松识别哪些查询运行最频繁、最慢或扫描最多的文档，以便进行优化。

[![PostgreSQL Hosting Deployment - Slow Query Analyzer - ScaleGrid DBaaS](img/2367d9ea36fc9c946bf31a219e9bf46f.png)](https://scalegrid.io/blog/wp-content/uploads/2019/07/MySQL-Slow-Query-Analyzer.gif)

数据库也因 PostgreSQL 扩展而广受欢迎，这些扩展使用户能够扩展他们数据库的功能。ScaleGrid 允许您无限制地安装任何第三方 PostgreSQL 扩展，包括流行的 pg_stat_statements、pg_repack、pgaudit 和 PostGIS。在他们的[比较 PostgreSQL 提供者](https://scalegrid.io/postgresql/hosting-comparison.html)页面上了解更多关于 ScaleGrid 的优势。

随着 PostgreSQL 的加入，ScaleGrid 现在支持四个开源数据库， [MySQL 托管](https://scalegrid.io/mysql.html)， [MongoDB 托管](https://scalegrid.io/mongodb.html)，以及 [Redis 托管](https://scalegrid.io/redis.html)在流行的云提供商 AWS、Azure 和 DigitalOcean 上。
