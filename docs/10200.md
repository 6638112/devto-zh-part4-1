## [](#positives-astrophysics-biology-and-sports)阳性:天体物理学、生物学和体育

*Kirit thad aka 在[风筝博客](https://kite.com/blog/python/future-of-data-science/)上的一篇新文章的一部分*

数据科学对技术影响我们生活的方式产生了巨大的积极影响。这些影响有些是好的，有些是不好的。*看脸书*但是，技术本身没有好坏之分，技术就是……技术。我们使用它的方式会产生好的或坏的结果。

我们最近在天体物理学上有了突破，有史以来第一张黑洞的照片。这有助于物理学家确认一个多世纪以来围绕黑洞和相对论的纯理论工作。

为了捕捉这张图像，科学家们使用了一个像地球一样大的望远镜(事件视界望远镜或 EHT)，通过组合来自八个地面射电望远镜的数据，并理解所有这些来构建图像。分析数据，然后可视化这些数据——听起来就像是数据科学。

关于这一点，有一个很酷的旁注:哈佛大学的 Andrew Chael 开发了一个用于 EHT 成像的标准 Python 函数库，用于模拟和操作 VLBI(甚长基线干涉测量)数据，帮助创建黑洞图像的过程。

康奈尔大学的 Olivier Elemento 利用大数据分析来帮助识别基因组中导致肿瘤细胞扩散的突变，以便可以更早杀死它们——这是数据科学对人类生活的巨大积极影响。你可以在这里阅读更多关于他令人难以置信的研究。

Python 被他实验室的研究人员用于测试统计和机器学习模型。Keras、NumPy、Scipy 和 Scikit-learn 是一些顶尖的 Python 库。

如果你是英格兰超级联赛的粉丝，你会欣赏莱斯特城在 2015-2016 赛季赢得冠军的例子。

在赛季初，博彩公司认为莱斯特城赢得 EPL 奖的可能性比发现尼斯湖水怪的几率低 10 倍。要更详细地描述这个故事的意义，请阅读这个。

每个人都想知道莱斯特是如何做到这一点的，事实证明数据科学发挥了很大作用！由于他们在分析和技术方面的投资，该俱乐部能够在球员训练时测量他们的健康水平和身体状况，以帮助防止受伤，同时根据球员的能量水平评估比赛中使用的最佳战术。

所有的训练课都有基于球员真实数据的计划，因此莱斯特城是那个赛季所有俱乐部中球员受伤最少的。

许多顶级球队使用数据分析来帮助球员表现，发掘人才，并了解如何为某些对手制定计划。

既然我们已经讲述了数据科学发现的一些惊人的事情，我也要谈谈一些负面的东西——批判性地思考技术以及它如何影响我们是很重要的。

不可否认，技术对我们生活的影响会随着时间的推移而增加，我们不应该在没有意识到它可能带来的积极和消极影响的情况下限制我们的理解。

这篇发表在 NIPS 上的论文讨论了如何在数据科学中经常使用的单词嵌入中应对性别偏见。

对于那些不熟悉这个术语的人来说，单词嵌入是一种表示单词的聪明方法，以便神经网络和其他计算机算法可以处理它们。
