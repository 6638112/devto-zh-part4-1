谁能告诉我用 js 写类方法最好的方法是什么，有什么好处和坏处吗？

# [](#first-approach)第一次接近

```
class Dog {

    constructor(name) {
         this._name = name;
    }

    bark() {
         console.log(`hello ${this._name}`);
    }

} 
```

# [](#second-approach)第二种方法

```
class Dog {

    constructor(name) {
         this._name = name;
    }

}

Dog.prototype.bark = function bark() {
     console.log(`hello ${this._name}`);
} 
```
