# 每日挑战#29 - Xs 和 Os

> 原文:[https://dev . to/the practical dev/daily-challenge-29-xs-and-OS-12mj](https://dev.to/thepracticaldev/daily-challenge-29-xs-and-os-12mj)

你能检查一个字符串是否有相同数量的 x 和 o 吗？

输入/输出示例:

```
XO("ooxx") => true
XO("xooxx") => false
XO("ooxXm") => true
XO("zpzpzpp") => true // when no 'x' and 'o' is present should return true
XO("zzoo") => false 
```

注意:该方法必须返回一个布尔值，并且不区分大小写。字符串可以包含任何字符
