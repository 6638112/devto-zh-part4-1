# 走出我的舒适区

> 原文:[https://dev.to/tngeene/getting-out-of-my-comfort-zone-59hj](https://dev.to/tngeene/getting-out-of-my-comfort-zone-59hj)

我最近在网上部署了我的网站。拥有自己的网络小天地一直是我的梦想，在那里我可以让自己超越家庭和国家的限制...对这个世界。成为创造者而不仅仅是消费者。然而，当一个目标看起来太遥远，我们无法想象一个清晰的路径来达到它，我们总是伸手去拿同一个抽屉:
[![Excuses](img/6c547595f744ce6010a2b6e6051110ae.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--yHnROFpK--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/hlfn6yd28wh8i6kaqqp2.jpg)

我似乎有很多这样的想法，真实的或想象的(大部分是想象的)，所以我把这个项目推迟了大半年。也许对互联网的恐惧和我可能得到的回应在我的拖延症中起了很大的作用。
除此之外，我决定建立这个网站，目的是:
把自己放在那里
，积极创造关于我喜欢做的事情、编程和谈论技术的内容...我们如何利用它作为一种工具为我们服务，而不是取代人际关系
无论如何，我不是专家，但我真的希望激励大多数人，如果不是所有人，参与我的内容。
如果你碰巧看到我的任何一篇文章，我会很感激你的反馈，无论是正面的还是批评性的。

一路平安。
