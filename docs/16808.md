# [](#vacation-goals)休假目标

今天过后，我将休假近一周。我和女儿一起去了#campusmoviefest，她正在放映一些她创作的短片。我为她感到非常兴奋，希望她能和那里的人们建立联系。

对于我来说，我刚刚买了一台新的笔记本电脑，我已经用 Godot 引擎、GDevelop、Android Studio 和 Visual Studio 代码(以及 Steam 和 Gog)为它做好了准备。在陪伴女儿的同时，我想为自己做几件事，包括:

*   读完 [Rust Book](https://doc.rust-lang.org/stable/book/) (还有 [Rust 和 Webassembly](https://rustwasm.github.io/docs/book/) 以及[Rust Async](https://rust-lang.github.io/async-book/)——我刚刚得知那些书是用 [mdbook](https://github.com/rust-lang-nursery/mdBook) 创作的...今天已经学到了新的东西！
*   读完 [Godot](https://docs.godotengine.org/en/3.1/) 文档。
*   在 Godot 中处理#ittybittyrpg。从头开始。会疯狂迭代。在过去的一年里，我对 Godot 了解了很多，这让我想用 Godot 来构建自己的工具。获得一个闪屏，并关闭编辑器和视图屏幕。让它输出一个简单的 HTML 文件。
*   在 GDevelop 的# dungeon 午后工作——让一个简单的人在地图上走来走去。0 级。

很多，对吧？啊我希望保持超级忙碌。或许还可以玩玩我积压的大量电脑游戏。
