# 追求第一份开发工作

> 原文：<https://dev.to/jimtryon/going-after-that-first-dev-job-3jgk>

在过去的 6 个月到一年里，我已经下定决心要把我的时间真正集中在学习 JavaScript 上，特别是尽我所能了解 React 以及相关的工具和技术。我从西雅图的就业市场和招聘启事中了解到，JavaScript 工作需求量很大。但是最近我对 C#编程和微软技术栈产生了兴趣。我不想养成这样的习惯:在没有掌握好一门语言的情况下，不断地从一门语言转向另一门语言。

我在职业生涯的因果报应上，正在做 21 天的挑战。这对我保持责任感和制定计划很有帮助。我脑子里有很多想法，关于如何进入寻找开发人员的公司，向他们推销自己。我也认为你不需要花 500-1000 美元请一位导师来教你如何找到一份开发工作。

我也不想落入冒名顶替综合症的阵营。我应该继续走 JavaScript 这条路还是转到 C#可以吗？我在想找一份 C#工作可能更容易。我没有任何数据支持这一点，但我很想听听其他人的想法！
