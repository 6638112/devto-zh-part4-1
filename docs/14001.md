# 国际经验:我从哪里开始？

> 原文：<https://dev.to/fabcat555/international-experience-where-do-i-start-15hb>

嗨！请注意，尽管标题可能暗示了什么，这不是一个自我宣传的帖子，也不是一个“请雇用我！”-喊救命😅另外，这可能有点跑题，但我很确定社区中的许多开发人员都和我有同样的情况，可以用一些建议来指出正确的方向！

我目前在米兰的一家金融服务软件开发公司担任全栈开发人员。在我看来，我每天处理的技术非常酷(我们与 Spring 和 Angular 7 合作)，自从我开始使用它们(大约 6 个月前)以来，我有机会学到了很多东西。
不过，我一直都是用特别的眼光看待外国的(至少，有一部分！).
尽管我热爱我的祖国，但我觉得海外经历可以极大地促进我的职业发展，同时从文化角度丰富我的人生。

在这方面，我必须说我对从哪里开始感到有点无所适从。我已经把国家列表缩小到几个(英国/美国/加拿大/西班牙，偏爱大城市)，但是下一步是什么？我应该开始建立关系网吗(以及如何建立关系网？)?我会随机向随机的公司发送简历吗？我应该放弃一切，不加思索地去冒险吗？:D

欢迎分享您的建议，或者更好的是，分享您的经验:)
