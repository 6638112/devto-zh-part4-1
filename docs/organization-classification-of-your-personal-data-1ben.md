# 您的个人数据的组织、分类

> 原文：<https://dev.to/flrichar/organization-classification-of-your-personal-data-1ben>

最近我和我弟弟聊了一会儿。从我记事起，我就对信息技术感兴趣。我们做的很多事情都是关于数据的组织。你如何看待数据，它是如何在你的头脑中，在你的思想中被可视化的？

我建议的事情之一是为分类创建约定。这适用于任何地方，不仅仅是代码或服务器，也适用于您自己的本地工作站。

我最喜欢的惯例之一是主题或类型，以及日期。我的备份看起来像是按年份排序的相同类型文件的目录。如果我在 2016 年搬家，把所有东西都扫描成 PDF 文件，这是找到它的超级快捷的方法。

我的 chrome 书签也是如此。Chrome 允许你将当前所有标签加入书签。我有一个名为 Stamped 的特殊组，其中我当时关注的当前标签是年-月-日格式的。我可以在 20140521 上告诉你我在读什么，或者关注什么，因为历史都在那里。你会对自己当时的想法有一个整体的了解，从那以后你去了哪里，你是如何成长的，也许还有你出于某种原因掉在地上的一些话题。

这个看起来很傻，但是对我来说已经用了很多年了。我知道大多数拥有编码技能的人可能很好地处理了他们自己的惯例。你用什么，有什么独特的吗？为什么呢？
