# 谷歌代码之夏采访 Aerogear

> 原文:[https://dev . to/ankitjey/interview-with-aero gear-for-Google-summer-of-code-1 HMC](https://dev.to/ankitjey/interview-with-aerogear-for-google-summer-of-code-1hmc)

已经花了两个多月的时间开发 Graphback，这是一款帮助开发者轻松构建 GraphQL 服务器的工具。从社区结合期开始到现在，这一直是一次很棒的经历。我相信最后一个月也会很棒。

最近，我接受了我的组织的采访，分享了我的经验。这是抄本。

你能简单介绍一下你自己吗？
我是 Ankit Jena，目前在 CET Bhubaneswar 攻读计算机科学本科。一年来，我一直兼职做全栈开发人员。

空闲时间你喜欢做什么？
如果我不常和朋友在一起，我喜欢在空闲时间看电影或玩游戏。此外，我还喜欢阅读有关技术创新的书籍。

你能告诉我们你的 GSOC 项目吗？
Graphback 是一个项目，旨在帮助开发者通过使用 GraphQL 规范中定义的数据模型来构建后端应用。它允许开发人员从许多可用的模板和扩展中选择一个命令来引导 GraphQL 服务器。它生成一个完全可定制的服务器，并允许用户根据自己的数据模型管理自己选择的数据库。它允许用户更专注于交付商业价值。

选择这个主题的原因是什么？
我从各种教程学习 GraphQL 的时候，设置项目有非常高的入门门槛。当这个想法出现时，我立即在脑海中喜欢上了它，因为我也面临过同样的问题，并认为这可以帮助人们。

你的项目对 AeroGear 社区有何帮助？
该项目将允许 aerogear 的开发人员根据他们的用例快速启动 GraphQL 服务器，并更加专注于提供特定于他们业务的实施，包括服务器、web 和移动客户端。

你的项目中最具挑战性的元素是什么？
项目最具挑战性的部分是最初决定构建项目核心的正确方法。我在几周内做了大量的研究，并决定采用后来碰巧成功的方法。

你的 GSOC 有什么搞笑、刺激或惊喜的瞬间吗？
真正让我兴奋的事情是当我们发布 Graphback 的第一个工作版本时，我们收到了来自开发者的评论，要求提供一些功能。这真的让我很兴奋，因为这是我的第一个开源项目，而且评论是积极的。

一个令人惊讶的时刻是，我做了一份公关，我的导师要求我做出改变。这些都是我做的，当我早上醒来时，我的导师说这是他见过的最好的公关之一。这真的很令人惊讶，我很高兴随着时间的推移我有所进步。

做这个项目时，你遇到过什么问题或障碍吗？
在社团结合期，我所在的州遭受了一场严重的气旋风暴(*范尼*)，造成了很大的损失。因此，在修复工程中出现了大量的电涌，导致最初的进度中断。还有，我的考试因为这个被重新安排到了 GSoC 的编码期。

第二个月的时候，我的笔记本电脑主板短路，不得不借朋友的笔记本电脑，在里面工作了差不多 20 天。

继 GSOC 之后，你是否计划参与红帽/AeroGear 项目？
是的，如果有机会，我愿意继续为 Graphback 和其他红帽/Aerogear 项目做贡献。
