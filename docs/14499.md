### [](#rank-1-woocommerce)*排名 1* - **WooCommerce**

[![](../Images/da0cea84b423598fa768d6c8c62369bf.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--nIs7u6U2--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/bkcreayke500isjdc15e.png)

WooCommerce 是一个免费的 Wordpress 插件，允许在线商家将他们的 Wordpress 网站变成在线商店。

作为全球最受欢迎的电子商务平台之一，WooCommerce 拥有自己的安全支付处理器和购物车。对于拥有 Wordpress 开发资源的小品牌来说，这个开源平台是一个不错的选择。

尽管它是免费的，但当你想完全整合购物车时，它需要一点钱。如果你打算扩大业务规模，你可以选择一个有些不同的电子商务平台——woo commerce。
