# 每日挑战#17 -双重麻烦

> 原文：<https://dev.to/thepracticaldev/daily-challenge-17-double-trouble-1ee6>

早上好！我们又回来了另一个挑战，来自 Codewars 的 AlexisHappy。今天，你被要求写一个程序，返回将在自动售货机上喝第 n 杯可乐的人的名字

> 谢尔顿、伦纳德、佩妮、拉杰什和霍华德在排队买“双份可乐”饮料自动售货机；队列中没有其他人。第一个排队的人(谢尔顿)买一罐，喝了它，然后加倍！由此产生的两个谢尔登到队列的末尾。然后下一个排队的人(伦纳德)买了一罐，喝了它，然后以两个伦纳德的身份排到了队伍的最后，以此类推。
> 
> 例如，Penny 喝了第三罐可乐，队列将如下所示:
> 
> 拉杰什，霍华德，谢尔顿，谢尔顿，伦纳德，伦纳德，佩妮，佩妮
> 
> 写一个程序，返回喝第 n 杯可乐的人的名字。输入数据由一个至少包含 1 个名字的数组和一个整数 n 组成，整数 n 可以是你所选择的语言支持的最大数字(当然，如果有这样的限制)。输出数据返回单行—喝第 n 罐可乐的人的名字。罐子从 1 开始编号。

祝你好运，并快乐编码！
