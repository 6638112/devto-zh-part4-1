# 你知道任何个人项目的社交网络吗？

> 原文:[https://dev . to/gmartigny/do-you-know-any-social-network-for-personal-projects-1n 17](https://dev.to/gmartigny/do-you-know-any-social-network-for-personal-projects-1n17)

大家好，

我们中的许多人(不是每个人)都喜欢创建兼职项目，而且这些项目往往永远不会完成，或者永远见不到一个用户。

这么多的时间和动力都被浪费了，这是一种耻辱。所以我在为数码制作者寻找某种社交网络。无论是艺术、代码、声音等等，你都可以分享你的工作并得到公正的反馈。与此同时，它还可以将愿意合作、拥有互补技能的人联系起来。

外面有这么多社交网络，但我找不到任何致力于正在进行的项目的[我似乎不是唯一的一个](https://www.reddit.com/r/AppIdeas/comments/8sc63n/a_social_network_for_side_projects/)。

你知道类似的事情吗？
