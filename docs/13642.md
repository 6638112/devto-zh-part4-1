# 朱莉·布施

[![Julie Busch](img/fc3e1be419696a36e50a097ad26a70a1.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--SxPZqdIv--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://2018.cascadiajs.com/speakers/julie.jpg)

Condé Nast 是《连线》、《纽约客》、《GQ》、《Vogue》和《名利场》等杂志的出版商，它整合了 20 多个网站的代码库，以提高开发速度和产品灵活性。多个工程团队从头开始构建新软件的各种互连组件；同时，设计团队创建了一个系统，允许出版物共享相同的代码，并保留其独特的外观和感觉。本演示探讨了这一变化的早期挑战，以及我们工作流程的演变如何简化了设计和开发流程。
