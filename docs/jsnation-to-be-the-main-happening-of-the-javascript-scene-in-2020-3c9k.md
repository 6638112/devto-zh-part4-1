# JSNation 将成为 2020 年 JavaScript 领域的主要事件

> 原文：<https://dev.to/gitnation/jsnation-to-be-the-main-happening-of-the-javascript-scene-in-2020-3c9k>

## JSNation 大会将于 2020 年 6 月推出全新版本

*如果你有幸参加 [JSNation 2019](https://jsnation.com) ，你会知道我们的活动会带来什么:众多知名演讲人、鼓舞人心的技术讲座、令人惊叹的场地、无数的交流机会、[进步艺术展](https://medium.com/@amsterdamjs/these-were-the-artists-showcased-at-jsnation-2019-52fbfa4d1743)，以及一场 JS 驱动的派对。然而，不仅仅是这样。即将到来的 JSNation 大会将是 2020 年 JavaScript 领域的重大事件，这是你不能错过的。*

对于那些没有机会参加去年版本的人来说，这里是你错过的:阿姆斯特丹市中心的一个令人惊叹的历史场馆，为期三天的富有洞察力的讲座和实践研讨会，分为 general 和 Node.js 轨道，来自世界各地的 500 多名与会者，由 Node.js，WebGL 和 WebAudioAPI 支持的 js 驱动的视听行为派对后计划。

《你不知道 JS》的著名作者 Kyle Simpson 发表的[扣人心弦的主题演讲在 Zuiderkerk 教堂引起了轰动，一个又一个强有力的演讲进一步推动了气氛:Henry Zhu 讨论了他的](https://www.youtube.com/watch?v=ZrKjgSfpppM)[艰苦的开源之旅](https://www.youtube.com/watch?v=fpQRi5ZIq1Q)，Tobias Koppers 解释了[即将在 Webpack](https://www.youtube.com/watch?v=p-MhcctQBlY) 中发生的变化，Andre Staltz [现场编码功能镜头](https://www.youtube.com/watch?v=5R3l2r1XxKI)，Mat Groves 展示了他用 PixiJS 和 WebGL 进行的[策划艺术创作，以及](https://www.youtube.com/watch?v=98Psw5-Qkrc)

还有许多其他有见地的谈话和对话，不能在一个两个段落的总结中涵盖；但是，相信我们，这很有趣。错过了也不用担心，因为 JSNation 又回来了！

## 看看我们的新场地

你准备好在阿姆斯特丹度过又一个美丽的夏天了吗？你准备好被演讲者、流行图书馆的核心维护者、书籍作者、著名播客和其他 JavaScript 名人无穷无尽的知识源泉所鼓舞了吗？

先前场地的神圣庄严和受人尊敬将让位于合作氛围，这是活动新场地不可或缺的一部分，即 [Kromhouthal](https://kromhouthal.com/en/) 。由于这个地方离市中心稍远，你将有机会乘坐渡轮往返于会场，从而欣赏 IJ 海峡两岸的迷人风景。

工业氛围，对 JavaScript 的深度关注，所有的谈话都是对新旧 JavaScript 技术的重新审视和改造，源源不断的富有成效的谈话，战略性地插入了出色的想法，JS 音乐和视觉艺术的炼金术实验室，更不用说免费的饮料和许多乐趣了——这是我们可以 100%保证的。

## 关于 GitNation

GitNation 是一个通过组织关注开源软件的活动来促进技术发展的基金会。我们组织有意义且有趣的 JavaScript 会议，将有才华的工程师、研究人员和重要库和技术的核心团队联系起来。

除了提供[单个会议门票](https://gitnation.org/#events)，该组织还出售 GitNation Multipass，提供对多个[远程 JavaScript 会议和研讨会](https://portal.gitnation.org/multipass)的折扣访问。
