# 为什么您应该立即进入编码领域

> 原文:[https://dev . to/misselliev/why-you-should-the-leap-into-coding-right-now-420h](https://dev.to/misselliev/why-you-should-take-the-leap-into-coding-right-now-420h)

这么说你已经考虑编码有一段时间了，但是一直犹豫不决。可能你觉得太难了。也许你认为你太老了。也许你认为你没有这个能力。

让我们一步一步地了解是什么阻碍了你。

### [](#blocker-1-its-too-hard)拦截器#1:太难了。

你知道这样做有什么好处吗？练习。然后再多练习一些。再多一点。最终，它会进入并变得有意义，就像生活中的一切一样，但首先你必须实际上 ***做*** 那件事。我建议你尽早和解的一件事是，你需要适应失败，并把它重新定义为成长的机会。舒适区不是你想去的地方，因为科技领域总是有新的东西。你会不断地学习，第一次没有做对也没关系。

### 拦截者 2:我需要知道所有的事情。

的确，你可能想知道即将到来的新技术，但这并不要求你知道所有的东西，并完美地编码所有的东西。我们都在学习。(见上文)

### [](#blocker-3-you-have-to-be-super-smart-to-code)拦截器#3:你必须超级聪明才能编码

我认为，更重要的是，要变得聪明，你需要坚持不懈和富有创造力。你需要创造习惯并培养它们。如果你刚开始写代码，保持一致的最好方法之一就是加入 twitter 上的#100DaysOfCode 对话，每天写一点代码。你不必每天都开发一个成熟的应用程序，但是你必须练习。(见上文)

### [](#blocker-4-im-xx-years-old-its-too-late-for-me)拦截者#4:我已经 XX 岁了/对我来说太晚了

在这一点上，我和你是同病相怜的。我也认为我错过了我的机会，因为我没有更早的投入进去。然而，一旦我决定全押，我发现唯一认为为时已晚的人是我。编码社区有各种年龄范围和各种背景，所以如果你不是来自某个地方或持有某个学位，不要气馁。

如果你能编写出需要的代码，如果你充满激情并坚持不懈，那就是你所需要的。你所要做的就是…没错，练习。一开始肯定会有一些阻力，因为你正在走出自己的舒适区，但从小处着手是没问题的。我支持你！你能做到的！

### [](#blocker-5-eli-but-i-dont-have-a-cs-degree)拦截者#5:伊莱，但我没有 CS 学位。

别担心，我的朋友！谢天谢地，互联网上有很多你可以自学的资源。你所需要的就是每天安排一点时间，然后投入进去。保持一致是关键。一个非常适合初学者的免费资源是 [FreeCodeCamp](//www.freecodecamp.com) ,或者如果你长期从事这项工作，一个编码训练营。

在你找到工作之前，有很多选择是不需要付费的，所以如果你认真对待这件事，你不会有任何损失，反而会有所收获。从我在训练营的个人经历来看，我可以推荐你去体验一下微信。

我推荐这两个选项，因为它们已经制定了课程，这让我想到了…

### 拦截器#6:好吧好吧，但是我该从哪里开始呢？/我不是应该学 X 而不是 Y 吗？

我知道跳上“无论什么是最新的”马车是非常诱人的，但是从基础开始总是最好的。我建议从 HTML、CSS 和 Javascript 开始，然后从那里开始构建到你想去的地方。把你的积木垒好，然后，你就可以更具体地了解你的旅程要带你去哪里。

### 拦截者#7:我很感兴趣，但我不认为我适合做这个。

嗯，你只有试了才会知道。正如我之前所说的，编码社区有各种不同背景的人，可能他们中的很多人一开始都有相同的想法。我鼓励你加入并找到与你相关的人。不用一个人流浪！我们都支持你！

这些只是我记得自己遇到的一些阻碍，但是总会有一百万个理由让你不应该做某事。

总有一个你应该这样做的理由，那个“为什么”不会让你完全抛弃这个想法。追求那个为什么。只要你决定对自己说“是”，你就会成功。你成功了！
