# Web 开发

> 原文：<https://dev.to/kevinsarav/web-development-40ol>

在这篇文章中，我将讨论到目前为止我所学到的东西。我一直在 Udemy 上网络开发课程。在课程中，我复习了 HTML 和 CSS。我学习了新的设计工具，比如 Bootstrap、Flex、Grid，以及使用在线模板来设计网站。

在这个课程中，我还复习了我的 JavaScript，并学习了如何使用它和 React 来制作可重用的组件和一个响应式网站。现在，我正在做最后一个项目，我正在建立一个人脸识别网站。你输入图像地址，它会显示图像和包含在图像中的任何人脸周围的方框。

我专注于 web 开发的主要原因是因为我将在项目的前端提供帮助。我的团队现在正在考虑探索其他更快完成的途径。我之前提到过，我们正在考虑建立一个求职网站。然而，经过大量讨论后，我们意识到这需要大量的研究、资源和时间来完成，所以我们会想出其他可以追求的想法。

我对学习 React 感到非常兴奋。我在网上看到很多工作需要 React 的经验，所以如果我用我学到的东西来做最终项目，并做自己的项目，我相信我会成为一名出色的申请人。我也希望重新开始学习应用程序开发。虽然我正在学习的东西可以让我不必过多地涉猎 HTML 和 CSS 来制作网站，但我仍然更喜欢应用程序开发。

在我的下一篇文章中，我希望谈谈我将从 Udemy 课程中完成的最终项目。我也希望谈谈在学习 app 开发方面的任何进步。
