# 每日挑战#75 -设置闹钟

> 原文:[https://dev . to/the practical dev/daily-challenge-75-set-alarm-2739](https://dev.to/thepracticaldev/daily-challenge-75-set-alarm-2739)

编写一个名为 setAlarm 的函数，它接收两个参数。第一个参数 employed 在您被雇用时为真，第二个参数 vacation 在您休假时为真。

如果您正在工作而不是在度假，该函数应该返回 true(因为在这些情况下您需要设置一个警报)。否则它应该返回 false。

*举例*:

`setalarm(true, true) -> false`
`setalarm(false, true) -> false`
`setalarm(false, false) -> false`
