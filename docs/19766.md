# 什么让你早上起床？

> 原文：<https://dev.to/taillogs/what-gets-you-out-of-bed-in-the-morning-3h6n>

有时候，起床开始新的一天是一个挑战。有时我工作压力很大，起床意味着更多压力的开始。有些日子我睡眠不足。有时候我觉得自己糟透了。但我最后总是会起床。

与我的狗和我的女朋友共度时光是我每天都可以期待的两件事。也有更多短暂的事情，比如一个令我兴奋的新项目，或者一天中晚些时候发生的我迫不及待的事件。

在特别艰难的日子里，我只记得我正在努力的所有潜在的美好经历。即使某一天并不突出或感觉不同寻常，它也是一个更大的过程的一部分，会带来长期的幸福。

所以我的问题是，是什么让你早上起床？
