# 关于计算的书籍

> 原文:[https://dev.to/peteratticusberg/books-about-computing-3n9b](https://dev.to/peteratticusberg/books-about-computing-3n9b)

*   代码解释了电报是如何演变成计算机的
*   [系统设计初级读本](https://github.com/donnemartin/system-design-primer)解释了从以不同能力运行的机器网络中组合出单个应用程序的设计模式
*   Unix 操作系统的设计解释了 Unix 如何提供软件到硬件的接口
*   [网络简介](https://www.amazon.com/Introduction-Networking-How-Internet-Works-ebook/dp/B00YD7M92S/ref=sr_1_3?crid=W6HCN4KHWVC&keywords=introduction+to+networking&qid=1561161870&s=books&sprefix=introduction+to+networking%2Cstripbooks%2C191&sr=1-3)解释了数据是如何编码并通过互联网发送的
*   [破解编码面试](https://www.amazon.com/Cracking-Coding-Interview-Programming-Questions/dp/0984782850/ref=sr_1_3?crid=1BPMZ8RWH3S70&keywords=cracking+the+coding+interview&qid=1561161894&s=books&sprefix=cracking+the+coding+interview%2Cstripbooks%2C160&sr=1-3)解释了构成算法的数据结构
*   [掌握比特币](https://www.amazon.com/Mastering-Bitcoin-Programming-Open-Blockchain/dp/1491954388/ref=sr_1_1_sspa?keywords=mastering+bitcoin&qid=1561161993&s=gateway&sr=8-1-spons&psc=1)解释了一个公共供电的对等网络如何使用一种协议来持续可靠地达成共识
*   [沉默在线](https://www.amazon.com/Silence-Wire-Passive-Reconnaissance-Indirect-ebook/dp/B008FRNHVY/ref=sr_1_1?crid=2AQUB3YLRTCK8&keywords=silence+on+the+wire&qid=1561162014&s=gateway&sprefix=silence+on+the+wire%2Caps%2C172&sr=8-1)解释了广泛使用的工具、库和硬件组件如何容易被利用
