**JavaScript:**
JavaScript 是跨平台或 web 平台中著名且使用最多的脚本语言之一。它很容易学习，也使用 html 和网络。

几周前，我在研究 JavaScript 的**‘getElementsByName()’&‘getElementsByClassName()’**方法，在**的社交媒体平台如:脸书和 LinkedIn 上实现后，我发现这两个函数有些特别。这两种方法帮助我在脸书接受了好友请求，在 LinkedIn 接受了邀请。**
