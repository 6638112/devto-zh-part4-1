# 为每个人提供一个交互式的、实用的网页可访问性介绍

> 原文:[https://dev . to/chinchang/an-interactive-and-practical-introduction-to-web-accessibility-22o 1](https://dev.to/chinchang/an-interactive-and-practical-introduction-to-web-accessibility-22o1)

自从我理解了可访问性的重要性，我总是努力有意识地分享我不断学到的东西。这样每个人在制作网站的时候都可以开始关心至少基本的可访问性。

没有什么比真正面对现实更能让你接近现实。这是我最近尝试传播网络无障碍意识的前提。

呈现👉🏼[introtoaccessibility.com](https://introtoaccessibility.com/)

这个项目是一系列章节，以测验的形式。每一章都给你看一些东西或者让你做一些事情，然后问你一个问题。这些章节基本上是不可用的用户体验的极端情况，其设计使得即使是有能力的用户也无法访问它们。

每一章都让你体验一个特殊的可访问性问题。然后，它会告诉您该问题如何影响互联网上的各种用户，为什么会影响他们，以及您可以做些什么来防止它。

我对这个项目的意图是，它可以成为向任何人和每个使用网站的人介绍网页可访问性的第一步。不管他们是开发商、设计师还是营销人员！我希望我能引起我们每个人的共鸣，让网络成为每个人更好的地方。
