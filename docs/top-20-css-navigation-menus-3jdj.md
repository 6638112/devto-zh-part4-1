# 前 20 个 CSS 导航菜单🚀

> 原文：<https://dev.to/webdeasy/top-20-css-navigation-menus-3jdj>

一个好的用户流量对一个成功的网站有着巨大的贡献。因此，内容、定位和外观都应该经过深思熟虑。在这里你会找到一些你自己的导航菜单的灵感！

在[前 20 个 CSS 按钮](https://webdeasy.de/top-css-buttons-en/)被你很好的接受并且我得到了很多积极的反馈之后，这里是继续:**前 20 个 CSS 导航菜单！**

有时候这个列表中会有“普通”菜单，原因很简单:可用性第一，外观第二。即使是“普通”的菜单栏也可以看起来非常别致。有了正确的效果和动画，他们也成为一个真正的眼睛捕手。

这同样适用于移动表示，也称为“汉堡包”。桌面和移动菜单都可用。

所有导航菜单都发布在 [codepen.io](https://codepen.io/) 上，是**不是**我的。因此作者总是被链接。我们走吧！
