嘿大家好！我一直打算给 [Fish](https://fishshell.com) 一个严肃的评价，作为我的默认 shell，因为它似乎比 Bash 提供了足够多有意义的生产力增强，几乎没有设置(例如命令自动完成)。也就是说，我很好奇其他人在用什么，他们的确切设置是什么，以及他们最喜欢它的什么。特别是，您发现哪些事情实际上对您的生产力和一般开发人员的乐趣有重大影响(而不仅仅是“酷”)？

你主要使用 Bash、 [Zsh](https://Zsh.org) 、Fish、PowerShell 还是其他什么？你使用框架/插件管理器吗，比如 [Oh My Zsh](https://ohmyz.sh/) 或者 [Fisher](https://github.com/jorgebucaran/fisher) ？有没有什么特别的建议是你觉得每个人都应该知道的，以便充分利用他们的外壳？

请随意分享关于您的个人 shell 设置的任何和所有细节，以便我们都可以获得尝试新事物的灵感，并可能提升我们的日常开发工作流！此外，如果您包含您的设置和最喜欢的 shell 功能的截屏/gif，还有加分🚀
