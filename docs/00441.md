## [](#what-is-exercismio)什么是 exercism.io

[关于 exercism.io 的第](https://exercism.io/about)页说:

> Exercism 是一个在线平台，旨在通过实践和指导帮助您提高编码技能。

但是 exercism 到底是什么，它与其他几十个编码练习平台有什么不同？

对我来说，作为一个学习者，这是我迄今为止看到的唯一一个你可以学习任何语言的习惯用法的平台。如果你是一个 python 开发者，看过这样的代码，你会有这样的想法:

```
my_list = ['dog', 'cat', 'hamster']
for i in range(len(my_list)):
    animal = my_list[i]
    print(animal) 
```

按照应该使用的方式使用语言是值得的。而 Exercism 通过建立一个导师社区来实现这一点，这些导师会就你的解决方案的可读性、性能和习惯性向你提供反馈。这不是让代码通过测试。

所以基本上 Exercism 就像编码学习平台中的 Archlinux。

而且我甚至没有提到它不是基于 web 的，他们有 CLI，所以你可以使用你最喜欢的 IDE 或 vim。

一切都是为了学习，但为什么要在那里指导呢？

## [](#reasons-to-mentor)给导师的理由

### [](#1-read-more-code-from-learners)1。阅读更多来自学员的代码

为什么要看别人的人码？因为这是学习编程最快的方法之一。其他人是不同的，你可以吸收他们的观点，他们的错误，他们的想法。跳到 Linux 内核源代码可能会让人不知所措，所以最好从小处着手。

作为 Exercism 的导师，你可以比较同一个问题的多种解决方案，看看人们犯了什么错误，他们使用什么风格，它如何适应特定的语言。

这就是你变得流利的方法。

### [](#2-practice-your-communication-styles)2。练习你的沟通风格

所以程序员不仅读和写代码，他们还和其他人交流。在我看来，这实际上是这份工作的重要组成部分之一。好主意可能会因为沟通不当而夭折。bug 被引入，人们称技术行业有毒，很多事情发生是因为开发人员没有掌握人类语言的技能。

这是一个很好的成长机会，在这里你可以学习和练习如何向其他开发者表达你的想法。你会从他们那里得到反馈，甚至导师也有懈怠。这就是你如何脱颖而出的方法。**我不是说你会成为 10 倍的开发者，但你会变得更好**

### [](#3-sandboxed-discussions)3。沙盒讨论

如果你喜欢在酒吧或咖啡机旁谈论编程，但不是每个人都有你的热情，这不是问题。

关于 Exercism 的大多数解决方案讨论都是关于样式、约定、微优化和可读性。你没有截止日期、业务优先级等。享受纯编程就好。

### [](#4-make-the-world-a-better-place)4。让世界变得更美好

你对世界上有多少优秀的开发人员负责。
就这么简单:

更多优秀的开发人员= >更好的软件= >更少的错误和压力。

谁知道呢，也许你帮助的某个学生会创建另一个开源框架或项目，这将产生数以千计的工作场所。

### [](#5-pay-it-forward)5。让爱传出去

有人在 StackOverflow、Github 或这个随机论坛上帮助你的频率如何？你也可以做到这一点，这不会像投身于一个大型开源项目或在 StackOverflow 上竞争那样困难。Exercism 是回馈社区的好地方。

有人可能会说

> 不，Python/Javascript/Java 有足够多的导师，不需要我的帮助

但那不是真的，甚至不是关于其他导师的。这是关于你关心你的手艺。
