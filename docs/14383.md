# 当人们说“编码很难”时，这对你意味着什么？

> 原文：<https://dev.to/treddson/what-does-it-mean-to-you-when-people-say-coding-is-hard-20al>

在 Twitter 上，在这个网站上，在 bootcamps 上，以及几乎所有编程存在的地方，你都会发现那些说“编码很难”的人。请允许我补充两点——编码很难！我想对于不同的人来说，因为不同的原因，这是很难的，但肯定有我们都同意是困难的领域。

对我们人类来说，说 10100100 语言当然不是自然的。是的，我们写的代码从二进制中抽象出来了很多，但是仍然需要我们的大脑去思考！

经过无数小时的练习，理解语言和语法变得更加容易，但对我自己来说，编程中的逻辑最具挑战性。当然，逻辑是构成编程的基础。但和大多数事情一样，我不断提醒自己，这是一块必须坚持锻炼的肌肉。

当我听到有人说“编码很难”时，我首先想到的是逻辑。逻辑之后想到的是**创意**。

我的意思是，*思考*要建造的东西。从空白画布开始。网上有很多资源可以帮助你构思项目，但是脱离😖教程地狱🤯开始完全原创的东西可能是初学者最大的挑战。

我知道，作为一名新的甚至是经验丰富的开发人员，我并不孤单。我很好奇，当你听到有人说“编程很难”时，你会想到什么？
