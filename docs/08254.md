# 伦敦大学一学期的远程学习

> 原文:[https://dev . to/tux boy/a-学期远程学习-w-伦敦大学-31e3](https://dev.to/tuxboy/a-semester-of-distance-learning-w-university-of-london-31e3)

2019 年 4 月，我报名参加了由伦敦*大学*提供的计算机科学学士课程；这篇文章总结了我第一学期的经历。尽管人们对学位的必要性存在争议，但我还是选择谨慎行事，拿到了一个学位。

## [](#background)背景

我在南北大学(一所实体大学)读本科第一年就开始全职工作了。我的雇主愿意让我在工作时间上课。然而，我不能每学期选修很多课程。全职工作就是全职。

结果，当我把自己定位为组织中的高级资源之一时，继续我有严格出勤要求的大学课程变得不可能了。我承担了更多的责任，更艰巨的挑战。我努力在大学考试中取得好成绩，成绩一落千丈。有一段时间，我远远落后于我的同龄人，我无法在 3 年内完成一半的课程要求。全职工作和在米尔布尔(达卡)的交通中战斗意味着，我从早上 7 点到晚上 8 点都呆在户外。筋疲力尽。一直以来。

我不能为了上学而停止工作。我家负担不起。孟加拉国的学生贷款不像美国，所以那也不是一个选项。最重要的是，我必须工作！全职工作 3 年后离开这个行业对我来说是不切实际的。

### [](#choosing-the-university)选择大学

在这些在线课程中，伦敦大学因几个原因而脱颖而出，

1.  它在孟加拉国和我想去工作的国家都很有名并被接受。
2.  与同类大学相比，费用明显较低。
3.  Coursera 是 VLE(虚拟学习环境)的合作伙伴。所以我知道我会对内容有很好的体验。
4.  我的两位同事已经在 UoL 大学学习过，他们的推荐增强了我的信心。

这个项目是由伦敦大学的金史密斯学院设计的。这是一个 3 年的项目(如果是全日制学习的话)。你最多需要 6 年时间来完成这个项目。

我写了一篇关于[为什么选择 UoL](https://adhikary.net/2019/07/06/uol-bsc-cs-why/) 的帖子，详情请看！

测验有两种类型:练习测验(不评分)和评分测验。我可以在网上参加测验。分级测验通常有时间限制。在课程进行到一半时，可能会有一次期中测验，内容涵盖到目前为止所教授的主题。

### [](#essential-book-readings)必读书籍

在课程中，经常推荐基本的书籍阅读。这些书可以在大学提供的虚拟图书馆里找到。这些可以在电脑、平板电脑上阅读，甚至可以打印出来(合法)。

### [](#assignments-and-projects)作业和项目

几乎所有的课程都有编码作业或书面作业。每份作业都用现代剽窃检查器检查。所以任何不良意图都应该受到检查！作业通常占课程分数的很大一部分(因课程而异)

### [](#deadlines)截止日期

有期限的！某些任务有最后期限。使用提醒/日历/待办事项系统来记录课程和作业是明智的。

### [](#proctored-exams)监考

有些课程(大约一半)需要监考。你去当地的考试中心参加考试条件下的笔试。这些中心遍布世界各地。在印度地区，考试由英国文化协会主办；在新加坡，由 RELC。

## [](#first-courses)第一道菜

我在大学里重新开始。所以我的第一门课是，

1.  编程导论 I
2.  数字数学
3.  计算机是如何工作的
4.  Web 开发

### [](#introduction-to-programming-i)编程入门 I

这基本上是对 JavaScript 和 p5.js 编程的介绍。在整个课程中，我们必须构建一个 HTML5 游戏并完成一些编程任务来解决刑事案件！本课程完全在线，不需要监考笔试。

### [](#numerical-mathematics)数值数学

这是我上过的最棒的数学课。本课程教授隐写术、加密和其他 CS 相关主题作为课程内容和作业。它非常关注计算机科学在数学上的应用。这让它变得非常有趣。它涵盖了统计学、概率、线性代数和微积分等。本课程 50%的分数来自在线评估，其余来自监考笔试。

### [](#how-computers-work)计算机如何工作

这门课程侧重于对计算机科学的基本理解。主题涵盖了网络、网络、机器内部、抽象、机器学习和其他计算机科学概念。这主要是一门概念性的课程。除了在线评估之外，还有监考笔试。

### [](#web-development)网页开发

这门课也是一门相当基础的课程。其中 100%的分数来自于项目！30%来自小组项目，70%来自个人项目。这个课程实际上教会了我很多关于语义和可访问性的知识。通常被开发者忽略的两个话题。这不需要监考。

更多关于课程的信息可以在网站上找到。

## [](#flexibility-devil)柔韧性魔鬼

这是必须讨论的。引导我选择远程教育学位的灵活性是我最大的敌人。让我解释一下。

因为我几乎有无限的时间来完成我的课程。我只是拖延时间。我把我的任务一推再推，一推再推。很快，学期结束了。我不得不处理我跳过的所有东西！

[![TODO List](../Images/0cd25ab08bb21f90a5f6b193ee460285.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--wy68BB-3--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/d65ufdk0kar75lkswbbi.png)

作为一个解决方案，我必须制定一个完整的学习计划，并严格遵守计划。谢天谢地，我现在差不多完成了恢复学业。我用一个 todo 应用程序来计划每一天。平衡我的学习作为每天的习惯。

我记得一位海豹突击队员的忠告:纪律就是自由。老实说，如果灵活性没有得到正确处理，它很快就会变成无政府状态。
