# 我总是在 Windows 上安装的 6 个工具

> 原文：<https://dev.to/0xkoji/6-tools-i-always-install-on-windows-442a>

我贴这个是因为收到了来自 [@tetri](https://dev.to/tetri) 的评论

说实话，我并不是真的在用 Windows 的机器。有时会用到。

如果你知道什么有用的，请告诉我！！！

### 1。Visual Studio 代码

我尝试了一些用于 Windows 的编辑器/IDE。最终，到达 vscode。

[https://code.visualstudio.com/download](https://code.visualstudio.com/download)

### 2。Cmder

Windows 机器有命令提示符，但是不允许我使用基本的 linux/unix 命令，所以不要用。

[https://cmder.net/](https://cmder.net/)

### 3。可视化工作室

我不写 c#和用。但我需要 Visual Studio for openFrameworks 和构建代码。

[https://visualstudio.microsoft.com/](https://visualstudio.microsoft.com/)

### 4。VirtualBox(虚拟方块)

一般不喜欢点击按钮，避免多次点击即可。

[https://www.virtualbox.org/](https://www.virtualbox.org/)

### 5。7-Zip

自从`7z`以来，我一直希望 Mac 有这个可以节省我的存储空间。

[https://www.7-zip.org/](https://www.7-zip.org/)

### 6。CCleaner

尽可能保持我的机器干净，但是不确定这个软件真的有帮助。然而，我曾经搞砸了注册表键，度过了一段艰难的时光。所以不要碰它们，哈哈。

[https://www.ccleaner.com/](https://www.ccleaner.com/)
