## [](#learning-through-doing)边做边学

我强烈主张通过做来学习。我看不出在使用之前学习*所有看起来有点复杂的东西(比如 flexbox)有什么意义。*

当你一开始只想创建几个列时，学习横轴和`flex-basis`有什么意义呢？

因此，本课程是关于在实践中学习。我之所以使用这种方法，是因为当你完成课程后，你不仅会对课程中所涵盖的内容感到舒适，而且会对 CSS 有实际的了解。

你将从这门课程中熟练掌握 CSS。我们不能面面俱到，但是**你通过学习建立的基础将会在你继续学习和建立网站的过程中为你的成功奠定基础**。

## [](#a-more-detailed-breakdown-of-the-modules)更详细的细分模块

### [](#module-1-css-fundamentals)模块 1 - CSS 基础知识

模块 1 从我的免费 [HTML 和 CSS 速成班](https://scrimba.com/g/ghtmlcss)停止的地方继续。因此，它从一个非常基础的水平开始，但很快齿轮！

在本模块中，我们将探索一些基础知识，如理解盒子模型，以及确保我们正确理解继承和特殊性等许多人(甚至是有经验的开发人员)感到沮丧的事情。

尽管重点是在基础上，我们添加了谷歌字体和一些设计技巧和窍门，开始让一些简单的布局看起来不错。

在我们深入更复杂的材料之前，这是一个重要的模块，确保我们都理解 CSS 的基本概念。

### [](#module-2-starting-to-think-responsively)模块 2——开始有针对性地思考

这个模块是一个很大的模块，它是课程的核心。它介绍了单元、新单元、媒体查询，并开始关注 flexbox。

它还深入探讨了如何在编写一行代码之前分析布局。这是一项基本技能，也是许多*人苦苦挣扎的事情。*

除了提前计划，我们还开始探索如何命名我们的类，以避免在项目规模扩大时遇到问题。

本模块最后构建了一个三页网站，并确保它完全响应。

[![screenshots of the different pages at small and large screens](../Images/33b74ef06807c45e293eb5ccaf3b79f4.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--CuYfwB-X--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/vilgqqpei6c0f4sr6w3k.jpg)

### [](#module-3-stepping-up-our-style)模块 3 -提升我们的风格

这一篇是关于看一看 CSS 更微妙的一面。

它从继续上一个项目开始，以帮助它变得更好，然后进入另一个设计，学习一些额外的东西，如背景图像，渐变，视口单位，一些排版的细节，以及创建和设计一个基本的形式。

**这是关于在更深层次上理解 CSS，并开始释放它的一些令人敬畏的潜力**。

这个模块以构建一个简单的、响应迅速的启动页面结束，但是它迫使我们继续提升我们对 CSS 的了解。

### [](#module-4-taking-flexbox-to-the-next-level)模块 4 -将 flexbox 带到下一个级别

**模块 4** 换个节奏。

至此，我们已经创造了很多。我们已经开始熟悉它的工作方式，所以是时候深入了解 flexbox(CSS 中的两个布局工具之一)了。

我们来看看它的细节，比如`flex-basis`、`align-self`，flexbox 内部的自动边距是如何变化的(等等！).

一旦你完成了这个模块，你将学会如何真正正确地控制页面上的每一个元素。

### [](#module-5-css-grid-the-ultimate-layout-tool)模块 5 - CSS 网格:终极布局工具

然后是时候进入 CSS Grid 了，这是 CSS 最新的布局工具。这是 flexbox 的一大进步。它允许我们构建更复杂的布局，同时简化我们的 HTML！

这个模块探索了它是如何工作的，以及像`auto-fit`、`minmax()`和网格区域这样的东西是如何使创建响应性网站*、*、*变得更加容易的！*

我们将研究如何用 grid 重新创建一些旧的布局，并看看几个简单的页面设计，以及 grid 在查看大图和组件级别时如何变得非常有用。

### [](#module-6-taking-it-to-the-next-level)模块 6 -更上一层楼

模块 6 是最后一个项目，我们把所有东西放在一起，建立一个 3 页的网站，使用我们到目前为止学到的所有东西，包括 CSS Grid。

在此期间，我们将通过探索新的事物将事情推进到下一个层次，比如伪元素、一些巧妙的阴影技巧，以及对 JavaScript 的一个非常小的探索。
