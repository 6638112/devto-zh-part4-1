## 典型的应用技术堆栈:看起来如何？

简而言之，技术栈是软件工具和编程语言的组合，用于将网络/移动应用程序带入生活。事实上，web 开发包括两个方面，如后端(服务器端)和前端(客户端)。此外，后端包括应用程序(和支持它的后端编程语言)、数据库和服务器本身。

各种技术彼此高度依赖。这可能是因为应用程序的每一层都建立在下一层之上，形成了一个堆栈。要开发一个 web 应用程序，你需要选择**你将要使用的服务器、数据库、编程语言、框架和前端工具**。

如果你仍然对术语感到困惑，让我们来定义它们。因此，**编程语言**是编写程序的语言，**框架**是编程语言的运行环境。换句话说，框架是允许你开发应用程序的程序集合。**一个数据库**将允许收集存储和组织的信息，以便可以轻松访问、管理和更新。web 服务器是一种“功能强大的计算机”,它根据请求组成并分发网页。

## [](#frontend-programming)前端编程

前端或客户端包括为网络应用程序构建网页和用户界面。它通常为你在使用网站或网络应用程序时在屏幕上看到的一切实现结构、设计、行为、动画。客户端是一个视觉部分，帮助用户看到应用程序的这一部分并与之交互。由于许多编程工具，上述交互是可能的。面向客户端的 web 应用程序通常使用 JavaScript、HTML 和 CSS 的组合来构建。

HTML(超文本标记语言)是一种告诉浏览器如何显示网页内容的编程语言。

**CSS(层叠样式表)**是一种样式表语言，描述用 HTML 编写的文档的外观和格式。换句话说，CSS 允许改变页面元素的外观，比如字体、文本、图片、表格等等..HTML 是第一位的，它为你的页面奠定了一定的基础，而 CSS 则紧随其后，创建页面的布局。

JavaScript (JS) 是另一种前端技术，它使网页具有动态性和交互性。JS 提供简单和复杂的网页动画，这极大地促进了积极的用户体验。对于更容易的 web 开发，有 **JS 库**(如 jQuery、React.js 和 Zepto.js)以及**框架**(如 Angular、Vue 和 Backbone)。库和框架之间的区别在于，框架是应用程序的一种“框架”,它定义了应用程序的特性，而库是类定义的集合，它执行特定的操作来填充应用程序的“框架”。

在 2muchcoffee，为了开发复杂的多层产品，我们使用 **TypeScript** ，作为前端工具包的一部分。TypeScript 允许实现编程的动态方法和适当的代码结构。

### [](#frontend-technologies)前端技术

前端技术是包含预先编写的标准化代码的包，这些代码以文件和文件夹的形式组织。它们为开发人员提供了构建功能代码的基础，以及更改最终设计的能力。因此，它帮助开发人员节省时间，因为他们不需要从头开始编写每一行代码。

现在前端技术市场分化很大。我们想提一下最常用的技术(jQuery、Backbone.js、Ember.js ),并关注一下在开发人员中非常流行的前三项技术(Angular、React、Vue)。

因此，前端技术包括:

*   **jQuery** 是一个 JS 库，旨在使网页活跃，所以变化可能会动态反映。**优步，Reddit，Hootsuite，OpenTable** 都在用 jQuery，原因是跨浏览器的可能性，轻量级和功能丰富的元素。

*   另一个 JS 库是 **Backbone.js** ，它被设计用来构建 JavaScript 代码以简化其操作。它用于创建单页应用程序，其中所有的通信都发生在该页面内，而不涉及整页刷新，例如像 **Gmail** 。当客户端可以轻松处理任何更新，并且 web 应用程序的大小非常重要时，它适用于任何复杂的项目。

*   一个开源的 JS web 框架 **Ember.js** 提供了一个完整的解决方案，其中包含数据管理和应用流程。这个框架消除了您或开发人员通常必须手工完成的所有繁重工作，因此您可以专注于构建杀手级功能和 UI。Ember 在许多现代网站上都有练习，包括**苹果音乐，雅虎！、LinkedIn、TinderBox、网飞、Groupon、Discourse** 。

然而，有**前 3 名的 JavaScript 技术**由于其与众不同的特性而成为领导者，如 Angular、React 和 Vue。

**角度**

Angular 最初是由 Google 开发的，后来成为了一个包罗万象的开发解决方案。Angular 更适合构建动态 web 应用和大规模功能丰富的应用。

在 2muchcoffee，我们专注于棱角分明的网页开发。它包含了开发者所需要的一切:由于模块系统，框架是结构化的和可扩展的，它为整个开发过程和 JavaScript 代码的结构提供了高度的简化。你可以在 <u>[Angular 官网](https://angular.io/)</u> 找到开发教程、基础和技巧。然而，我们的开发团队基于个人经验产生了 Angular 的最佳实践。它收集了 Angular 团队的建议、常用的最佳实践以及长期的技术专业知识，可以根据 Angular technologies 的规范开发出一流的产品。

事实上，当涉及到复杂项目的开发时，组件方法、清晰的流程和延迟加载会有所不同。我们团队成为开源软件 <u>[Ngx-restangular](https://github.com/2muchcoffeecom/ngx-restangular/stargazers)</u> 的作者和维护者。它负责 API 和复杂的前端 web 应用程序之间的通信。此外，我们相信，通过对开源做出贡献，我们在开发过程中插入了一种新的方法，因此每个人都有机会扩展他们的专业水平以及分享他们的知识。

您应该使用 Angular 的原因是它的可定制性、灵活性、易测试性和电荷依赖性。这就是为什么当你想开发**视频流应用、用户评论或社交应用、旅游或天气应用、用户生成的内容门户或电子商务和移动商务市场**时，该框架是最佳选择。Angular 在许多现代网站上都有实践，包括 **Youtube、Vevo、Freelancer、Istockphoto、Weather、technoSky Store** 。通过使用 Angular framework，上述网站受益于大量的各种功能，有机会创建更快、响应更快的应用程序。

此外，基于特定的前端技术，您可以构建移动应用程序来扩展您的业务。使用基于 Angular 的 Ionic 框架，你可以开发一个跨平台的移动应用。请注意**跨平台开发**你可以有一个代码库，能够为多个平台创建移动应用，确切地说是 iOS 和 Android。

**反应**

React 是由脸书创建和开发的，致力于构建有效的用户界面。使用 React，您可以构建包含动态内容的应用程序，其中没有太多格式。脸书、Instagram、纽约时报、WhatsApp、Codecademy、PayPal 等公司都是在 React 上发展起来的。

我们的开发团队使用 React 来构建处理大量信息的快速应用程序。使用 React 的主要目的是其简单、快速和可伸缩的特性。此外，通过社区卷，未来的维护和定制变得更加容易。React 最适合构建复杂的 web 应用程序，例如电子商务平台、市场等。

在 2muchcoffee，我们相信对开源的贡献是团队技术能力的最好体现。由于这个事实，用开源贡献提供一些新的东西意味着投资将在数千个前端项目中使用的东西。此外，我们的 React 开源项目 Redux-xs 证明了这一点。它集中在许多开发者发现跟随 Redux-pattern 是困难的想法上。所以 Redux-xs 项目将有助于使这项工作更容易理解。此外，所有 redux 插件都可以与之兼容。

要开发跨平台移动应用程序，您可以使用 React Native。它使用与 React 相同的设计，这意味着您可以编写丰富的移动用户界面。成千上万的公司在他们的应用程序中使用 React Native，例如 Instagram、脸书、Skype、Pinterest 等..

Vue.js 是另一个轻量级的快速框架。它易于学习和集成，这使得它在开发人员中非常流行，以建立用户界面。如果你想构建一个易于修复的应用，Vue.js 是最好的选择。如果你计划在未来扩展你的团队，Vue 将是合适的，因为如果你的团队已经被改变，它是最容易开始的框架之一。由于其多功能性，该框架允许您从一个简单的应用程序开始，并随着项目的扩展快速添加额外的功能和模块。

WizzAir、Grammarly、EuroNews、Adobe 等公司委托 Vue.js 开发应用程序。可以说，Vue.js 是一个主要的解决方案，如果你想建立一个单页面的应用程序，但你不确定将来是否需要添加一些额外的功能。事情是，作为一个初创公司，你需要快速开发具有基本功能的 MVP，为此，Vue.js 是最好的框架。前面提到的使用 Vue.js 的公司都遵循着相同的创业发展模式——启动具有基本特性的 MVP，然后扩展其功能。

## [](#backend-programming)后端编程

服务器端编程或后端对用户来说是不可见的，但是，它是应用程序开发的核心元素，因为它加强了客户端。强大的后端相当于建筑的基础。没有一个好的和强大的基础，任何建立在基础之上的东西都将无法执行。

主要的挑战是为进一步的开发选择正确的后端技术。后端编程语言旨在创建网站和应用程序的逻辑，而框架提供了多种工具，用于更简单和快速的编码实践。每种语言都有自己的强项。在构建应用程序时，有许多事情需要考虑，选择合适的语言来开发应用程序是一项复杂的任务。在选择时会考虑语言的利弊，最重要的是，性能、可伸缩性、成本、支持和可维护性是主要考虑因素。

最流行的后端语言(以及括号中包含的框架):

**Ruby (Ruby on Rails)**

Ruby on Rails 建立在 Ruby 编程语言的基础上，为构建 web 应用程序提供了一个强大的开发生态系统，非常适合于交易平台、数据服务、社交网站、社交网络、全球在线市场等。该框架的工具包括数据库集成和控制器，这使得端到端的 web 开发变得更加容易。通常用于开发桌面应用和 web 应用。运行在 Ruby on Rails 上的热门应用包括 **Airbnb、GitHub、彭博、Fiverr、Groupon** 。

**Python (Django，Flask，Pylons)**

Python 可用于开发不同的应用程序，如 web 应用程序、基于图形用户界面的应用程序、软件开发应用程序、科学和数字应用程序、网络编程、游戏和 3D 应用程序以及其他商业应用程序。在现代 IT 行业，Python 被广泛用于**数据科学和机器学习**。Python 的主要优势之一是每分钟能够处理数千个请求。此外，由于这是一种面向对象的语言，它能够有效地生成代码，这意味着软件可以操纵自己的功能和可伸缩性。在 Python 上运行的热门应用包括 **Instagram、Spotify、亚马逊、脸书**。

**PHP (Laravel，CodeIgniter，Symfony，Yii，Zend，Phalcon，CakePHP，FuelPHP)**

PHP 有助于提供统一的变量语法、敏感的上下文和抽象语法，这使得语言一致，尽管在标准库问题上存在问题。像**维基百科、Tumblr、Slack、Etsy** 这样的公司在应用程序开发过程中信任 PHP。PHP 将是创建博客或软件即服务应用的正确解决方案。

**Node.js(快递。JS，Koa。JS，Nest。JS，Loopback.io)**

Node.js 是一个开发服务器端 web 应用的开源平台。它的初衷是让实时网站的推送功能变得更容易。它还包括本机模块，可以管理从文件到计算资源和安全的一切。

在 2muchcoffee，我们更喜欢使用 Node.js 作为实时应用程序的伟大解决方案。由于其速度、并发性和密集的数据交换，许多大公司，如**网飞、Linkedin、沃尔玛、Trello、优步、PayPal、Medium、易贝**受益于这项技术。事实上，Node.js 的功能为应用程序提供了一致性和流畅的性能。它适用于**流媒体服务、类似优步的应用、在线电子商务平台、发布平台**。

后端编程的机制相当简单:web 服务器是后端的一部分，它接受来自浏览器的请求，按照一定的逻辑处理这些请求，如果需要的话转向数据库，并发回相关内容。

**一个数据库**将帮助你存储应用程序的数据。数据库通常包括数据记录或文件的集合。例如，在电子商务开发中，这些记录或文件将是销售交易、产品目录和库存以及客户资料。

在众多数据库中，**我们会提到数据库 2muchcoffee 通常与**配合使用。对于 web 开发，我们通常使用以下数据库:

*   MySQL -提供高性能和可伸缩性，这对于严重依赖多行交易的应用程序来说至关重要，例如典型的**银行应用程序**。

*   **PostgreSQL** -该数据库适用于**金融、制造、研究和科学项目**。它具有强大的分析能力，并拥有强大的 SQL 引擎，可以轻松流畅地处理大量数据。

*   MongoDB -主要的功能之一，比如基于文档的数据模型。当涉及到**计算距离**和计算客户的地理空间信息时，这是一个很好的选择，因为这个数据库有特定的**地理空间特征**。它还是电子商务、活动和游戏应用的技术堆栈的一部分。

**网络服务器**允许服务器通过互联网传送内容。原则是所有的网站必须托管在一个网络服务器上。当你在网上浏览互联网时，你打开网页。当您导航到某个网页时，web 服务器会在后台查找您所请求的页面。它将组成网页的文件传送给您，这样您就可以查看网页并与之交互。有两种主要的 web 服务器:

*   Apache -是最受欢迎的网络服务器之一，可以让你流畅地运行一个安全的网站。这是希望在网上出现的小型企业最常选择的方式。使用 Apache 的公司有 **Slack、HotelTonight、Etsy、Linkedin、Comcast、Yahoo！**

*   Nginx -帮助交付具有高性能、可靠性、安全性和规模的网站和应用。该公司提供屡获殊荣的综合应用交付平台，在全球超过 3 亿个站点上使用。世界各地的公司都依赖 NGINX，通过高级负载平衡、web 和移动加速、安全控制、应用监控和管理等功能来确保完美的数字体验。使用 Nginx 的公司包括 **Airbnb、优步科技、Spotify、Instagram、Medium、Dropbox** 。
