# OK 平台

> 原文：<https://dev.to/jondouglas/ok-plateau-4dmo>

我们大多数人每天至少花一个小时练习，但我们并没有注意到多少进步。

20 世纪 60 年代，心理学家发现我们会经历三个阶段。

认知阶段,在这个阶段，我们在找出任务的同时，发现新的策略来做得更好，并犯下大量的错误。这是我们关注我们正在做的事情的地方。

在**关联阶段**，我们犯的错误越来越少，慢慢变得越来越好。

自主阶段,我们打开自动驾驶仪，把技能转移到我们的脑后，不一定要集中注意力。

当我们觉得自己足够擅长一项任务时，我们就简单地打开自动驾驶仪，开始我们的快乐生活。因此，我们都达到了 OK 平台期。

心理学家认为，无论是精神上的还是身体上的障碍，都是我们无法克服的。例如，过去没有人能在 4 分钟内跑完一英里。

然后一个人跑了 4 分钟一英里，最终有人跑快了一秒。现在，每个认为自己是合格跑步者的人都可以跑 4 分钟一英里。就好像闸门降低了这个障碍，简单地通过有人直接突破。

是什么让那些对自己的工作感到惊奇的专家与众不同？有什么事情你可以学着做得更好并变得令人惊奇吗？答案是肯定的。在为什么他们的实践养生法给了他们想要的专业知识的领域中，有一套被专家使用的通用原则。

如果你想在某件事上做得更好，当你处于自主阶段时，你无法做到。发挥专长太难了。使用这些策略来远离这个阶段。

*   他们在自己的舒适区之外运作，并研究自己的失败。

例如，如果你想在健身房举更多的东西，试着比平时多举 20%,看看你在哪里失败了。你会花更多的时间去了解你失败的地方，以及如何改进。最好的实践者尝试真正困难的事情。刻意练习自然是很难的。

*   人们也会步那些比他们更有能力的人的后尘。

不要想你投入了多少练习，想想你做了多少学习来缩小你和比你更有能力的人之间的差距。

*   专家渴望即时和持续的反馈，并从中受益。

直觉没有给你足够的指南针来知道哪里需要改进。更确切地说，拥有一个封闭的反馈回路将为你提供方法，来调整你下次可以改进的地方。

*   专家像对待科学一样对待他们所做的事情。

他们收集数据，分析数据，创造理论，证明什么可行，什么不可行，然后测试这些理论。这有助于我们创建最佳实践。

有一个克服 OK 平台的概念会让你从一般到惊人。
