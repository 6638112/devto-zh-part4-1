1.-当你学习计算机逻辑的本质，它独立于任何语言。

在这一阶段，你使用的语言不应该是重要的，你要记住的是不要在基本内容上增加一层复杂性。

好吧，我已经知道要点了，我想毕业进入职场还是克服个人挑战，我该怎么办？

好吧，那么你现在需要的是分析，你需要知道你站在哪里，你需要知道什么技术支配着你的领土。

[![](img/f1b8b1a034933a2ab602b11fa283a5ce.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--eYAtFifJ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/y87y6ittbcj0wzmmtp82.png)

一些这样的文章中没有提到的一点是，您需要知道哪些技术更多地使用您国家的公共和私营机构，以便您可以轻松找到工作，因为如果您学习 C#(编程语言)编程，但在您的国家提供的大多数工作都是 Java(编程语言)

我不是说你不能多学一种语言，只是你需要知道你的背景并适应它，这样你才能更快地完成所学的东西。

就我而言，在我的国家 C#被国家和私营部门使用，但我喜欢 Python(编程语言)-我...。我学会了 C#和微软技术，这样我才能生存下去，最后，我没有忘记 Python，我的个人项目和附近的客户，我是在他身上做的。

Python 对我以后要迁移到的其他国家的工作很有吸引力，我现在和 C#一起工作，每天都在努力改进，甚至开始喜欢它。

结论是，你首先调查并根据它作出决定，你可以进入求职页面，根据软件开发、web 开发筛选它，询问一个亲近或有联系的人，告诉你开发技术使用较多，但始终面向你的国家。

现在，我会给你一个语言列表来学习逻辑编程:

1.- Python
2。- JavaScript
3。- Php
5。- C#
