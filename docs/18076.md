# 你如何教你的孩子编程？

> 原文:[https://dev . to/Amit _ savani/how-do-you-teach-programming-to-your-kids-2 npb](https://dev.to/amit_savani/how-do-you-teach-programming-to-your-kids-2npb)

我的大儿子上六年级，到目前为止，他的课程包括基础、Logo、kturtle、edubuntu。

上周末他让我帮他学习 Ruby。基本上，我的职业是 Ruby 程序员，而他对使用 Ruby on Rails 创建的应用程序非常着迷。

我从事编程工作大约有 12 年了。在我的职业生涯中，我设计了几门课程，并在不同的组织中培训了数百人。但我发现很难从哪里开始，如何训练孩子。

我确信你们中的许多人已经通过并找到了一条道路。分享你的经验需要一篇完整的文章，但是尽你所能分享，这肯定会有助于 kickstart。
