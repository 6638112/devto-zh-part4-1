# 宠物项目，保持自我更新的秘密

> 原文:[https://dev . to/ya shints/pet-projects-the-secret-to-keep-yourself-updated-51e 0](https://dev.to/yashints/pet-projects-the-secret-to-keep-yourself-updated-51e0)

对我们(开发人员)来说，最令人恼火和麻烦的过程是让我们自己跟上时代，因为每 2 秒钟就有一种新的语言/工具/技术从天而降。我们知道，如果我们想跟上潮流，不想在交谈中感到过时，就必须跟上他们。

这对于顾问来说是非常糟糕的，因为他们将在不同的项目上花费一些时间，他们必须了解他们将面临的一切，如果他们想在这一领域领导客户，还需要了解更多。

然而，大多数人都有家庭/个人事务要做，通常没有足够的时间来阅读他们每天面临的一切，他们也不应该这样做，因为从长远来看，这被证明会降低生产力。所以问题是我们应该做什么？

这个问题的答案是让我们自己养成一个习惯，每天花一些**短**但**固定**的时间在一个副业项目(通常被称为宠物项目)上，让我们的手保持干净，并通过实例学习。

宠物项目是开发人员学习和成长的一个很好的方式，无论是个人的还是专业的。他们让你走出舒适区，学习新技能，锻炼你的创造力。但是，在应付日常工作、孩子、朋友、家庭和无数其他承诺的同时，很难完成任何事情。

过去，我曾试图将兼职项目塞进我日历中其他项目的缝隙中。这充其量是令人沮丧的；最坏的情况是完全无效。我一直觉得我在欺骗我最热衷的*项目。那么接下来呢？*

 *# 从小事开始，但要坚持做下去

开始一个项目肯定很难，过一段时间再回到这个项目，当你已经在做一些完全不同的事情时，再重新开始就更难了。

所以每天做一点可以让你保持在正轨上，而不会感到无聊或疲劳。如果你每天坚持这样做，你会很容易从你停下来的地方找到它，因为它还在你的脑海里。

# [](#no-deadlines-or-pressure)没有期限或压力

不要给自己的副业设定期限或压力。这不是一个客户约定，你不想很快失去你的兴奋。保持每周专注于一两项任务，在轻松的氛围中完成。

慢慢来，喝杯果汁(不是咖啡😁)或者新鲜水果得心应手，让小长假的感觉战胜日常的敏捷心态。关闭[脸书](https://www.facebook.com/)，停止滚动你的[推特](https://twitter.com)信息，专注于你承诺自己每天要做的事情。

一旦习惯形成，压力就消失了，你开始为做自己热爱的事情而感到快乐。如果你对养成习惯感到无力，读一读类似这样的文章。这些观点不是来自我，而是来自我过去看过的经过验证的科学文章和资源。

# [](#track-your-progress-and-praise-yourself)追踪自己的进步，自我表扬

记录已经取得的成果和将要完成的工作。这样你就能感觉到你在里程碑方面做得如何。

如果你觉得自己落后了，不要付出额外的努力，而是专注于保持你的日常习惯。这将自动解决问题，让你的精力集中在这个项目上，并创造一种美好的感觉，那就是你花了一些时间为自己做你喜欢的事情。

我自己也有在一个客户身上工作很长时间而没有学到任何新东西的问题，但是在我养成这个习惯后，我感觉学习新东西很棒，并且有那种男人屈服的感觉。此外，我开始吃水果而不是晚餐，这对我帮助很大，解决了我的失眠问题。

我习惯了去邦宁斯为我的另一个宠物项目(我喜欢木工)买点东西。

# [](#shift-the-project-time-to-time)将项目时间移动到时间

不要在一件事情上工作太久。当我们看到每天都有新的语言和技术引入时，我们应该相应地改变我们最喜欢的项目来使用不同的语言/技术。

我特别喜欢前端，但也试着关注安全(网络和其他领域)、azure、IoT、Hololens 等等。来保持我兴奋水平。

我的方法是完成一个我开始着手的项目，然后用我想熟悉的新技术创建一个新项目。

如果我想在节省的空间里学习一门新的语言，我会用这门语言重写我的上一门语言，这有助于我面对不同的挑战，用不同的语言实现一些东西。

# [](#talk-about-your-pet-project-somewhere)在某处谈论你的宠物项目

这一个与你的宠物项目没有任何关系，但是帮助你发现弱点和新的挑战，因为它接触到爱问问题和学习新事物的人。

即使它是一个内部的棕色袋子会议，或演示，或你正在谈论一个外部事件，如聚会或会议，该项目可以帮助你轻松地收集内容，并演示一些即使演示神也无法阻止的事情(因为你以前已经玩够了)。

# [](#summary)总结

有东西玩，学习新东西。如果你刚刚开始(我讨厌说三年级学生)，这会给你很大的帮助，提升你的自信。

希望这能帮助你保持更新，如果你发现有什么可以补充的，请留下你的评论🙏🏼。*
