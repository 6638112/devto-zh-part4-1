# [](#southern-dev-conference-2019-post-mortem)南方发展大会 2019 马后炮

2019 年首届南方发展会议被视为一次成功。为了在成功的基础上更进一步，应该讨论实际发生了什么，我们收到了什么样的反应，以便我们能够更有效地分配明年的工作。首先，我想说这只是我的观点，也是我最大的努力来传达我直接经历的和间接听到的。然而，作为一个活动的组织者，我知道我的直接体验将会不同，因为人们不太可能直视我的脸，给我关于活动的诚实反馈。俱乐部的团队非常希望得到建设性的批评。作为一个非营利组织，我们的使命之一是建立社区，为了实现这一目标，我们需要变得平易近人和包容。所以，请让我们知道你的想法，并希望与社区中那些倡导技术社区的人合作。

## [](#successes)成功案例

#### [](#engagement)订婚

我们最大的成功是组织者、与会者、赞助商和演讲者彼此建立了牢固的关系，这种关系在会议结束后仍会继续。每个人都传达了一些关于新友谊的非常积极的信息，并加深了想要参与社区建设的感觉。对话继续在网上、公开场合和通过直接信息进行。这告诉我这是一件重要的事情，也告诉我为什么我们要尽可能地继续组织这个活动。因为我认为参与是我们最大的成功，所以我想交流可能导致这一结果的过程的不同方面。我们支持当地公司，并致力于发展壮大。我认为我们也为每个人建立了一个安全、包容和支持的环境。

## [](#initial-thoughts-towards-improvements)对改进的初步想法

我们了解到，我们可以在信息传播部门做得更好。改善这种情况的一个方法是对组委会的角色有一个更好的定义。比如我们没有传播会议 wifi 是什么。它可能看起来很小，但可能有与会者只是使用了数据，因为他们可能不愿意提问。我希望对我们会议的组织以及我们如何取得成功保持透明。我认为，我们在 2019 年的组织中存在的一个缺陷是，任务、目标以及如何完成任务都不明确。今年，我们将进一步明确我们的目标。

# [](#sponsor-committee-organizer-details)赞助商委员会组织者详细信息

#### [](#communications)通讯

每个赞助商都需要感觉到他们的捐赠物有所值。我们不仅想让我们的赞助商了解情况，还想与他们接触，并希望传达他们对会议成功的重要性。当我们了解到关于场地、筹备日、营销推广等方面的新进展时，我们想让我们的赞助商知道。本次会议的一个主要目标是通过赞助支付整个会议的后勤费用。如果我们能成功做到这一点，那么门票收入将被捐献出来，用于下一年的会议和当地的非营利组织。

#### [](#sponsor-package)赞助商套餐

赞助商包是一份传达我们活动的愿景和后勤要求的文件。该文件的创建将由赞助负责人领导，需要活动策划和营销部门的合作。这份文件将作为一个参考和真实的初始来源，以帮助指导我们的赞助商在他们的旅程中发现我们的活动。

#### [](#sponsor-recruitment)赞助商招聘

为了资助我们的活动，我们将依靠当地、地区和国家行业领先组织的金融投资。委员会发起人部门的责任是建立筹资机制，这看起来像是与潜在合作伙伴联系/建立网络。

# [](#speaker-committee-organizer-details)演讲者委员会组织者详细信息

#### [](#speaker-communications)扬声器通讯

我们的演讲者对这次活动非常有价值。演讲者可以通过与全球观众分享他们的见解来成就一番事业。2019 年，我们了解到这种有价值的见解可以激发一场城市运动。他们对活动和我们与会者的参与向我们的与会者、赞助商和组织者阐明了我们活动的愿景。这需要我们主动与演讲者沟通，让他们提前知道我们的意图，并在日程安排有任何变化时让他们了解情况。这也意味着传达演讲者的关注和问题。

#### [](#speaker-travel-and-lodging)演讲者差旅和住宿

成为会议发言人需要一种特殊的人。专业演讲者飞往世界各地分享他们的见解。我们的期望是，我们应该尽最大努力让这个过程对他们来说尽可能无缝。我们来自该地区以外的演讲者将分享他们在国内的经历，因此尽我们所能提供最好的经历是很重要的。我们将努力做好预算，以支付旅行、住宿、演讲晚宴和会议当天的费用。

#### [](#schedule-day-of-speaker-activities)安排当天的演讲人活动

活动当天的演讲者通常都很忙。有时，这看起来像是演讲准备、幻灯片编辑、演讲练习、独处时间或在活动现场与与会者交谈。因此，我们的活动需要一个安全安静的空间，让演讲者有时间准备。我们还希望明确我们的期望，了解我们的发言人可能有兴趣参加的活动，并努力为他们参与这些活动提供便利。

#### [](#call-for-papersspeaker-line-up)征集论文/发言人排队

2019 年，全球各地的演讲者对我们的活动表现出了极大的兴趣。我们有大约 100 份提交材料要处理。这些提交的内容从小段落到深思熟虑的研究论文摘要不等。选择演讲者的时间需要提前通知申请人，以便他们可以安排航班和物流。我们的主要策略是促进演讲者阵容的多元化和包容性。我们还将与代表我们目标受众的社区建立战略合作伙伴关系。

## [](#event-planning)活动策划

#### [](#events)事件

2019 年，我们举办了会议本身以及在此日期之前的几次经历。会议本身是在 4 月 27 日，这是与会者能够买到票的时间。购买门票进入会场，会谈，早餐，小吃和午餐。个人也有机会购买贵宾票，这发生在主要会议之后。在会议的前一天晚上，我们开始了编程比赛，比赛一直持续到会议本身。与此同时，我们进行了演讲练习，并举行了一次晚宴。前一天，我们举办了一场活动，介绍编程竞赛及其构建平台。我们希望扩大活动目录，以便更深入地吸引观众。

## [](#marketing)营销

我们正在努力代表整个地区，为此我们需要努力提高我们在整个地区的代表性。营销团队应该在我们的社交媒体平台上与我们的社区互动。与我们的观众保持联系是极其重要的。我们的社区很棒，如果我们保持与他们的关系，他们将为我们提供宝贵的反馈，让我们的活动变得更好。我们也将尽可能地支持我们当地的技术领域。这看起来像开源网站，利用当地的机器和设备，如果可能的话，使我们的徽章。它将是开源的，因此社区中的开发者将有一种方式能够参与会议并为会议本身的发展做出贡献。

* * *

## [](#finance)金融

我们专注于激发社区的建设。在我看来，这意味着我们正在鼓励竞争和可持续发展。我们有兴趣激活我们的与会者，激励他们继续参与这些年。这一理念促使我们的融资计划围绕开放的概念。我们不会以一种不容易被我们的参与者识别的方式得到我们的资金。就电子邮件营销和数据收集而言，这看起来像是选择加入，而不是选择退出策略。

# [](#conclusion)结论

会议的成功不是基于活动当天的进展。成功是基于我们社区的参与。我们正试图通过培育强大和可持续的技术社区来建设南方的经济。要做到这一点，我们需要保持参与和始终如一的参与和积极。这是一项艰巨的工作，但我们俱乐部相信，支持我们的社区，让世界变得更美好，这是必不可少的。
