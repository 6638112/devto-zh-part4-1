# [](#the-frustration)挫折😖

这个故事开始于你(至少是我)最喜欢的代码编辑器:VSCode。VSCode 最棒的地方是社区提供的所有工具，让您可以将代码编辑器变成自己的小泡泡。

✨:你可以下载大量的**扩展**，添加**片段**，改变布局，当然，改变**主题**！✨

这是我的 VSCode: [![](../Images/2dea2d8faa08196eb65e834326298ad2.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--EeoGs0tE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/um2y9r4qyg2rgo0r7bqc.png)

高亮选择很难看到，当我想快速选择一个错别字或更改名称时，我很快选择了整个单词，并因此感到沮丧。

# [](#the-research)研究💻

我已经修复了高光问题，但是颜色不是我真正喜欢的。所以我在市场上寻找一个有很好的高光对比度的主题，但是找不到一个符合我口味的。

相反，我找到了一篇来自 Sarah Drasner 的文章，关于如何[创建一个 VS 代码主题。](https://css-tricks.com/creating-a-vs-code-theme/)

喜欢这篇文章，所以我决定做我自己的主题:一个好看的高对比度主题。

首先，我在 Azure DevOps 平台上注册开发人员时遇到了一些问题。

所以，也许我花了 20-30 分钟才明白我需要什么，而不是 5-10 分钟，但现在我在这里，我可以在上面发表。⏱

因此，如果你遵循 css-tricks 的文章，你很容易找到如何生成一个主题模板，并开始创建。此外，VSCode 文档制作得非常好，可以定制一切，你可以像浏览浏览器一样查看 VS Code，因为它只是一个电子应用程序！👌

我希望我的主题有高对比度，我在 colorsafe.com 选择了 AAA 级的颜色。只有一个是 AA 级，让我的主题更漂亮。

所以这里是:**拯救眼睛 HC 主题**👁！
