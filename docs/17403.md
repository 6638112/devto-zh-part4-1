## 图书馆家族

```
Quark    TypeClass
    ↘    ↙
   Witchcraft
       ↓
     Algae 
```

| 名字 | 描述 |
| --: | --- |
| [T2`Quark`](https://hex.pm/packages/quark) | 标准组合子(`id`、`compose`、& c) |
| [T2`TypeClass`](https://hex.pm/packages/type_class) | 在内部用于生成类型类 |
| [T2`Algae`](https://hex.pm/packages/algae) | 实现`Witchcraft`类型类的代数数据类型 |
