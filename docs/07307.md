# 开源 CRM 项目(VueJS，Symfony)

> 原文:[https://dev . to/inshop _ group/open-source-CRM-project-vue js-symfony-gad](https://dev.to/inshop_group/open-source-crm-project-vuejs-symfony-gad)

嘿伙计们！

我们正在做开源 CRM 项目。也许有人会对它感兴趣。
客户端部分已完成#vuejs、#vuetify、#vuex 和使用#symfony 和#api-platform 创建的 API

API:[https://github.com/inshopgroup/inshop-crm-api](https://github.com/inshopgroup/inshop-crm-api)
客户:[https://github.com/inshopgroup/inshop-crm-client](https://github.com/inshopgroup/inshop-crm-client)

演示:[https://demo.inshopcrm.com/signin](https://demo.inshopcrm.com/signin)

它可以作为独立产品使用，也可以作为新项目的核心。CRM 有很多功能，比如 JWT 授权、用户管理和角色限制等等。

我们很乐意回答任何问题并讨论新功能。
