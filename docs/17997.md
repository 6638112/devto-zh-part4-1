# 重建个人品牌

> 原文:[https://dev.to/shellscape/rebuilding-a-personal-brand-e4l](https://dev.to/shellscape/rebuilding-a-personal-brand-e4l)

自满。这是最糟糕的。喝杯咖啡，我们就要登上[号故障船](https://i.chzbgr.com/full/1324739328/hC8866755/)了。我们都知道公司品牌至关重要。但个人品牌往往被忽视。它们体现了你的能力，是一份实时的、互动的、功能性的简历。你的品牌充分说明了你是谁。当一个个人品牌半途而废会发生什么？我得到了惨痛的教训。

> 在额外的一英里上永远不会拥挤。韦恩·戴尔

我的个人品牌在 2012 年达到顶峰。我参与了 jQuery UI 项目，我参加了一些活动(这在很大程度上要感谢我当时的雇主)，我偶尔做一些演讲，我在网上的表现从未如此丰富。然后一切都变糟了。

## [](#ground-up)绞起来

2000 年代初，我开始因自己编写的一些 Windows 应用程序在免费软件领域声名鹊起，这也是我大学毕业后的第一份工作。我围绕它建造了 shellscape.org，并把它作为我所有项目的登陆点。随着我添加更多的项目，我在网站上添加了一个小博客。

真正对我的品牌建设帮助最大的是*“jQuery 公司”*。我是那些认为 Twitter 只是一时的时尚，这个概念很傻的 T2 人之一。但是公司很肯定地要求我报名并积极宣传公司和我自己的工作。他们让我在小型集会上发言，并让我参加会议，与前端开发领域的重量级人物交谈。

我有很多推特粉丝。我的网站有足够的流量，可以在 AdWords 上每月赚 200 美元。

## 我犯了一个可怕的错误

关键是，我变得自满了。我从前面提到的会议促成型雇主转到了一家非常有趣的初创公司，凭借着我之前职位的成功和我的个人品牌。这是一次大幅度的加薪，我做的正是我想做的工作。我很激动。

在这家初创公司工作的 9 个月里，我开始让自己的品牌滑落。我没有写博客，也没有真正为任何著名的开源项目做出贡献，我停止了使用 jQuery UI。我不再参与我所在领域的 Twitter。我对自己的工作和收入很满意，买了一栋可拆卸的投资房，它占据了我业余时间的每一分钟。我是在成人，但我忽略了让我的事业有今天的东西。

不幸的是，这家初创公司在一个随机的周三倒闭了。我没有准备好。幸运的是， ***很大程度上是因为*** 我在科技同行中的知名度，我很快就被一个我以前从未说过话的 Twitter 追随者发现了。经过几周的面试和几份竞争性的工作邀请，我开始在 Gilt 工作。

我再次获得了增加福利的奖励，而且我正在做着非常酷的工作。我指的是那些让你一周都不想睡觉的东西，那些让吃饭看起来不方便的东西。在那里，我再次陷入了对我的品牌更深的自满。

## [](#the-sky-is-falling)天要塌了

吉尔特一直是一个混乱的行业；这在无数的商业网站和博客上都有详细的记录。2015 年末，该公司经历了大规模裁员。 ***恐慌模式*** 。然后在 2016 年初，Gilt 被一家更大的公司收购。不细说——很多人*非常*不开心。

在坚持了第一季度后，我决定谨慎地开始观察外面的情况。我真的很兴奋；有很多很棒的公司以有竞争力的薪水招聘员工，尽管远程工作的领域很窄(我从 2009 年就开始远程工作)，但看起来我状态不错。一晃两个月过去了，我还在寻找。我被许多公司拒绝了。有些人直接通过了，有些人认为我没有其他本地候选人那么耀眼，不足以保证被录用。更不用说我已经陷入了日常工作的维护文化，这使我处于轻微的劣势，因为我没有跟上最新的技术。但是我总是能够让人们相信我可以很容易地学会我不知道的东西(并且我可以/仍然可以)。

某处有赤字，我必须找出它。

## [](#mirror-mirror)镜子镜子

经过大量的反思和与同事的交谈，我集中精力研究了影响我招聘过程的三个条件:

1.  我远远落后于最新的技术。
2.  我已经变得不擅长面试了。
3.  我的个人品牌不存在。

现在，你看，我不需要给出一个工作实例或者执行一个测试项目来进行评估——从来不需要。在我整个职业生涯中从来没有过。当然，我做过一些现场编码，但这没什么大不了的。突然之间，公司要求我为面试做项目。只有少数人花时间查看我的 Github 账户和那里的工作。我失去了我的品牌，我失去了令人惊叹的因素。另一个奇怪的异常是，我没有收到一个推荐。我整个职业生涯中的调动都是通过推荐。

我不是一个有吸引力的候选人。没有什么能让我鹤立鸡群。我看起来不像是公司想要什么的权威，因为在我目前的职位上，我只是专有技术方面的权威。我失去了优势。

因此，我给那些从未听说过我的人打电话进行面试，那些负责招聘的人对我所做的工作并不熟悉。见鬼，他们中的一些人甚至不知道什么是“远程员工”。

## [](#the-lessons-they-have-me)吸取教训，他们有我

我不再寻找新的机会。相反，我决定利用我通常用来看电影或去酒吧的时间，为重建我失去的东西建立一个基础。所以我花了几周时间重建了这个网站。我重新调整，把它当作每周的优先事项。我和许多在不同领域成功的博客作者交谈过，得到了可靠的建议。我和一些有创造力的人讨论了如何为网站创造独特的感觉。

我又开始出版了。我开始开源并围绕我日常工作的各个部分创建项目，最重要的是:无论我在业余时间做什么。我开始在 Twitter、LinkedIn、Medium 和脸书上发帖。我全力以赴。

## [](#important-things-are-important)重要的事情重要

科技领域的个人品牌非常重要。这是了解你是谁和你在做什么的窗口。对于任何想了解你的人来说，这是一个发射台，在你知道之前，先和你谈谈。才过了几个月，我已经惊讶于我的作品是如何流传开来的。这是一个很小的观众群，但这是一个观众群，我只会继续努力增加这个观众群。

有线电视不要做[我，不要做这个我。 ***如果你没有现成的品牌，那就创建一个。*** 如果你有一个，保持它！](https://www.youtube.com/watch?v=wl2uDi6Zju8)
