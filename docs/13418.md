# 收藏夹:给初级开发人员的建议

> 原文:[https://dev.to/mr_steelze/advice-to-juniour-developers-4k1p](https://dev.to/mr_steelze/advice-to-juniour-developers-4k1p)

我读过很多建议初级开发人员如何在这个领域取得成功的文章。当我可以的时候，我通常会把它们写下来或者试着记在脑子里。

以下是我最喜欢的建议，排名不分先后:

1.  提问是好的，寻求帮助也是好的。

2.  分解它，再分解一些。

3.  沟通是关键。

4.  像最终用户一样思考。

5.  除了学习工具、语言或框架，编程不仅仅是写代码。

6.  找个导师。

7.  掌控你的进步。

8.  保持简单。

9.  了解如何编写干净的代码。

10.  学习如何阅读代码。

11.  错误消息是你的朋友。

12.  锻炼身体。

13.  专注于基础。

14.  当你去开会时，一定要带笔和纸。

15.  不要想你什么时候能从大三毕业，当你毕业的时候你会知道的。

16.  拥抱变化。没有什么是永恒的。

17.  面试不公平。不要让他们挫伤你的积极性。

18.  即使你在晚上和周末做其他事情，你也可以成为一名优秀的开发人员。集中注意力，让这些工作时间有价值，然后离开你的办公室，在更广阔的世界里度过你的空闲时间。

19.  贡献开源。

20.  写一篇博客。
