# 熨斗学校第一周。

> 原文：<https://dev.to/valenadelv/1st-week-flatiron-school-4nc2>

我非常兴奋地说，我已经到了代码训练营的地步！

我是谁？
我叫 Valena，我正在成为一名全栈式 web 开发人员。几个月前，我对编码知之甚少，甚至一无所知。我男朋友给我发了一个申请暑期黑客项目的链接，我考虑了一下。我没有参与这个项目，但正是这个链接让我看到了一个全新的想法:学习编码。从那以后，我加入了职业因果家庭，并在# 21 日挑战赛中与许多人建立了联系。这个挑战帮助我走出了我的内向城堡，真正地去了解人们。在挑战中，我开始学习 Javascript、HTML 和 CSS 的基础知识。我非常喜欢通过熨斗的预备课程来学习操作 DOM，以至于我仍然会在脑海中想象“阿达·洛芙莱斯”的画面。我以为我很酷！

**快进到本周**
当新兵训练营的第一天到来时，事情开始变得真实起来。我不由自主地兴奋起来，而*真的*紧张起来。我的脑海里立刻充满了对未来的积极想法(当我找到一份好工作的时候！)，并怀疑我是否在薯片上放了太多蘸酱。我是健康科学的学生。除了研究和阅读预科课程，我对自己将要进入的领域一无所知。我觉得这不仅仅是感觉我“不属于”..我甚至没有那种感觉。我觉得好像所有的机会都对我不利，整个世界都在等着我失败。这种感觉是真实的！但我正以此为动力，证明那些怀疑是错误的，并让自己感到自豪。
