# 作为一名 web 开发人员，最让我反感的是什么

> 原文:[https://dev . to/bensassialadin/what-offenses-me-most-as-a-web-developer-169g](https://dev.to/bensassialadin/what-offends-me-the-most-as-a-web-developer-169g)

我在 web 开发行业呆的时间够长了，已经变成了一个尖酸刻薄的老人，他会冲着邻居的孩子大喊大叫，让他们离开他的草坪。所以我决定写一个小清单，列出每次我遇到的让我绝望的事情。

客户联系我建立一个网站，然后当我告诉他们我的价格时，他们害怕地离开，只是因为他们认为我会向他们收取和 Wordpress 模板差不多的价格。真的吗伙计？真的吗？

其他 web 开发人员认为他们比你更神圣，仅仅因为他们以写代码为生。这些都是一类人，他们到处告诉每个人他们的智商，他们有多聪明，只是因为他们是“开发者”。初级开发人员拒绝接受比他们更有知识和经验的人的建议，因为他们强烈的自尊心阻止他们学习任何新东西。

伪企业家希望你免费开发他们的应用程序，以换取股票。这些人其实是想奖励你的努力，用不值钱的股票，除非你造出一个产品，给他们价值。我仍然不能相信这些人会为人类的 DNA 库做贡献。

什么都争论的开发人员，实际上什么都争论。从 spaces vs tabs 到 Linux vs Mac OS，以及介于两者之间的一切。我靠，这些人真烦人。

自称为设计师的人，受雇于某项工作时会着手购买和安装主题。不要误解我，购买主题没有错，但是如果你卖大麻，你就不能称自己为药剂师。

我知道，我在派对上很有趣。
