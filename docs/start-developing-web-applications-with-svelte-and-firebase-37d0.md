# 开始用 Svelte 和 Firebase 开发 web 应用程序

> 原文：<https://dev.to/jorgealvarez/start-developing-web-applications-with-svelte-and-firebase-37d0>

Svelte & Firebase 是一个[模板，可以使用 Svelte 和 Firebase](https://svelte-firebase-template.web.app/) 轻松创建 web 应用程序。

它集成了您启动项目所需的所有工具和代码，为您打下了坚实的基础。

## 目标

如果你对这个模板感兴趣，你可以很容易地学会如何:

*   使用 Svelte 创建一个 web 应用程序。
*   使用 Firebase 身份验证。
*   用规则保护 Firebase 数据库。
*   使用 Firebase 函数云。
*   创建公共和受保护的页面。
*   将其作为新项目的基础，以加快开发速度。

## 特性

您可以[下载模板](https://github.com/jorgegorka/svelte-firebase)并立即开始使用。只需检查[的指示](https://github.com/jorgegorka/svelte-firebase/blob/master/README.md)。

*   用户认证
*   公共部分
*   登录页面
*   注册页面
*   管理部分
*   团队管理(例如 CRUD)。
*   员工管理(例如 CRUD)。
*   可重用组件(表单、模式、按钮...)

有一个[现场版](https://svelte-firebase-template.web.app/)以防你想看它的动作。

## 技术

这个项目使用了所有这些惊人的技术。

### 火基

这是谷歌的一套工具来帮助你创建网络和移动应用。它给你一个数据库，托管，认证，分析等。

### 身材苗条

苗条是一种全新的构建用户界面的方法。尽管 React 和 Vue 等传统框架在浏览器中完成大部分工作，但 Svelte 将这些工作转变为编译步骤，在构建应用程序时进行。

### 实体化

基于材料设计的现代响应式[前端框架](https://materializecss.com/)。

由谷歌创建和设计的材料设计是一种设计语言，结合了成功设计的经典原则以及创新和技术。

### [T1】validate . js](#validatejs)

[Validate.js](https://validatejs.org/) 提供了验证 javascript 对象的声明方式。

### 纤薄路由器 SPA

[苗条路由器 SPA](https://github.com/jorgegorka/svelte-router) 为您的苗条应用添加路由。它是专门为单页应用程序(SPA)设计的。
