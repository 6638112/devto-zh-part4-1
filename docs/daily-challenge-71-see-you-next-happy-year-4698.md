# 每日挑战#71 -明年快乐时光再见

> 原文：<https://dev.to/thepracticaldev/daily-challenge-71-see-you-next-happy-year-4698>

当你的朋友对你说“下一个快乐的新年再见”时，你正在和他们说再见你微笑着挥手，说你会看到他们的。但是等等，下一个快乐的一年是什么时候？

给定一年，写一个函数，返回你将会见到你朋友的最近的一年，*下一年，所有唯一的数字*。

年份将始终表示为正整数。通过函数度过的一年不一定是快乐的一年。

**例子:**
`nextHappyYear(7712) ==> 7801`
`nextHappyYear(1001) ==> 1023`
`nextHappyYear(2018) ==> 2019`
