# 15 天的#100DaysofCode 换来了一个完整的 Noob——我有没有实现任何相关的东西？

> 原文:[https://dev . to/nanythery/15-days-of-100 days of-a-complete-noob-had-I-achieved-any-relevant-4d HC](https://dev.to/nanythery/15-days-of-100daysofcode-for-a-complete-noob-have-i-achieved-anything-relevant-4dhc)

我刚刚看了一篇关于对 30 多岁后转行的人的建议的帖子。我必须说这鼓励了我在 dev.to 写我的第一篇文章。

## [](#a-story)一个故事

让我给你介绍一个小故事。一个政治学毕业生如何放弃她在 CMO 的工作，从零开始成为一名开发人员。

正如我的简历所描述的，我从 21 岁开始就在西班牙的房地产行业工作，大部分时间是在银行工作。最后，在另一家公司，我成了首席运营官，后来成了 CMO。我必须说我真的很喜欢我的工作。但是有些事情并没有满足我。必须承认，在过去的一年里，由于许多原因，我真的筋疲力尽了。在那段时间里，我的同事们，今天的朋友们，我的磐石。

所以今年三月，我决定放弃一切，真正去追求我的梦想。当我意识到数学不是我的那杯茶时，我年轻时(我说的是 10 岁或 12 岁)的热情就被抛在脑后了。不知何故，我认为如果我不和他们相处，我永远也不会成为一名开发人员。所以我选择了相反的道路，生活只是把我带到了我之前提到的地方。老实说，没什么专业的。尽管我总是努力做到最好。什么都不重要。

我不仅仅是自学。我还在西班牙攻读 Web 应用程序开发的正式学位。我意识到，在这个行业，学术头衔不再那么重要了。但我需要觉得自己有资格申请这个行业的工作。

## [](#so-the-100daysofcodefinally)所以，#100DaysofCode...最后

我已经完成了我在学校的第一个季度(不知道怎么称呼)，相当成功。所以现在，我决定自学 JavaScript 代码，并提高我的 web 开发技能。这就是我们如何参加#100DaysOfCode 挑战赛的。

至此，我们大概都知道是关于什么的了。不过挺自圆其说的，呵呵。我从 15 天前开始，这是我到目前为止取得的成绩(记住我几乎是从 0 开始的):

*   我在 Codepen.io 建立了一个简单的投资组合[你可以在这里查看](https://codepen.io/nanythery/)
*   我在 FreeCodeCamp.org 获得了第一个响应式网络开发认证。
*   我的个人网站有了第一个版本
*   我发现了 Dev.to，我完全被吸引住了。
*   我扩大了我的联系网络，有了新的同事。试图用我有限的知识去帮助别人。
*   我设法制作了两个简单的 JavaScript 应用程序:[https://codepen.io/nanythery/embed/WBVVMQ?height=600&default-tab=result&embed-version=2](https://codepen.io/nanythery/embed/WBVVMQ?height=600&default-tab=result&embed-version=2) 灯泡[https://codepen.io/nanythery/embed/QXwLwb?height=600&default-tab=result&embed-version=2](https://codepen.io/nanythery/embed/QXwLwb?height=600&default-tab=result&embed-version=2) 骰子游戏
*   我参加了在阿利坎特庆祝的 VueJS 日。

哇，写这个列表非常鼓舞人心。这让我真正正确看待我在短短 15 天内真正取得的成就。而且很多。然而，这些是我经历过的一些感受和疑惑:

*   我研究得越多，就发现了我不知道的东西。不知道我是否能了解他们！
*   当我在通用仪表板中看到那些令人惊叹的代码笔时，我觉得我离那里很远。需要多长时间？我有吗？
*   害怕最终来到一家公司却无法解决问题。

所以，如果你和我现在的情况一样。**请随意分享你的感受和迄今为止的成就**。坚持住。

感谢您的阅读！
