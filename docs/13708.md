* * *

[计算机科学专业离散数学导论](https://www.coursera.org/specializations/discrete-mathematics)

由于我没有太多的数学知识，这让我担心学习计算机软件，这就是我想学习一点它的原因。

专业包括五门课程；一般的数学思维(主要是谜题)，组合学和概率，图论，数论和密码学。起初，我听说这些课程没有先决条件，除了基础数学和对主题的好奇心，还有一些 Python 知识。然而，对我来说，一些数学主题对我来说有点难，所以我不得不看几次视频，并对细节做一些额外的研究。这些课程提供了许多谜题、练习和测验，以便对主题有更深的理解。特别是教师展示了 Python 代码的例子，这不仅有助于更好地理解，也让学生在更熟悉语言的同时看到了语言本身的潜力。在录音练习中，他们提供了一些基本的结构来开始，所以这对他们自己来说并不是绝对困难的。老师们也非常友好地在论坛上帮助提问，如果在学习中有任何问题或障碍，也可以提供帮助。

我很高兴我现在对许多有趣的话题有了一点点的经验，所以当我将来面对问题时，我可以有一个更好的心态。我希望我现在对我的学习没有那么恐惧，我可以有更多的机会学习更多关于离散数学和相关主题的知识。

* * *

这门课是由几乎相同的导师从[数据结构和算法专业](https://www.coursera.org/specializations/data-structures-algorithms)上来的，这是一个很大的项目，其中一门课是我从[退学的](https://dev.to/ljuneaul/review-of-algorithmic-toolbox-24go)。
这实际上是那些课程的先决条件。

这个也在变化，但即使对于像我这样数学知识水平不高的人来说也是可行的。

最后，iirc，专业化的某些部分使用 Jupyter Notebook 和 NumPy，当时我不知道那是个“东西”。
