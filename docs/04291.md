# 每日挑战#57 -身体质量指数计算器

> 原文:[https://dev . to/the practical dev/daily-challenge-57-BMI-calculator-57fb](https://dev.to/thepracticaldev/daily-challenge-57-bmi-calculator-57fb)

写一个计算身体质量指数的函数
注:bmi =体重/身高^ 2

> 如果 bmi <= 18.5 return "Underweight"
> 如果 bmi < = 25.0 返回“正常”
> 如果 bmi < = 30.0 返回“超重”
> 如果 bmi > 30 返回“肥胖”
