## 什么是邮件营销？

对于开发者来说，电子邮件营销就是使用电子邮件让人们使用你的软件。软件是全新的还是只是有更新都没关系。这是与使用您的软件并想知道您何时发布新软件(或只是更新)的人保持联系的最佳方式。

每个人都抱怨他们每天收到太多的电子邮件，但电子邮件营销仍然是接触你当前用户的最佳方式。

如果你正在制作软件，你可以**使用电子邮件营销来建立信任和权威**同时让新的和潜在的用户来检查你的软件。

如果你只是想为另一家公司写代码，这是让领导有兴趣雇佣你的一个好方法。

电子邮件营销最重要的部分是你已经在他们的邮箱里了。这就是谚语所说的“迈出第一步”。与脸书、推特或其他社交媒体平台不同，你不必担心电子邮件会消失。

并不是说这些特殊的社交网络会很快走向何方，但是...你现在会买 MySpace 的股票吗？Friendster 怎么样？有时网站会失去人气或倒闭。

你的电子邮件列表永远不会被淘汰。除非他们退订或者干脆关闭他们的电子邮件账户，否则你永远都只能迈出这一步。

## 为什么电子邮件营销对开发者有效。

当有人访问你的网站时，他们并没有做出任何承诺。他们只是一个路人，看看有什么可看的。一个游客。

他们正在阅读你的内容，希望能喜欢，但是**他们只需按一下后退按钮，就再也看不到你的网站了**。这是一个决定，一次点击，不到一秒钟，你可能会失去一个潜在的用户。

这是我们希望尽可能避免的事情。我们希望他们使用我们的软件，在他们信任我们之前，他们不会购买(或租用)我们的软件。

他们必须相信软件会物有所值。用户必须相信我们的产品可以解决他们的问题。

我们的网站访问者不会信任我们，直到我们证明我们知道如何制作优秀的软件。对大多数人来说，这不仅仅是访问你网站上的一个页面。我们希望他们回来。

电子邮件是让他们回来的最好方式。

一旦你进入他们的收件箱，**有很多机会让他们回到你的网站并使用你的软件。**

如果你想让客户(或公司)雇佣你为他们开发软件，我建议你在教育方面提供更多的内容。这会建立信任。

如果你在卖软件，给他们发一些免费的内容。预告或者一些发展的背景信息。你有一个绝佳的机会来提供你已经创作的软件之外的内容。

我一般不建议提供折扣，但如果你想在你的业务中这样做，电子邮件是向回头客或那些只是观望并想尝试你的软件的人提供信息的方式。

## 选择电子邮件简讯提供商。

有很多不同的电子邮件营销提供商，所以当你第一次开始查看所有这些提供商时，选择一家可能会非常令人生畏。当你对电子邮件营销一无所知时，尤其如此。

你不想只使用普通的电子邮件服务，比如 Gmail，以及 CC 或 BCC everyone。那会引起很多头痛。

电子邮件服务提供商也可能不允许手动运行时事通讯。

当我们谈论它的时候，把 Gmail 作为专业邮件使用也是一个坏主意。获得一个网站和一个[firstname@yourwebsite.com](mailto:firstname@yourwebsite.com)电子邮件地址。或者出于安全原因使用类似 Protonmail 或 Tutanota 的东西。

我的主要建议是，在大多数情况下，你会得到你为 T1 所付出的。

我个人使用并推荐 [ConvertKit](https://garrettmickley.com/convertkit) (附属链接)。这不是免费的，但如果你真的想制作软件并把它送到用户手中，这是一项负担得起的投资。

ConvertKit 很棒，因为* *它超级容易使用，并且具有真正强大的自动化功能* *。

我是自动化的超级粉丝，因为我不喜欢一件事情做一次以上。这对于设置自动收入也很有用(我稍后会教你怎么做)。

谁不想在睡觉的时候赚钱呢？自动化的智能电子邮件营销可以让你比你想象的更快地到达目的地。

我最喜欢的功能之一是 Sequences，我用它创建了一个很酷的欢迎电子邮件序列，实际上产生了一些收入。通过一些调整，它将在不久的将来获得更多的收入。

对于任何认真对待软件业务的开发人员来说，这是我的建议。无论你是想被雇佣来开发软件，还是想出售自己的软件，这都是我使用和推荐的服务。

作为一个免费的替代品，我曾经使用 MailChimp。MailChimp 是免费的，但我发现它很难使用，而且一点也不友好。

对于刚接触电子邮件营销或数字营销的人来说，你可能会发现这更加难以理解。我知道这一点，因为这是我第一次使用的服务。

这不值得你开始头痛，并试图弄清楚一切。

但是，既然有这么多令人头痛的事情...我会给你我最初的欢迎邮件使用的模板，让你开始。

## 您的欢迎邮件。

欢迎邮件是注册后收到的第一封邮件。这非常重要，因为**是给人留下良好第一电子邮件印象的地方**。

显然，你已经给用户留下了某种第一印象，因为他们已经决定足够信任你，给你他们的电子邮件地址。现在，你需要给你的邮件留下另一个第一印象，这样他们就不会后悔，也不会退订。

失去订户可不好玩。

我已经设置了**一个模板，您可以按照**的说明进行每个步骤:

1.  首先欢迎他们加入邮件列表，并提醒他们注册的原因。有时候人们会忘记。
2.  如果你为注册提供了某种奖励，比如免费的软件演示，给他们一个访问它的链接。
3.  告诉他们一点你的情况。为什么你有资格向他们发送简讯、你过去开发的软件以及你曾经参与的任何专题(杂志、播客等)。
4.  他们可以从未来的时事通讯中得到什么。谈论你发送什么样的内容以及发送的频率(提示:你应该每周至少发送一次有价值的内容)。
5.  告诉他们可以关注你的其他地方，如 Dev.to、Github、脸书、Twitter 等。
6.  问他们一个问题。你需要确定你使用的电子邮件服务允许回复。如果你正在通过你的博客和时事通讯教授一些东西(你应该这样做)，这是一个询问人们他们正在努力做什么的好时机，这样你就可以得到写新内容的想法(或者，如果是你已经写的东西，用一个链接回复他们！).

使用这个模板，你就可以开始写一封好的欢迎邮件。随着时间的推移，你可以随时调整它。

然后，您需要设置您的电子邮件服务来自动发送欢迎电子邮件。我把我的欢迎邮件设置在他们注册一小时后。

## 设置自动装置。

我有一个坚定的信念:如果你不得不做一件事不止一次，你应该让它自动化。幸运的是，你的电子邮件简讯服务很可能提供了某种程度的自动化。

例如，ConvertKit 有两种不同类型的自动化:即**“序列”**部分和**“自动化”**部分。

我确实感到有点困惑，一个实际上被称为“自动化”,而另一个不是，但是**两者都是自动化的形式**,一会儿你就会明白如何。

**序列是一系列在预定时间自动发出的电子邮件。**例如，我的滴滴简讯序列有许多电子邮件，每封邮件间隔一周。

我有一份[自动每周简讯](https://garrettmickley.com/how-to-set-up-an-automatic-newsletter-without-wasting-time/)。基本上是定了就忘了。

我还使用 sequence 创建了[免费电子邮件课程](http://gamingforfunandprofit.com)。

有人订阅接收电子邮件课程后，第一封电子邮件会立即发出。之后， **ConvertKit 在最后一封邮件**之外的一天发送每一课。

在 ConvertKit 中，电子邮件序列可以设置为相隔几小时、几天、几周。这很方便，取决于你需要什么。

在我以前的电子邮件课程中，有一封邮件问了一个问题，用户可以从三个选项中选择一个。如果他们没有选择一个选项，他们将不会收到下一封电子邮件。它被要求继续这门课程。

如果他们没有点击其中一个选项来继续课程，我会在一周、一个月和六个月设置提醒。

自动化部分有很多特性，有一天我会写一篇关于它的博文。这些功能包括标记订户、集成其他平台和服务等等。

为了这篇已经很长的文章，我只打算讨论标记订阅者，因为我发现这很有用。

标记订户有助于您识别哪些订户对您的业务和时事通讯的哪些方面感兴趣。

例如，假设我的网站和时事通讯通常包含两个主题:建立软件业务和我的个人软件。我还提供编码服务，并在未来将提供购买产品的方式。在我的欢迎邮件中，我可以问订阅者:

> 您是否希望收到以下方面的消息:
> 
> *   学习建立一个软件企业
> *   我的软件和其他项目
> *   都是！

然后我在每个页面上设置了一个链接，将他们带到一个单独的感谢页面，上面有关于这个主题的更多信息。

我使用 Convertkit 的自动化来标记哪些订户点击了哪些。

当我发送关于建立软件业务的电子邮件时，我会将这些电子邮件分段，只发给那些点击了该选项的人(“两者都要！”不获取链接，只是默认设置；不需要给那些人贴标签)。

这样，**用户就不会收到与他们兴趣无关的电子邮件，他们将来也不太可能退订**。

## 让人们注册你的时事通讯。

这是最难的部分:让人们注册你的邮件列表。

这是他们私人生活的开始。电子邮件收件箱是神圣的，不只是被大多数人随意分享。

然而，有些人有几十万甚至几百万的电子邮件用户。他们是怎么得到它们的？有一些营销人员使用的技巧是经过检验的，是真实的。

### 获得首批 100 名订户:

**第一:随便问问。**

就这么简单。打开手机，翻看联系人列表。你认为那里有谁会感兴趣，就去问他们。给他们发一条短信，内容如下:

> “嘿，我正在为我的网站创建一个时事通讯，在那里我分享我的编码项目，我想你可能会感兴趣。我可以把你加入名单吗？”

如果他们说不，感谢他们，不要为此烦他们。如果他们说是，回答如下:

> “牛逼！很高兴你能看到我在做什么。什么邮件最适合你？”

和脸书的朋友、推特粉丝以及你能想到的任何地方重复这个过程。无论你在哪里有你认为可能感兴趣的联系人，就问他们。

不要在你知道不会感兴趣的人身上浪费时间。

### 获取您的下一个 100 名订户:

接下来要做的是**让所有访问你网站的人注册**。

你的电子邮件简讯服务应该提供一些简单的复制和粘贴注册表格，你可以直接在你的网站上输入代码。

许多营销人员建议你在边栏(如果你有的话)上注册，在标题上注册(如果你可以在不妨碍标题其余部分的情况下注册)，在每个博客帖子和页面的底部注册。

我更喜欢跳过标题注册，以保持顶部的一切更加整洁。相反，如果有意义的话，我会经常在博客帖子的更高处注册。我不会试图把一个东西塞进一个不合适的地方，无论是物理上还是环境上。

如果你在 Wordpress 上使用 ConvertKit，你可以用他们的 Wordpress 插件很容易地做到这一点，这正是我所用的。

1.  你需要去 Wordpress 安装并激活 ConvertKit 插件，
2.  然后转到您的 ConvertKit 帐户，获取您的 API 密钥和 API 密钥。
3.  然后，在 Wordpress 的设置下，你会找到 ConvertKit 插件。
4.  去那里把你的两把钥匙放进去。
5.  单击“保存更改”,它将刷新您当前表单的下拉列表。
6.  然后，您可以选择您想要的默认设置。

在 Wordpress 中，在 Appearance 下，转到 Widgets，你可以将 ConvertKit widget 拖放到任何你想要的边栏中。

要将它放在文章的底部，您可以在每个单独的文章中选择该选项，该选项位于文章文本框下方的页面底部。

它会自动将您设置的表单作为默认表单，但是您可以很容易地将其更改为另一个表单，或者根本不更改。

这也适用于 ConvertKit 登录页面。

## 在你的全新电子邮件简讯中发送什么。

您已经遵循了上面的所有说明。你注册了 ConvertKit。你设置了一封欢迎邮件。你甚至设置了一些标签来划分你的列表。

但是你送什么呢？

**更新！时事通讯！你正在做的项目**！

大多数情况下，只是与你正在做的事情相关的东西，你的订阅者希望看到的东西。请记住您已经标记他们要接收的首选内容。

如果你在用你的博客向人们传授你所知道的，这也是你应该做的，那么一定要在你的电子邮件中发出这些信息。

要确保的一件事是，你不会不断地用你的软件和/或其他产品的广告轰炸你的用户。那是垃圾邮件，而且是不好的。

一定要推销自己，但不要过度推销。

## 何时发出你的简讯。

你需要每周至少发送一次简讯。这能让你留在人们的脑海中，增加你获得新销售的机会。

研究表明，你发送的电子邮件越多，你的点击率就越高。不过，确保你不会给人发垃圾邮件是很重要的。这是失去订户的一种快速方式。

营销人员一直在研究一天中的时间和一周中的哪一天发送电子邮件，这些时间和日期变化很大。

[CoSchedule 对比了 10 项不同的研究](https://coschedule.com/blog/best-time-to-send-email/)发现**周二是发邮件的最佳日**，如果你一周发两封邮件，周四是发第二封邮件的最佳日。

周三也是一个受欢迎的日子。

至于一天中的时间，他们发现上午 10 点和 11 点是最佳时间点，晚上 8 点到午夜之间的任何时间也是如此。

我通常在星期二发出我的时事通讯，但是每个星期的时间是不同的。

当然，**这都取决于你的受众**。

您的用户可能会在不同的时间操作。

例如，如果你是一个游戏开发者:你的观众主要是视频游戏玩家和/或向你学习如何制作视频游戏的人，他们可以在非正常工作时间工作(例如通宵玩你的游戏的大学生)。

但是，像数据分析这样的另一个行业可能有一个主要是白天工作的人的受众，因此在上午 9 点到下午 5 点之间打开最多的电子邮件。

最好的办法是* *尝试不同的日子和时间，找出你的听众更喜欢什么。*

## 法律货色。

在美国，根据名为《CAN-SPAM 法案》的反垃圾邮件法律，**你必须在你的电子邮件简讯中有一个有效的实际邮寄地址**。这个地址不一定是你的家或办公室。可以是邮政信箱。

它必须是一个能联系到你的地址。你不能在谷歌地图上随便选一个加油站地址。

如果您使用的是 ConvertKit，他们会为您提供一个可以使用的免费地址。

出于安全原因，我建议你不要使用你的家庭地址，除非你已经在互联网上公开了你的家庭地址。一般来说，这根本不是一个好主意，所以如果你能把它拿下来，建立一个办公室或邮政信箱，那将是一个更安全的做事方式。

还有，**不要用假地址**。违反 CAN-SPAM 法案的一封电子邮件可能会让您损失高达 16，000 美元。

你还需要在电子邮件中有一个取消订阅按钮，这样用户就可以很容易地将自己从你的列表中删除。

一些营销人员使用许多聪明的技巧(通常称为黑暗模式)来迷惑试图退订的人。

最好的策略是让它变得容易。无论如何，想要退订的人不是你的目标受众，否则他们不会想要退订。
