# 学习 Java 和相关技术的 20 本书

> 原文:[https://dev . to/javinpaul/20-books-to-learn-Java-and-related-technologies-KGM](https://dev.to/javinpaul/20-books-to-learn-java-and-related-technologies-kgm)

大家好，今天，我将分享一些过去几年中最好的 Java 书籍，你们现在可以阅读这些书籍，以便更好地学习 Java 和相关技术。这些是我在过去几年里读过的书。所以，如果你没有在这个列表中找到你最喜欢的书，不要失望，我可能没有读过它，但请留言，我很乐意加入我的阅读清单。

这个列表包括学习 [Java 8](http://www.java67.com/2014/09/top-10-java-8-tutorials-best-of-lot.html) 、 [Java 9](http://www.java67.com/2018/01/top-10-java-9-tutorials-and-courses.html) 、 [Spring 5](http://www.java67.com/2017/11/top-5-free-core-spring-mvc-courses-learn-online.html) 、 [Groovy](http://javarevisited.blogspot.sg/2017/12/10-things-java-programmers-should-learn.html#axzz53ENLS1RB) 、 [Scala](http://javarevisited.blogspot.sg/2018/01/10-reasons-to-learn-scala-programming.html#axzz550Ppgfxg) 、 [Hibernate](http://javarevisited.blogspot.sg/2018/01/top-5-hibernate-and-jpa-courses-for-java-programmers-learn-online.html#axzz55mTqgbD8) 、 [Spring](http://javarevisited.blogspot.sg/2016/12/top-5-spring-and-hibernate-training-courses-java-jee-programmers.html) 、 [Agile](http://javarevisited.blogspot.sg/2018/01/top-5-free-agile-courses-for-programmers.html#axzz54v9Z26qM) 、 [Scrum](https://www.java67.com/2020/06/top-5-books-to-become-scrum-master-in.html) 、[微服务](https://medium.com/javarevisited/10-best-java-microservices-courses-with-spring-boot-and-spring-cloud-6d04556bdfed)、[算法](https://medium.com/javarevisited/7-best-courses-to-learn-data-structure-and-algorithms-d5379ae2588)等 Java 开发者的重要素材。你不需要读所有的书——这个列表会给你一些关于你想学的有趣的书的想法——但是如果你像我一样是个书呆子，并且有时间看书，特别是在上下班途中，那么这是一个非常有趣的列表。

这是我过去几年读过的 20 本书的清单，今年你也很有可能会读到它们。 [**有效 Java 第三版**](http://www.amazon.com/dp/0321356683/?tag=javamysqlanta-20) 应该是你今天读的第一本书，但是如果你确实有空闲时间，你可以用这个列表挑选一两本书。

### [](#1-modern-java-in-action)1。现代 Java 在行动

[这是 2016 年我最喜欢的书之一](http://www.amazon.com/Java-Action-Lambdas-functional-style-programming/dp/1617291994?tag=javamysqlanta-20)。2017 年，我一直在定期阅读这篇文章，同时学习 JDK 8 的功能。如果你还没有读过或者还没有开始阅读 JDK 8，今年是阅读这本书的好时机。如果你需要 Java 8 的教程，你也可以在这里查看我最喜欢的 Java 8 教程列表[。](http://www.java67.com/2014/09/top-10-java-8-tutorials-best-of-lot.html)

这本书将教你用 Java 编写代码的所有现代方法，如 lambdas、Stream、函数式编程以及反应式编程。不过，如果你更喜欢在线课程而不是书本，我建议你看看 Udemy 上的[](https://click.linksynergy.com/fs-bin/click?id=JVFxdTr9V80&subid=0&offerid=323058.1&type=10&tmpid=14538&RD_PARM1=https%3A%2F%2Fwww.udemy.com%2Fjava-the-complete-java-developer-course%2F)**完整的 Java 大师班课程，这是最近为 Java 11 更新的。**

 **[![](../Images/4674c7a1bb74309b25fa0fc0207c5042.png)T2】](http://www.amazon.com/Java-Action-Lambdas-functional-style-programming/dp/1617291994?tag=javamysqlanta-20)

### [](#2-clean-architecture)2。干净的建筑

这是我最近发现的最好的书之一。我是鲍勃大叔作品的忠实粉丝，已经阅读了[的《干净的代码](http://www.amazon.com/Clean-Code-Handbook-Software-Craftsmanship/dp/0132350882?tag=javamysqlanta-20)和[的《干净的编码者](https://www.amazon.com/Clean-Coder-Conduct-Professional-Programmers/dp/0137081073?tag=javamysqlanta-20)。这些书更多的是谈论如何让你的架构正确。这是一本独一无二的书，如果你想了解不同软件架构的优缺点，这是值得一读的书。

[![](../Images/5b88f10db11b43e3ed0911738ba6a037.png)T2】](https://www.amazon.com/Clean-Architecture-Craftsmans-Software-Structure/dp/0134494164?tag=javamysqlanta-20)

### [](#3-grokking-algorithms)3。搜索算法

[这是另一本很棒的书](https://www.amazon.com/Grokking-Algorithms-illustrated-programmers-curious/dp/1617292230/?tag=javamysqlanta-20)我去年读了这本书，可能是算法方面最好的书。尽管就其对[算法](https://medium.com/javarevisited/20-algorithms-coding-problems-to-crack-you-next-technical-interviews-23191f229788)和[数据结构](https://www.java67.com/2019/07/top-10-online-courses-to-learn-data-structure-and-algorithms-in-java.html)的覆盖面而言，它非常短，但它所覆盖的内容非常有趣和有用。

它通过与现代例子相关联，赋予旧概念新的生命，比如脸书可能如何存储其用户。

顺便说一句，如果你的目标是学习数据结构和算法，那么我也建议你参加一个综合的在线课程，比如 [**数据结构和算法:在**](https://click.linksynergy.com/fs-bin/click?id=JVFxdTr9V80&subid=0&offerid=323058.1&type=10&tmpid=14538&RD_PARM1=https%3A%2F%2Fwww.udemy.com%2Fdata-structures-and-algorithms-deep-dive-using-java%2F) **[Udemy](https://medium.com/@udemy) 上使用 Java**T3】进行深度学习

[![](../Images/0c3381939adeeda01efe3c2b2aa2a4b9.png)T2】](https://www.amazon.com/Grokking-Algorithms-illustrated-programmers-curious/dp/1617292230/?tag=javamysqlanta-20)

总之是今年想学算法的初学者必读的书之一。如果你需要更多的建议，在这里查看我的[列表](http://www.java67.com/2015/09/top-10-algorithm-books-every-programmer-read-learn.html)。
