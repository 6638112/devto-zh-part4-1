所以。我有个愚蠢的想法。它最初应该是为处于工作间隙的广播创意人员和技术人员准备的，大概是这样的...创意人员、技术人员聚在一个房间里，为一个节目提出想法。如果所有部门都同意，他们就制作节目。每个人都永远得到一份收入。

唯一的问题是:没有人为内容付费，当然，还有 0 美元的 x%。

因此...我在想。又来了。(我知道，对吧？我真的不需要做那么多)为什么开发者不这样做？如果你什么也没穿，为什么不聚在一个房间里，分享一些想法呢...如果大家都喜欢，那就做吧。先整理出所有权的规则，然后着手去做。不一定是世界上最大的事情，让我们面对它$ x/月/用户比$0 好。

只是一个想法。
