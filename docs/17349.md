。使用响应图像

这个功能是默认提供的，是在 WordPress 4.4 版本中引入的。它利用 srcset 属性为浏览器提供不同大小的各种图像列表以供选择。

这种方式浏览器将根据用户设备的大小加载适当大小的图像，而不是每次都下载并带来沉重的图像文件。

例如，如果用户在他们的移动设备上浏览，他们将在 srcset 属性中收到较小尺寸的图像。它通过设置 srcset 代码来自动添加图像。这个过程很受欢迎，因为它不会干扰你的 WordPress 站点的任何其他插件或主题。

。使用图像优化插件

在 WordPress 插件库中，有各种各样的图片优化插件。你可以很容易地让他们压缩或优化你的图像。图片大小调整或者懒人加载，一切都可以通过 WordPress 插件轻松实现。

举几个例子，reSmush 和 WP Smush 是图像优化最流行的。然而，还有其他多种方法可用。你可以选择一个最符合你要求的。
