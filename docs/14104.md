# 在 Ubuntu 中创建一个可引导的笔式驱动器

> 原文:[https://dev . to/anuragrana/create-a-bootable-pen-drive-in-Ubuntu-p08](https://dev.to/anuragrana/create-a-bootable-pen-drive-in-ubuntu-p08)

Ubuntu 有一个默认程序来创建一个可引导的笔式驱动器。
按照以下步骤创建相同的。

1.  从他们的官方网站下载最新 Ubuntu 的 iso 镜像文件。
2.  现在将一个空的 USB 驱动器连接到系统。
3.  启动“启动盘创建器”程序。

[![](../Images/0fa61242f66cfa888929de4175a9a506.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--ojDCVYuN--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/3iszecgakv594dvxlrjr.png)

1.  选择下载的 iso 映像文件作为源文件。

[![](../Images/8295586cf88163b6ddb4f5614c71e195.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--33rsfS9_--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/twu9yykm6nxfty5ov4p8.png)

1.  选择 USB 设备作为“要使用的磁盘”。
2.  选择小于 1 GB 的保留磁盘空间。单击“制作启动盘”。

[![](../Images/192977834ac1a7fbc7572c605661737b.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--ovN1vQfE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/40k11rsydekeqvu0x47o.png)

1.  文件将被复制到磁盘。一段时间后，当磁盘上的操作系统安装完成后，单击“退出”按钮。

[![](../Images/1d671563eb54d2965c035ab6885f7b47.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--ylPQu4yj--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ro3ayombufczevsyr85a.png)

[![](../Images/b4f876954e55222a1afe4f3988f2f822.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--uxg1CFfE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/blcwzu84btix2acioc2l.png)

1.  卸下 u 盘，在任何系统上安装 Ubuntu。
