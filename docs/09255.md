# 收到第一个报价

> 原文:[https://dev.to/torianne02/first-offer-received-hmn](https://dev.to/torianne02/first-offer-received-hmn)

已经一分钟没写关于找工作的帖子了。我真的很喜欢写技术博客，以至于很难停下来休息一下。然而，本周我要离开他们，因为我终于收到了我的第一份工作邀请！

这几个月来，我把找工作作为我的全职工作，这绝对是疯狂的几个月。谢天谢地，我一直在非常努力地记录每一件事，所以我有一些关于我找工作的统计数据，我很乐意与你们分享。在我开始之前，我想分享一些我在 133 天的旅程中学到的经验。

## [](#the-lessons)吸取教训

*   我很清楚找工作是多么的令人疲惫、耗时和沮丧，所以我认为庆祝一路上的小胜利非常重要，这有助于保持你的动力和高昂的情绪。第一次面试成功时，拍拍自己的背，第一次进入第二轮面试时，也这样做。总的来说，只要**为自己**的进步感到骄傲。

*   你会遭到很多拒绝。这只是过程的一部分，你不能把他们当成个人。当我第一次开始找工作时，每封拒绝邮件都让我感到刺痛。我让很多人给我同样的建议，就像我刚刚给你的一样，但是我的身体和思想仍然有这样的反应。对于像我一样的人，请知道这种感觉最终会消失。你会得到很多，这是理所当然的，他们会越来越少打扰你。

*   记录你的求职过程，这样你就能知道什么适合你，什么不适合。大约两个月之后，我决定根据我似乎从哪些招聘网站上得到更多回复来重新评估一下。当我实现我需要的改变时，它给我带来了相当大的不同。

*   我要提到的最后一课就是要有耐心。几个星期以来，我一无所有。不接受采访。没有编码挑战。没什么。我会开始感到有点气馁，然后突然在接下来的一周左右，我结束了这么多的事情，我几乎不能管理它，并想知道为什么我曾经担心摆在首位。所以，对自己和过程要有耐心。*在这个情绪疲惫的时期保持积极的心态并不容易，但这是可行的！*

我想指出的是，我在求职过程中学到的这些经验和技巧并不总是适用于所有人。你需要有自知之明，了解什么对你最有效！

我知道你可以用一些技巧来找工作。在任何人试图告诉我我发出的申请数量是荒谬的之前，请记住这是平均每天 *2.3 份申请*。我能够日复一日地发布高质量的应用程序，所以我做到了。

这篇文章的目的是提醒所有正在找工作的人，这个过程需要时间和耐心。不要气馁。不要放弃。你最终会得到第一份工作，这会让你感觉像卸下了肩上的重担。永远记住**如果你觉得不合适，你不一定要接受第一个提议**。是的，已经找了很长时间了，但是现在你有了一个，你可以再找一个。**保持乐观。高昂起你的头。你可以这样做**。
