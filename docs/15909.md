# 第一步

> 原文：<https://dev.to/oieeaaaa/1-first-step-like-react-59ib>

我们迈向目标的旅程总是从这些微小的进步开始。而那些微小的碎片就像一块积木，用来创造一些东西，一些在那些总是着眼于整体而不是像素的人眼里看起来太复杂的东西。

我作为一名前端 Web 开发人员在学习方面的经验是，它总是从一小步开始，比如我们如何在 React 中构建我们的组件，它总是从一些非常小的事情开始。一些我们可能没有注意到的事情。这些小步骤是迈出更大步的关键，一步又一步，一步又一步，突然你就有了一辆车，这样你就不会因为走太多步而感到累了。

老实说，除了使用组件、状态和道具或者用它们构建一个待办事项列表之外，我对 ReactJS 不是很了解。但是我写作。为了传播我的天赋、才能、经验、知识和智慧，我从艰苦的磨练中学到了这些。
