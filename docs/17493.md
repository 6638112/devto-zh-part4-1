# Golang PayU

> 原文:[https://dev.to/connectsteven/golang-payu-4878](https://dev.to/connectsteven/golang-payu-4878)

使用 Golang 快速集成 PayU

PayU 是在印度、美国以及其他国家广泛使用的在线支付网关。

您可以将 PayU 整合到您的 Golang 网站，以便轻松支付。您必须验证从 PayU 官方网站收到的商家密钥。

在这里查看我们如何进行检查的代码: [Golang- PayU](https://github.com/Mindinventory/Golang-PayU) 。
