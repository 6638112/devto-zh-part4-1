# [](#1-start-easy-discussions)1。开始简单的讨论

通过向你的成员提问，让他们习惯于交谈。一般的 chatter 会在你有了好的用户基础后开始流动，但这需要一些时间。让你的问题可以被任何人回答，并且是开放式的。以下是我过去一个月问的一些问题:

*   在过去的一周里你完成了什么事情？
*   你最近不得不解决的一个奇怪的问题是什么？
*   你想了解更多的是什么？
*   你今天/这个星期在做什么？
*   在过去的一个月里，什么挑战了你？

与此同时，用你的问题建立良好的节奏。我通常每周做大约 2-3 个问题，当我早上上班的时候我会把它们踢开。

# [](#2-follow-up-on-everything)2。跟进每件事

你的成员正在回答问题。很好。让我们让他们继续谈下去。提出你的问题后，留下来，对*的每一个回答提出后续问题*。没错。每一个答案。用相关表情符号做出反应。从人们口中套出细节。这使得讨论更具互动性，而不是人们只是对着空间大喊大叫。

例如，如果有人发帖说他们正在学习一门新的编程语言，我会跟进并询问他们为什么选择这门语言。为了工作？他们在做什么项目？如果有人说他们解决了一个问题，我会问他们在解决这个问题上花了多长时间。任何让人们继续交谈的事情。

其他成员会注意到并最终跟进彼此对讨论的回答。很高兴看到成员们这样互动。"嘿，我*也*刚刚开始做一个 React 项目！"

## [](#3-become-an-expert-on-your-members)3。成为你会员的专家

你的成员更多地参与讨论。他们在谈论他们正在做的项目或他们学到的东西。吸收这个。当成员们觉得与社区联系更紧密时，他们会留下来，所以现在你可以扮演柏拉图式的媒人，把人们联系起来。

当 Sally 发布她正在进行一个新的 React 项目时，我可能会标记 Megan——她上周提到她也在学习 React。嘣。即时的朋友。

当 Paula 贴了一张她新养的哈巴狗的照片后，我可能会给 Joann 和 Tabatha 贴上标签，他们也有哈巴狗。嘣。即时的朋友。

你所做的是在你的平台上人性化每个成员的数字存在。有时候依靠文字交流感觉很冷，会让会员脱离你的平台。然而，如果 Paula 刚刚和另外两个成员谈论过关于养一只哈巴狗的事，她更有可能留下来，稍后再贴更多她的狗的照片。

## [](#4-make-new-members-feel-welcome-immediately)4。让新成员立即感到受欢迎

加入一个新的社区可能会很可怕！尤其是如果这些成员还没有亲自见过很多其他人。通过有意识地让新成员感到受欢迎，确保轻松过渡到平台。

这可*自动化*了，这让 SRE 的心漏跳了一拍！我配置了[欢迎机器人](https://slack.com/apps/A2LJCT3GR-greetbot)以两种方式欢迎新成员:在我们的#通用频道打招呼和私信。

开放的问候很简单，只是标记成员。这很好，因为新成员不仅感觉准备好加入讨论，而且让其他成员知道我们有了新成员。聚会之后，我们经常会有五个或更多的新成员连续加入。真的很高兴看到#将军在挥手致意。

## [](#5-poke-existing-members)5。戳现有成员

你做得很好。你有一个坚实的会员基础，他们经常在#general 和其他一些频道聊天。现在，我对这一步的时间线没有硬性规定，所以我要说的是*你会知道什么时候做这个*。

偶尔，在#general 上发布类似“在过去的几个月里，我们有了很多新成员。我们都重新介绍一下自己怎么样？”通过这种方式，你可以让很多现有成员参与进来，并更新每个人的信息。也许他们找到了新工作。也许他们改变了爱好。即使他们仍然是同一个人，把介绍作为一个连续的过程也是一种很好的氛围。
