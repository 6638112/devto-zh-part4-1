# 每日挑战#6 -奶奶和她的朋友

> 原文：<https://dev.to/thepracticaldev/daily-challenge-6-grandma-and-her-friends-23kd>

新的一天，新的挑战。今天，你的任务是帮助你的祖母拜访她的朋友。

这个挑战来自 CodeWars 上的用户 [@g964:](https://www.codewars.com/kata/5536a85b6ed4ee5a78000035)

> 你的奶奶住在 X0 镇，她有朋友。这些朋友以数组的形式给出，例如:朋友数组是
> 
> 【“A1”、“A2”、“A3”、“A4”、“A5”】。

注意:朋友的顺序是这个数组，它不能改变，因为这个顺序给出了他们将被访问的顺序。

> 这些朋友居住在城镇中，你有一个朋友和城镇的数组，例如:
> 
> [ ["A1 "，" X1"]，["A2 "，" X2"]，["A3 "，" X3"]，["A4 "，" X4"] ]
> 或
> [ ("A1 "，" X1 ")，(" A2 "，" X2 ")，(" A3 "，" X3 ")，(" A4 "，" X4") ]
> 或
> (C)
> {"A1 "，" X1 "，" A2 "，" X2 "，" A3 "，" X3 "，" A4 "，" X4"}
> 
> 这意味着 A1 在 X1 镇，A2 在 X2 镇...有可能我们不知道其中一个朋友的城镇。

你奶奶想去拜访她的朋友，想知道她要走多少英里。你能帮她弄明白吗？
