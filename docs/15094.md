# 你的“多元化雇佣”离开的真正原因

> 原文:[https://dev . to/angelamchunu/the-real-real-reason-your-diversity-hire-leaves-2op](https://dev.to/angelamchunu/the-real-reason-your-diversity-hire-leaves-2op)

与上面的标题相反，人们离开一家公司的原因有很多；薪酬、离家距离、*插入通用的糖衣离职理由*。但以我的经验来看，多元化雇员离开是因为公司并不像他们在面试中承诺的那样真正相信多元化。

一家公司会声称关心多样性，但休息室里的所有游戏主要是由男性享受的。所有高管都是男人，大多数活动都是为男人设计的，所有高管都与男人打成一片。这创造了一种明显为男性设计的文化，几乎没有给女性留下适应任何地方的空间。我并不是说有某种形式的睾丸激素报名费，我只是说这种文化符合 brogrammer。没有任何积极的努力去包含任何不是典型开发人员的人。

许多公司很少表现出偏离常规的意愿——如果它没坏，就不要修理它。他们还没有体验到多元化团队的真正效果。

我要用好莱坞来解释我的意思。让我们以布鲁克林九九为例，他们确实是一个多元化的演员阵容。每个角色都有一个同等重要的角色，他们的角色与他们的种族或性别无关。如果罗莎或霍尔特队长离开了，你会注意到，没有他们中的任何一个，一切都不一样了。他们可以制作一集，只有他们两个，这仍然很有趣。他们的不同真正增加了这部剧的幽默，从而使它变得伟大。

相比之下，如果你像《另一个女人》一样拍一部电影，我所爱的妮琪·米娜对这部电影绝对没有任何影响。你可以去掉她所有的镜头，这样电影就有意义了。她显然是为了一些配额去的，我想好莱坞已经开始强加了。许多声称关心多样性的公司最终都是这样。

在技术世界里，Nicki 的对等物应该是那些总是要做文档的人，用已经为他们写好的脚本得到一份报告。他们总是被赋予相同类型的世俗任务，而他们的同龄人则被督促或指导。他们被排除在设计会议之外，技术升级决策的制定没有他们的参与，只是整体上的微妙排斥。

多元化雇员梦想找到一份布鲁克林的朝九晚九的工作，但经常发现自己处于“另一个女人”的境地。基本上，你的多元化雇员意识到，他们应该得到更好的待遇，而不仅仅是一个时髦的黑人朋友，提供很好的建议和有趣的笑话。他们会离开，因为他们太有资格不被重视。他们将离开去追求一个挑战，这个挑战目前对他们的同龄人来说是一种奢侈。他们会离开，因为节目很恶心，没人想看。

除非公司真正理解多元化的好处或需求，而不仅仅是 BEE 分数的需求，否则他们很可能会一直雇佣多元化员工。
