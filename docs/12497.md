# 可变字体列表

> 原文:[https://dev.to/vuild/list-of-variable-fonts-4393](https://dev.to/vuild/list-of-variable-fonts-4393)

OpenType 可变字体于 2016 年推出，作为 OpenType 规范的扩展。从技术上讲，它们允许一个字体文件存储一系列连续的设计变体。简单来说，它意味着可以跳舞的字体。能够让你的文字变得栩栩如生是一件很新奇的事情&如果使用得当，一定会引起注意。
