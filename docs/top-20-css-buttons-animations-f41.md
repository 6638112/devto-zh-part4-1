按钮不仅有利于可用性，也是网站极其重要的设计元素。为此，这里收集了最好的 CSS 按钮！

无论是在你的主页上加粗还是在页脚小而不显眼，按钮对于网站的用户流来说都是一个非常重要的设计元素。对于一个公司网站，通常使用更谨慎的设计，而创意产业则使用更醒目和“怪异”的 CSS 按钮。让所有行业都得到同等服务，你会发现这里有很多不同的风格和组合。

我——以及许多其他 web 开发人员/设计人员——也对悬停或聚焦的动画设置了很高的值，这就是为什么下面所有的按钮都带来了很好的动画。但是现在我们直接开始！
