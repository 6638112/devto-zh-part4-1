# 不要让事实妨碍一个好故事

我第一次听说杰瑞·宋飞的“生产力系统”是在 2013 年。这个系统很简单:

1.  买一个大挂历
2.  在你完成任务(比如写作)的每一天画一个大大的红色 *X* 然后，在完成任务的几天后，你就只有一项工作了:
3.  不要弄断链条

虽然我从未买过挂历，但多年来我一直努力养成写作的习惯，我把杰里的习惯记在了心里。我写作的日子——那些是美好的日子。我不喜欢的日子——它们没那么好。它也在工作。今年是我有史以来最好的一年，就写作天数而言，上个月是我个人最好的一年。

这是真的，尽管我知道 Jerry 从来没有使用过这个系统，他被认为是这个系统的创始人是互联网知识的一个意外。

事实上，他并没有真正想出这个系统并不重要。故事因为有杰瑞做主角而变得更好。简单地定义一个系统(就像我上面做的)不会强迫人们。告诉他们杰瑞·宋飞用它来培养自己作为喜剧演员的技能。

这不是对说谎的认可。如果你有事实，就使用它们。但是如果观点值得一听，不要让他们妨碍你讲一个好故事。
