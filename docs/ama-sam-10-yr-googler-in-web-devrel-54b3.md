# AMA，山姆 10 年谷歌在网络发展

> 原文：<https://dev.to/samthor/ama-sam-10-yr-googler-in-web-devrel-54b3>

截至 2019 年 6 月，我已经在🇦🇺悉尼的谷歌工作了十年，从我大学毕业开始。所以我会通过回答你的问题来庆祝！🔟🎉

我是 Web 开发人员关系团队的一员。我们提供关于性能、工具、新的 web 特性，...你说吧！我们也做一些类似会议网站和内部工具的项目工作。我的目标是支持“长尾”🐍:我的视频、[演讲](https://www.youtube.com/playlist?list=PL5vCN4IjFTd40C6QRiHfs6h4MOdjbRo3B)、[博客](https://dev.to/samthor)、[帖子](https://medium.com/@samthor)等，都是以*网络平台*能为你做什么为主题的。🤔💬

* * *

**你应该问什么问题**？好吧，问问你的开发者关于网络的问题，尤其是它的未来！或者关于我参与的一些项目:

*   [**圣诞老人追踪器**](https://santatracker.google.com) :我是工程负责人，我们建造像圣诞老人这样有趣的东西来展示网络能做什么(因为圣诞老人！🎅🎄🎁)
*   [**Emojityper**](https://emojityper.com) :因为当你的工作是谈论 PWAs 的时候，有一个来演示是很好的！😂
*   **事件**:我从事 Chrome 开发峰会和 I/O 等活动🎫

还有一些过去的亮点，在 Web DevRel 之前——作为一名软件工程师:

*   地图:我在 iOS 版谷歌地图上工作，在它发布前六个月加入这个团队。对，那个 app！🍎📱

*   **Keep** :打造 [Google Keep](https://keep.google.com) 团队的一员。我记得根据每个🧓中有多少文本，构建了第一个具有不同字体大小的笔记原型

*   **波**:对，那波！🌊我建立了[波协议](https://youtu.be/v_UyVmITiYQ?t=4132)。如果你有一个小时(或者 30 分钟在 2x 上！)，整个视频是一个伟大的手表，即使多年后😎
