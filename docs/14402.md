# 巴塞罗那最活跃的软件开发社区

> 原文:[https://dev . to/Apium _ hub/best-active-software-development-communities-in-Barcelona-b59](https://dev.to/apium_hub/best-active-software-development-communities-in-barcelona-b59)

成为软件开发社区的一员比你想象的更重要。

您对编程社区的选择通常会决定您在特定语言或专业领域的进步速度。社区提供了独一无二的学习体验。编程社区可能会为您提供最新的信息和问题解决方案、有见地的答案和新观点、面向所有级别程序员的提示和技巧、资源链接、讲座和研究论文。但同样重要的是要强调的是，社区重新团结了代码伙伴、潜在的合作伙伴和志同道合的人。软件开发社区通过事件、开源项目和其他计划来推动改进和创新。

认识到有数百个在线和离线软件开发社区，我决定与 Apiumhub 开发者交谈，列出巴塞罗那最好和最活跃的社区。

**巴塞罗那最活跃的软件开发社区**

*   [软件工匠](https://www.softwarecrafters.org/)

软件工匠社区旨在吸引和联系软件开发专业人士，他们对自己的工作充满热情，并分享工匠运动的价值观和原则。这是一个社区，是为那些关心他们所做的事情，想要提高和掌握他们的手艺的人服务的。

*   [巴塞罗那足球俱乐部](https://www.barcelonajug.org/)

这是巴塞罗那的一个软件开发社区。BarcelonaJUG 召集了来自巴塞罗那的极客，他们希望分享与 IT 世界和 Java 语言相关的经验和知识。在这个社区中，你可以聊天、讨论和组织研讨会来实践和测试新技术。

*   [巴塞罗那。网络核心](https://www.meetup.com/netcorebcn/)

这个社区有开放源码的想法，并愿意探索开放源码的跨平台现代 web 开发。网芯。社区成员喜欢现代软件架构和设计，包括 DDD、CQRS、事件源、函数式编程、微服务、Docker、K8s 等等。

*   [领域驱动设计巴塞罗那](https://www.meetup.com/dddbcn/)

这个社区致力于领域驱动的设计。成员通过 4 种形式尽可能多地学习 DDD:现场编码、研讨会(实践/动手)、小组/公开讨论、问答(Q&A)和讲座。

*   [敏捷-巴塞罗那](https://www.meetup.com/Agile-Barcelona-Meetup/)

在这个社区中，成员们分享与敏捷以及在软件开发中使用敏捷方法相关的知识和经验。

*   [BarcelonaJS](https://www.meetup.com/BarcelonaJS/)

巴塞罗那。JS 是一个专注于 JavaScript 和相关主题的社区。他们每月在巴塞罗那市中心组织聚会，分享经验，讨论最新趋势。

*   巴塞罗那的反应

这个社区位于巴塞罗那，它是关于 React.Js 的。他们的目标是组织活动，涵盖 React.js、Flux 和所有其他与之相关的库等主题。

*   [巴萨安卓](https://www.meetup.com/Barcelona-Android-Developer-Group/)

该社区的使命是通过预定的会谈和视频会议，将来自巴塞罗那的 Android 开发人员聚集在一起，分享经验、知识，共同成长为专业人士。

*   [巴塞罗那詹金斯](https://www.meetup.com/Barcelona-Jenkins-Area-Meetup/)

Jenkins 是实现敏捷持续集成、持续交付和 DevOps 实践的首要开源技术。该社区将专业人士联系在一起，分享实践、建立关系网和探索高级主题。

*   [Scala 开发者](https://www.meetup.com/Scala-Developers-Barcelona/)

这个社区是为那些热爱 Scala 编程并希望在巴塞罗那地区与其他 Scala 开发者交流这种编程语言的人准备的。社区成员组织演示、编码 dojos 等。专注于 Scala 和函数式编程。

*   巴塞罗那码头

这个社区是为那些想了解更多关于 Docker 的人准备的。社区成员组织活动，了解如何使用 Docker 来确保所有分布式应用的敏捷性、可移植性和控制力。

*   [女性技术人员](https://www.womentechmakers.com/)

Women Techmakers 不断推出全球可扩展的计划和试点新项目，以支持和增强技术和软件开发行业中的女性。

*   [女性编码](https://www.meetup.com/Women-Who-Code-Barcelona/)

Women Who Code 是最大、最活跃的工程师社区，致力于激励女性在技术职业中脱颖而出。他们组织各种活动，免费参与技术活动、学习小组、小组讨论、闪电谈话和主题演讲，并邀请有影响力的技术行业专家、创新者和投资者参加。

[![Apium Academy](../Images/239906b081fca2bd3f9320bc5c00ce49.png)T2】](https://apiumacademy.com/)

*   [巴塞罗那自由软件](https://www.meetup.com/Barcelona-Free-Software/)

这个社区是为巴塞罗那的自由软件爱好者服务的。这个社区根据不同的场合为开发者和用户举办面向 FLOSS 的讲座。

*   [ns 巴塞罗那](https://www.meetup.com/nsbarcelona/)

这是一个非常棒的 iOS 和 OSX 开发者社区。这个社区大约每月聚会一次，讨论苹果的所有事情——从 Swift 和 Objective-C 编码实践，到操作系统的变化和最佳实践。

*   [软件测试人员& QA 专家德巴塞罗纳](https://www.meetup.com/BarcelonaTesters/)

这个社区面向在巴塞罗那从事软件测试和质量工程的专业人士。这个社区定期组织业余活动来分享关于软件测试的信息和经验，并促进巴塞罗那本地测试社区的发展。

*   [软件架构社区](https://www.meetup.com/apiumhub-software-architecture/)

这个社区面向软件架构爱好者，他们组织关于软件架构的聚会和活动，其中包括[全球软件架构峰会](https://gsas.io/)。健壮和可伸缩的软件是每一次讨论和谈话的中心，这使得它成为在软件开发世界中为质量而战的人们的理想场所。
