# 瞄准 Cloud Foundry 认证开发者(CFCD)认证？这里有一些提示。

> 原文:[https://dev . to/dchucks/aiming-for-the-cloud-foundry-certified-developer-cfcd-certification-here-some-tips-5292](https://dev.to/dchucks/aiming-for-the-cloud-foundry-certified-developer-cfcd-certification-here-are-some-tips-5292)

> 转载自[空指针](http://nullpointer.debashish.com/cloud-foundry-certified-developer-certification-tips)

我在 2019 年 3 月通过了 Cloud Foundry 认证开发人员(CFCD)认证，并想到分享一些技巧以造福他人。CFCD 是专业的云原生开发者认证。对于开发人员来说，这是一个很好的方式来展示他们一般的云技能和具体的云铸造专业知识。这是一个远程监考的基于性能的在线考试，除了回答一些 [MCQ](https://fullforms.com/MCQ) 问题(每个问题 15 个)之外，您只需要使用 Cloud Foundry CLI 来解决 Ubuntu 虚拟机上的问题。完成考试的总时间为 3 小时(相信我，这是必需的)。

> 不要脸的塞:你在 AWS、Kubernetes、Docker、Cloud Foundry 上的技术有多好？在 [TQ](http://tq.debashish.com) 检查你的技术商数。

# [](#recommended-steps-for-attaining-the-cfcd-certification)获得 CFCD 认证的推荐步骤:

通读:

*   [考生手册](http://www.cloudfoundry.org/wp-content/uploads/2017/01/CFFCertCandidateHandbook.pdf)
*   [考试标题](https://www.cloudfoundry.org/wp-content/uploads/2017/03/CFCD-Exam-Rubric.pdf)
*   [官方考试提示](https://cloudfoundry.org/wp-content/uploads/2017/01/Tips-for-CFCD.pdf)
*   [官方学习指南](https://www.cloudfoundry.org/wp-content/uploads/CFCD-Study-Guide.pdf)
*   使用 Cloud Foundry 获得实践经验，这是无可替代的。事实上，考试的重点是测试实践技能(被恰当地称为基于表现的考试)。

# [](#additional-tipsinformation)附加提示/信息:

*   请确保在参加考试之前运行[系统兼容性检查](https://www.examslocal.com/ScheduleExam/Home/CompatibilityCheck)，这样您就不会在考试过程中遇到问题。你需要一台带摄像头的笔记本电脑。
*   确保在这 3 个小时里保留一个相当隔离的地方。你的工作站应该是干净的，监考人员甚至不允许在桌子上放一个水瓶，他们会要求你摇动你的网络摄像头来展示房间，甚至桌子底下:)。也摆脱任何可穿戴设备。
*   请随身携带政府颁发的带照片的身份证明，以供核实。
*   请注意，如果出现任何技术问题，Linux 基金会可能会取消您选择的考试时间。但是他们会提前通知你。如果你在旅行，在考试前一定要检查你的电子邮件。
*   考试结果只提供了总体通过率，但没有按主题细分的表现。
*   您应该熟悉 Java、NodeJS 或 Ruby 的应用程序开发(exam 提供了使用其中任何一种的选择),并且能够熟练使用基本的 Unix 命令和文件编辑器。

使用以下补充材料进行准备:

*   [PCF 学习指南&复习题](https://djitz.com/tag/pivotal-cloud-foundry/)(免费)
*   Linux 基金会的 LFD232:面向开发者的云计算平台( 299 美元)
*   Linux 学院的 [CFCD 课程](https://linuxacademy.com/devops/training/course/name/cloud-foundry-certified-developer)提供的模拟测试(基于订阅)
