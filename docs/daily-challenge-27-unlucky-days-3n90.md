# 每日挑战#27 -不幸的日子

> 原文：<https://dev.to/thepracticaldev/daily-challenge-27-unlucky-days-3n90>

13 号星期五或黑色星期五被认为是不吉利的一天。计算一年中有多少个不吉利的日子。

你能找到给定年份中 13 号星期五的数字吗？祝你好运！

输入:整数形式的年份。

输出:一年中黑色星期五的整数个数。

例子:

```
unluckyDays(2015) == 3
unluckyDays(1986) == 1 
```

**注**:红宝石年将从 1593 年开始。
