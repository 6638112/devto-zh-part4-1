# 总是阅读评论

> 原文：<https://dev.to/ionic/always-read-the-comments-3pfk>

“永远不要看评论”是一句熟悉的格言，告诉制造产品的技术人员。

有道理，对吧？用户通常只在他们经历了非常糟糕或积极的经历后才留下反馈。

然而，如果你把网上的评论框定得正确，阅读这些评论是有价值的。

## 确定新产品冠军

与热情推销你产品的人建立牢固的关系是发展业务的一个重要策略。你经常可以在评论里找到它们！超越传统的“评论箱”,看看社交媒体、论坛、闲散小组——任何社区聚集的在线空间。他们回答关于你的产品的问题，分享你的内容，并在会议上找到你。

一旦你确定了他们，许多机会就会出现。公开表扬他们对社区的贡献。送他们礼物。让他们尽早接触新产品。

你为支持他们付出的任何努力都会得到十倍的回报。

## 获得产品洞察力

忽略明显有害的评论，你会发现新的见解。负面评论通常是沮丧的表现:这个人希望你的产品成功，但是遇到了问题，所以在时刻做出反应*。*

这里的关键是要超越愤怒。设身处地为评论者着想:他们想告诉你什么？这种方法让我发现了我工作过的每家公司产品中的许多差距。

## 提高你接受反馈的能力

由于匿名性，在线评论可能是你收到的最糟糕的反馈。然而，学会忍受它是一种建立心理韧性的低摩擦方式。摆脱负面反馈的能力对个人和职业发展非常有价值。

在一个理想的世界里，每个人在任何时候都是尊重的。现实是永远不会这样。当然，如果你发现自己被一个特别下流的评论淹没了，休息一下，继续前进。
