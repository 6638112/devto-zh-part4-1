# 显示您自己的/使用页面

> 原文：<https://dev.to/nikoheikkila/show-your-own-uses-page-160j>

你有没有在 Twitter 或其他社交媒体上发布一段代码，然后立刻被询问你使用的酷字体或编辑器主题的人包围？我知道。你有没有变得特别喜欢某些工具，并且喜欢向全世界赞美它们的荣耀？我有。

受 [@wesbos](https://dev.to/wesbos) 和 [@aspittel](https://dev.to/aspittel) 的启发，我发布了自己的`/uses`页面，解释我在日常 web 开发人员生活中使用的生存工具。查看我的页面 [**这里**](https://nikoheikkila.fi/uses/) 。

如果这还不是一场运动，那就让它成为一场运动吧！在评论中发布你自己的热门`/uses`页面。
