# Chrome 插件-关闭我的标签

> 原文：<https://dev.to/kambanthemaker/chrome-plugin-closemytabs-3ka1>

我曾经打开过多的浏览器标签，然后发现很难关闭它们。当我想关闭旧的插件时，很多插件都把重点放在了暂停标签上。由于有太多的浏览器标签，我构建了这个来查看和关闭基于最后活动时间的浏览器标签。

## 特性

1.  根据上次访问时间关闭多个选项卡
2.  可调持续时间
3.  徽章计数
4.  基于标题或 URL 进行搜索的搜索栏
5.  从列表中打开标签
6.  忽略特定选项卡
7.  多窗口支持
8.  上次访问的详细信息
9.  便于识别的图标

在谷歌 Chrome 网络商店下载: [CloseMyTabs](https://chrome.google.com/webstore/detail/closemytabs/pjjdjbdoipimhkcfidhnekpfcgahfjca)
