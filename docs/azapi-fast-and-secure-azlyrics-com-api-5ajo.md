# azapi:快速安全的 AZLyrics.com API

> 原文：<https://dev.to/elmoiv/azapi-fast-and-secure-azlyrics-com-api-5ajo>

很多 AZLyrics 的 API 会导致用户被禁止，所以我采用这种方法，使用代理和随机用户代理来避免被禁止。

亮点:

*   获取艺术家的歌曲列表和专辑、年份...等等
*   将歌词保存为. txt 文件或任何你喜欢的格式。
*   使用代理和多用户代理避免 BAN。

我为`python >= 3.4`发布了 100%构建成功的版本
