## [](#styled-components)样式组件

浏览器控制台上有一些错误。
我又谷歌了一下，一些文章说使用`.babelrc`。
所以我添加了`.babelrc`，它解决了这个问题。

```
{
"presets": ["next/babel"],
"plugins": ["styled-components"]
} 
```
