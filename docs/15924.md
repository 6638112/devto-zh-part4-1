# 如何给别人的“杯子”续杯？

> 原文:[https://dev . to/cubiclebudha/how-to-fill-someone-s-cup-13 in](https://dev.to/cubiclebuddha/how-to-refill-someone-s-cup-13in)

有时我们太依赖我们的朋友和同事，以至于我们经常忘记确保他们得到他们所需要的。这里有几个快速补充能量的小贴士。

B 型人格通常是非常好的倾听者。事实上，有些人把 B 型人格描述为与人相处时给予能量的人，把 A 型人格描述为从与人相处中获得能量的人。如果是这样的话，那么没有人有无限的能量供应。

所以不管你是什么性格类型，尽你所能给那个人斟满酒:

1.  **直接提出听他们的。**为什么要直接？因为有些人会被一连串的问题冒犯，即使你只是“试图成为一个好的倾听者”所以，主动提出倾听他们，你就给了他们说“不，谢谢”的机会或者，如果他们说“是”，那么他们可以完全自私地利用这段时间。
2.  给他们买个小礼物表示感谢。我会带一块糖、一杯咖啡，甚至一份水果。不，我不认为钱能解决一切，但有时一个简单的手势表明你在想他们。如果他们感到有联系，这可能足以扭转他们的一天。
3.  减轻他们的负担。如果是同事，为他们承担一点工作，这样他们可以休息一会儿。只是要小心先问他们，因为你选择的任务可能是他们真正感兴趣的。此外，透明度很重要，因为它再次给了人们说“不”的机会。例如，新父母被告知在最初几个月要非常清楚他们想要什么样的帮助。但是你必须直截了当，精确，清楚地表达你的意图。无论是新父母还是祖父母，都不想对这一发现感到惊讶，那就是通宵的拜访毫无帮助。你所要做的就是以一种让你的朋友可以轻易拒绝的方式询问是否需要帮助。
4.  别管他们。下次你认为你要通过和他们说话来“从他或她的水库中取用”时，问问自己是否真的需要社交。[如果你需要帮助来关闭你多嘴的大脑，我最近写了另一篇文章，描述了如何避免喋喋不休的技巧。](https://cubiclebuddha.com/?p=204)但简而言之，如果你从一个听你说话的朋友那里获得了一些东西，不要让他们听你的话到筋疲力尽的地步。
5.  与他们分享你知道他们会感兴趣的东西。可以是一篇你认为他们会感兴趣的简单文章。我发现给某人发送链接(只要你不强迫他们阅读)是一种向某人表达你对他们的想法的低承诺方式。
6.  加分点:真诚地说“谢谢”就行了。没有什么能像让一个人知道他们的时间投资是值得的那样，填补一个人的情感储备。

给你的朋友斟满酒是建立健康社区的一部分。毫无疑问，你们两个都会因为你表现关心的行为而感觉更好。
