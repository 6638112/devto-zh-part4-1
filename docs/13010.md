介绍

有兴趣创建自己的网站吗？有没有想过拥有一个具有登录/注销功能的网站有多酷？想学习如何在数据库中创建、读取、更新或删除(CRUD)记录？你是否已经忘记了之前的教程？好吧，我将教你如何从头开始创建一个，在那里你将知道代码如何工作的每一个细节。如果你是后端 web 开发的新手，这篇教程是为你准备的。我会详细解释每一件事，这样你就不必研究一些正在使用的特殊方法。我们不会使用任何框架来简化事情。此外，我不会把重点放在网站设计上，因为我们追求的是功能，但实现设计很容易。当用户登录时，将做的是一个简单的项目列表。
