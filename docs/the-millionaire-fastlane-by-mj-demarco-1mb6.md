# MJ·德马科的《百万富翁浪子》

> 原文：<https://dev.to/sandordargo/the-millionaire-fastlane-by-mj-demarco-1mb6>

[MJ DeMarco](https://www.mjdemarco.com/) 在[他的书](https://amzn.to/2XHPsZ7)中提出要*【破解致富密码】*让你*【富活一辈子】*。这听起来像个诱饵，不是吗？

太多时候，这些书是由那些过着体面生活的人写的，但仅此而已。他们一点也不富裕。这根本不是问题，没有人必须成为富人，但是如果你写如何变得富有，你宁愿成为富人。

德马科很富有，他是靠自己成为富人的。

他真正讨厌的是那些慢慢变富的计划。每月存几块钱，到你 80 岁时，由于复利，你会变得富有。他不觉得这有吸引力，更不用说性感了。

不是说他不重视复利，而是说这不是致富的好方法。一旦你成为富人，让你呆在富人区真的很有用，但要到达那里实际上毫无用处。

他区分三种类型的人。

# 步行者

他们是不太了解钱的普通人，不管他们挣多少钱，他们都会花光。在最糟糕的情况下，他们花费甚至超过他们拥有的，并建立了一个庞大的债务。侧向者是一群对变得富有或拥有健康的财务状况没有太大希望的人。

# 慢吞吞

慢行族是那些明白他们必须存些钱的人。他们必须定期储蓄一些钱，以便有一天他们可以退休，然后他们可以相对富裕地死去。问题是，你必须大量工作，在大多数情况下，一周工作 5 天才能得到属于你的 2 天，致富之路不仅漫长，而且也不那么有趣和节俭。先不说这两天你做了多少家务之类的事情。

我们中的绝大多数人要么是步行者，要么是慢行者。事实上，德马科声称，在某些时候，甚至很多慢性子的人会变成慢性子，因为他们厌倦了努力工作和为未来存钱。

我自己也是一个慢行者，我时常觉得自己有成为一个慢行者的趋势——至少不是一个衰弱的人。幸运的是，对于我的妻子，我们对长期财务有着相似的愿景，这有助于保持控制。就像我们用来积累储蓄和投资的系统一样。

顺便说一下，德马科并没有说做一个慢性子是不好的。他说，如果你真的喜欢你做的事情，那完全没问题，但你必须意识到这一点。你不必变得富有。顺便说一下，有钱并不一定意味着你住在豪宅里，像吝啬鬼一样用金币洗澡。这只能意味着你有时间和金钱去做任何你想做的事情。它可以简单到一年带你的父母去度假一次，邀请你的朋友去吃大餐。

# 飞毛腿

他们是那些懂得致富公式的人。根据德马科的说法，这个公式相当简单。你应该把财富和时间分开！这次离婚意味着什么？这意味着，例如，为小企业建立越来越多的网页将不会超过一定的规模。假设你在一周内构建了一个，并收取 2k 的费用——这是否现实并不重要。这样的话，你一年可以赚 10 万左右。你可以减少你需要建立它的时间和/或增加你的费用。但两者都有极限。这不是一个可扩展的模型，你不能变大。你不可能像这样赚几百万。

相反，你应该专注于开发有巨大潜在市场的产品。产品不会为你带来更多的工作，但会带来更多的利润。重要的是你能在规模和程度上影响多少人的生活。你能卖给越多的人越贵的产品，你就越富有。

你要么为数百万人服务，要么只是少数人，但数量很大。

然后你就可以用赚到的钱通过安全的投资和复利来赚更多的钱。

当然，事情不仅仅如此。他解释了不同形式的公司的一些技术细节，并提出了用他的话说，为了在生活中获得成功，什么是有用的性格特征。

# 结论

变得富有很简单，但不容易。这也不一定是每个人的目标。另一方面，理解它背后的数学和原理是有用的。

我真正喜欢《百万富翁浪子》的一点是，它是由一个靠自己致富的人写的。顺便说一句，这不是某个一点也不富有的人破解的公式，但是当然，如果他想变得富有，他也可以变得富有...它的风格是令人愉快的，它不是很长。虽然它重复了我们许多人已经知道的生活中的一些重大事实，但我认为这没什么。被提醒你正在掌控你的生活总是好的，你将会得到什么是你唯一的责任。

你想在生活中获得什么？
