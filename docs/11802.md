# 我的自学之旅(目前为止)

> 原文：<https://dev.to/pachicodes/my-self-learner-journey-so-far-3nm6>

大约 10 个月前，我开始了我的编码之旅。我的哥哥是巴西(我的家乡)的一名开发者，他决定将我和我的弟弟变成开发者。看了很多机器人先生后，我非常兴奋！让黑客来吧，我告诉他 lol
但是他所做的只是在 HTML 和命令行上分享一些代码学院教程。完全没有黑客行为！=/

但是，当我开始从头开始创建我的第一个非常简单的 HTML 页面，并为它们添加 CSS 时，我感觉非常棒。HTML 和 CSS 是我最喜欢的组合！我还学习了 UI 的基础知识，响应式编码和一点更高级的 CSS(我喜欢 CSS ),这些都是在前 6 个月，每天学习不超过 45 分钟到 2 小时，通常每周 6 天。

然后我发现我心爱的 HTML 和 CSS 并不是真正的编程语言:((仍然爱着它们，尽管< 3)，我的目标是(现在仍然是)成为一名优秀的开发人员，所以我开始催促我的导师，也就是我的哥哥，让我学习 Javascript。但是他却让我去学计算机逻辑！如果你是一名程序员，你可能会认为哥哥是明智的，因为这是一个好计划。但是我不同意你看...我想编程！！！尽管如此，我还是看着他和我一起上的课(也许没有给予他们应有的关注)直到结束。

最后他给我分享了一个 JavaScript 课程！！！我太兴奋了！！！第二天我醒得更早(我是一个早起的人，晚上学习对我来说不是很好)来开始课程...我一点也不喜欢！哦，骗局。我觉得自己很失败。我根本无法集中精力听课，说实话也没学到多少东西。但是弟弟做到了。我不能放弃！

当我们到达课程的某一点时，大哥让我们停下来做一个项目。我当然很害怕，但最终，在他的伟大指导下，我做到了:
[https://pachicodes.github.io/Products/](https://pachicodes.github.io/Products/)

 I was SOOO happy when it was done, even if I wasn't even that sure how I made somethings work, they were working!

在那之后，我们开始了另一门 JS 课程，老师和伙计都不一样，差别太大了！！！我更快地完成了那门课程，学到了很多东西。这位新教授的教学方法对我来说要好得多，我真的很想学！
现在，我把这件事说得好像是很久以前的事了，但实际上是上个月(2019 年 6 月)的事了。所以，是的，我几乎是个新手。但是我可以用 JavaScript 编写一些漂亮的静态页面和一些基本的很酷的应用程序。

学习编程有三种主要方式:大学、训练营和自学。

大学从来就不是我的事；我喜欢训练营的想法，但是在我人生的这个阶段，时间承诺和财务投资对我来说不合适。自学是我的选择。它不像大学那样组织有序，也不像训练营那样快，但有了导师、计划和奉献精神，它对我很管用。

在我的下一篇文章中，我会写一些让自学变得更容易的事情。
