# 重塑你的 IT 简历，获得更多面试机会

> 原文：<https://dev.to/stetsenko_me/facelift-your-it-resume-to-get-more-interviews-2mih>

通过对你的 IT 简历做一些小的改进，你可以将面试的次数增加几倍。相信我:我每天都会看到几十份科技简历，其中很多都缺乏一些小小的升级来给他们的主人带来更多的面试和工作机会。

这些升级包括:

1.  专注于你最近的项目。你过去 3-4 年的经历是招聘人员最感兴趣的。不要详细描述更老的项目。
2.  用数字和成就充实你的简历。不要写“我们正在用[X]和[Y]构建一个 web 应用程序”，而要写“领导了[X]功能的开发，将它集成到[Z]个产品中，带来了额外的[Y]收入”。
3.  避免“虚假的成就”。这些是你的职责，用过去式描述。除非你表现出影响力，否则它没有意义。
4.  **包含相关关键词。仔细阅读职位空缺，在简历中加入你所拥有的所有必要技能。招聘人员会将你的技能/头衔与他们的职位相匹配，以及你在特定领域(SaaS、电子商务、金融科技等)的经验。**
5.  建立并展示网上形象。你在 GitHub、StackOverflow、HackerRank 等上的活动。展示你的技能，增加你被考虑从事某项工作的机会。如果你有一些值得注意的简介和链接，把它们放在你的简历里。
6.  不要用模糊的百分比和/或类别来给你的技能打分。在技术面试或测试任务中，你的潜在雇主会客观地评估你的技能和知识。至于简历中的自我评价，往往会导致应聘者和招聘人员之间的误解。
7.  检查你简历中的错别字。即使是 Jquery 或 Nginx 这样的技术的不正确拼写也会让你的简历看起来不专业。
8.  省略总结和目标部分。在 10 个案例中，有 9 个案例的总结和目标并不令人印象深刻。与其浪费时间创建定期总结，我建议省略它。
9.  使用现代字体。永远忘记 Comis Sans 和类似的字体。相反，尝试现代字体，如 Palanquin，Merriweather，Lato 和 Poppins。
