# 实际 10 倍的工程师

> 原文:[https://dev.to/elmuerte/actual-10x-engineers-n22](https://dev.to/elmuerte/actual-10x-engineers-n22)

本月早些时候，法布里斯·贝拉发布了 T2 的 QuickJS。这是一个 Javascript 运行时，几乎是 ES2019 完成，你可以嵌入到你的应用。这不是一个小壮举。

在过去的一周左右，有很多关于 10x 工程师的讨论。虽然 10 倍工程师的概念在许多方面都有缺陷，但还是有一些软件开发人员有着令人惊叹的工作记录。

Bellard 发布的 QuickJS 在 hackernews 上引发了一场关于伟大开发者的讨论，这些开发者完成了惊人的工作。

# [](#fabrice-bellard)法布里斯·贝拉

让我们从法布里斯开始。他的[网站](https://bellard.org/)列出了他创造/开创的伟大事业。他的工作不仅仅局限于一个领域。

他创建了 FFmpeg，这是很多音频视频软件的基础。很明显，现在有一个完整的项目团队在不断改进这一点。

Bellard 还创建了 QEMU，一个通用的机器模拟器和虚拟器。它一直是虚拟机主机的主要参与者。

此外，还有一大堆其他项目，如 LTE/NR 基站、JavaScript PC 仿真器、ASN1 编译器等。

# [](#dan-j-bernstein)丹·伯恩斯坦

又名 [djb](https://cr.yp.to/djb.html) 。他不仅起诉了美国政府关于[围绕密码术](https://en.wikipedia.org/wiki/Bernstein_v._United_States)的出口法律，并且赢了，他还创造了许多伟大的软件。

他的主要领域是密码学。创建算法/密码，如 Sals20、ChaCha20、Curve25519。以及一大堆研究。

在他的软件简历上有很多真正安全的软件，尽管多年没有任何更新，这些软件仍然经常被使用。它包括: [qmail](https://en.wikipedia.org/wiki/Qmail) ，一个 SMTP 服务器；djbdns，一个 dns 服务器；一个邮件列表服务器。

qmail 可能是最值得注意的。它已经很久没有收到重大更新了。只发现了几个漏洞，没有一个被认为是安全漏洞。

# [](#linus-torvalds)李纳斯·托瓦兹

可能是这份名单中最知名的人。他开发了 Linux 内核，并且仍然在积极地开发和管理这个项目。

但是他也创造了 Git T1，因为他需要一个适合他需求的分散版本控制系统。

# [](#richard-stallman)理查德·斯托尔曼

可能最出名的是他的激进主义。但是当他最初开始 GNU 项目的时候，在他的 T2 自由软件的概念中并没有很多他可以使用的软件。

他必须编写自己的编译器(后来成为 GCC)、自己的调试器(GDB)、标准 C 库(glibc)以及 GNU 用户世界中的一大堆其他部分。他还创建了 GNU Emacs。

# [](#ludvig-strigeus)路易·斯特里特

名单上的一个年轻人。

他是 LucasArts Scumm 游戏引擎的重新实现[scumvm](https://www.scummvm.org/)的原始开发者之一。这些天来，scumvm 已经发展成为一个通用的 VM，用于各种旧的图形冒险游戏，而不仅仅是基于 Scumm 的游戏。

Ludvig 还启动了 OpenTTD 项目，这是克里斯·索耶视频游戏《运输大亨豪华版》的逆向工程实现。社区继续他的工作，OpenTTD 现在可以作为一个完全独立的视频游戏使用。

MS Windows 上最受欢迎的 BitTorrent 客户端之一可能是 uTorrent。这个客户端也是 Ludvig 写的。

然后就是这个叫 Spotify 的东西。Ludvig 从一开始就在那里开发这项技术。
