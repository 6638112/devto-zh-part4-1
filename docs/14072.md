# 完整机器学习生命周期的平台

> 原文:[https://dev . to/codegeekaz/platform-for-complete-machine-learning-life cycle-a0l](https://dev.to/codegeekaz/platform-for-complete-machine-learning-lifecycle-a0l)

ML 开发带来了许多超越传统软件开发生命周期的新的复杂性。与传统的软件开发不同，ML 开发人员希望尝试多种算法、工具和参数来获得最佳结果，他们需要跟踪这些信息来重现工作。此外，开发人员需要使用许多不同的系统来生产模型。为了解决这些问题，许多公司正在构建定制的“ML 平台”来自动化这一生命周期，但即使是这些平台也仅限于少数受支持的算法和每个公司的内部基础设施。

在本次会议中，我们将介绍 MLflow，这是 Databricks 的一个新的开源项目，旨在设计一个开放的 ML 平台，组织可以使用他们选择的任何 ML 库和开发工具来可靠地构建和共享 ML 应用程序。MLflow 引入了简单的抽象来打包可复制的项目、跟踪结果并封装可以与许多现有工具一起使用的模型，从而为任何规模的组织加速 ML 生命周期。

通过一个简短的演示，您将看到一个完整的 ML 模型生命周期示例，您将带着它离开:模型、实验和项目的 MLflow 概念和抽象如何在模型训练期间使用跟踪 Python APIs 开始 MLFlow 使用 MLflow UI 直观地比较和对比具有不同调优参数的实验运行并评估指标
