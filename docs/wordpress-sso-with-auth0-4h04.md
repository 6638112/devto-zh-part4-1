# 带有 Auth0 的 WordPress 单点登录

> 原文：<https://dev.to/auth0/wordpress-sso-with-auth0-4h04>

Auth0 是一个身份平台，提供全面的身份认证解决方案，覆盖各个市场领域的客户。我们最受欢迎的功能之一是单点登录，简称 SSO。这项功能让用户只需登录一次，就可以访问不同的网站、手机和网络应用程序。

为了通过 SSO 的灵活性和强大功能来帮助保护世界上最受欢迎的 web 平台之一，Auth0 为 WordPress 开发者提供了 Login by Auth0 插件，该插件在任何其他平台上的不同 WordPress 安装和应用程序之间提供 SSO。

对于刚接触 SSO 的开发人员和站点管理员来说，SSO 的概念可能会有点混乱。所以在这篇文章中，我们将探索与 SSO 相关的不同概念，以及我们的 WordPress 插件如何帮助你。
