# 善良很重要:传播正能量的提醒

> 原文:[https://dev . to/sarahscode/kindness-matters-a-reminder-to-spread-正能量-1m5o](https://dev.to/sarahscode/kindness-matters-a-reminder-to-spread-positive-energy-1m5o)

**注意:这篇帖子涉及精神健康、自杀和轻微的网络欺凌。如果这些话题中的任何一个对你来说很难读懂，你可能想跳过这个博客。给每一个人送去大量的爱，不管你是否选择阅读这篇文章。**

我试着将我的开发生活和我的个人生活分开(虽然它们确实融合在一起了，只是因为我喜欢分享我的生活如何影响我的开发体验，反之亦然)，但本周我必须将两者结合起来，因为除了最近在跑步社区爆发的丑闻，我很难思考任何事情。上周，人们发现一名男子可能在洛杉矶马拉松比赛中作弊，以创世界纪录的时间赢得了他所在年龄组的冠军。几天后，他被发现死亡，死因被确定为自杀死亡。在我的跑步团体中，这被普遍认为是一场悲剧。不管这个人的行为是否合乎道德，他是一个人，他的死是悲惨的。这也导致了一些分歧和许多关于精神健康的讨论，以及鉴于作弊丑闻，网上对他的评论是否可能促使他决定结束自己的生命。讨论结束时，我们都同意善良很重要。

我们不知道是什么让这个人认为自杀是他最好的选择。我们不知道网上说的不厚道的话能不能有贡献。我们不知道他的任何朋友和家人最近是否检查过他，以了解这一丑闻如何影响他的心理健康。我们能做的就是以此为戒，在世界上传播正能量。

我们不能总是阻止自己去想不友善的想法。当有人欺骗或撒谎时，生气并不反常，尤其是当欺骗影响到你的时候。仅仅因为你有这些想法或感觉，并不意味着你需要把那些消极的能量传递到这个世界上。通常保持沉默比传播消极情绪更好。另一方面...一定要分享善意的想法。如果你看到某人擅长某事，告诉他们你印象深刻或感到自豪。如果你喜欢某人的服装，赞美他们。如果有人为你做了好事，感谢他们的帮助。如果你对某人有好感，分享出来。如果你只是想发一串关于你的朋友有多棒的微博，那就发吧。向世界传递更多的积极。

另一个有助于传播积极情绪的重要方法是和你的朋友保持联系。这不必是一个漫长的过程，只需要给他们发一条简短的短信，说“我知道你经历了很多，我想让你知道我在想你。”如果你有空和一个正在挣扎的朋友出去玩(或者只是电话长谈)，让他们知道。尊重自己的需求和时间安排(毕竟不先帮助自己就无法帮助别人)，但如果能为了朋友而在，那就去做吧。有时候，只是坐下来和朋友喝咖啡就能让你的两天都大不相同(因为通常你从帮助别人中获得的满足感会给你的一天带来精神上的提升)。如果你知道你的朋友患有精神疾病，或者在一年中的某些时候(例如，在假期或亲人去世的周年纪念日)，这一点尤其重要。当你伸出手告诉别人你在想他们的时候，你总是被感激的。

鼓励和支持你周围的人也很重要。对每个人来说，鼓励和支持看起来是不同的(你觉得鼓励的事情可能不会对其他人产生影响)，但是找到一种方法来支持你的朋友(有时是随机的互联网陌生人)，并向他们展示你有多相信他们，这是一种向世界传递积极能量的好方法。很容易感到孤独和不被支持，尤其是在经历困难的时候，一句“你能行的！”可以帮助人们度过难关。(这是我从跑步中学到的另一个教训，任何时候你看到有人在跑步或比赛中挣扎，总会有另一个跑步者或观众为他们加油，鼓励他们继续前进——我多次受益于此。)

每一天(有时是每一分钟)我们都必须决定我们是要把积极还是消极带到这个世界上。有时候消极是必要的，但是当有选择的时候，传播积极是很重要的。我们都希望世界变得更美好，而这必须从我们开始，分享美好的事物。传播积极性...成为你想在这个世界上看到的正能量。
