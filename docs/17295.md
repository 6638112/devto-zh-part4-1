# 成为社区的一部分

> 原文:[https://dev.to/vblazenka/being-a-part-of-community-2m6l](https://dev.to/vblazenka/being-a-part-of-community-2m6l)

很长一段时间以来，我都想写一些关于社区、聚会和公共演讲的东西。最近我找到了一个做这件事的借口，这是我系列文章中的第一篇，献给每一个需要动力迈出成为社区一员的第一步的人。

最近，我收到了一个提到公开演讲的帖子，我不得不回应，因为我记得我第一次在当地集会上做公开演讲，以及它对我的意义和此后对我的影响。

我的回答是:

> 2016 年，我在当地的 meetup 上做了我的第一次公开演讲，这给我的职业生涯带来了巨大的变化，因为我接触了很多很酷的人，扩大了我的网络，并获得了不错的邀请。从那时起，我有了更多类似的工作，通过这些工作，我学会了如何更好地沟通，学会了如何更有效地学习和教学，更好的演讲技巧反映了我如何与经理和客户交谈，从而获得了更好的工作职位。直到今天，在演讲前我总是很紧张，我仍然会犯很多错误，但演讲后的感觉总是值得回来。

但是我忘了说，现在我是我举行第一次演讲的同一个聚会的共同组织者，我还帮助其他一些很酷的项目，例如 [NodeSchool](https://nodeschool.io/zagreb) 。

## [](#interested-in-giving-a-talk)有兴趣做个讲座？

我想鼓励每个人打开一个 meetup.com，搜索你所在城市的聚会，并试着去拜访他们几次以了解他们。试着找到组织者，问他们一些他们想听的问题。记住，一些你过去解决过的难题，试着从中找到一个主题，并应用到聚会中。

最近，在我们的 JavaScript 萨格勒布#49 meetup 上，一位同事讲述了他如何使用无服务器、无节点和 AWS 编写了一个脚本，帮助他在萨格勒布找到了一套公寓。观众的反馈是积极的，他们中的大多数人喜欢这个演讲，因为它简短而实用。记住，人们不能长时间集中注意力，如果你讲太多细节超过 25 分钟，他们会感到厌烦。

如果你展示的所有东西都能在 15 分钟内被在线阅读，那就不要做演讲。

## [](#interested-in-attending)有兴趣参加吗？

即使你没有说话(你应该尝试)，成为社区的一部分也是一种很好的感觉，因为你会认识新朋友，学习新东西，增加你的同理心，稳定你的自我。同理心是一种理解和感受他人的能力。没有比和其他人在一起并练习这些技能更好的方法来提高你的同理心技能。
自我是另一件大事。我们都知道自我是如何让我们感觉不好或者做错事的。想象一下，你周围有很多聪明的人，他们可以作为你在职业生涯中所处位置或你拥有多少知识的参考
。这些信息可以帮助你与他人进行比较，你可以看到你需要朝哪个方向前进，你需要提高哪些技能。

> 不要害怕别人根据你的知识来评判你。没有人什么都知道。没人知道他们在做什么。你可能知道他们不知道的事情！；)

## [](#introverted)内向？

我以前也坐过这趟火车，但请记住，战胜某种事物的唯一方法是直面它并取得胜利。有时我会避免去拥挤的地方，试图独处，和别人在一起我永远不会感到舒服，以至于我认为我不喜欢人。但当我开始工作时，它迫使我与我的同事和客户交谈，我爱上了它，做第一次公开演讲让一切都变得不同。现在我无法想象我以前是谁，我想更多地和人们一起工作，我每天都在这样做。这个帖子也是我尝试和更多人交流的方式。
