我们非常自豪，Franziska 同意成为 JS Kongress 2020 的主题演讲人，并期待着一场鼓舞人心的演讲。

我们先问了她几个问题，这样你就可以了解她了:

**你为什么期待 JS Kongress？**

我有幸在 2016 年第一届 JS Kongress 上发言。那时候已经是一个梦幻般的会议了，我真的很期待再次参加这个活动。慕尼黑是一个可爱的城市，当然，我希望那里会有一些威斯敏斯特香肠和炸薄饼。

**你对我们 2020 年的口号“扩展 JS——挑战极限:大规模平台、数据驱动架构和现代 API”有什么看法或联想**

JavaScript 在过去的 20 年里取得了长足的进步——甚至 NASA 也用 JavaScript 编写代码。不仅语言进化了，有人知道吗？)，还有 JavaScript 引擎(即时编译器)，在我看来，最重要的是工具和框架(这里的 vim 长期用户更喜欢 VS 代码)。我们可以用 JavaScript 解决复杂的问题，因为它快速且可维护。有了无服务器和云中的功能，我们几乎不需要担心可扩展性。我很期待看到 JavaScript 还能做些什么。

在你返回慕尼黑期间，你会去参观一些慕尼黑最受欢迎的景点吗？有知情者分享给与会者？

市中心的英国花园很棒。尤其是沿着 Isar 再往北一点。
