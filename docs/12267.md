# 当你尽了最大努力(在你已经擅长的方面)却没有成功时...

> 原文：<https://dev.to/briwa/when-you-try-your-best-at-what-you-re-already-good-at-but-you-don-t-succeed-3if>

我周围的一些人知道我非常喜欢篮球。我的确是看着《灌篮高手》长大的(这是我一直以来最喜欢的动画之一)，但我并不怎么喜欢这项运动，是个电视迷，甚至不是个运动员。我在大学的时候才开始学习和实践。即使这只是最基本的:射击。记得学长跟我说的一件事；如果你能把球投好，你就可以认为自己足够优秀了。这句话不知何故在我的脑海里停留了多年。

这是真的。我确实在某个时候变得很擅长射击。我和我的同学们一起玩，当我拿到球时，他们开始尊重我，紧紧地保护我。这基本上是我的奖励，是我坚持下去的动力。每当我们练习的时候，除了射击，我对别的什么都不感兴趣。每当我自己练习的时候，大部分时间也只是练习投篮。我是说，那是我擅长的，不如专注于此。

然后更大的场景来了。我和不同的人打过球，全场 5 对 5。这是一个完全不同层次的篮球，我大吃一惊。我几乎拿不到球，即使我设法投篮，我也很少能投进，这太激烈了。每个人都开始告诉我这个和那个。我没有很好的耐力，我应该传球更多，我应该防守更好。我对自己说，他们不明白；*你们这些家伙*应该把球给我，突破防守，我本可以得分更好的！我知道我在做什么！

我是对的。在练习中，我可以投篮。我在实际比赛中错过的相同地点重复了相同的投篮，我可以投中。然而，在比赛过程中，我的投篮没有再下降。我的队友开始传球给别人，而不是我。一开始我很困惑，但后来我想，管它呢，我不会再玩了。我是说，这很令人沮丧。我已经练习过了，我已经表现出了很大的进步迹象，我尽了最大努力，但这还不够。总有其他人要求的东西。我的拍摄很成功，为什么他们不能欣赏我能做的，而欣赏 T2 的努力呢？

* * *

在那个时候，我已经学了至少 5 年的篮球，尽管它并没有那么激烈。大概每周一两次，时断时续。然后我突然想到。5 年...而我只知道怎么开枪？即便如此，我仍然在比赛中努力投篮？在过去的五年里，我都在做些什么？

所以我开始看篮球比赛。NBA。看看他们打得多有组织，多套进攻和防守，他们如何在球场上跑来跑去。那是我在那 5 年里意识到哪里出了问题的地方:除了在篮球中投篮，还有很多事情要做。当然，我的学长一开始说的是对的，但我不应该停止射击。

更重要的是，我也意识到我确实不够好。天才篮球运动员和我*之间的技术差距不得不被承认。一直对自己重复“我的投篮很好，我尽力了”是错误的，而忽略了我的其他问题。我需要走出那个外壳，开始倾听和改进。*

这并不是一次有趣的经历。我记得我练习了几个月的运球和上篮，开始锻炼我的耐力。比以前更努力、更频繁地练习。我基本上是在做初学者做的事情，尽管起步比他们早。然而，我的比赛感觉更完整了，之后我真的可以和我的队友一起工作，开始以某种方式投篮。

* * *

你可能会想，“这些和编程有什么关系呢？这是 dev.to，这篇文章里没有#watercooler 标签！”我的错，我可能有点忘乎所以了(老实说，我非常喜欢篮球)。然而，当与你的工作经验进行比较时，也许你们中的一些人已经有了这种感觉:在这一领域工作多年，已经在某种程度上擅长插入编程语言，但是不知何故，你的主管或你的同事(甚至你自己？)在告诉你，你不够好？

我认为，要克服你不够好的事实，你还有很多事情要赶上，重要的是首先*承认*你不够好的事实。我知道这看起来像是你在搅动已经沸腾的冒名顶替综合征锅，但我不认为你可以在不承认需要改进的情况下改进。说“我尽了最大努力，为什么你看不到我的努力”或“我已经很擅长 X Y Z”并没有帮助，你可能已经从我 5 年的篮球练习中学到了；它仍然没有解决实际问题，而这些正是你所缺乏的。然后，从中制定可行的项目，并开始改进。

我不得不说，接受你不够好的事实对于提高自己来说是非常重要的，至少对我来说是这样。它教会你降低自我，避免被自己的观点束缚。也让自己能够适当的倾听批评者；如果你发现自己在得到反馈时很容易被冒犯或产生抵触情绪，可能是因为你一直在回避你正在得到反馈的事实。首先承认它们，也许休息一下，开始倾听，然后看看反馈是否有意义。

此外，偶尔让自己看起来像初学者也是可以的，尤其是如果你是初学者，并且你已经承认或接受了你是初学者的事实。如果有人告诉你，尽管你已经在 it 行业工作了很多年，但你在 Git 方面还不够好，那么*很好*去拿那本“如何为傻瓜 Git”的书，并弄清真相。在我看来，这并没有让你少了一个人，只是你碰巧不特别了解那件事。那就是[超正常](https://overreacted.io/things-i-dont-know-as-of-2018/)。这种经历会让你成为一个更谦逊的人，总体来说这很好。

* * *

无论如何，已经有很多关于冒名顶替综合症的文章，以及你如何摆脱那种不够好或不值得你现状的感觉。我不想听起来像一个破纪录者，但目前这是 NBA 的休赛期，没有太多的篮球话题可谈，所以我想为什么不呢😂。此外，在我看来，当谈到这个话题时，这仍然不是最显而易见的事情，至少对我来说，所以只是让这个词出来。

尽管如此，谢谢你读我的文章，我希望这在某些方面能帮助到一些人！
