# 按区域划分的本地松弛渠道

> 原文：<https://dev.to/thisdotmedia/local-slack-channels-by-region-59co>

有许多基于技术的 Slack 频道，但在本文中，我们按地理位置收集了一个正在进行的技术社区 Slack 频道列表。

这是一次促进当地技术社区的尝试，并鼓励所有人加入和参与他们的当地社区。

在 github [这里](https://github.com/ladyleet/tech-community-slacks)有一个更新的列表和提交 PR 来添加你的城市或技术空闲的能力。

**大西洋中部**

如果你住在特拉华州、马里兰州、新泽西州、纽约州、宾夕法尼亚州或华盛顿特区，那么看看这些频道是值得的:

*   DC : [DC 理工大学懈怠](https://dctech.slack.com/)

*   纽约州:[博罗吉斯](https://borojs.slack.com/)和[纽约科技](https://nyctech.slack.com/messages/C037T0K3D/)

*   PA : [费城创业领袖](https://phillystartupleaders.slack.com/messages/C03DP9M5Y/)
