# 为什么我进入了 Web 开发(latepost)

> 原文:[https://dev . to/mpissanos/why-I-get-into-wed-development-late post-5a al](https://dev.to/mpissanos/why-i-got-into-wed-development-latepost-5aal)

哦，时间过得真快！这是我刚开始熨斗学校的在线沉浸式项目时的一篇博文。回过头来看很有意思。幸运的是，我今天仍然感到和那时一样的兴奋和快乐....

从我最早的记忆开始，我就一直着迷于机器的内部运作，事物是如何建造的，以及人类使用它们的方式。我的祖父经常提醒我，小时候为了让我吃饭，我不得不解释液压电梯和缆索电梯之间的区别。13 岁那年，父母犹豫不决地把电脑组件作为生日礼物送给了我。那天晚些时候，他们发现我在上面玩游戏，非常震惊。

我对艺术和音乐也有热情和亲和力，直到今天我还在培养这种热情和亲和力。在我的一生中，我一直在寻找挑战自己精神的东西，以及奖励创造性思维的东西。我在一家五星级酒店担任泳池和海滩部门主管的经历教会了我大量的团队合作和领导技能。我最喜欢的部分是通过收集数据、利用电子表格(和基本编码)以及关注细节来安排和优化我们的运营效率。

在遭受了一次令人虚弱的伤害后，我再也无法保持我的双手在位置上，但幸运的是，我在一家酒店管理公司获得了一个行政职位。这个新角色是公司内部责任和曝光度的一大飞跃。我发现自己在培训经理，在每月的领导会议上在 100 名员工面前讲话，并且每天负责大量的现金资产。

尽管如此，缺乏真正能够驾驭我创造高效系统的热情，以及公司的政治，开始挫伤我对工作的热情。在这一点上，潜在的感觉，有更多的东西在那里等着我，是强有力的拉动。我只是不知道那是什么……还不知道。

我决定进行一次巨大的信仰飞跃，给自己留点时间，恢复体力，简单地生活。我游历了北加利福尼亚和南俄勒冈地区，沿途遇到了许多人。我的一个旅伴给我起了个绰号叫“工程师”。我可以(几乎)总是用最少的材料或资源解决或解决任何挑战。在这个过程中，我突然想到，我的逻辑、解决问题的能力、创造性思维和对技术的热爱会让我成为一名优秀的软件工程师。

一回到家，我就开始尽我所能地研究如何实现成为科技世界贡献者的目标。我花了数不清的时间着迷于代码的力量，仅仅通过输入就可以构建如此重要的东西。不用说我被迷住了！我知道我需要教育，我能够完成的自我学习似乎是一个巨大的进步，但这还不够。

当我发现新兵训练营的概念时，我知道它非常适合我。我做过各种各样的工作，比如景观美化、服务员，以及任何我在经济上能够投入时间让自己完全沉浸其中的工作，同时利用网络上的免费资源学习基本的 JavaScript、HTML 和 CSS。

我今天来到这里，即将开始熨斗学校在线沉浸式项目的第三周。在如此短的时间内，我学到了如此多的东西，真是令人难以置信。我将使用这个平台来记录我的旅程，并就我一路上学到的关键概念提供教程。

下次再见，一路平安！
