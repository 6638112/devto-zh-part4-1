# **[开关](https://ahkohd.github.io/switch-desktop/)**——一个用`Electron` & `Node.js`构建的带有一些使用`N-API`的原生绑定的 keycombo 窗口管理器。

在您最喜爱的应用程序之间快速切换。(Windows、macOS -正在开发中)

# 特性🔥

*   在应用程序之间快速切换。
*   最喜欢的应用程序加载到你的`Num`键`0-9`上。
*   添加收藏的 dock。
*   通过使用`Alt-left`或`Alt-right`，方便`left`和`right-handed`用户的使用。
