# 你每天的信息来源是什么？

> 原文：<https://dev.to/gac/what-is-your-daily-source-of-information-4o0b>

所以就在几天前，我停用了我的大部分社交媒体账户，现在只使用 Twitter(但也试图减少它的使用)。所以我注意到的一件事是，当我无聊或有任何空闲时间时，我会拿起我的手机，尝试打开 fb 或 twitter 滚动槽来消磨时间。

所以你可以想象，这不是很有成效，也没有帮助我在任何方面。

所以我想听听你们的意见，你们最喜欢访问哪些网站来了解最新的新闻(最好是科技方面的)，教育网站，播客和任何其他网站/应用程序，以尝试获得最新的信息和/或提高你的知识和技能？

如果你也注册了一些你确实想收到的时事通讯，请把它们写在你的评论里。或者任何其他你认为对某人有用的链接。

由于我在社交媒体上花了太多时间，我以为我总是知道发生了什么，但这与现实相去甚远。所以我决定改变一些旧的(坏的)习惯，并试着更多地关注自己。

这些是我访问的一些网站:

*   推特
*   Dev.to
*   不正常的
*   TheHackerNews.com
*   我当地的报纸(几乎没有什么有趣的内容)

谢谢大家:)
