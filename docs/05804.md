# 每日挑战 50 号邻居

> 原文:[https://dev . to/the practical dev/daily-challenge-50-number-neighbor-44jm](https://dev.to/thepracticaldev/daily-challenge-50-number-neighbor-44jm)

给定一个整数长度的电话号码`N (1 ≤ N ≤ 10)`作为一个数字串(例如，`555-555-5555`将作为`"5555555555"`被传递)，返回一个所有电话号码的数组，作为将被认为是该电话号码的邻居的数字串(可以是任何数字 1)。

> 例如:电话号码为`555-555-5555`的人有邻居`555-555-5554`和`555-555-5556`。
