# React tldr 为什么在 Redux 中应该对列表使用对象而不是数组

> 原文：<https://dev.to/maniekm/react-tldr-why-you-should-use-an-object-and-not-an-array-for-lists-in-redux-cco>

*   使用对象解决方案，您不会得到重复的项目。
*   使用一个对象将为你未来的自己节省一些时间和错误。
*   CRUD 操作不会每次都遍历所有项目。
*   如果你需要一个项目数组，只需添加将对象转换为数组的选择器。
