# 什么是 Selenium 语言绑定？

> 原文:[https://dev . to/tone theman/what-a-selenium-language-binding-2mib](https://dev.to/tonetheman/what-is-a-selenium-language-binding-2mib)

Selenium 是一个比其他任何东西都更重要的规范。

它是一组规则/方法/接口(在这里使用你自己的有意义的语言)，定义了你如何自动化一个网络浏览器(像 Chrome、Firefox 或 Safari)。

浏览器供应商(谷歌、Mozilla、苹果、微软..)都同意/了解该规范，并同意实施。当 Selenium 首次启动时，情况并非如此...

如果你想看的话，有一个关于 Selenium 的官方 w3c 规范。看这里:[https://www.w3.org/TR/webdriver1/](https://www.w3.org/TR/webdriver1/)

实际上，这是一种自动化浏览器的方式。Selenium 支持一些官方语言:Java 和 Python 是我使用最多的两种语言。还有像 Javascript 或 C#这样的...

因此，Selenium 为开发人员提供的代码(库)被称为 Selenium 语言绑定。

这些绑定试图符合您正在使用的语言的习惯。这意味着 Python 绑定看起来像您所期望的 Python。Java 绑定看起来像普通的 Java 代码...C#绑定看起来像表现良好的 C#等等。

因此，基于这一点，绑定会有所不同。但精神是一样的。
