# 寻找导师

> 原文:[https://dev.to/jondouglas/finding-a-mentor-5bdk](https://dev.to/jondouglas/finding-a-mentor-5bdk)

当你在软件行业日复一日地工作时，你会开始感到有点停滞不前，甚至有时停滞不前。毕竟，你已经付出了 100%的努力，但是如果 100%的努力不够呢？也许你希望得到提升，承担更多的责任，甚至考虑一个完全不同的角色。

捷径来了，找一个导师。在你的生活和日常生活中有一个好的导师会给你带来启发，这种启发可能需要几个月，甚至几年才能自然到来。如果你想充分实现你作为一名开发人员的真正潜力，你需要有人在这一路上帮助指导你。

你可能会问...说起来容易做起来难，找一个导师并不容易。这并不总是那么容易，也不会是一项令人生畏的任务，即足够脆弱地告诉某人你相信他们会是你的理想导师。

要记住的一点是，你的导师不会是完美的。我的意思是，每个导师可能精通某项技能，而在其他方面有所欠缺。世上没有完美的导师。相反，你可以接受多位导师，通过了解每位导师的贡献，以及你如何充分利用这些贡献来磨砺你的思维。

一个好的导师有什么特质？如果你知道某人:

*   了解全局
*   积极倾听
*   鼓励需要的地方
*   表现得像你最大的粉丝
*   激励你做得更好
*   让你走上正轨

因为这些技能存在于人们身上，不管他们的技能如何，在职业生涯的早期找到这些人来寻求安慰是有益的。你可以通过为他人提供指导来回报这种帮助。

导师不一定是你心目中的英雄，他们可以是和你一样正在为同样的问题而奋斗的同龄人，但他们就像你最大的粉丝一样，鼓励你做得更好。你甚至会发现自己是你的导师之一。听起来很疯狂，你是你最大的粉丝，你确保你理解大局，你让自己每天都在正轨上。

从小事做起，首先问一问你欣赏的人他们的工作、职业道德，甚至他们的个性，问他们是否愿意每月一次在午餐时指导你。最重要的是，确保**你做了识别你认为你可以改进的地方**的工作，这样你的导师在了解你的同时，对你今天努力的领域也有所了解。
