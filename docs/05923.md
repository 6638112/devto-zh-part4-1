# 如何将图片添加到帖子的提要中？

> 原文：<https://dev.to/cavezza/how-do-you-add-images-to-the-feed-for-a-post-2l2d>

我尝试在元数据中使用`cover_image`。它将一张图片应用到帖子本身，但该图片并没有出现在该图片的提要中。例如，图像如下所示:

[![Alt Text](img/e8e1b3998131bddc1a5f3248d831a7b0.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--Gwhgh-q3--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/eh5ujzxf4tjbdwxigjxo.png)

为了做到这一点，你还需要做些什么吗？

以下是我的猜测:
1。)封面图像需要固定一定的宽度/高度限制才能出现在提要中
2。)图片需要得到批准才能出现在提要中
3。)我需要在帖子中添加另一个元数据标签。

任何帮助都将不胜感激。
