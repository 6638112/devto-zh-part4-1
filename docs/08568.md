## [](#ipconfig-)**ipconfig :**

ipconfig 是 windows 中一个重要的基本命令，可以用来显示你的机器的网络信息，如 ip 地址，DNS 等..

**例子:**
**- >打开你的 window 终端，运行‘ipconfig’命令。**
[![](../Images/64d7f1fa5cccee1ecadcec3ee5911c2a.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--buI_CuiK--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/dwx6i9ko3u3b6t2m21x3.JPG)

## [](#ping-)**平:**

ping 是 windows 中更常用的网络实用程序，用于检测网络上的设备和排除网络问题。

**例子:**
**- >打开你的窗口终端，运行‘ping’命令。**
[![](../Images/d7d3abafc983d0ab63b339b27901725b.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--0Gv52gCL--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/tlj5hm4yiuyj1ptbd6z9.JPG)

## [](#netstat-)**netstat :**

netstat 是非常有用的命令。它用于显示系统中的 udp 和 tcp 连接和端口。它还可以通过显示未知端口来检测系统中的任何病毒类型。

**例:**
**- >打开你的窗口终端，运行‘netstat-a’命令。**
[![](../Images/bd8d14dd361711f7c1ce7020d7ddba12.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--CZGmeKZL--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/4awtwvhxp7bmxsvm9b73.JPG) 
**注意:-a 用于系统中的所有端口。**

## [](#tracert-)**tracert :**

tracert 代表 traceroute，就像 ping 一样，它发送数据包来排除任何网络故障，但在数据包从一台服务器跳到另一台服务器时，它会跟踪数据包的路由。

**示例:**
**- >打开你的 window 终端，运行‘tracert IP _ address’命令。**
[![](../Images/29ad1cea2d1db9b2effd53d546513151.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--GA08QzvG--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/r0tszipsjlze396iz9lf.JPG)

## [](#nbtstat-)**nbtstat :**

nbstat 是一个使用 NBT(TCP/IP 上的 NetBIOS)显示协议统计和当前 TCP/IP 连接的实用程序。在道德黑客攻击中，这个术语叫做枚举，用于从系统中提取用户名、机器名、网络资源、共享和服务。
所以对于枚举有时 nbtstat 命令用来枚举共享服务。

**例:**
**- >打开你的窗口终端，运行‘nbstat IP _ address’命令。**
