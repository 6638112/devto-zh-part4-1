# [](#ruth-john)露丝·约翰

[![Ruth John](../Images/42cc998b5c86b6b8eba33fe970279924.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--0o_cxMGb--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://2018.cascadiajs.com/speakers/ruth.jpg)

快来感受浏览器中的音频视觉世界吧。这不仅仅是创造令人惊叹的音乐数据！我们看到了如何将数据实时传输到自定义属性中，以增强我们的视觉效果。Ruth 将重新阐述我们如何通过网络音频 API 分析声音，并提出一种更好的处理数据的方法来更好地适应音乐。更重要的是，我们还利用 Web MIDI API 进行全面的硬件控制。您可能没有想到，但这确实是一个包含大量要点和新技术的演讲，供您在编码生涯中考虑。
