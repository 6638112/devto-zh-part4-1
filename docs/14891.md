2.  [@material-ui/core](https://www.npmjs.com/package/@material-ui/core) - React 材质 ui 组件
3.  [react-Bootstrap](https://www.npmjs.com/package/react-bootstrap)-Bootstrap 4 个用 React 构建的组件
4.  [react-bulma-components](https://www.npmjs.com/package/react-bulma-components) -布尔玛框架的 react 组件
5.  [semantic-ui-React](https://www.npmjs.com/package/semantic-ui-react)-React 组件库
6.  [office-ui-fabric-react](https://www.npmjs.com/package/office-ui-fabric-react) -基于 react 的前端框架，用于构建 office 和 Office 365 的体验
7.  [reactstrap](https://www.npmjs.com/package/reactstrap) -引导程序 4 的无状态 React 组件
8.  [rebass](https://www.npmjs.com/package/rebass) - React 用 styled-system 构建的原始 UI 组件
9.  [@ appbaseio/React ive search](https://www.npmjs.com/package/@appbaseio/reactivesearch)-React UI 组件进行弹性搜索

## [](#css-in-js)CSS 中的 JS

1.  [styled-components](https://www.npmjs.com/package/styled-components) -组件时代的视觉原语。使用 ES6 和 CSS 的精华来设计你的应用，没有压力
2.  [情感](https://www.npmjs.com/package/emotion)-JS 库中的一个 CSS
3.  radium 是一套管理 React 元素内联样式的工具。它为您提供了强大的样式功能，而无需 CSS

## [](#http-clients)Http 客户端

1.  [axios](https://www.npmjs.com/package/axios) -浏览器和 node.js 的基于 Promise 的 HTTP 客户端
2.  基于 axios 构建的 http 客户端
3.  [请求](https://www.npmjs.com/package/request)——请求被设计成进行 http 调用的最简单的方式。默认情况下，它支持 HTTPS 并遵循重定向
4.  [superagent](https://www.npmjs.com/package/superagent) -小型渐进式客户端 HTTP 请求库，以及具有相同 API 的 Node.js 模块，具有许多高级 HTTP 客户端功能

## [](#routing)路由

1.  [React-Router-dom](https://www.npmjs.com/package/react-router-dom)-React 路由器的 DOM 绑定
2.  通用路由器(universal-router)一个简单的中间件风格的路由器，可以用于客户端和服务器端应用程序
3.  [connected-React-Router](https://www.npmjs.com/package/connected-react-router)-React 路由器 v4 和 v5 的冗余绑定

## [](#state-management)状态管理

1.  redux 是 JavaScript 应用程序的可预测状态容器
2.  react-Redux-Redux 的官方 React 绑定
3.  mobx 是一个经过实战检验的简单可扩展的状态管理库，它透明地应用了函数式反应式编程(TFRP)
4.  redux-saga - redux-saga 是一个旨在使应用程序副作用(例如，异步的数据获取和不纯的访问浏览器缓存)更容易管理的库
5.  [redux-thunk](https://www.npmjs.com/package/redux-thunk) -用于 redux 的 thunk 中间件
6.  [redux-persist](https://www.npmjs.com/package/redux-persist) -持久化和恢复 redux 存储
