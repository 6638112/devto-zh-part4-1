# 你知道你在优化什么吗？

> 原文：<https://dev.to/bovermyer/do-you-know-what-you-optimize-for-2ndb>

作为开发人员，我们花了相当多的时间来优化代码。

然而，我们也花了很多时间来优化我们的行为。我们常常意识不到这一点。

举个例子，如果你开车去上班的时候选择把车停在靠近门口的地方，你就是在为某件事优化。如果你选择把车停在停车场的后面，你是在为别的事情做优化。不过，你可能不知道你在优化什么，这就是这篇文章的重点。你可能只是在没有完全理解为什么的情况下决定了一些事情。

试着把注意力集中在你通常不假思索地做的一件特定的事情上。你这么做是为了什么优化？您可以针对哪些方面进行优化？
