最近，Lisa Crispin 与 Janet Gregory 合著了《更敏捷的测试:整个团队的学习之旅》和《敏捷测试:测试人员和敏捷团队实用指南》，他们在测试自动化大学发布了一门新的免费课程，以学习更多关于 DevOps 中测试自动化的知识。

在这个课程中，Lisa 提供了指导来学习一个团队可以使用测试自动化的不同方法，从而以可持续的速度成功地向客户交付商业价值。她涵盖了一些关于开发运维、持续交付和持续部署的基本概念和术语，以及可操作性和从生产使用中学习的技术。

Lisa 还谈到了 DevOps 的自动化策略，如何使用自动化——包括发布前和发布后的测试——来降低风险和指导开发过程。她还分享了一些缩短团队反馈循环的方法。
