# [](#introduction)简介

我想我首先想到的是票价。每当我偶然发现一个我非常想去的技术会议。

但是由于门票价格过高，我放弃了去参加那个技术会议的想法。

是的，我是那种不愿意为一场技术会议支付 300 美元或更多的人。

除非我相信那张会议票的价值与黄金同等重要。

我希望通过与你分享这些关于如何获得“免费”入场券的技巧。

这对你自己和你所在的开发者社区都有很大的帮助。

# [](#be-a-volunteer-for-conference)成为会议志愿者

今年是我第三次在新加坡帮助组织 Python 大会(PyCon)。

我再怎么强调这一点也不为过，那就是需要**志愿**你的时间或者**的专业知识**来获得你的“免费”入场券。

作为一名技术会议组织者，我一直在寻找志愿者来帮助组织一次技术会议。

通过分担负担来顺利运营活动，并获得足够的收入(在黑暗中)，以便我们可以在第二年举办另一场技术会议。

组织一次会议是一项令人精疲力尽的活动，从寻找赞助商、获得场地，到为与会者安排食物，或者仅仅是会议谈话的录像。

会议组织者感谢所有他们能得到的帮助，只要你花一点时间和精力去帮助实现它。

# [](#be-a-speaker-or-instructor)成为演讲者或讲师

这是从作为一名志愿者获得大会“免费”入场券的一步。

我建议任何刚刚开始开发生涯的开发人员或业内老手向会议组织者提交 CFP(征集提案)。

不要害怕提交您对某个特定技术主题的了解。

以下是我作为组织者如何审查来自 CFP 的提案的一般指南。

*   与会议主题的相关性？
*   提案是否详细说明了演讲或研讨会的内容？
*   与会者对这个主题感兴趣吗？
*   观众是否容易在短时间内消化？
*   对一个与会者有用吗？

# [](#seek-to-be-part-of-the-super-connectors-network)寻求成为超级连接器网络的一部分

我通过超级连接器拿到了一些科技会议的门票。

**超级联系人**是思想领袖或拥有巨大的&强大人际网络的人，他们通过专业工作或志愿者工作获得了培养。

他们可能是 meetup 的组织者，或者做一些对本地开发者社区非常有益的工作，比如为无法参加这些会议的开发者录制技术会议的演讲。

一个这样的超级连接器是工程师的创造者迈克尔·程。SG 这是一个非营利社区，帮助记录新加坡开发者社区的技术演讲和会议。

由于这些超级联系人的这种**“向前支付”**态度，他们将有机会偶然发现参加他们社区的技术会议的“免费门票”，以帮助他们的开发伙伴。

因此，成为他们网络的一部分没有坏处，但是请注意，如果你计划成为超级连接器网络的一部分。

你也应该在心理上采用这种**向前支付**来提供推荐、传播信息或者只是卷起袖子志愿帮助他人。

# [](#conclusion)结论

我并不谴责利用这些技巧获得“免费”门票的行为。

自从成为一名志愿者，一名演讲者或成为超级连接器网络的一部分。

这些技巧对开发人员的职业生涯和本地开发人员社区都非常有益。

对于任何一个看重自己的网络，有**“向前付出”**态度的开发者来说
