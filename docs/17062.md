# 新的树莓 Pi 4 发布了！你会用它来建造什么？

> 原文：<https://dev.to/downey/new-raspberry-pi-4-released-what-will-you-build-with-it-2cpg>

液体错误:内部

今天，树莓派基金会[宣布了全新的树莓派 4](https://www.raspberrypi.org/blog/raspberry-pi-4-on-sale-now-from-35/) ，起价仍然只有 35 美元！链接的公告帖子更详细，但一些值得注意的升级是:

*   可选的 2GB 和 4GB RAM 配置
*   不与 USB 集线器共享带宽的千兆以太网
*   基于全新 28 纳米工艺的更快 1.5 GHz 处理器
*   分辨率高达 4k 的双显示器支持

我对可选的更高 RAM 配置和千兆以太网特别感兴趣。我很想换掉我的玩具[基于 Raspberry Pi 的 Kubernetes 集群](https://downey.io/blog/how-to-build-raspberry-pi-kubernetes-cluster/)中的一些工作节点，因为 1GB 根本不足以在操作系统和 kubelets 运行所需的基础上运行任何有意义的东西。此外，有了千兆以太网，我觉得我现在可以用它来构建一个相当不错的 NAS。
