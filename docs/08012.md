# [](#my-recent-open-source-contributions-august-2019)我最近的开源贡献(2019 年 8 月)

最近我非常积极地为 GitHub 上的开源项目做贡献。部分原因是有必要推进其他项目，而提议的变更允许我这样做。另一个原因是，我非常钦佩这些软件包，并愿意做出贡献，使它们变得更好，不仅是为了我自己，也是为了其他所有人。

我最近对以下存储库做出了贡献:

*   [spatie/laravel-activity log](https://github.com/spatie/laravel-activitylog/pull/570)
*   [阿加洛斯/拉腊维尔-奥尼斯基](https://github.com/ageras-com/laravel-onesky/pull/19)

我很高兴我能以一种有意义的方式做出贡献，实际上是建议内部的改变，并让它们也为其他人服务。

## [](#my-own-packages)我自己的包

当然，我也一直在开发自己的包，增加新的特性，编写测试，修复错误。对于一些包，我还添加了与 TravisCI 的集成，以便能够自动测试包，并确保一切仍然正常工作。我自己的所有套装包括:

*   [roelofjan-elsinga/content-to-html-parser](https://github.com/roelofjan-elsinga/content-to-html-parser)
*   [roelofjan-elsinga/sitemap-generator](https://github.com/roelofjan-elsinga/sitemap-generator)
*   roelofjan-El singa/atom-feed-generator
*   [试管/模型分度器](https://github.com/roelofjan-elsinga/model-indexer)
*   [roelofjan-elsinga/flat-file-CMS](https://github.com/roelofjan-elsinga/flat-file-cms)
*   [roelofjan-elsinga/flat-file-CMS-GUI](https://github.com/roelofjan-elsinga/flat-file-cms-gui)
*   [roelofjan-elsinga/flat-file-CMS-publish](https://github.com/roelofjan-elsinga/flat-file-cms-publish)

roelofjan-elsinga/flat-file-CMS 和*roelofjan-elsinga/flat-file-CMS-GUI*是我目前最大的软件包。*平面文件 cms* 是一个简单的软件包，允许你在 Laravel 中拥有一个嵌入式平面文件 cms。 *flat-file-cms-gui* 只是一个管理仪表板，允许您在图形用户界面(gui)中与 cms 进行交互。会有额外的包来补充*平面文件 cms* ，像*平面文件 cms 自动发布*和*平面文件 CMS SEO*。这些将作为单独的包添加，因为我想保持核心包的干净，并专注于内容本身。GUI 包只是核心 CMS 包的图形表示，也可以被完全不同的图形实现所取代。它只是作为“官方 GUI”，不多也不少。

我希望你已经对我在过去几周的工作有了更好的了解，我希望你能回来更新。通过在 Twitter 上联系我，让我知道你对这种形式的博客帖子的看法。
