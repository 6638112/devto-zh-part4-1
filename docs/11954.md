# 反应性感容易样板

> 原文:[https://dev.to/nimahkh/react-sexy-easy-boilerplate-5a8b](https://dev.to/nimahkh/react-sexy-easy-boilerplate-5a8b)

嗨伙计们。你有没有担心过为每个 UI 工具包或状态管理创建一个 SSR react 样板文件？😓
是的，处理这些样板文件是我的挑战。
每次，我都必须克隆一个回购协议并使用`npm i`，这并不重要，最重要的是在我的项目中增加专业 react 开发人员的经验。
由于这个原因，我决定开发一个名为“React easy boilerplate”的包。😁 😁一个 CLI 包，为您的 Reactjs 项目调节和生成简单的样板文件。😋 😋它非常容易使用。
该包装有四个 UIKits

*   jss
*   材料-用户界面
*   postcss
*   蚂蚁设计

和两种状态管理

*   上下文 api
*   回家的

目前，redux 支持 postcss 和 jss，因为它是 UI 工具包。

其他样板文件我们会尽快准备好。

我希望这是有用的，可以帮助您生成简单的 SSR 样本。

### [](#github-heartbeat-heartbeat)github💓 💓

[反应轻松的样板文件](https://github.com/nimahkh/react-easy-boilerplates)

### [](#installation)安装

```
npm i -g react-easy-boilerplate 
```

### [](#usage)用法

1-在 cmd 或终端上运行以下命令

```
$ react-easy-boilerplate 
```

2- `cd`到您的项目文件夹

3-运行以下命令:

```
npm run start:dev   // run development server
npm run build:prod //build production 
npm run start:prod  // run production server (ssr) 
```
