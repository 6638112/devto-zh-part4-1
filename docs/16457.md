# 构建自动化股票交易者- 1

> 原文:[https://dev . to/nirajvinit/building-an-automated-equity-stock-trader-1-18dp](https://dev.to/nirajkvinit/building-an-automated-equity-stock-trader-1-18dp)

我一直想了解股票市场和股票交易，但一直拖延。然后去年我读了一些关于财务自由的文章，改善财务状况的渴望再次高涨。但是，我的心从来没有进入金融，或任何与钱有关的东西。我是一名程序员，我如此热爱编程，以至于我几乎忽略了其他一切——社交生活、金钱管理、时间管理。见鬼，有时候我想，我做我喜欢的事情是有报酬的，我可能为了一小袋花生就完成了我的工作，除了比萨饼更贵。

因为，我敏锐地意识到我的拖延症，我决定采取一个新的一年的决议，以更好地处理财务问题，了解更多，争取财务自由。我想有一个系统，让我的钱即使在睡觉的时候也能保持增长。

因此，我开始学习股票市场，通过做积极的交易，只要我能找到时间就阅读。

很快，我就迷上了。赚了点钱，亏的比赚的多。过了一会儿，我意识到这些-

*   我浪费时间和金钱在低价股上(大部分都是垃圾),希望有足够的时间，这些股票中的一个会成为多袋
*   我笨手笨脚的(搞砸了许多交易)
*   市场是由快速，有纪律，消息灵通，经验丰富的交易者控制的
*   关于股票的技术信息非常重要
*   最好做摇摆交易，不断轮换资金，以获得更好的收益率
*   我记性不好，记不住交易、股票和它们的价格
*   我不擅长管理我的投资组合
*   我超级懒
*   我很容易害怕
*   我承担不必要的意外风险

许多投资组合管理服务提供商已经和我接触过，但是我拒绝接受他们的服务——原因是——我想学习这个系统以便在这方面做得更好。我想建立自己的能力。

我一直想要一个自动系统，它可以推荐我买/卖哪些股票，并在适当的时候给我提醒。或许以后，系统可以自己做交易了。交易是可以学的，也是可以教的。成功取决于实践，更好的计划，更好的编码，更好的坚持。如果我能建立一个系统来做我的工作——在我睡觉的时候为我赚钱，那将是一个惊人的成就。

我敢肯定，有许多机器，这样的系统在那里，我可以使用他们的服务，而不是试图建立自己的。但是，我一直是一个实干家，我想利用这个机会学习更多的东西。我正在利用这次旅程学习新语言、新技术——人工智能/人工智能、大数据、数据科学、算法交易、阅读和理解金融数据、实时数据处理。这么多要学的东西和要做的事情，真令人兴奋。

就在最近，我完成了这个旅程的第一步——创建了一个程序来帮助我分析我以前的交易，并在价格下跌时通知我。这是非常重要的第一步，我会继续写更多关于我的旅程。
