# 想法

借用了谷歌 Chrome 的“无互联网连接”恐龙。

一年前我就这样做了，当时我想“如果，我应该做一个 3d 像素艺术，而不是 2d”。

所以，碰巧那周的“代码笔挑战”是关于恐龙的。

我首先想到的是谷歌 Chrome 的“无互联网连接”恐龙游戏。

我没有对 CSS 和动画 Javascript 使用任何预处理。

但是为了方便起见，我使用了 CSS 变量。我当时对 CSS 变量了解不多，对 HTML 前置处理器完全不了解。
