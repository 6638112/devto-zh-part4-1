# #CodepenChallenge 面条:餐厅主页布局概念

> 原文：<https://dev.to/takaneichinose/codepenchallenge-noodles-restaurant-home-page-44ai>

我这周的共同挑战。

本周的挑战是关于面条的。这个星期我想不出任何花哨的 CSS 艺术，所以我只是想到为一家餐馆制作一个示例主页。

它的功能是用 VueJS 编写的。

设计灵感来源于互联网上的很多餐厅网站。

标题部分只是一个简单的滑块。您可能会看到网站上的图片。

使用的技术:
