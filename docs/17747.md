#### 必备 Chrome 扩展

*   配合 [Chrome Vimium](https://lifelongprogrammer.blogspot.com/2019/04/awesome-tips-about-chrome-vimium.html) 使用[大吊杆](https://chrome.google.com/webstore/detail/the-great-suspender/klbibkeccnjlkjkiokjodocebajanakg/)，吊杆减少内存使用，使用 [Chrome Vimium](https://lifelongprogrammer.blogspot.com/2019/04/awesome-tips-about-chrome-vimium.html) 快速找到并转到特定标签页。

### Chrome 扩展阅读

#### [黑暗阅读器](https://chrome.google.com/webstore/detail/dark-reader/eimadpbcbfnmbkopoojfekhnkhdbieeh)

*   也可以在偏好设置->通用中启用 Mac 的黑暗模式

*   当页面(网站)很慢时，使用[阅读器视图](https://chrome.google.com/webstore/detail/reader-view/ecabifbgmdmgdllomnfinbmaellmclnh):加载大量广告，flash，使用大量内存，你将需要时间(2 分钟以上)来阅读它。

*   在其他情况下，使用[只需读取](https://chrome.google.com/webstore/detail/just-read/dgmanlpmmkibanfdgjocnabmcaclkmod)。

*   用[明确](https://chrome.google.com/webstore/detail/clearly/odfonlkabodgbolnmmkdijkaeggofoop)检查轮廓。

### 用于书写的 Chrome 扩展

*   [语法](https://chrome.google.com/webstore/detail/grammarly-for-chrome/kbfnbcaeplbcioakkpcpgfkobkghlhen)
    *   将`contenteditable="true"`添加到 div( `div class="post-outer" in blogger`)中，使语法在任何地方都有效。
    *   [也是一款独立的 Mac 应用](https://app.grammarly.com/)
    *   将它与 Chrome 自带的拼写检查器结合使用
    *   按需检查:将“网站访问”改为“点击访问”(以减少内存使用)
*   [让谷歌图片再次变得伟大](https://chrome.google.com/webstore/detail/make-google-images-great/kjlaimldplcbblpgenlhfjpobmolcblj)

### 共享的 Chrome 扩展

*   [缓冲器](https://chrome.google.com/webstore/detail/buffer/noojglkidnpfjbincgijbaiedldjfbhh/)
    *   在 chrome://extensions/shortcuts 中将快捷方式`Ctrl+Shift+B`分配给`Buffer -> Activate the extension`
*   [保存到口袋](https://chrome.google.com/webstore/detail/save-to-pocket/niloccemoadcdkdjlinkgdfekeahmflj)
    *   也安装移动应用程序，这样我们就可以随时随地保存到口袋里。
*   [Evernote Web Clipper](https://chrome.google.com/webstore/detail/evernote-web-clipper/pioclpoplcdbaefihamjohnefbikjilc)
