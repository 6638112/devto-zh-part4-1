# 关于格林奇

> 原文：<https://dev.to/glitch/on-glitch-this-week-8k1>

本周 Glitch 是一个常规系列，在这个系列中，我们总结了一些了不起的创作者在 Glitch 上完成的令人惊叹的项目。以下是本周吸引我们眼球的几个项目。

## [和朋友🖼️一起搞艺术](https://glitch.com/~inter-art-ivity)

正如“[互动](https://glitch.com/~inter-art-ivity)”的创作者 Vidhi Bagadia 所说:“创造力最好是在一起创造的。”这就是我喜欢它的原因。你可以和朋友一起创作，或者和网上的陌生人一起创作。选择一种颜色，输入你的名字，然后把你的杰作和其他作品一起画出来。你甚至可以下载一份拷贝。

[https://glitch.com/embed/#!/embed/inter-art-ivity?previewSize=100&path=index.html](https://glitch.com/embed/#!/embed/inter-art-ivity?previewSize=100&path=index.html)

## [玩多样性迎合宾果⭕](https://glitch.com/~diversity-pandering-bingo)

由[杰夫](https://glitch.com/@gapple)创作的“[多样性-迎合-宾果](https://glitch.com/~diversity-pandering-bingo)”是[拉妮·莫拉](https://twitter.com/ranimolla)的[众包](https://twitter.com/ranimolla/status/1138276871781531648)宾果板的互动版本，致力于科技公司用来解释为什么他们还没有更加多样化的无力借口。

这是她的文章《[为什么科技行业的女性是被 PS 进来的而不是被雇佣的》](https://www.vox.com/recode/2019/6/17/18678541/women-tech-photoshop-diversity)的补充，这篇文章深入研究了过去 5 年科技公司的多样性报告，看看有什么变化，或者更准确地说，有什么没有变化。

[https://glitch.com/embed/#!/embed/diversity-pandering-bingo?previewSize=100&path=index.html](https://glitch.com/embed/#!/embed/diversity-pandering-bingo?previewSize=100&path=index.html)

## [Zdog 入门🐶](https://glitch.com/~zzzdog)

Zdog 是 Metafizzy 的用于 canvas 和 SVG 的 3D JavaScript 引擎。它使您能够在 Web 上设计和渲染简单的 3D 模型，将矢量插图的简单性带入 3D。你可以在这里看到你能用它做的各种东西。然后使用[德里克·雷诺兹](https://glitch.com/@drk)的项目“ [zzzdog](https://glitch.com/~zzzdog) ”开始用它制作你自己的有趣场景。
