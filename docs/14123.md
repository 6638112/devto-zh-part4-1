# 为什么是软件工程？

> 原文：<https://dev.to/valenadelv/why-software-engineering-4h19>

我读过很多关于成为软件工程师的利弊。好消息是:

1.  他们赚了很多钱
2.  有些人可以选择远程工作
3.  对软件工程师的需求很大

所以，当然，我做了我的研究，并找到了开始我的旅程的方法。我过去觉得，现在仍然觉得，这是一段我和任何人都可以踏上的旅程。只要有耐心、毅力和投入工作的动力，我就会有一个好的开始。

我通过免费的在线课程直接学习了 HTML、CSS 和 JavaScript 的一些基础知识。不久之后，我想为什么不申请几个编码训练营，在那里我可以找到我所学的某种结构，这是我觉得我需要的。我一直渴望改变我的学习重点，软件工程立刻吸引了我的注意力。从那时起，我知道这必须是我的下一步行动。

对于正在阅读这篇文章的开发者，我的问题是，是什么让*你*想要成为今天的你？你是如何为你的旅行做准备的？
