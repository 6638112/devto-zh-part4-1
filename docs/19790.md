# 单页应用程序安全性(对于 Typescript 项目)

> 原文：<https://dev.to/zainabed/single-page-application-security-for-typescript-projects-1j1h>

[**@ zainabed/soteria**](https://github.com/zainabed/web-security-soteria)是单页面应用安全，它通过提供客户端授权来减少 HTTP 服务器调用量。它提供 API 来根据应用程序的安全区域验证用户权限，而无需咨询服务器 API。
这些验证可以在应用程序路由或 REST API 调用上执行。

它是`@zainabed/security`规范 API 的一个实现。
