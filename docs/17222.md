# Neat 独立项目第一名

> 原文：<https://dev.to/spekulatius1984/neat-indie-projects-no-1-49og>

## [T1】简介](#intro)

作为一名独立制作人，我和许多其他制作人一样面临着同样的问题:我不会做营销。我努力推销我的副业，并通过我的独立作品赚钱。我在 indiehacker 空间的许多对话中都看到和听到了这一点。为了有所帮助，我想谈谈我最近发现的一些优秀的独立项目。我们开始吧:

### “花得值的时间”——把定价看成工作时间。

查看非美元价格，查看以小时显示的价格。需要工作时间，才能赚到这个数目。在许多主要网站上工作，如亚马逊和易贝。仅作为 Chrome 扩展 atm 提供。查看由 [Greg Greiner](http://gregrgreiner.com/?ref=PeterThaleikis.com) 编写的 [Chrome 扩展“花得值的时间”](https://chrome.google.com/webstore/detail/time-well-spent-%E2%80%94-see-pri/fagkeimkebmhbeelogmpfpcfdiamhpoa)

### “ask blocker”——停止浏览器中的权限请求

默认情况下，有太多网站请求推送通知等权限。到目前为止，我不喜欢这种做法，并找到了[ask blocker](https://chrome.google.com/webstore/detail/askblocker/eelfgnmggfgncoipinopobfcabnfknho)([Firefox](https://addons.mozilla.org/en-GB/firefox/addon/askblocker/))作为解决方案。这是一个简单的浏览器扩展，阻止请求并允许你管理请求(通过允许和重载)。该扩展由[亚当·林奇](https://adamlynch.com/?ref=PeterThaleikis.com)构建，并作为[开源库](https://github.com/adam-lynch/askblocker)提供。

### “Cost me”——支付提供商实际上花了你多少钱？

你是否后悔当初选择了 SaaS 或副业的支付提供商，并想看看有哪些选择？或者你计划开始一个新的项目，使用直接支付作为一种货币化的方式？检查 [Fauzi Abdillah](https://fauz.io/?ref=PeterThaleikis.com) 项目到[比较支付提供商成本](https://costme.fauz.io/?ref=PeterThaleikis.com)！

### “full help”——自托管的帮助台和知识库软件

你的副业、SaaS 或社区需要一个全面的帮助部分吗？由 [Gerardo J. Báez](https://www.producthunt.com/@gerardojbaez) 开发的 FullHelp 允许您自己托管一个帮助台和链接的知识库应用程序。让您和您的团队有时间关注重要的事情，而不是每天处理登录问题。许可费包括一年的更新和增强功能，并可以折扣价续订。

### “录制屏幕”——直接从浏览器录制您的屏幕。

一个简单的在线工具[在你的浏览器](https://recordscreen.io/?ref=PeterThaleikis.com)中进行截屏，完全不需要安装。只需点击“录制”，然后“开始录制”并授予权限。这就是你所需要的。您还可以同时录制您的屏幕和网络摄像头，以制作 explainer 或查看视频。由 [Asad Memon](https://asadmemon.com/?ref=PeterThaleikis.com) 和 [Munir Usman](https://munirusman.com/?ref=PeterThaleikis.com) 完成。

### “Enso OS”——基本 OS 的替代方案

我是 Linux 的忠实粉丝。从 2k 以前开始使用它，当时一切都很糟糕，许多东西都不起作用😂️幸运的是，这些日子已经过去了。现在比以前发布了更多优秀的 Linux 发行版。Enso OS 就是其中之一。

一些科技的东西，如果你对这个感兴趣的话:Enso OS 是基于 Ubuntu 16.04 (LTS - Xenial)的定制版本。它使用 Xfce 作为桌面管理器。除此之外，Enso 还集成了 Gala 窗口管理器(来自初级项目)以及 Panther 发射器(一个弹弓叉)。“地球上最简单的码头”也开始运作了。

Nick Wilkins 也开源了这个项目。目前正在等待更多的投稿& bug 报告。源代码托管在 [GitHub](https://github.com/nick92/Enso-OS) 上，如果你想查看的话。请考虑用[推文](https://twitter.com/intent/tweet?text=I%27ve%20just%20found%20this%20neat%20Linux%20distro%3A%20enso-os.site%20by%20@Nick_Wilkins&via=spekulatius1984)或下载&试用来支持这个项目🙏️

### “push folio”——用几行 JavaScript 创建弹出消息

对于那些不想要对讲机这样复杂工具的人来说，Pushfolio 是一个解决方案。如果你只是想显示一些像样的弹出窗口来转换人们，这是你应该看看。

它基本上是一个 JavaScript 库，你可以把它放在你的网站上。它允许你使用代码定义和触发弹出窗口。目的是将更多的访问者转化为用户。在基本版本中，它是免费的，一次性支付 9 美元的计划支持着制作者[丹尼·波斯特马](http://www.dannypostma.com/?ref=PeterThaleikis.com)度过黑夜😀️
