# 每日挑战#32 -隐藏电话号码

> 原文：<https://dev.to/thepracticaldev/daily-challenge-32-hide-phone-numbers-ham>

对于这个[挑战](https://www.codewars.com/kata/5953e4a29e93cef2e90000aa)，写一个加密电话号码的函数。该方法应该隐藏每个电话号码的后六位数字。

> **例 1** (使用美国电话号码)
> *IN:*`201-680-0202`=>*OUT:*`201-6XX-XXXX`

您还可以让代码检查电话号码格式的有效性。

> **例 2**
> *入:* `145-201-680-0202` = > *出:* `false`

请随意将此挑战应用于美国以外的电话号码。
