
**安装**

```
$ git clone https://github.com/Drozerah/cli-starter.git 
```

然后全局安装:

```
$ npm install -g 
```

完成此步骤后，您需要:

```
$ npm link 
```

这个命令将在本地注册您的 CLI 项目，因此您现在可以在终端的任何地方运行下面的命令:

```
$ cli-starter 
```

该命令将在终端中提示一条消息，表示安装 *[cli-starter](https://github.com/Drozerah/cli-starter)* 一切正常:
