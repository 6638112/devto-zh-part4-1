# 重叠图像库视图

> 原文:[https://dev . to/connect Steven/overlap-image-gallery-view-4235](https://dev.to/connectsteven/overlap-image-gallery-view-4235)

OverlappingImageGalleryview 是一个灵活的库，可以帮助你为 Android 应用程序创建重叠图像。

你可以将它与最流行的图像库集成，如毕加索壁画，或滑翔。

想了解一下流程？你可以在这里查看代码:[重叠图库视图](https://github.com/Mindinventory/overlap-image-gallery-view)
