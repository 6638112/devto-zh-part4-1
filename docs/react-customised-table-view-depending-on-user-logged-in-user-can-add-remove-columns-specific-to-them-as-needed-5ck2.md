我想知道是否有人能帮我解决这个问题。

我有一个设置表。尽管内容不同，但每个用户都有相同的起始基本设置表，具有相同的列数。它将列 A 和列 B 作为该表的维度。

我有多个用户。

用户 1 可能想要通过新的列或维度添加到具有定制标题的表格中，他们可能想要 1 或 2 或 3 个新的列/维度

用户 2 可能希望向表中添加 10 个新维度，并为列添加他们自己定制的标题。

用户 1 和用户 2 的单元格内容会不同。

在存储方面，我将如何管理它？您会将用户 1 和用户 2 的“设置”分别存储在 Postres 数据库中，然后在用户选择表格组件时加载它们吗？

在这之后，我就有了一个事实表，用户将能够在其他设置表中为他们的事务用户标记唯一的维度。

最终，根据登录的用户，最终的设置表看起来会完全不同。
