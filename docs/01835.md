# 我刚刚使用 npkill 清理了 7 GB 的磁盘空间

在使用这个工具之前，您甚至不知道 node_modules 文件夹占用了多少空间！

npkill 将搜索您运行它的所有子文件夹，并允许您轻松清除 node_modules 文件夹。

当您需要再次运行文件夹中的项目时，您可以简单地运行 npm install 来恢复依赖项。
