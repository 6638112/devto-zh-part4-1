# 我第一次在聚会上讲话！

> 原文：<https://dev.to/valevstech/my-first-time-talking-at-a-meetup-5dhc>

正如标题所言，我在 meetup (Ruby Peru)上做了我的第一次闪电演讲，现在我仍然对此激动不已！那是一个多月前的事了，但是你知道...拖延是一件事。不管怎样！这是我一直想和大家分享的帖子。

我谈到了 Rails 上的模型级验证，因为我发现这是一个非常有用的主题，并且在创建 web 应用程序时可能需要。让我印象最深的是这样一个事实:我们不能信任用户，所以我告诉自己:“为什么不分享这一点，分享我们阻止邪恶用户破坏我们网络应用的方法呢？如果你问我，我会觉得《T4》真的很有趣。”因此...就是这么回事:)

非常感谢我的导师们，是他们让这一切成为可能:)尽管一开始我有点不确定，但一切都很顺利，:D，这绝对是一种荣誉！
