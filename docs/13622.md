> 原文:[https://dev . to/SF writer mdk/advanced-objects-course-comparisons-3 lch](https://dev.to/sfwritermdk/advanced-objects-course-comparisons-3lch)

碰巧的是，我在 Udemy 的三门 JavaScript 课程中处于相同的位置。这些是:

*   [现代 JavaScript Bootcamp (2019)](https://www.udemy.com/course/modern-javascript/)

*   [高级 JavaScript 概念](https://www.udemy.com/advanced-javascript-concepts/)

*   [JavaScript:理解奇怪的部分](https://www.udemy.com/understand-javascript/)

我将要学习和比较的主题是高级对象。我将花时间研究这些材料，并尽我所能给你们带来最客观的总结。

我想这将需要一周或三周的时间来浏览材料并写一篇关于它的帖子。

任何人都可以阅读该摘要，尽管它是面向自学者的，这些自学者希望先睹为快，了解 JavaScript 中同一主题的三门不同课程的异同。

也许你正在考虑服用一片或多片。记住，我也是一个自学者。我不期望你学到任何关于对象的新东西，尽管如果你学了，那会很棒！

我的目标是给你一个关于这些不同的教师的教学风格的提示，并向你展示我在理解材料时遇到的问题以及我对解决这个问题的想法，这通常包括在网上搜索更多的资源。

这些课程并不是 JavaScript 高级对象的唯一指南或教程。很明显！外面有这么多优秀的资源。我希望向你展示如何使用多种资源来增强你的学习体验。
