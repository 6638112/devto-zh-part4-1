# 需要了解机器学习吗？

> 原文：<https://dev.to/gilad/discuss-do-you-need-to-understand-machine-learning-2mob>

虽然大多数程序员能够在他们的代码中利用简单的机器学习或深度学习库/模块(例如 [scikit-learn](https://scikit-learn.org/) ),但许多人并不知道这些库是如何在幕后工作的，它们为什么工作等等。

很明显，如果一个软件开发人员想从事数据科学方面的职业，他需要学习算法和模型(以及相关的数学)。但是，*偶尔*需要在他的 web 应用程序中实现人脸识别/狗检测/你能想到的普通程序员怎么办？

你认为他需要**理解**他使用的模型吗，如果需要，理解到什么程度？或者仅仅理解算法的输出就足够了？
