# 计划误入歧途

> 原文：<https://dev.to/arleneandrews/plans-gone-astray-3om1>

有时候，事情会发生。我们怀着最美好的愿望来看待这种情况，但它就是不起作用。这篇短文花了我三个多小时才完成。

在抓住一个直接在肩关节上滑动的盒子(比预期的要重)后，我的打字受到了严重的限制，手写和画画更是如此。这快把我逼疯了。

我想说我已经做了大量的研究，并继续前进——但我没有。我一直高度关注一系列采访，试图梳理出那些使用另一种计算机语言或不同框架的人给我的有用信息。其中一个人非常乐于助人:让我想去键盘前打字，但是我不能。这让我和她都很沮丧。

我有一个更长的帖子，所以我希望下一个帖子能提供更多信息！
