# 如何修复 Wordpress 中的白屏问题

> 原文：<https://dev.to/maxwoody095/how-to-fix-the-white-screen-problem-in-wordpress-o2m>

WordPress 是最流行的网站构建器之一，广泛用于构建网站。它是当今用户的首选。在使用它时，你可能会突然遭遇到死亡的白屏。它使得访问者和管理员都无法访问。在这里，我们将讨论一些常见的错误，出现在 WordPress 白屏的死亡，最可能的原因和快速排除故障的解决方案。要修复 Word Press 中的白屏错误，您可以立即致电 [WordPress 支持](https://www.wordpresssupport.co/)团队。

## 为什么 WordPress 会出现白屏问题？

一个 WordPress 白屏死机主要是 PHP 错误或者内存极限耗尽造成的。首先，你需要决定你的网站管理员是否在工作。如果你的网站的前端被拉下，并且管理员正在工作，你有一个有缺陷的主题或插件的机会更高。在谷歌 chrome 等浏览器的基础上，死亡的白色屏幕可能会显示不同的结果。这实际上提供了一条警告消息，说明“此页面不起作用，无法处理应用程序。

## 解决 WordPress 中白屏问题的简单方法

### 1-清除你的缓存

清除你的缓存是修复 WordPress 中白屏死机的简单方法。当您可以轻松访问管理面板时，不幸的是网站的前端和后端出现了黑屏问题，您可以通过某些工具清除缓存。这个问题主要是由缓存插件引起的，可以通过管理面板快速解决。

### 2-检查其他网站问题

当你的服务器上又多了一个 WordPress 实例时，仔细检查一下其他网站是否也有类似的问题。它可以帮助你排除几种可能性。例如，一个 [WordPress 网站](http://wordpress.org/)上的问题，而不是其他的问题，会导致-

有缺陷的插件
糟糕的主题
自定义编码问题
黑客或恶意软件泄露文件
如果你所有的网站都显示白屏，服务器可能有很多问题。在这种情况下，您需要联系虚拟主机提供商或网络管理员。

### 禁用插件

错误的插件会给网站带来各种各样的问题。无论几个开发者试图创造兼容的一切，这些附加组件经常会彼此分裂。因此，网站可能会停止工作。有很多方法可以停用插件-

管理面板-你需要登录到管理面板，并前往“插件”按钮，并停用它们。

### 通过 FTP 删除

如果你不能登录到管理面板，你可以在 FTP 的帮助下删除插件。从这个问题中走出来可能是一个非常容易的方法。不过，如果你有任何问题，你可以立即打电话给 WordPress 支持团队。
