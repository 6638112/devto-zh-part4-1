# 为什么你办公室的温度比你想象的更有影响力

> 原文:[https://dev . to/tech globex/why-the-temperature-in-you-office-than-you-think-1l2j](https://dev.to/techglobex/why-the-temperature-in-your-office-is-more-influential-than-you-think-1l2j)

如果你在经营一家企业，或者领导一个部门或团队，可以合理地假设你至少会花一些时间评估关键指标或产出。这些输出指标可能包括销售完成、销售线索生成、里程碑事件、MVP 发布或一系列完全模糊的指标，这些指标只与您的特定行业或部门目标相关。不管怎样，俗话说，如果你不是在评估，你就是在猜测。

但是输入呢？测量某些输入可以很好地预测输出。例如，大多数组织测量缺勤率。如果人们不工作，他们就不太可能产生所需的产出。这是个简单的公式。销售电话也是如此。虽然作为拜访次数函数的成交数量是关键指标，但拜访次数是产出的重要预测指标。

然而，有些投入尽管一直是产出的可靠预测指标，却不太经常被衡量。舒适就是这样一种输入。不舒服的员工不是高效的员工。他们注意力分散，没有动力。伦敦一家物业维护公司在 2019 年进行的一项研究揭示了员工工作场所不适的主要来源。这些问题包括通风不良、干扰性噪音、难闻的气味——所有这些都是工作中低级分心的常见来源。然而，令人分心的不适的一个真正突出的来源是温度。几乎一半参与调查的人说他们的工作场所因为环境温度而不舒服。

卡里·库珀爵士 CBE 教授是关注工作场所的组织心理学和健康专家。他认为，在创造舒适的工作环境方面，雇主们没有把最基本的东西做好。“一个身体不适甚至不安全的工作场所显然不利于生产率和一个组织的外部形象。干扰性的噪音、令人不适的温度和普遍的衣衫褴褛等问题可能会分散注意力，但这也是雇主对员工的尊重。

“雇主不应该仅仅满足健康和安全要求，还应该确保工作场所尽可能舒适宜人。那些没有这样做的人向他们的员工发出了一个信号，即他们的身体舒适和不受干扰是次要的，这对任何员工来说都是令人沮丧的。这并不是说组织需要投资于高规格的装修和豪华的环境，而是应该专注于满足适合其行业的基本环境舒适度标准。”

太热或太冷都不利于生产力。其他研究证实了这一点。2019 年，德国研究人员发现，当女性被要求在不舒适的温度下工作时，她们在基本的数学和打字任务上的表现不如在舒适的温度下。由 [Aspect](https://www.aspect.co.uk/trades/air-conditioning-refrigeration/air-conditioning-in-the-workplace/) 进行的 2019 年研究还发现，女性受到不舒适温度的影响不成比例。参与研究的女性中，超过一半的人抱怨工作场所太冷，相比之下，仍有 42%的男性抱怨工作场所太冷。

所以下次当你考虑你是否在你的生意中测量正确的东西时，看看那个恒温器。而且，征求你周围人的反馈。它们舒服吗？你的团队中有没有人裹着毛衣试图抵御空调的寒冷？上述两项研究表明，在不损害生产率的情况下，采取一种气候适合所有政策几乎是不可能的。

但这并不意味着每个人都需要经历不适。企业现在应该寻找建立不同气候地区的方法，以适应所有的偏好。办公室已经有了安静区、休息区和协作区。生产力的下一个前沿领域很可能与环境温度有关。
