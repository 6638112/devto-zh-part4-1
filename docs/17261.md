# 一小时汇编编程视频。

> 原文:[https://dev . to/John found/video-of-one-hour of-assembly-programming-4h9m](https://dev.to/johnfound/video-of-one-hour-of-assembly-programming-4h9m)

这是我的汇编语言编程课的一个小时的视频。

很多时候，当我在讨论汇编语言编程时，我有一些感觉，对手根本无法想象整个过程，并认为这是某种非常不同的(而且非常难！)来自其他语言。

所以，我决定把我的编程过程截屏，以便对我在帖子中写的编程有一个大致的了解。

此视频显示了在新 IDE v3.x 项目中通过文件打开对话框打开多个文件的实现。

实际上，它是同时进行两个项目的工作:FreshLib 的 GUI 工具包(文件对话框和组件 TGrid 和 TListView)和 Fresh IDE 本身(它处于非常早期的阶段，我主要将它用作 GUI 工具包的测试项目。

GUI 工具包使用 FreshLib 的 OOP 宏库，因此，使用的一些指令可能看起来有点奇怪。

不幸的是，很多过程都留在了镜头后面——在我的脑海里。所以，你们中的一些人可能会觉得这个视频有点无聊。但希望你会喜欢音乐，如果不是视频。
