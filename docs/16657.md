# 发布 GraphQL:由李·拜伦、丹·斯查费和尼克·施勒克主演的纪录片

> 原文：<https://dev.to/honeypotio/release-of-graphql-the-documentary-w-lee-byron-dan-schafer-nick-schrock-1e11>

[https://www.youtube.com/embed/783ccP__No8](https://www.youtube.com/embed/783ccP__No8)

GraphQL:在 GraphQL 社区的帮助和支持下，这部纪录片在旧金山和柏林拍摄了几个月。

由李·拜伦、丹·斯查费和尼克·施勒克以及其他有影响力的 GraphQL 社区成员主演的《GraphQL:纪录片探索了 GraphQL 是如何以及为什么会出现的，并强调了开源社区的力量。

GraphQL 的成立是为了解决一个特定的问题，这个问题不仅影响了脸书(它最初是在那里开发的)，还影响了包括 Twitter、Airbnb 和 Github 在内的许多其他科技公司，这些公司都受到了关注。

我们制作开源纪录片，因为这是我们回馈我们和其他公司使用的开源技术以及围绕它们建立的社区的方式，也因为他们的故事令人向往，近距离观看令人惊叹。:)

这将是我们的第三部开源纪录片，继我们关于 [Ember.js](https://www.youtube.com/watch?v=Cvz-9ccflKQ&list=PLtEPUaeDclku1ECmuN3IsUimHApukWIOf&index=3) 和 [Elixir](https://www.youtube.com/watch?v=lxYFOM3UJzo&list=PLtEPUaeDclku1ECmuN3IsUimHApukWIOf&index=2) 的纪录片之后。

希望你喜欢它！
