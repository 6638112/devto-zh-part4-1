# 参加我的第一次聚会

> 原文:[https://dev.to/theryanfurrer/attending-my-1st-meetup-31a6](https://dev.to/theryanfurrer/attending-my-1st-meetup-31a6)

## [](#what-is-meetup)什么是 Meetup？

Meetup 的自我描述是“Meetup 是一个寻找和建立本地社区的平台。”这是一个网站，你可以在那里寻找你所在地区的团体，那里有很多和你有相似兴趣的人。这些小组倾向于以体育、即兴表演和表演为主题，或者以我昨晚参加的那个为例，以 JavaScript 为主题！

几周前，我加入了长岛 JavaScript 组，迫不及待地想参加他们的月度会议。所以，我们来多聊聊吧。

## [](#meetup-agenda)Meetup 议程

虽然每次聚会都是不同的，但据我所知，他们倾向于坚持某种议程。我参加的会议从下午 6:30 到晚上 9:00，包含两个演示文稿——第一个是关于 OWASP(开放 Web 应用程序安全项目)截至 2017 年的十大安全风险。第二个是当前 Bootcamp 参与者的第一个演示 web 应用程序的演示，这是一个使用 Ruby on Rails 和 JS 的锻炼跟踪器。

## [](#what-was-it-like)是什么样子的？

虽然这两次演讲都没有立即对我有用，但它仍然是一次很好的经历。我见到了真正的 web 开发者——在野外看到 web 开发者让我很兴奋。

社交是参加聚会的一个重要原因——昨晚 99%的与会者已经在这个领域工作，这可以在几个方面帮助我:

1.  那里的每个人都是我以前没有的新知识来源
2.  到了找工作的时候，认识能帮助我进入这个全新行业的人是至关重要的……对我来说是全新的。

如果你还没有参加聚会的话，光是人际关系网就足以让你开始参加聚会了。根据你的技能水平，有些内容可能会超出你的理解范围，但没关系！一个人应该参加这样的活动有很多原因。

人们很友好，容易交谈。没有人对我这样做仅仅几个月(在写这篇文章的时候)有丝毫的不屑。相反，每个人都非常支持我，这太好了！

> *附注:如果你还没有，请找到你所属的编码社区。超级傲慢的&令人讨厌的 1%正是所有程序员和开发人员的 1%。其他人都非常乐于助人、支持和鼓励，这正是**T2 所需要的。***

## [](#whats-next)接下来是什么？

我期待着更多的聚会！我已经找到了另一个名为 [Developer Deep Dives](https://www.meetup.com/developerdeepdives/) 的本地团体，我也非常期待这个团体。如果你害羞或社交焦虑，尽你所能走出去见见人！一个很好的第一步就是*参加*其中的一个，即使你不和任何人说话——你可以为你的第二个做准备。

我最终希望在一次集会上发言。作为一名前职业音乐家，在众人面前表演(或演讲)并不会困扰我。事实上，在那种压力下我会表现得更好，所以我非常期待呈现*一些*的东西。无论是我的作品集和项目，我正在开发的 JS 应用程序，还是谈论我觉得舒服的初学者话题。

在我短短几个月的旅程中，Twitter 上的这个社区、Discord、Meetups 等等已经给了我太多，我迫不及待地想要回馈。
