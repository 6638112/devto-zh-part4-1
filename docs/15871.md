# 1 年网络工程师经验。远离软件开发。

> 原文: [https://dev.to/djacales/1 年网络工程师-远离软件开发-47c5](https://dev.to/djacales/1-year-as-a-network-engineer-afar-from-software-development-47c5)

过去的一年，我做了一个最冒险的选择。我的一个朋友邀请我参加 CCNA 路由和交换课程。
我一直在软件开发界。所以在一个不同的地方开始新的旅程
是有风险的。

工作和学习新东西并不容易。在我的舒适区之外。但是我学到了一些现在可以应用的东西。回到我的软件程序。
也许我在网络的经历结束了。但是这种经历总是令人感激的。
