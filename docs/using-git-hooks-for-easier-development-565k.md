# 使用 Git 挂钩简化开发

> 原文：<https://dev.to/chrislydemann/using-git-hooks-for-easier-development-565k>

在这篇文章中，我们将介绍我推荐的使用 Git 钩子的方法，使用 lint-staged、TSLint 和 appearlier。我们还将回答这个问题:“什么应该作为 Git 钩子运行，而不是在 CI 上运行？”。
