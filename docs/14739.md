# 我得到了第一份软件工程师的工作！！！

> 原文：<https://dev.to/thekimmykola/i-got-my-first-ever-job-as-a-software-engineer-4l45>

我找到了一份软件工程师的工作！🥳🎊🥂🎈

今天，我非常兴奋地宣布我在 M&G 保诚集团的第一个职位&软件工程师助理。职业转变是一次冒险之旅；梦想真的会实现！

我以为没有计算机科学学位我永远也不可能做到这一点，但是支持我的了不起的技术社区证明了我是错的！

如果你有兴趣听到更多关于我职业转变的消息，请点击这里查看我的编码之旅问答。

## 特别感谢！

感谢迄今为止所有参与我的技术之旅的人！

我还想对 EVORA Global、Makers、Codebar、Code First: Girls、编写代码的女性、Tech，Inc .中的女孩、Tech for Good 和 Rails Girls 表示感谢，感谢他们支持我的技术之旅！

### EVORA 全球

我在可持续发展咨询公司的朋友是一群很棒的人，他们激励我每天去工作，做到最好。在咨询团队担任初级可持续发展顾问一年后，我有机会提出一些想法来发展公司的专有技术解决方案，该解决方案专注于可持续发展的房地产行业。该解决方案采用数据驱动的方法来支持商业房地产行业客户的环境数据管理和报告。

我非常感激公司对我的信任，给我机会成为他们新技术团队的一员。那时候这几乎是一张白纸，感觉有些东西是对的。虽然我犹豫着要不要离开我的可持续发展咨询工作，但我为自己能成为某件特别事情的一部分而欢欣鼓舞。

在两年多的时间里，我掌握了敏捷的概念，并通过实施敏捷产品/项目管理策略、业务需求收集和规范，与团队一起从事可持续性软件的软件开发。我学习了 scrum 迭代软件开发以及如何使用 Atlassian 的 JIRA 工具。

*感谢 EVORA 成为我技术之旅的一部分！*

如果你没听说过 EVORA，一定要看看他们的网站[这里](https://evoraglobal.com/)！

### 代码第一:女生

2017 年 9 月，我加入了《代码第一:女生 Web 开发初级班》，从此一去不返。我记得每周的一个晚上下班后，我飞奔到 Twitter 英国总部，筋疲力尽，但对编码感到如此兴奋和充满活力。我每周开一次会，内容涉及 HTML、CSS、JavaScript、jQuery、GIT、GitHub 协作、开发概念、Twitter Bootstrap 和响应式 web 开发，甚至有机会参与一个小组项目。

一结束，我就注册了 Code First: Girls Advanced Ruby 课程，在那里我了解了 Ruby 编程、Sinatra 框架、GET/POST 请求、开发概念、使用 Mailgun 的自动电子邮件、外部 API 和部署。我开始了解 Heroku 云托管服务上的应用程序部署，并探索了 Twitter API。

*先谢谢码:姑娘们！你真的帮我找到了激情。作为代码第一:女生校友，我觉得我可以改变世界！*

如果你没有听说过[代码第一:女孩](https://www.codefirstgirls.org.uk/)，一定要去看看他们的课程。他们有免费的社区课程和旨在让女性进入科技行业的专业课程。

### 制造者

当我在 2018 年末遇到[创客](https://makers.tech/)时，我确定软件工程适合我。我参加了**展示日**活动，被那里的学生创造的项目所震撼。对我来说， **Intro Cohort** 是一段很有用的时间，可以结识志同道合的人，一起编写代码。

进入创客对我来说就像家一样，我想象自己有一天会在那里。我实际上考察了英国的几个编码训练营，但被费用吓住了。幸运的是，我偶然发现了创客的奖学金项目并申请了！这是我做过的最好的决定。我的面试很有挑战性；尽管这是我参加过的最愉快的面试之一。
