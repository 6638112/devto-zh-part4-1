# 回应:当你面对一个编码问题时，你有没有感到完全无助？

> 原文：<https://dev.to/jacobmgevans/response-have-you-ever-felt-completely-helpless-when-facing-down-a-coding-problem-55a3>

原文：<https://dev.to/ben/have-you-ever-felt-completely-helpless-when-facing-down-a-coding-problem-5gad>

**TL；是的，这种情况偶尔还是会发生。尤其是当我无法控制工作中问题的依赖性或周围问题时。在这场马拉松旅程中互相扶持。**

当我在编码训练营时，无助或冒名顶替综合症更容易发生。即使周围有成百上千的其他学习者经历着同样的挣扎，即使在巨大的开放空间里所有的人都在你旁边；你周围有成千上万的学生、导师、老师和助教...然而，我有时仍然会感到无助，你可以要求的所有帮助实际上都在那里，你仍然会感到孤独和失落，有时确信你永远做不到。

长话短说坚韧不拔，用自我激励克服自我怀疑(无论你能从哪里获得)是前进的关键。不管事情变得多糟糕，不管有多困难，不管有没有骗子综合症...你推过去！让那些支持系统帮助你，让那些朋友和家人告诉你，不管听到你认为自己有多累，你都可以做到，让那些导师和老师带你一步一步地思考模型和过程。

让我们在最艰难的时刻相互扶持，即使是白板练习问题或企业级生产缺陷。
