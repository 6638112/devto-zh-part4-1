# 什么学位对上大学的前端开发人员最有用？

> 原文：<https://dev.to/silencerweb/what-degree-will-be-the-most-useful-for-frontend-developer-who-goes-to-college-54c1>

首先，我想说，我是一名自由职业的前端开发人员，在我 17 岁的时候有 3 年的经验，事情是这样的，我**必须**去另一个国家(加拿大，如果这很重要的话)上大学。是的，我**有**。

我的问题是我不能决定我应该选择哪个教育项目。我还没有发现任何只针对前端的程序。我查了一些“网页设计和开发”项目，我可以肯定地说，我不会学习 XML、PHP 和 MySQL。

我在考虑计算机科学，但我是一名前端开发人员，我想在未来继续做前端开发人员，我想用 React 工作，而不是一些编程语言，我害怕数学课！但是我仍然要去上大学，所以我正在努力寻找最有用的课程，它将帮助我在这个领域建立一些良好的联系，并提升一些在未来生活中有用的其他技能。

任何建议，想法和意见都非常欢迎和感谢，谢谢:)
