# 边编码边看电视？分心还是动力？

> 原文：<https://dev.to/andrewbrown/tv-while-coding-distraction-or-motivation-4dgd>

当我编码时，我喜欢完全的安静，但不是所有的工作都是一样的。当我在做一件我已经做过上百次的事情时，我可以放心地让《星际迷航》作为背景音乐。

此外，并不是所有的节目都一样，我可以放心地把《星际迷航》放在背景中，因为我已经看过几百遍了。

如今，一个不受干扰的人越来越难找到了。我不得不容忍 elmo sing-a-long 在后台。
