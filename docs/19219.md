# 不要使用你雇主的电子邮件帐户来协商任何合法的事情

TL；大卫:因为他们可以删除一切。

我所说的雇主的电子邮件账户是指类似于`you@youremployersdomain.com`的东西。

我最近和以前的雇主分道扬镳了。一切都进行得非常顺利，我的前雇主非常好，非常有同情心。然而，过了一会儿，他们自然关闭了我的邮箱。我差一点就丢失了一些重要的文件，比如合同、硬件收据、促销协议等等。幸运的是，我的电子邮件客户端上有所有内容的副本，我可以保存备份。但这让我意识到，我的雇主一直对我的账户拥有绝对的权力。我相信我以前的雇主有这种能力，他们是很好的人，但是当你知道的时候为什么要相信呢？

显然，使用公司的电子邮件进行工作相关的交流是正常的，但当涉及到谈判你的合同时，对方不应该有权删除你的论点。当你和你的雇主谈判你的薪水、合同或任何不共享的利益时，使用你的个人电子邮件。

这对于远程员工尤其重要，因为他们的合同和谈判大多是数字化的。
