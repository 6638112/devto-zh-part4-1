# 每日挑战#60 -找到丢失的字母

> 原文：<https://dev.to/thepracticaldev/daily-challenge-60-find-the-missing-letter-49bn>

编写一个方法，将一个由连续(递增)字母组成的数组作为输入，并返回数组中缺少的字母。

你总是会得到一个有效的数组。总会少一个字母。数组的长度总是至少为 2。该数组将总是只包含一种情况下的字母。
