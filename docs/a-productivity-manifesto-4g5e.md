# 生产率宣言

> 原文：<https://dev.to/pascal/a-productivity-manifesto-4g5e>

*本文最初发表于[我的个人博客](https://pascalprecht.github.io/posts/a-productivity-manifesto/)。在那里找到更多有趣的内容。*

在阅读了相当多关于生产力、生活方式和正念的书籍后，并在日常生活中应用、整合和实践了一段时间我的知识，我认为写下我对一天中做好事情的想法是有用的。

那里有大量关于生产力的文章和资源，我确信我的一些经历与你在别处读到的有所重叠。然而，这不应该只是另一个关于生产力的帖子。

相反，**这是我的个人宣言**,我可以回头看，提醒自己如何保持专注，保持在正确的轨道上。这份宣言只包括几条规则，而这些规则被分成更小的部分。我努力让它尽可能集中和切题，不受干扰和不重要信息的影响。

你也可以把它作为你的个人生产力宣言。我会保持这一个更新，并且随着我的进展和感觉需要而改变它。

## 生活方式

为了最大限度地利用我们自己，我们的身体和思想必须处于良好的状态。这就是为什么我认为让我们的个人身心健康成为全天高效工作的重中之重是非常好和重要的。

1.  平静的头脑是强大的头脑
    如果我们不能清晰地思考，我们的头脑被太多分散注意力的想法、想法和情绪淹没，就很难完成任何事情，也很难专注于正确的事情。我们也很难以一种合理的方式引导或跟踪与同事的讨论。每天冥想一两次不仅有助于我们保持头脑清晰，还能让我们欣赏“现在”，并对我们所做的事情更加专注。也不用很久。即使是 5 分钟的当下，也会有很大的不同。

2.  **定期锻炼**
    每天锻炼至少半个小时已经有助于给我们更多的能量，因为它能促进血液循环，减轻压力。无论是全身锻炼还是仅仅是一次提高心率的跑步，这都不重要。保持纪律性，定期做。我个人不是那种可以去健身房举重的类型。我一点也不兴奋。这就是为什么我大约一年前开始巴西柔术，从那以后我每周训练 3-4 次。找到适合你的方法！

3.  **睡眠**
    说到生产力，休息是最重要的因素之一。事实上，睡眠提高了我们的记忆力，巩固了我们一整天所学的知识(这就是为什么睡前阅读可能是最好的阅读时间)。缺乏睡眠和休息永远不会对我们有好处，即使我们认为通宵工作会更有效率。它只会伤害我们的身体和健康。试着每晚至少睡七个小时。

4.  保持水分
    我们的大脑在洗热水澡时工作得最好。喝大量的水是更好地集中注意力的关键。此外，当我们整天坐在电脑前时，它有助于防止头痛。所以，喝水。很多。或者茶。

## 聚焦和执行

想法和完成事情的意图是伟大的，但只是故事的 10%，如果我们不坐下来做我们想做的事情，我们将一事无成。另外 90%是自律和全天保持专注的艺术，所以我们可以一步一步地接近我们的目标。

这是我坚持下去的动力。

1.  根据你典型的一天是什么样子，以及你要处理多少不同的事情，把你的想法写在纸上，给你的大脑腾出空间来集中注意力，这可能会很有帮助。我开始写子弹日记，这对我很有帮助，让我清楚地知道我必须做什么，我每天、每周和每月的目标是什么。如果你在工作中经常和你的团队开会，记录你所做的事情也是非常有用的！加分点:它让我放松。

2.  **关注“现在”**
    我们常常担心过去发生了什么，以及我们在不久或遥远的将来可能会面临什么样的挑战。接受这一点并不重要。专注于当下的事情，以及在当前情况下需要做的事情。然后继续下一件事。

3.  从最重要的任务开始
    当你知道你需要或想做的事情很无聊、很难或者只是让你脱离了舒适区时，你很容易拖延。在这种情况下，我们通常会开始欺骗自己，做一些现在不重要的事情，这些事情不会像此刻真正重要的任务那样有太大的影响。不要回避它，而是从最重要的任务开始你的一天。你花一整天在上面也没关系。往往最重要的也是最难的任务。但这也是推动事情向前发展的因素。

4.  **开始小**
    你不必一次做完所有的事情。你可能甚至不能，这也没关系。从小处着手。**走一步看一步。迭代。**一旦你开始，事情就会变得容易。

5.  说不
    说不没关系。我们不必一直都有空。我们不必马上回复那封邮件。我们不必每年都在这个会议上发言。我们不必参加每一次会议。消除一切让你无法完成自己重要任务的干扰。**少即是多**。

**就这样**。我知道，这些都是很少很简单的规则，但是其他声称让我更有效率的事情都可以很容易地从这些规则中推导出来。如前所述，我会保持更新，并在尝试时添加新的规则。
