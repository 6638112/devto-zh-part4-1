# 告诉我们你对开源的想法

> 原文:[https://dev . to/digital ocean/tell-us-your-thoughts-about-open-source-5fd 8](https://dev.to/digitalocean/tell-us-your-thoughts-about-open-source-5fd8)

开源软件无处不在，为全世界各种规模的应用程序和小工具提供动力。如果你从事技术工作，尤其是如果你是一名开发人员，你肯定每天都在使用大量的开源软件。您可能已经使用 GitHub 检查了一些代码，或者了解了如何使用某个库/ API。但是你有没有问过自己**为什么世界上有那么多人花费那么多时间来构建和贡献开源项目？**

我们想听听你的意见。你为开源做过贡献吗？你做那件事的主要动机是什么？这对你的职业生涯有积极的影响吗？你最大的挑战是什么？

我们将向参与我们关于开源的最新调查的每个人发送限量版的 Sammy 贴纸。花几分钟时间分享你对这个主题的想法，帮助我们了解并报告 2019 年开源的状态。这正好赶上即将到来的 Hacktoberfest。

此外，我们欢迎任何人在 Twitter 上分享他们的#OpenSourceStory，或者在本帖中发表评论。我们将在我们的社交媒体渠道上展示最好的评论，以便让人们了解开源工作的好处和困难！
