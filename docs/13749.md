# 优秀管理者的三个方面

> 原文：<https://dev.to/kyleljohnson/three-aspects-of-a-good-manager-cng>

在我漫长的职业生涯中，我遇到过好经理和坏经理。我试图把好的方面/品质从好的方面/品质中提取出来，使它们成为我管理风格的一部分。迄今为止，根据我的年度团队调查，我是一名优秀的经理。我把我的伟大归功于三件事。

# 1.公平

经理应该对他们的员工非常公平。他们不应该以不同于自己的标准来要求人们承担责任。

# 2.沟通

经理应该定期(每月)与员工交流。在一个月中留出时间来倾听你的团队成员的问题、担忧或抱怨。

# 3.视力

这很重要，并且伴随着交流。管理者应该有远见。你希望你的团队达到的乌托邦式的(长期)目标。目标应该是对团队有明确而巨大的好处，而不是对管理者。达到目标的奖励也应该被传达。经理应该始终如一地传达这一愿景，并将其保留在团队的记忆中。

# 结论

在我的职业生涯中，我遇到过苛刻的经理，但如果他们公平并倾听，我会欣赏他们。你可以为世界上最好的公司工作，赚很多钱，但如果你有一个糟糕的经理，那就惨了。另一方面也是如此。我在一个工资低的地方工作，但因为我有一个好经理，所以我留了下来(直到最后)。

如果我错过了什么，请评论。
