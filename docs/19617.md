# 介绍

> 原文：<https://dev.to/raulcote/introduction-edo>

开始吧，因为这是我的第一篇文章，所以我要介绍一下我自己。我的名字是劳尔，我出生在阿尔赫西拉斯(加的斯)，也许你想知道这个地方在哪里，它在世界的尽头，至少我是这么觉得的。实际上，它在西班牙的尽头，是乘船去非洲的最后一站。安达卢西亚的尽头，地中海和大西洋的交汇处。

现在我 33 岁了，住在一个稍微有名一点的城市，巴塞罗那。这样一个伟大的城市，却像所有大城市一样更加紧张和困难。这也是一个昂贵的工作，你必须做很多工作，但当你寻找鼓舞人心的东西时，这也是值得的。

当我十几岁的时候，回到 Geocities 和 Soulseek 的日子，我曾经制作了一个朋克摇滚网站，并开始学习基本的 HTML，那是一段有趣的时光。我认识很多人，我的网站也做得很好，我持续做了将近 3、4 年，但是当博客时代迟早到来的时候，我停止了，直到一年前我才忘记了 web 开发。在我大学学习期间，我获得了图书馆学学位，我遇到了我的女朋友，我在一些朋克摇滚乐队演奏。我从来没有做过图书管理员，相反，当我搬到巴塞罗那寻找更好的机会时，我开始在一家西班牙银行工作，那真是一段有趣的时光，我从来没有想过我能享受它，但我做到了，我的同事很棒，我非常擅长它，我有很多美好的回忆，但一段时间后，工作本身变得单调乏味，我想改变，但我不知道该做什么。

有一天，我在 2018 年(或者更早一点，我真的不记得了)在网上随机看到 Windows Vista 即将停止支持。我讨厌那样，我从 10 年前就有了我的笔记本电脑，它对我当时做的事情来说还是可以的，所以我开始思考我能做什么。我当时身无分文，没有机会买新的，我也不想买，所以我考虑安装另一个操作系统。

一段时间后，我决定尝试 Linux，我决定去 Linux Mint XFCE，我喜欢它，它真的改变了我，有几天我觉得自己像一个黑客与这个美丽的终端的事情，几个月后，我开始更多的控制，我想更多，更多的时间在终端上，更多的东西要学，我需要编码，所以我决定也许我必须学习一门编程语言。所以我开始搜索，并决定显而易见，JavaScript 似乎是一个新手的好选择。我已经知道一些基本的 html，所以它不会很难，我喜欢制作网页。我玩了 Codecademy，感觉真好。我决定自己一个人去，但几个月后我意识到我做不到，我几乎没有空闲时间，有时下班后我很累。我读了很多关于训练营的东西，这很可怕，也许它不起作用，但在一天结束时，我跳上了马车。我参加了 Ironhack，那是一段美好而有趣的时光。我研究了很多，学到了很多，完成后几周，我开始作为 React / Redux 前端开发人员工作。

这太可怕了，我只有基础水平，我可以处理这些任务，但是太紧张了。现在 6 个月过去了，我觉得我做出了正确的选择。我喜欢编码，我喜欢 React 和它的整个生态系统，我也喜欢围绕它的整个社区。我在空闲时间玩盖茨比，我在工作时玩 React、Redux 和很多 Canvas，我也玩很多 Javascript OOP。我仍然每天都在阅读和学习。
我获得了高超的技能，做了一年前我做梦也想不到的事情。如此激动人心的时刻，我甚至喜欢 CSS。现在，我也想加入这个伟大的社区，我想把我所获得的一切回报给这个社区，我想用我的作品帮助别人。

所以我来了，在我 30 岁的时候做了一个巨大的改变。我想现在这种事经常发生，但我对此感到自豪。永远不会太晚。
