## 20 大数据工具

## [](#1-apache-hadoop)1。Apache Hadoop

它是一个库框架，允许我们在不同的计算机集群上对大型数据集进行分布式处理。它可以扩展到处理数千台服务器。它可以检测故障并在应用层处理它们。

**特性**

*   用户可以很容易地在分布式系统上编写和测试。

*   它自动地在机器之间分配数据，并且可以利用 CPU 内核的并行性。

# [](#2-apache-spark)2 阿帕奇火花

根据定义，它是一个快速、开源、通用的集群计算框架。‘API’可以用 JAVA、Scala、R、python 语言开发。这个框架支持在不同的计算机集群上处理大量的数据。它可以扩展到管理和支持单个服务器到大型服务器。

Spark 可以覆盖大量的工作负载，如交互式查询、流、批处理应用、算法迭代等等。它可以减轻管理多个工具的负担。

# [](#3-apache-storm)3 阿帕奇风暴

这是一个开源的实时大数据计算系统，也可以免费使用。它可以分布式实时处理无限的数据流。

# [](#4-tableau)4 表

表格是有史以来最强大的工具，它有助于将原始数据简化成易于理解的数据集。Tableau 工作性质很容易被组织中任何级别的专业人员理解。它连接并提取各种来源的数据。

# [](#5-apache-cassandra)5 阿帕奇卡桑德拉

apache cassandra 可以有效地管理大型数据集，而不会影响它为您提供的可伸缩性和高性能。Cassandra 是容错的、分散的、可扩展的、高性能的。

# [](#6-flink)6 个好人

它也是另一个开源的分布式大数据工具，可以流畅地处理数据。

# [](#7-cloudera)7 Cloudera

更快、更简单、高度安全的现代大数据平台。它允许用户在一个可扩展的平台上从任何环境中获取数据。

# [t1】8 个 HPCC](#8-hpcc)

由 LexisNexis 风险解决方案开发。它在具有单一编程语言支持的单一平台上提供数据处理。

# [](#9-qubole)9 曲波

它是一个自主的大数据平台。将自我管理，自我优化，它允许企业专注于更好的结果。

# [](#11-couchdb)11 CouchDB

它是唯一一个在 JSON 文档中存储数据的大数据工具，它提供了超容错的分布式扩展。它允许通过治疗床复制工具访问数据。

# [](#12-pentaho)12 个五分体

这种大数据工具可用于提取、准备和混合数据。它为企业提供可视化和分析。

# [](#13-openrefine)13 开放精炼

Openrefine 也是另一个大数据工具，它可以帮助我们处理大量杂乱的数据。

# [](#14-rapidminer)14 Rapidminer

它也是另一个开源的大数据工具。用于数据准备、机器学习和数据模型部署。

# [](#15-data-cleaner)15 数据清理器

这是一个数据质量分析工具，在数据清理器中有一个强大的数据剖析技术。
