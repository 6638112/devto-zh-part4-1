# 简化物联网设备开发的容器化工具链

> 原文：<https://dev.to/supersophie94/containerized-tool-chain-to-simplify-iot-device-development-43kh>

VS 代码中 Azure 物联网设备工作台扩展中启用的容器化设备工具链的公开预览版今天发布！

整个目的是通过将编译器、设备 SDK 和基本库封装在容器中，为使用 Azure IoT 的嵌入式 Linux 设备(如 Debian、Ubuntu、Yocto Linux……)的设备开发人员简化设备交叉编译工具链的获取工作。你只需要安装或升级物联网设备工作台(v.0.3.1)和 Docker 运行时。和树莓派之类的目标设备。然后一切工作就像今天你正在使用一个本地环境。
