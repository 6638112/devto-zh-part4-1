# 使用 openCV 的最快图像扫描仪

> 原文：<https://dev.to/0xprateek/fastest-image-scanner-using-opencv-5f2g>

链接:[https://github.com/0xPrateek/Photoroid](https://github.com/0xPrateek/Photoroid)
photo roid 是最快、轻量级、易于使用的图像扫描工具。Photoroid 是用 OpenCV 和 Python 制作的。它可以用来从图像列表中找到一个相似的图像，而不需要花费太多时间。当要扫描的图像数量较大时，Photoroid 在查找相似图像时非常有用。这也有助于找到重复的图像。
