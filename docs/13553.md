# 错误/警告的旧颜色

> 原文:[https://dev . to/Tony/vs code-good-old-of-errors-warnings-colors-50gj](https://dev.to/tony/vscode-good-old-of-errors-warnings-colors-50gj)

将此字段添加到 settings.json 文件中，以返回版本 1.36 之前的颜色

```
"workbench.colorCustomizations": {
    "editorWarning.foreground": "#58995F",
    "editorError.foreground": "#B65E54"
} 
```
