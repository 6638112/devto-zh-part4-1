# 去和你的顾客谈谈

> 原文：<https://dev.to/samjarman/go-talk-to-your-customers-n59>

最近是我公司发布产品的第四个生日，昨天团队做了一些有趣的事情！除了在当地餐厅享用令人惊叹的午餐之外，我必须赞扬我的首席执行官和首席技术官所做的事情，我一直认为这是最好的领导力练习之一:他们带领整个团队(包括开发人员)亲自会见客户。我们了解了许多使用我们软件的角色类型，以及他们对软件的不满和他们的采用程度。

最好的事情当然发生了:我们发展了更多的同理心。我们很快开始在脑子里写代码来解决他们的问题，我们有周一要研究的东西的笔记。

开发人员经常忽略客户，这是可以理解的，但不是可以原谅的。这很有帮助。我 100%推荐在你工作的地方做类似的练习。

如果你读过[@ Adam grant](https://twitter.com/AdamMGrant)的《惊人的给予与索取》，你会知道当你向你的员工展示他们的工作成果时，动力会增加。许多研究从经验上证实了这一点。所以是啊，有趣的，激励人心的一天。谢谢老板！
