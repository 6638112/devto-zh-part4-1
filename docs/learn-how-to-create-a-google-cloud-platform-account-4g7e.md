# 了解如何创建谷歌云平台帐户

> 原文：<https://dev.to/educostadev/learn-how-to-create-a-google-cloud-platform-account-4g7e>

阅读时间:1 分钟

云提供商的魅力在于他们都有某种动机开始使用他们的服务。他们让事情开始变得简单。

谷歌云平台(GCP)免费层让你可以访问所有的谷歌云产品。你可以使用 VM 实例，Kubernetes 和其他很酷的服务，其中一些是永远免费的。新用户获得 300 美元的优惠券，有效期为 12 个月，用于在 GCP 服务中消费。[一些服务](https://cloud.google.com/free/docs/gcp-free-tier)在有限的使用范围内总是免费的，这足以让开发者开心。

利用这种激励并开始使用您自己云环境学习很酷的东西是值得的。在下面的视频中，我用 1 分钟时间展示了创建一个 GCP 账户是多么简单。在注册过程中，您的信用卡信息将被要求验证您的身份。免费期间你不会被收费，谷歌还有一个不需要信用卡的[模式。](https://cloud.google.com/free/)

[https://www.youtube.com/embed/bLXhykbdxpE](https://www.youtube.com/embed/bLXhykbdxpE)

一旦你有一个帐户，玩得开心！探索可用的服务，并测试您在在线教程中学到的服务。虽然在免费期间你不会被收费，但是你要养成使用后关闭服务的习惯，这一点非常重要。我建议创建一个账单提醒。通过这种方式，您将收到是否收费的通知。

不要忘记查看[另一篇文章](https://educosta.dev/blog/learn-how-to-create-a-billing-alert-on-google-cloud-platform/)并创建一个账单提醒。通过这种方式，您将收到是否收费的通知。
