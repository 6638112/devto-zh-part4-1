# 每日挑战#49 -美元和美分

> 原文：<https://dev.to/thepracticaldev/daily-challenge-49-dollars-and-cents-3en>

今天的挑战要求您创建一个函数，该函数将接受一个浮点数并返回以美元和美分格式化的金额。

> 比如:`6.2`变成了`$6.20`
