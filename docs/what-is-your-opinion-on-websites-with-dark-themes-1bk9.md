# 你对主题阴暗的网站有什么看法？

> 原文：<https://dev.to/spacerockmedia/what-is-your-opinion-on-websites-with-dark-themes-1bk9>

很长一段时间以来，我从不喜欢主题阴暗的网站，尤其是个人网站或博客网站。他们看起来都很糟糕。回过头来看，我敢肯定，他们可能有太多的方式与黑色背景的白色文本的对比。但是我拉不起来所以我不记得细节了。

多年来我只记得黑暗主题大多很糟糕。个人博客更是如此。

今天，我觉得我可能会动摇这个决定。我一直在为 DEV 使用黑暗主题，我喜欢它，尽管我认为可能会有一些微妙的改进(我不是设计师，所以我可能没有权利问这些会是什么)。但随着编辑器黑暗主题的变化，浏览器，终端，不和谐，等等；我觉得黑色主题会有更好的机会。

但是这种感觉很难被动摇。

为什么你认为深色主题是好的(尽管深色背景上的浅色文字看起来更舒服)？深色主题还有一个问题是在明亮的光线下更难看到，通常是在户外的手机/平板电脑上。

那么，你会如何改变我对糟糕网站的黑暗主题的看法呢？有哪些网站有很好的主题和很好的内容可视性？为什么他们总是对个人博客、产品或营销网站感觉如此糟糕(我曾经为一家有黑色主题营销网站的公司工作，但内部没有人喜欢它)？
