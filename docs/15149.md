# 杰克(可能)对被解雇的想法

> 原文:[https://dev . to/just zeros/what-jack-possible-thought-on-be-fired-4nbi](https://dev.to/justzeros/what-jack-probably-thought-about-being-fired-4nbi)

星期三，我提到我突然解雇了杰克，而不是给他一点颜色看看。

他没有领会我的“表现暗示”，所以他从来没有真正知道有问题。

或者，至少，他不知道问题有多严重。

我没有使用任何正式的警告程序(例如，正式的口头、书面警告)，所以我让他继续认为事情很好。

我不知道如果我发出哪怕一个正式的警告会发生什么。

也许这会引起他的注意，唤醒他，告诉他他有危险。

也许这就是他改变心意，扭转乾坤的全部代价。

如果你还记得我的 PIP 故事/播客，它给了我一个正式的警告，让我醒悟到现实。

对我来说，全心全意地决定“不惜一切代价”保住这份工作。

我不打算粉饰太平，杰克后来可能认为我是个混蛋。

这是一个愚蠢的举动，但这是我自己唯一的举动。

## 不要把自己封闭起来

学习使用员工纪律系统可能会让人不舒服，但后果会更糟。没有它们，你除了惊讶、恐惧和愤怒什么都没有了。

你如何对待一个员工会通过小道消息或玻璃门影响到其他人。

即使你已经暗示过，你的团队中有谁可能不知道他们正处于危险之中？

你如何做最仁慈的事情:清楚地表明他们处于危险之中，并公开讨论下一步措施？

你可能是老板，但你不必犯和我一样的错误。
