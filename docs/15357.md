**反应**

*试验 ReactVR 米凯拉·莱尔如何开始*

Angular 开发者米凯拉·莱尔对 React 产生了兴趣，因为 ReactVR 最近的发展。ReactVR 是为使用 React 创建 VR 应用程序而设计的，允许开发人员轻松构建 VR。

编写传统的虚拟现实应用程序涉及到大量的代码，因此很难跟踪事情的发展，也很难看到代码发生了什么。然而，使用 ReactVR，一切都按照 React 应用程序的方式进行组织，并且 VR 按钮等本机组件的存在使整个 VR 应用程序开发过程更加简单。

要开始使用 ReactVR，您可以下载包含代码的 zip 包，或者通过 npm 安装它。这些文档为示例项目提供了一个框架，因此您可以轻松地亲自尝试！

*使用来自 MLS 的 Peggy Rayzis 的开源项目增强您的 React 开发*

在美国职业足球大联盟(MLS)的一个小团队中，Peggy Rayzis 有机会使用 React 进行大量的 UI 实现工作。MLS 使用相当多的开源项目来构建他们的应用程序。

数据可视化，更具体地说，统计数据可视化，是 MLS 的主要关注点。由于该团队拥有大量关于球员、比赛和比赛统计的数据，他们一直在尝试寻找新的方式与球迷分享这些信息。MLS 使用的一个开源解决方案是 Victory(由强大的实验室提供)。Victory 是一个构建在 D3 之上的库，由可重用、模块化和可组合的 React 组件组成。该库几乎完全跨平台，95%的代码可以在 react 和 react native 之间重用。

React-Native-Create-Bridge 是一个自动生成桥模块的 CLI 工具。这是 Peggy 在 MLS 开发的一个开源项目。正如她在 npm 中所说，如果你是一个编写第一行本机代码的 JavaScript 开发人员，或者是一个希望从 React 本机工作流中消除样板文件的更有经验的开发人员，那么这个工具就是为你准备的。

Interact 是 Peggy 不仅使用，而且参与的另一个项目。Interact 是一个开源的电子应用程序，它允许你创建代码单元或降价单元，并在线运行它们。它类似于 Jupyter 笔记本，但用户体验更好。

React 开源社区发展迅速，随着 React-Navigation、React-Native-Web、Exponent 和 React-Router v4 等版本的发布而不断变化。
