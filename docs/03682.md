# 2019 年图书运输

> 原文：<https://dev.to/jmspace/2019-book-haul-5b5h>

我在 2017 年制定了一个目标，要多读书。我有更多的空闲时间。2018 年读了 2 本书。现在是 2019 年，我读了 11 本书。阅读可能是一件好事。它可以打发时间。前一秒你还在读一页，下一秒你就知道这本书已经读了一半了。

找到一本好书可能很难。在不掏钱的情况下接触新书会让这变得更难。我一直在用这个应用程序，Libby。这是一个通过当地图书馆的数字借阅系统，你可以在那里借书和阅读。这就像去图书馆一样，但是没有滞纳金。如果你使用 Kindle，你借的书可以在 Kindle 应用程序中阅读。
