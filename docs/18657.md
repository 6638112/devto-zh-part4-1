# 女人们，跟上程序

> 原文:[https://dev . to/cyd stumpel/women-get-with-the-programming-1078](https://dev.to/cydstumpel/women-get-with-the-programming-1078)

当你想到程序员时，你会想到什么？你能想象一个人坐在桌子后面，一只手用力打字，另一只手拿着无麸质百吉饼吗？

你想象的是男人还是女人？如果你想象了一个坐在桌子后面的人，不要难过，当我想到一个开发者时，我仍然是这样想象的。为什么编程会被感知为这样一个男人的世界？

我的老师实际上提出了这个问题，当时他问我和一名辅修 web 开发的女同学，为什么我们认为我们班 30 名学生中只有 5 名是女性。我们对此进行了思考，并说了这样的话:“辅修课被宣传为课程负担非常重”，“这可能是因为女性认为自己不够好”。当然，最终我们责怪社会让女孩缺乏安全感，让每个人都有性别偏见。

虽然我们对此一笑置之，但实际上让我震惊的是，在我整个上学期间，发展课程只有一位女老师。甚至我们在辅修期间每周谈话的讲师都是男性。

## [](#gender-biases)性别偏见

许多人认为编程是男人更擅长的技能，这当然不是真的。男人和女人有相同的认知能力，但是男人和女人如何看待他们的技能有一些关键的不同。

*世界上第一台计算机上工作的女性；埃尼阿克—陈楚翔*

这也与榜样有关，因为尽管阿达·洛芙莱斯被广泛认为是科学计算的创始人，格蕾丝·赫柏发明了第一台编译器，欧娜·胡佛通过开发计算机化的电话交换系统彻底改变了现代通信，但你不太可能在学校里学过这些。

根据最近的研究，性别角色、刻板印象和态度形成的年龄比以前想象的要早得多。

> “早在一年级，即使在数学成绩没有差异的情况下，女生也认为自己的数学能力不如男生。”(安东尼、考斯&沙拉拉出版社，2017 年)。

他们还指出，女孩认为自己的识字能力与同龄男孩处于同一水平。但是不仅仅是孩子，他们的父母也认为男孩比女孩更擅长数学。

> “这些自我形象的模式很重要，因为在幼年时表现出数学能力较低的孩子，在高中和以后的大学里，可能不太可能报名参加高等数学课程。”女生参与高等数学课程的减少似乎对有兴趣成为科学家、工程师或程序员的女性数量产生了直接影响(Antony，cause&Shalala，2017)。

# [](#self-image)自我形象

在大学里，编程入门是我学习的一门核心课程。据我的许多女同学说，这是最难的课程之一。就好像他们已经决定他们不够聪明去理解它，甚至在课程开始之前就已经放弃了。

当然，不是每个人都会成为编码界的传奇，但是只要有足够的信心和决心，任何人都可以学会。

很重要的一点是，女孩从小就被教导，她们在任何一门学科上都和男孩一样优秀。性别偏见不应该阻碍女性认为自己能做什么和不能做什么。第一步是提高认识。第二是创造一种心态，让我们不再把性别作为行为的借口。我们真的没什么不同。第三步是使用更多的女性榜样，邀请更多的女性来谈论和教授编程。因此，当一个人想到程序员时，他们不会自然而然地想到计算机背后的人。
