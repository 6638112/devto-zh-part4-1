# 通过网站建设者练习 HTML 对孩子和初学者来说很棒

> 原文：<https://dev.to/annebalad/practicing-html-through-website-builders-great-for-kids-and-beginners-7m9>

作为一名与开发人员和程序员一起工作的数字营销人员和 UX 从业者，我想在学习和职业发展方面实现的事情之一是知道如何编码。我认识的大多数开发人员和程序员都是自学的，如果他们能够从基础开始在线学习编码和编程，并最终学习复杂的代码结构，如 PHP 和 Java，那么我也可以这样做。

与 10 年前不同，现在有许多免费的资源和应用程序来学习如何编码——其中最基本的代码是 HTML。所以我想我可以先从那个开始，最终获得知识的构建模块，这将允许我理解更难的编码语言。

几周前，我通过学习平台 [Code Academy](https://www.codeacademy.com) 开始学习 HTML，这是一个为那些想在自学环境中学习编码或编程的人提供的编码平台。它就像一个在线学习教室，面向未来的网站开发者和任何想自学编码的人。编码模块包括 HTML、CSS、Java、R 和 PHP 等等。

他们的大纲很容易理解。他们先教你代码是做什么用的，给你举例说明在哪里用，怎么用，之后你就可以通过实践最终应用代码的编码原则了。练习区是学习模块中的一块空间，您可以在课程的每个方面结束后立即进入。不用说，这是一次很好的学习经历——您学习材料，看看它是如何应用的，然后通过实践学习来应用代码。

然而，对我来说，应用于平台上每一课的少量代码是不够的。我想我最终寻求的是一个顶点项目，我最终可以在一个页面或一个项目中应用 HTML 中的所有学习模块。

考虑到这一点，我决定通过一个网站构建器来应用我在 HTML 方面的学习。我选择了免费的建站软件，因为它没有网站升级的附加条件，而且平台是完全免费的。虽然有一个页面编辑器，您可以轻松地放置富文本和上传文件，但在构建网页时，您也可以选择使用他们的 HTML 编辑器。这对我来说是一个完美的平台，因为我的目标是在网页上应用 HTML 代码。

我能够很容易地通过 HTML 代码构建我的网页，并且能够看到整个过程。当我犯错误时，我会在网页上看到它们，然后自己改正错误。

当谈到用 HTML 或 CSS 代码创建自己的顶点项目时，我认为网站建设者是应用这些知识的一个很好的平台。对于初学者和希望通过创意项目看到自己成果的孩子来说，这很好。此外，他们可以自己做这个项目。因此，他们不再将网站视为家庭作业或编码要求，而是能够通过网站建设和创建自己的网站来感受 HTML 的乐趣。

另一方面，它也是一个很好的免费学习工具，附带 HTML 和 CSS 编码资源。我强烈建议初学者、业余爱好者和孩子都尝试一下。我现在正在享受学习 HTML 的乐趣，不久我也会在网页上尝试 CSS。
