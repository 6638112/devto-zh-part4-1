# 产品经理在开发团队中的角色是什么？

> 原文：<https://dev.to/pavelegorkin/what-is-the-role-of-a-product-manager-in-a-development-team-2369>

这是一个伟大的说法，他是他正在制造的产品的首席执行官。那是一段时间的事实。不再是了。在现在的大多数团队中，PM 除了他的专家权重之外，没有任何优势。这是在试图找到商业、发展和用户快乐之间的平衡。那么我们是谁，我们做什么，我们的角色是什么？

我们收集关于行业、开发、用户和利益相关者期望的知识。那些知道如何通过各种研究(市场，UX，任何需要的手段)找到信息的人

我们将这些积累的知识作为文档和答案提供给开发团队。

统治不是我们的职责。我们的目的是引导所提供的知识。有哪些我们不知道的，需要做哪些实验才能知道更多？数字和最佳实践是什么？作为一名专家，你承担了团队成员的认知负荷。如果你知识渊博，能够为自己的论点辩护，团队会支持你。

(提示:开发人员和企业都擅长数字)
