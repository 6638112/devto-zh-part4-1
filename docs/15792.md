# 别担心，我们和其他人一样沮丧

> 原文:[https://dev . to/aidiri/don-t-worry-we-re-just-as-depressed-as-else-6kj](https://dev.to/aidiri/don-t-worry-we-re-just-as-depressed-as-everyone-else-6kj)

网络开发社区中的心理健康。有问题。你甚至可以说这是有问题的。冷酷无情。至少，我看的所有关于它的文章好像都是这么说的。我开始研究这个行业中的抑郁和焦虑的话题，因为这是我自己遭受的痛苦。在我的情绪最近一次暴跌之后，我想把它写下来；一种自我治疗。所以，作为一个专业人士(书呆子)，我首先做了一些研究。我很好奇，在我自己说之前，想看看编码社区有什么要说的。

然而，我读得越多，我就越了解编码生活方式是如何让抑郁症成为可能的。编码工作是孤立的；我们整天坐在室内看着屏幕。这份工作压力很大，期限不断。冒名顶替综合症和职业倦怠非常普遍。抑郁症和更有创造力的人之间甚至有联系。网络开发可以是一项非常有创造性的工作。

说我感到担心……是一种保守的说法。有很长一段时间，我完全相信。我注定了自己的职业生涯将彻底摧毁我已经脆弱的精神健康。我将永远活成一个孤独、痛苦的艺术家，最后期限悬在头上，潜在的崩溃在每个角落等着我。是的，在众多悲观和沮丧中，有一篇积极的文章表明，也许，仅仅是也许，网络开发者并不像人们想象的那样沮丧。很明显，我马上就拒绝了。我已经向前看，看它到底有多糟糕。

stack Overflow 2018 年对网络开发者的调查发现，近 20%的开发者自我报告患有焦虑、抑郁或两者兼而有之。为了证明这一切到底有多糟糕，我寻找了一些关于普通人群心理健康的统计数据。相反，我发现了一个几乎相同的美国其他地区的统计数据。真烦人。

很明显这是不对的，所以我继续调查。我不断发现更多像另一个积极的博客一样的博客，类似地说网络开发者真的没有那么沮丧。事实上，一些人说，他们可能比其他工作领域的人更快乐。但是，如果这是真的，为什么要强调这个问题呢？

我可能同意一些文章的观点，认为这可能有点耸人听闻。但我不知道我是否同意 web devs 实际上比其他人更不抑郁。我不认为 web 开发人员比其他人更沮丧。原因如下:

做开发者不会让你孤独。强有力的研究表明，孤独与环境关系不大，更多的是与你的基因有关。孤独有遗传倾向。就像你被一群人包围时会感到孤独一样，独自工作并不一定意味着你孤独。

长屏幕时间是每个人的问题，不仅仅是开发者。随着智能手机和电脑的普及，说这是工作特有的问题是不公平的。如果开发人员因为整天呆在室内，长时间坐在屏幕前而更加抑郁，那么其他人也一样。

说工作压力大也有类似的问题，因此开发人员更容易因此而抑郁。它设定了一个预期，即你可能从事的任何开发工作都将是高压力的。除此之外，你的工作是否压力大取决于你的雇主对你的期望。

倦怠、冒名顶替综合征、隔离、长屏幕时间；这些都影响着科技行业之外的许多人。我在大学的时候就意识到了冒名顶替综合症。时至今日，我以前的大多数同学都感受到了这一点，并对此抱怨不已，不管他们是哪个领域的。这些都是不特定于任何一个领域的症状。

我想说的是:相关性不等于因果关系。还有:如果我们作为网络开发人员感到沮丧，那是因为我们和其他人一样沮丧。

因此，没有理由恐慌或相信成为一名网站开发者会增加你患精神疾病的几率。但这仍然是一个值得谈论和牢记的重要话题。你的心理健康和你的身体健康一样重要，甚至更重要。

作为一名网站开发人员，你能做些什么来保持心理健康？

*   管理你的身体健康。伸展一下，经常离开你的椅子，四处走走。关于这一点，趁着还有阳光，出去走走吧。
*   管理你的屏幕时间。是的，我们确实在我们的屏幕上工作，但是如果它占用了你太多的空闲时间，尽量限制你的社交媒体消费。
*   培养你编码之外的爱好。你的生活不必全是代码。让你的大脑每隔一段时间换一次车道。
*   把你的工作留在办公室。或者是那些可能把工作带回家的人的办公室角落。给自己设限，这样你就不会过度伸展自己，如果这是你可能会做的事情。
*   注意你的饮食习惯。注意到你吃了那盘意大利面后感觉有点虚弱吗？也许当你不需要解决问题的时候就把意大利面留下。
*   求助。生活是艰难的，有时我们只是耗尽了精力。不要害怕要求跳跃。
