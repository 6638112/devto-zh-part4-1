# 使 Github 动作有用的资源

> 原文：<https://dev.to/msurguy/resources-to-make-github-actions-useful-27ld>

随着新的 Github Actions 功能在全球范围内推出，我想花一点时间整理一个简短的资源列表，帮助您开始使用 Actions，并充分利用它们带来的强大功能。

以下是我建议你在开始行动时查阅的资源:

*   查看动作圣经，又名“工作流语法”来浏览动作的语法:[https://help . github . com/en/articles/Workflow-Syntax-for-github-Actions](https://help.github.com/en/articles/workflow-syntax-for-github-actions)
*   浏览“令人敬畏的行动”列表，寻找一些灵感:[https://github.com/sdras/awesome-actions](https://github.com/sdras/awesome-actions)
*   当您想要使用现有的动作或查找动作的示例时，请查看动作市场:[https://github.com/marketplace?utf8=%E2%9C%93&type = Actions&query =](https://github.com/marketplace?utf8=%E2%9C%93&type=actions&query=)
*   如果您正在将动作从旧语法迁移到新语法:[https://help . github . com/en/articles/migrating-github-actions-from-HCl-syntax-to-YAML-syntax](https://help.github.com/en/articles/migrating-github-actions-from-hcl-syntax-to-yaml-syntax)
*   要了解触发工作流的事件:[https://help . github . com/en/articles/events-that-trigger-workflows](https://help.github.com/en/articles/events-that-trigger-workflows)

希望这些资源对你有用，玩得开心，保持活跃！
