# 用 S.M.A.R.T .的方式管理你的计划

> 原文:[https://dev . to/vorakl/managing-your-plans-in-s-m-a-r-t-way-32m 6](https://dev.to/vorakl/managing-your-plans-in-the-s-m-a-r-t-way-32m6)

在规划中使用 SMART 标准的好处在过去几十年中已经为人所知。它们被如此广泛地应用，被整合为管理组织目标和目的的成熟技术，并被如此多的贡献者改进，以至于目前很难命名唯一一个“SMART”的确定定义。对于不同的人，这个首字母缩写词意味着不同的事情。甚至“目标”和“目的”这两个词，在某些情况下，也有相反或相同的意思。然而，原始想法的知识有助于从整个方法中获得最大收益。

1981 年，乔治·t·多兰的文章《有一种 S.M.A.R.T .的方式来写管理的目标和目的》发表。他合理地指出，尽管有所有可用的文献和研讨会，“*大多数经理仍然不知道什么是目标，以及如何制定目标*”“*大多数美国公司并没有真正有效的目标设定/规划过程*”，以及“*制定目标的过程是焦虑的主要来源，许多人希望没有*。

这是一个令人沮丧的现实，原因是各个层面的"*公司官员、经理和主管*"都缺乏适当的教育。乔治·t·多兰也提到，“*目标设定必须成为一种生活方式*，并提出了解决这一问题的有效方法。当要定义目标或写下目的时，你必须*想出首字母缩写词 SMART* ，并且清楚术语‘目标’和‘目的’之间的区别。

"**目标** *代表独特的执行信念和哲学。它们通常是一种持续且长期的形式*。"**目标** *，另一方面，给管理层的信念*以量化的支持和表达，"*使一个组织能够专注于问题，并给公司一种方向感*"，"*要实现的结果的陈述*"。

在最初的形式中，缩写词本身具有以下含义:

*   **具体**，针对某个具体领域进行改进
*   可测量的，量化或至少建议一个进度指标
*   可分配的，指定谁来做这件事
*   **现实**，陈述在给定可用资源的情况下，实际上可以实现的结果
*   **与时间相关的**，指定何时可以达到结果

考虑到所提出的解决方案是针对组织的这一事实，很少有“字母”改变其含义以满足较小的团体或个人的需求就不足为奇了。例如，对于个人使用，由于没有其他人可以分配，因此“A”和“R”将其含义更改为**可实现的**(类似于现实的)和**相关的**(适当且与上下文相关)。

有两点值得注意:

*   所提出的技术并不要求所有的目标都是可测量的或可量化的。在某些情况下，这可能导致“*为了获得量化*而失去更抽象目标的好处”
*   不要求设置所有五个标准。"*然而，我们越接近 SMART 标准作为指导方针，我们的目标就越明智*。

让目标和目的协同工作的一个方法是将它们加入一个行动计划。行动计划可以根据你的实际情况，通过短期目标来实现长期目标。
