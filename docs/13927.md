# 你的习惯是什么？

> 原文:[https://dev.to/emma/what-s-your-keystone-habit-7e4](https://dev.to/emma/what-s-your-keystone-habit-7e4)

我有几个我想养成的习惯——每晚睡 8 小时，每天读书——但有时我发现很难保持完成任何事情的意志力！

在《习惯的力量》一书中，查尔斯·杜希格提到了一个基本习惯的概念，即一旦养成了一个习惯，它就会和你的其他习惯产生连锁反应。

对我来说，如果我早点上床睡觉(结果自然会起得更早)，我会发现把一天中所有的习惯都划掉需要更少的意志力。当我开始工作时，额外的收获是感到高效和快乐！

你有拱心石的习惯吗？我很有兴趣知道！
