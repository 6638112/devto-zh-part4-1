## [](#what-is-ieo-)什么是 IEO？

首次交易所发行是一种类似 ICO 的筹资技术，但 ICO 和 IEO 的区别在于，IEO 依赖交易所进行整个筹资过程。在 IEO，经过合法化程序后的代币直接进入顶级交易所上市。

在交易所上市代币不需要单独营销代币，因为交易所已经有相当数量的交易者。交易所对整个 IEO 进程作出回应，甚至筹款也在交易所进行。

## [](#ieo-and-ico-difference)IEO 和 ICO 的区别

ICO 和 IEO 的主要区别在于，在 IEO，现有的交易所管理整个过程，而在 ICO，则由创造代币的机构来管理。

## IEO 的好处

### [](#centralized-boost-trust)集中-增强信任

IEO 的一个关键优势是，在这里，众筹是在一个交易所进行的，这个交易所就像一个中间人和交易对手，监控着每个项目的进程。交易所基本上筛选 IEO 项目的过程，以维护他们的良好声誉。这种集中化的再利用可以被视为与最初创造比特币和开放区块链的基本理念背道而驰，这是一个完全的 P2P 网络。

### [](#marketing-strategy-makes-ieos-more-easier)营销策略——让 IEOs 更简单

与 ICO 相比，IEO 的主要优势在于，交易所将拥有一个现有的观众群体，IEO 可以加以利用。该交易所现有的营销资源使得一个新项目更容易被更大的投资者发现，从而获得巨大成功。

### [](#legal-protection-increases-security)法律保护——增加安全性

代币创建者不必担心筹资安全，因为交易所将通过智能合约管理整个 IEO 进程。交易所将有一个组织良好的法律结构，使其成为监管机构，并为通常在其中处理 IEO 的新初创企业提供保护。一些发展成熟的交易所提供法律建议，并能接触到完整的法律团队，以帮助新创公司度过区块链行业的风险。
