# 寻找反馈

> 原文:[https://dev.to/taillogs/looking-for-feedback-45hn](https://dev.to/taillogs/looking-for-feedback-45hn)

最近，我在 Dev.to 上写了很多内容。显然，我知道每个帖子有多少人阅读，但这不一定是我应该制作什么内容的最佳指标。

相反，我想知道你们将来最喜欢看什么类型的内容。我的技能相当一般，所以不要觉得有必要建议我已经介绍过的话题，任何事情都是公平的游戏！

这也应该被认为是一个开放的论坛，给我关于我的写作(风格，内容等)的反馈，包括建设性的批评(不要隐瞒)。

我还想感谢在我开发的大约 1.5 个月里支持我的每一个人，这个网站有一个真正独特的社区。

TL；速度三角形定位法(dead reckoning)

你想看更多的主题是什么？你不希望看到的话题有哪些？
我的写作怎么提高？
我该如何改进？
还有别的吗？
