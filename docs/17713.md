# 作为一名老兵如何让我成为一名更好的开发人员

> 原文：<https://dev.to/meghanduprey/how-being-a-veteran-makes-me-a-better-developer-457c>

我来自纽约北部的一个小镇。我高中成绩很好，但是家里没什么钱，也不知道自己上大学是为了什么。我 17 岁的时候加入了延迟入伍计划下的空军。当我在 1998 年报名时，我并没有考虑爱国主义或为我的国家服务，那些事情是后来才发生的。军队是一条生命线，我要抓住它，尽可能地坚持下去。

在基础训练中，我们学习了空军核心价值观:

*   诚信第一
*   先服务自己
*   我们所做的一切都是卓越的

我在新兵训练营的时候学到了这些名义上的价值观，在以后的军队生活中也学到了这些精神上的价值观。这些核心价值观塑造了我作为一名飞行员和一名女性的形象。

我的兽医编程之旅即将结束，作为一名初级开发人员，我可以看到空军的核心价值观正在融入我的生活。当我发现问题时，我会解决它。当我解决这个问题时，我会尽我所能做到最好。而当我不知道怎么解决的时候，我就大声说出来！不知道答案的时候我不怕承认，因为我知道有时候团队合作才是更好的答案。

无论我是在做日常的代码挑战，与我的导师一起工作，还是尝试在这个过程中写博客，我都在实践这些价值观。我在军队的另一个特点是我的幽默感和融入新团队的能力。好奇和知道如何自嘲是学习如何编码的无价之宝。

编写代码的兽医是一次惊人的冒险。这是我的旧世界和新世界的伟大融合，我很感激我所学到的一切，包括技能和对自己的了解。我比我想象的要聪明，能够和他们中最好的人一起解决问题，我期待着下一步的发展。
