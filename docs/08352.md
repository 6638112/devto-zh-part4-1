# 什么是云原生？

> 原文:[https://dev.to/educostadev/what-is-cloud-native-44en](https://dev.to/educostadev/what-is-cloud-native-44en)

云原生是一个你以前听过的常见术语。它指的是设计为在云上运行的应用程序。不仅运行在云上，而且利用云平台提供的所有资源。这些应用程序被构建为本机使用 IaaS 或 PaaS 服务。这意味着组件是为处理故障、可伸缩性和可用性而构建的。这些应用程序的所有部分都是完全自动化的。

IaaS(基础架构即服务)侧重于计算节点、网络和存储。在这种模式下，企业可以利用云的弹性。可以根据传入的需求或负载轻松添加或删除计算节点。

采用 PaaS(平台即服务)包括识别企业的潜在平台功能，并用云提供商提供的等效平台服务替换它。这些服务使企业可以省去所有的运营工作。例如数据存储备份、可用性、可伸缩性和冗余性。他们可以用提供所有这些功能并专注于产品业务的托管服务来取代它们。
