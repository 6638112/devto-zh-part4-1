# 我从程序员愤怒中恢复过来

> 原文：<https://dev.to/jaymeedwards/my-recovery-from-programmer-anger-4496>

在反思我发布的帮助软件行业人士的内容后，我意识到我仍然需要帮助来解决愤怒问题 。🤦‍♂️

在过去的 5-8 年里，我基本上避开了社交媒体，当我开始发表我的想法时，我慢慢地被引诱到了愤怒文化中，而没有意识到这一点。

我参与过许多失败的项目。

我在家庭和身份上也有个人问题。

我制作了这个视频，为我的愤怒和我有时给人的“无所不知”的印象道歉。🙏

为了帮助你理解我是如何恢复的。

从现在开始，我会更加注意如何分享我的观点和建议。

也许通过理解我的挣扎，如果你在职业生涯中也曾与愤怒或怨恨作斗争，你会找到平静。

软件开发人员有赚大钱的潜力，尽管我的大部分经济和物质需求都得到了满足，但我内心还是很空虚。

两年前，在经历了一场慢性失眠之后，我开始回到教堂，并开设了我的 YouTube 频道。

此外，我已经开始[再次写歌](https://www.youtube.com/watch?v=xMkEHW0d0Pk&list=PLvEAMZJI1A11NZia9fRouE4Qk3o9IVigB&index=2&t=0s)为自己提供一个比软件更好的创作出口。

当你在公司工作时，如果他们把你放在一个盒子里，不允许你做最好的工作，这真的会令人沮丧。 ***我在人生错误的地方寻找创造的机会*** ！

**感谢你在我寻找一种方式将我的职业奋斗转变为对你的指导的过程中给予我如此大的支持**。❤️

我只是想分享我的成功和错误，以提供软件开发行业的一个诚实的观点——我从来没有想到你们中有这么多人在同一个问题上挣扎！
