#### [](#quick-intro)快速介绍:

嘿！👋我是陈丹，钦奈 SRM 大学 2019 级计算机专业毕业生。我在这里谈谈我在 communication us 工作的经历。

每个人都推荐你在大学期间的一个重要经历是实习。我想在一家公司实习，让我了解真正的开发人员是什么样的。当大学实习开始时，我正在寻找一家能给我这种经历的公司。communication us 就是其中之一。Arbor Program (TAP)是 communication us 的一项举措，旨在培养年轻人才，并在他们还在大学时对他们进行技术开发技能培训，以便他们在毕业时准备好正面解决现实世界的问题。当我还在学习的时候，这个项目还得到了公司一封很有吸引力的录取通知书的支持。这让我很兴奋，因为我正在寻找一条职业道路，发展新的技能，这将使我在就业市场上占据优势。

#### [](#hiringprocess-)招聘流程:

选择的程序很简单。从包括在线测试和简单作业的预筛选环节开始。接下来是一轮案例研究，他们给我们一个公司在其合作伙伴的产品中遇到的问题，并要求我们解决这个问题，以评估我们解决问题的技能。最后一轮是与几位经验丰富的开发人员和[communication us](https://www.commutatus.com)的首席执行官进行的一次轻松的面对面面试。我觉得这是一个面试应该如何进行的完美例子。整件事感觉像是一次谈话，而不是一次采访。

#### [](#beginning-of-awesomeness-)牛气冲天的开始:

当我收到公司的录用函时，我非常激动，并在面试后的第二天确认了我的选择。在我实习的第一天，当我到达换向公司的办公室时，我充满了复杂的情绪，焦虑和兴奋。我由另外五名学生陪同。这是我在办公室的第一个职位，我不知道该期待什么。然而，换向公司的环境相当宽松。我们进行了入职培训，以了解公司 it 愿景的核心原则。在我的初始阶段，我被分配了一些小的每周任务，这些任务有助于提高我的网络开发技能。

#### [](#the-journey-so-far-)旅程至此:

我在 Commuatus 的实习教会了我很多我想象不到的东西。这一切都是从简单的 HTML/CSS 网站到大规模的高级单页应用程序开始的。你在公司的角色(前端/后端/全栈开发人员)是根据你的兴趣和在分配给你的每周任务中的表现决定的。由于我热衷于学习 Javascript 并对构建美妙的用户体验表现出兴趣，我被分配了与前端开发相关的任务。在我完成核心概念和高级 javascript 的培训后不久，我被介绍给了我将共事的团队。在这一点上，我非常高兴能够加入 React 团队。(对于那些不知道 React 是什么的人来说，它是一个现代的 javascript 库，用于构建 web 应用程序，如脸书、Instagram、网飞、Codecademy 等。,).
