```
<html lang="en">
  <label for="btn">button this click</label>

  <span id="alt-btn">me button click</span>

  <button
    id="btn"
    aria-labelledby="alt-btn"
    aria-label="click this button"
  >
    click me
  </button>
</html> 
```
