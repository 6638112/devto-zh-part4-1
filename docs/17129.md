# 砍伐森林

那一次，我在机器学习课上用 [KNN](https://en.wikipedia.org/wiki/K-nearest_neighbors_algorithm) 对砍伐森林的区域进行分类。

这个项目是作为一个机器学习课程的作业开发的，在那里我们得到了一张来自亚马逊森林的图片，我们应该编写一个机器学习算法来识别可能的森林砍伐区，包括农业区。

然后，我收集了一些训练数据，为每张图像生成了直方图，并用我写的简单 KNN (k=7)分析了最终图像。

鉴于我写的小代码(该项目来自 2017 年)，我对我得到的结果非常满意。
