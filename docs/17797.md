# 你在听什么音乐？

> 原文:[https://dev.to/samba_code/what-music-are-you-coding-to-2c7n](https://dev.to/samba_code/what-music-are-you-coding-to-2c7n)

抛开关于你是否应该戴上耳机或者让自己变得平易近人的争论，这周你听了什么音乐(或者一般来说)来进入心流状态？

我是美元的超级粉丝，尤其是“专注和创造力的音乐”和“放松的音乐”。

Eluvium 也有一张很棒的专辑叫做“错误解读”。他们还有一张名为“shuffle drones”的专辑，适合你只需要一些白噪声的时候。

我也喜欢 Spotify 上的“深度聚焦”播放列表。

另一方面，我喜欢新的建筑师专辑，当我在会议后情绪激动时:)
