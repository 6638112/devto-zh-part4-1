在这个 Angular 8 教程中，我们将学习使用 NgSwitch 指令。我们将通过真实世界的例子来理解`ngSwitch`结构指令。NgSwitch 是一个内置的 Angular 8 模板指令，它的工作方式与 JavaScript 的 Switch 语句相同。该指令有助于根据条件语句显示或隐藏模板。

为了使 **NgSwitch** 工作，在主 HTML 元素上声明 NgSwitch 结构指令。在 NgSwitch 容器中定义了 ***ngSwitchCase** 语句，`*ngSwitchCase`语句引用了多种可能性树中的一种可能性。如果一个语句与表达式不匹配，它将转到另一个 ngSwitchCase 语句，如果它与表达式不匹配，它将显示定义了*ngSwitchDefault 语句的元素的结果。
