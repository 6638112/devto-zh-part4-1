# 我的第一份开发工作

> 原文:[https://dev.to/tekgadgt/my-first-development-job-47mh](https://dev.to/tekgadgt/my-first-development-job-47mh)

我最近开始了我在开发领域的第一份工作。我是 Maxx 潜能的初级学徒。虽然我们在技术领域有许多不同类型的项目，但我主要从事开发和代码维护项目。

到目前为止，我已经与 Docker Compose、Docker Swarm 和 vagger 合作开发环境。他们肯定很有趣。如果你真的想学习 Docker，把一个定制的 Docker 排版设置转换成 Docker Swarm 友好的。

就语言而言，我主要用 PHP 来扩展 WordPress 插件 Elementor。Elementor 非常强大，但是它并没有被很好地记录下来，所以它也是一次相当不错的学习经历。除了学习 Elementor，我还被要求学习 PHPUnit 单元测试。毫无疑问，这是这个过程中最困难的部分，但是它迫使我认识到我需要能够测试我自己的代码。

这篇文章更多的是关于我现在正在做的事情的更新，不幸的是并没有真正涵盖任何有趣的代码工作。所以我想我会把它标记为“讨论”,并询问是否有其他人刚刚开始他们在这个行业的第一份工作，并想谈谈它。我很快会有更多的期待，仍然在解决这个博客的事情。
