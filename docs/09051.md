现在，一旦我迈出了开发的第一步，或成为一名开发人员的第一步，windows 就开始发挥作用了。事情发生了，事情结束了，我开始处理它。病毒时代来了，USB 出现了，窗户坏了，所有的曼波 Jambo 都很沉重，但我活了下来。

截至目前！

### 窗口

我一直用 windows 作为游戏系统，玩堡垒之夜，尼尔自动化，最终幻想十五。我通过 USB 和铁拳 7 连接到 Playstation 3 控制器，真人快打 11 很有趣。现在说到写代码...啊哈。它太尖了，太僵硬了，获取环境路径有点痛苦。

### MacOS

现在当我开发的时候，我用我的 Macbook，因为我觉得为 MacOS 设置环境和库很流畅。有点像滴答滴答，然后就完成了。界面，我喜欢动画和桌面上的停靠栏。它简单高效。

### Linux

现在，再一次提醒我一些关于 Mac 和它的工作空间，设置环境，都很好。现在，我想说这是一个负担得起的苹果电脑。
