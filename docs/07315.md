> 原文:[https://dev . to/dzhavat/vs-code-extension-that-let-you-open-the-MDN-docs-of-a-CSS-property-gj6](https://dev.to/dzhavat/vs-code-extension-that-lets-you-open-the-mdn-docs-of-a-css-property-gj6)

**更新**:从 [v1.38](https://code.visualstudio.com/updates/v1_38#_mdn-reference-for-html-and-css) 开始，VS 代码也原生支持该扩展提供的功能！我对此很高兴，因为我不知道他们会实现类似的东西。更棒的是，它们还显示了 HTML 元素的 MDN 链接！现在去更新到最新版本的 VS 代码。很值得！

* * *

上周发布了我的第一个 VS 代码扩展之后，我有了另一个想法。这个可以让你打开 CSS 属性的 MDN 文档。相当简单！悬停时会出现该链接。

该扩展被方便地称为 [CSS 到 MDN](https://marketplace.visualstudio.com/items?itemName=dzhavat.css-to-mdn) 。

这里有一个演示:

[![Demo](../Images/89420e968782b6d954355a6b58ed9417.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--CLULl-EH--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ushkj9qa74ls3z6k8x5z.gif)

例如，在进行扩展之前，如果我想打开`background-size`的文档，我会切换到浏览器，打开一个新标签，搜索类似“mdn bkg size”的内容，然后单击第一个结果。

有了这个扩展，我就不必经历这些步骤了。我只需将鼠标悬停在某个属性上，然后单击链接。很好，不是吗？😉

如果您使用 CSS，请尝试一下。希望你觉得有用。
