# React: React Fiber，React Gamekit，无服务器架构，React Loadable

*对塞巴斯蒂安·马克贝奇、肯·惠勒、詹姆斯·凯尔、索菲亚·舒梅克的视频采访*

在过去的几个月里，React 社区经历了许多激动人心的事情和创新。这些想法和项目包括 React Fiber、React Gamekit、无服务器架构和 React Loadable。React Router 等项目的更新也对社区产生了巨大的影响。

### [](#react-serverless-architecture)**React +无服务器架构**

*Sophia Shoemaker 分享了关于无服务器架构的信息，以及她是如何开始 React 之旅的*

众所周知，服务器用于保存应用程序的数据和逻辑。然而，随着画面中的 React，一场运动出现了。这些相同的组件现在越来越倾向于客户。

随着这种转变，出现了使用无服务器架构和 Lambda 来创建应用程序的想法。在使用服务器创建应用程序时，没有经常涉及的监控、平衡或调整，这种方法被证明在用户交互方面更有效，总体上更好。

对 Sophia 来说，React 是一个框架，它让她真正增强了对 javascript 的知识和理解。比 JQuery 和 Backbone 更重要的是，在决定研究其他技术之前，她已经研究了相当长一段时间。她解释了 React 的其他好处，以及随着它的不断改进，这个 JS 库的前景将会如何。

[https://www.youtube.com/embed/oF8dAbnV8No](https://www.youtube.com/embed/oF8dAbnV8No)

### [](#react-music-react-gamekit-and-spectacle)**React 音乐，React Gamekit，和奇观**

*Ken Wheeler 表达了他对 React 项目创意的热爱*

肯·惠勒深入研究了他参与过的许多项目，如《奇观》、《光滑的旋转木马》、《反应音乐》和《反应游戏包》。他谈到了在使用 Keynote 之后，他是如何想出“奇观”这个创意的，并详细阐述了他从这个创意中获得的好处。

React Gamekit，允许开发者使用 React 或 React Native 构建游戏。它打开了一个平行的物理世界，个人可以使用自己的角色在其中充分探索。

当被问及 React 的许多伟大之处时，Ken 提到了虚拟化、React 硬件和 React 本机。他还分享了一点关于创造移动控制武器的知识(他觉得这很有趣！)还谈到了 React 社区如何拥有一些最友好和最具创新精神的人。

### [](#react-react-loadable-and-the-react-community)**React，React 可加载，以及 React 社区**

*詹姆斯·凯尔谈 React、React Loadable 和开源社区*

作为一名开发人员，当您的应用程序变得足够大时，您需要将它分成不同的路径，并分别异步加载它们。React Loadable 就是由 James 编写的。通过使用该库，开发人员可以在组件级别分解他们的代码。James 继续围绕这个项目进行构建，希望创建一个能够帮助更有效地交付包的应用程序。

由于参与了如此多的 javascript 项目，James 将开源描述为他最热衷的一个社区。能够使用 OSS 来提供世界各地的人们可以用来解决他们的问题的工具是他不断前进的动力。他还谈到了 React 和 React Native 之间的细微差别，以及人们呆在一起的重要性。能够发现协作的方式并让每个人都围绕标准聚集是有价值的。除非人们愿意合作分享解决方案，创造解决方案，否则这不可能发生。

[https://www.youtube.com/embed/615O4etSkE4](https://www.youtube.com/embed/615O4etSkE4)

### [](#react-fiber-and-web-assembly)**反应纤维和纤维网组装**

*React 核心团队的 Sebastian Markbage，React 纤维和纤维网组件*

Sebastian 向我们讲述了他在进入 React 之前，在脸书工作、研究数据可视化框架以及研究 web 解决方案的故事。一旦他发现了 React，他就开始将其用于艺术和图形。Sebastian 描述了他在 React 首次引入 JS 社区时的经历。一开始有很多批评和反对，一旦围绕框架建立了一个更大的生态系统，这些批评和反对就平息了。对 Sebastian 来说，在创造新技术时，拥有一个生态系统和统一是至关重要的。

经过对脸书的大量实验，反应纤维问世了。他描述了 React Fiber 的关键特性之一是功能性——一切都是异步和不可变的。采访中简要讨论的另一个项目是 ReactVR，以及它是如何开始的。

最后，Sebastian 被问及他对 Web Assembly 的兴趣以及 React 即将发布的消息。
