# VirtualBox:配置“主机密钥”

> 原文:[https://dev . to/nabbi sen/virtualbox-configuring-host-key-3 akl](https://dev.to/nabbisen/virtualbox-configuring-host-key-3akl)

[VirtualBox](https://www.virtualbox.org/) 提供主机键的各种功能，如主机和访客之间的切换或截图。

默认的主机键是“右 Ctrl”，而有些键盘没有这个键。
在这种情况下，我们可以通过预引用中的键盘设置来更改映射:

[![keyboard settings](../Images/4277c366679e5fe5c9929014561c826f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--dRFxRmin--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/al74xp25t9vcu7m3lqxq.png)

只需将焦点放在“主机组合键”的“快捷方式”区域，并键入您想要设置为主机键的键。

此外，当您已经运行虚拟机时，仍然可以通过右键单击来宾窗口的右下方来配置它:

[![guest window](../Images/3fb89eefb89e9605393b923038367929.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--EQsU1sUk--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/fjngu4iodb9o2fnvxo9h.png)
