# 大规模的高级打字工具

> 原文:[https://dev . to/JavaScript _ tuto/advanced-typescript-tooling-at-scale-303 I](https://dev.to/javascript_tuto/advanced-typescript-tooling-at-scale-303i)

工具一直是 TypeScript 关注的焦点，也是它成功的重要原因。

TypeScript 代码智能可用于许多编辑器，甚至是基于 web 的工具，所有这些都是由编译器本身自带的相同服务 TypeScript 提供的。本演示介绍了 TypeScript 如何通过 tsserver 和 tsserver 协议为许多不同的客户端提供同样出色的体验，以及它如何影响和适应语言服务器协议等新的努力。我将展示如何使用它来提供代码智能特性，比如跨存储库级别的跳转到定义和查找引用，使用 TypeScript 的一些最新特性，比如声明映射，并将其与 JavaScript 包管理工具中不太为人所知的有趣特性相结合。在本次演讲的最后，听众将了解 TypeScript 工具是如何工作的，以及它如何与生态系统中的其他工具相结合来提高开发人员的工作效率。目标是通过展示什么是可能的以及如何实现来激发 TypeScript 工具生态系统中的创新
