# Delphi 数据访问组件现在与 macOS 64 位兼容

> 原文:[https://dev . to/devart software/Delphi-data-access-components-are-now-compatible-with-MAC OS-64-bit-1pf](https://dev.to/devartsoftware/delphi-data-access-components-are-now-compatible-with-macos-64-bit-1pf)

Devart 最近发布了其 Delphi 数据访问组件的新版本，现在支持 macOS 64 位。

[![Delphi Data Access Components](../Images/8cd009f71434f6a6e87af9a0f0a18410.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--53n1Lkq3--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i.ibb.co/QKrM0Qf/entity-dac-screen.png)

Devart 推出了新版本的 Delphi 数据访问组件，允许用户用 Delphi 为 32 位和 64 位版本的 macOS 开发应用程序。RAD Studio 用户会对该版本特别感兴趣，他们决定将其软件升级到最新版本 10.3.2，该版本支持 macOS 上的 64 位 Delphi 应用程序。

另外值得一提的是，新版 DAC 与 macOS 64 位上的 RAD Studio 10.3.2 的兼容性已经过全面测试。但是，此 DAC 版本需要安装 Embarcadero RAD Studio 10.3 的版本 2。

**Devart Delphi 数据访问组件**允许在 Windows、Linux、macOS、iOS 和 Android 上用 Delphi、C++Builder、Lazarus(和 FreePascal)开发多平台应用程序。这些工具提供了对流行数据库的直接访问，如 Oracle、Microsoft SQL Server、MySQL、InterBase、Firebird、PostgreSQL、SQLite。

如需此版本中变更的完整列表，请访问 [DAC](https://www.devart.com/dac.html) 页面。
