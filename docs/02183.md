# Dart 中的 If-Else 语句

> 原文：<https://dev.to/jay_tillu/if-else-statement-in-dart-5396>

If-else 是一个条件语句。当我们想检查情况时，就用它。但是除了 ***if 语句*** ，这里我们还有 ***else*** block too。如果语句 评估为假，那么 ***的条件将执行哪个？***

### 语法

* * *

```
if (boolean_expression) {
  //Statements to be executed if the condition is true
} else {
  //Statements to be executed if the condition is false
} 
```

*   如果布尔表达式评估为 ***真*** ，那么 If 语句内的代码块将被执行。

*   如果布尔表达式评估为 ***假*** ，那么 else 的代码块将被执行。

*   这里记住 ***否则*** 在这里是可选的。

#### 样本代码

```
void main() {
  int number = 35;
  if (number > 5) {
    print("Number is greater than 5");
  } else {
    print("Number is less than 5");
 }
}

Output
Number is greater than 5 
```
