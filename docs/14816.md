编程的整个要点归结为一个不可避免的结果，即我们编写指令来操纵/控制数据。

Python 提供了许多数据结构来存储数据，Dictionary 就是其中之一，它将数据存储为一个键-值对。根据官方文件，有许多定义，但是，我发现下面的定义很容易理解。

> 字典是一种数据结构，通过关键字进行索引。它也可以被定义为一组无序的键:值对。像字符串和数字这样的不可变(非动态)类型总是可以作为键。*
