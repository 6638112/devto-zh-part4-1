# 做一名女程序员:对你来说怎么样？

> 原文:[https://dev . to/ilonacodes/being-a-female-programmer-it-how-for-you-i7d](https://dev.to/ilonacodes/being-a-female-programmer-how-is-it-for-you-i7d)

这大概取决于我们生活和工作的地方，但就我个人而言，在我几年的职业生涯中，我并没有经历过作为一名女性程序员有什么负面的东西。

像其他人一样，我有面试机会。只要我通过了技术挑战和非技术面试，我就会得到一份合同。在工作中，我得到了团队的认可。

只要你热衷于与技术人员合作，他们实际上是简单而实用的。

即使是开发团队中的女性也仍然很少，尽管在其他技术岗位上有很多女性(我的大多数项目经理都是女性)，UX 专家、设计师、QA、商业分析师等等。所以团队里女性比较多。

唯一让我感到有点失落的是没有加入男生俱乐部。你知道，所有那些家伙都在聊天，这很好，因为我对这样的话题不感兴趣。此外，如果团队中大多数是女性，只有少数男性，类似的事情也会发生。否则，我们相处得很好，我们可以与话题广泛的人进行小组对话，他们对我很友好，很尊重。

然而，我采访的许多女性都提到，女性在软件工程职业生涯中仍然有一些障碍，这些障碍每天都会出现，非常难以识别，比如:

*   用一个全是男性的面试团队面试一位女性，然后评价她们“不够自信”女人被训练成当她们单独和一群陌生男人在一起时要小心谨慎。让一个女人在房间里接受采访，即使为此你必须带你的女朋友。

*   **有弹性的。**如今，初创公司和其他公司都是为了福利。无尽的小吃、免费饮料、办公室中央的一张大乒乓球桌——随着雇主提高门槛以吸引更多有前途的候选人，这样的例子不胜枚举。大多数公司通常认为不提供的极具吸引力的额外津贴是灵活性。出于各种原因，女性经常需要比男性更特殊的住宿条件和更灵活的工作时间表。通过提供在家工作日或兼职工作，公司可以实现不可能的事情:他们可以留住有才华的摇滚明星员工(其中许多是女性)。企业从留住员工中受益，女性找到了解决“拥有一切”难题的方法。

*   当管理层想要提拔一位女性担任领导职务时，即使她可能还没有准备好。如果她失败了，管理层可能会以此为借口，不再为女性提供多样化的角色。即使她成功了，她的队友可能会怨恨她“偷”了一个她没有资格获得的机会，因为她是一个女人。

*   工资率。女人总是比男人挣得少，因为她们更少参与谈判。心中有一个固定的预算，只有在极端情况下才会超出预算，其他人都会理解或同意。如果你不断听到求职者说他们需要更多的钱，那就提高这个职位的薪水。但是你也必须为现有的员工这样做。

总而言之，在好的公司里，男性确实在努力帮助女性，但有时他们并不理解，而了解科技行业女性的最好方法就是和你团队里的女性聊聊，看看她们有没有什么问题。告诉他们你想支持他们，让他们知道你站在他们一边。
