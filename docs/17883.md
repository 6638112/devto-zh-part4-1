# 作为前端开发人员，投资组合仍然是必要的吗？

> 原文：<https://dev.to/jamesthomson/are-portfolios-still-necessary-as-a-front-end-developer-1geg>

我不得不经历求职过程已经有几年了，我很好奇，作为一名前端开发人员，拥有一个投资组合网站仍然是一种规范吗？

我们中的一些人，包括我自己，在几年的时间里都在开发一个 web 应用程序，所以最近的作品组合将会非常稀少。当然，在网站上已经做了很多工作，所以不同的方面都可以有特色，但有一个这样的组合是不是有点过头了？

我看到的前端开发人员的招聘启事中没有一个提到发送您的作品集，这就是为什么我想知道它是否仍然是必要的。

编辑:感谢大家的反馈。人们似乎一致认为，拥有一个个人网站是值得的，也是一个收集你过去的经验、实验和一般想法的好地方。这些对获得潜在的就业机会都非常有用。
