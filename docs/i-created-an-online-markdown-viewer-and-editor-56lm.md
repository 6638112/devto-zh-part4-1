### 轻量级降价查看器和编辑器

在我的一篇博客文章中，我正在寻找一个在线的减价查看器和编辑器。有许多功能强大的编辑器，带有无用的功能，但似乎没有一个能轻松快速地处理所有基本功能。

这就是为什么我创建了一个在线 markdown 查看器和编辑器工具: [Marcdown](https://liyasthomas.github.io/marcdown) 以便在旅途中轻松创建、编辑、查看和保存 markdown 文件。 [Marcdown](https://liyasthomas.github.io/marcdown) 就是所见即所得！

> 所见即所得的缩写。在计算中，WYSIWYG 编辑器是一种系统，在该系统中，内容可以以非常类似于其作为成品打印或显示时的外观的形式进行编辑，例如打印的文档、网页或幻灯片演示。

[![Dark mode](img/d6a99855faad79fa006894d0c263a1a7.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--FLwBumKc--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://raw.githubusercontent.com/liyasthomas/marcdown/master/icons/screely2.png)

忘记与复杂的编辑一起工作的所有麻烦。了解如何在几分钟内创建、编辑、查看和保存降价文件！

Marcdown 是一个开源的 markdown 查看器和编辑器，针对变化和适应性进行了优化，以满足您的确切需求。

[![Light mode](img/5f9508458aba3b5036bc701044bfa8ff.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--o2_5c5-P--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://raw.githubusercontent.com/liyasthomas/marcdown/master/icons/screely.png)

有了[markdown](https://liyasthomas.github.io/marcdown)，现在我可以免费查看和编辑各种大小的 markdown 文件；我可以在任何地方查看、编辑、创建降价文件并下载它们。

查看和编辑 markdown 文件现在就像 1，2，3 一样简单！

1.  在左侧粘贴或键入降价。

2.  在右侧查看预览。

3.  下载降价文件。

奖金:🌗我还增加了一个黑暗模式的选项开关，这样在晚上阅读 markdown 文件就不会伤害你的眼睛了！
