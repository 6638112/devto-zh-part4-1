# 关于网络钓鱼攻击及其解决方案，您需要了解的一切

> 原文：<https://dev.to/2muchcoffeecom/everything-you-need-to-know-about-phishing-attacks-and-their-solutions-i0k>

网络钓鱼是黑客最喜欢使用的技术。这部分是因为它非常简单，任何具有基本 it 技能的人都可以做到。另一部分原因是它受到高度影响，每天都有数百万互联网用户成为网络钓鱼攻击的受害者。

考虑到网络钓鱼攻击的流行，人们期望每个互联网用户都知道什么是网络钓鱼。不幸的是，大多数人不知道。

## 什么是钓鱼？

<u>[网络钓鱼是一种黑客技术](https://www.expressvpn.com/blog/what-is-a-phishing-attack/)</u> ，黑客利用它来收集敏感数据，如电子邮件地址、个人账户密码等。这是最古老、最有效的网络钓鱼技术之一。它也是最简单的一种——简单到业余黑客可以在黑暗网络上找到现成的钓鱼工具。

网络钓鱼可以追溯到互联网诞生之前。据说“网络钓鱼”一词是由“phreaking”一词创造出来的。Phreaking 是黑客用来从他们的手机服务提供商那里获得免费通话时间的一种技术。黑客会在他们的手机上播放独特的声音来解锁免费通话时间。

网络钓鱼出现在 90 年代中期。黑客会瞄准 AOL 用户，让他们分享自己的登录信息。如今，网络钓鱼也一样简单，但风险要高得多。如今，黑客的目标是更敏感的信息，如银行账户和网上银行平台的密码。网络钓鱼还被用来收集个人信息，这些信息可以被出售牟利或用来勒索受害者。如今的网络钓鱼技术也比以往任何时候都多。

## 钓鱼攻击技术的种类

黑客总是想出新的方法从毫无戒心的互联网用户那里窃取敏感信息。然而，目前有三种主要技术被广泛使用:

**电子邮件钓鱼**

电子邮件是黑客最常用的网络钓鱼平台。它更可取，因为它为黑客提供了欺骗受害者所需的匿名性。它也很容易操纵，以便模拟任何实体。

电子邮件网络钓鱼包含欺骗。黑客注册的电子邮件看起来和听起来就像真实实体的真实电子邮件。注册的域名与原始域名非常接近。电子邮件也被设计成看起来像原件。然后，黑客使用假域名向不知情的受害者发送假邮件。

电子邮件网络钓鱼被用来以多种方式收集数据。最简单的情况是，黑客要求收件人的个人数据，希望他/她会上当并回复。另一种情况是，电子邮件中嵌入了恶意软件，这些恶意软件会感染设备并收集敏感数据——键盘记录器等程序被用来记录击键甚至截图。

最复杂和最常见的电子邮件网络钓鱼技术包括发送带有虚假网站链接的虚假电子邮件。在这种情况下，黑客创建了该公司的电子邮件和网站的副本。邮件敦促收件人点击链接并登录，以避免即将发生的问题。该网站然后收集敏感的登录信息，这些信息随后被用于实施其他形式的网络犯罪。

***如何躲闪邮件钓鱼***

如果你仔细观察，假邮件很容易被发现。域名和电子邮件地址乍一看可能是原始的，但总有一些不同之处。因此，当你收到网上银行平台和政府等敏感实体的邮件时，一定要保持警惕。

你也应该避免点击电子邮件中的链接。如果你必须登录一个网站，那么直接通过浏览器登录。同样不言而喻的是，你永远不应该回复假邮件或与任何人分享敏感数据。

**网络钓鱼**

互联网上有成千上万的假冒和恶意网站。这些网站中有许多是专门为网络钓鱼设计的。它们被设计成处理敏感问题(如私人通信和金融交易)的流行网站。

网站使用两种网络钓鱼技术。最常见的策略是诱骗访问者登录他们的账户，在这种情况下，网站会收集这些登录信息并发送给黑客。第二种技术包括用恶意软件和间谍软件程序感染用户的浏览器。然后，这些程序跟踪用户的在线活动，收集大量敏感数据。

***如何躲闪网站钓鱼***

你会惊讶于真实的钓鱼网站是如何出现的。然而，它们都有一些泄露秘密的迹象。首先，URL 通常不安全——它不包含 HTTP 和 HTTPS 前缀。此外，该网站的某些页面要么是不活跃的，要么是肤浅的。

避免可疑网站的最好方法是使用一个像样的网络安全程序。包括 Chrome 和 Firefox 在内的许多浏览器会自动屏蔽可疑网站。

**电话钓鱼**

黑客仍在使用手机进行网络钓鱼，但他们不是为了免费通话时间。任何呼叫者 ID 被公开列出的号码都容易受到欺骗。黑客伪造目标号码，用它们从毫无防备的目标那里窃取信息。

电话网络钓鱼就是花言巧语。例如，黑客可能会伪造与银行相关的号码，并用它们来联系银行的客户。这本书最古老的玩法通常是提醒客户他们的账户有问题，并要求他们提供账号和其他敏感信息，以便“修复”所谓的问题。不知情的受害者最终会分享他们的信息。

***如何躲闪电话钓鱼***

避免成为电话钓鱼攻击受害者的最好方法是不要通过电话与任何人分享您的敏感信息。如果你担心任何事情，那么总是挂断电话，用你知道并相信是真的号码回电。你也应该经常核实任何打电话给你的人的身份。

## 网络钓鱼和社会工程

网络钓鱼正在越过互联网和智能设备，进入现实世界。黑客在攻击中变得厚颜无耻，现在有足够的信心与受害者进行更密切的接触。

网络钓鱼和社会工程没有太大区别。在这两种情况下，黑客都试图从受害者那里探听敏感信息。然而，社会工程更复杂，更有说服力。

社会工程攻击通常是有针对性的。黑客花时间研究他们的目标，然后想出一个精心设计的计划来欺骗他们。大多数目标是小公司，但个人、大公司甚至政府机构都有资格。一旦黑客了解了目标的所有相关信息，他/她就会继续了解他们的联系人和熟人，并冒充一个。

社会工程攻击通常发生在电话上。它们类似于电话网络钓鱼，只是攻击更加复杂。通常，黑客会冒充受害者的联系人之一，然后打电话询问信息或金钱。最常见的策略是冒充一家公司的客户，目前为该公司做一些工作，并要求使用不同的账号预付款项。黑客还打电话询问敏感信息，然后利用这些信息来欺骗公司。

在某些情况下，黑客厚颜无耻地走进一家公司，亲自实施这些社会工程攻击。例如，一个黑客可能冒充一个 IT 承包商，扮演一个角色，包括获得制服、徽章、工具箱和你期望从一个 IT 人员那里得到的一切。在许多情况下，黑客会为你需要他们的服务制造借口，例如，他们可能会发起网络攻击，破坏你的计算机服务。一旦他们来“修复”问题，他们也安装间谍软件，然后可以收集尽可能多的信息。

社会工程攻击大胆多变。事实上，任何使用虚假手段强迫他人提供敏感信息的企图都可以被描述为社会工程攻击。这些攻击正在上升，因为它们利用了人类的弱点，而与网络安全程序相比，人类的弱点更容易通过。

## 它可能发生在任何人身上

世界上每天都有数百万人成为网络钓鱼攻击的受害者。这包括精英，各大组织，甚至政府。以下是迄今为止最突出的三种网络钓鱼攻击:

**费西合唱团·弗里行动**

<u>[行动费西合唱团 Phry](https://www.forbes.com/sites/billsinger/2012/05/15/feds-catch-their-illegal-limit-in-operation-phish-phry/#20eff4726265)</u> 是有史以来记录在案的最大一起网络钓鱼案件。它涉及一个由数百人组成的高度组织化的网络。这次攻击的目标是数百名银行和信用卡用户。攻击者发送看似真实的电子邮件，并将受害者引导至虚假的银行和信用卡网站。上当受骗的人继续登录他们的账户，最终成为攻击的受害者。

据报道，费西合唱团·弗里行动诈骗了受害者 150 多万美元。然而，袭击者并没有逃脱，因为联邦调查局展开了大规模调查，抓获了数百名袭击者。

**针对沃尔特·斯蒂芬的骗局**

<u>[沃尔特·斯蒂芬](https://www.reuters.com/article/us-facc-ceo-idUSKCN0YG0ZF)</u> 作为单次网络钓鱼攻击中损失最大的个人而载入史册。沃尔特·斯蒂芬是奥地利航空航天公司 FACC 的前首席执行官，该公司为空客和波音制造飞机零部件。

在斯蒂芬担任首席执行官期间，一名黑客成功猜出了他的电子邮件地址，并制作了一个类似的电子邮件。黑客随后向一名低级会计人员发送了一封电子邮件，授权向一个外国银行账户转账 4700 万美元。据称这笔款项将用于支付一个预定的采购项目。那个职员上钩了，把钱转了出去。

不幸的是，斯蒂芬和其他几名高级官员在这次袭击后被解雇。更重要的是，该公司只追回了被盗资金的一小部分，黑客也从未落网。

**目标骗局**

<u>[针对目标](https://www.usatoday.com/story/money/2017/05/23/target-pay-185m-2013-data-breach-affected-consumers/102063932/)</u> 的网络钓鱼攻击是 2013 年最广为人知的网络攻击事件之一。这次大规模攻击影响了超过 1.1 亿用户，并泄露了与超过 4100 万个零售卡账户相关的详细信息。然而，当局没有提供更多关于攻击者偷了多少钱，甚至他们是否被抓住的细节。

## 结论

欺骗攻击的终极解决方案:将您的敏感信息保密。

与其他利用网络安全程序弱点的网络攻击相比，电子欺骗依赖于人们的脆弱性。因此，既然您知道什么是网络钓鱼，您就可以通过保护您的敏感信息来阻止任何攻击。此外，请确保遵循针对每种类型的网络钓鱼攻击的突出提示。
