# 宣布！！

> 原文：<https://dev.to/aishablake/announcing-title-of-conf-29nb>

大约一个半月前，我在罗马尼亚的 [JSHeroes](https://jsheroes.io/) 遇见了无与伦比的 [Anjana Vakil](https://twitter.com/AnjanaVakil) 。我刚刚加入了 [Mozilla Tech Speakers](https://wiki.mozilla.org/TechSpeakers) 项目，她也是其中的一员，我有幸与她和许多其他优秀的演讲者共度时光。会议结束后，我们在一个组织者的家里共度了最后一个晚上。

在所有的谈话、饮酒和欢乐中，Anjana 分享说她正在为[准备一个演讲！！那实际上更像是一场音乐剧表演。好奇吗？我也是！！我曾幻想过写一篇演讲，不知何故也包括我演唱的歌曲，但我从未花时间和精力去真正写这样的东西。现在，这位出色、聪明、有趣的女士将这个想法提升到了一个新的高度！](http://bangbangcon.com/)

我们开玩笑说，如果整个会议完全由音乐技术讲座组成，那该有多棒。

虽然我计划去纽约看 Anjana 的演出，但我错过了买票的窗口。不用说，我崩溃了。相反，在一次拜访我男朋友的母亲时，我与 wifi 较劲，以便我能确保赶上直播。想象一下，当我现场观看(和唱)这部杰作时，我是多么的高兴:

看到尾部呼叫优化:这部音乐剧第一次让我的灵魂感受到了当我自己即将走上舞台时的一些电流。我非常兴奋，我知道我必须带着这种兴奋做点什么。

我知道我必须让我们在罗马尼亚某个人的厨房里开玩笑的那个会议成为现实。

## 自拍点

当我把这个半生不熟的计划带给我组织 self.conference 的同事 Amber Conville 时，她的第一反应是建议在 2020 年的 self.conference 上尝试一下。这样，我就可以使用我们现有的 self.conference 系统，并评估人们对这种新形式的兴趣。

原来，安珀一直在密谋，我们都不知道！她已经启动了一个名为 [selfdot](https://selfdot.org/) 的全新非盈利组织，该组织将在未来推出 self.conference 以及其他一些项目。目标是提供计算机扫盲讲习班和其他技术支持。会议和共同工作空间(当然必须有一个共同工作空间)将有助于组织的收入。

## <会议名称>

所以我想向你们介绍，它是以我最喜欢的音乐剧之一<a href="https://en.wikipedia.org/wiki/Title_of_show">【节目名称】</a>命名的。<会议名称>将是一项实验，以新的方式教育和挑战科技专业人士。讲座将以音乐表演的形式进行，可能是像 Anjana 那样的模仿，也可能是完全原创的作品。

与会者将受益于歌剧演唱家兼声乐老师克里斯蒂娜·斯旺森的指导。你将学会如何利用你自己身体和呼吸的力量来增强你的舞台表现力，为你的演讲加油(台上台下)！我的希望是带给你一些不同的、毫无歉意的*乐趣*，它仍然教会你有价值的技能和工具，你可以用来支持你的团队，提升你的技能，或者让你的生活更轻松。
