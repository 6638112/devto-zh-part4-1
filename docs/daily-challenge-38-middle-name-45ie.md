# 每日挑战#38 -中间名

> 原文：<https://dev.to/thepracticaldev/daily-challenge-38-middle-name-45ie>

你今天的[挑战](https://www.codewars.com/kata/5768a693a3205e1cc100071f)是初始化一个人的中间名(如果有的话)。

> 例如:
> “杰克·瑞恩”= >“杰克·瑞恩”
> “露易斯·玛丽·莱恩”= >“露易斯·m·莱恩”
> “迪米特里”= >“迪米特里”
> “爱丽丝·贝蒂·凯瑟琳·戴维斯”= >“爱丽丝·b·c·戴维斯”

祝你好运！
