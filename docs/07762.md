# 每日挑战#42 -凯撒密码

> 原文：<https://dev.to/thepracticaldev/daily-challenge-42-caesar-cipher-43k8>

凯撒密码或移位密码是最著名的加密技术之一。它的工作原理是将信息中的字母替换为字母表中某个固定位置的字母。

你的挑战是编写一个函数，接受一个字符串和一个 shift 键。它将返回加密的文本。如果字符串中的某些内容不在字母表中(例如标点符号、空格)，请保持原样。
