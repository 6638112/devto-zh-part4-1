## [](#play-scattergories-with-friends)[与好友✍玩散兵游勇🏽](https://glitch.com/~scatterg)

对于我们当中的幸运儿来说，本周享受一个漫长的周末，与朋友和家人一起出去玩，[斯蒂芬·莫内蒂](https://glitch.com/@smonette)的“[散射](https://glitch.com/~scatterg)”游戏可能会派上用场。这是一个简单的玩法。找一个或几个朋友，一支笔，一张纸。然后使用她的应用程序随机生成一个字母，点击计时器上的播放，并在每个列出的类别中写下单词。

重新混合，加入你自己的想法——改变类别，或者修改规则，让事情变得更有趣。

[https://glitch.com/embed/#!/embed/scatterg?previewSize=100&path=index.html](https://glitch.com/embed/#!/embed/scatterg?previewSize=100&path=index.html)

## [](#map-with-me)[带着我的地图🌍](https://glitch.com/~mapwithme)

## [](#make-card-art)[制作卡艺🃏](https://glitch.com/~canvas-cards)

卢克·巴顿的[画布卡](https://glitch.com/~canvas-cards)是一个用 JavaScript 画布创作的卡片设计图库。每周更新，这是一个不断增长的由数学、建筑、自然和其他事物启发的生成艺术集合。

更重要的是，每张卡都包括一个可编辑的代码块，你可以调整它来做出你自己的设计，或者了解更多关于每张卡是如何工作的。
