# 📄备忘单——如何推广你的开源项目？

> 原文：<https://dev.to/kefranabg/cheat-sheet-how-to-promote-your-open-source-project-4flh>

你创建了一个开源项目？太好了。但是当你的项目足够成熟可以使用的时候会发生什么呢？您已经付出了很多努力来构建一个可以帮助开源社区的了不起的项目，但现实是如果您不传播它，几乎没有人会看到或使用它。

在过去的几个月里，我和一些同事经历了很多开源项目的推广，我们发现了一些帮助我们的开源项目获得流行的好的实践。你猜怎么着？我们制作了一份备忘单来总结我们所学的一切😎。
