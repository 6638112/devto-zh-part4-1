# 不平衡-学习 0.5.0 已过时

> 原文：<https://dev.to/chkoar/imbalanced-learn-0-5-0-is-out-4p3o>

unbalanced-learn 是您最喜欢的 python 包，它提供了许多重采样技术，这些技术通常用于显示类间强烈不平衡的数据集中。

这是一个小版本，包含了大部分的 bug 修复，已经可以通过 pip 和 conda 获得了！

您可以使用 pip
来安装它

```
pip install imbalanced-learn -U 
```
