# 科技对孩子不好

> 原文:[https://dev . to/marines 0210/technology-is-bad-for-children-2p9d](https://dev.to/marines0210/technology-is-bad-for-children-2p9d)

科技的使用如何影响你的孩子。今天的技术非常有用，它正在成为我们日常生活的一部分，我们都在使用技术，包括儿童。他们可以用它来学习不同的东西，例如，通过 YouTube 上的一个简单视频，您的孩子可以学习演奏乐器、数学、艺术等。他们也可以使用应用程序来学习；然而，正如生活中的一切事物一样，它有优点也有缺点。不幸的是，接触科技的孩子可能会有危险。由于缺乏成人监控，儿童可能会沉迷于技术，成为恋童癖者的猎物，并成为网络犯罪分子欺诈的受害者。

当儿童接触智能手机时，他们可能会对技术上瘾，因为他们开始使用不同的应用程序，例如视频游戏，这些应用程序可能会上瘾。沉迷于电子游戏会形成一种病态、精神和身体状态，在这种状态下，儿童需要赢得他正在玩的游戏，以获得幸福感。这导致孩子们对他们的小玩意有一种可怕的痴迷，因为丢失不是一个选项。

恋童癖者是掠食者，因此他们在技术上是一个巨大的危险。2019 年，著名的 Tik Tok 应用程序允许未成年人访问，人们可以上传视频。该应用被举报是因为恋童癖者用淫秽语言评论儿童视频。

共享信息对于成年人来说是一项艰巨的任务，因为他们在向第三方提供机密信息(如电话号码、信用卡信息等)时可能成为欺诈的受害者。所以，如果这对成年人来说很难，对一个孩子来说就更糟了；他们可能会被诱骗点击恶意横幅和弹出广告。这些广告展示孩子们感兴趣的东西。然后，他们被引诱到一个共享信用卡数据的页面，以获得他们的奇思妙想。

总之，孩子们可以从科技的使用中广泛受益，但是有必要让一个成年人来监督他们。另外，如上所述，他们很容易沉迷于技术，成为恋童癖的受害者，并且因为不了解更多而参与欺诈。
