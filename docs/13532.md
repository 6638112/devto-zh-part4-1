# [](#introduction)简介

作为开发人员，我确信您可能会被推进到不时需要您的社交技能的社交环境中。

从与同事的饮水机会议、聚会、与上级开会、参加会议、向客户推销。

这就要求你站着思考，运用你的社交技能去完成一些事情，或者作为一个个体去茁壮成长。

# [](#kid-travels-100-of-miles-to-be-back-home)孩子跋涉 100 英里回家

从前有个孩子被宠坏了，对陌生人很害羞。因为这种害羞，他的妈妈觉得他可能会变成一个害羞或反社会的成年人。

因此，他的妈妈想出了一个“绝妙”的主意来帮助这个男孩的害羞。一天，当我离家旅行时。

她在一个偏僻的地方把他的孩子从车里踢了出来，并告诉她的孩子找一条回家的路，身上什么都不穿。

幸运的是，他能够通过学习与陌生人聊天和不舒服的情况导航回家。

这一课给了他信心，使他开始疯狂的冒险。

从建立自己的商业帝国，乘坐热气球环游世界到建立自己的私人太空旅行公司。

如果你知道他是谁，请在评论中告诉我。他是我最喜欢的企业家之一，我真希望能和他一起喝酒。

# 做一些让你害怕的事情

这是学习成为人类的本质，我们需要能够走出我们的舒适区，进入社交场合或做一些让你害怕的事情。

尽管写了十几篇文章，我仍然有同样的恐惧和写作的麻烦，因为我 7 岁就学会了用英语读写。

当然，如果可以选择的话，我更愿意躲在壳里，过几天不与人交往。

这可能是你选择成为一名开发者的原因之一。

不管你是打算成为一名独立的贡献者还是晋升为经理。

作为一名开发人员，我们必须学习社交技能来帮助我们成长。

# [](#how-to-be-human)如何做人

学习社交技能是一种**技能**,就像我们作为开发人员为了保持相关性而必须学习的任何新技术一样。

总有一种方法可以帮助你在社交技巧方面变得更好，帮助你在职业上成长，成为一个更好的人。

## [](#be-a-volunteer)做一名志愿者

志愿服务帮助我们不要把注意力放在自己身上，而是迫使你把注意力放在你所帮助的人身上，通过你自己的时间、金钱或努力来帮助他人，从而增加价值。

## [](#become-a-voracious-learner-for-social-skills)成为对社交技能如饥似渴的学习者

关于公众演讲、情商、社会心理学、伦理学、说服力，有大量的课程、书籍或研究。

所以，慢慢来，挑选一些你喜欢的东西，帮助你在这个过程中变得更“人性化”。

以下是我挑选的这些年来真正切中主题的书籍。

*   [出售是人类作者丹尼尔·H·平克](https://www.amazon.com/Sell-Human-Surprising-Moving-Others/dp/1594631905)
*   影响:罗伯特·B·西奥迪尼的说服心理学
*   丹尼尔·戈尔曼的《情商》
*   [如何赢得朋友&影响人戴尔·卡耐基](https://www.amazon.com/How-Win-Friends-Influence-People-ebook/dp/B003WEAI4E/ref=sr_1_1?crid=1XKZHG583AN2T&keywords=how+to+win+friends+and+influence+people&qid=1563117418&s=digital-text&sprefix=how+to%2Cdigital-text%2C355&sr=1-1)
*   理查德·布兰森
*   《每个人都在说什么:前联邦调查局特工速读指南》，作者乔·纳瓦罗，马文·卡林

## [](#practice-makes-perfect)熟能生巧

练习社交技能的方式可以是参加聚会或者仅仅是科技会议。

我记得近年来桌游团体有所复苏。也许只是买一个很棒的游戏，邀请朋友来玩桌游之夜也能帮到你很多。

不要害怕失败，当你和别人一起练习社交技巧时，你会变得更好，并有希望在这个过程中结交新朋友。
