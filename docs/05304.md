# 代码故事 E4:肖恩·华盛顿谈冒名顶替综合症

> 原文：<https://dev.to/codestory/code-story-e4-sean-washington-on-imposter-syndrome-245c>

[https://www.youtube.com/embed/2QZegt_8NTM](https://www.youtube.com/embed/2QZegt_8NTM)

现在，人们通过多种途径了解软件开发。令人惊讶的是，许多其他行业也在思考和创造代码——教学、项目管理……甚至音乐。肖恩·华盛顿通过创作音乐和图形设计，走了一条非常规的软件之路。从那以后，作为一名全栈网络开发人员和技术播客主持人，他一直在向前冲刺。他最近的冒险是成为设计集体的首席技术官和首席开发人员——这不仅让他有机会了解这个平台——而且让它成为自己的平台，并定义如何提升产品的愿景。

在上面的视频中，肖恩·华盛顿讨论了我们很多人可能都很熟悉的东西:冒名顶替综合症。如果你喜欢你听到的，看看下面完整的一集。
