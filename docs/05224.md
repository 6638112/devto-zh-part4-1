# dev.to 在 Reddit 上被屏蔽了

> 原文:[https://dev.to/gajus/dev-to-is-shadow-banned-on-reddit-15pf](https://dev.to/gajus/dev-to-is-shadow-banned-on-reddit-15pf)

昨天我写了第一篇关于 dev.to 的文章，并在 Reddit 上分享。[我在 Reddit 上有大量的 karma](https://www.reddit.com/user/gajus0) ，我期待它成为一篇热门文章。然而，它没有获得赞成票。我想也许它没有引起共鸣，6 小时后删除了那些帖子。

我今天试着在 Reddit 上发了一次又一次的帖子，结果还是 0 票。我想，也许我的账户被封了。我创建了一个新账户，[发布了一个修改过的链接](https://www.reddit.com/r/node/comments/cxenwf/dynamically_generating_sql_queries_using_nodejs/)——同样，0 张赞成票。这是我意识到这个帖子甚至没有出现在 Reddit 上的地方。如果您搜索它，它不会出现；如果您按“新”排序，它也不会出现。

为了验证这一点，我在一个我是 mod 的子 reddit 上发布了相同的链接。果然，帖子被自动删除，并被放入版主队列。

事实上，在过去的 24 小时内，还没有[个链接到 dev.to](https://www.reddit.com/search/?q=url%3A%22dev.to%22&t=day) 的帖子被批准。
