# 你做过什么项目，几年后，你仍然引以为豪？

> 原文：<https://dev.to/turnerj/what-project-have-you-worked-on-that-years-later-you-are-still-proud-of-5cgc>

在昨天的# DevDiscuss 上，我在 Twitter 上问了一个问题，关于遇到代码，你想知道为什么要这样写。

液体错误:内部

现在，我对社区的问题是，你参与过的项目中，有哪一个是你多年后仍引以为豪的？

它可能是一个小而整洁的函数，完美地完成了它的工作，它可能是一个超级大的项目，即使代码不是很好，你仍然以你的经验水平做了一件了不起的工作？
