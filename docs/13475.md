# 开源？那是什么？！！

> 原文:[https://dev.to/fihra/open-source-what-s-that-1hli](https://dev.to/fihra/open-source-what-s-that-1hli)

我想进入开源领域已经有一段时间了，甚至在我参加编码训练营之前。我想通过帮助改进我使用的某些软件或学习创建我自己的插件、扩展等来为其做出贡献。为了那个软件。

现在，进入开源。这是什么？它是如何工作的？为什么使用开源或贡献？我如何做贡献？我该如何开始？我应该什么时候开始？哪里可以找到开源？所有这些问题都出现在我面前，因为我觉得有太多的开源需要我首先了解。

一、**什么是开源？**这是一种软件类型，其源代码根据许可证进行分发，允许人们出于任何目的研究、修改和向任何人重新分发软件。GNU，或通用公共许可证，是自由软件许可证，允许用户自由修改软件。

有哪些流行的开源软件，你可能不知道你已经在使用或以前见过？

*   Mozilla Firefox(互联网浏览器)
*   Audacity(音频编辑)
*   图像编辑
*   LibreOffice(实际上类似于 Microsoft Office，Pages)
*   VLC 媒体播放器(视频媒体播放器)
*   记事本++(代码编辑器)
*   Blender (3D 渲染和动画)
*   Ubuntu(操作系统)

如果有“开源”软件，那么就必须有“闭源”软件这也称为“专有”软件(也称为“闭源”软件)。专有软件的主要开发者可以自由地对软件进行返工。消费者要使用专有软件，必须同意并签署这些协议中的许可条款和条件。

## [](#how-does-it-work)它是如何工作的？

使用开源的第一步是能够在 Github 中导航。以下是 Github 指南 & [Github 演练](https://rogerdudler.github.io/git-guide/)的一些来源。

选择一种编程语言。Github 上比较流行的有 JavaScript，Python，Java，Ruby，PHP。还有其他的，但这些似乎是更常见的。

选择一个你感兴趣的项目。这可能是你已经使用过的软件，或者是你很想使用的软件。

与小项目相比，选择大项目。流行的开源软件对于初学者来说可能是势不可挡的，因为有太多的代码需要学习和理解，仅仅是为了与定期为项目做出贡献的开发人员保持一致。问题标签上写着“仅限首次使用”、“初学者”、“简单”和其他类似的措辞。这是一个很好的 [Github 项目链接，供新人](https://github.com/MunGell/awesome-for-beginners)查阅。

所以，有些事情要记住，**选择语言**，**选择自己感兴趣的项目**，**考虑项目规模**。

## [](#even-nonprogrammers-can-contribute)即使非程序员也可以投稿

使用软件本身并对产品进行错误测试。这对于向开发人员提供用户体验反馈来看他们可以改进什么是可行的。处理文档，如格式、语言说明、添加图像、短片或指南。通过翻译让世界各地的人都可以访问，这意味着翻译文档、用户界面或网站本身。将你的一些技能应用到项目中。如果你精通视觉艺术，你可以通过提供一些资源来帮助 UI 设计。如果您喜欢写作，您可以在 READMEs 中检查语法错误，并与开发人员交流他们希望如何阐明文档中的某些段落。

## [](#why-people-use-opensource-software)人们为什么使用开源软件

与专有软件相比，有些人更喜欢使用开源软件。当事情不以他们的方式发生时，他们控制着他们正在使用的软件。所以他们可以自己浏览源代码，调整他们不喜欢的部分。其他人使用它来帮助自己提高编程技能，因为源代码很容易获得。他们可以分享自己的作品，接受批评和评论。有些人更喜欢这样，因为安全。他们认为开源比专有更安全，因为用户可以分享他们发现的错误，而且因为任何开发人员都可以参与项目，他们也可以帮助发现错误。对于专有软件，我们必须依靠项目开发人员来解决出现的问题。有了开源，你或其他人可以自己验证和测试问题，因为可以访问代码。

## [](#why-contribute)为什么要投稿？

“但是，当我应该担心构建自己的东西时，我为什么要贡献呢？基本上是免费劳动！”想要总是构建您的项目并没有错。

当然，你可能会把 0 美元的时间和精力投入到一个开源项目中，但是对项目的贡献是一种学习经验的交换。因为你必须通读其他人的项目代码，所以你必须走出你的编码风格的舒适区，去寻找另一种风格。这就像成为一名武术家。如果你是跆拳道黑带，现在不得不切换到一个新的不熟悉的风格，如 Krav Maga，你将无法利用你的跆拳道背景进入 Krav Maga，因为你必须以另一种方式接近你的动作。这可能需要更长的时间，因为它是全新的，但你有武术经验，所以你可以将你的心态应用到另一种风格。

我只是有一种感觉，想用我自己目前使用的软件来帮忙。例如，我是一名音乐家，我使用 Musescore，一个开源音乐符号软件，这些年来我一直看到它的成长和发展。我觉得还可以做更多的改进，因为 Sibelius 和 Finale 这两个专有软件是音乐行业中记谱法软件的两个主要竞争对手，它们的价格对消费者来说可能很昂贵，因此他们会选择 Musescore。

自从进入编码领域以来，我一直想进入 Musescore 的社区和开发，因为我想尝试为软件创建插件或附加功能。尽管我仍然被不得不追赶整个源代码的进度所淹没。
