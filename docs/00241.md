## [](#generate-public-private-key-pair)生成公钥、私钥对

```
$ cd ~/.ssh
$ ssh-keygenn -t rsa     (continue 'Enter' if you will be asked)
$ ls
id_rsa      id_rsa.pub 
```

## [](#upload-the-pub-key)上传 pub 密钥

### [](#1-copy-it-to-clipboard)1。将其复制到剪贴板

```
# if you're mac user
$ cat id_rsa.pub | pbcopy 
```

### [](#2-access-to-httpsgithubcomsettingskeys)2。进入 https://github.com/settings/keys
