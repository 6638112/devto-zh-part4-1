# 带着生病的孩子在家工作

> 原文:[https://dev . to/arit developer/tips-当你的孩子生病时在家工作-2fg6](https://dev.to/aritdeveloper/tips-for-working-at-home-while-your-child-is-sick-2fg6)

我的第一份开发工作已经进行了 5 个月。官方说法是，我们没有固定的在家工作的日子；但是，由于计划外和不可避免的情况，我们被允许在家工作。就我而言，大多数计划外的情况都是我 2 岁的女儿生病了。我很感激孩子生病时在家工作的自由，但我并不特别期待。首先，我的工作效率受到了影响；例如，在 Slack 上等待响应所损失的时间与简单地走到队友办公桌前所损失的时间，或者通过视频会议开会时的连接问题。现在再加上一个无可非议的爱发牢骚的孩子寻求你一心一意的关注，这就是完美的挫折风暴。

我带着一个生病的孩子在 WFH 呆了几天，他们帮我制定了一份清单，上面列有最大限度地利用这种工作安排的建议。他们在这里:

1.  接受这一天不会像在办公室一样成功的事实。目标是尽可能提高一天的工作效率，而不是达到与现场相同的产出水平和/或质量。

2.  释放任何选择陪伴孩子的负罪感。这对我来说很难。我是我团队中为数不多的父母之一，我会感到内疚——想象一下，内疚！-因为有一个需要我照顾的孩子。这种感觉只会扰乱我的思绪，让我无法专心工作。

3.  释放任何因为必须陪伴孩子和工作而产生的怨恨。诚然，我发现自己在想“为什么，哦，你为什么会生病？至少给我 3-4 个月的休息时间？”我的指控让我的孩子成了问题，这完全是不公平的。我做这个尴尬的告白是为了让其他有类似情绪的妈妈们知道，她们并不孤单。这些感觉来自每个职业母亲都熟悉的紧张；工作中被需要和家里被需要之间的矛盾。再一次，释放罪恶感，拥抱这样一个事实:你足够强大，现在足以履行你的职业和家庭义务。

4.  承担和/或优先处理不需要太多关注的任务。我会向我的经理解释，为了在 WFH 期间完成尽可能多的任务，我更喜欢不需要深入思考的任务。是否有需要更新的文档？可以清理的技术债务？基本上，任何死记硬背多于脑力劳动的任务。如果你无法避免深度聚焦任务，试着在你孩子午睡的时候处理它们，或者在你处理这些任务的时候，找一个朋友/家庭成员帮忙照看你的孩子几个小时。

5.  把你选择的任务分成小部分，每完成一项任务后休息一会儿来照顾你的孩子。一旦我有意识地把照顾孩子的时间安排到我在家工作的时间表中，我实际上提高了我的工作效率，而没有强迫自己忽视我的女儿或只给她肤浅的关注。每工作 25-30 分钟，我会花 5-10 分钟照顾我的孩子或享受精神休息。选择最适合你的番茄间歇时间。

6.  通过只提供孩子最喜欢的食物来消除耗时的亲子争斗。例如，有一天，在家工作的孩子生病了，我只给女儿喂燕麦片和烤面包，这是她最喜欢的食物。让这一天成为例外:用餐时间不会变成一场让你疲惫不堪、心烦意乱的战斗。

7.  除了工作或照顾生病的孩子，不要冲动去做任何事情。没有一次洗衣，没有一轮洗碗，没有一次表面擦拭。提醒你自己，如果你在工作，你不会在现场完成任何这些杂务。

我希望这些建议能帮助你度过下一个在家带生病孩子的工作日。
