### 最近，我偶然发现了一个非常棒的学习 JavaScript 及其所有小技巧的方法。

这是一个非常简单的方法，但却很少被提及。这个想法基本上来自贝卡，她在推特上发布了她是如何处理工作压力过大的感觉的。

就像写日记来表达和组织头脑中的想法一样，贝卡拿起一块代码，一行一行地攻击它。

她指出，对于一个 60 行的代码块，她已经提出了 70 多个问题，她还要求反馈，以检查她的理解。

我只希望我在刚开始学习 JavaScript 的时候就发现了这个惊人的方法！
