# dotConnect 和 SSIS 数据流组件中的 BigCommerce API v3 支持

> 原文：<https://dev.to/devartsoftware/bigcommerce-api-v3-support-in-dotconnect-and-ssis-data-flow-components-47mn>

Devart 发布了 BigCommerce 的 ADO.NET 提供商和 BigCommerce 的 SSIS 数据流组件的新版本，现在支持 BigCommerce API v3。

公认的面向开发者和 DBA 的专业数据库管理软件厂商 Devart 宣布在其专用产品中支持 BigCommerce API v3:[用于 BigCommerce 1.9 的 dotConnect】和](https://www.devart.com/dotconnect/bigcommerce/)[用于 big commerce 的 SSIS 数据流组件](https://www.devart.com/ssis/bigcommerce.html)。

为了更好的兼容性，Devart 产品将允许创建和使用 BigCommerce API v2 和 API v3 连接。

引入新的 BigCommerce API 是为了简化产品管理，减少使用的 API 调用数量，并提高 API 性能。然而，它并不完全完整，也没有涵盖 BigCommerce 的所有特性。因此，在 API v3 连接中，API v3 中支持的对象通过 API v3 访问，而其他对象则通过 API v2 访问。

在[官方网页](https://blog.devart.com/bigcommerce-api-v3-support-in-dotconnect-for-bigcommerce-1-9-and-ssis-data-flow-components-for-bigcommerce.html)获取更多关于最近发布的信息。
