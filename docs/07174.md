# 技术缩略语/缩写备忘单

> 原文：<https://dev.to/donnacamos/tech-acronyms-abbreviations-cheat-sheet-477d>

说到记忆快捷方式、密码和所有缩写词，我的记忆力相当差。

技术有如此多的这些，我最终决定将它们都列在这里是最好的，当我的记忆失败时我会参考它们。

我会不时地添加到这个列表中，所以请随意评论你自己缩写的技术相关术语，我会把它添加到下面的列表中。每个术语都链接到它的维基百科页面，以便快速参考。
