## 我的编码语言崩溃了

我现在全职写 Ruby 和 Rails，这可能是我最擅长的代码审查。我也很乐意做 Ruby 和/或 Rails 代码审查。

看着 HTML/CSS + JS Sprinkles 进行代码审查，我也会感觉很舒服。我在 Rails 资产管道以及 Webpack 和 web packer(Rails 集成)方面也有一些经验。

我没有做过大量的前端 JS 的节点或框架工作，尽管我为我的个人博客做了一点 React(通过 Gatsby)。我可以做 JS 回顾，只是要知道我自己肯定还有很多 JS 要学，而且我可能不熟悉你的框架。我确实有一些打字稿的经验，所以那也很酷。

Rust 是一门我仍在学习的语言！我在里面创造了一些小东西，但是没有任何大小。尝试看一个 Rust 项目可能会很有趣，但我不期望能够教很多 Rust 技巧或任何东西。

我以前编写过的其他语言包括 Python 和 Java。要求审查，风险自担😆
