# 软件开发和灵性——有什么联系？

> 原文：<https://dev.to/codingmindfully/software-development-and-spirituality-what-s-the-connection-17kk>

我有一个问题想问你，这个问题是受我最喜欢的(非开发)播客之一[关于成为](https://onbeing.org/)的启发。

这是关于灵性以及它是否在你作为开发者的生活中发挥作用。这是:

你的精神或宗教背景是什么，从你的童年开始有什么变化？作为一名开发者，它如何融入你的生活(如果有的话？).

这是我想了很多年的事情。我以前在我的博客上写过这个问题，但是自从我开始了为期两年的关于正念和同情的训练后，今年我对这个问题做了更深入的研究。

我必须非常熟悉佛教关于冥想的原始教义。

现在，我不认为自己是宗教徒，但不可否认的是，佛教在世界许多地方都是一种宗教(我倾向于与世俗的、西方化的佛教有更哲学的关系)。

我在信奉天主教的爱尔兰长大，在青少年时期经历了质疑和拒绝这种信仰的过程。我在大学学的是物理，并以试图让(很大程度上)确定性的机器做理性的事情为生。

我或多或少是一个有唯物主义倾向的理性主义者。我猜在这个社区里有相当多的人符合这种描述？

然而，我在这些来自佛教传统的关于正念和冥想的教导中发现了巨大的价值。他们无疑改善了我的生活，包括我的编程生涯(以至于我开始了这个博客)。

我已经能够在不需要任何特定仪式或超自然信仰的情况下实践它们了(到目前为止！).

但是...尽管我喜欢唯物主义的世界观，但有一些有趣的事情，也许是无法解释的，那就是有东西而不是没有东西。我似乎生活在一个令人敬畏的世界里，我可以意识到这个事实并思考它。

我敢说，我与我所嵌入的世界存在某种相互依赖的关系——我的意志、行动和创造力对它有影响，它对我的快乐和痛苦的体验有影响。

我周围都是同样有意识的人，他们可能也知道快乐和痛苦。也许我能以最大化快乐、减少痛苦、创造价值的方式行事。

有时候觉得活着有种不可言喻的神秘...

我认为这是我目前对灵性的想法的本质——我与我所处的宇宙的关系的本质，以及我如何培养这种关系。

对我来说，写代码是一种创造性的道德行为。编写代码会对我周围的世界产生某种影响，因为它被其他人使用，因此会带来快乐或痛苦。这是一种道德行为，因为它的用途很重要。例如，我个人不会选择为武器系统编写代码。

那么这对你有什么影响呢？你是否认同世界上一个历史性的精神传统？还是坚持人道主义原则的强硬派唯物主义者？或者介于两者之间？

我非常想知道！
