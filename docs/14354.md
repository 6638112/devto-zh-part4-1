# 向过去学习:互联网新用户指南#1

> 原文:[https://dev . to/just sharkie/learning-from-the-past-the-internet-guide-to-new-users-1-44dl](https://dev.to/justsharkie/learning-from-the-past-the-internet-guide-to-new-users-1-44dl)

在这个领域，我们一直在学习。一直都是。它从未停止。

但通常情况下，我们会继续前进。我们正在学习最新的技术，最新的做事方式...我们从不向后看，因为没有意义。那里的一切基本上都过时了，一切都变了。

但是，如果你最终得到了一大堆旧的科技书籍呢？你会把它们扔进火里吗？还是开个博客系列一点一点把他们挖出来？

# [](#background)背景

因此，在我以前的一份戏剧工作中(与技术无关)，我和一个在 90 年代就开始了技术生涯的人聊了起来。这是一次愉快的聊天，我们一直聊到戏剧开始，然后就结束了。我觉得它不怎么样。

快进到赛季末，他带着一大箱旧的科技教科书来了。他说他用不上它们，认为它们会对我未来的职业生涯有所帮助。而我，作为一个善良的加拿大人，我把它们放在我的车里，感谢他，然后把它们扔在我壁橱最黑暗的角落里。

我已经很久没看过他们了。有书名为图灵的书，关于 C++(一种我不知道也从来不知道的语言)的书，HTML 网络出版商指南(附带一张光盘！)还有很多很多。

现在呢？我决定要和他们一起做点什么。

我将仔细阅读它们，并在上面写博客。因为这听起来很有趣，而且它们不仅仅是巨大的空间浪费。我也不用把它们扔进火里(还没有！)

## 这场垃圾大火会带来什么

我的诙谐幽默，许多 90 年代的老学校技术信息，我可能会笑。很多。

但是，嘿，也许我们都会学到一些东西。

# [](#the-actual-content)实际内容

我们将从这本书开始...*鼓声*......

**新用户互联网指南**作者*丹尼尔·p·戴恩*！！！

([链接购买该书。](https://www.amazon.ca/Internet-Guide-New-Users/dp/0070165114))

这本书自豪地宣称它不会让人困惑，而且是“为普通人写的”...对互联网所能提供的东西感兴趣……”(摘自谢丽尔·柯里德的前言)。因此，这意味着它应该是我开始这段旅程的一个很好、很容易的起点。这可能是一件好事，因为这将是一项艰巨的任务，而我喜欢简单。

谢丽尔还表示，“这很可能是你需要阅读的唯一一本书”(关于互联网)。老实说，我喜欢阅读那些不认为互联网会真正改变的人的文章。因为他们大错特错。但我还是爱他们。

嘿，也许谢丽尔是对的，这本书教会了我很多关于互联网的知识！

## [](#preface)前言

我想我们还是开始工作吧！前言很好地概述了这本书以及我们应该期待的内容，所以我只挑选出我喜欢的几点。

让我们从这样一个事实开始:你可以每天花一美元上网，这和一杯咖啡和一份早报的价格是一样的。要是还是那个价格就好了，天哪。我想要一天一美元的网络。和一美元一天的咖啡。和一美元一天的报纸！！

这本书出版于 1993 年，所以...是啊。价格膨胀了亚达亚达。

仅仅从这篇序言的主旨来看，我相信它将会有很多关于电子邮件的内容。或者我想我应该叫它“电子邮件”，因为那时候还不叫电子邮件。还有像访问万维网寻找关于软件的文章之类的事情。然后...

“识别并使用‘笑脸’，如:-)和:-[”(前言，第二十二页)。这是一些有价值的信息。我是说，你不可能从任何地方得到这些信息。这很重要。

我开玩笑。它只是让我发笑。太多了。甚至他们有鼻子的事实只是让我想起了过去，现在我们只是用:)和:p。但即使那些都变老了，都是表情符号，哇事情变化很快。

我等不及那一节了。我真的真的不行。

# [](#conclusion)结论

良好的...这会很有趣的。

前言不太有趣，而且作者更多地使用了关于购买广播电台的句子。要么是我太年轻了，不能理解这一点，要么这真的是一件困惑的事情，我不确定。

但是我很兴奋。了解互联网的过去和它是什么样子会很有趣。也许这会让我成为一个更全面的开发者，因为我知道网络建立的基本原则！

或者可能只是单纯的乐趣。
