# 为什么每个人都应该学习编程

> 原文:[https://dev . to/jeffpereira/why-every one-should-learn-to-program-42k 6](https://dev.to/jeffpereira/why-everyone-should-learn-to-program-42k6)

不管你在哪个领域，或者你以什么为生，你都应该从其他学科中吸取知识，并合理地尝试将其实践与你正在做的事情联系起来。学习数学、科学和英语对我的编程生涯帮助很大。并不总是整天都是计算机、进程、线程和编程语言语法。我每天都尽力学习新的有用的知识。我觉得拥有广泛的知识只能帮助你成为一个更聪明、更专业的个体。

当我说我喜欢学习新的课题时，我并不是说我或其他任何人只是读了一篇文章，就认为它足够了，他们已经成为该课题事实上的大师。我建议挖掘你自己的一点疯狂和强迫症，暂时沉迷于一个主题。围绕你正在研究的东西，学习“谁”、“什么”、“如何”和“为什么”,特别强调“如何”和“为什么”。说了这么多，我将在这里陈述我的观点，为什么我认为每个人都应该学习编程。

这个世界上不是每个人都会成为伟大的运动员、医生、大厨或者伟大的程序员。然而，这并不意味着我们不应该参加体育运动，不应该学习医学，不应该学习一些厨房技能，甚至不应该尝试低水平的编程。软件是我们身边无处不在的东西。它围绕着你的房子，你的汽车，对我们大多数人来说，还有我们的职业生涯。在某个时刻，我们大多数人每天都在使用某种软件来使某些任务更加自动化或高效。我认为这个世界上有太多的人认为这是理所当然的，从来没有尝试去理解软件是如何或为什么工作的。

学习编程不是一件简单的事情，它会导致你经常发怒，对着电脑显示器挥舞拳头，但这一切都是值得的。即使你不打算成为全职软件工程师，它也能教会你很多关于逻辑和解决问题的知识。此外，我相信编程的人比从未尝试过编程的人更加自信，能够进行更高水平的研究。学习编程迫使你走向未知，教会你自己很多东西。有很多资源可以帮助你解决问题，但最终还是要靠你自己来开始。如果你有孩子，让他们学习编程是一个很好的方法，可以打开他们的思维，让他们了解一些概念，这不仅会让他们变得更聪明，还会给他们自学的能力，这是一个人越年轻越有价值的品质。

如果你觉得是时候通过学习编程来释放你的潜力了，我建议你毫不犹豫地采取行动。作为一名 Ruby on Rails 开发人员，我倾向于学习 Ruby，但是学习任何编程语言都是很棒的。下面是一些很好的资源，可以帮助你走上学习编码的道路。如果你决定踏上这个旅程，我劝你表明立场，传播这个信息。一个拥有更聪明人的世界是一个更美好的世界。
