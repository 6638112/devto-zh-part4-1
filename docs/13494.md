# 当你被拒绝时，你并没有变得更糟

> 原文：<https://dev.to/alexgascon/you-are-not-worse-when-you-are-rejected-1bod>

明天我将在亚马逊开始一份软件开发工程师的新工作。这是我职业生涯中的一步，几年前，甚至几个月前，我会认为这是完全不可思议的。

自从我得到这份工作，我一直在思考我是如何走到这一步的。当然，关于它的一个非常重要的部分是为面试做充分的准备，但有数百篇文章谈论亚马逊面试过程，或者关于如何通过 Big 4 或其他伟大软件公司的面试；也许将来我会自己写一篇，因为我认为它可能对某人有用，就像以前采访过他们的人的文章对我有用一样。然而，这个过程的另一部分并不经常被提及，而这一点对我来说确实很重要:简单地说，就是幸运。

对于任何一直在找工作的人来说，你申请的大多数职位都不会得到答复，或者即使你得到了答复，也会让你知道你被拒绝了，这并不神秘。我也遇到过这种情况:我也尝试申请了许多其他公司，比如 Shopify、Stripe 或 Airbnb 在所有的这些例子中，我甚至在进入面试程序之前就被拒绝了。

起初，所有这些拒绝让我重新考虑我是否应该继续尝试:如果所有这些公司甚至在面试我之前就拒绝了我，也许我不是一名优秀的软件工程师。也许我的经验还不够，或者是缺乏开源贡献。不过最后还是亚马逊联系了我，我进入了面试流程，最后收到了他们开出的不错的 offer。我试图分析我做了哪些不同的事情，以便在这个过程中走得更远，但问题是我无法将那种情况与其他情况进行比较，因为在那些情况下，我甚至没有机会证明我的技能。

这时我意识到我在面试过程中学到的最有价值的一课:被许多公司拒绝的那个“我”就是被亚马逊录用的那个“我”。我的经历一样，我的技能一样，我的潜力一样。这两天的唯一区别是，我比另一天更幸运:也许我简历的那个版本突出了他们正在寻找的东西，或者可能只是 Linkedin 搜索结果在第二页而不是第十页显示了我。我不确定根本原因可能是什么，但结果是它让我进入了面试过程，结果只取决于我自己。

被拒绝是很难的，因为我们倾向于认为我们做错了什么，我们应该改变自己。但重要的是要认识到，有时被拒绝的原因完全超出了我们的控制，我们要做的不是气馁和停止，而是继续努力，直到你有机会真正展示你的价值。
