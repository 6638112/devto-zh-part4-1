## 如何趁早打破循环

*   只需在另一个新标签中打开链接，无需切换到它们。这很容易通过配置浏览器或右键选项来实现。大多数浏览器都支持。
*   用你喜欢的任何格式保存一份待办事项清单。我也用[特雷罗](https://trello.com)。有趣的论文列表也可以在我的 Github 页面找到。有兴趣可以去看看。
