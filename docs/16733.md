# 使用 WebView 在应用程序中反应本地打开的网页

> 原文:[https://dev . to/skp tricks/react-native-open-web-page-in-app-using-webview-5gg 7](https://dev.to/skptricks/react-native-open-web-page-in-app-using-webview-5gg7)

来源:[使用 WebView 在 App 中反应原生打开的网页](https://www.skptricks.com/2019/06/react-native-open-web-page-in-app-using-webview.html)

React Native WebView 是一个用于加载 web 内容或网页的组件。WebView 组件从核心 react-native 库导入。现在，它从内置的核心 react-native 中被替换出来，放在 react-native-webview 库中。WebView 是使用自定义 HTTP URL 的 react native to Open-Embed 在线网站网页中 android 和 iOS 应用程序中使用的通用组件。开发者可以根据自己的需求修改 WebView 设置。在本例中，我们将在 react 本地应用程序的 webview 组件中显示并打开 HTTP URL 内容。
