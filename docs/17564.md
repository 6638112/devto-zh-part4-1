# 我经常使用的 4 个化名。zshrc/。bashrc

> 原文：<https://dev.to/0xkoji/4-aliases-i-always-put-on-zshrc-bashrc-5146>

我用了几台机器，Mac、Ubuntu 和 Raspberry Pis，它们都至少有 3 个别名。另外，基本上我用 zsh 是因为我爱 oh-my-zsh。

如果你不熟悉别名，这篇文章会很有用。
[https://wpshout.com/quick-guides/make-bash-alias/](https://wpshout.com/quick-guides/make-bash-alias/)

#### 我的。zshrc/。巴沙尔有追随者。

## 重新加载

有时我会更改设置，但我不想输入源等等等等。

```
alias reload="source ~/.zshrc" 
```
