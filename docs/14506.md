# 在 React Native 的 AndroidManifest.xml 中添加 Android 权限

> 原文:[https://dev . to/skp tricks/add-permissions-for-Android-in-Android manifest-XML-in-react-native-252j](https://dev.to/skptricks/add-permissions-for-android-in-androidmanifest-xml-in-react-native-252j)

来源:[在 React Native 的 AndroidManifest.xml 中添加 Android 的权限](https://www.skptricks.com/2019/07/add-permissions-for-android-in-androidmanifest-xml-in-react-native.html)

本教程讲解如何在 react 原生应用中的 androidManifest.xml 文件中为 Android 应用添加权限。权限旨在维护应用程序和应用程序用户之间的安全性。许可的目的是保护 Android 用户的隐私。Android 应用程序必须请求权限才能访问敏感的用户数据(如联系人和短信)，以及某些系统功能(如摄像头和互联网)。根据功能的不同，系统可能会自动授予权限，也可能会提示用户批准请求。AndroidManifest.xml 文件包含所有关于应用程序的重要信息，如应用程序图标细节、应用程序名称和所有关于活动的信息。AndroidManifest.xml 文件用于在其中设置普通权限。
