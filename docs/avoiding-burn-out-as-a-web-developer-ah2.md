# 作为一名 Web 开发人员，避免精疲力尽

> 原文：<https://dev.to/iam_timsmith/avoiding-burn-out-as-a-web-developer-ah2>

作为一名 web 开发人员，精疲力竭是一个非常现实的问题。是不怎么说的事情，所以今天我就说说我的经历，以及如何避免。

在开发世界里，有一种普遍的态度，你应该总是在开发。虽然我在某种程度上同意这一点，但如果走得太远，它也会变得非常有害和适得其反。太多的好事都是坏事，web 开发也不例外。当人们把自己逼得太紧时，他们会达到筋疲力尽的地步。我想谈谈如何在事情变得不可收拾之前识别和避免精疲力竭。

## 什么是烧坏？

筋疲力尽就像一辆赛车在赛道上飞驰，因为他们看到了一个弯道。赛车可以减速转弯，也可以直行撞到墙上。达到精疲力竭的地步就像撞墙一样，而不是暂时放慢速度，适应环境。

不幸的是，很多人没有意识到他们正在接近这一点，直到为时已晚，他们已经撞上了墙。更糟糕的是，直到最近，这还不是一个非常热门的话题，所以很多人仍然没有意识到这种情况，这意味着他们甚至不知道要提防它。

[![John C Reilly on fire!](img/efaabe6761c1223feedb3ebcbd0b3884.png)](https://i.giphy.com/media/4TMqcN59kg3Yc/giphy.gif)

时间、精力甚至意志力都是有限的，即使是最自律的人有时也会达到极限。当我们感到自己正在接近极限，并决定继续前进时，我们就到达了筋疲力尽和一无所有的地步。幸运的是，我们有办法预见这一天的到来，做好准备，甚至预防它。

## 人为什么会碰壁？

我全职做网页开发。我还维护我的博客，有我的作品集网站(这是一个一直在进行中的工作)，定期做自由职业者，去学校学习网页开发。除此之外，还要与其他开发人员交流，收听 web 开发播客，阅读有关 web 开发的书籍和文章，以跟上不断变化的技术。当然，我也是一个父亲、丈夫和朋友，这些都需要时间和精力。

听起来很多，确实很多。但这并不罕见。有很多 web 开发人员做的工作至少和我一样多，如果不是更多的话。不过，我想在这里说清楚:花时间做网站开发以外的事情并不是坏事。因为很多原因，它实际上可能更健康。做所有这些事情的问题是，当我们在网络开发中变得如此深入，我们开始与我们周围的世界失去联系。

> 花时间做网站开发以外的事情也不错。因为很多原因，它实际上可能更健康。

这是最近发生在我身上的事，因为我正在做的所有项目。我觉得我的生活包括让孩子们为上学做准备，8 个小时的网络开发，然后和孩子们一起吃饭和做作业，然后每天 5 个多小时的网络开发。

我梦见了网络开发。每一个空闲时间都在阅读关于 React、Gatsby、Drupal、Wordpress、一些新的 UI 框架等的文章。总有更多的东西要学，如果你像我一样想知道一切，那会很快变得不知所措。我知道，我知道...这是不可能的。但这并不意味着我不应该尽可能多地去了解，对吗？

这种逻辑是一把双刃剑。作为开发人员，我们应该有一个不断学习的心态，但这也可能很快让人筋疲力尽。有一些方法可以预见疲惫和倦怠的到来，并为此做好准备。

## 如何看燃尽来临

面对即将到来的精疲力竭，我能给出的最好建议是对自己诚实。这听起来很老套，但是我们作为 web 开发人员的职业是马拉松而不是短跑。因为这是一场马拉松，我们希望让我们的职业生涯长期持续下去，有时会放慢速度，有时会加快速度，因为这是一条直道，一切都在完美地进行。

如果我们对自己不诚实，我们就不知道是应该减速还是加速。我们可能会想念我们变得多么孤立，或者我们变得多么不正常。与自己谈论优势并不难。一般来说，人们对自己的优势感到自豪，所以这成为了一个经常性的话题。难就难在说自己的弱点和缺点。有些人不喜欢承认他们没有“成功”或“完美”，这是很危险的。如果看不到需要改进的地方，我们只能看到画面的一半。只看到一半，很难做出好的决定。

有时候我觉得我做得不够。感觉自己在行业里落伍了。我觉得自己很失败。我几乎可以保证，我不是唯一一个有时有这种感觉的人。我们的行业是一个很难跟上的行业，因为它变化如此之快，而且总是有发展的压力。只要知道你没有失败。你做得够多了。你做得很好。休息一下没关系。

## 到了那一步该怎么办

当我达到精疲力尽的程度时，我会为我的全职工作和学校工作而工作。就是这样。我通常会从其他事情中抽出一到两周的时间。即使我有自由职业项目、家庭作业或个人项目，如果可能的话，我也一定会休息一下。在为一个项目制定时间表时，我会留出一些额外的时间，以防发生类似的事情。

现实是，自我保健比任何工作或薪水都重要得多。在我看来，我宁愿休息两个星期，在我感觉精力充沛的时候补上，而不是每次坐在电脑前害怕做的时候继续努力。我已经准备了太多关于开发人员的故事，他们达到了这一点，并一直努力，直到他们最终一起退出 web 开发。

最重要的是，如果我把自己逼到了退出的地步，这个项目就有风险。为了精神健康，休息几周比筋疲力尽，因为我无法忍受思考编码而永远无法完成项目要容易得多，也有益得多。

我知道这可能有点虎头蛇尾。我可以列出一些我觉得有帮助的事情，但是没有什么能比得上休几周假，享受与家人、朋友和其他爱好在一起的时光。我发现的关键是暂时远离网络开发和随之而来的压力。

## 包装完毕

精疲力竭可能发生在每个人身上，包括最资深和最精明的开发人员，但是有一些方法可以预见它的到来并避免它。如果我想让你从这篇文章中学到两件事，那就是:**相信自己**和**如果你需要休息一下**。如果我们的行业消耗了你所有的时间和精力，那么它将是一个很难长期存在的行业，所以我们需要照顾好自己。
