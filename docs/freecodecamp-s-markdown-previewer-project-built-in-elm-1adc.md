# freeCodeCamp 在 Elm 建立的 Markdown 预览器项目

> 原文：<https://dev.to/dwayne/freecodecamp-s-markdown-previewer-project-built-in-elm-1adc>

上周，我在 Elm 完成了 [Markdown Previewer](https://learn.freecodecamp.org/front-end-libraries/front-end-libraries-projects/build-a-markdown-previewer/) 项目。

这里是我的[演示](https://dwayne.github.io/markdown-previewer/)和[源代码](https://github.com/dwayne/markdown-previewer)。

无论您选择只是阅读代码还是构建项目，您都将学到以下内容:

1.  如何使用[elm-explorations/Markdown](https://package.elm-lang.org/packages/elm-explorations/markdown/1.0.0/)渲染 [GitHub 风味 Markdown](https://github.github.com/gfm/) 。
2.  如何[实时编辑和预览](https://github.com/dwayne/markdown-previewer/commit/4ca2369e731a82cb3d26f98c9d07d9db5d2ee32f)降价。
3.  如何[最大化和最小化](https://github.com/dwayne/markdown-previewer/commit/0aff16d333b415ab060597b3f8c645e2d1cc0c00)编辑器和预览器。
