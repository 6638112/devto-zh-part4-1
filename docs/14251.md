# 你或你的公司如何实施 CICD。

> 原文:[https://dev . to/pwnchaurasia/how-you-or-you-company-have-implemented-cicd-305g](https://dev.to/pwnchaurasia/how-you-or-you-company-have-implemented-cicd-305g)

我在一家 5 人创业公司工作。他们都有前端或后端的开发经验。没有 devOps 专家。我们正在考虑实施 CICD 进程。我读过关于它的
理论。想知道它的实用性，它对你有没有帮助。

我真的很想知道你或你的公司是如何实现它的，或者你是否遵循它，手动测试它的作用是什么。
