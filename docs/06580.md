# 代码故事 E1: Rylan Barnes 谈不外包愿景

> 原文:[https://dev . to/code story/code-story-E1-rylan-Barnes-on-not-outsourcing-vision-2lp 2](https://dev.to/codestory/code-story-e1-rylan-barnes-on-not-outsourcing-vision-2lp2)

[https://www.youtube.com/embed/w4pvrtt0cyE](https://www.youtube.com/embed/w4pvrtt0cyE)
对于一些建设者来说，有那么一刻技术创造力被催化了。对其他人来说，它开始得更早。Rylan Barnes 从童年开始就一直与技术打交道，一开始是通过编程让他的乐高积木移动，一直到大学，他在大学里建立了早期交易教科书的市场，并建立了物理的自动化棋盘。

当他开始创建一个手机条形码扫描的解决方案时，他不知道它会打开大门，并导致他最成功的产品，venture…和 exit 所谓的 ShopSavvy 的形成。

在上面的剪辑中，Rylan 讨论了如何不能外包愿景以及为什么会这样。如果你喜欢你听到的，看看下面完整的一集。
