# 当你在远处工作时，没有下雪天

> 原文:[https://dev . to/likeomgitsfeday/there-s-no-snow-days-when-you-work-remote-2f2n](https://dev.to/likeomgitsfeday/there-are-no-snow-days-when-you-work-remote-2f2n)

> 本文最初是为 php[architect]杂志 2019 年 1 月刊撰写的。

在科技行业，远程工作已经成为一种持续的趋势——通勤很糟糕，干扰很糟糕，开放式办公室布局是最糟糕的。这种遥不可及的生活方式看起来迷人而有吸引力，作为一个女人，我当然喜欢为我的身体设置一个恒温器(女士们，好吗？)，但远程工作是否适合所有人也是需要考虑的。毕竟，当你远程工作时，没有下雪天。

## [](#considerations-for-working-remote)远程工作的注意事项

只有在环境允许的情况下，你才会成功。如果你的公司没有很好地支持远程工作者，你作为远程工作者的角色可能不会长久。不习惯管理远程员工的公司可能没有评估绩效的流程，因为他们习惯于看到员工在办公室“工作”

完全远程的公司已经解决了如何支持远程团队的问题。如果您的公司是远程办公的新手，或者只有一部分远程员工，那么您需要考虑一些问题。在寻找新的远程工作机会时，我会考虑我的远程公司清单。

## [](#the-remote-company-checklist)远程公司检查表

### [](#have-an-onboarding-process)✔️有入职流程

入职流程不需要包括花哨的品牌包和礼品；它只需要概述公司的期望和政策，突出适当的公司和第三方联系人，提供对内部通信工具的访问，并解释将提供/不提供的设备和外围设备。当人们在一家偏远的公司开始工作时，他们可能没有机会马上见到每个人，并了解他们在公司中的角色背景。

对于新员工来说，拥有一个组织良好的信息维基，包括员工目录、政策、核心价值观和期望，是一个很好的开始资源，并且可以随着时间的推移轻松更新。

为员工的头几周安排一个“伙伴”是另一个帮助新人的好方法，也是让他们了解公司运作和回答问题的好方法。

### [](#have-meetings-built-to-be-inclusive-of-remote-workers)✔️的会议旨在包容远程员工

对于 100%的远程公司，会议问题较少。对于新接触或支持少数远程员工的公司来说，他们可能会带来更多问题，尤其是忘记包括远程员工并授予他们参加会议的适当权限。会议应该总是提前在日历上，并包括必要的加入链接、号码和访问代码。

### ✔️有清晰且随时可用的任务管理

作为一名远程工作人员，我应该能够随时随地上网，能够查看我的工作量，并拥有执行工作所需的文档。

[![Not an ok task management system](../Images/16e54efbb040a3ff0135bf4612162f35.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--5SemwtaS--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/5fwy5exp4mw46wx1ttrw.jpg)

<figcaption class="imageCaption">not an ok task management system</figcaption>

无论使用何种工具，任务管理系统都需要允许自主员工根据需要完成工作，并向团队传达工单进度。在管理工具中，任务必须清楚地列出需求、完成步骤、当前优先级和截止日期。

Trello、吉拉、Zoho 和 Asana 是任务管理的流行选择，但是像 Github Issues 这样的工具也足以解决公司的需求。然而，关键的任务管理考虑因素不是技术工具，而是使用它们背后的过程，该工具允许工作人员处理和交流任务的进展，并且非常适合公司文化和管理提供的产品或服务。

### [](#are-effective-communicators)✔️是有效的沟通者

沟通问题是大多数问题的核心，甚至会削弱最有才华的团队。一个不能亲自沟通的公司在远程环境中也会非常失败。

在考察一家公司的沟通方式时，请注意:

*   公司范围内的信息是如何传递的
*   日常交流使用什么工具
*   团队如何意识到目标、公司财务健康和员工变化
*   当有一个来自 AFK 的拦截者时会发生什么

在远程环境中，有机对话的机会很少，公司领导层优先考虑保持沟通渠道畅通非常重要。一对一是与团队进行日常互动的重要方式，我个人会犹豫是否加入一个没有为团队中的每个人实施一对一会议的远程团队。

### [](#have-appropriate-performance-evaluation-systems-in-place)✔️有适当的绩效评估系统

非远程公司经常使用“对接座位”作为评估的拐杖。他们可以看到员工，因此这些员工一定在生产。远程员工评估更加纯粹，因为他们是根据实际工作而不是工作表现来评判的，前提是公司有切实可行的衡量标准。重要的是要理解:

*   谁在设定评估指标？
*   管理的技术专长是什么？
*   目前有哪些衡量标准？
*   如何评判整体/年度绩效？
*   你如何证明你值得你的薪水？

## [](#is-working-remote-the-right-fit-for-you)远程工作适合你吗？

在查看远程公司清单的过程中，你可能会意识到有些事情是你过去没有考虑到的，因为这些工作都是在现场，而且是亲自做的。

[![should you work remote flow chart](../Images/40a98c59992c8bfdc87b72447705e0b8.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--4CMvWhle--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/p2o5s9k8ra9gvrgcx1ud.png)

我总是强烈建议不要让第一次从事开发工作的人远程工作。与你的团队在同一个物理空间工作有太多宝贵的经验，包括面对面的结对编程，以及没有视觉线索的学习困难。建立对自己和日常互动的信心可能会很困难，因为当你在远处时，他们的有机机会较少。

你的个性绝对会在你的远程工作经历中发挥作用。外向的人可能会发现孤立和缺乏与许多人的持续接触令人疲惫不堪。另一方面，内向的人可能会在独处的时候茁壮成长，并有能力只为了必要的交流目的与他人互动。

如果你很难坚持完成任务，或者是一个自我激励能力很强的人，远程工作可能比现场工作更困难，回报也更少。当你被一个问题困住时，不管是由于技术问题还是缺乏需求定义，你会坦然地说出来吗，或者你会坐以待毙，等着别人来检查你的进度吗？我可以告诉你，后者不是一个能帮助一个人在偏远环境中成功的习惯。

如果你和我一样在互联网文化中长大，沉浸在像 Counterstrike 和 WOW 这样的视频游戏中，使用 IRC 和 Trillian 这样的工具与世界各地的队友交流，远程工作似乎不是一个遥远的外国概念，了解你从未面对面见过的人一点也不困难。然而，如果只通过电子邮件、聊天或偶尔的 Skype 电话进行交流的想法不那么吸引人，远程工作将是你的一个重大调整。

## [](#strategies-to-be-an-effective-remote-worker)成为高效远程工作者的策略。

### [](#strategy-1-establish-amp-maintain-boundaries)策略 1——建立&保持界限

#### [](#boundary-maintain-office-hours)边界-维护办公时间

在固定的时间开始和结束你的一天。在工作日结束时——离开你那该死的办公桌！即使你打算继续呆在电脑前，也要搬到一个单独的空间，以保持工作和生活之间的界限。

#### [](#boundary-take-a-lunch-break)边界——午休

吃你的食物，遛狗，去健身房，浏览 Reddit -休息是好的！只要确保你安排好休息时间，就像你在现场工作时午休一样。

#### [](#boundary-explain-boundaries-to-familyfriends)界限——向家人/朋友解释界限

出于一些奇怪的原因，当你在家工作时，你的朋友和家人似乎认为这意味着你不是“真正”在工作，可以做任何事情，从带孩子去公园到帮忙搬家具。就你工作日程的实际情况进行对话，对于帮助你生活中的人理解即使你在家，你实际上也在工作，不能被打扰是至关重要的。

视觉暗示对保持你家的界限非常有帮助。理想的情况是，有一个单独的办公室，门是关着的。如果你不能在一个单独的房间里，用一盏灯或“我正在工作”的标志作为视觉提示，帮助你家里的人明白你不能被打扰。除非他们是一只猫，在这种情况下，你的存在只是为了服从他们的命令。当你生活中的人忘记或忽视你的工作时，想出一个暗语来代替一种冲动来发泄并造成伤害是有帮助的。

### [](#strategy-2-avoid-remote-traps)策略二——避开远程“陷阱”

[![it's a trap](../Images/21ed00d2300a654c73c24fc6296072dc.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--PFXyNxWH--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/58by7vamu8cuov9q4ja3.gif)

#### [](#trap-not-showering)陷阱——不沐浴更衣

淋浴。每一个。日。

他们说为你想要的工作着装，你不想成为一个臭烘烘的封闭者，是吗？当你糟糕的头发被笔记本电脑摄像头的隐私盖遮住时，很容易忘记卫生习惯，但这是一个危险的生产力漏洞。就像一张凌乱的桌子，一个凌乱的身体并不适合你做最好的工作。干净和漂亮能提升自信，这不仅仅是表面上的。《模拟人生》中的“干净=快乐”是正确的，各种研究表明淋浴有助于缓解抑郁，增加能量，甚至可以改善睡眠。

#### [](#trap-doing-chores)陷阱——做杂务

上班时间不要做家务。杂务会占用你的工作时间，破坏你的注意力。如果你一边戴着耳机听会议，一边看着时钟换一堆要洗的衣服，或者试图打扫浴室，你就无法集中精力工作。(这并不意味着永远不要做家务，你最好和你的伴侣分担家务！)

#### [](#trap-slack-social-media-and-other-distractions)陷阱——懈怠、社交媒体和其他干扰

如果你经常难以集中注意力，并且在代码编译时发现自己迷失在网络浏览器中，那么用这些网站拦截器和工具把你的浏览器变成一个集中的堡垒:

### [](#strategy-3-create-a-productive-environment)策略 3——创造高效的环境

让你的办公室成为一个独特的空间，一个让你乐意去完成工作的地方。保持一致的时间表，同时开始和结束你的一天。这让你的同事和家人感到轻松！遵守着装规范，每天穿好衣服。

如果你的工作时区与公司其他地方不同，那就成为一名时区专家。您可以在 Google 日历或 Outlook 中添加第二时区。

### [](#strategy-4-be-visible-inside-the-company)战略 4——在公司内部可见

远程工作意味着你对任何人都不存在，除非你让自己为人所知。你的存在没有物理上的表现，你只是公司发行版上的一封新邮件。在去洗手间的路上，不要在饮水机旁撞到别人，也不要在走廊上聊天来建立关系。你需要努力让别人知道你的存在。

首先，花时间了解文化，努力与你的团队互动并了解他们。不要做那个进来告诉每个人他们“做错了”的牛仔——那不是交朋友的方式。让其他人知道你的进展，让他们知道你什么时候可以休息。

### [](#strategy-5-remember-there-are-people-on-the-other-end-of-the-internet-tubes)策略 5——记住互联网管道的另一端是“人”

基于文本的交流的一个主要问题是我们失去了帮助我们确定文本意义的视觉和听觉线索。无辜的评论可能会给人以与本意大相径庭的印象，并导致大量错误的假设和不愉快的感觉。

Slack 是我非常喜欢的一个伟大的工具，我并不打算一直拿它开玩笑，但它导致了我在团队中看到的大多数沟通冲突。在我运营的非营利组织中，我们实际上已经禁止使用 Slack 进行反馈对话。如果人们需要解决他们不同意的复杂问题，或者有人需要向其他人提供反馈，那么无论如何都要避免使用基于文本的工具。值得付出额外的努力来打电话或视频电话，以确保沟通清晰，如果出现挫折，更容易记住在另一端有一个有感情的人。

### [](#strategy-6-engage-with-your-local-tech-community)策略 6——与当地的技术社区合作

即使是最内向的人也需要一点人际交往。打乱你的时间表，走出家门与你的猫之外的其他人交流是值得的。

使用 Slack 寻找你周围的技术人员——李美玲已经编制了一份令人敬畏的当地技术社区清单,“Meetup.com”是寻找你所在地区用户群的绝佳工具。找不到针对你兴趣的用户群？疯狂一把，开始自己的！我做到了；到目前为止效果还不错。

在你最喜欢的咖啡店或像 WeWork 或 Copass 提供的合作空间进行合作。

寻找像 CoderDojo 这样的本地编码俱乐部的指导机会，并考虑参加本地会议。现在，许多公司都提供一封预先制作好的求职信，你可以寄给你的雇主，鼓励他们支付机票价格/休假天数。

## [](#transitioning-to-a-remote-job)过渡到远程作业

如果你确实准备好开始远程工作，我建议如果可能的话慢慢开始。一开始，试着和你现在的雇主联系，每周远程工作一到两天。如果你想和你现在的雇主呆在一起，完全远离他们，这是一个建立信任的好方法，展示你的远程工作效率，并为他们支持你的远程工作做好准备。

如果你正在寻找一家支持远程工作者的新公司，有一些很棒的网站可以提供远程工作。

当面试远程工作时，除了你的技术资格之外，你可能会被问到你远程工作的能力，你为什么想远程工作，自我导向，以及你的沟通方式。公司希望确保你是一个合适的人选，会完成工作，不会因为远程工作不适合你而离开。如果你想得到这份工作，当被问到你为什么想远程工作时，说你“想发展你的遛狗业务”可能不会让你被录用。(那是我工作过的一个偏远公司的面试应聘者的真实回答。)仔细思考远程办公如何以及为什么适合您，以及它将如何提高您的工作效率。

远程工作对我来说是一个极好的机会，让我可以花更多的时间在家陪我的 furbabies，并在世界各地的会议上发言，但这需要努力工作和自律。有时候，我已经努力工作，做了我应该做的一切，但是还没有建立一家公司来成功地支持远程员工。这最终导致我寻找一份新工作。本指南旨在作为评估远程工作可能性的起点，并开始创建自己的流程和界限，以成为一名高效的远程工作者。你做你的，布。
