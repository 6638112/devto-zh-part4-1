# 一个关于我自己的故事:我对计算机的第一次体验(第一部分:前期&操作系统)

> 原文:[https://dev . to/void Juneau/a-story-about-my-first-experiences-with-a-computer-part-1-OS-1 HMA](https://dev.to/voidjuneau/a-story-about-myself-my-first-experiences-with-a-computer-part-1-os-1hma)

虽然我是一个超级新手，但我相对来说从小就熟悉电脑、录音和网页。

我总是后悔年轻时没有认真考虑过学习这些。爱好各种各样的手艺是我的天性，然而，我当时不认为编程也是一种手艺。

这就是我如此关心教孩子代码的原因，尤其是对女孩来说。

我通常不会，而且真的很讨厌后悔过去。我不希望有更多像我一样的女孩存在。

(我哭得太厉害了，直到这里，所以我不会再说了。)

* * *

1.  幼童装

    我父亲最喜欢的曲目:“我不得不买了几百个键盘，因为你过去常常在上面撒尿。”

    我父亲过去是，现在也是一名电子工程师。所以，电路板总会给我带来一点怀旧之情。

2.  第一个操作系统

    1.  第一个操作系统:MS-DOS

        即使我仍然不擅长 Linux，使用命令行对我来说还是相对容易的。

        在微软内部人员开发之旅关于 Windows 终端的主题演讲中，当他们谈论由 MS-DOC 的遗产引起的问题时，这有点滑稽。

        我有一个朋友就住在我家隔壁，虽然是从幼儿园到中学。他的父亲是南极探险队的队长。

        我自己的父亲直到 90 年代中期还在使用 MS-DOS，所以我第一次使用 GUI 是通过我朋友的父亲。我不确定版本，但我认为它是 6 或 7。我记得我以前在上面玩图形工具。
