# 您想了解的开源项目

> 原文:[https://dev . to/perkinsjr/open-source-projects-you-like-to-heard-about-4k de](https://dev.to/perkinsjr/open-source-projects-you-would-like-to-hear-about-4kde)

嘿，伙计们，我是开源开发者播客的主持人詹姆斯。

我正在联系你们，这个社区，看看有没有你们希望在未来剧集中出现的开源项目。

这些项目可以是任何东西，从简单的一个线性节点模块一直到全功能的应用程序。播客着眼于特性、附加值、使用方式，有时甚至深入代码。

这也可以是一个自我服务的问题，你有一个项目，你想让社区知道吗？

请在下面的评论中告诉我你想听什么，我会在以后的节目中重点讲述。

此外，如果你不知道播客是什么，在你最喜欢的应用程序上搜索开源开发者播客，并从今天开始收听！
