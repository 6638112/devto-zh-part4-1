# 2021 年如何选择雇佣软件开发人员？

> 原文：<https://dev.to/gabriela/how-do-you-choose-to-hire-software-developers-20mo>

[![Alt Text](img/ebb0aa05ab907742460f2ff4a6afabb4.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--HPQy7LgV--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/rj19d8itxrd1tjqcjm6z.png)

我写这篇文章是因为我真的很好奇世界上的公司/人们是如何招聘新开发人员的:初级、中级、高级。

这个问题的范围是给那些不练习*精疲力尽 _no* 步骤面试的人以光明和发言权，这更多是给寻找其他开发人员合作的开发人员，他们知道在面试中增加更多的层次只会疏远人们。

每个人都选择自己想要的，我相信听到其他招聘方式也很好。没有必要为你的招聘过程辩护，更重要的是向别人解释这是什么，这样合适的人可能会来找你。

如果你想和我们分享你人性化的、友好的、没有焦虑感的招聘面试，请在下面这样做(记住你对初级、中级和高级职位的要求)。我建议你也写一些关于技术栈的文章，因为谁知道呢，你可能会发现一些开发者对跳槽很好奇。
