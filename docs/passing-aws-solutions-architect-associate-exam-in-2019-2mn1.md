# 你想成为一名解决方案架构师，是吗？

不久前，我的经理给了我一个目标，让我在第一次尝试时通过 T2 AWS 解决方案架构师助理考试。我们的公司， [OLX 集团](https://www.olxgroup.com/)正在大量使用 AWS，既然公司在 AWS 上下赌注，它的工程师也应该在 AWS 上下赌注。

一点背景:我是一名 JavaScript/React 开发人员，有 7 年的前端经验。这意味着我不是网络、数据库、安全等方面的专家。(我仍然不是，但几个月前更是如此)。

根据 AWS 网站，该考试旨在:

> AWS 认证解决方案架构师助理考试面向的是担任解决方案架构师角色并拥有一年或一年以上在 AWS 上设计可用、经济高效、容错和可扩展的分布式系统的实践经验的个人。

这并不是说:“你已经做了两年的 Backbone.js，这对你来说是轻而易举的事情”。

剧透:不是的。

# 学习路径

我们的项目基于无服务器堆栈(包括但不限于:DynamoDB、无服务器框架、Lambda、S3、SQS、CloudFormation)，所以我有一个*位*的开端。一般来说，你对 AWS 的实际经验越多越好。

谈到通过实际考试，我已经从一门优秀的 AWS 认证解决方案架构师助理 2019 课程开始，这门课程是关于云专家的[。大约有 12 个小时的内容，视频制作得非常好，](https://acloudguru.com/browse-training)[瑞安](https://twitter.com/KroonenburgRyan)是一个非常棒的老师。

学完这门课程后，我想我已经为考试做好了准备。不幸的是(至少对我来说)，浏览视频是不够的。我的第一次模拟考试很尴尬，我显然需要学习更多:“他妈的是一种运动，那是一种植物吗？”

为了了解更多信息，我参加了 [Linux Academy](https://linuxacademy.com/course/aws-certified-solutions-architect-2019-associate-level/) 的 AWS 认证解决方案架构师助理级别课程。它比一部[云大师](https://acloudguru.com/browse-training)的**长得多**(超过 40 小时的视频)，我强烈建议以 1.5 倍的速度观看视频，尤其是如果这不是你第一次使用 AWS 的话。Linux Academy 也有很棒的实验室，你可以在那里实际构建和配置 AWS 中的东西(例如——你自己的 VPC 和负载平衡器)。

**编辑:**与此同时，Linux 学院和云大师联手，详情请查看此网址:【https://acloudguru.com/browse-training】T2

好的，两门课之后，我 100%准备好考试了，对吗？**没有**。

我无法在模拟考试中取得超过 69%的好成绩。在棘手的 VPC 和我的问题上失去了太多的分数。要是有一个免费视频网站，让我可以看着了解更多就好了...

我已经开始在 YouTube 上观看 **Re:Invent 2018** 视频，这对**帮助很大**。请务必观看这个精彩的[云演讲](https://www.youtube.com/watch?v=2DF_EXmxbLM)中的保护您的虚拟数据中心，以及[在 60 分钟或更短时间内成为 IAM 策略大师](https://www.youtube.com/watch?v=YQsK4MtsELU)。没有那些视频我是不会通过考试的。

很多人告诉我去阅读 AWS 白皮书和常见问题(包括著名的架构良好的框架一)。这是一个很好的建议，但我个人却无法在不睡着的情况下浏览 S3 常见问题。相反，我浏览了[教程 Dojo 备忘单](https://tutorialsdojo.com/aws-cheat-sheets/)，它们是一样的，但是更短。

# 通过考试

如果你只听我的一条建议，听听这条:

购买 Udemy 上设置的这个考试，并尝试通过所有考试。我花了 50 兹罗提(大约 12 美元，可能比你这周在星巴克花的还少)。

我再怎么强调这一点也不为过，不仅这些考题与真题处于同一水平，而且如果你在某个问题上选错了答案(你会的)，会有一个非常详细的解释，说明哪个是正确答案以及**为什么**。

这并不容易，可能需要一整个周末，但这 100%是值得的。

# 好了，我怎么选择正确答案？总是λ吗？

嗯，算是吧。

考试的时候肯定有一些模式和东西是你可以期待的。我将列出一些我记得的事情，但是出于对 jQuery 的热爱，不要在没有阅读实际问题的情况下选择 Lambda 作为答案，因为我告诉过你。

*   看完整个问题。两次。然后再一次。
*   您应该知道如何从 EC2 访问 DynamoDB(提示:通过电子邮件向某人发送 DB 密钥不是一个好主意)
*   (几乎)从来不是第三方。如果您可以使用 AWS 服务或第三方产品做一些事情，答案不太可能是第三方。出于某种原因，除了 Splunk
*   确保记住各种 EBS 存储类型。你会得到几个问题，免费分！
*   如果能把 API Gateway，DynamoDB，Lambda 连接起来，就是正确答案。这三样东西就是 AWS lovechild，贝佐斯(以及创建测试的人)似乎真的为它们感到骄傲。
*   AWS 将考试作为一种营销工具，因此“X 在 AWS 中是不可能的”不太可能是正确答案。
*   我个人对将内部数据中心连接到 AWS 没有什么疑问，这可能是另一种营销策略。*当你可以把所有东西都转移到**云**T3】时，你为什么要把数据保存在肮脏的内部*
*   一家公司希望将他们现有的 MQ 消息系统迁移到 AWS——答案是 Amazon MQ，而不是 SNS。SNS 与 RabbitMQ 之类的东西并不真正兼容，将这部分内容迁移到 AWS 需要额外的开发。
*   数据库约占考试的 30%,确保理解它们之间的区别。这真的很奇怪，因为我肯定期待更多的网络内容，而不是三个关于极光的问题和五个关于红移的问题。
*   了解预留、预定预留、按需预留等之间的区别。EC2 实例。同样，这个**将**放在考试中，免分
*   如果一个问题提到的事情花了不到 15 分钟，它可能是λ
*   请务必理解 SQS 长期投票和短期投票的区别
*   理解网络 ACL 和安全组之间的区别。考虑在你的前臂上纹一个“SG 有状态，NACLs 无状态”的纹身
*   了解不同的 S3 储物选项。如果你需要存储数据直到半条命 3 发布，使用冰川
*   出于某种原因，你不能使用亚马逊控制台直接上传数据到 Glacier
*   如果你可以复制粘贴一些东西或者使用云形成，它总是云形成
