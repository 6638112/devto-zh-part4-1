# 如何不断提高编程技能

> 原文：<https://dev.to/spukas/how-to-keep-improving-programming-skills-8k3>

Web 开发是一个非常动态的工作领域。你应该通过提高你的编程技能来跟上并感受它的脉搏。如果这不是跟上聪明同龄人的理由，那就为了更好的自己。

## 有什么好犹豫的

对我来说，是缺乏动力和懒惰。当我没有一个学习/训练计划，指导我，不让我思考下一步该做什么时，就会发生这种情况。而且，在一段紧张的时期后，无论是短跑、特写镜头还是仅仅是缺乏睡眠，在工作中筋疲力尽。为了重新站起来，欺骗自己想要学习更多，我收集了一份资源清单，这些资源在过去对我很有帮助，也许能帮助你找到学习的动力。

## 资源

#### 挑战与困惑

挑战，谜题，算法。我的最爱，爱死了！总比学习拖延症强。花一点时间，让你的思维变得敏锐，尤其是在早上。我玩的平台有两个，可以推荐:

*   [代码大战](https://www.codewars.com/)
*   [代码信号](https://app.codesignal.com/)

任何编程水平的套件，每个人都将面临挑战。记住排序算法、数据类型和数学的好方法。

#### 写着

在床边放一本书也很有效，尤其是如果你养成了睡前阅读的习惯。注意不要选择太技术性的材料；这将很难阅读，但更容易入睡。挑些轻松的来读，比如《发展中的软技能》。最近，我读了约翰·桑梅兹的两本书:

*   完整的软件开发人员职业指南
*   软技能

书好读；作者的语气既吸引人又有趣——提供了大量有意义的建议。

如果书籍太多或者当时不想看，可以通过每周订阅来了解科技新闻:

*   [节点周](https://nodeweekly.com/)
*   [JavaScript 周刊](https://javascriptweekly.com/)
*   [CSS 周刊](https://css-weekly.com/)

#### 框架

选择新的/流行的框架来学习。做教程或者在线课程。你会看到新的编程模式，项目结构，从不同的角度看问题。
比如不久前我挑了 Gatsby 框架，学习了如何用 GraphQL 生成从 MD 文件中读取信息的动态网页。现在[我的博客](https://spukas.lt)就是基于它。
首发名单可能是这样的:

*   反应
*   Next.js
*   盖茨比（姓）
*   某视频剪辑软件
*   苗条的

#### 个人项目

建立一个个人项目，尤其是当你厌倦了教程的时候。你不一定要完成它或者长期坚持。仅仅通过开始和创造，你将不得不独立思考(与教程相比)，跳出框框，遇到从未有过的问题，并有机会创造性地解决它们。让我们集思广益，看看会是什么:

*   去健身房？-锻炼记录器
*   写博客？-包含您的作品集和博客的网页
*   喜欢看电视剧？-当您最喜爱的系列节目播出时，微服务会通知您。

你明白了。

#### 聚会

寻找并参与当地的聚会、研讨会或会议。这是了解新技术话题、不同观点、与志同道合的人交谈和交朋友的绝佳机会。
首先在 [Meetup](//www.meetup.com) 网站搜索您身边的活动。

#### 个人发展计划

最近，我们的团队领导介绍了个人发展计划。这个计划包括你落后的技能，你需要更多的关注。它针对你的日常职责和产品的技术堆栈。因此，这不仅能提高你的技能，还能在工作中检验它们，为公司带来更多价值。

尝试向团队领导或资深同事寻求帮助来制定您的发展计划。问你哪里落后，需要迎头赶上？你能学到什么，做什么来增加公司价值？

## 总结起来

学习和提高你的开发技能不应该是复杂或乏味的。计划，思考可能的选择，然后去做。而最重要的是找到什么是好玩的，让你兴奋的。
