# Python 学习资源的巨大列表

> 原文：<https://dev.to/petercour/giant-list-of-python-learning-resources-2a27>

你几乎可以用 Python 制作任何东西:网络应用、桌面软件、脚本、机器学习程序等等。作为一门通用语言，它太棒了！

要不要编 Python？你可能会对外面的所有东西感到不知所措:博客、书籍、课程、课堂、小抄等等。

你应该从哪里开始？下面是我最喜欢的资源列表。

*   [全栈 Python Flask 资源](https://www.fullstackpython.com/flask.html)
*   [使用 Python 的 Flask Web App(初学者教程)](https://pythonspot.com/flask-web-app-with-python/)
*   [RealPython 烧瓶教程](https://realpython.com/tutorials/flask/)
*   [Python Flask Web Apps 课程](https://www.udemy.com/python-flask-make-web-apps-with-python/)
*   [用烧瓶静置原料药](https://www.udemy.com/course/rest-api-flask-and-python/)
*   [姜戈训练营](https://www.udemy.com/python-and-django-full-stack-web-developer-bootcamp/)
