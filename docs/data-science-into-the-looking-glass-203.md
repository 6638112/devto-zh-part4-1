# 发展中的数据

> 原文：<https://dev.to/ztimmons/data-science-into-the-looking-glass-203>

拉瓜迪亚机场的 C 门，早上只有一家咖啡店。两万人将涌入 Au Bon Pain，为获得一杯每两小时煮一次的三美元咖啡的权利而大打出手。

如果你到得足够早，你可以得到一把内置充电器的椅子，并在主要等候区占有你的权利。我知道 C 门的哪个部分有坏的充电器，我知道去哪里远离噪音，同时仍然让我的手机充满电。这不是收集你的想法的最好地方，但是那天早上我觉得我有很多事情要思考。

我的工作是一名旅行软件培训师。与我们这个时代的许多行业一样，就数据如何影响决策过程而言，商业房地产正在经历一场革命。我负责与客户合作，从年轻的大学生到几十年的老手，向他们解释我们软件背后的工具和概念，这些工具和概念为房地产估价行业设定了标准。

对我们的客户来说，学习软件意味着他们拥有了开发影响他们底线的预测所需的工具。对我来说，这份工作给我上了一堂真实的课，让我了解了组织良好的数据在塑造决策过程中的力量，尤其是当涉及数百万或数千万美元时。

大数据的兴起象征着我们经济模式的转变。它很大程度上是不可见的，难以组织，经常存储在“云中”，并且它正迅速成为美国各地董事会的中心舞台，成为尚未开发的繁荣之钥。同样，随着新兴技术的发展，收集数据的能力也呈指数级增长。数据分析的力量已经在支付我的账单，我甚至不知道它是如何工作的。我如何利用知识和对数据的掌握来追求我的兴趣？

我知道我需要做更多的研究。

我有很多问题要问自己:
谁是最让我兴奋的专业人士？他们做了什么让他们如此有趣？那些回答别人懒得问的问题的人有什么令人振奋的？人们是如何利用技术壮大自己的关系网，解决看似无法解决的问题的？

我从几个人那里听说过熨斗学校，也知道休斯顿市长创新办公室的建议。我觉得将数据科学作为一个领域将涉及以新的方式挑战自己，从理解编程语法中涉及的关系，到增长我在高级统计方法方面的知识，到开发一个识别 Python 和其他数据科学语言试图解决的业务问题的 ear。一想到要开始一场转型运动，从一个严格的前端、面对客户的角色转变为一个我目前的技能可以补充但不构成我的工作主体的角色，我就很兴奋。

很少有人能准确判断前方的路。然而，在数据科学领域，你可以确定一些事情。

你可以确信你明天不会面临和今天一样的问题。

可以肯定的是，不仅对您的组织，而且对您的客户来说，都有大量的机会来改善和优化体验。

你可以肯定的是，有时你将不得不依靠其他人以前没有尝试过的方法来解决业务问题，无论这些业务问题是新的还是一直存在的。

你可以肯定的是，每一个行业和每一次经历都将提供额外的智慧，这些智慧将发展成为创造越来越有市场的工具的能力。

我在熨斗学校经历了几个星期，那里有免费的拿铁咖啡和休斯顿市中心的风景，那里的老师不仅渴望解释代码，还渴望解释这个不断增长的领域的商业方面，这更容易为我在这里的努力提出真正的价值主张。我练习的代码越多，我就越能看到创造力和基本效率的结合，我希望这标志着提供 21 世纪价值的未来的开始。

我希望你能从我的博客中获得和我写博客时一样多的乐趣和价值。

扎克
