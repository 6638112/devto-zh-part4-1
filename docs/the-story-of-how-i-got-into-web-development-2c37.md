# 我如何进入网络开发的故事

> 原文：<https://dev.to/themarcba/the-story-of-how-i-got-into-web-development-2c37>

我不确定是否有人对此感兴趣，但不管怎样，我们开始吧。

# 高校网站开发

我在奥地利福拉尔贝格州的一所小型应用科学大学获得了信息服务/嵌入式系统的学士和硕士学位。在学习期间，我们主要接受 C/c++/Java/汇编程序开发的培训。这很好，因为我学到了很多对我以后有益的东西。但是我们仅仅用基于 Java 的工具(JSP，JSF 等)触及了 web 开发的皮毛。).

然而，我一直对分布式系统感兴趣，并希望创建人们能够轻松使用的东西(只需访问一个网站)。足够愚蠢的是，我没有付出额外的努力去自学网络技术。

# 我无法拒绝的提议

2011 年，我毕业几周后，通过我的关系，我得到了一份有趣的工作——“嘿，马克，你为什么不到墨西哥的这所大学工作呢？我们计划在这所大学建立一个学生管理系统。我们需要训练有素的人来做这件事”。

我花了几天时间考虑这个提议。我会交出使用什么技术。“我**应该**对此做好充分准备，毕竟我拿的是这个硕士学位。”，我想。

由于我在卢森堡没有任何其他计划，我接受了邀请，并开始向墨西哥当局办理移民手续。

在做了三个月的临时移动开发员以节省一些现金并等到我的工作签证被批准后，我前往东南部韦拉克鲁斯州的 Coatzacoalcos 镇。

# 墨西哥历险记

啊，是的，墨西哥。炎热的太阳，美味的食物，非常好的人。墨西哥的一切都不同于我在欧洲所知道的 T2。人们对待彼此的方式，做事的方式，人们说话的方式。见鬼，就连厕所的工作方式也不一样。

所以头几个月，我“适应”了我的新环境。事实证明，我在大学里学的两门西班牙语课程不足以达到专业水平，所以我“在街上”学习西班牙语，熟悉大学的基础设施、利益相关者和未来的用户。掌握了墨西哥的习俗以及如何“融入当地”(欧洲人在墨西哥东南部很难做到)。

我也试图让自己接受这个新的任务，这个任务在当时对我来说几乎是不可能的。

# 第一个 web 应用

软件系统的要求如下:

> 我们想在学生相应的学期中用数字方式记录他们的名字。然后，教授会记录学生的成绩，学生会对他们的老师进行评估。我们的付款部门应该能够登记每个学生的付款，如果有些付款逾期，他们应该会收到某种通知。它还应该连接到银行的系统，使其自动登记付款。

从那时起，我就非常清楚:我们需要构建一个分布式应用程序。我们将构建一个 web 应用程序！

所以我开始探索 web 开发是如何完成的，当时 Ruby on Rails 和 Django 似乎非常流行。我找到了一个帮助我构建 Twitter 应用程序的 Rails 教程。伙计，太迷人了。

一切只是“有点工作”,我不知道为什么。Rails 背后的魔力。我花了两个星期来编程，当然是在教程的帮助下。事实证明，这让我有足够的信心开始着手处理手头的任务——开发大学的软件。

# 巨石

我不太记得我编程的顺序，但是我很确定我是从使用[device](https://github.com/plataformatec/devise)的用户管理系统开始的。然后，我写了学生注册。

我的西班牙语开始变好，所以我开始雇人帮我开发软件。他们大多是受资助的学生，需要在大学的某个部门帮忙“支付”他们的奖学金。

几个月之内，整个事情变得如此复杂，以至于我无法相信仅仅在不久之前，我还在和书战斗，只是打书上说的任何东西。通过编程，我理解了 Rails 更深层的工作原理。

> 我认为对初学者来说有一个有趣的收获 -获得一些基础知识，然后开始编码。为了开始，你不需要知道所有的事情。

# 开始生产

在第一个可用版本准备好(在本地运行)之后，我们将它部署到 AWS 服务器上。说起来容易做起来难。

我们建立了一个 Ubuntu 服务器，并安装了运行我们的应用程序所需的所有东西。我不太记得确切的问题是什么，但我们遇到了相当大的麻烦，让它在 prod 环境中正常工作。但是我记得我们在开发中只使用 SQLite 是一个很大的问题。

> **也许这里还有另一个要点** -尽快投产。因为我向你保证，你等得越久，问题就会越多。

当它最终在生产中顺利运行时，发现大学的互联网太慢，无法与平台很好地配合。所以我们不得不对它进行调整，使用更多的 AJAX 和更少的站点重载。这是我们最大的建筑重新设计之一。

# 总结

我可以继续说下去，但可能会很无趣。简而言之:我从零开始，通过开始写代码，与一个团队一起构建成熟的、高效的(被 1000 多个用户使用)web 应用程序。

也许这并不是什么值得骄傲的事情，但这在当时是必要的，这让我对 web 开发产生了极大的兴趣。

> **重要的是**当你边走边学时，你需要不断地重构你的代码，以反映你在这个过程中学到的最佳实践。
