# 什么时候开始使用 Python 是个好选择？

> 原文:[https://dev . to/ctg 123/when-is-python-a-good-choice-to-start-4j 14](https://dev.to/ctg123/when-is-python-a-good-choice-to-start-4j14)

你是编程新手吗？对从哪种语言开始比较好有疑问吗？

这实际上更像是个人偏好，取决于你想要实现的项目和解决方案。然而，对于初学者来说，你有很大的灵活性去试验哪一个适合你。Python 是我喜欢使用的一种编程语言，其他人也发现它很有用。我推荐从 Python 3 开始，因为它有更多的社区支持，可以在未来几年开发和使用。

全世界可能有数百万开发人员使用 Python 来构建项目和工具，并使用它来教授其他人编程的基础知识。你可以在众多领域中做各种其他令人惊奇的工作。

#### [](#helpful-online-resources-amp-books)有用的在线资源&书籍
