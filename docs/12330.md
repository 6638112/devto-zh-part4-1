# 简介

[![pentest](img/6cf0f98acf1e62637128591deddb4c1c.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--se753BGB--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/figt9quvhqcrnpnkmg0j.png)

渗透测试(又称 pen 测试)是对计算机系统及其物理基础设施发起**授权**和模拟攻击以暴露潜在安全弱点和漏洞的实践。

渗透测试旨在回答以下问题:

> 我当前的安全措施对熟练攻击者的有效性如何？

这里有一些进行钢笔测试练习的最佳工具。我们可以在这里免费找到大部分列出的工具，而其他的将是入门免费版的主要功能，需要支付许可费才能使用所有功能。

## 1。[卡莉 linux](https://www.kali.org/)

Kali Linux 是基于 Debian 的开源发行版，专注于提供渗透测试和安全审计工具。

这篇文章中提到的大多数工具在 Kali 中都有。它包括许多工具，用于信息收集、漏洞分析、无线攻击、web 应用程序、开发工具、压力测试、嗅探和欺骗、密码破解等等。

[![kali](img/81c1ec93fa7028c364e896905eaa033a.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--BNEkItRI--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ut05b3idtt60h1l2t8fo.PNG)

## 2。 [Wireshark](https://www.wireshark.org/)

这是一个网络监控工具。Wireshark 收集并显示所有网络流量的信息，并详细提供 IP 地址、协议、请求、数据包等信息。

## 3。[sqlcmap](http://sqlmap.org/)中的一个

Sqlmap 是自动化 SQL 注入最强大的工具之一，它完全支持许多数据库，如 Mysql、SQL Server、Oracle 等。

在本教程中，我一步一步地解释了如何使用这个工具。

## 4。[打嗝组曲](https://portswigger.net/burp)

Burpsuite 可用于拦截 web 浏览器和 web 服务器(如代理)之间的流量。与 Sqlmap 配合很好，您可以将请求导出到 Sqlmap，例如，轻松利用 HTTP POST 请求的 SQL 注入。

然而，它不仅仅是一个代理。Burpsuite 可以用作 web 应用程序安全扫描器、对 web 应用程序执行自动攻击的工具、对整个网站进行蜘蛛搜索以识别攻击面的工具以及其他功能。

## 5。 [Metasploit 社区](https://github.com/rapid7/metasploit-framework)

它是与 [Rapid7](https://metasploit.com/) 共同开发的开源工具。它是一个非常流行的各种渗透工具的集合，包括发现漏洞、管理安全评估和制定防御方法。

## 6。 [Nmap](https://nmap.org/)

它是一种扫描系统或网络漏洞的工具。它可用于安全扫描，简单地识别主机正在运行的服务、主机正在使用的防火墙类型，或者快速清点本地网络。

## 7。 [Aircrack-ng](https://www.aircrack-ng.org/)

Aircrack-ng 是一个全面的实用程序集合，用于使用各种监控、攻击、测试和破解方法来分析 WiFi 网络中的弱点。

此外，如果你想评估你的 WEP 和 WPA-PSK 密钥的可靠性，你可以使用这个工具破解它们。

## 8。[纳苏斯](https://www.tenable.com/products/nessus/)

该工具是一个漏洞扫描器，允许笔测试人员通过扫描互联网协议(IP)地址的范围并使用一系列插件识别漏洞来审计他们的网络。它发现的一些漏洞包括配置错误、不正确的密码和开放端口。

## 9。 [Openvas](http://www.openvas.org/)

OpenVAS 是 Nessus 的一个分支，但是它的提要是完全免费的，并且是 GPL 许可的。该工具允许您编写自己的安全插件并将其集成到 OpenVAS 平台，即使当前引擎附带了超过 50.000 个网络漏洞测试，可以扫描许多不可想象的场景。

## 10。[开膛手约翰](https://www.openwall.com/john/)

它是最流行的密码破解工具之一，结合了几个不同的破解程序，可以在暴力和字典攻击模式下运行。它可以针对每个操作系统上的各种用户帐户运行各种各样的密码破解技术，并且可以编写脚本在本地或远程运行。

## 结论

渗透测试对于任何重视安全性的公司或组织来说都是至关重要的。如果一个渗透测试者设法危害你的应用程序或网络，那么一个真正的黑客也可以。
