# 辅助项目想法#7 -完成任务-导师聊天应用程序

> 原文:[https://dev . to/bin console/side-project-ideas-7-get-shit-done-mentor-chat-app-3lb 0](https://dev.to/binconsole/side-project-ideas-7-get-shit-done-mentor-chat-app-3lb0)

这是我昨天想到的另一个主意。建立一个每周清单，当你没有完成这个清单时设定一个价格。添加导师或好友，并在应用程序中与他聊天。你是他的导师，他也是你的导师。在导师模式和工作模式之间切换。

在一周结束时，导师会评估你是否完成了这些事情，并会提供证明聊天作为帮助。

当你通过时，什么也不会发生，当你失败时，你将被收取你之前设置的金额。
