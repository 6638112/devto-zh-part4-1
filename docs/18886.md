# 另一个最简单的无服务器功能🤷‍♀️ ‍—却严肃地说💖

> 原文：<https://dev.to/upkarlidder/another-simplest-serverless-function-but-seriously-4c7b>

我看到了一些标题为 ***最简单的无服务器功能*** 的帖子，我最初的想法是…

1.  如果你必须写一个配置文件来发布一个 ***hello world*** 函数，事情就没那么简单了🤦‍♂ !
2.  现在将这个功能扩展到更有用的东西有多容易？

所以我们又来了……这个例子是基于 ***Apache OpenWhisk*** ， ***一个开源的分布式无服务器计算平台*** 。然而，台式机/笔记本电脑上的无服务器系统与运行 nodejs express 应用程序或 flask 应用程序没有什么不同。 ***你错过了 OpenWhisk*** 的规模、性能和分布式本质。为了使它更简单，我们将使用 IBM Cloud Functions，一个 Apache OpenWhisk 的托管版本。
