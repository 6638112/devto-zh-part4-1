有请 Golo，JS Kongress 2020 计划委员会的首批成员之一。从 10 月 16 日开始征集论文截止，委员会将开始审查所有提交的论文，并选出最符合主题的论文。首先，我们问了他几个问题:

**你为什么期待#jskongress？**

JS Kongress 是一个复杂的社区会议，近年来令人印象深刻地证明了当志同道合的人联合起来，共同创造美好的事物时会发生什么。在一个拥有共同内在动机和激情的团队中开发一些东西——还有什么比这更好的呢？

你如何看待或联想到 2020 年的格言“扩展 JS——挑战极限:大规模平台、数据驱动架构和现代 API”？

JavaScript 从早年兴起。我们用它来构建大型复杂系统，从而迎接新的挑战。此外，用户的期望也发生了变化:应用程序必须始终可用，快速反应，与其他应用程序集成，等等…要应对这些挑战，您不仅需要正确的技术，还需要关于开发有效解决方案的适当概念和方法的扎实基础知识。我认为这种尖端技术和良好基础的结合非常令人兴奋。

你对这个项目的愿景是什么？

我希望看到一个多元化和平衡的议程，涵盖许多主题，但所有的贡献都有一个共同点，那就是人们可以以一种有趣的方式学习很多新的东西。这就像一次技术性的深度潜水，仍然很有趣，而且非常注重细节——我喜欢这样！

你还记得你第一次接触 JS 的时候吗？原因是什么，或者是谁？

我最初来自美国。NET 世界，厌倦了不得不处理两种语言——后端的 C#和前端的 JavaScript。在尝试将 C#引入浏览器失败后，2010 年我决定在后端使用 JavaScript，并转向 Node.js。当时和今天真正给我留下深刻印象的是你可以基于 JavaScript 和 Node.js 设计和构建解决方案的轻便和快速。

你对所有演讲者和所有想提交演讲想法的人有什么激励吗？

根据我的经验，每个人都有一个激动人心的故事要讲。大多数人害怕他们的故事很无聊，因为它们是他们日常生活的一部分——然而，对其他人来说，这些故事是对外国世界的难得洞察，而且几乎总是令人兴奋和有趣的！
