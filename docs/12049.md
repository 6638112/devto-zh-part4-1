# 2019 年的无服务器框架

> 原文:[https://dev.to/austen/serverless-framework-in-2019-5265](https://dev.to/austen/serverless-framework-in-2019-5265)

[无服务器框架](https://www.serverless.com)使每个人都能以极低的开销和成本构建应用程序。

所谓低成本，我指的是每次代码运行的成本只有 0.0000002 美元。

该框架通过在新的云基础设施(如 AWS Lambda)上构建应用程序来实现这一点，这些应用程序可以自动扩展，并且只在运行时收费。也就是说，当你的应用程序不被使用时，它不会花你一分钱。

无服务器框架已经成为构建无服务器应用程序的主要工具。今年，我们将其扩展为不仅仅是一个开发工具。

该框架现在包括监测，测试和安全功能，以帮助您操作您的无服务器应用程序，是免费的！
