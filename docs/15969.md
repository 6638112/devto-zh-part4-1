## 如何用 React 钩子和上下文 API 替换 Redux

> “在 React 中处理共享应用程序状态最流行的方式是使用 Redux 之类的框架。最近，React 团队引入了几个新特性，包括 React 钩子和上下文 API。这两个特性有效地消除了大型 React 项目开发人员一直面临的许多挑战。最大的问题之一是嵌套组件中常见的“支柱钻孔”。解决方案是使用像 Redux 这样的状态管理库。不幸的是，这是以编写样板代码为代价的——但是现在，可以用 React 钩子和上下文 API 替换 Redux。
> 
> 在本文中，您将学习一种在 React 项目中处理状态的新方法，而无需编写过多的代码或安装一堆库 Redux 就是这种情况。React hooks 允许您在函数组件内部使用本地状态，而上下文 API 允许您与其他组件共享状态。"

➤阅读了[如何用 React 钩子替换 Redux 和上下文 API](https://www.sitepoint.com/replace-redux-react-hooks-context-api/) 。

## 限制你成长的 7 个最糟糕的 UX 错误

> 对于那些“有所作为”的企业来说，增长往往是重中之重。“他们已经找到了让顾客心动的东西，他们的特殊配方，现在他们已经准备好向全世界展示了。
> 
> 然而，扩展不仅仅是扩展成功。
> 
> 如果 UX 中有摩擦、缺陷、技术限制或任何其他类型的 UX 缺陷，这些缺陷会随着产品规模的扩大而放大，这就是为什么最成功的企业是那些从容不迫、尽量不要增长过快的企业。"
