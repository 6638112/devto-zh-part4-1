# Clippy 复兴

> 原文:[https://dev.to/cgreinhold/clippy-revival-bmo](https://dev.to/cgreinhold/clippy-revival-bmo)

对许多人来说，Office 的 Clippy 是聊天机器人出现之前的第一个虚拟助手。许多人仍然错过它，它是随机提示。

Clippy.js 是一个旨在复兴 Clippy 的项目，目的是“让任何网站都能立即怀旧”。

基于这个 JS 库，并使用每个人都喜欢讨厌的 javascript 框架 Electron，我创建了[桌面 Clippy](https://github.com/CGReinhold/desktopclippy) ，这是一个 Clippy 改编程序，每 10 分钟就会给你一个随机报价。

我敢肯定，对于电子应用来说，这不是最好的主意，但对于一个有趣的周末项目和尝试不同的技术来说，这很合适。
