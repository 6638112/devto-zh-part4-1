# [](#about-func)关于 func

最近，我创建了一个现代化的命令行框架，其理念是更容易的设计和开发人员体验，我称之为`func`。

您可以使用 func 轻松创建命令行工具。[在 GitHub 了解更多信息](https://github.com/WittBulter/func)。

它比以前的框架有更多的优势:

*   极小包装的体积
*   优雅的语法和项目结构，没有任何思想负担
*   非常少的依赖性，最大的优化运行速度
*   有了完整模板的支持，一键就能开发
