2.  通过带有浏览器数据 AJAX 从页面接收帖子
3.  提取内容和属性
4.  构建 HTTP 数据的 JSON 文件
5.  存储它以接受手机或现代浏览器中的位置，您必须运行 SSL。所以我为这个 HTTP 请求添加了。

使用 openssl 创建您的 2048 RSA X509、PKCS12、JKS 密钥库、导入信任存储并在浏览器中导入
