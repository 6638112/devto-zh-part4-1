# 记得 Rust 和 Anki 吗

> 原文：<https://dev.to/jonathans/remember-rust-with-anki-58j8>

我以前读过一次《T1》这本书，但那是很久以前的事了，在某个时候，我也读过一本奥莱利写的关于铁锈的书。在读那些书的时候。感觉这种语言很有意义。我肯定会记得我读过的东西。

几个月过去了，我觉得我想在 Rust 做点什么，深入一些。所以我坐在电脑前，茫然地看着。rust 中再次进行 for 循环的确切语法是什么？更不用说如何创建一个特征受限的泛型结构了。

听起来相似吗？嗯，我决定对此做点什么，所以我创建了一个有 212 张卡片的 Anki 卡片组(更新于 2019-06-21)。安奇甲板覆盖铁锈书。如果你没读过这本书，这本书就没什么用了。但我要说，一旦你读过这本书，它肯定会帮助你回忆起来。

如果你从未听说过 Anki 简而言之，这是一个使用[间隔重复](https://en.wikipedia.org/wiki/Spaced_repetition)来帮助你记忆的闪存卡程序。因此，即使我在将来的某个时候远离 rust，我也会记住语法，现在你也有了一副牌，你也会的！
