本周 Glitch 是一个常规系列，在这个系列中，我们总结了一些了不起的创作者在 Glitch 上完成的令人惊叹的项目。以下是本周吸引我们眼球的几个项目。

## [](#mix-up-synthier-things)[把合成器的东西混在一起🎛️](https://glitch.com/~synthier-things)

随着最近发布的《陌生事物》第三部，你可能会受到启发，尝试一下埃里克·伍德的[synthier-Things](https://glitch.com/~synthier-things)——一个只能播放《陌生事物》主题的网络合成器。synth 由一些现成的网络音频组件组成。您可以使用自动琶音器、振荡器、滤波器和衰减器来随心所欲地扭曲那些标志性的 80 年代灵感的电子琶音。

[https://glitch.com/embed/#!/embed/synthier-things?previewSize=100&path=index.html](https://glitch.com/embed/#!/embed/synthier-things?previewSize=100&path=index.html)

## [](#pick-your-fave-queen)[挑选你最喜欢的皇后💅🏽](https://glitch.com/~favorite-queen)

如果你喜欢基思·库尔森，你是鲁保罗飙车赛的粉丝，那么你可能会喜欢“[最受欢迎的皇后](https://glitch.com/~favorite-queen)从 RPDR 的每个季节中选择你最喜欢的女王，并在社交网站上分享你的选择，看看你的朋友是否同意。确实是选择。

[https://glitch.com/embed/#!/embed/favorite-queen?previewSize=100&path=index.html](https://glitch.com/embed/#!/embed/favorite-queen?previewSize=100&path=index.html)

## [](#doodle-in-the-depths)[深处的嘟嘟🐙](https://glitch.com/~shadowy-depths)

罗文·梅莱伍德的“[阴影深处](https://glitch.com/~shadowy-depths)”是一个 CSS 涂鸦，有着触手的一天。如果你曾经花时间使用过影子 DOM，我想你会有同感。
