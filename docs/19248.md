# 是改变的时候了

> 原文:[https://dev . to/nfairbairn/looking-for-a-change-and-finding-a-way-make-it-occurrence-4h5o](https://dev.to/nfairbairn/looking-for-a-change-and-finding-a-way-to-make-it-happen-4h5o)

意识到我想要一些不同的事情发生...

我想对我来说，那是我觉得卡住的时候。当我意识到我不喜欢我的工作，但不知道我还能做什么的时候。我在大学里进进出出了几个学期，没有什么是坚持。我喜欢学习，但我不能忍受整天听课。

[![sleep](../Images/7437bdd24eace1e2032a11ca85fd7ef4.png)T2](https://i.giphy.com/media/l378AEZceMwWboAQE/giphy.gif)

然后呢…？

如果我不去上学，那我肯定要工作。在每个职位描述都要求“经验”和无法通过工作获得经验之间的长期矛盾中，我在一家汽车经销商那里找到了一份销售零件的工作。刚开始的几个月真的很开心！我开始学习整个零件编号系统，一个用于维修订单和发票、目录和满屋子待发现零件的计算机系统。

[![experience meme](../Images/d9e718bd4caa0f40e7969b12f6ae36cc.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--0zW4Ytnm--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/yhtcepb3447a1a5bt8pg.png)

**残酷的现实...**

这样过了一年，如果我不得不再次在早上 7 点醒来，然后整天储存零件，我就处于崩溃的边缘。我已经学到了很多东西，并且沉迷于问问题，但是当我达到目前工作描述的顶峰后，我留下了更多的渴望。当我终于攒够钱为自己争取时间时，我辞掉了工作，去一家餐馆做全职服务。

[![sherlock](../Images/c39f8665a40dfd7b2f2025c9bf5897d0.png)T2】](https://i.giphy.com/media/yoJC2HGt4h8G8zgbaU/giphy.gif)

**香香加班**

餐馆是我有幸置身其中的最有活力、最多样、最不可思议的工作环境之一。我非常喜欢每天去上班。我喜欢会见顾客，了解食物、葡萄酒和鸡尾酒，并成为真正相互依赖的团队的一员。我可爱的同事很快提醒了我他们在工作之外有多努力。我发现大多数人都在努力完成大学学业，或者为他们的乐队存钱，或者为他们的艺术提供资金。不管是什么，我从他们身上学到了很多，这让我思考，“我在为什么工作，不仅仅是现在，而是 5 年后。?"。

[![ponder](../Images/3efbe6e85cb2e9647e917e5a430fcdbb.png)T2】](https://i.giphy.com/media/kPtv3UIPrv36cjxqLs/giphy.gif)

或许让我们试试编程？

我爸爸和我叔叔都是开发人员，虽然我之前接触过编程以及编程存在的事实，但我从未真正研究过它。我从 HTML 和 CSS 的免费 CodeCademy 课程开始。我说“嘿，也许你可以建立一个网站，人们需要网站，你可以从你的笔记本电脑上做！”。一旦我完成了这些语言，我就被提示开始 JavaScript 课程。我很快意识到这是一门完全不同的语言，但是我继续努力去了解编程到底是怎么一回事。

[![css](../Images/269fd81b094e1de6a911a54c171312fb.png)T2】](https://i.giphy.com/media/yYSSBtDgbbRzq/giphy.gif)

他有了一个主意

一直有传言说科技行业是世界上发展最快的行业，但我没有意识到这是一个比我想象中更容易进入的行业。我没有时间、金钱或注意力来完成 4-6 年的大学学业以获得计算机科学学位。但是当我听说训练营的时候，我想“伙计，这是我的入场券。我会尽我所能，尽可能快地学习，我会在西雅图做到这一点。”。我开始寻找如何尽快进入科技行业的方法，以及有什么比打包行李横跨全国搬到西雅图参加训练营更好的方法。六个月后，我收拾好一个行李箱、一个手提行李和一个背包，搬到了西雅图，去追寻我成为一名软件工程师的疯狂想法。

[![aladdin](../Images/4e6bec6a70b8b24af454bfc3ac83669f.png)T2】](https://i.giphy.com/media/yyvSeRGVj4C64/giphy.gif)

要是我能早点意识到这一点就好了！

所有这些经历中不变的是，我永远无法动摇我的学习欲望(有时尽管我尽了最大努力)。对我来说，转行或重返校园最困难的部分是开始。我现在喜欢说，“任何过程中最难的部分都是从 0 到 1。”。我的意思是“一旦你迈出了第一步，你就有了动力。”。这是一种滚雪球效应，一旦你采取了正确的行动，并继续为同样的目标而努力，一切就开始变得明朗起来。我发现激情能做更多的事情，因为说服自己放弃激情比其他任何事情都难。

[![Steps](../Images/811f40b1601c610842ce1568c68ffbe7.png)T2】](https://i.giphy.com/media/l41JGpEtXuozyNbvq/giphy.gif)

**要旨**

我想我想说的是，做出人生的改变，却不知道另一边会发生什么，这是很可怕的。我只是想让读到这里的人知道，这是非常有可能的，而且是非常值得的。如果你能找到学习的激情，并加强追求知识的动力，无论是什么，一切皆有可能。这需要做很多工作，而且看起来很可怕，但是做你的研究，尽可能多地了解你想做什么，并充分利用你所拥有的。你能行的！从键盘后面的陌生人到你，你得到了这个！
