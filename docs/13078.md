# 为什么没有更多的开发者编辑维基百科？

> 原文：<https://dev.to/shushugah/why-don-t-more-developers-edit-wikipedia-4959>

这不是咆哮的开始。我真的很好奇为什么更多的开发者不为维基百科做贡献，无论是关于技术话题还是任何其他兴趣。对许多人来说，一个常见的障碍，即过时的维基标记不太适用于开发人员。虽然大多数开发人员**并不**为开源软件做贡献，但许多人确实会写错误报告，或推动他们的公司使用开源软件，因为在许多情况下这是标准化和可转移的知识。

维基百科上的知识没有同样的动机或必要性。

我们如何改变这种情况？附上我的[英文维基百科简介](https://en.wikipedia.org/wiki/User:Shushugah)。平我，如果你想变得更活跃！也许我们可以在这里组织一次虚拟编辑马拉松。
