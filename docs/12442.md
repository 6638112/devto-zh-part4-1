算法是语言不可知的，任何称职的程序员都应该能够将它们转换成自己选择的编程语言的代码，但同时，如果示例以自己选择的编程语言呈现，学习起来会更容易。

不幸的是，我遇到过几个真正擅长编程语言的程序员，比如 T2 Java T3 或 T4 Python T5，他们知道 API 的细节和语言的复杂性，但是对算法和数据结构的基础知识却知之甚少。

只要让他们实现任何流行的排序算法，如[快速排序](http://java67.blogspot.sg/2014/07/quicksort-algorithm-in-java-in-place-example.html)或[合并排序](http://www.java67.com/2018/03/mergesort-in-java-algorithm-example-and.html)，他们就会崩溃。如果你希望他们知道更高级和复杂的算法，如[字符串算法](https://dev.to/javinpaul/top-20-string-coding-problems-from-programming-job-interviews-493m)、[图算法](http://bit.ly/2NAbaIR)、[树遍历](https://javarevisited.blogspot.com/2016/10/post-order-binary-tree-traversal-in-java-iteration-recursion.html)或贪婪算法，准备好检查面试，否则，你可能会有一些惊喜。

有一次，我遇到了一个非常好的核心 Java 高级开发人员的候选人，他擅长 Java、[多线程](https://javarevisited.blogspot.com/2018/06/top-5-java-multithreading-and-concurrency-courses-experienced-programmers.html)，但是他的数据结构和算法技能与他的经验和能力相比实在是太差了。

我问他，为什么不来面试之前，不花时间刷一下自己的算法和解题技巧？他的借口是“那些算法只是为了面试，从来没有真正用于实际编码。在我 6 年的 Java 开发生涯中，我从未使用过它们。

他说得有些对，你不需要用 Java 实现哈希表，你总是可以用 Python 中的 [HashMap](http://www.java67.com/2017/08/top-10-java-hashmap-interview-questions.html) 类或字典来实现这个目的，但是他没有认识到在[提高编程技能](https://javarevisited.blogspot.com/2014/01/10-tips-to-improve-programming-skill-become-better-programmer.html)中更长期的改进算法和数据结构。

算法是开发编程解决技能和编码意识的工具，这是将用户需求转换成代码行(也称为计算机程序)所必需的。

他们也无处不在，如果你使用脸书，你有没有想过他们是如何如此容易地找到你的朋友？或者网飞如何推荐你想看的电影？这些只是 [**机器学习算法**](https://pluralsight.pxf.io/c/1193463/424552/7490?u=https%3A%2F%2Fwww.pluralsight.com%2Fcourses%2Fmachine-learning-algorithms) 的几个例子，它将算法带到了另一个层次。

### 面向初学者和有经验的程序员的关于 Java、Python、C 和 C++的十大数据结构和算法书籍

对于那些认为算法是数据结构的人来说，另一个黄金提示是给那些想在亚马逊、T2、谷歌、脸书、英特尔或微软工作的人的，记住这是唯一永恒的技能，当然除了 UNIX、SQL 和 C 编程语言。

> 编程语言来来去去，但编程的核心，也就是算法和数据结构是不变的。

如果你知道哈希表是如何工作的，那么你可以在任何编程语言中使用它们的实现，比如 JDK 的 [HashMap](http://www.java67.com/2013/02/10-examples-of-hashmap-in-java-programming-tutorial.html) ，T2 的【Python】Dictionary，或者 T4 的【c++】boost library 的 HashMap。

所以，如果你对编程很认真，并且现在意识到算法和数据结构不是可选的，这里有一些学习算法的好书。

> 你们中的一些人可能已经读过了，但是它们值得再读一遍。顺便说一句，如果你用这篇文章中的链接买书，我也会得到报酬。

**1。**[**Thomas h . Corman《算法导论》**](http://www.amazon.com/Introduction-Algorithms-Edition-Thomas-Cormen/dp/0262033844?tag=javamysqlanta-20)
这是最受欢迎的算法书籍之一，但要注意它包含了大量的理论。这本书的当前版本是第三版，我强烈建议每个程序员都应该在他们的书架上放上这本书，但只是为了简短阅读和参考。

一口气读完这本书是不可能的，你们中的一些人可能会觉得很难读懂，但不要担心，你可以将你的学习与在线课程结合起来，如本书中的[数据结构和算法:使用 Java 的深度探索](https://click.linksynergy.com/fs-bin/click?id=JVFxdTr9V80&subid=0&offerid=323058.1&type=10&tmpid=14538&RD_PARM1=https%3A%2F%2Fwww.udemy.com%2Fdata-structures-and-algorithms-deep-dive-using-java%2F)。

这就像两全其美，你可以在网上课程中快速学习基础算法，然后通过阅读这本书进一步巩固知识，这对于你来说更有意义，因为你已经完成了一门课程。

我推荐这本书作为第一本关于算法的书的另一个原因是因为它的语言不可知论，并附有讲座

**2。** [**罗伯特·塞奇威克的算法&凯文·韦恩**](http://www.amazon.com/Algorithms-4th-Edition-Robert-Sedgewick/dp/032157351X?tag=javamysqlanta-20)
这是我很长一段时间以来首选的算法资源，现在仍然是，但现在我看到它的次数比以前少了。你会学到很多关于算法的背景知识，现在甚至这本书的特定版本也可以用于不同的编程语言，如 [Java](https://javarevisited.blogspot.com/2018/05/top-5-java-courses-for-beginners-to-learn-online.html) 和 [C++](http://www.java67.com/2018/02/5-free-cpp-courses-to-learn-programming.html) 。

这本书还有几个免费的 Coursera 在线课程， [**算法第一部分**](https://coursera.pxf.io/c/3294490/1164545/14726?u=https%3A%2F%2Fwww.coursera.org%2Flearn%2Falgorithms-part1) 和 [**算法第二部分**](https://coursera.pxf.io/c/3294490/1164545/14726?u=https%3A%2F%2Fwww.coursera.org%2Flearn%2Falgorithms-part2) ，很好地补充了这本书。太棒了。这也是我向 Java 程序员学习算法的首要推荐。

在 Coursera 上，它也有一些最好的数据结构和算法课程，涵盖了每个重要的主题。它们被捆绑在一起成为 [**Coursera 专攻算法**](https://coursera.pxf.io/c/3294490/1164545/14726?u=https%3A%2F%2Fwww.coursera.org%2Fspecializations%2Fdata-structures-algorithms) 。如果你热衷于深入学习算法，那是我建议你去的地方。

信不信由你，但是如果你已经知道一种编程语言，那么看一个关于这种编程语言的算法的例子也可以减少学习曲线。*你也可以在网上免费阅读这本书的第四版*[*这里*](【http://algs4.cs.princeton.edu/home/】T2)

**3。** [**《算法设计手册》作者史蒂夫·s·斯基亚纳**](http://www.amazon.com/Algorithm-Design-Manual-Steven-Skiena/dp/1849967202?tag=javamysqlanta-20)
这是另一本关于计算机算法的优秀书籍，书中用大量代码讲述了大量算法。我特别喜欢这本书的地方是，他实际上给出了他在实践中使用算法(或其变体)的例子；它真的可以帮助你看到一个特定的算法(或一系列算法)可以解决的问题类别。

代码在 [C](https://www.java67.com/2020/07/5-free-courses-to-learn-c-programming.html) 中，但不是很深奥，很容易理解。我离开学校也有一段时间了，这帮助我很快掌握了一些[](http://bit.ly/2NAbaIR)**图形算法。这本书我已经买了将近 10 年了，我仍然会时不时地看看它**

 **[![](../Images/ea86f31dd92b8758a1de32519c3e6d5e.png)T2】](http://www.amazon.com/Algorithm-Design-Manual-Steven-Skiena/dp/1849967202?tag=javamysqlanta-20)

**4。** [**算法概括地说**](http://www.amazon.com/Algorithms-Nutshell-In-OReilly/dp/059651624X?tag=javamysqlanta-20)
奥赖利的算法概括地说，是一本非常好的学习编程算法的书，尤其是对于 Java 程序员来说。它描述了算法，重点是实现它们，而不是经典算法书籍中使用的复杂数学。

所有算法都以模式的形式呈现，并附有使用它们的动机、给出高级概述的图片和伪代码，以及工作代码(在 [C](https://dzone.com/articles/5-free-courses-to-learn-data-structure-and-algorit) 、 [C++](http://www.java67.com/2018/02/5-free-cpp-courses-to-learn-programming.html) 、 [Java](https://javarevisited.blogspot.com/2018/05/top-5-java-courses-for-beginners-to-learn-online.html) 和 [Ruby](http://www.java67.com/2018/02/5-free-ruby-and-rails-courses-to-learn-online.html) 中)。

他们也有基准来证明算法的理论性能。总之是程序员学习算法的最佳书籍之一。

[![](../Images/7529dddb634414da13b731dc4ba39612.png)T2】](http://www.amazon.com/Algorithms-Nutshell-In-OReilly/dp/059651624X?tag=javamysqlanta-20)

**5。** [**面试的算法**](http://www.amazon.com/Algorithms-For-Interviews-Adnan-Aziz/dp/1453792996?tag=javamysqlanta-20) \
阿德南·阿齐兹(Adnan Aziz)的《面试的算法》是一本关于算法的必读书籍，是从牢记[编程面试](https://hackernoon.com/10-data-structure-algorithms-and-programming-courses-to-crack-any-coding-interview-e1c50b30b927)的角度来写的。

封面本身表明，如果你仔细观察封面上用名人缩略图绘制的图像，这本书会多么有趣，这本书还解释了如何开发这种算法。很有趣，不是吗？

嗯，我喜欢这本书，因为它的方法和目标，有时用不同的对象学习同样的东西有助于更好地理解它。

[![](../Images/9b7af0a9a80ae1388c9ec9793d3db908.png)T2】](http://www.amazon.com/Algorithms-For-Interviews-Adnan-Aziz/dp/1453792996?tag=javamysqlanta-20)

**6。**[**Kleinberg&Tardos**](https://www.amazon.com/Algorithm-Design-Jon-Kleinberg/dp/0321295358?tag=javamysqlanta-20)\
算法设计这其实是继 Thomas Cormon 的[算法导论](http://www.amazon.com/dp/0072970545/?tag=javamysqlanta-20)之后算法方面第二好的书。它不是真正的算法入门，更适合有经验的程序员。

对于熟悉基本算法的开发者来说，更多的是算法设计。你应该从介绍罗伯特·塞奇威克的算法书或算法开始，然后继续读这本书。

顺便说一句，如果你喜欢，你也可以将你的学习与在线课程结合起来，如 [**算法和数据结构-第一部分和第二部分**](https://pluralsight.pxf.io/c/1193463/424552/7490?u=https%3A%2F%2Fwww.pluralsight.com%2Fcourses%2Fads-part1)[plural sight](https://medium.com/u/50a6c7ef7431)。在进入算法设计主题之前，熟悉基本算法和数据结构是一门很好的课程。

[![](../Images/7ff517bf4fdd0d6e70553fabc17015f5.png)T2】](https://www.amazon.com/Algorithm-Design-Jon-Kleinberg/dp/0321295358?tag=javamysqlanta-20)

7 .**。** [**《算法简介:一种创造性的方法》**](https://www.amazon.com/Introduction-Algorithms-Creative-Udi-Manber/dp/0201120372?tag=javamysqlanta-20)
《算法简介:一种创造性的方法》乌迪·曼伯写的是另一本自学的好书，因为它充满了数百个问题和例子。
