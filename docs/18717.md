# [](#my-journey-launching-jets-to-helping-businesses-take%C2%A0off)我的旅程:开飞机帮助企业起飞

经常有人问我从现役空军到前端开发人员的转变。今天我将把这个问题和另一个我经常被问到的问题联系起来，那就是为什么我们需要编程训练营，因为现在有这么多免费和便宜的教育内容。

我的旅程是在德国驻扎期间开始的。我开始修补一些 HTML 和 CSS 文件，并迅速向我的妻子展示了我能够在浏览器中显示的红框和蓝圈。至少可以说，她真的很兴奋😆。

鉴于现代网站和网络应用程序的复杂性，我知道我需要学习一门真正的编程语言来为网站添加功能。所以我用谷歌搜索了一下，以为一个明确的答案会出现在搜索结果的顶部…但是没有🤷‍♂.每个开发人员，每个知道什么是代码的人似乎都有一个非常强烈的想法如何开始。Python 对我来说听起来不错，几周之内，我就通过艰苦的努力学习了 Python，并对我的技能发展方向感觉良好。完成后，我知道了一些基本知识，做了一些很酷的游戏，但仍然有一些问题，比如…

> "我如何把这种语言和网站联系起来？"。

我有一些很酷的技能，但无法将它们结合起来真正创造出任何东西。我休息了一段时间，直到大约 6 个月后我搬回美国。作为一个播客爱好者，我偶然发现了一个关于编程的播客，并了解到这个兽医训练营，VetsWhoCode，听起来很有前途。
[![](../Images/327c721d464276048bffd7ac992fecb5.png)T3】](https://res.cloudinary.com/practicaldev/image/fetch/s--wrtdstON--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1600/1%2AYh61FLB1naD5ZnmKbIxLfg.png)

这是我真正的发展旅程开始的地方。VetsWhoCode 给了我一个过渡到新职业的路线图。他们不仅提供了高质量的指导，并指引我找到我们行业中的顶尖人员正在使用的资源，而且他们还将您带入了一个由真正工作的开发人员组成的生态系统，这些开发人员确保他们能够交流进入开发世界所需的工具和技能，而不仅仅是知道基本的编码技能。

知道如何用 JavaScript 编码或连接到数据库以存储安全数据是一回事，但专业人员还依赖于更重要的东西，如工具、构建流程以及如何有效地与经验丰富的开发人员合作，以便在公司中保持有效的工作流程。在一些新兵训练营中，这经常被忽视，作为一个老兵，这是一个非常需要的优势🔪在面试第一份工作的时候。

所以现在你可能在想，有时间做这件事，学习一项有价值的技能，而没有生活的阻碍，这一定很好，对吗？尽管我已经退役了，但我的妻子仍在服役，这意味着作为配偶的我，如果我的妻子接受了训练、部署或军方可能给我们带来的任何事情，就应该承担起几乎所有与家庭相关的事情。再加上一个令人惊讶的 4 岁儿子，作为一个正常的男孩，他每 5 分钟就会遇到麻烦。

我们最终在弗吉尼亚州找到了我们的首选基地，并兴奋地购买了我们的第一套房子。我们开车到弗吉尼亚州，找到一个家，回到佛罗里达州，兴奋地开足马力，生活是伟大的，当…我们发现我的妻子患了癌症。说这让我们止步不前，措手不及是一种保守的说法。

还在搬去弗吉尼亚吗？没人知道。

支持医疗预约等？最近的家庭在 1500 英里之外。

但是，当逆境袭击家园时，军队可以展现出它的本色和光芒，我们有来自各个分支的军官进行沟通，确保我的妻子得到最好的医疗护理，同时确保我们选择的基地为我们的未来提供保障。

优先考虑成为一名看护者，确保将我的儿子转到一所新学校，同时在同一个网络中找到我的第一份发展工作，这被证明是一份全职工作。有一个了不起的妻子，她确保我不会因为这个挑战而退缩，这证明了拥有一个支持系统并且没有放弃的选择肯定是生活中任何事情的秘密调味汁。作为一名应届毕业生，我证明了自己能够在其他训练营指导其他训练营的学员，并且用 500 多个小时磨练了我的教学技能。

为了得到一份可以帮助开发面向用户的产品的工作，我去了一家为跨国公司开发软件的公司。当我有机会回馈社会并投入个人时间指导学生学习 VetsWhoCode 时，这是一个简单的答案。毕竟，分享高质量的信息和提高彼此的技能是在技术领域保持领先并创造改变世界的公司的最佳方式🌎。
