# 每日挑战#31 -统计 IPv4 地址

> 原文:[https://dev . to/the practical dev/daily-challenge-31-count-IP v4-addresses-487j](https://dev.to/thepracticaldev/daily-challenge-31-count-ipv4-addresses-487j)

你的[挑战](https://www.codewars.com/kata/526989a41034285187000de4)是写一个函数，接受开始和结束 IPv4 地址，返回从开始到结束的 IP 地址数，不包括结束地址。所有输入都将是字符串形式的有效 IPv4 地址。

> 例子:
> `ipsBetween("10.0.0.0", "10.0.0.50") => 50`
> `ipsBetween("10.0.0.0", "10.0.1.0") => 256`
> `ipsBetween("20.0.0.10", "20.0.1.0") => 246`

如果有人想尝试为 IPv6 创建同样的功能，那将会令人印象深刻。不过，你可能需要处理一些大数字。
