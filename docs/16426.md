### UI 设计的配色方案

对于我的一个项目重新设计，我正在寻找一个在线颜色组合生成器。有许多颜色选择器、生成器等。但是似乎没有一个能够生成适合我的 UI 的颜色组合。

这就是为什么我创建了一个在线设计工具: [Colorbook](https://liyasthomas.github.io/colorbook) 来创建可以很好地搭配的颜色组合。使用[配色系统](https://liyasthomas.github.io/colorbook)创建伟大的颜色组合调色板。

[![UI](../Images/8d2a9e45df5e1e238a55fd94dfcd7f72.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--vcRYLjBq--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://raw.githubusercontent.com/liyasthomas/colorbook/master/icons/screely.png)

Colorbook 是一个开源的配色方案生成器，针对用户界面进行了优化，如字体、背景、边框等。

有了 [Colorbook](https://liyasthomas.github.io/colorbook) ，现在我可以为我的 UI 创建调色板，并测量任何颜色组合的可访问性。

挑选配色方案现在就像 1、2、3 一样简单！

1.  选择一种颜色。

2.  查看示例 UI 的预览。

3.  复制颜色代码。

[![Colors](../Images/956846e2b18c47a6cdd9a0240c017808.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--3EW-Q3sf--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://raw.githubusercontent.com/liyasthomas/colorbook/master/icons/screely2.png)
