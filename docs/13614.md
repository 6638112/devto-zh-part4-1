# 资源列表-你关注哪些开发人员的 Twitch 流和 YouTube 频道？

> 原文:[https://dev . to/adriansandu/what-twitch-streams-and-YouTube-channels-for-developers-do-you-follow-jph](https://dev.to/adriansandu/what-twitch-streams-and-youtube-channels-for-developers-do-you-follow-jph)

> 我在帖子的底部为开发人员整理了一个 YouTube 频道和 Twitch 流的列表，以省去每个人导航评论的麻烦。我想把这个变成一个资源帖，让大家都能找到自己喜欢的开发者创作者。
> 
> **最后编辑:2019-07-18** :将杰森·伦托夫加入 Twitch 名单

他们说一张图片胜过千言万语。如果是这样的话，一个视频值多少钱？

的确，开发人员更喜欢以书面或音频的形式消费信息。文章、书籍、有声读物、播客...这些信息大部分可以在做其他事情的时候消耗掉。然而有些东西只有视频才能提供。

不要脸的外挂——我目前正在 YouTube 上的 [Front End Nexus](https://www.youtube.com/channel/UCgACtqiDmnSaskDIBsK54ww) 主持一个每周新闻节目。我知道这里另外两个人也做视频:[蒂姆·埃尔米洛夫](https://dev.to/yamalight)和[布莱恩·罗宾逊](https://dev.to/brob)。

在万维网上，有许多关于珍·西蒙斯频道的精彩视频， [Layout Land](https://www.youtube.com/channel/UC7TizprGknbDalbHplROtag) ，尽管它最近很安静。明天我会翻翻我的书签和订阅，然后自己带一份清单过来。

你订阅了专门面向开发者的 YouTube 频道吗？你最喜欢的有哪些？请与我们分享。更好的是，你有这样的渠道吗？

抽动彩带呢？我知道许多开发人员正在现场编写代码。你有正在关注的人吗？
