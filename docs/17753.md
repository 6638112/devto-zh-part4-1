# 在 React Native 中使用动画动态添加/删除组件

> 原文：<https://dev.to/skptricks/dynamically-add-remove-component-with-animation-in-react-native-3n1b>

Source : [在 React Native 中用动画动态添加/删除组件](https://www.skptricks.com/2019/06/dynamically-add-remove-component-with-animation-in-scrollview-in-react-native.html)

本教程解释了如何在 react 本地应用程序中添加和删除动画组件。在这个例子中，我们将在 scrollview 中创建添加和删除功能。为了在滚动视图中使用这些功能，您必须执行以下操作:

1.  当用户点击添加按钮时，它会在滚动视图中添加新的条目。
2.  当用户点击删除按钮时，它会从滚动视图中删除特定条目。
