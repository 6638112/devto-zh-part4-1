# lassi:Android 平台的媒体选择器库。

> 原文：<https://dev.to/connectsteven/lassi-a-media-picker-library-for-android-platform-gk0>

Lassi 是选择媒体的最简单方法。要么是图像，要么是视频。

以下是使用 Lassi 的一些优势，

1.  简单实现
2.  设置您自己的自定义样式
3.  按特定媒体类型过滤
4.  按最短和最长时间过滤视频
5.  从媒体选取器启用/lass option 照相机
