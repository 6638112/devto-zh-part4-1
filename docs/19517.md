# 开发商可能如何投资？

> 原文:[https://dev.to/conw_y/how-might-developers-invest-3f40](https://dev.to/conw_y/how-might-developers-invest-3f40)

作为开发人员，或者其他类型的技术人员和数字人，我们的薪水相当不错。

有了这么多收入，我们如果不考虑为未来储蓄和投资，那就太傻了——为了我们未来的自己，也为了我们关心的人，比如配偶、孩子、亲戚、朋友。

在过去的几十年里，一系列投资者的一个重要观点是，金钱会随着时间的推移而贬值。这是由于通货膨胀和机会成本的综合作用。通货膨胀降低了我们货币的购买力，所以 5 年后 20 美元买不到今天 20 美元能买到的东西。机会成本意味着只是坐在那里无所事事的钱不会增长，而以某种富有成效的方式投资的钱在未来可能会增长。

另一个关键的见解是**你应该投资于你所知道的**。最好投资于你很了解的领域，这样你就能更好地判断哪些投资物有所值，哪些不值得。在这方面，投资类似于任何其他类型的购买。如果你想买一辆车，做一些研究是值得的，以便了解你买的是什么，并确保你得到一个好的交易，无论是短期还是长期。

将这两种观点结合在一起，我得出了以下问题:

为了投资和增加财富，开发人员/技术人员可以通过哪些方式利用他们的特定知识和技能？

在将这个问题提交给 DEV.to 社区之前(这是我非常想做的！)，我想给你简单介绍一下我用过的或者正在探索的一些想法。

*   **投资技术以节省时间/金钱**。这是最简单最容易摘到的水果。作为软件/技术人员，我们通常知道如何自动化我们生活的某些方面，并使用计算机和其他设备来节省时间和金钱。我们通常是技术的早期采用者，因此我们可以比大多数人更早实现节约。举几个例子:当 Skype 和 Google Voice 还是新技术的时候，我通过使用它们节省了海外通话费用。当它还是新事物时，我通过网上订购来节省小额采购。我开发了自己的食物卡路里/成本跟踪器，这样既省钱又减肥。这些投资继续为我省钱。

*   **投资在线课程增加我们的工资**。利用我们的技术知识，我们可以选择我们投入时间和金钱的课程，从而最终获得适合我们自己和市场的专业知识，从而获得更高的工资。

*   **投资在线财产，如域名**。通过购买一个适销对路的域名，并在此基础上建立一个在线存在(或付钱给其他人这样做)，我们可以拥有互联网的一部分，并通过我们捕获的流量产生广告或其他收入。

*   **建立一家初创企业**。不是对胆小的人，但机会确实存在，对那些有好主意、合适的网络、对时机和市场有良好感觉的人来说。如果我们生活节俭，挣得一份体面的工资，我们就能存下足够的钱，从日常工作中抽出 6-12 个月的时间，尝试创办自己的公司。它可能是一个付费软件产品或 SAAS，一家咨询公司，一个可以通过支持或付费服务货币化的开源项目，或者一个可以通过广告货币化的博客或 YouTube 频道。

*   **风险投资**。这是一个我仍然需要了解更多的领域。据我目前所知，没有多少人可以成为风投(风险资本家)，部分原因是为了获得杠杆而需要大量现金，部分原因是因为法规要求投资者必须拥有一定数量的额外现金才能从事某些类型的投资活动。然而，我渴望进一步探索这一领域，并寻找可能利用我的技术专长和判断力投资于我对其成功充满信心的公司和项目的方法。

*   **选股**。我看不到这一领域的大的短期机会，但凭借对科技市场的一些具体见解和长期的“买入并持有”策略，我想一些科技人可以从少量精选的科技股中获得可观的利润。

所以现在我想把接力棒传递给我的开发人员和技术人员。对于像你这样知识渊博的人，你会考虑哪些有趣的投资想法？这篇文章中有什么观点让你觉得是好是坏，为什么？你是否创办或投资过风险投资公司，或者你是否考虑过这样做，为什么/为什么不？

渴望得到你的想法！
