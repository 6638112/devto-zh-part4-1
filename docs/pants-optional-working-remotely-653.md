# 裤子可选:远程工作

> 原文：<https://dev.to/shellscape/pants-optional-working-remotely-653>

我已经远程工作了将近 7 年，我无法想象再一次每天走进办公室。这是一个神奇的设置，提供了难以置信的自由，以及更多的责任。我喜欢远程工作，我将和你分享我在自己的办公室工作的策略。

> 我不能在家工作，我什么也做不了。

这是我通常从任何人那里得到的关于我的设置的回应。我已经学到了几门关键的课程和技能，这些课程和技能让远程工作成为一种乐趣，以至于我怀疑自己是否会再从事一份必须向办公室汇报的工作，如果我能帮上忙的话。

## 永不言弃

人们对“我永远做不到”的反应并不奇怪。我们大多数人在家最不想做的事情就是考虑工作。事实上，许多聪明的人会告诉你这样做是不健康的。对于那些按照典型的*朝九晚五*规律工作的人，我可能会同意。但是我过着软件开发人员的生活，并且我已经找到了一种方法，这种方法使得远程成功变得非常容易。

## 工作时不工作

如果工作是工作，你就不应该在家做。我很幸运，可以做一些我相当擅长的事情，做一些我真正喜欢的事情。这真的很关键。享受工作让你可以随意开关，而不会让你的家被“办公室”污染。你将能够让想法日夜流动。你物理办公室的正式时间将成为更多的指导方针，你需要保持可用，但你不会受到约束。

## 裤子其实是可选的

如果你打算开视频会议，别忘了它们。你不想成为那种不得不起床关窗，无意中炫耀那些新的冒险内衣的人。

但说真的，放松点。选择一个最适合你的工作空间。记住你不是在办公室里！我个人选择在我的餐桌上工作，享受佛罗里达的清新空气。我的一些同事更喜欢被封锁的地下室办公室。你的工作空间应该是让你一天中最舒适的地方。

一个伟大的工作空间的一部分就是让你周围都是你喜欢的东西。我的厨房离我的笔记本电脑大约有 20 英尺远——我内心肥胖的孩子喜欢这样。这也意味着我可以随时获得水和其他饮料。我还在墙上挂了一台立体电视，通常会让网飞一整天都呆在家里。对我来说，某种白噪音和偶尔近距离的精神休息让我一整天都保持清醒。

## 生物特征

养个宠物吧！它们是一种难以置信的减压方式，也是解决问题的绝佳途径。不要害怕成为那个和他们的动物说话的疯子。Leelu 经常帮我摆脱精神困境。

## 交流，傻

交流。交流。交流。你必须不怕麻烦地与你的同事、你的上司，甚至是在办公室接待处工作的人交流。当然，这是有原因的:

### 沟通建立信任

同龄人对你的信任是关键，如果我们都以自己的优点来评判，那就太好了，但许多人在没有正常舒适的交流基础之前是无法信任的。

### 沟通建立融洽关系

除非你在办公室里，有机会与和你一起工作的每个人面对面交流，否则你就是匿名(或单面化身)的互联网人，显然是由同一家公司每两周支付一次工资。你可能很幸运，为一家可以使用视频会议的公司工作，但即使这样，你仍然是一个橱窗里的浮头。良好的沟通能让你与同事建立亲密关系，最终有助于合作。

### 频繁沟通创造价值

频繁的交流是最重要的。长时间消失会产生负面影响。记住你不在办公室，人们看不到你在做什么。据他们所知，你在游泳池里漂浮在一只巨大的充气天鹅上。当你一整天都伸出手时，你在提醒人们你在那里。当你经常分享你正在做的事情时，你在提醒人们你正在做的工作。

## helloooooooo

远程工作有时[会感觉像是 28 天后在*的一个场景*](https://www.youtube.com/watch?v=eCdRFMp8Xwo)。孤立感是每个远程工作人员每天都需要克服的。有时候，这是办公室里的一个缓慢的一天，而其他人则是因为物理办公室太忙了。在这种频繁的情况下，很容易感到气馁、沮丧，或者只是沮丧。这是一个离开并改正你的船的好机会。

因为我们的遥控器在一天中也被忽视了社交，所以当一天的工作结束时弥补这个不足。我是一个喜欢社交的人，呆在家里或与世隔绝太久会让我发疯。我努力在下班后参加社交活动。结交一些调酒师或其他在朝九晚五周期之外工作的人。如果你有一个家庭，去拜访一些邻居或者带孩子出去和其他人一起玩。

## 立定，然后着火

休息一下。吃午餐。睡午觉。谢绝会议。跑腿。和狗一起玩。去散步。去跑步。培养一个爱好，然后沉迷其中。看一集*行尸走肉*。如果你想喝成人饮料，就来一杯。我个人从不喜欢连续工作超过两个小时。磕磕绊绊让我精神饱满，头脑清醒。

停下来喘口气是我工作的关键之一。我前面提到的那些难以置信的自由在这里发挥了作用。当然，这取决于你为之工作的公司以及你的上级或同事，但是要利用这种能力去做那些在办公室里做不到的事情。请记住，你对这项工作的责任并没有结束。如果你要逃课，记得晚些时候腾出时间来完成你的任务。

## 拓展你的组织能力

当没有人可以聊天时，你并不总是有停下工作等待的奢侈。你必须发展扎实的组织技能，让你重新审视需要讨论的事情，或者继续你被阻止的任务。无论你在什么样的管理结构中工作，高度的自我管理都是必要的。谨慎的独立水平将大有帮助。你不会总是有时间或机会与你的同龄人讨论选择、问题或解决方案。这就把我们带到了...

## 对等依赖

(对于阅读这篇文章的书呆子来说，这是一个可怜的 NPM 双关语)

你的同事将意味着远程工作的成败。面试时，确保你也在问问题。你需要采访那些做采访的人，以确保他们知道如何与远程开发人员合作。一些公司刚刚涉足招聘遥控器，另一些公司经验丰富，还有一些公司不知道他们将进入什么行业。管理层应该参与进来，并看到远程员工的价值。这也有助于和有遥控器工作经验的同事一起工作。和将要管理你和你的项目的人谈谈，感受一下他们期望如何和你一起工作。永远不要害怕在早期分享你的想法。
