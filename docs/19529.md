# 如何像专业人士一样过滤掉不良客户——自由职业者指南

> 原文：<https://dev.to/prahladyeri/how-to-filter-out-bad-clients-like-a-pro-a-guide-for-freelancers-mla>

无论是 Upwork、Guru、自由职业者还是任何其他市场，坏客户的数量比比皆是，在这个时候，他们甚至可能远远超过好客户！

各种各样背信弃义的事情正在发生，这给自由职业这一高尚的职业带来了坏名声。有人以提供工作的名义获取你的信用卡号码或其他信息，有人试图从你那里榨取免费工作，甚至有人试图让你注册各种服务，以完成他们的销售目标。

你需要时刻保持警惕，意识到这些事情。一个流氓客户不仅欺骗你，还可能让你对自由职业失去信心。对于许多自由职业者来说，一次糟糕的经历可能会让人麻木。他们对这个职业失去信心，回到朝九晚五的工作岗位，一去不复返。但只有当他们对这些事情更加警惕和清醒时，只有当他们小心翼翼地处理那个自由职业项目时，他们的未来才会如此光明，他们今天就会实现他们的梦想。

我将列出一些我第一次见到新的潜在客户时经常使用的技巧。

1.  过滤掉因小失大的人:你必须做的第一件事是剔除那些试图与你讨价还价的人。现在，很有可能你提供的潜在解决方案价格有点过高，可以提供折扣，但在提供之前，观察他们如何讨价还价或谈判。一个客户真诚地想要一个公平的价格折扣，另一个客户想要几乎免费地完成所有事情，这两者之间存在行为差异。这两者之间有很大的区别，前者仍然重视你的技能，并且只在产品价值上争论，而后者只是把你当作不必要的花费。有了经验，就很容易识别这两种类型的客户端之间的差异。

2.  阅读过去的评论:一个好的客户或雇主通常会受到大多数自由职业者或与他们共事过的员工的喜爱，他们会写下他们与客户的互动以及他们与客户共事的感受。尽可能多地阅读评论，随机挑选，因为现在很容易伪造评论。还要查看所有平台的评论，而不仅仅是像 Upwork 或 Fiverr 这样的平台。如果他们是新客户，并且没有任何过去的回顾，你就必须更加小心地关注接下来的两个步骤。

3.  阅读社交媒体活动:这些天，几乎每个人都在社交媒体上，他们中的许多人将他们的整个生活故事都倾注在上面！尝试在 quora、twitter、facebook、linkedin 等网站上搜索该客户。看看他们的公开帖子怎么说。你会从中了解到客户的本质和真实性。

4.  慢慢来:从第一次互动就判断人性并不容易，不要匆忙通过你的提议并立即说好。慢慢来，和他们交流，问很多问题(特别是，如果你在上面两个步骤中没有得到你想要的)。坦率地询问你将要使用的项目管理工具(github/trello/等)。)、你的沟通方式(电子邮件/Skype/jitsi/等)。)，等等。如果他们提出任何要求，考虑他们是否有效。一个 NDA 通常是好的，但如果他们要求你在某个地方注册或问你要钱，那么不要这样做。客户没有理由向承包商要钱，就这么简单。在 0.001%的概率中，客户确实是真实的，仍然不值得冒这个险。拒绝这个提议，着手下一个提议。

你的办公桌上不断有提案出现，但这种糟糕的经历有可能让你放弃在那张桌子上工作，所以要了解这些风险，并始终保持警惕。
