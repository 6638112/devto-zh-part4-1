要在 JavaScript 中创建正则表达式，可以通过将表达式括在斜杠内并将其赋给变量
来按字面方式

`const urlSlug = /^[a-z0-9-]+$/;`

创建 regex 的另一种方法是使用 RegExp 对象的构造函数:

`const urlSlug = new RegExp(/^[a-z0-9-]+$/);`

请注意，使用构造函数创建 regex 时，不需要将其置于斜杠之间。

建议使用构造函数，例如，当您从外部源(如用户输入)获取 regex 时。
