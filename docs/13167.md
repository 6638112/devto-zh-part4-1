# 智能合同如何影响行业？

> 原文：<https://dev.to/errnausa/how-smart-contracts-can-impact-industries-1bcl>

智能合同是一种计算机程序，用于以数字方式启用、验证或实施合同的协商或执行。智能合同能够在没有第三方的情况下实现可信交易。智能合同的总体目标是履行常见的合同条件，减少恶意和无意的异常，并减少对可信中介的需求。

可编程合同将先进的合同法交易与在线商务中使用的协议相结合，可以在分布式账本系统(也称为区块链)上毫不费力地执行。

智能合约的区块链吸引人的一个特点是，因为分散的网络位于所有签署方之间，所以无需支付中间人，总是节省宝贵的时间并防止冲突。智能合同的某些部分可以在区块链上进行编码，当这些条件得到满足时，合同就完成了，并在区块链的分类账中进行不可逆的分录。

智能合同不仅像旧式合同一样概述了协议的指导方针和结果，而且机械地实现了需求。

区块链作为安全、分散的记录保存系统的出现，使得智能合同的实施成为可能。因为之前阻碍他们接受的关键问题——扩大信任——最终得到了解决。区块链是防篡改的。它们可以减少对可信第三方的需求，因为事实上，它们是无可争议的，并且可以提供比银行更大的安全性。

### 智能合约是如何在以太坊上执行的？

只要代码能够访问无限的资源和充足的时间，这些智能契约就能够计算事务。此外，这些代码根据需要循环多次，以完成代码中登记的指令。

由于这种能力，智能合约要么简单，要么非常复杂，取决于它们被编程执行的操作数量。值得注意的是，合同完成的每一项功能都附带一笔油费。

在以太坊上，gas 是估算计算工作难度的单位。举例来说，一个合同的运作可能需要 2 个天然气——这可以转化为一定量的乙醚——而另一个合同可能需要多达 5 个天然气。智能合约的指令越复杂，客户为执行它支付的汽油就越多。

请记住，乙醚的价格是浮动的，气体单位的存在是为了让矿工选择调整乙醚与气体的交换率，因为他们认为合适，但在以太坊上，每次操作的气体价格将保持不变。

以太币区别于其他货币的另一个重要特征是，它不使用未用完的产出。作为替代，它保持帐户余额的共享状态。这消除了交易输出的跟踪和编译，并使智能合约能够在不同账户之间转移资金。

### 使用区块链的智能合同可以影响的行业

#### 选举/政府

智能合同开发服务将通过提供一个明显更安全的系统来缓解选举过程中的所有担忧。受分类账保护的投票必须被解码，并且需要极高的计算能力才能访问。因为一个人不太可能拥有那么强的计算能力，所以要黑掉这个系统需要很多人！此外，智能合同可以消除低投票率，因为大多数时候，沮丧来自于混乱的系统，包括排队，显示身份和填写表格。通过智能合同，选举机构可以将投票转移到网上，千禧一代将集体投票给他们的领导人。

#### 抵押

手动处理抵押贷款可能很耗时。这个过程包括开放、融资和服务；有很多财务和财产数据需要核实，任何错误都可能导致可怕的后果。

然而，通过智能合同，可以避免在抵押处理中增加成本和导致延期的无能和系统问题。使用计算机化和重新设计，通过提供共享访问确认的计算机生成的文件和外部资源，如土地登记，智能合同可以为抵押贷款公司及其客户创造利益。

#### 医疗保健

使用智能合同，个人健康记录可以用私钥编程并存储在区块链中，该私钥只允许授权人员访问。同样的策略可以用来确保医学研究通过 HIPAA 法律进行。

还可以将手术收据存储在区块链上，并自动发送给保险提供商作为交货证明。该分类帐也可以用于全面的医疗保健监督，例如管理药物、遵守法规、检查结果和处理医疗保健用品。

#### 房地产

房产买卖通常在线下进行，是一个漫长的过程，通常涉及一群中间人。一旦房地产档案被数字化，所有房产的详细信息和业主的详细信息都将在一个安全的区块链上提供，该系统很可能会得到政府机构的支持，使每个人都能通过创建自己的智能合约在区块链进行交易。

存储在区块链上的记录将是不可变的，并相互链接，以便买家可以看到所有以前购买的细节，并确保交易透明和快速。传统的方法很像这样，只不过涉及到文书工作，太多的中间人，很多时间都浪费在来回奔波中。

随着智能合约的出现，许多行业都将受益匪浅，从在线投票到房地产和医疗保健，智能合约在区块链上的使用几乎是无限的。尽管智能合同显示了前景，但计划抓住其有前途和变革性机会的企业应该在实施行动战略之前进行广泛的研究。

目前，全球只有极少数开发人员能够编写智能合约。然而，有了像以太坊这样的有效项目，就有了呈现基础设施层的意图。编写这些合同将变得越来越容易，并使智能合同区块链成为不久的将来需要考虑的一股力量。
