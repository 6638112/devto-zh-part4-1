# 介绍学习身份！

> 原文:[https://dev.to/auth0/introducing-learn-identity-5f2c](https://dev.to/auth0/introducing-learn-identity-5f2c)

厌倦了用遥控器和屏幕电视键盘在智能电视上输入凭证吗？你已经和它打过太多次交道了，你可以打赌你的顾客也不得不这么做。如果您只需在他们的手机上轻敲几下就可以帮助他们登录，会怎么样？

你好，来自 Auth0 的 Dan 将为您带来 OAuth 2.0 设备流:一种快速、简单、安全地登录智能电视和其他输入受限设备上的应用程序的方法。没有键盘？没有浏览器？没问题！

普通登录可能会很繁琐，而且错误百出。OAuth 2.0 设备流是轻量级的，可以在几秒钟内验证您的身份，不仅可以与电视一起使用，还可以与游戏控制台、CLI、打印机等等一起使用！

Auth0 为开发人员提供了 OAuth 2.0 设备流的完全兼容的实现，以轻松解决这一用户体验痛点。让我们进一步了解什么是输入受限设备，这种授权如何工作，以及 Auth0 如何提供帮助。
