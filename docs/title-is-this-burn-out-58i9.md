# 这烧坏了吗？

> 原文：<https://dev.to/sloan/title-is-this-burn-out-58i9>

这是一个不愿意透露姓名的会员发来的匿名帖子。请仔细考虑你的回答，因为这些通常是很难写的帖子。如果你想留下匿名评论，请发电子邮件至。

我正带领一个团队制作另一个产品。有大量的工作，很大的商业影响，有足够的时间让我完成一大堆。但是我感觉没有写代码的动力。相反，我用寻找更多的知识来分散自己的注意力。我一直希望灵感会出现。

我喜欢迎接技术挑战，这样做通常会引发一连串的编码。但是一旦完成，我似乎再也不想做类似的事情了。尽管如此，我还是设法发布了多个成功的产品。但在长期职业生涯后，似乎很难找到挑战来激励我做我的雇主实际上需要做的工作。

我甚至为来上班感到内疚，因为我知道我今天不会带来那么多价值。所以我几乎没有假期了。但是当我起飞的时候，我也为没有在那里帮助球队而感到内疚。

我总是乐于帮助球队，我大部分时间都在这样做。我给他们出主意，帮助他们克服技术障碍，让他们对我们正在做的事情保持清醒的认识。但这并不能填满我的一天。我还有很多时间盯着屏幕，努力说服自己去写我知道需要写的代码。人们期待我写的东西。但我不想，我为此恨自己。“就写点什么吧！”

我带着未能“出色完成工作”的挫败感回家。第二天醒来，我知道同样的命运在等着我。起床是困难的。在上班的路上，我试着让自己做好准备，开始写代码。但是事情总是马上发生。当我终于坐下来编码时，动力消失了。

事情并不总是这样。我有多产的季节，像这样的季节。

我只是累坏了吗？以前有人经历过或见过这种模式吗？请大家分享一下这方面的经验。
