# 在线快速计算 rem 的方法

> 原文：<https://dev.to/hurricaneinteractive/fast-way-to-calculate-rem-online-3m2j>

> 使用 [JustREM](https://justrem.xyz/)

让我们把显而易见的事情弄清楚，这不是超级原创。有趣的事情！

这个项目的目标是创建一个网站，我可以很容易地将一个`px`值转换成一个`rem`值。我发现其他在线工具本来是用来做这件事的，但是又慢又笨重，用起来很痛苦，而且它们加载了 4 亿个广告。

## 🎉介绍 JustREM！！！

JustREM 是我想用的工具；

*   它会在您键入时计算出`rem`
*   您可以更改附加到值中的内容(这是我在使用设计系统时需要做的事情)
*   您可以更改基本值
*   加载速度快，工作速度更快
