# 一个组织想从你这里得到什么？

> 原文：<https://dev.to/doomsday/what-an-organization-wants-from-you-2h8k>

假设明天你要去一家像“xyz 私人有限公司”这样的中型公司面试，尽管你做了充分的准备和技术复习，你可能还是会在某个地方落后。你能想到吗？有些人可以说他们的英语不好，另一方面，你的借口可能是你对此没有做好准备。我们都不得不承认，至少 70%的我们已经接受过或即将接受的采访中，这种事情总是发生在我们自己身上。那么，是什么秘密魅力让近 60%的公司雇佣你呢？

1.  说服的力量(说服力)。
2.  团队合作。
3.  基本的计算机操作。

是啊！你没看错。这是你必须发展的三个关键素质，以增加你在大部分组织中超过 30%的被录用机会。
