## 简介

前端 web 开发全是用定制的`Html`、`css`和可能的一点点用于交互的`javascript`来创建网站的日子已经一去不复返了。

随着框架和库的流行，前端开发人员的工作要求变得越来越模糊。

从长远来看，这对初学者来说可能是一个问题，因为他们可能希望在匆忙学习框架或库之前花更少的时间学习基础知识(html、css 和 javascript)。

当你看一看招聘启事时，你会发现几乎所有的工作要求都包括初级职位的框架或库。你会发现很难找到只需要`html`和`css`的工作。

由于这个行业的口味经常变化，我决定列出 7 项技能，我觉得这些技能不会让初级 web 开发人员在获得第一份前端工作时看起来像离开水的鱼。

# **①。** **Html，Css & javascript**

这是前端开发人员的组成部分。和任何职业一样，在钻研难的部分之前，先掌握好基础知识总是恰当的。

*   超文本标记语言(HTML)是设计用于在 web 浏览器中显示的文档的标准标记语言。不是一种编程语言，尽管你可能会发现一些不这么想的开发者。

`html`的主要用途是结构化网页的内容。`HTML`由不同的元素组成，可以用来封装或包装内容的不同部分，使其以某种方式出现。

*   CSS(层叠样式表)用于呈现在`html`中创建的文档。它通过提供有助于操作页面布局、颜色、字体等等的功能来帮助提高可访问性、灵活性和控制力。

*   Javascript 也称为 ECMAScript，是一种支持网页之间交互的编程语言。它在构建 web 应用程序中起着重要的作用，并且受到 web 浏览器的良好支持。`Html`、`Css`、`javasrcipt`是万维网的核心技术之一。

对于一个前端开发人员来说，这三项技术的扎实背景是必不可少的。

# [**去**](#2-git)

`Git`是一个版本控制系统，用于跟踪软件开发过程中代码的变化。它使程序员能够在项目上合作，从而以更快的速度完成任务。

作为当今时代的前端开发者，git 的重要性怎么强调都不为过。几乎每一个招聘职位都要求具备 git 的基础知识。

要使用 git，您可能需要熟悉命令行，所以如果您害怕命令行，您需要克服恐惧并开始学习基础知识，从在本地机器上的文件之间导航，到使用它来发出第一个 pull 请求。

# **三)。** **响应式设计**

所以，你现在已经熟悉了`html`、`css`和`javascript`。你可以建立一个用户友好的网站。

随着不同屏幕尺寸的智能手机和平板电脑的流行，在你创建的每个网站或应用程序中，将响应式设计作为优先事项是很重要的。

人们主要在桌面上浏览网站和应用程序的日子已经一去不复返了。当今世界，平均每人每天至少花 4 个小时在智能手机上。

大多数人可能会从移动设备上查看你的应用程序或网站，因此需要扎实的`css`基础知识来帮助用户获得更好的用户体验。

# **隐射)。** **测试驱动开发**

这在初级水平的开发者中可能不会太受欢迎。我将这一点包括在列表中，因为编写可测试的代码对于初级水平的开发人员来说是非常必要的。

对于一个现代的初级 web 开发人员来说，知道测试驱动的过程是如何工作的是有好处的。在我的第一份工作中，我是一名 web 开发人员，我发现自己处于一个高度测试驱动的环境中，我很高兴我在职业生涯的早期阶段接触到了整个过程。

测试有多种形式(端到端、单元和集成)测试。你可以阅读它们，熟悉它们的术语和含义。

# **⑤。** **Javascript 库/框架**

作为一名现代前端开发人员，如果你想增加找到工作的机会，你最终将不得不学习 javascript 框架或库。

流行的 javascript 框架和库包括`Angular`、`React`、`Vue Js`和`Svelte`。

大多数开发人员发现很难先学习哪个框架或库。我认为这个决定应该基于你目前居住的位置。例如，在美国，React 非常受欢迎。所以如果你来自美国，你可能想先学习反应。

在我生活的国家，大多数公司都使用 angular，所以我首先开始学习 Angular。所以你可以做一些调查，看看你所在的地方最常用的是哪个框架或库，你可以从那里开始。

# **⑥)。** **服务器端渲染/脚本基础知识**

不久前，前端开发人员所需要做的就是制作一个显示网页内容但没有太多功能的网站。

如今，前端开发人员应该对`Api's`和`servers`如何与正在创建的应用程序相关联地工作有一个基本的了解。

对服务器端如何工作有一个基本的了解可以让前端开发人员在代码质量方面创建一个更好的架构，并且熟悉应用程序中的身份验证过程。

# **隐射)。** **为人处事**

是的，技术技能非常重要，但我们生活在一个人们容易分心的时代，并且发现不看他们的智能手机很难长时间交谈。

作为一名网站开发人员，你需要有人际交往技能，使你能够清楚而恰当地与他人交流。

在您的设备上查看一些通知时，您不应该与您的产品经理联系。您应该知道如何向同事或高级开发人员寻求帮助，解决您觉得难以完成的任务。

沟通的定义看起来很简单，但却是最难掌握的技能之一。

**结论**

达到你的目标水平的道路是漫长的。随着越来越多的技术被发布，你不应该觉得必须一次学会所有的东西。

选择一个 web 堆栈，在尝试学习其他东西之前，对它足够精通，以构建一些有形的东西。

作为开发人员，你越老，你就越有价值。不要让自己处于试图跟上每一项新技术发布的压力之下。
