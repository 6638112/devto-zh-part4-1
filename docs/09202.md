# 我是一名前酒保/服务员，现在是一名 web 开发人员和 DevOps 工程师，问我任何问题！

> 原文：<https://dev.to/pencillr/i-m-an-ex-bartender-waiter-who-is-now-a-web-developer-and-devops-engineer-ask-me-anything-14gb>
