# 从 postman 触发 PHP XDebug

> 原文:[https://dev.to/dechamp/trigger-php-xdebug-from-postman-1hab](https://dev.to/dechamp/trigger-php-xdebug-from-postman-1hab)

当使用 postman 时，可以很容易地触发 xdebug 为 PHP 运行。您只需要简单地将`XDEBUG_SESSION_START=PHPSTORM`添加到您的 url 查询参数中。

当然，如果不同，你需要用特定的触发键替换掉`PHPSTORM`。
