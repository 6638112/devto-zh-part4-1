# 为什么 Node.js 被认为是 web app 开发领域的后起之秀？

> 原文:[https://dev . to/saurabhkumarqss/why-node-js-is-consided-a-rising-star-in-the-web-app-development-space-497 a](https://dev.to/saurabhkumarqss/why-node-js-is-considered-a-rising-star-in-the-web-app-development-space-497a)

JavaScript 不仅被认为是最广泛使用的编程语言之一，而且是无处不在的软件应用程序开发框架。
从技术上来说， [Node.js app 开发](https://www.qsstechnosoft.com/technologies/node-js-app-development/ "Node.js app development")不是一个框架或者 librar，而是一个建立在 Chrome 的 V8 JavaScript 引擎上的运行时环境。
