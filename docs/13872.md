# 要是我有更多的时间就好了

> 原文:[https://dev.to/justzeros/if-only-i-d-had-more-time-36ho](https://dev.to/justzeros/if-only-i-d-had-more-time-36ho)

昨晚我和一些同事一起吃饭，快结束的时候，其中一个对我说，“我注意到你描述你的生活时有一个主题。你似乎会腾出时间来思考。”

“这是真的，但并不总是如此，”我回答道。

“在我的职业生涯中，大部分时间我只用来做事情，而不是思考。但这让我有一种模糊的不安的感觉，如果我有更多的时间，大多数事情都可以做得更好。我并没有变得优秀，而是在生活的许多方面都还过得去。我不想再那样了。”

虽然我远非完美，但我发现两个变化帮助我找回了思考时间。

首先，我每天早起。

一旦我的家人醒了，我的时间就不是我自己的了。因此，我每天早上 4:55 起床，阅读、写作、喝咖啡和思考。这持续 60-90 米。

第二，我定期写作。

我*开始*这个邮件列表，因为我听说这是一个很好的营销策略。

我继续做这件事，因为它迫使我坐下来深入思考每一天。

通过给你写信，我知道了我的信仰，并对这个世界有了新的看法。

我没想到会这样。我想我应该写下我所知道的。相反，我通过写作找到了我所知道的。

你可能会说我“通过写作来思考”，因为写作的过程就是我的思考过程。

我没想到会有这样的好处，但这是我继续写作的原因。

作为一名领导者，你需要思考时间——你必须找到对你有用的方法。

我不知道这些想法对你是否有帮助，但我可以说，自从我留出时间思考后，我对自己的生活更满意了。而且，我认为我做得更好，对结果也更满意。

这也是你纠结的事情吗？

极品，马库斯
