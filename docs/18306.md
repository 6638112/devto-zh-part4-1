你好，首先让我感谢你邀请我来这里。我去年六月从 bootcsmp 毕业。一年后的今天，我仍然是个乞丐。在间隙中编码一点帮助都没有。一个有家室的男人，一份到处加班的工作。这些事情妨碍了我的日常工作。我已经记不清有多少次试图把 JS 的基本原理记下来了。就像今天，我计划下班后写代码。我回到家，意识到我需要去一趟杂货店，这样我这周就不会在工作上花钱了。似乎总有一些事情。我如何克服这些障碍并每天编码，而不觉得我在动摇我的妻子、儿子和这个世界🌍
