# 小型企业机器学习的常见应用

> 原文：<https://dev.to/vincajayi/common-applications-of-machine-learning-for-small-scale-businesses-3oki>

当我第一次听说机器学习(ML)时，我认为只有大公司才会应用它来探索大数据。在互联网上搜索 ML 的含义时，我发现维基百科将其定义为人工智能(AI)的一个子集。特别是，它涉及计算机系统用来有效执行特定任务的算法和统计模型的科学研究，而不使用显式指令，而是依赖模式和推理。除了诸如语言翻译之类的自然语言处理之外，互联网上可用的 ML 应用的一些常见示例包括皮肤癌检测、面部识别、流失预测、糖尿病眼病的诊断。此外，在亚马逊和网飞等公司应用推荐引擎来预测和建议任何给定用户可能想要购买或观看的内容的方式中，ML 发挥了作用。

我从来没想过 ML 对小规模企业有用。令人惊讶的是，几乎所有我见过并讨论过 ML 概念的人也持同样的观点。我一直持有这种观点，直到我注册了一个关于 ML 的短期培训课程。随后，我被分配了一个项目，开发一个 ML 模型，可以有效地改善一个慈善组织的营销活动的成本，这完全改变了我的观点。

本文旨在讨论 ML 对于小型企业的重要性，并举例说明如何使用 ML 算法来估算成本。

小型企业机器学习(ML)的六个常见重要功能

1.  趋势和模式识别:几个小规模企业的所有者维护一个销售簿和一个会计簿，其中他们记录来自不同商店的客户的姓名、销售量、现金交易等。这些记录生成的数据可以用来分析客户的购买模式，以及推动销售或影响客户偏好的其他因素。
2.  建模和预测:销售手册中的信息可以用来估算成本、预测销量、衡量收入、利润以及预期的市场份额。优势在于，企业的可持续性和成功取决于这种预测的准确性。
3.  安全性:ML 可以帮助分析数据和识别数据中的模式。此外，该流程可用于识别可疑交易行为、跟踪错误和检测欺诈。因此，企业主可以很容易地立即采取行动，以弥补漏洞，并防止其在未来发生。
4.  信息提取:ML 可用于从其他数据库中提取有价值的信息，并鼓励操作协调。事实是，没有一个企业主能够成为知识的孤岛。为了做出正确的商业决策，企业所有者需要外部信息，以对这些信息进行适当的数据分析(例如，关于天气、通货膨胀、利率等的信息)。).
5.  良好的商业环境:ML 创造了一个适合小企业成长和变得高效的环境；它还为员工提供了新的技术来更好地发挥作用。例如，ML 推荐引擎可以模仿客户行为，推荐额外的产品并促进追加销售。媒体公司使用 ML 来识别嘴唇运动的模式，并将其转换为文本。
6.  广告和营销:值得注意的是，75%的企业利用 ML 将客户满意度提高了 10%以上，而四分之三的组织利用 AI 和 ML 将新产品和服务的销售额提高了 10%以上。在几秒钟内，ML 应用程序就可以联系到数百万客户，告知他们新产品的信息，以及现有产品优于替代产品的原因。

机器学习如何用于估计成本的一个例子
一个慈善组织依靠其好心人的慷慨来支付运营成本，并提供必要的资本来从事慈善事业。由于来自不同国家不同地区的捐赠者数量增加，通过明信片筹集资金的成本逐年增加，10%的记录显示，通过明信片收到的个人平均捐款为 15 英镑，而制作和发送明信片的成本为 2 英镑。如果一个慈善组织选择向每个认同该组织的人发送一张明信片，预计这一成本可能会增加。因此，该组织需要聘请一名数据科学家来开发一个经济高效的模型，以确定最有潜力和最有可能捐款的捐赠者。

从研究中，我们观察到决策树分类器(CART)优于其他预测模型，因为该模型与其他选择的模型相比具有最高的均值。因此，我们应用 CART 模型进行预测。结果如下:

对于包含 3875 个家庭的分类报告，我们测试了可能为慈善组织捐赠资金的家庭的实际数量。在我们的报告中，有两种可能的预测类:“0”和“1”。对于两个预测结果，“1”表示可能捐款的实际家庭数量，而“0”表示不太可能捐款的实际家庭数量。

根据混淆矩阵，N = 3875(家庭总数)，决策树分类器预测 1076 个家庭可能捐赠(FP +TP)和 2799 个家庭不太可能捐赠(TN+FN)。现实中，样本中有 973 户家庭会捐款(实际“1”= TP+FN)，而有 2902 户家庭可能不会(实际“0”= TN+FP)。
要计算成本，请回忆以下内容:

单位成本= 2
单位平均收入= 15
为了最大限度地降低成本，该组织可以避免向每个表示有兴趣的人发送明信片，而只向那些最有可能捐款的家庭发送。

在这种情况下，总成本将是(TP+FN)*单位成本= 1946
收入= TP *单位收入= 4200
利润=(TP *单位收入)–([TP+FN]*单位成本)= 2254

这个结果意味着，如果慈善机构只给那些有可能捐款的家庭寄明信片，它将花费 1946 英镑，收入 4200 英镑，产生 2254 英镑的利润。
