# 在开发过程中，您是否在您的机器上运行所有相关的服务？

> 原文:[https://dev . to/kishoreannapureddy/during-development-do-you-run-all-the-dependent-services-on-your-machine-1kb 7](https://dev.to/kishoreannapureddy/during-development-do-you-run-all-the-dependent-services-on-your-machine-1kb7)

开发需要运行许多依赖服务，如数据库、缓存服务器、队列、其他依赖(微)服务等。

你在你的机器上运行它们吗？或者您使用其他服务器基础设施来运行它们，同时只在您的机器上运行您正在开发的特定服务？
