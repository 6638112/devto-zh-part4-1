# 省道中的线圈

> 原文：<https://dev.to/jay_tillu/loops-in-dart-2bji>

当我们想要多次重复某个代码块时，在编码时会出现许多情况。现在有两种方法来完成这项工作。

1.  编写代码块的次数。
2.  使用循环。

在第一种情况下，问题是你的代码会变得过于冗长和老套。过了一段时间后，维护代码会变得非常困难。正如在编程中有一个普遍的规则，那就是 ***【永远不要重复你的代码】*** 。

所以在这种情况下，为了拯救你，循环就出现了。为了重复一些语句的次数，我们使用循环。

循环体内的语句将不断重复，直到条件变为假。

循环的重复在编程中称为迭代。

*   编程中有两种类型的循环:

1.  一定的
2.  模糊的

**定循环-** 迭代次数固定的循环，其中在进入循环之前知道确切的迭代次数，则称为定循环。for ex- for 循环

*   [为循环](https://dev.to/jay_tillu/while-loop-in-dart-21og)

**不定循环-** 迭代次数不固定的循环，其中在运行前你不知道确切的迭代次数，则称之为不定循环。Ex — while 循环& do-while 循环

*   [While 循环](https://dev.to/jay_tillu/while-loop-in-dart-21og)
*   [Do While 循环](https://dev.to/jay_tillu/do-while-loop-in-dart-33ee)
