**简介**

读者们好，我的名字是穆罕默德，我从来没有想过要从事这个行业，但不知何故，在 90 年代初，当电脑游戏行业开始让它付出代价时，这种新的娱乐形式对孩子来说就像日出中闪烁的晨星一样明亮。这是一种药物，一种真正令人上瘾的药物，接触它意味着，父母必须对孩子施加限制，因为他们的头脑虚弱，所以他们可能不理解，但你能说什么呢，孩子的头脑是一个充满可能性的美丽宇宙，无限的可能性。所以你限制了一个孩子，最终你得到的是一个孩子试图寻找替代方法来绕过限制。你没有为家长事先制定的安全协议提供基础，从而把孩子培养成了黑客。然后它发生了，到处摆弄，拥有整个世界的时间，发现一个玩具的路径发生了，那个玩具是什么个人电脑。我记得，它是一个奔腾 200mhz 处理器，配有 MMX 处理器，天哪！，我有 128mb 的内存，这确实让我喜笑颜开。我买电脑的时候，记得卖家说


**概述**

我从预建的 CMS 开始我的道路，我们将强调，如 PHP 中的 Wordpress，Joomla 和 Magento，然后我从这些 CMS 转移到我自己的 PHP 定制 CMS，这起初似乎是不可能的，但最终只是我对不可能的恐惧阻碍了它。我认为你开始害怕的是变成你不可能的事情。我将给出我使用 PHP 的经验，以及我为什么完全转向 Java 的原因。有些优点要比缺点好得多。我将包括一些练习，如果人们喜欢，可能会有一些评估，然后还会指导使用 git，我更喜欢 GitLab。我们将继续创建内容。祝我好运，干杯！。永远欢迎贡献者。
