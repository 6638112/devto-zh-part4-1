就编程而言，如果你发布一些不太切题的东西，这并不是世界末日，但我们会努力为不太关注编程的帖子保留这个空间。

我们都是开发人员，无论是职业还是爱好，但是我们都有代码之外的生活。如果你对任何与代码无关的话题感兴趣，可以去饮水机旁聊聊。

当然，[行为准则](https://dev.to/code-of-conduct)在 watercooler 标签中的应用和在其他地方一样。

*如果你**不想**看到太多来自 watercooler 的帖子，目前的方法是先**关注**标签，然后**在 [/dashboard/following](https://dev.to/dashboard/following) 中给它一个负权重**。*

我知道这有悖常理，但这是目前最好的方法。
