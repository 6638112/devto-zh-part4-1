如果你是一名 Java 程序员，正在考虑学习一种新的编程语言来扩展自己的知识和技能，但不确定选择哪种编程语言，那么你来对地方了。

在这篇文章中，我将分享 Java 程序员可以学习的 3 种 JVM 语言以及为什么你应该学习它们。你不需要学所有的，理想情况下，你选择最适合你的。例如，我非常喜欢 [**Groovy**](https://javarevisited.blogspot.com/2020/06/top-5-courses-to-learn-groovy-and-grails.html) ，这是我学到的，因为它在我们的构建工具和自动化测试中被大量使用。

此外，成为一名通晓多种语言的开发人员是一种良好的素质，在面试中通常会受到高度重视。这也拓展了你的思维，因为不同的编程语言有不同的特性，使得某些事情变得非常简单。

学习一门新的编程语言也让你有了比较两种编程语言优劣的经验和知识，这本质上让你能够写出更好的代码。

今天，我将向 Java 开发人员推荐的所有编程语言都是基于 JVM 的，因为它们对 Java 程序员来说更容易学习。

它们以这样或那样的方式与 Java 相关，这就是为什么它们有助于拓展任何 Java 开发人员的思维。我还包括了相关的书籍和在线课程，你可以用来学习这些语言，如果你从我的链接中购买，我也会得到报酬。

## [](#scala-kotlin-and-groovy-3-jvm-languages-java-programmers-can-learn)Scala、Kotlin 和 Groovy-3 Java 程序员可以学习的 JVM 语言

下面是我列出的 Java 开发人员应该学习的三种编程语言。为什么我只包含了 3 种语言，而不是 10 种或 5 种？

> 嗯，学习一门新的语言并不容易——学习一门新的编程语言需要时间和精力，即使你只能学会一门，那也绰绰有余了。

### [](#1-scala)1。斯卡拉

Scala 已经存在很长时间了，当它第一次出现时，被吹捧为取代 Java 的最佳语言。嗯，这并没有发生，尤其是在 Java 8 中引入了一些函数式编程特性之后，例如 [lambdas](http://javarevisited.blogspot.sg/2017/01/3-ways-to-convert-java-8-stream-to-array.html) 和 [stream](http://www.java67.com/2014/04/java-8-stream-examples-and-tutorial.html) ，但 Scala 仍然是 Java 开发人员需要学习的语言。

Scala 在过去几年中不断发展，越来越多的公司和初创公司开始使用它，如优步、索尼、亚马逊、Autodesk 等，这使它成为最受欢迎的技能之一。

流行的 Scala 框架如 Play、Akka 和 [Spark](https://javarevisited.blogspot.com/2017/12/top-5-courses-to-learn-big-data-and.html#axzz6qJRj7kdw) 也在帮助公司采用 Scala 进行网络开发和[大数据](https://dev.to/javinpaul/top-5-courses-to-learn-big-data-and-hadoop-for-beginners-6g8)解决方案。Scala 开发人员平均收入超过 12 万美元也就不足为奇了。

如果你正在寻找一些令人兴奋的工作并学习一门函数式编程语言，那么 Scala 应该是你的首选。

### [](#2-kotlin)2。我的锅

Kotlin 是由 JetBrains(IntelliJ IDEA、PyCharm 和许多其他顶级 ide 的制造商)在 2011 年推出的，但没有多少程序员关注它，直到谷歌在 2017 年宣布它是 Android 开发的官方语言。

像 [Groovy](http://javarevisited.blogspot.sg/2017/08/top-5-books-to-learn-groovy-for-java.html) 和 [Scala](http://javarevisited.blogspot.sg/2018/01/10-reasons-to-learn-scala-programming.html#axzz550Ppgfxg) 一样，Kotlin 也运行在 JVM 上，并向下编译成 Java 字节码；您可以在现有的 Java 或 Android 项目中开始修改 Kotlin，一切都会运行良好。

所以，如果你对[学习 Android 开发](https://medium.com/hackernoon/top-5-courses-to-learn-android-for-java-programmers-667e03d995b4)感兴趣，想让你的 app 安全的避免 NullPointerException，想给你的函数一些纯度，写一些高阶函数，或者一些函数式编程，那么你就应该学习 Kotlin。

### [](#3-groovy)3。绝妙的

几年前，我探索了 Groovy 领域，从那次经历中，我可以说每个 Java 开发人员都应该学习 Groovy。

它以其简洁的语法和强大的编程特性真正补充了 Java，比如集合文字、定义任何东西的 def 和多行字符串。

学习 Groovy 之后，我意识到我可以用 50 行代码做什么。与 Java 相比，它大大减少了做某些事情所需的代码，Groovy 最好的一点是它**是** Java。

我学习 Groovy 的一个原因是 Spock 框架，这是一个基于描述和规范的单元测试框架。它使得 Java 中的单元测试更加容易，并且扩展了 T2 JUnit 的功能。

如果你决定学习 Groovy，那么 Udemy 上完整的 Apache Groovy 开发者课程 是一个不错的起点。本课程用简单的语言解释了 Groovy 的基本特性。

这就是 Java 开发人员为了成为一名通晓多种语言的程序员可以学习的 3 种编程语言。如果你想转向函数式编程方面，那么 Scala 是最好的选择。如果你对开发手机游戏和应用感兴趣，那么 [Kotlin](https://javarevisited.blogspot.com/2018/02/5-courses-to-learn-kotlin-programming-java-android.html) 会帮助你，如果你想写脚本，那么 Groovy 是一个很棒的语言。它在单元测试和构建自动化方面也有很大帮助。由于 [Groovy](http://javarevisited.blogspot.sg/2017/08/top-5-books-to-learn-groovy-for-java.html) 编译成 Java，它有效地补充了 Java 编程语言。
