没错。一个空数组。有道理。我想。

所以套路。我把它留给其他人去测试(我已经测试过了，它不是特别有性能):

```
function Bruce_ObjectKeysReduce(string) {
  return Object.keys(string).reduce(function (acc, cur) {
    acc = string[cur] + acc;
    return acc;
  }, "");
} 
```
