# 数据、归纳推理和程序犯罪剧:为什么我决定从事数据科学

> 原文:[https://dev . to/MDA ni38/data-inductive-reasoning-and-procedural-crime-drapes-why-I-decide-to-purchase-data-science-53i 6](https://dev.to/mdani38/data-inductive-reasoning-and-procedural-crime-dramas-why-i-decided-to-pursue-data-science-53i6)

在过去的一个月里，我决定参加一个数据科学训练营。这个决定看起来似乎是突然冒出来的，但有许多原因驱使我进入令人兴奋的数据科学世界。

虽然数据科学家的职业机会越来越多，越来越有吸引力，但我一直对使用数据结合编程来解决问题很感兴趣。在过去的几年里，我断断续续地尝试过使用编程和不同的技术来处理我作为一名科学家在工作中使用的数据。这个支点代表了我可能的职业道路上的一个变化，但它仍然忠实于我一直以来的兴趣。

我一生中的大部分时间都对追求科学感兴趣。从年轻的时候起，我就一直对科学工具以及我们如何利用它们来更多地了解我们周围的世界感兴趣。科学方法驱动我如何与周围的所有信息互动。

我对科学的兴趣驱使我获得了地质学学士和硕士学位，并发表了我对南极洲罗斯海冰盖沉积物下地形的地震重建的研究。过去的追求已经给了我大量分析、操作和可视化大型数据集的经验。

那么，为什么我特别决定进入数据科学领域呢？对我来说，就是**归纳推理**、**基于数据的问题解决**、**结果的有效可视化**这些概念。这些概念在另一部《我爱侦探剧》中也有表现。

## [](#detective-shows-what-does-that-have-to-do-with-anything)侦探秀？这有什么关系？

[![Alt Text](img/630c5ff71acfca319a9caa7101d75b2d.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--5G4rOrb_--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/yxm2xwf5eyd3iwfxabik.gif)
<sup id="fnref2">[2](#fn2)</sup>

是的，侦探剧或程序犯罪剧与我从事数据科学的兴趣有关。虽然这些类型的电视节目很有趣，我也看过大部分(CSI 是我最喜欢的)，但我对侦探节目的热爱与我对逻辑推理的使用息息相关。

虽然每一集都不一样，但程序是一样的。这些角色(比如上图中的杰西卡·弗莱彻)面临着一个谜或问题。然后他们必须检查犯罪现场的证据。此外，可以收集关于该情况周围环境的更多信息。他们需要确定哪些证据是有用的，哪些证据是无用的。然后他们必须根据他们所掌握的信息得出结论。

对我来说，这是一个类似于数据科学的工作流程。问题、目标和挑战总是不同的，但数据科学家必须使用他们拥有的数据。然后他们决定是否有新的数据可以收集来帮助解决问题。必须确定在大型数据集中关注哪些信息是有用的，哪些数据是无用的。最后，他们可以分析这些数据，并使用不同的技术对其进行建模和可视化，以得出结论。
