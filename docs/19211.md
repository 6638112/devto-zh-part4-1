## [](#the-application-state)应用状态

状态将在`App`组件中管理。使用`state lifting`技术，它将通过 son 组件。
此刻，我只能想象这个状态会是什么样子:一个 todo 的集合，其中每个 todo 都有一些属性。

```
...
{
  id: 1,
  description: "Lorem Ipsum",
  done: false
},
{
  id: 2,
  description: "Another Lorem Ipsum",
  done: true
},
... 
```
