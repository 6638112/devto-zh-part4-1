# DEVpreneur 播客

> 原文:[https://dev.to/charlieinthe6/the-devpreneur-podcast-5abl](https://dev.to/charlieinthe6/the-devpreneur-podcast-5abl)

本周，我和我的好朋友兼前同事汤姆·萨拉戈萨发布了一个名为《企业家 T2》的播客。18 个月前，我和汤姆在郁金香零售公司做软件开发时相识。我们很快发现我们有相似的兴趣；我们都喜欢在业余时间制作 SaaS 产品。

我们开始[DEVpreneur 播客](https://charliereese.ca/devpreneur-podcast)来谈论我们对企业家精神和软件开发交集的共同热情。我们涵盖了常见的误解，如何快速构建用户想要的产品，等等。我们的第一集有点回音，但我想我们已经摆脱它了。谁知道音频是这样一个棘手的业务-不是我。

此外，我们在我的个人网站上托管[播客的媒体文件和 RSS 提要。如果你对如何为自己的播客做这件事感兴趣，我在我的博客上写了关于](https://charliereese.ca/devpreneur-podcast/rss)[如何主持自己的播客 RSS 提要](https://charliereese.ca/blog/45-how-to-host-your-own-podcast-rss-feed)——你应该看看！
