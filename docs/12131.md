# 作为 WordPress 开发者学习 JavaScript 的初学者指南

> 原文:[https://dev . to/codeinwp/初学者-学习指南-JavaScript-as-a-WordPress-developer-4523](https://dev.to/codeinwp/beginner-s-guide-to-learning-javascript-as-a-wordpress-developer-4523)

如果你在过去的几个月里一直在关注 WordPress 社区发生的事情，你可能已经意识到 JavaScript 日益增长的重要性。

首先，WordPress.com 桌面应用 Calypso 脱离了标准的 PHP 路线，使用 JavaScript 进行编码(这是 CodeinWP [最近谈到的](https://www.codeinwp.com/blog/december-2015-wordpress-news/))。如果这还不够，在 2015 年 12 月的年度“国情咨文”演讲中，马特·莫楞威格直截了当地说:

*“深入学习 JavaScript。”*

自然地，越来越多的 WordPress 开发者开始转向 JavaScript，实际上，2016 年将会是 JavaScript 之年，至少从 WordPress 开发者的角度来看是这样。

如果你是一个现有的 WordPress 开发者，你应该如何开始你的 JavaScript 之旅？更重要的是，这种多样化的编码语言的哪些方面是你最应该关心的？在这篇文章中，我将尝试回答这些问题。

作为 WordPress 开发者学习 JavaScript 的初学者指南

### [](#introducing-javascript-to-wordpress-developers)**向 WordPress 开发者介绍 JavaScript**

同样，如果你一直活跃在 WordPress 社区，你可能已经熟悉了 JavaScript 的基础知识，因此，我们可能不需要提醒自己 JavaScript 不同于 JAVA。

现在，转到另一个问题:**对于 WordPress 来说，JavaScript 真的那么陌生吗？**

不完全是。

事实上，甚至在 WordPress 4.4 之前，JavaScript 就已经在 WordPress 中发挥了作用——在 *wp 中的 js 文件夹——包括*和 *wp-admin* 代表 JavaScript，到目前为止，JavaScript 主要用于主题开发。因此，尽管 JS 已经在 WordPress 的世界里存在了很长一段时间，现在有了 REST API 的前半部分和 Calypso 这样的应用程序，它的角色已经上升了许多。

任何开始使用 JavaScript 的 WordPress 开发者可能会感到困惑，因为 JavaScript 有自己的世界。例如，举三个与 JS 相关的非常常见的术语: [jQuery](https://www.codeinwp.com/blog/learning-javascript-for-wordpress/) ，AngularJS 和 Node.js。

[![Alt text of image](../Images/b2bb43a10565981cb46b8650aec6e740.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--v5WPfzr3--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://www.codeinwp.com/wp-content/uploads/2016/01/jQuery-AngularJS-Node.js_.png)

所有上述三个实体，尽管与 JavaScript 相关，但彼此完全不同。JQuery 是 JavaScript 库，而 AngularJS 是 JavaScript 框架，Node.js 是 JavaScript 运行时环境。

有些程序员认为从普通的 JS 开始更明智，而其他人更喜欢依赖框架或库。谈话并没有就此结束。如果您决定从 JavaScript 库开始，您还必须了解它的附加组件或插件(是的，有扩展库的核心功能的插件，就像库扩展 JavaScript 的核心特性一样)。

已经困惑了？请继续阅读！

### [](#know-the-fundamentals)**知道基本面**

如果你真的很重视你的 JavaScript 技能，你必须非常重视基础。这一点尤其重要，因为作为一名 WordPress 开发者，你必须已经了解 PHP，而 JS 在各个方面都不同于 PHP。

大多数 PHP 开发人员都习惯于 PHP 的函数式方法，尽管 PHP 确实有许多面向对象的原则，但它在很大程度上是一种函数式语言。另一方面，JavaScript 是一种严格面向对象的语言。Mozilla 开发者文档详细讨论了 JavaScript 的 [OOP 基础。](https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Objects)

一旦你掌握了 JavaScript 的基础知识，你就可以开始使用框架了。

### [](#pick-a-javascript-framework)**挑选一个 JavaScript 框架**

如果您希望尽快开始构建 JS 应用程序和项目，学习 JavaScript 框架是至关重要的。如果你在 WordPress 中有过一些主题开发的经验，你可能已经熟悉 HTML 和 CSS，更重要的是，你会意识到一个好的框架可以带来的编码的简易性。

当谈到为初学者挑选完美的 JavaScript 框架时，专家和编码人员意见不一。就个人而言，对于熟悉 MVC 框架的用户，比如 concrete5 开发者，我一直推荐 [AngularJS](https://angularjs.org/) 作为更好的选择。

Angular 支持 MVC 结构，确实很容易上手。然而，WordPress 编码人员有时并不十分熟悉 MVC 结构，因此，React 似乎是一个更容易和更合适的选择。此外，因为 [React](https://github.com/reactjs) 只使用视图组件，所以它在设计师中也很受欢迎。

还有其他选择，但是在大多数情况下，JavaScript 框架世界的任何旅程都应该从 React 或 AngularJS 开始。我曾经写过一篇关于这两个框架的比较文章，更详细的内容可以在这里找到那篇文章。

### [](#decide-what-you-need)**决定你需要什么**

就像任何其他技能一样，学习一门新的编程语言是一个需要和要求的问题。毕竟你在努力学习 JavaScript，因为你需要它。

在 JavaScript 中，库和框架是根据您的需求定义的。例如，Node.js 非常适合那些必须处理服务器端编程的人。同样，尽管 React 对于 web 开发来说是一个很好的选择，但在移动应用程序开发方面却很欠缺。

对于 WordPress 开发者来说，需求基本上可以分为后端和前端。如果你正在处理前端开发，像 [AngularJS](https://angularjs.org/) 、 [React](https://reactjs.org/) 或者 [Backbone.js](https://backbonejs.org/) 这样的框架将会满足你的需求。点击，你可以找到一个更长的列表，里面有更多的选项[。](https://github.com/collections/front-end-javascript-frameworks)

### [](#where-to-learn)**在哪里学的**

感谢互联网，对于任何愿意学习新事物的人来说，资源并不短缺，JavaScript 也不例外。你可以在 Stack Overflow 上讨论你的疑问，关注 Udemy 或 Lynda.com 的教程，当然，也可以参考一些 WordPress 博客上的优秀文献。

[JavaScript 很性感](http://javascriptissexy.com/)无论你是初学者还是进阶学习者，都是学习 JavaScript 的好资源。不幸的是，这个网站已经有一段时间没有更新了，所以你需要小心行事。也就是说，某些线程比如 JavaScript 中的 [OOP 仍然是有用的和相关的。](http://javascriptissexy.com/oop-in-javascript-what-you-need-to-know/)

Codecademy 的 [JavaScript Track](https://www.codecademy.com/learn/introduction-to-javascript) 在初学者中相当受欢迎，你应该尝试一下。同样，如果你热衷于学习 jQuery，代码学校的 [Try jQuery](https://jquery.com/) 计划是个不错的选择。

对于学习 React，我发现 [React for 初学者](https://reactforbeginners.com/)是计划和结构最完善的资源。 [Superhero.js](http://superherojs.com/) 是另一个惊人的资源和教程集合，你可以从中学习。

在书籍方面，选择很多，你可以在这个 GitHub 线程上找到一些最有用和最受欢迎的书籍。

### [](#conclusion)**结论**

作为一名 WordPress 开发者，你学习 JavaScript 的过程或多或少会和你学习 PHP 的过程一样。

PHP 的实用知识足以调整 WordPress 和主题或插件，并从中获得更多，但是如果你希望认真构建一些大的东西，PHP 基础知识是必不可少的。类似地，虽然掌握一个 JavaScript 框架会让您快速入门，但是您还需要掌握 JS 的基础，以便充分利用它。
