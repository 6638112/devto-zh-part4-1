# 使用 Angular Universal 构建应用程序或网站

> 原文:[https://dev . to/angular/applications-or-websites-build-with-angular-universal-p4e](https://dev.to/angular/applications-or-websites-build-with-angular-universal-p4e)

自从 2016 年 10 月我第一次了解这项技术以来，我一直在使用 [Angular Universal](https://angular.io/guide/universal) 用于[我的博客和个人网站](https://samvloeberghs.be/)！

你用 Angular 或者更确切地说是 Universal 建立过哪些网站？您面临过哪些挑战？到目前为止你觉得怎么样？

对于那些还没有听说过它的人来说，Angular Universal 是一种允许在服务器上渲染 Angular 应用程序的特定技术。这给了你一些主要的好处，比如

*   更好的性能和加载时间
*   更好的搜索引擎优化和爬虫支持
*   [还有很多..](https://angular.io/guide/universal#why-use-server-side-rendering)

点击这里和[这里](https://github.com/angular/universal)了解更多信息。如果你有具体的问题，请随时提问！
