# 面对你作为开发者的恐惧，获得最佳奖项

> 原文：<https://dev.to/blarzhernandez/face-your-fears-as-developer-and-get-the-best-award-3llc>

你有没有问过自己，作为一个开发者，什么让你害怕？如果答案是肯定的，你是否已经决定真正面对和思考如何克服那些不允许你发挥最大潜力的事情，或者你是否还在处理？我敢说，对我们大多数人来说，答案是否定的，让我在这篇文章中向你解释为什么我们作为开发者或人类必须承认、拥抱和面对我们的恐惧，以及最重要的事情——最终获得最佳奖项。我完全相信没有一场战斗是没有任何回报的，其中一些你将会征服，而你将会从其他人那里学习。

我们大多数人都有恐惧。为什么？因为所有的恐惧都是习得的。没有人生来就有恐惧，我们伴随着恐惧长大，这样不好吗？不，那很好，真正的斗争是如何面对和忘记它们。在我的一生和职业生涯中，我还没有遇到过不害怕做让他们害怕的事情的人，这有一个科学的解释。我们的大脑被设计来保护我们免受任何对我们来说是危险的事情，所以每次你想做让你害怕的事情时，你的大脑都会以一种防御的方式行动。

这篇文章的目的是鼓励你面对你的恐惧，无论是作为开发人员还是其他任何角色，并向你展示恐惧并不危险。我知道这不是一件容易的事情:公开演讲，决定开始一些有挑战性的事情，让你走出舒适区，战胜失败。正如我之前提到的，所有这些恐惧都是后天习得的，因此，开始忘掉它们是一个好主意，好消息是你不仅会克服它们，还会获得最佳奖项，在帖子结束时，你最终会明白我在说什么。

每个人，包括我在内，都会觉得做简单或众所周知的事情很舒服，正如我上面提到的，你的大脑不会让任何痛苦和危险的事情发生，所以如果你做一件简单的事情对它有好处。

另一方面，我们中的大多数人不愿意做艰苦的事情，例如挑战自己在 X 时间内学习新的东西，并分享这一进展，以暴露你，当你知道你将被评判、批评或拒绝时表达观点。这是因为我们的世界将失败和犯错视为垃圾、愚蠢或愚蠢的人的同义词。

以上不真实。为什么？举一个很好的例子，如果我们回顾历史，看到托马斯·爱迪生发明的白炽灯，他在发明之前失败了 10000 次，我们中的任何人都会指出他是失败的，总结一下这一点，他说，“生活中的许多失败都是人们在放弃的时候没有意识到他们离成功有多近”。

在之前关于恐惧的大量介绍之后，我想谈谈你应该知道的一些具体的点和技术，也适用于作为一个开发者面对你的恐惧。我真的不喜欢谈论我没有亲身经历过的事情，所以我的观点和意见有 80%是基于专家的观点或文章。

# 新的挑战，同样的恐惧

两个月前，我在一家领先的软件开发公司 plateado Studios 担任开发人员，开始了我人生和职业生涯中新的一页。第一天很棒也很有挑战性，因为我的一个新同事邀请我去玩电子游戏，我接受了，但是我很害怕，因为我从来没有学过玩电子游戏。我在城市之外长大，远离那些东西，所以我从来没有机会这样做，无论如何，重新专注于我之前的观点，我真的在努力按下手指碰到的任何视频游戏控制按钮。这只是举一个例子，说明有时恐惧和未知的事情是如何对我们不利的。就对他们说，我是来打败你的。

# 恐惧让你远离目标

你的恐惧巡视着你舒适区的边界——大卫·杜舍明

当你决定面对你的恐惧时，就有一个信号表明你正在跨越你的舒适区。开始时，你感到不知所措、害怕是正常的，但正如美国成功的企业家和演说家汤姆·比利尤所说，做一件事最重要的步骤是现在就开始，这意味着要学会如何拥抱和面对这些恐惧，一点一点地，这个想法是从你的舒适区跨越到不适区。这时你会意识到你是如何进步的，你不仅会克服你的恐惧，还会有一个故事来计算和激励其他人也这样做。当我们有能力用自己的故事激励他人时，我们会变得更好。

作为一名开发人员，我的目标是每天都成为一名新版本的开发人员，每天早上开始成为一名初学者，而不是因为我直到现在都不擅长开发；这个头衔我不能给自己，但也因为我不想失去学习和变得更好的渴望，所以，在一天结束时，我想成为从 1.0.1 版本到 1.0.2 版本的我，这不仅仅是我们在工作中所做的，实际上我是在笼统地说，这包括你从未完成的附带项目，开始做课程来教别人，支持别人，评论其他帖子，写技术帖子，或任何会挑战你的技术事情。对我来说，最大的问题是恐惧驱使我去实现根据我的经验和后来的技术文章写一些有帮助的故事的目标。即将推出！

恐惧让我们觉得我们没有为某事做好准备，没有人愿意冒险去做。你意识到并认为你的技能还不够，这种恶性循环永远不会结束，这是一个继续推迟你的伟大事业的陷阱，也是让你远离你的目标的陷阱。

雅虎前首席执行官玛丽莎·梅耶尔(Marissa Mayer)在接受美国有线电视新闻网(CNN)采访时说，“我总是做一些我没有准备好去做的事情”

# 自我意识——了解恐惧从何而来

*开药方，你得知道诊断！*

一路走来，我遇到了不同的挑战时刻，这些时刻教会了我如何变得越来越强大，成为一个更好的人，并认识到我现在是谁，未来我想成为谁。他们迫使我以不同的方式思考，分析我行为和反应的每一个部分，我比昨天更了解自己，这意味着这不是短跑而是马拉松。在开始的时候，我非常痛苦地面对恐惧，在观众面前展示我的观点和看法。他们有时仍然在那里，但我觉得处理他们并把他们送进地狱更舒服，因为我意识到他们来自哪里

如果你能够理解你恐惧的根源，我保证你能够战胜它们。关于这一点，我的意思是你需要了解自己，就问我在开始时提出的问题，我最大的恐惧是什么？他们从哪里来？它们在什么情况下出现？

例如，我不喜欢问其他开发人员，因为我认为这些问题很愚蠢。事实是，我失去了学习的机会，也失去了犯错并从中吸取教训的机会。所以我确定了它们的来源并克服了它们，这并不意味着我没有恐惧，这代表着一场胜利。

# 面对恐惧，保持活力

当你决定面对你的恐惧，那一刻你不是选择存在，你是选择生活，是当你意识到生活的滋味和滋味，活着的兴奋。当你面对它们时，你会经历不同的生理变化，神经系统会命令我们的身体释放肾上腺素和皮质醇，正如我之前提到的，它们会引发身体变化。这样做的好处是它们提醒我们还活着，它有助于我们保持动力、警觉和专注。

迫使自己面对恐惧，哪怕只是一点点，也会让你看到自己真正的潜力。可能你和我都生活在对我们开发技能的怀疑中，试图在我们作为开发人员所做的事情上成为完美主义者。当你试图在观众面前解决一个编码问题时，你是什么感觉？这个特殊的时刻既具有挑战性又令人兴奋，对此我称之为，这就是生活！

# 你不会被你的恐惧所定义

事实上，你有或多或少的恐惧并不意味着它们很少定义你的生活，它们并没有定义你到底是谁。我记得当我开始自学英语时，我设定了第一个大挑战，那是我一生中最大的挑战，90 天英语挑战我邀请你看一看，因为我的英语不是我的第一语言，我有一个目标，它克服了困难，我记得我很害怕。

现在我看到了它，当然，我看到了很多错误，但我也看到了自己的承诺，并以一种伟大的态度面对暴露自己的恐惧，我们需要留心并认为恐惧定义了我们的生活。

不幸的是，在某些文化中，当你想向其他人展示你的开发技能和工作、你的 web 应用程序想法、你的项目、你的想法时，就在你这样做之后，会出现一个人，他正等着你的第一个错误被批评和评判。这一定是因为我们不能像世界上其他人一样发现自己真正的天赋和成长。所以现在你不仅要与你的大脑斗争，还要与批评和评判引起的不好的感觉斗争。

# 攻击你的恐惧，与它们共舞

*“如果你跌倒，让它成为舞蹈的一部分”*

恐惧是不可避免的，要用一生的时间来忘记它们，所以当它们出现时，正面攻击它们，放松你的马，并邀请它们与你共舞。这不是一朝一夕就能实现的，当然，你必须像本文前面提到的那样一点一点地努力。

在这一点上，我想说服你在它们出现时接受它们，我会写一篇完整的帖子来了解我们如何系统地克服它们。

# 欣赏自己的勇气，在一场战斗后奖励自己

拥有勇气不是一件容易的事情，所以欣赏这种勇气，每天激励自己继续这样做，感觉很好你尝试过，这将成为你保持下去的动力。此外，在面对恐惧或赢得战斗后奖励自己，不要低估每一场战斗都有助于赢得战争。

# 如此等等，还有最佳奖项？

毕竟之前的事情。我将获得的最佳奖项是什么？。当恐惧飞走的时候，你会有力量和能力去面对更强烈的恐惧，那时你会更强大，更快乐，更确信无论你面前是什么，你都有能力去面对它们。

不要害怕展示你所知道的东西，作为一个开发人员，我认为总有人在等待那些知识，不管是基础的还是高级的。总之，我完全赞同“失败，失败，但失败得很快”的哲学。最终，你将从一个版本转向一个新的版本。

最好的奖励是你每次都会成为一个更好的版本。作为一个开发者，一个更好的人，一个更好的朋友，一个更好的爸爸，一个更好的专业人士，你还在等什么呢？
