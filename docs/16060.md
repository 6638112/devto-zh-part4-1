# 用软件修复世界

> 原文：<https://dev.to/jorinvo/fixing-the-world-with-software-5gk9>

当今世界面临着如此多的大问题。我们每天都在新闻中听到他们的消息。你想在这个世界上做些好事。你想做出改变。但是你喜欢构建软件。
软件开发人员能为*做些什么，让世界变得更美好*？

可能大多数人类在他们的时代都有大问题。以这样或那样的方式，我们可能总是为生存而战，为我们的存在而战。如今我们的问题是全球性的，这是独一无二的。我们面临的挑战从未如此复杂，作为我们信息驱动的社会的一部分，媒体不断提醒我们面临的问题。与此同时，我们的能力和行动范围从未如此之大。

如今，我主要看到三个明显的挑战:

1.  **保持全球人类社会为了防止不同人类群体之间的致命冲突。**即使在自由和多样化的社会中，我们也需要找到和平解决冲突的方法。现代战争已经发展到一个地步，任何错误的行动都可能使整个地球无法居住。
2.  寻找控制人类健康和预防流行病的方法。令人惊讶的是，我们对构成我们的生物知之甚少。一种病毒就能毁灭全人类。
3.  设法与我们的环境和谐相处。人类对地球的影响已经变得如此之大，以至于我们必须找到保持地球生态系统平衡的方法。我们需要找到可持续的方法来处理碳排放、空气污染、能源消耗、废物、水污染、食品生产和许多其他问题。

最可怕的是，这些问题似乎太大了，任何人都无法解决，甚至不知道从哪里开始。

对于实现复杂的目标，约翰·凯有一些很好的建议:

[https://www.youtube.com/embed/_BoAtYL3OWU](https://www.youtube.com/embed/_BoAtYL3OWU)

但是软件如何帮助解决这些问题呢？

这些问题似乎只有政治家和决策者、科学家和工程师、公司经理和记者才能解决。

我认为这是关键所在:

我不相信软件能改变世界。人类可以。我认为我们作为软件开发人员的角色是支持那些能够做出改变的人。我们构建工具和平台来帮助他人。

如果你正在开发软件，并且想要解决关键问题，问问你自己你在为谁开发软件。想一想你如何帮助他人。
