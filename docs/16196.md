# 如何平静地开始你的一天

> 原文：<https://dev.to/lucpattyn/how-to-start-your-day-calmly-1mdf>

1)早起

2)不要碰手机，也不要接任何电话。确保昨晚入睡前关闭了 WiFi/数据。

3)保持新鲜

4)去散步

4.1)进行几轮

4.2)徒手练习

4.3)冥想/祈祷任何有效的方法

(顺序可能从 4.1 到 4.3 有所不同)

5)花些时间和老年人在一起，最好是明智的

(就我而言，是一位上了年纪的路边茶摊老板)

6)先不要碰手机

7)读点书(也许是报纸)

8)准备好

8.1)淋浴

8.2)早餐

(顺序可能从 7 到 8.2 有所不同)

9)走出去

*   现在，您可以打开 WiFi/数据并接听电话了

对于不稳定的人来说，这很难在日常生活中遵循，所以我很难平静地开始一天:|
