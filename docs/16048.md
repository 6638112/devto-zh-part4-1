## [](#hey-folks)嗨乡亲们！

基于我之前建立多个 JavaScript 全栈架构的经验，我创建了自己的固执己见的极简样板:

[JavaScript 全栈入门套件](https://github.com/mi-kas/fullstack-starter-kit)🚀

它基于后端的 Node.js、Express、MongoDB、Mongoose & Swagger 和前端的 React(用 create-react-app 构建)等框架。

### [](#features)特性

*   带有 Express 的 Node.js 后端
*   Swagger API 规范和验证
*   MongoDB 数据库
*   反应前端
*   使用 Babel 的下一代 JavaScript 不可知设置
*   多集装箱码头设置
*   使用方便的 Makefile 命令实现简单的 Docker 编排
*   单一回购方法
