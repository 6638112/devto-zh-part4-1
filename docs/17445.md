# Dev.to 很酷

> 原文：<https://dev.to/triptych/dev-to-is-cool-7hg>

我最近经常看到[dev to](https://dev.to)在附近。我也在那里张贴了一些！我刚刚发现 Dev.to 可以“转贴”你自己网站的 RSS 提要(如果你想订阅，我的是这里的)。 [Write.as](http://write.as) 也可以自动发布到[乳齿象](http://mastodon.social)关注我的博客—>@[triptych @ write . as](mailto:triptych@write.as)或我自己的账号—>@[triptych @乳齿象. social](mailto:triptych@mastodon.social) 。

这个真的很厉害。RSS 已经存在很久了，虽然谷歌不再喜欢它(很难在 feed 中投放广告),但网络的其他部分可以并且**应该**仍然使用它！我很高兴像 Dev.to 这样的网站正在接受这一点，并允许我们所有人有更大的自由。

我每天发布一篇“开发博客”,从现在开始，我会再发布到 Dev.to！

Write.as 甚至自动将此发布到 Twitter 上。所以真的，如果你创建了一个 Write.as 账户，你可以让它成为你所有社交帖子的“主编辑”。
