> 原文：<https://dev.to/tumee/rich-text-editing-in-react-with-draft-js-course-3jmk>

我很高兴地宣布,“在 React with Draft.js 中编辑富文本”课程现已上线！这是我创建的在线课程。这是我最近一直在做的事情。本课程是 Draft.js 的基础课程，因此如果您还不熟悉 Draft.js 的基础知识，那么本课程就是为您准备的！

本课程涉及的几个主题是

*   如何将富文本编辑器组件添加到 React 应用程序中
*   如何在编辑器中添加粗体和下划线等功能
*   如何将富文本内容保存到数据库
*   如何从数据库加载富文本内容
*   如何编辑保存的内容
*   如何显示保存的富文本内容，例如 HTML
