这里有一些初学者友好的 Python 资源推荐给我。

## [](#free-college-courses)免费大学课程

[edX 介绍计算机科学和使用 Python 编程](https://www.edx.org/course/introduction-to-computer-science-and-programming-using-python-2)
这是一门 14-16 小时/周的课程，持续 9 周，由麻省理工学院提供。这门课程是免费的，但是你可以为分级考试和作业以及证书支付 75 美元。

[卡耐基梅隆大学编程基础](https://www.cs.cmu.edu/~112/schedule.html)
课程结束时，你将能够阅读、编写、设计和调试 Python 代码。

## [](#online-resources)在线资源

[现代 Python 3 训练营](https://www.udemy.com/the-modern-python3-bootcamp/)
该课程包括 29.5 小时的视频以及可下载的资源、超过 121 篇文章和 135 个编码练习。Udemy 课程定期打折，所以如果你有兴趣参加的话，请关注一下这个课程！

[真正的 Python](https://realpython.com/)
各种教程，让你从头开始学习 Python。

Traversy Media
的 Python 速成班在一个半小时内涵盖了 Python 基础知识。

## [](#books)书籍

Zed Shaw
通过阅读这本书，你将完成 52 个编码练习。它还包括一张 5 小时以上教学的 DVD。

阿尔·斯威加特
用 Python 自动化枯燥的工作你将学习如何编写程序，只需几分钟就能完成手工通常需要几个小时的工作。阅读前无需任何编码经验！

马克·皮尔格林
深入研究 Python 3 每一章都以一个真实的代码样本开始，并对其进行分解和解释。

## [](#streams)溪流

Brian Clark 和 Cecil Phillip
Brian 和 Cecil 每周三美国东部时间上午 11 点左右开始学习 Python，他们在那里记录他们学习 Python 的旅程。
