# 什么是机器学习？

> 原文:[https://dev.to/jay_tillu/what-is-machine-learning-3m4n](https://dev.to/jay_tillu/what-is-machine-learning-3m4n)

人工智能是一项非常广泛的技术，基本上有两个主要的子部分:

1.  机器学习
2.  深度学习

所以我们可以说，如果你想学习和理解人工智能，那么你应该拥有机器学习和深度学习的明确知识，因为人工智能在内部依赖于这两个主要的子部分。

### [](#machine-learning)机器学习

* * *

机器学习是人工智能的一种应用，它为计算机提供了自动学习和根据经验改进的能力，而不需要明确地编程。

[![](../Images/e9fb1631e510eb92d6d060977e9face2.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--hUhkMVQi--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2APAqzvCxPjpDN8RC9HQw45w.jpeg)

在机器学习中，我们显然专注于 ***如何创建这样一个机器，它可以通过观察数据*** (如图像、位置、物体、事实等)来自己学习一些东西。).

机器学习过程可以分四个阶段完成:

1.  获取关于要应用机器学习模型对象的大量数据。
2.  定义该对象的特性、特征和功能等。
3.  选择要使用哪个机器学习模型来获取过滤数据。
4.  通过机器学习模型传递整个数据(如图像、特征、功能、特性)并获得最终答案。

[![](../Images/890f074cb2617efe7dd294c2276240d9.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--rLafT2Zv--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2AH8utsyq-MugrtL9Ol_peFQ.jpeg)

让我们借助一个例子来理解以上四个阶段。如果我想让我的电脑识别这张照片是不是一只猫。那么让我们来看看这个东西是如何工作的:

1.  我将拍摄大量的照片，照片中有许多不同姿势、不同品种、不同颜色、不同情况的猫，等等。

2.  然后我会定义一只猫的特征。就像它的基本外观，它的眼睛形状，它的平均高度，它的功能等等。

3.  然后，我将选择我想要应用于数据以获得结果的机器学习模型。

4.  在这些最后的步骤中，我将把这些数据放到机器学习模型中，以获得最终的答案。

所以现在在应用了所有这些步骤之后，我的电脑就可以识别它是否是一只猫的图像了。所以现在你可能已经明白这是一个多么复杂的过程。这个刚好适用于一只猫，试想一下现实世界里有多少物体，有多少情境，有多少事情。所以我们可以说我们有很多工作要做。

谷歌镜头、谷歌地图、人脸解锁、基于人工智能的相机是机器学习应用的最新例子之一。

[![](../Images/3228bbec2d52e8dd657bbf077110f066.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--nX_b7wHz--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2ALlyic_isecP3wi2bLHFR5Q.jpeg)

今天，我们有一个巨大的数据集和机器学习库，用于创建更强大的机器学习模型。

但是，机器学习仍然需要更多的人类互动来完成他的工作。机器学习的主要目标是 ***创造出这样一个可以自我学习的系统*** 。所以我们需要找出机器学习的上述四个阶段自动化的方法！！计算机可以自动获取数据，理解其特征并对其进行过滤，选择正确的机器学习模型并应用它，自己理解一些东西，这就是未来的朋友…
