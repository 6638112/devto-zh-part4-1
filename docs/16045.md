# 视频:如何创建讲述故事的数据可视化

> 原文:[https://dev . to/hamatti/video-how-to-create-data-visualizations-that-tell-a-story-2n 25](https://dev.to/hamatti/video-how-to-create-data-visualizations-that-tell-a-story-2n25)

现在制作图表很容易。然而，仅仅扔进一些数据和选择一个图表类型不会飞得很远。William Cleveland 从 80 年代开始在图形感知和数据可视化领域的研究是一个很好的起点，可以帮助人们制作易于理解的图表，更好地向读者传达信息。

在我的演讲中，我将带您浏览克利夫兰论文《用于分析科学数据的图形感知和图形方法》的重要部分，并分享现实生活中一些好的和不好的可视化，以及数据可视化的最佳实践。
