# 现在，您可以在 start.spring.io 查看 springboot 生成的项目

> 原文:[https://dev . to/mehdishahdoost/now-you-can-take-a-peek-in-spring boot-generated-project-at-start-spring-io-55k 0](https://dev.to/mehdishahdoost/now-you-can-take-a-peek-in-springboot-generated-project-at-start-spring-io-55k0)

对于基于 spring 框架开发项目的开发人员来说，使用 spring 的 initializr 在线工具来生成项目的支架是非常常见的。这个工具在 spring framework 生态系统中棒极了。
[![](../Images/ae1080c5c16120c681be452ff62b71cf.png)T3】](https://res.cloudinary.com/practicaldev/image/fetch/s--zLlGW6Po--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/7kfbxddd1cqba7yaf73l.png)

有时我使用这个工具来为我现有的项目找到正确的 POM 配置，我只下载生成的 zip 文件并在本地提取它，然后检查 POM 文件或其中的其他文件以应用于我的项目。

现在在 spring initializr 的新版本中，这个在线工具提供了一个很棒的特性，无需下载就可以探索你的项目。
