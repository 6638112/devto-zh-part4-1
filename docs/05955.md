随着当今顶级域数量的增加，很难验证电子邮件。
使用 RegEx(正则表达式或正则表达式)，我们只能检查电子邮件中是否包含@以及@前后的任何文本。更重要的是发送确认电子邮件。

写这个函数的另一个较短的方法是:

```
const isEmail = text => /^.+@.+$/.test(text); 
```
