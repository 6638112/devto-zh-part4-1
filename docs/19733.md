我曾经想知道拥有单独的页面和单一的滚动站点有多重要。在了解 DOM 如何工作之后，我意识到页面更适合搜索引擎索引。

功能联系页面通常包括:

1.  访问者可以向您发送消息的表单
2.  电子邮件地址
3.  你的社交媒体处理
4.  你的工作地点(或者至少是你的国家)
5.  包括工作时间在内的必要信息
6.  任何其他形式的联系
