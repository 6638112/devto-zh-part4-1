# 找工作

> 原文：<https://dev.to/pickleat/finding-a-job-4335>

我很幸运地找到了我的第一份开发工作，当时我刚刚离开之前在咖啡行业的工作，头脑中没有任何具体的目标，我联系了一些熟人寻求一些指导，他们最终聘用我在他们的初创公司工作。我不是从发展开始的，但我已经慢慢地走过来了。环境促使我去找另一份工作，但我不知道从哪里开始。我认为这是我第一次真正的软件求职。

我很想看看其他人是如何在软件行业找到第一份工作的。也许对我这样的人会有一些智慧。

问题:
你是如何找到你的第一份开发工作的？
它是远程的、现场的还是其他的？
在你得到这份工作之前，你有什么经验/资历？
在你申请之前，你真的满足了所有列出的要求吗？

说出你的故事！我很想读它！
