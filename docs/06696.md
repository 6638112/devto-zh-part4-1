### [](#a-guide-to-submitting-your-idea-for-js-kongress-2020)提交 JS Kongress 2020 创意指南

这就是我们正在寻找的——通过[sessionize.com/js-kongress-2020:](https://sessionize.com/js-kongress-2020)提交的指南

JS Kongress 2020 的重点是**扩展 JS——推动极限:大规模平台、数据驱动的架构和现代 API**。

编程语言 JavaScript 比以往任何时候都更重要，并且在服务器和硬件、本地应用程序和游戏中无处不在。

随着**增长**，每个公司和项目都增加了规模扩张的痛苦。和缩放问题是困难的问题。

> “您负责哪个领域的缩放工作”？

我们希望发现**各种扩展领域**，并探索这些领域中的问题和优化:

*   如何**优化编码和架构**以实现扩展
*   JS 引擎是如何发展的，V8、SpiderMonkey、JavaScriptCore 中的性能优化方法是什么
*   如何在 React、Angular 或 Vue.js 中**缩放大规模前端**
*   如何**提高 Node.js 性能**
*   如何**扩展团队并与数百名工程师一起工作**
*   如何**水平扩展**以及如何使用**现代云基础设施**如 Kubernetes 进行扩展
*   如何用 JavaScript 构建大规模数据驱动的平台
*   如何处理 Javascript 服务上**巨大的并发和不断增加的负载**

> “跟踪”

**扩展影响每个人**:前端、后端、架构师和团队。让我们关注一下**解决方案**和**最佳实践**。从 4 个领域中选择 1 个，将您的演讲归入一个主题—如果涉及多个领域，请选择最合适的一个。

> 基本信息

我们也需要知道，你要详细谈些什么。因此，我们要求一个**标题**和一个**描述**。可以参考去年的日程举例(如[https://2019 . js-kong RESS . com/sessions/how-to-combine-event-sourcing-architecture-in-your-node-js-server/](https://2019.js-kongress.com/sessions/how-to-combine-event-sourcing-architecture-in-your-node-js-server/))。利用这些信息吸引人们参加你的会议。

> 面向参与者的深入信息

从您的会议中有哪些关键收获？与会者之后会知道什么，他们会获得哪些知识，这对哪种项目/问题有用？

> 计划委员会的深入信息

为什么你认为，你的演讲**符合 2020 年的焦点**？请解释您提交 JS Kongress 2020 演讲的动机。始终牢记座右铭**“攀登 JS——挑战极限”**。项目委员会将利用这些信息来了解你演讲的背景，也有可能评估你的演讲是否能与共同主题无缝衔接。

> 附加信息

我们正在寻找的进一步信息是您的**个人信息**，该信息将在计划上线后用于演讲者个人资料。讲座将由项目委员会以匿名方式进行评估，因此只能根据其内容进行评估。

此外，我们计划今年再次举办**研讨会**。请考虑在实践小组中进行演讲。你是否也准备好举办一场研讨会不会影响对讲座的评价。只会有少数的研讨会，无论哪种方式，不是每个演讲者都可以成为纯粹组织原因的研讨会培训师。

我们期待着创建一个**多元化的阵容**，尤其是我们鼓励第一次在**演讲的人**也通过征稿来提交论文(有一个请求支持的复选框)。所有会谈都将使用**英语**。此外，还将有一名德国手语翻译。

JS Kongress 最关注的是**社区**。因此，我们鼓励你**回顾其他演讲者的演讲/幻灯片。**作为一名演讲者，每一次试讲都会让你对自己的演讲和话题以及 25 分钟的发言时间感到更加自如。作为评论者，如果思路清晰，你为他们提供了客观见解的绝佳机会。此外，你将获得如何呈现、如何论证一系列思路的新想法，以及修辞动机的灵感。

–**退款政策:**我们会退还差旅费(详情见下文)，并在演讲者酒店住三晚。一旦你被录取，我们可以退款或者给你一张免费票的优惠券。

最多可享受三晚酒店住宿。报销差旅费的上限:— DACH 地区:150 欧元—欧洲境内:最高 300 欧元—欧洲境外:最高 900 欧元—大洋洲:1200 欧元

–所有投稿必须使用**英语**。

–会谈计划持续 **25 分钟，没有提问&和个人介绍**。介绍将在讲座开始前与司仪一起完成。

–我们的 **CFP 评选过程是不公开的**，这意味着所有的论文直到评选过程的最后一刻都是匿名的。我们的计划委员会将对您的提案进行排名和评论，该委员会由 IT 领域的专家和社区人士组成。

–请注意，提交您的想法即表示您同意遵守 JS kong RESS【https://js-kongress.com/code-of-conduct/[的**行为准则**](https://js-kongress.com/code-of-conduct/)

如果你**有进一步的问题**，你发现把你的想法输入工具，请不要犹豫，随时通过[【hello@js-kongress.com】](mailto:hello@js-kongress.com)以及社交媒体**联系我们。请不要担心保存您的草稿，以便以后能够继续工作。在 2019 年 10 月 15 日论文征集截止之前，可以更改您的参赛作品。只有这样，项目委员会才会开始评估。**
