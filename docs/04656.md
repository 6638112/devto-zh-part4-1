以下是更长的功能列表:

*   管理人员
    *   登记
    *   登录/注销
*   管理组织
    *   用户可以创建组织
    *   一个组织有一个名字，可以有一个标志和自定义的 CSS
    *   管理人员的访问权限
*   管理项目
    *   用户可以在选定的组织中创建项目
    *   项目有标题和描述
    *   可以存档/恢复
    *   管理人员的访问权限
*   管理文档
    *   用户可以在选定的项目中创建文档
    *   文档有标题、版本和描述
    *   查看呈现的文档
    *   发布文档
    *   查看发布的文档
    *   下载 pdf 格式的文件
    *   管理人员的访问权限
*   管理文件
    *   用户可以创建降价文件或将降价文件、图像或其他资源上传到选定的文档
    *   管理文档中文件的顺序
    *   编辑降价文件
