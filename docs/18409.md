# Jupyter Key shortcuts

> 原文:[https://dev.to/mraza007/jupyter-key-shortcuts-48j6](https://dev.to/mraza007/jupyter-key-shortcuts-48j6)

有一些 jupyter 快捷键/技巧可以帮助开发人员保持更高的工作效率。

*   运行单元`Shift + enter`或`ctrl + enter`。
*   jupiter 笔记本还提供制表符补全功能

[![image](../Images/8f3619a8213a8298421ad6e813f6e37d.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--ua2oal_U--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/http://i68.tinypic.com/w9cn7a.jpg)

*   你也可以通过点击三次`shft + tab`在`jupyter-notebook`内获得文档，或者你可以通过点击函数末尾的`?`获得`docs`。

[![image](../Images/b8f7a456f9cc9e000b478f94834f1c01.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Qr2CFfeA--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/http://i67.tinypic.com/2lc62he.png)

*   `M`表示`markdown`模式，`Y`表示命令模式。
*   `%env`这是要打印的环境变量。
*   `%pdb`激活`pdb`调试器。
*   `%pylab`允许与`numpy`和`matplotlib`交互工作。
*   `%%debug`激活单元中的调试模式。
*   `%%html`将单元格呈现为 html。
*   `%%latex`将细胞渲染为乳胶。
*   `%%sh`外壳脚本。
*   `%%time`时间命令，查看运行命令花了多长时间。
