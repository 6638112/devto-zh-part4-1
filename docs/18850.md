# 你开发 Dapp 遇到的最大问题是什么？

> 原文:[https://dev . to/Anderson chenog/what-s-the-largest-problem-you-occurs-developing-dapp-ILD](https://dev.to/andersonchenog/what-s-the-biggest-problem-you-encounter-when-developing-dapp-ild)

我想知道 Dapp(去中心化应用)开发者面临的主要问题是什么。我为 Dapp 开发了 crypto wallet app，希望能为他们做点什么。我已经收集了一些:

1.  支持多种钱包(MetaMask，Ledger，Trezor，Mobile Wallets，Wallet Connect 等。)
2.  要求用户安装钱包
3.  自己管理用户钱包(我看到一些 Dapp 不喜欢使用第三方钱包)
4.  支持不同的链(以太坊，EOS，TRON 等。)
5.  测试和部署智能合同
6.  其他的
