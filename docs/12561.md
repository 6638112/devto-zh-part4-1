# 超级星系际帮派中的时间扭曲(还有一般的电子游戏？)

> 原文:[https://dev . to/Martin _ cerdeira/time-distort-in-super-galactic-gang-in-video-games-in-general-1gb](https://dev.to/martin_cerdeira/time-distort-in-super-intergalactic-gang-and-in-video-games-in-general-1hgb)

当我开始开发 SIG 时，我只清楚几件事:

*   这将是一场枪战。
*   它将有一个慢动作力学，好像它是近地天体和著名的“子弹时间”。

*视频演示*
[![IMAGE ALT TEXT HERE](../Images/89d94fb3e8cfdbf5f3350582ff29c905.png)](https://www.youtube.com/watch?v=4zkn1-7K1c0)

我们通常习惯于以慢动作观看事物，以此来重复高速发生的事件，并能够清晰地看到它。在运动、打斗或车祸等快速动作中，以比正常速度慢的速度看事物有助于更好地欣赏发生的事情。

例如，看到梅西以慢动作比赛是非常有吸引力的，并且注意到他似乎比他的对手受这种时间扭曲的影响更小，即:他比对手移动得相对更快。

当然不是这样，只有他更大的速度和实时的技能才是达到这种效果的。

因此，由于时间扭曲是我游戏中的主要内容，它必须感觉良好和令人满意，让玩家感觉像一个伟大的技巧，可以躲避子弹，在敌人之间滑动，并射向他们的脸。

这就是为什么我清楚地知道，我不能仅仅把它实现为时间减速，它必须以特定的方式影响不同的对象，以便最大化效果。

我做的第一件事是在我的游戏中定义一个代表时间速度的全局变量:

`globlal TIME_SPEED = 1;`

[![](../Images/d3fd0c6517a82325a342ea6e8859a880.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--BHEO3DAE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/0ax8vp5yl3js3i9ly7v8.png)

也就是说，它被用作乘数，正常值为 1，例如，如果我想让一切都以一半的速度运行，值应该是 0.5。

然后，我在每一个与时间有关的过程中使用这个乘数。

例如:敌人以与时间相关的给定速度前进和移动，因为行进给定的距离，更少的时间意味着更快的速度。此外，在大多数情况下，他们的攻击是基于一个计时器，该计时器规定了射击或攻击的频率。对于 bosses，计时器参与改变状态，例如:

`passive >> attack 1 >> passive >> attack 2`

子弹的速度、动画的帧数、角色的移动也与时间有关。

随着全局变量到处增加，唯一剩下的事情就是改变它的值，在“时间扭曲模式”下一切都会自动变慢。

剩下的就是让玩家把它当成实时失真来体验，并利用它来为自己谋利。

让每件事都平稳地慢下来是很直观的，但是，通过实验和大量的测试，我确定的是，这只是一种缓慢的感觉。它不会给人一种力大无穷的感觉，只会让一切都变得更慢，甚至不会给人一种很大的优势，因为尽管。比如:子弹走得慢，人物也是。

在这里，我认为可以有成千上万种方法来解决这个问题，我不认为有一种方法是最佳的，但我认为我使用的方法非常有效，它是:

**敌人:**
所有的敌人、boss 和敌人子弹都一样受到慢动作的影响，使用值 0.3 作为乘数，即不到正常速度的一半。
这会影响你的移动速度，你的动画，你的射速，以及任何改变行为或对事件反应的计时器。
这意味着敌人“思考”比正常模式慢。

**玩家:**
人物的移动受到的影响比敌人小，使用值 0.5。
子弹的速度也会受到影响，但不会影响射速，射速与正常速度相同。
后者听起来有点奇怪，但通过这个实现的是玩家可以继续射击，子弹累积形成更密集的一排，当时间扭曲和加速时产生“矩阵”效果。

简而言之，在时间扭曲期间，玩家有更多的时间做出反应，感觉更加敏捷，并且能够在敌人的子弹之间滑动。间接来说，这就像有一个速度的提升和更多的射速，但在一个稍微更有趣和隐蔽的方式。

然后，为了强调时间扭曲的感觉，产生了一种鬼影轨迹，以延迟的方式跟随玩家的移动，并将 SFX 和音乐的音调改为一半。

我认为这种方法足够通用，可以在任何其他游戏或流派中使用，所以这就是为什么我认为分享这种思想和技术对任何试图实现类似目标的游戏开发者都是有用的。
