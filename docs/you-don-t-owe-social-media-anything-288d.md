# 你不欠社交媒体任何东西！

> 原文：<https://dev.to/rizwanjavaid/you-don-t-owe-social-media-anything-288d>

最近，我在社交媒体上休息了一段时间。我不想在任何地方张贴任何东西。我没有计划，我只是一时失态了。在远离 Twitter、LinkedIn、脸书和 Instagram 几天后，我感到一种平静笼罩了我。我能够专注于生活中重要的事情，而不是总是寻找下一张照片或帖子来分享，然后查看我的帖子收到了多少赞。我想是时候休息一下，想想我和社交媒体的关系了。我并不是说社交媒体都不好。它也有好的一面。我学到了新的想法、声音和观点，否则我不会学到，但负面影响是强烈的，可能是破坏性的。

离开社交媒体一段时间帮助我理解了它的力量。它消耗我们的时间、精力和行动。但不一定非要这样。

你不欠社交媒体任何东西。

你不欠它你的时间。

你不欠它你的能量。

你不需要关注它。

你不欠它你的健康。

你不欠它你的自尊。

你不欠它你的精神健康。

你不欠它你的隐私。

把你的时间、精力和注意力花在你爱的人身上，最重要的是先照顾好自己。
