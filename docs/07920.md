本教程解释了如何在 react 本地应用程序中创建验证码。如今，这种验证码被广泛用于表单验证。验证码是一种不可预测的挑战文本代码，以字母数字字符编写，用于区分病毒(垃圾邮件)和真实的人。验证码只能由人眼读取，并且只有人类才能回答正确的验证码。它将阻止垃圾邮件对网络应用程序中的登录、唱歌和评论页面的攻击。现在，验证码也用于移动应用程序，使其免受垃圾邮件机器人的攻击。
