# 使用 Vue.js 实现无服务器化

> 原文:[https://dev . to/JavaScript _ tuto/going-server less-with-vue-js-4k ek](https://dev.to/javascript_tuto/going-serverless-with-vue-js-4kek)

随着无服务器架构的兴起，开发人员有能力用更少的资源做更多的事情。他们现在可以专注于驱动其应用程序的业务逻辑，而不是花费时间和精力来部署和管理服务器。此外，无服务器函数允许划分你的应用程序，所以你的逻辑被隔离在一个函数中，可以自动伸缩，没有任何担心或忙乱。此外，您只需为您使用的东西付费！

在这个视频中，我们将研究在 VueJS 中用无服务器思维构建应用程序的策略。具体来说，我们将通过真实生活中的用例来了解我们如何依靠无服务器功能来完成 API 调用和与第三方服务通信的工作。
