大家好，我的一个读者几天前问我——你介意像你为 [Java](https://medium.com/javarevisited/10-articles-java-developers-should-read-this-week-7648edd18a22) 和[算法](https://medium.com/javarevisited/10-data-structure-and-algorithms-articles-programmer-should-read-this-week-585404a9403b)所做的那样，为 web 开发文章列出一个每周列表吗？嗯，我非常乐意研究*的*请求，因为我也喜欢阅读 Web 开发文章，所以对我来说，向你们推荐一些值得阅读的文章并不困难。

所以，我这里有几篇文章关注前端开发、 [Javascript](http://www.java67.com/2019/01/top-10-javascript-frameworks-and-libraries-for-web-developers.html) 、 [Web 开发](https://javarevisited.blogspot.com/2018/01/top-10-udemy-courses-for-java-and-web-developers.html)以及那些独角兽框架，如 [Angular](https://javarevisited.blogspot.com/2018/06/5-best-courses-to-learn-angular.html) 、 [React](https://hackernoon.com/the-2018-react-js-roadmap-4d0a43814c02) 和 Vue.js

> 顺便说一句，这个列表绝不是一个详尽的列表，我希望看到你们的一些行动。如果你有一篇关于 web 开发的文章，不管是在媒体内部还是媒体外部，请随时与我们分享。我很想读读它们。

为了不浪费你更多的时间，这里是我列出的 10 篇本周末值得一读的关于 Web 开发的文章:

### 【AP hinya Dechalert】成为一名高效的角度拓展师需要学会的 19 件事

从《角度的上帝》到《两个晚上学会反应 JS 》,在过去的几天里，我读了很多她的故事，这是我最喜欢的一个。如果你正在使用 Angular 或者正在学习 Angular，这是你的必读文章。

### 2021 年你应该知道的 7 个前端 JavaScript 趋势和工具[乔纳森·萨林](https://medium.com/@JonathanSaring)

到目前为止，你们中有多少人用过 Async 和 Await of JavaScript？你们中的许多人是对的，但我敢打赌，读完这篇文章后，你一定会学到一些新东西。我确实学到了。常见错误和陷阱部分是这篇文章最精彩的部分。

### [javinpaul](https://twitter.com/javinpaul)针对初学者的 GraphQL 前 5 名课程

我在 [Hackernoon](https://medium.com/@hackernoon) 的另一篇文章。GraphQL 真的成为了 Web 开发人员的重要工具，现在可能是学习 GraphQL 的好时机，否则就太晚了。本文包含了初学者和 web 开发人员学习 GraphQL 的一些最佳课程。

我已经分享了 2021 年学习 GraphQL 的五门优秀课程，但如果你问我最喜欢的，那肯定是现代 GraphQL 训练营 (有 Node.js 和 Apollo)，不是别人，正是[安德鲁·米德](https://medium.com/@andrewjmeade)，还记得他的 [Node JS 课程](https://click.linksynergy.com/deeplink?id=JVFxdTr9V80&mid=39197&murl=https%3A%2F%2Fwww.udemy.com%2Fthe-complete-nodejs-developer-course-2%2F)，Udemy 上的史诗课程之一。

### TDD 改变了我的人生[埃里克·艾略特](https://twitter.com/_ericelliott)

这是一个改变了许多开发人员生活的原则，我强烈推荐给每一个编码人员。第一次看起来令人畏惧，但是随着你的学习和经验的增长，TDD 不仅会节省大量的时间，而且会产生更高质量的代码。这篇文章是一个很棒的故事，我强烈建议每一个编码者阅读它。

并且，如果你热爱 TDD 或者想了解更多关于它的知识。我也强烈推荐经典的[测试驱动](http://www.amazon.com/Test-Driven-Acceptance-Java-Developers/dp/1932394850?tag=javamysqlanta-20)的书，我读了很多关于 TDD 和测试的书，但是没有人接近这本书。真是太神奇了。

如果想买，这里有链接: [**测试驱动:面向 Java 开发者的 TDD 和验收 TDD**](http://www.amazon.com/Test-Driven-Acceptance-Java-Developers/dp/1932394850?tag=javamysqlanta-20)
