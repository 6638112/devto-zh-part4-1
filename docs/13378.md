# 在 Youtube 上学习 jQuery 动画制作方法

> 原文:[https://dev . to/yog yogi/learn-jquery-animate-method-on-YouTube-4 dog](https://dev.to/yogyogi/learn-jquery-animate-method-on-youtube-4dog)

众所周知, [jQuery Animate](https://www.yogihosting.com/jquery-animate/) 方法用于在网页上轻松制作出色的动画效果。它也很容易学，几乎不会花你 15 分钟的时间。

## [](#syntax)语法

`$(selector).animate({params},speed,callback)`

## [](#example)举例

让我们看一个简单的例子，我将从左到右制作一个正方形的动画。

这里的代码是:
 `$("#div1").animate({ left: '250px' });` 
这里的“div1”是正方形，你可以使用普通的 CSS 向它提供任何大小或颜色。
