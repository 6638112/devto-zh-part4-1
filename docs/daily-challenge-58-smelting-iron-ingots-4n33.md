# 每日挑战#58 -熔炼铁锭

> 原文：<https://dev.to/thepracticaldev/daily-challenge-58-smelting-iron-ingots-4n33>

《我的世界》已经成为一个非常受欢迎的游戏。毫无疑问，许多阅读这篇文章的人都自己玩过或者看过别人玩过。在《我的世界》，收集到的矿石必须在熔炉中进行熔炼，以将资源转化为可用的形式。

每一锭花 *11 秒*生产。史蒂夫有以下几种燃料可以添加到熔炉中:

`Buckets of lava, each lasts 800 seconds`
`Blaze rod, each lasts 120 seconds`
`Coals, each lasts 80 seconds`
`Blocks of Wood, each lasts 15 seconds`
T4】

编写一个函数，计算生产一定数量的铁锭所需的最小燃料量。

Ruby: `{:lava => 2, :blaze_rod => 1, :coal => 1, :wood => 0, :stick => 0}`
JavaScript: `{lava: 2, blazeRod: 1, coal: 1, wood: 0, stick: 0}`
