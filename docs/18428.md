主文章:**增强浏览**

此列表不完整；你可以通过扩展来帮助它。

*   Bookmarklets(JavaScript-last update:always working)可以在任何页面上执行任意 JavaScript，但需要用户点击它们，而不是自动运行
*   Firebug 允许对页面的 DOM 进行任意实时修改
*   **语法**(浏览器扩展-最后更新:2017)语法，抄袭和拼写检查，可以检索标记单词的定义和同义词
*   Greasekit 为 Safari (WebKit)提供用户脚本
*   **Greasemonkey** (火狐扩展-最后更新:2017)在网页内容显示后立即改变其输出
*   **iMacros** (浏览器扩展-最后更新:2016)允许用户记录和重放“互联网宏”，用于网络自动化、网络抓取或网络测试
*   **修道浏览器**(浏览器内最新更新:2019)用**修道浏览器** `Markdown-first`重建纯真网络
*   Proxomitron (代理级语言-最后更新时间:2003 年)自 20 世纪 90 年代后期开始使用，早于 JavaScript 主动浏览，并使用类似正则表达式的匹配语言为所有浏览器提供类似的功能
*   **时尚的**(火狐扩展-最后更新:2017)网页内容的客户端操作
