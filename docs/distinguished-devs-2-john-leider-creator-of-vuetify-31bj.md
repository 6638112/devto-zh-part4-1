# 杰出的开发者#2:约翰·雷德 Vuetify 的创造者

> 原文：<https://dev.to/bengineer/distinguished-devs-2-john-leider-creator-of-vuetify-31bj>

约翰·雷德是《虚拟世界》的创作者。

几年前，John 对快速构建原型和开发新应用程序的可用选项感到沮丧。他开始将 Vuetify 作为自己的一个副业项目，并且从未打算发布它。在一个朋友说服他将其作为开源项目后，Vuetify 已经成为 Vue 最受欢迎(可能是最受欢迎)的前端框架之一。

在这次采访中，John 谈到了学习、工作流程以及他是如何进入编程领域的。我们还讨论了 John 运行大型开源项目的方法，以及它与其他方法的不同之处。
