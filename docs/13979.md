# 如何平衡你的工作和生活

> 原文：<https://dev.to/mtdvio/balance-sustainability-227j>

最初发表于[我们的元开发博客](https://metadevelopment.io/blog/how-to-balance-your-work-life)。
如果你想从我们的社区获得新鲜的内容，你可以[为我们的时事通讯](https://digest.metadevelopment.io)歌唱！

在 Metadevelopment，我们非常关注你的个人生活和工作生活平衡的可持续性。我们不认为你的职业和生活是独立的实体。

你的工作影响着你的幸福和安康，就像你的一般生活方式一样。

我们关注的是**心态**和[行动](http://www.deconstructingexcellence.com/the-two-most-common-mistakes-of-optimizers/)。你可以做几件事来平衡你的生活。

1️⃣带着你的**愿望和需求**开始工作。
你需要一个参照点来衡量生活各个方面的重要性。问题是:你[想自己解决什么问题](https://markmanson.net/question)，什么问题不是必须的？

2️⃣开始弄清楚你的价值观、目标或使命。
至少，思考你的使命，尝试你的生活方式和工作来找到你的目标，会给你人生一个大致的方向和动力。从阅读一本名为[《高效能人士的 7 个习惯》](https://www.goodreads.com/book/show/36072.The_7_Habits_of_Highly_Effective_People)的书开始。

3️⃣学习一些关于幸福的基本知识。
选[这个课程](https://www.coursera.org/learn/the-science-of-well-being)强烈推荐！这是基于证据和当代的，不会花你太多时间。如果你想深入了解，请阅读[“为快乐而解决”](https://www.goodreads.com/book/show/30354426-solve-for-happy)。

4️⃣弄清楚自己与**控制**和**缺乏控制**的关系。
你无法控制生活中的很多事情:环境、天气、他人的想法。你甚至不能控制你的情绪，但你可以控制你的行为——你对情绪的反应。
斯多葛主义哲学是一个很好的长期思维模式的例子，这种思维模式基于缺乏控制和世界的不断变化。《美好生活指南》是一本很棒的书，概述了斯多葛派的原则以及如何在现代生活中应用。

5️⃣对自己的平衡负责。
一旦你意识到你最终要对发生在你身上的一切负责，你的生活就会开始改变。照顾好自己非常重要。

如果你不优先考虑你的生活，别人会的。
-本质主义。格雷格·麦克欧文

本质主义是一种很好的心态，开始掌控自己的生活并承担责任，找出正确的优先事项。承担责任是基于做出选择和接受后果。

6️⃣开始做一些实验和[挑战](https://medium.com/the-mission/how-to-be-incredible-comfort-zone-challenges-3e2b21d7d5b1)找出你的恐惧和**限制信念**。
你的拖延和理想主义与你的恐惧紧密相连。你的潜意识大脑基于你的恐惧向你隐藏了一些信息。你因为恐惧而错失了身边的机会。

7️⃣开始有意识地选择**动作**和**不动作**。
你需要承诺做某事，也需要承诺不做某事。事情是这样的——你已经在生活的某些方面选择了无为，你只是在不知不觉中做到了。

8️⃣开始根据你的价值观、承诺和愿望设计你个人独特的生活方式。
有意识地选择你的**抱负**并根据周围的现实设定正确的**期望**是可持续生活的关键。探索现实和提高你的心智模式是成长和自我实现的唯一务实态度。
