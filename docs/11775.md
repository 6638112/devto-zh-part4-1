# 第一天:向外看

> 原文:[https://dev.to/rahulchowdhury/day-1-looking-out-2eih](https://dev.to/rahulchowdhury/day-1-looking-out-2eih)

你好，

我叫 Rahul，在过去的三年半时间里，我一直是一名 Android 工程师，在印度的各个初创公司工作。

今天，我开始为同样的角色寻找新的工作机会。对我来说，要么是偏远的地方，要么是在 T2 的工作。

找一份新工作可能是一个令人精疲力尽的过程。长达数周至数月的面试过程和拒绝很容易让人疲惫不堪。

我去过那里，那里糟透了。

# [](#a-little-about-this-series)稍微介绍一下这个系列

这不是第一次有人公开他们的求职之旅。我主要读到的是那些成功找到工作的人的故事，以及他们是如何找到工作的。

我想让这次不一样。让这成为我旅途的现场记录。
