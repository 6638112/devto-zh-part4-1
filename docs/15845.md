### [](#why-replicate-this-template)为什么要复制这个模板？

我选择这个模板作为起点有几个原因:

*   很独特。
*   这可能不是一个你认为 Tailwind CSS 可以很好工作的页面。
*   在其他地方做一个预先制作好的模板意味着你可以像那些主要的竞争者一样做好创建一个网站的工作。
*   这看起来像一个有趣的挑战。
*   这是你也可以跟随我一起创造的东西。

这个版本是一个虚构的电影制作公司的单页登录页面。这里的目标是利用 Tailwind CSS，而不需要添加太多自定义 CSS。至少，我添加了一些自定义样式，为页面底部的循环视频背景图像和行动呼吁背景图像设置定义的高度。

乍一看，这些设计几乎完全相同，但我根据自己的偏好选择了一些替代设计。源代码非常简单明了，所以我邀请您加入并跟随我的脚步！
