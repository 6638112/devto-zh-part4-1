# 每日挑战#25 -双倍可乐

> 原文:[https://dev . to/the practical dev/daily-challenge-25-double-cola-4pm](https://dev.to/thepracticaldev/daily-challenge-25-double-cola-4mpm)

今天的[超凡脱俗的汽水机](https://www.codewars.com/kata/551dd1f424b7a4cdae0001f0)来自 CodeWars 上的 AlexIsHappy。

> 谢尔顿、伦纳德、佩妮、拉杰什和霍华德在排队买“双份可乐”饮料自动售货机；队列中没有其他人。第一个排队的人(谢尔顿)买一罐，喝了它，然后加倍！由此产生的两个谢尔登到队列的末尾。然后下一个排队的人(伦纳德)买了一罐，喝了它，然后以两个伦纳德的身份排到了队伍的最后，以此类推。
> 
> 写一个程序，返回喝第 n 杯可乐的人的名字。
> 
> 例如，Penny 喝了第三罐可乐，队列将是这样的:`Rajesh, Howard, Sheldon, Sheldon, Leonard, Leonard, Penny, Penny`
> 
> 输入数据将由一个数组组成，该数组至少包含一个名字和一个整数`n`。返回单行——喝第 n 罐可乐的人的名字。罐子从 1 开始编号。
> 
> `who_is_next(["Sheldon", "Leonard", "Penny", "Rajesh", "Howard"], 1) == "Sheldon"`
> `who_is_next(["Sheldon", "Leonard", "Penny", "Rajesh", "Howard"], 52) == "Penny"`
> T2】

多奇怪的队伍啊，这一个和其他挑战很不一样。祝你好运！
