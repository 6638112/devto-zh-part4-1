# 我建立了一个工具来帮助人们理解递归！

> 原文：<https://dev.to/zchtodd/i-built-a-tool-to-help-people-understand-recursion-1g0h>

递归是其中一个主题，我认为很多人从阅读到掌握它在实践中有一段艰难的时间。我自己知道学习递归的核心术语很容易，比如基本情况、前序、按序、后序等等，但是我仍然发现这个概念在实践中很难处理。

我们有时会说人们是视觉学习者，我相信我自己也是这样。我可以了解所有的概念，但如果我看不到它，这个想法就更难真正实现。

当我试图跟踪递归函数时，可视化是我的第一步，但是当函数很复杂，或者有太多的调用要跟踪时，这可能会很困难。

所以，为了满足自己的需求，我开发了一个小工具，可以在递归发生时画出草图。你写一个函数，工具会帮你画出来。让我们来看一些例子。希望它能帮助你递归点击！
