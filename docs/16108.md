#### 做一个(获取)获取请求

*   这应该发生在 componentDidMount()中

```
 componentDidMount() {
    fetch(your_url)
    .then(res => res.json())
    .then(data => {
      this.setState({your_state:data})
    })
  } 
```

#### 迭代

*   您仍然可以在 render()中使用循环

```
 render(){

    let listItems = this.props.myList.map(item => {
      return(
        <Item
          id={item.id}
          name={item.name} />
    )
  })
    return (
      <div>
        {listItems}
      </div>
    );
  } 
```
