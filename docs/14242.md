使用 Firebase 的 Ionic 社交登录

我们将把这个过程分成以下几个步骤。

1.  将 firebase 添加到您的 ionic 应用程序中。
2.  在 firebase 控制台中设置和 android 应用程序。
3.  启用 firebase 动态链接。
4.  在 firebase 控制台中启用 google 登录。
5.  在您的 cordova 项目中安装所需的插件。
