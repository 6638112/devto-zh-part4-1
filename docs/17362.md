# [](#welcome-to-the-6th-entry-of-this-series)欢迎来到本系列第 6 期词条！

和家人一起度假！！！这意味着我妻子委托我为她创作的东西，并没有进行太多的编码/学习。如果我迟到了，希望她能原谅我。:-D 我在利用一些安静的时间写这篇帖子。

本周发生的事情:

*   周六——在经历了两周的军事恶作剧后，开始享受与家人和朋友共度的时光。
*   星期天——父亲节！享受了一次不错的教堂礼拜，然后和我的女儿们一起依偎在沙发上度过了大半个下午。没有比这更好的方式来度过我的一天了。
*   周一到周三——做我的日常工作，花时间为假期做准备。周二，我开始通过一个朋友推荐的课程刷新自己对 flexbox 内部工作的认识，这个课程也是我在 [Let's Code](https://lets-code.info/) discord 服务器上的导师。

> [Flexbox 僵尸](https://mastery.games/p/flexbox-zombies)简直**牛逼！**主要目标是逐步学习 flexbox，同时在僵尸攻击中幸存下来。你应该认真检查一下。大概是我学习 flexbox 的新宠课程吧。

*   周四-前往佐治亚州泰比岛度假！会很有趣的！学习了更多 flexbox 课程，这很棒。
*   星期五——这是我们又一次每周综合报道的最后一天。我仍然在享受我的假期，我真的开始确定 flexbox 了。

感谢您抽出时间跟随我的旅程。我知道这个综述更像是我生活的更新，这也是为什么我更加欣赏你的观点。我期待着更经常地回到编程中来。必须把这些应用程序从纸上写下来，写进代码里。
