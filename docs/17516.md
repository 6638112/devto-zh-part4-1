# 代码区

> 原文:[https://dev.to/sensedeep/the-code-zone-2egg](https://dev.to/sensedeep/the-code-zone-2egg)

编写一个新的功能有时很有趣，有时很痛苦。但是我发现有一个区域编码变得更加高效和快速。让我解释一下。

在每天上下班、午餐、会议、咖啡和行政管理的普通日子里，你通常只有几个小时的有效工作时间。即使这样，这些时间也经常被打断。这一切都很好，除了编码和设计。这些干扰会打断你的注意力，让你脱离状态。

我发现如果我连续几个小时不间断地写代码，我的效率会提高。如果我这样做了一天，没有会议，没有分心，没有旅行，生产力继续建立，编码区开始增长。一旦进入这个区域，代码自由流动，简单优雅的解决方案似乎就出现了。

如果我持续这样做几天，那么我的潜意识就会启动。睡觉、开车、洗澡时，我的潜意识似乎一直在解决这个问题。我坚信我的潜意识是我大脑中更聪明的部分。当区域像这样工作时，就像在问题上有两个大脑。

有趣的是，最终的结果。当在区域中编码时，产生的代码更小、更干净并且更易于管理。

我确信，对于那些经常扼杀这一区域的公司来说，这是一个教训。
