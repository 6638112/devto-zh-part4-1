Kotlin 是一种很棒的语言，但我记得当我看到它没有三元运算符时，我真的很难过。我是一名 Java 开发人员，我经常使用这个操作符，因为我认为它写起来很快，读起来也很容易。

等等，等等，是的:我知道`if`表达式返回值，但是我觉得在主体只有一条指令的情况下，写一个`if`是多余的。

例如:

我宁愿写这段代码

