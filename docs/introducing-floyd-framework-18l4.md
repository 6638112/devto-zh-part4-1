# 弗洛伊德框架简介

> 原文：<https://dev.to/faraazahmad/introducing-floyd-framework-18l4>

**Floyd** 是一个构建 GraphQL 后端的框架，对 Typescript 的支持一流。

## 查看[这里](https://floyd-framework.gitbook.io/docs/)。

## 为什么

构建 REST APIs 是重复和枯燥的，对于前端开发人员来说更难处理。GraphQL 消除了开发人员在构建和使用 API 时面临的大部分问题。点击查看 GraphQL [的更多优势。](https://www.apollographql.com/why-graphql)

## 特性

### 1。零配置

你不必花时间搜索和下载软件包，然后设置它们，这样你就可以开始了。Floyd 将为您完成所有这些工作，这样您就可以立即投入运行并开始编写您的应用程序逻辑。

### 2。成分

Floyd 会将你的应用程序代码分割成独立的组件，这样逻辑就容易理解和维护。
