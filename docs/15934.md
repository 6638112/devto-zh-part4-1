# 修复 Python 缩进

> 原文：<https://dev.to/t4christ/fixing-python-indentation-2fp6>

在用 python 编写代码时，我们中的一些人有时会面临缩进的问题。奇怪的是，尽管您认为您在代码编辑器(通常是 visual studio 代码)中提供了正确的缩进，但您仍然会遇到缩进错误。这可能非常令人沮丧，因为我可以自信地告诉你，我已经经历过很多次了。好消息！我有一个解决你在用 python 编码时可能遇到的缩进问题的方法，这个方法一直有效，我是说一直有效。我已经试验过了，我可以自信地说它一直都有效。

你应该做的第一件事是选择所有的代码块，按 shift + tab 将所有的代码块向左对齐，然后按 tab 键你想向右缩进的次数。就这样，你的缩进问题得到了解决。对于多个代码块，重复上述步骤，按 shift + tab 将代码块向左对齐，并按 tab 键向右缩进所需的次数，这样就解决了缩进问题。
