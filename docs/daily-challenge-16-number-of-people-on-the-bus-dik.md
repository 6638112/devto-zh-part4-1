# 每日挑战#16 -公交车上的人数

> 原文：<https://dev.to/thepracticaldev/daily-challenge-16-number-of-people-on-the-bus-dik>

今天的[挑战](https://www.codewars.com/kata/5648b12ce68d9daa6b000099)来自 Codewars 上的雅利安-菲鲁齐安。在下面的场景中，你要算出公共汽车上有多少人:

> 城市里有一辆公共汽车在行驶，它在每个公共汽车站让一些人上下车。
> 
> 为您提供了一个整数数组(或元组)的列表(或数组)。每个整数数组都有两项，分别代表在一个公交车站上车的人数(第一项)和下车的人数(第二项)。
> 
> 你的任务是返回在最后一个公共汽车站后(在最后一个数组后)仍留在公共汽车上的人数。

祝你好运，并快乐编码！
