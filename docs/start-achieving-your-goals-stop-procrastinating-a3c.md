# 开始实现你的目标吧！停止拖延

> 原文：<https://dev.to/lvtdeveloper/start-achieving-your-goals-stop-procrastinating-a3c>

你有过脑子里有很多目标的那些日子吗？有你想做但还没开始的项目吗？
你的想法是从一个想法到另一个想法？
你是否有过这样的时刻，你想得太多，直到头疼，躺在床上看网飞，玩游戏，或者只是做一些与实现目标无关的事情？xD
当我们处于这种情况下，我们往往会感到压力太大，以至于我们的大脑拒绝一切产生压力的事情，所以我们最终会拖延。

## 当我们遇到这些情况时该怎么办？

首先，呼吸。然后，吃一顿丰盛的早餐，找一个你觉得舒服的地方，开始写一份清单，上面列有你想做的所有事情，列出你心中所有的活动、项目和目标。当你完成清单时，开始将它们分为短期和长期利益，例如，你想为客户或特定目标开发一个项目，但你也想学习一门引起你注意的新语言。第一个需要几个星期，但会带来直接的好处(短期的好处)，第二个会花你更少的时间，但好处现在看不到，因为你目前没有一个具体的项目(长期的好处)。

一旦你把清单分成两块，保留那张能在短期内让你受益的清单，这并不意味着你不打算做其他的事情，只是现在，你将把你所有的注意力集中在能在短期内让你受益的活动上。

记住这些目标，开始写一个实现这些目标的日常活动清单。把它们写下来是很重要的，因为这样你会更有动力和责任感，每次你完成一项活动，你就可以把它从你的清单上划掉，然后继续下一项。那种完成一项活动的感觉会让你感觉良好，会让你保持动力。

✔️，把你的想法整理好，明确你想做什么。✔️最好是在一天开始的时候，在丰盛的早餐之后。✔️写下你想要实现的目标和目的。✔️将这些目标分为短期和长期收益。
✔️保持短期目标。✔️列出实现每个目标的日常活动(你可以包括时间表)。
✔️开始执行每一项活动，尽自己最大的努力。
