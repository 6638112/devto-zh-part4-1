# 通过 AWS 认证改变我人生的旅程

> 原文:[https://dev . to/vel 12171/my-life-changing-journ-to-be-AWS-certified-2njn](https://dev.to/vel12171/my-life-changing-journey-to-being-aws-certified-2njn)

这是老生常谈；我绝对喜欢挑战自己。否则，这位前大型机开发人员、后来成为业务分析师的人怎么会获得 **2x AWS 认证，在撒哈拉以南非洲最大的银行之一担任云工程师**。
我的 AWS 认证之旅始于大约 3 年前。我的前雇主将云作为一种战略，并随后与 AWS 合作，这给我带来了一种“新的、闪亮的东西”，对我来说意义重大。云是数字化转型的核心，确保我不会落后让我非常兴奋。

**我是怎么学会 AWS 的？**
我前面提到过，我的前雇主采用了云。然而，我当时的工作并不涉及任何形式的云工程，所以我学到的很多东西都是自学的，而且大多是在工作时间之外完成的。
我学习 AWS 的早期是由无数个小时观看网络研讨会组成的——由于创新的速度，我仍然会观看很多网络研讨会，总会有新的功能或产品需要学习。我也花了很多时间阅读白皮书。我还开了一个 AWS 账户，开始尝试各种可用的服务。

我很快决定验证我认为我知道的东西，很快就和一位云专家签约了。我注册了解决方案架构师助理课程。我还预约了考试时间，以帮助我集中注意力。我第一次尝试就写作并通过了。那种成就感是无法用语言表达的，获得这个认证对我来说意味着一切。我很快就意识到，这并不是机会的突然保证。许多拒绝接踵而至——我想，一些非常令人沮丧的拒绝也只是为了让我坚强一点。

**云工程师的职业**
我已经下定决心要从事全职的云工程师职业——云迁移战略、云架构设计是我的主要兴趣领域。求职六个月后，我现在的雇主提供了一个机会。大约 4 个月前，我作为一名云工程师加入，主要关注我之前提到的主要兴趣领域。
在我开始新工作的两个月里，我决定追求开发者助理认证。我采用了与准备解决方案架构师助理相同的方法—我也写了并通过了。
我现在正在学习 SysOps Admin 助理，希望能尽快写下这篇文章。

**下一步是什么**
在过去的 3 年里，我一直在提升自己的技能，并在云中寻找机会。没有人能说他们已经到达，但是我为我已经走了这么远而自豪。我总是喜欢尽我所能帮助他人，所以我现在开始指导那些希望踏上自己的云之旅的人。
几个月前，我们在工作中举办了 AWS 游戏日，在那里我有机会指导一个团队——释放激情！:-)
