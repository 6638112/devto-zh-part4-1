这里有一些有趣的东西:**注释**列。也许我们可以使用该列来过滤并删除未使用的或**禁用的**快照？是的，我们可以！我用下面的脚本来做:

```
#!/bin/bash
set -eu

snap list --all | awk '/disabled/{print $1, $3}' |
    while read s r; do
        sudo snap remove "$s" --revision="$r"
    done 
```
