# 做一个 iOS 或者 Android 的 App 要多少钱？

> 原文：<https://dev.to/yalantis/how-much-does-it-cost-to-make-an-ios-or-android-app-4874>

对于一个创业公司来说，做一个 iOS 应用或者 Android 应用需要多少钱的问题？肯定会提出来。但答案至少需要对项目有一个粗略的估计。

想象一下，你想翻新你的整个公寓，你问你的承包商要花多少钱，却没有告诉他们你有八间卧室。在他们查看您的空间以确定工作范围之前，我敢打赌他们不会给你一个准确的估计。

如果没有估计，也很难想象 iOS 或 Android 开发者需要花费多少时间来开发一个应用程序。我们上一篇关于构建一个应用程序的成本的文章更全面地描述了应用程序开发的成本以及它所依赖的东西。

如果你想知道应用程序设计的成本，看看我们关于设计一个应用程序需要多少成本的文章。

在本文中，我们关注构建一个 iOS 应用程序和一个 Android 应用程序的平均成本，并考虑 iOS 和 Android 开发在时间和金钱方面是否有任何差异。
