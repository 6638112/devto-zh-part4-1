一个典型的例子是无意中允许公众访问 S3 的文件。

最近几起备受关注的数据泄露事件都是由松懈的 S3 安全措施造成的。

其他攻击使用受保护程度较低的服务的 AWS 凭证来下载文件，而这些服务本来就不应该访问 S3。
