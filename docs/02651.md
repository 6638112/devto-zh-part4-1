# 为什么我离开海湾去沼泽——一个开源多样性的故事

我离开了许多人认为梦寐以求的工作，在旧金山湾附近的甲骨文公司负责开发人员营销，加入了一家领先的 DevOps 初创公司，该公司认同青蛙和沼泽。以下是我在高科技云行业学到的东西，以及我为什么选择回归本源。

150 年前，在旧金山湾区，常见的景象是麋鹿、鹿和叉角羚混杂成群在山上吃草，而灰熊则在附近的河流和小溪里捕捞鲑鱼。现在，你更有可能看到云计算解决方案的广告牌和远处正在建造新科技大楼的起重机。鹿仍然存在，但已经被降级为从住宅花园偷花和西红柿，同时躲避匆忙的足球妈妈(像我妻子)。

与城市化如何给旧金山湾区的生物多样性带来压力类似，开源创业公司也在由云计算巨头主导的技术行业中艰难生存。他们通过在云中包装开源软件，在专业支持下吸引开发人员采用他们的基础设施，创造了一个成功的商业模式。如果亚马逊、微软、谷歌和甲骨文等大型云公司继续窃取开源软件，并剥夺积极开发和维护这种软件的公司的盈利机会，创新将局限于这些科技巨头认为有利可图的开源项目。

JFrog 是一家逆潮流而动的公司，它有着巨大的创新，并为开源做出了巨大的贡献。十多年来，他们一直在支持 DevOps 工作流(在开发人员甚至不知道工件和包管理为什么重要之前)，并通过收购 Shippable、Conan 和 Cloudmunch 来领导云原生革命，以构建统一的工件管理、安全和部署解决方案。然而，他们成功的秘密是开源投资和社区支持的文化。10 年前，在我成为 Java 摇滚明星之前，我是“沼泽”社区的一员，是早期 Artifactory 客户，这一直是 JFrog 今天成功的驱动力。

JFrog 正在加倍努力，为越来越多的语言社区提供免费的模块库，包括 Go ( [GoCenter](https://gocenter.io) )、Java (JCenter)和 C/C++ (Conan-Center)。此外，任何开源项目都可以利用云中的免费 Artifactory Pro 实例，这些实例涵盖了上述语言以及 JavaScript、Python、Ruby、R、Lua、Scala、Fortran、Objective C、Swift、。网等。有近 100 个开源项目，包括 Helm charts 和一个开放的命令行界面，他们正在[回馈社区](https://jfrog.com/open-source/)。最后，通过邀请一位高级开发者关系负责人(你真正的),这显示了他们对开源和社区投资的持续承诺。

因此，下次你想简单地使用你的云提供商为你方便捆绑的一键式服务时，想想带来创新的开源开发者，给他们一个打动你的机会。特别要感谢 MongoDB 的联合创作者[艾略特·霍洛维茨](https://twitter.com/eliothorowitz?lang=en)、[尼哈·纳尔克赫德](https://twitter.com/nehanarkhede)、【汇合】卡夫卡的联合创作者、[萨尔瓦托·桑菲利波](https://twitter.com/antirez)、Redis 的发明者、 [Mitchell Hashimoto](https://twitter.com/mitchellh) 、Terraform (Hashicorp)的创作者、 [Shay Banon](https://twitter.com/kimchy?lang=en) 、Elasticsearch (Elastic)的创作者、 [Yoav Landman](https://twitter.com/_yoav_?lang=en) 和[弗雷德·西蒙](https://twitter.com/freddy33?lang=en)[这些技术领导者都成功地创立了推动 DevOps 和云原生运动的公司，值得我和你们的支持。](https://twitter.com/freddy33?lang=en)

我很高兴加入这些开源领导者的行列，推动我们的行业向前发展。期待更多来自 JFrog 开发者博客的优秀开源和技术内容。
