# 我如何找到第一份工作的真实故事

> 原文:[https://dev . to/coding mind ly/the-true-story of-how-get my ' s first-job-5ee4](https://dev.to/codingmindfully/the-true-story-of-how-i-got-my-first-job-5ee4)

有一天，我在办公室向一些人讲述了这个故事——我们在谈论我们的编程之路。

即使我已经写了很长时间的代码，直到我必须得到我的第一份工作，我才知道我想成为一名开发人员。

(事实上，在得到这份工作后，我花了一段时间才意识到这是我想做的事情...但是我跑题了)。

我放弃了攻读物理学博士的想法(这个决定我至今仍感到高兴)，于 2001 年获得了高性能计算硕士学位。这有点像网络公司崩溃的中期，但是人们仍然雇用任何可以疯狂编程的人。

随着我的硕士课程接近尾声，我开始在谷歌上搜索软件公司的信息(当时谷歌可能只有 3 岁？Linkedin 关闭了)。

我找到了一个网站，它有点像爱尔兰软件业的分类区。产品、服务和公司广告的随机选择。

顶部似乎有一个登录按钮。我点击它，出现了一个表单。

我天真地尝试登录。

用户名:测试
密码:测试

....

你知道什么！我进去了！

我看到了一个允许我创建广告的界面。所以我做了！我登广告招聘一名毕业程序员。

两天后我接到一个电话。这是网站运营者的来信，他们想知道我是如何投放广告的...

他们很想知道答案！出于某种原因，他们想和我谈谈给我提供一份工作。

也许，知道我对他们的“安全”做了什么，我不应该接受。但是那是另一回事...

你呢？
