# REST 架构——第 1 部分:构建 API

> 原文：<https://dev.to/auth0/rest-architecture-part-1-building-the-api-37oi>

这是展示如何通过实际应用 REST 架构原则来构建客户机-服务器系统的两篇文章系列的第一篇。在这一部分中，您将学习如何通过为 Auth0 管理 API 实现一个简单的仪表板包装器来创建 RESTful API。在第二部分中，您将学习如何实现一个使用 API 的 RESTful 客户机。

为了构建应用程序，您将使用 Express.js 和 React.js，但是这些原则对任何语言和平台都有效。
