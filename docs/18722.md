# 一个非常简单的命令行工具，作为学习围棋的辅助项目。

> 原文:[https://dev . to/flrnd/a-very-simple-command-line-tool-as-a-side-project-to-learn-go-1 gif](https://dev.to/flrnd/a-very-simple-command-line-tool-as-a-side-project-to-learn-go-1gif)

### [](#the-story)故事

我已经用 [Larder](https://larder.io) 作为我的书签管理器大约一年半，或多或少。在 Larder 之前，我一直在使用 [Buku](https://github.com/jarun/Buku/) ( *一个用 Python3 和 sqlite3* 编写的强大的书签管理器——来自 Buku github 页面)。由于已经习惯了 Buku，larder 中我缺少的一个东西是直接从命令行进行搜索的 cli，既然我喜欢命令行工具，还有什么更好的理由在 go 中启动它呢？

### [](#starting-small)起点小

每次我学习新东西时，我喜欢找小项目，因为保持小想法有助于我专注于特定的任务。另一个从小处着手的原因是因为在合理的时间内完成项目更容易，而且一旦完成后感觉很棒，这也是继续下去的巨大动力。

### CLI

一个月前，larder-cli 就是这样诞生的，以 Buku 为灵感。cli 的当前状态离被称为完整的工具还差得很远，但这不是它的目的。在这个小软件中工作非常有趣，我打算添加更多的功能。我现在想学习的一件事是管理命令行参数的正确方法。

另一件很酷的事，我在 goreportcard 得了 A+！
