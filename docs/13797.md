# 本周 JavaScript:与 Peter Cooper 的对话

> 原文:[https://dev . to/deleteman 123/JavaScript 与 peter-cooper-4lh9 进行每周一次的对话](https://dev.to/deleteman123/javascript-of-the-week-a-conversation-with-peter-cooper-4lh9)

大家好，欢迎来到我称之为*“本周 JavaScript”*的新一期节目。在这篇文章中，我想介绍一个来自我们亲爱的 Node.js + JavaScript 社区的人，向他们提出一些有趣的问题，并为新来者提供一些建议。

希望展示这些有巨大影响力的人开始就像其他人一样，如果他们成功了，你也可以！

本周我们请来了《JavaScript 周刊》的编辑 Peter Cooper(在 Twitter 上也叫 T0 @ peterc T1)，他来自 https://cooperpress.com/的，这是一份我们都非常熟悉的出版物。

我们来看看他说了些什么，好吗？

## [](#1-tell-me-a-bit-about-yourself-hobbies-education-etc)1。告诉我一些关于你自己的情况(爱好、教育等)

我 16 岁离开学校，一直从事编程工作。

我有三个孩子，所以谁有时间搞业余爱好？哈哈。我经常玩 FIFA(足球视频游戏),只是为了确保除了编程之外，我还有别的话题可以聊..

## [](#2-how-old-were-you-when-you-started-programming-and-what-language-was-it)2。你几岁开始编程的？那是什么语言？

五点左右。我不记得了，但是我父母有照片！我第一次记得编程是在 6 岁左右，但那只是非常基本的屏幕打印，等等，没什么新奇的！

## [](#3-how-long-have-you-been-working-with-javascript)3。你从事 JavaScript 工作多久了？

我第一次在网上发布关于 JavaScript 的帖子是在 1996 年，23 年了？但是它时断时续，当然，我并不是一个专业的 JavaScript 开发人员。

## [](#4-what-got-you-started-with-it)4。是什么让你开始的？

我记性不好，但我最早的在线帖子都是关于给 HTML 表单添加额外功能的。基本验证之类的东西，在内置验证(随 HTML5 一起出现)之前的时代。那时候 JavaScript 真的只擅长这些，因为它真的很慢而且很有限。

## [](#5-if-you-could-redefine-the-language-what-would-you-change)5。如果可以重新定义语言，你会改变什么？

这将是有争议的，但是如果这是一个历史性的修订(而不是现在每个人都习惯于 JavaScript 的修订)，我会避免原型继承，用一些不太习惯的东西。

## [](#6-what-would-you-say-is-the-best-feature-of-javascript)6。你认为 JavaScript 最好的特性是什么？

可达性。我不太喜欢这种语言的许多特性，但是人们似乎发现它很容易学习和使用。

## [](#7-what-advice-would-you-give-to-someone-whos-just-starting-to-learn-jsnode)7。你会给刚开始学 JS/Node 的人什么建议？

订阅 JavaScript 周刊？哈哈。

## [](#8-any-particular-learning-resource-youd-like-to-recommend)8。你有什么特别想推荐的学习资源吗？

[MDN](https://developer.mozilla.org/en-US/) ，韦斯博斯的课程，永远的好书(比如阿克塞尔博士的那些)。

## [](#9-is-there-a-project-website-something-youd-like-to-promote-while-youre-at-it)9。有没有你想推广的项目/网站/东西？

这里没有给出回应，但是如果你还没有，我建议你去看看 JavaScript Weekly！
