听起来“dev”已经足够了，因为现在我觉得我离它还很远，但我将分享我的挣扎和我的一些建议，希望我能帮助至少一个挣扎中的程序员，也许在这个过程中帮助我自己。正如我之前提到的，我在本地和全球范围内都是开发社区的新手。我目前正在参加全日制课程，学习 React 的奇妙框架。上学之前，我对反应一无所知。当然，我知道 javaScript，但老实说，直到大约 2 年前，我一直认为 javaScript 是“黑客”用来窃取你的信息和加载病毒到你的电脑上。否则为什么‘IE’会把`props.script.allow === false`作为默认值呢？`I have a feeling that syntax maybe wrong but you get the point`。，然后我听说 facebook 创建的开发团队有了反应。哇，我觉得 facebook 太棒了。它非常容易使用，反应非常迅速，即使你的互联网连接有问题，它也总是可用的。“反应应该是轻而易举的”，我想，“来吧，让我们做一些反应”。第一天来了道具，类和组件，在一天结束时，我删除了我的 facebook，并认为“f$%^”Facebook 和趾高气扬的“#@$”开发团队。😜稍后我将讨论我与 React 的斗争。这超出了本文的范围。

JavaScript 和我有一种爱恨交加的关系，而我最讨厌的通常是我的应用程序无法呈现的时候。其他时候我喜欢它，我完全被它能做的事情惊呆了。当 js 做你想让它做的事情时，无论它是在屏幕上呈现“Hello World”还是 api 调用 nasa 获取当天的照片，这都是一种令人敬畏的感觉。如果你像我一样，想把你的电脑扔出窗外，你的代码会无法工作。你的控制台错误百出。JavaScript 基本上是叫你白痴，告诉你哪里做错了，但你还是不明白，还是想不明白。不要担心，即使是最有经验的程序员也会遇到这种情况，至少我是这么听说的。不要放弃，继续努力，如果你在学校的学习要求更多？s，如果你问的人，”如果没有答案，问问他们上面的人。和其他努力学习的学生一起建立自己的学习小组。成为社区的一员是学习编程最重要的一部分，因为还有其他人正在奋斗，已经奋斗了，或者只是知道这一切，他们非常愿意提供帮助。我花了大约两个月的时间才弄明白，尽管这是开学第一天告诉我的。😉如果你花时间读完这篇文章，再次感谢你。
