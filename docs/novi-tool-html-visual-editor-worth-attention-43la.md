# Novi Builder 是什么？

基本上，Novi Builder 是一个 HTML 拖放生成器。

正如你已经猜到的，这个工具旨在帮助你不用编码就能建立一个网站，这真的很酷。从我自己使用 Novi 会员的经验来看，我可以告诉你，这并不意味着你**不用编码就能建立一个功能齐全的网站**。

然而，这个工具确实使创建网站的过程更加容易和快捷。此外，拖放式构建器并不是 Novi 的全部功能。
