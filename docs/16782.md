# 如何赢得咨询面试

> 原文:[https://dev . to/dannycatalano/how-ace-your-stride-consulting-interview-2n4h](https://dev.to/dannycatalano/how-to-ace-your-stride-consulting-interview-2n4h)

无论你是主动还是被动地寻找新机会，求职过程都会充满压力。难道你不希望公司能帮助你成功找到一份工作吗？好了，不要再看了！

Stride Consulting 是一家位于纽约的敏捷软件开发咨询公司。我们曾与 NBC、Casper、Peloton、IBM、Codecademy、Warby Parker 和许多其他公司合作解决各种类型的技术问题。我们与语言无关的工程团队与敏捷方法和极限编程同吃、同呼吸、同睡眠。

你为什么想在这里工作？很简单——我们的工程文化融入了学习和成长，我们的员工享受敏捷开发和 XP 的效率和生产力，这里的员工很棒，我们的客户是纽约最受欢迎的公司之一！

碰巧的是，如果你是一名软件工程师，你很幸运——我们正在招人！去吧，看一看。你喜欢你所看到的吗？很好！我们喜欢你喜欢你所看到的。花一点时间申请——提交你的简历并回答两个简单的问题。

好奇，但不感兴趣？继续读。

我们希望我们所有的候选人不仅表现出色，而且感觉他们在每个阶段都尽了最大努力。所以，我们组织了一个资源来帮助。

以下是如何在 Stride 咨询公司赢得面试的方法:

#### [](#demonstrate-strides-five-core-values)展示 Stride 的五大核心价值观

我们雇佣优秀的人，从失败中吸取教训的优秀的人，愿意帮助他人，并实践诚实、谦逊和尊重。这些不仅转化为理想的团队成员，也转化为成功的现场客户顾问。

#### [](#read-this-article-on-phone-interview-etiquette)[阅读此条电话面试礼仪](%5Bhttps://www.linkedin.com/pulse/how-ace-your-next-phone-interview-julia-berchem/%5D)

如果你不知道，现在你知道了。

#### [](#know-your-agile-xp-and-tdd)了解你的敏捷、XP 和 TDD

这些实践是我们 Stride 工程文化的核心。当我们深入了解软件开发最佳实践时，面试过程有两个部分:在电话屏幕上和在集中的远程敏捷和咨询面试中。我们发现，候选人有时能够谈论他们使用这些最佳实践的经验，但不能理解它们为什么重要。“为什么”让你进入下一轮。

#### 确保你有故事

和每次面试一样，为开放式问题提供清晰的例子非常重要。回想一下你的经历，回答典型的面试问题，这些问题总是以“告诉我一个……的时候”开始，即使他们不是这样措辞，我们仍然希望你提供清晰简洁的答案。当我思考如何回答开放式问题时，我总是使用 PAR(问题-行动-结果)方法。

#### [](#get-ready-to-highlight-your-tech-skills)准备好彰显你的科技技能吧！

如果你通过了电话筛选，你将被发送一个简短的代码测试。别担心，这不是黑客排名计时评估。我们寻找干净的、经过良好测试的干代码。确保您花时间阅读所有文档，观看电子邮件中的视频，并展示您的技术最佳实践。我建议利用分配的时间来整理一些突出你工作的东西。每个级别的漫游者都在某种程度上编写代码，让我们看到你能做什么是很重要的。

#### [](#be-yourself-and-have-fun)做自己，玩得开心

这么说你已经到达现场了！准备好和我们一起度过整个工作日。典型的面试礼仪适用于此:准时，穿着舒适，带上你的笔记本电脑。无论是在去办公室的路上，在火车上，还是在出租车上，花点时间集中注意力。有些人通过听他们最喜欢的歌曲，重复他们最喜欢的咒语，或者只是深呼吸几次来做到这一点。最终，我们希望为你雇用你。做你自己，享受这个过程。这让我们在一起的时间更多地变成了交谈，而不是采访。
