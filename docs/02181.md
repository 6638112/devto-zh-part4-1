# Dart 中的条件句

> 原文:[https://dev.to/jay_tillu/conditionals-in-dart-35n9](https://dev.to/jay_tillu/conditionals-in-dart-35n9)

*   在编程中，有各种各样的情况需要在代码执行前检查条件。

*   有许多方法可以检查条件并执行代码。这完全取决于我们在什么条件下选择什么方式。

*   dart 中共有五种方法来检查条件:

1.  [if 语句](https://dev.to/jay_tillu/if-statement-in-dart-2k91)
2.  [if-else 语句](https://dev.to/jay_tillu/if-else-statement-in-dart-5396)
3.  嵌套 If 语句
4.  [else if 语句](https://dev.to/jay_tillu/else-if-ladder-statement-in-dart-15dl)
5.  [开关盒](https://dev.to/jay_tillu/switch-case-in-dart-1p7c)
