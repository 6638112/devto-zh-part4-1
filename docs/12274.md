# 你对 React 测试库有什么看法？

> 原文:[https://dev . to/kentcdodds/what-do-you-think-of-react-testing-library-1d1e](https://dev.to/kentcdodds/what-do-you-think-of-react-testing-library-1d1e)

大约一年半前，我写了 [React 测试库](https://testing-library.com/react)，因为我在测试工作室教书，我对酶不满意。我觉得有太多东西我必须告诉人们不要在酶中使用(就像 API 是某种雷区)。所以我创建了 React 测试库，我对它非常满意。

我很想听听别人的想法。你有机会使用其他的测试库家族成员吗，比如 DOM 测试库，Cypress 测试库，Vue 测试库等等？我也很想听听你对此的看法！
