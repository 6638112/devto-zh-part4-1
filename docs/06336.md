# Dev 家长，你是怎么做到的？

> 原文:[https://dev.to/sloan/dev-parents-how-do-you-do-it-2616](https://dev.to/sloan/dev-parents-how-do-you-do-it-2616)

这是一个不愿意透露姓名的会员发来的匿名帖子。请仔细考虑你的回答，因为这些通常是很难写的帖子。如果你想留下匿名评论，请发电子邮件至。

我感觉自己正处于职业生涯的黄金时期。我在工作中粉碎了它，并在公司中获得了更高的领导职位。我的演讲事业飞速发展，不断有人邀请我在会议上发言。鉴于这一切，我一直在考虑要孩子。我从未想过我想要孩子，但现在我在重新考虑。我今年 32 岁，我肯定想成为一个更年轻的妈妈，所以我认为机不可失，时不再来。

然而，我担心如果我现在做，我的事业会受到影响，我将没有时间工作和讲话。我也担心别人会如何看待我。如果人们不把我当回事。我热爱我目前的生活，但我想知道是否还有更多，我是否因为没有孩子而错过了。

做科技工作的家长们，尤其是边工作边说话的，你们是怎么做到的？你会推荐吗？有哪些利弊？请不要隐瞒，我需要诚实的真相，这样我才能做出最明智的决定。
