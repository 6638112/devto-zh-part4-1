**什么是 Anync Await 关键字？**
在 JavaScript 中，我们使用了**承诺**，承诺的代码看起来像异步代码。

通过使用 async await 关键字，我们可以使异步代码看起来像同步代码。

**定义:**
在标记为 async 的函数中，允许将 await 关键字放在返回承诺的表达式前面。当放置 await 关键字时，异步函数的执行被暂停，直到承诺被解决/拒绝。
