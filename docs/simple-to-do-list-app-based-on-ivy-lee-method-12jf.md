# 基于李锦梅方法的简单待办事项 app

> 原文：<https://dev.to/tuckerschreiber/simple-to-do-list-app-based-on-ivy-lee-method-12jf>

大家好，

今天我要分享一个我已经开发了一段时间的应用程序，叫做 Ivy，它的灵感来自于“李锦梅方法”。这并不完全是李锦梅方法，但它迫使你每天只关注六件事。

2014 年，我被诊断出患有广泛性焦虑症。那时我还在上学，我试图用两份兼职工作来管理我在大学的时间。这绝对是我生活中的一个混乱点，为了度过这一切，我不得不一天一天地对待事情——把我的精神和身体健康放在首位。我通过写详细的日志和待办事项清单做到了这一点。

时间快进一点，我发现自己从大学退学，去了两家科技公司工作，这是一次相当疯狂的经历。我尝试了很多不同的方法来管理我的一天并保持高效，但唯一真正坚持的是李锦梅方法。我开始在家里使用贴纸和白板，每天完成六件事情。

这就是艾薇的切入点。该应用程序目前只有 iOS，但我想在未来建立一个网络和 Android 应用程序。

请看看它，让我知道你的想法！
