### [](#weave-ignite-et-weave-footloose-dans-scaleway-quand-les-machines-virtuelles-se-prennent-pour-des-containers-et-inversement-)在 Scaleway 中穿着点燃式服装和脚镣:当虚拟机认为自己是容器时，反之亦然……

[![](../Images/77db6dae81ab44e0d0e70912502b3578.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--A4dFJ-KO--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/749/0%2AnaSE2FI_HbGQ-0dK.png)

Weave 在 Firecracker 的基础上设计了一个点火装置它可以作为一个码头来管理 runC 容器。Ignite 从 OCI 映像构建虚拟机映像，能够有效地管理多个虚拟机。

[![](../Images/47adefdd901c1545ca6db548f02caba8.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--0R9EsSTb--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/742/1%2A54k7ytMLIGcVxls87Y7KoQ.jpeg)

我们的想法是，点火会使火饼干 VM 看起来像坞站容器。使用的映像是 Docker 映像，但根映像文件系统不是在容器中运行，而是作为具有专用内核的实际虚拟机运行，而/sbin/init 则作为 PID 1 运行。

[鞭炮](https://firecracker-microvm.github.io/)

网络将自动建立，VM 将获得与主机上任何坞站容器相同的 IP 地址。虚拟机的构建和启动只需几秒钟。有了打火机，我们就可以马上开始使用萤火虫……

[weaveworks/ignite](https://github.com/weaveworks/ignite)
