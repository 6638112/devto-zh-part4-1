# 笔记本电脑编码要求

> 原文:[https://dev . to/Angela RAE 63/laptop-requirements-for-coding-3l0n](https://dev.to/angelarae63/laptop-requirements-for-coding-3l0n)

朋友们好！

我真的爱这个社区！这么多人回复了我的反馈请求，我非常感谢！不过，我还有一个请求....

在我的开发旅程中，我需要买一台适合编码的笔记本电脑。过去，我既有 Mac 电脑，也有 Windows 电脑。对我来说，他们各有所长。我从未使用过 Linux，但我渴望尝试一个更简单的发行版。

我很想听听大家对我买的任何东西的规格的看法。有你喜欢的品牌吗？在编码方面，你最喜欢什么操作系统？我需要多少存储空间？处理器速度会是一个因素吗？最后，如果你明天必须更换笔记本电脑，你会买什么？

我可能应该提到，自从我开始我的 webdev 课程以来，我一直使用我儿子的华硕 ROG 笔记本电脑，它工作得非常好，但我买不起 2600 美元。

到目前为止，你们没有误导我...迫不及待想看看你的想法！
