> ***什么是插件？***
> 插件是给现有软件增加特定特性或功能的软件。

在大多数情况下，插件被用来给现有软件增加一些功能，使其更加强大和有用。

Plug-in、add-in、add-on 或 extension 是插件的不同名称。

例如，你的 chrome 是浏览器，它的工作是帮助你浏览互联网。chrome 在这方面做得很好。

但是当你浏览互联网的时候，有太多的广告打扰你。所以，在这种情况下，你会去 chrome 网上商店下载广告拦截器。你的工作完成了。现在你可以享受无广告浏览。

在上面的例子中，你想要一个特定的功能，那就是一些可以阻止网站广告的软件。在这里，广告拦截器是一种插件，可以帮助您实现这些结果。chrome 是一个安装了插件(广告拦截器)的现有软件。

广告拦截器只是给 chrome 增加了一些功能。

##### [](#applications-support-plugins-for-two-main-reasons)应用程序支持插件有两个主要原因:

1.  **在平台上添加新功能—** 插件使第三方开发者能够轻松地在平台上添加新功能。

2.  **减小软件大小—** 插件有助于减小软件大小。您可以创建功能有限的软件，并通过商店提供各种插件。所以当用户需要时，他们可以下载。就像 Chrome 一样，Mozilla 也在这么做。

## [](#different-scenarios-of-plugin-usage)插件使用的不同场景

* * *

*   媒体播放器使用插件来支持不同的文件格式。

*   电子邮件客户端使用插件进行加密和解密。

*   图形软件使用插件来支持不同的文件格式和高级图像处理。

*   文本编辑器和 ide 使用插件来支持编程语言和一些常用的功能。

*   音频编辑器使用插件来处理和分析声音。

*   网络浏览器使用插件来增加新的特性和功能。
