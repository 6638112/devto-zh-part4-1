# 首先，你需要知道——绝对和相对长度单位👀

我相信你知道 CSS3 中的单位是什么，最流行的是 **px** ，我们用这个单位来声明字体、容器、图像、边框、边距等的长度。**像素**是一个**绝对**单位，因为**是固定的**，用这个表示的任何长度都会有一个精确的大小。由于 **px** 存在其他绝对单位:cm、mm、in、pt 和 pc *(但现在我们不关心这些)*

而相反的一个**相对长度**单位是**柔性**，因为它的值不是固定的，这个值被**引用**为其他值😵保持冷静，你会明白的。

最流行的相对单位是%和 **em** ，如果你已经使用过 **flex-box** 概念，也许你在你的元素中使用%来表示长度值为**宽度**或**高度**，并且这些有一个父元素的参考长度值。

# 那么……em 单位是什么？😐

与%单位不同的是，em 单位用于字体大小，其值是之前指定字体大小的最接近元素的引用

> **em** 是最接近的特定字体大小的相对单位。

# 去举例！❤️

```
<nav style = "font-size: 16px;">  <!-- inital size 16px --> 
    <ul style = "font-size: 1em;">  <!-- 1em = 16px -->
        <li style = "font-size: 2em;"> <!-- 1em = 16px, 2em = 32px -->
          <a href= "" style = "font-size: .5em;"></a> <!-- 1em = 32px, .5em = 16px -->
        </li>
    </ul>
</nav> 
```

1.  在我们的 **nav** 元素中，我们指定字体大小为 **16px** ，这是我们的**初始**大小。

2.  由于我们的**初始**大小是 16px，所以当我们在 **ul** 元素中声明一个 **1em** 的字号时，这个就取 16px 为 **1em** 。*(记住，em 单位取最近元素的字体大小值，在本例中是 nav)*

3.  在 **li** 元素中我们声明了一个 **2em** 的字号，正确！2em 的值是 **32px** ，爽！

4.  注意，在四行中，我们为我们的**声明了一个字体大小为 **.5em** 的**元素，对吗？但也许你现在在想，为什么字号值是 **16px** ？发生这种情况是因为在最靠近的**元素中声明的最后一个值是 **32px** *(不管是否用于 2em，现在 1em 的新值是 32px)* 那么，. 5em 等于 1em 的中间值，所以最终的字体大小是 **16px** 。**

# 什么时候应该使用 em 单位？👌

我们已经知道， **em** 单位是相对的，就像 flex-box 中的%,我们可以在进行响应设计时使用 em。

这是一个很好的实践，因为当我们进行响应式设计时，我们只需在 **px** 中声明主元素的**字体大小**，在 **em** 的子元素中声明，因此，当我们使设计适应不同的设备尺寸时，我们只需改变主元素的字体大小，子元素就会适应。
