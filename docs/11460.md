# 2019 年底前参加 5 场 JavaScript 会议

> 原文:[https://dev . to/wolk software/5-JavaScript-conferences-to-attend-2019-7cg](https://dev.to/wolksoftware/5-javascript-conferences-to-attend-before-the-end-of-2019-7cg)

现在我们中的许多人已经结束了假期，回到了工作岗位，我们已经开始计划如何在今年剩下的时间里保持最新和最新的技能。下面是 2019 年底前要参加的 5 个会议！

## [](#jsconf-us-us-august)JSConf 美国(美国，8 月)

[![](../Images/a715db4a7dbad1c9d7bef81e08ee8d87.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--eap79haz--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/bzsjxa441uzcw0crd8mn.png)

JSConf US 已经将 JavaScript 推出了它的舒适区，即网络浏览器，并进入了服务器、无人机、机器人和视频游戏的前沿。

“选择你自己的冒险”日是 JSConf 区别于其他会议的地方。它为你提供了一个非同寻常的机会，在一个新的不同的环境中与我们社区中最聪明的人交往。JSConf US 为那些想学习如何为物理世界编程的人提供了 NodeBots 和 NodeBoats 等物联网研讨会，为那些想远离计算机走到外面的人提供了冲浪课程或圣地亚哥动物园野生动物园之旅！

早鸟票起价 850 美元。您可以了解更多关于此次活动的信息，并在[https://2019.jsconf.us/](https://2019.jsconf.us/)领取门票。

## [](#jsdayie-ireland-september)JSDayIE(爱尔兰，9 月)

[![](../Images/ac47e0ca9744b35358c74a60f9a68af0.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--PbJK6nun--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/2gvpcjsk4w0gtgayff7d.png)

JSDayIE 2019 是一个为期一天的单轨技术会议，专门面向爱尔兰的 JavaScript 社区，有超过 450 名与会者和一些爱尔兰最好的 JavaScript 专业人士和组织。

JSDayIE 是爱尔兰的第一个 JavaScript 会议，将于 2019 年 9 月 20 日在都柏林道森街大厦的圆形房间举行。

Dev.to 社区成员可以享受 15%的折扣[在这里](https://ti.to/wolk-software-limited/jsdayie-2019/discount/Dev)。

早鸟票起价 116 欧元。您可以了解更多关于此次活动的信息，并在[https://www.jsday.org/](https://www.jsday.org/)领取门票。

## [](#tsconf-us-october)TsConf(美国，10 月)

[![](../Images/760c01109fc5a9cd965b1c6f08fec7e6.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--3e1ro9xi--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/fluy5utd95lp2vqws5st.png)

有史以来第一次会议完全聚焦于 TypeScript！除了典型的 TypeScript 讲座介绍之外，在 TsConf 上，您将花一天时间讨论该语言的最佳实践和高级功能，包括来自 TypeScript 团队主要成员的信息。一切都在一个非正式和有趣的环境中！

去年的 TSconf 演讲阵容震撼全场。在这次会议上，介绍性的话题退居次要位置，我们让 JavaScript 专家工程师来驾驶我们深入到类型脚本的领域。

早鸟票起价 350 美元。您可以了解更多关于此次活动的信息，并在[https://tsconf.io/](https://tsconf.io/)领取门票。

## [](#nodeconf-eu-ireland-november)欧盟节点会议(爱尔兰，11 月)

[![](../Images/42035b2edb6ec334d2556170051aa6e2.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--TejvZzMc--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ayt8l9kkxq3xi2rxm03t.jpg)

NodeConf EU 是欧洲重要的 Node.js 活动，为 Node.js 社区提供了一个论坛，该会议已经成为 Node 社区的传奇，这是有充分理由的。在每年的几天时间里，Node 的一些关键人物会聚集在爱尔兰的会议上庆祝 Node。在这个地方，你会找到一些关于 Node 主题的最好的演讲者和专家，讨论关于协作，以及成为 Node 产品的贡献者。

早鸟票起价 799 欧元。您可以了解更多关于此次活动的信息，并在[https://www.nodeconf.eu/](https://www.nodeconf.eu/)领取门票。

## [点(巴黎，12 月)](#dotjs-paris-december)

[![](../Images/4e125acca87909629f0e2adb3eedb383.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--masMvoPO--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/nqzl05kcu4xjmusv3jwv.jpg)

DotJS 将于 2019 年回归，这将是第一次为期 2 天的会议！

第一天将是关于前端开发(UI 框架，浏览器，...).第二天将讨论后端开发和 JavaScript 语言(Node.js、ES2019、编译器、...).

DotJS 2019 将继续邀请社区中最优秀、最聪明的黑客来探索新的视野。你可以期待一些惊喜&交 1400 多个新朋友！

早鸟票起价 499 欧元。您可以了解更多关于此次活动的信息，并在[https://www.dotjs.io/](https://www.dotjs.io/)领取门票。
