# 为什么安全自动化是值得的？

> 原文:[https://dev . to/wegotpop/why-it-worth-automating-security-2hof](https://dev.to/wegotpop/why-is-it-worthwhile-automating-security-2hof)

我有时想你可以把客户分成三类:

*   那些关心安全的人
*   那些想在安全上打勾的人
*   那些还不知道他们关心安全的人

好消息是，创建自动化的安全基础设施有助于这三者。

如果你的客户关心安全问题，他们想要证明你正在做你所说的事情，并且你定期检查和监控它。自动化提供了所有的报告和证明，而您无需为此做任何事情。

这些相同的报告允许你在方框中打勾。如果您没有这些信息，那么您将不得不通过手动流程来挖掘信息，并根据审查流程的要求临时提供这些信息。

之前，我们有一些自动化，我们现在这样做，这正是我要做的。寻找正确的信息可能会很痛苦，有时它会过时，或者会揭示出一个需要在我们做出反应之前解决的问题，因为我们没有足够频繁地回顾事情。

通常，所有这些都是在时间压力下发生的，因为安全审查与我们试图获得的销售机会有关(通常娱乐行业喜欢在截止日期前工作，讨厌做任何不必要的事情)。这让一切都变得不必要的紧张。

一旦你实现了自动化，我发现我们已经将一些潜在客户的审查过程缩短到了几天。我们可能会花更多的时间讨论云基础设施，而不是安全方面。

最后，那些还不关心安全性的客户怎么办？根据我的经验，这些客户乐于承担他们正在做的事情的风险，但当事情出错时，往往会发现他们确实对您提供的服务有安全预期，但未能表达出来，因为他们通常不理解技术是如何工作的。

好消息是自动化也帮助了他们，因为构建自动化的成本实际上都在设置中。从运营上来说，我们已经能够保护我们的客户和边际额外成本。你所需要的是一个真正关心安全的客户，你可以对他们所有人进行一步改进。
