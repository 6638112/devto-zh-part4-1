# 在云中运行企业构建

> 原文：<https://dev.to/tryexceptpass/running-enterprise-builds-in-the-cloud-4fjp>

有许多构建代码的解决方案，其中一些可以作为云服务使用，另一些运行在您自己的基础设施、私有云或以上所有的基础设施上。它们可以轻松创建定制管道，以及简单的测试和打包解决方案。有些甚至提供开源的功能受限的“社区版”来免费下载和运行。

以下是我在为依赖云服务的组织决定构建系统时需要考虑的重要方面的经验。最初的意图是包括各种在线解决方案的详细审查，但我决定留到下次。相反，我更多地是从企业的角度来说的，这比通常的“如何做”更接近大组织的现实。我们在这里讨论这样做的意义和可行性。
