# 学习如何用 coregym 用 java 编写你的第一个游戏

> 原文:[https://dev . to/codegym _ cc/learn-how-to-codegym-2220](https://dev.to/codegym_cc/learn-how-to-code-your-first-game-in-java-with-codegym-2220)

## 队长，我们有麻烦了！

大多数自学编程(或其他东西)的尝试都失败了。每一个自学成才的人都有可能因为各种原因失去动力。例如，一些编码主题可能看起来过于复杂和难以理解。或者，恰恰相反，你走得太容易了，以至于有人怀疑你走的方向是对的。评估你自己的成功是很困难的。怎么做？去哪里？此外，新手程序员经常会因为看不到他们工作的最终甚至中间结果而失去动力。

## 这里太暗了…我看不到我工作的结果！

这个问题不仅在学生中很典型，在职业程序员中也很典型。软件开发人员在大型团队中从事大型复杂项目，为一个庞大的程序编写一小段代码。这件作品的效果并不总是可见的，更多的时候，它只是迷失在那里，淹没在他们的同事制作的相同作品中。

编程学生的问题略有不同。大多数时候，他们研究一些课题，解决一些任务，以巩固脱离现实的知识，但总体上看不到他们学到了什么，他们实际上能做什么或不能做什么。其实结果和专业人士一样:看不到自己努力的结果。

## 是的，这的确是个问题…你该怎么办？

放弃编程学习不是一个好主意。更好的办法是用别的东西稀释。对你来说有趣的事情，结果将是可见的。

在大型外包和产品公司工作的程序员经常通过创建自己的项目来“娱乐”自己。我们为高级学员推荐同样的方法。大多数时候，新手开发者在经典的基础上编写小游戏，有时也会实现自己的想法。

但是编码菜鸟该怎么办，那些自己还没准备好做一个相对有趣的项目的人？现在在网上，你可以找到足够的分步说明。跟随他们，一个学生在更有经验的开发者的帮助下创建他们自己的项目。一个很棒的例子就是免费使用 [CodeGym 游戏版块](https://codegym.cc/projects/games)。在那里，你可以在专业人士的明确指导下创建自己的经典游戏版本。现在你可以试着写自己对 6 款游戏的解读:《扫雷》、《2048》、《贪吃蛇》、《太空入侵者》、《登月者》和《赛车》。

[![Alt Text](../Images/65a6c3e79bd9b889b816a9527203621d.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s---zaLsSK7--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ql9hp1ovdttw2wwq8457.png)

要编写这样的游戏，你只需要有 Java 的基础知识，这可以很快获得:关于类和对象的初步知识，基本的原始数据类型，字符串，循环和分支，数组和数组列表结构的初步知识。CodeGym 的作者建议完成[课程](https://codegym.cc/quests)的前 5 到 10 个级别来获得这些知识。如果你觉得不够自信，这里有一个有用的[理论](https://codegym.cc/groups/posts/140-games-section-on-codegym-useful-theory)，可以帮助你解决问题。

然而，CodeGym [游戏部分](https://codegym.cc/groups/posts/137-new-section-on-codegym--games)是主课的一部分，每个人都可以参加，不仅仅是 CodeGym 的学生。

每个游戏被分成 24 个子任务和一步一步的说明。首先，获得子任务和提示，然后编写代码。当您完成时，CodeGym 自动验证系统正在检查您的代码。如果一切正常，你进入下一步。

[![Alt Text](../Images/537c802ef53cb2161c1adcacd58b1338.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--tJcaMLdI--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/mu0la0bpahbuf5xzv6gs.png)

当你完成最后一步，你得到一个现成的游戏。你可以自由地将它发布在网站上，并与你的朋友分享或修改代码以改进它或提出一些原创的旧想法。所有的游戏都使用简单的 CodeGym 游戏引擎，你可以在这篇文章中读到。

CodeGym 游戏部分是一个很好的方式来编写你的第一个完整的游戏项目，修改它，并了解如何自己开发独立项目。在这样的练习之后，你准备好继续你的学习，并在不久的将来想到你自己的原创项目。
