# 我闹鬼的衣橱

> 原文：<https://dev.to/justzeros/my-haunted-closet-2jj2>

我本周在加州圣克鲁斯的海滨别墅的前室有一个小衣橱。

壁橱里没有电灯开关。相反，内部灯是在一个运动传感器上。

凌晨 2 点，我起来喝了点水，然后僵在原地。

我可以看到壁橱灯亮着，明亮地照射在门下。

谁或什么打开了灯？？？

房子里有人吗？

或者，一种动物？

或者，是鬼？

我仍然不知道——我没有足够的勇气开门。

*咳咳……*

我们每天都看到人们的行为在思想的紧闭的大门下“闪闪发光”。

很容易猜测他们内心深处正在发生什么。

有人早上的脱口秀迟到了吗？"他们对团队不忠诚。"

有人犯了草率的代码？“他们就是不在乎质量。”

有人在会议中保持沉默？"他们在工作中被检查。"

试着抓住你自己。

记住，你能看到的只是光，而不是门后发生的事情。

比我勇敢些——敲门。

问发生了什么事。

保持好奇——并付诸行动。

一个好的开始问题是，“请多告诉我一些。”

极品，马库斯
