其行为方式的原因是因为 ID 直接附加到 heading 元素上。这不是必需的。

这是一件小事，但它能让我的齿轮啮合，因为修理很容易:

当您想要创建命名锚点时，创建具有相对位置的辅助元素，并添加具有偏移绝对位置的锚点链接。像这样:

```
<div style='position: relative;'>
  <a name='foo' style='position: absolute; top: -20px;'>  
</div> 
```
