# 让 Android studio 看起来更酷。

> 原文:[https://dev . to/edkluivert/making-Android-studio-look-better-1 klc](https://dev.to/edkluivert/making-android-studio-look-better-1klc)

嘿伙计们！，今天我将向你展示如何让你的 Android 工作室看起来更漂亮。

第一步。打开 Android studio，进入设置。

[![Alt Text](../Images/5e46cd41834e3227e66c47045fcded13.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--HQ5hZvzs--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/uxzxgkc9azf0s9mv1wxf.png)

第二步。寻找插件，然后点击它。

[![Alt Text](../Images/f86cadc9416c4e0e06ad745b1b99a691.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Qfv6IFtU--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ddawyps6g3pcelmmaq8j.png)

第三步。你应该看到一个标签(市场，安装和更新)，点击市场和搜索“材料主题用户界面”。下载插件。

第四步。android studio 会要求重新启动，然后重新启动和插件将被激活，然后你可以在 Android studio 上选择你想要的主题，你也可以在外观设置(设置)上自定义它。

希望你喜欢？。那好吧。祝你有美好的一天....
