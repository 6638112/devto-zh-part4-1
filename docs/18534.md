# 六大业务理由您需要更加多元化的数据科学团队

引用美国第一位首席数据科学家 DJ Patil 的话来说，最成功的组织变革是基于数据科学的方法以及人工智能和人工智能的应用，作为一项团队运动。

希望扩展到这个领域的团队必须包括广泛的角色、技能和职责，而不仅仅是一个拥有计算机科学博士学位的人。

这对我们企业的影响:

1.  这消除了任何一个人交付“人工智能”的压力。

2.  它投资于员工学习能为他们的职业和团队带来价值的新技能，以最适合他们的方式学习。

3.  它需要团队合作开发先进的产品，这些产品可能会产生某种程度上的影响，而这种影响是单个视角所无法察觉的。如果你是一家全球性公司，你必须有全球性的代表权。

4.  它通过鼓励思维和问题解决的多样性来挑战如何“最好地”解决问题，从而降低了潜在的风险。

5.  它让一个团队的所有成员都沉浸于理解人工智能和人工智能能解决什么样的问题。这是公司采用人工智能和人工智能的*关键*，也是最难教授的:如何识别这些算法可以解决的问题类型，以及如何将一个问题框定为人工智能或人工智能实验，是理解如何构建、测试和设计这些系统的第一步。

6.  这迫使我们质疑我们召集谁来解决问题。部署中特定的算法变化会导致与某个功能交互的用户减少吗？你需要来自用户体验团队的人来帮助你理解你对问题的限制是如何影响你想要的结果的。
