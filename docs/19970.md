## [像素化你的图像](https://glitch.com/~pixelator)

当我们在像素艺术，尝试由[贾罗德 Overson](https://glitch.com/@jsoverson) 的 [pixelator](https://glitch.com/~pixelator) 。将图像拖放到应用程序中，它会将图像像素化。然后调整色调，分辨率，亮度和饱和度，以完善你的像素化杰作。

## [创建自己的古腾堡积木](https://glitch.com/~gutenberg-block-kit)

Wordpress 背后的团队 Automattic 创建了" [gutenberg-block-kit](https://glitch.com/~gutenberg-block-kit) " -这是为 Gutenberg block editor 构建模块的最快方式。

Gutenberg 是 Wordpress 中一个新奇的编辑器，它基于内容块，比如段落和引用或者嵌入的 YouTube 视频和推文，而不是自由格式的文本。它带有一堆预制的积木，但你也可以自己制作，这就是"[古腾堡积木套件](https://glitch.com/~gutenberg-block-kit)"的用武之地。重新混合项目并开发你的 block，然后下载你的 block 作为 WordPress 插件在你自己的网站上使用。

如果你是块开发的新手，有一本[手册](https://developer.wordpress.org/block-editor/)和[教程](https://developer.wordpress.org/block-editor/tutorials/)可以帮助你入门。

[https://glitch.com/embed/#!/embed/gutenberg-block-kit?previewSize=100&path=index.html](https://glitch.com/embed/#!/embed/gutenberg-block-kit?previewSize=100&path=index.html)
