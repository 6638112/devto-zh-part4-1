# 使用 Chromebook 进行 Web 开发

> 原文:[https://dev . to/Angela RAE 63/using-a-chrome book-for-web-development-32pc](https://dev.to/angelarae63/using-a-chromebook-for-web-development-32pc)

正如你们中的许多人所知道的，从今年五月开始，我就一直使用借来的笔记本电脑来学习编程课程和项目，并且一直在寻找一台自己的笔记本电脑！我写了一篇文章，请求专家就编码对[笔记本电脑的要求提供建议。你们都没有让人失望！关于我应该寻找什么以及每个人在他们的开发设置中使用什么，我有一些非常好的建议。但是，没有什么比来自](https://dev.to/angelarae63/laptop-requirements-for-coding-3l0n)[马修·克莱尔蒙特](https://dev.to/clairmont32)的评论更能激起我的兴趣了:

> 我主要从事安全工具开发，并参与了几个开源项目。也就是说，我在我的三星 Chromebook Plus 上做了大量的工作，因为当我通勤上班或坐在沙发上时，它有一个内置的 Linux 发行版。

你在“内置 Linux”上骗过了我我已经爱上了 Chromebook OS 的简单性和安全性，但是内置了 Linux？！那听起来好得令人难以置信！于是，我的研究开始了！

起初，我看到了很多关于使用面包丁从本质上越狱 Chromebook 的教程，这似乎对我没有任何吸引力。但是，随着我的进一步研究，我发现谷歌添加了内置功能，可以在使用独立于 Chrome 容器的容器的虚拟机上运行 Linux。太棒了。Tyler V. 写了一篇很好的文章，讲述了这是如何工作的[在这里。](https://dev.to/tvanblargan/chromebook-coding-kickoff-4g72)。
