# 拖延的隐藏的一面

通常，懒惰、拖延和缺乏决心的本质是我们的恐惧和限制信念。你不是真的懒惰的几率很高。

实现我们想要的事情的最大阻碍不是我们懒惰或者意志力薄弱或者没有时间。最大的阻碍是我们的恐惧和我们有限的信念。

### [](#beliefs)信仰

我们的信仰似乎影响了很多事情:

1.  信念影响[动机](https://www.stanforddaily.com/2010/10/25/willpower-can-be-an-unlimited-resource-study-says/)。
2.  信念影响压力。
3.  信念影响我们的自制力。
4.  信念影响我们学习和改变的潜力。

一个[自我限制的信念](http://www.stevepavlina.com/blog/2012/07/dissolving-limiting-beliefs/)是你认为是真的东西，它**限制了你的进步**。它可以表现为消极的自我对话或对世界的悲观看法。一些例子是:“*我赚不到更多，因为这个数字在我们的行业里是玻璃天花板*”、“*像我这样的人不可能成为企业家*”、“*我总是被告知我数学不好，因此我是个糟糕的程序员*”。

我们的大多数信念都是我们认为是事实的东西，不需要我们这边的任何主动验证。
好消息是**信念可以改变**。一本名为[习得性乐观](https://www.actionablebooks.com/en-ca/summaries/learned-optimism/)的书中有一个很好的解释，通过重塑你的信念来改变态度。另一个很好的来源是一本名为[感觉良好:新情绪疗法](https://www.goodreads.com/book/show/46674.Feeling_Good)的书，这是你自己开始[认知行为疗法](http://cbtsanfrancisco.com/cbt-is-for-hackers/)的最佳第一步。

### [](#fears)恐惧

我们的恐惧做着类似的事情:它们阻止我们做我们想做的事情。
恐惧的问题在于他们**撒谎**，并且他们**逃避**。

恐惧可能非常微妙，并可能导致看似不相关的行为——你看了一系列，而不是写一篇你思考了一整天的文章。为什么？**因为你怕这篇文章不完美。**人家不会喜欢的。还不够好。它甚至还没有写出来，但它已经不够好了！

我们的大脑在估计可能性和概率方面非常糟糕，尤其是在对手的情况下——当事情可能出错的时候。恐惧让我们厌恶风险，让我们错过完全可以利用的机会。

消除恐惧不仅能帮助你，还能帮助一个更大的实体——一个团队。

### 那你能做什么？

首先，有一个巨大但有争议的想法:**不要相信什么是真的，而是相信什么是有帮助的**。甚至迷信[也会有帮助](http://journals.sagepub.com/doi/abs/10.1177/0956797610372631)。
第二，故意迈出[小的非理想的步伐](https://practicingruby.com/articles/low-cost-approach-to-side-projects)而不是跳入巨大的雄心勃勃的项目。变化不会立即大量发生。实际上恰恰相反:你越努力去改变，阻力就越大。

第三，走出舒适区，直面你的恐惧。例如，接受[拒绝挑战](https://www.rejectiontherapy.com/100-days-of-rejection-therapy/)来面对你对拒绝的恐惧，看看它是什么感觉。

总的来说，开始质疑你的信念，通过**进行实验来验证它们**。

马克·曼森的《自我认知指南》是一个很好的起点。
