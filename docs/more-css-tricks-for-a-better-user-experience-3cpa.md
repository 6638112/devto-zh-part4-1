# 更多 CSS 技巧带来更好的用户体验

> 原文：<https://dev.to/vesikodi/more-css-tricks-for-a-better-user-experience-3cpa>

改善用户体验是当今网页设计的主要目标之一。随着新趋势的形成和对更好的 UX 的需求越来越高，网站开发人员需要设计一个不仅好看，而且在高负载下也能运行良好的网站。

还有一个事实是，良好的用户体验会影响网站的其他方面，包括数字营销的成功和 SEO 的表现。幸运的是，你可以做很多事情来提供更好的用户体验，从我们将要讨论的下面几个 CSS 技巧开始。

SVG 动画

给网页添加触觉动画是目前市场上最热门的趋势。动画元素在另一个层面上真正吸引了观众。当你与网站互动时，触觉动画会给你带来满足感。

更好的是，你不再需要依赖 gif 或者更糟的 Flash 动画来传递那种触觉。如今的 CSS 动画比以往任何时候都更加健壮，尤其是在与 SVG 结合使用时。您可以使用常规的 CSS 标签，如[过渡和变换](https://www.hongkiat.com/blog/svg-animations/)，甚至是关键帧动画，来制作页面上 SVG 元素的动画。

SVG 基于多行代码，让您对如何制作动画有更多的控制。例如，您可以提取 SVG 的特定部分，并独立于文件中的其他元素制作动画。可能性是无限的。

高级滚动效果

直到最近，干扰用户如何浏览你的网站还被认为是禁忌。虽然有一些有趣的框架提供了平滑滚动，但它们并不总是适用于所有浏览器，尤其可能会妨碍移动用户的体验。

然而今天，你可以用滚动做一些事情。是的，您仍然希望避免打乱用户滚动页面的速度和滚动本身的行为，但是您可以添加高级滚动效果来使您的页面变得生动。

WSI 数字网络公司是一家著名的网页设计公司，在公司网站上利用高级滚动效果方面做得非常好。你可以从公司过去的项目中看到，有很多有趣的效果可以通过滚动来触发。

使用选择器向下钻取

CSS3 还提供了新的方法来设计特定元素的样式以及全局添加样式。当您想要对整个站点的元素进行样式化时，*选择器尤其方便；例如，您可以使用*p 选择器来更改段落的显示方式。

子元素，尤其是列表中的子元素，可以使用:n-child(n)选择器来定位。您可以使用:before 和:after 将元素甚至内容添加到站点的特定部分。你还有@support，方便[检查浏览器支持](https://www.creativebloq.com/features/css-tricks-to-revolutionise-your-layouts)。

显而易见，CSS3 比以往更加健壮。总的来说，你有更多的选择器可以使用，更多的元素可以添加到你的站点，还有新的方法来改善用户体验——以及用户如何与你的站点交互。
