# 为什么我还在学基础？

> 原文：<https://dev.to/restroika/why-iam-still-learning-basic-2eah>

为什么？这是因为我的大脑需要它，它对我来说就像维生素，有点嫉妒新来的人，但我有很高的语言技能。Android 和几个 js 技术仍然是学习的首选，我知道怎么做，直到我在 office 上得到一些用 php 重写 web 应用程序的项目，我必须服从 Android 和朋友。

我是一个专注的工作者，讨厌知道一些客户在计划中改变技术，讨厌知道其他人在 js 生态系统的新来者中已经熟练。

我应该做什么？提醒自己要写我的 optims，看看官网上的教程或手册，为了什么？从基础学起！在我得到这个领域的一些工作之前，我应该需要建立自己的基础，然后给我一个项目😎
