# JavaScript、jQuery 和其他可以用 JavaScript 编写的东西之间的区别

> 原文：<https://dev.to/simomazil/difference-between-javascript-jquery-and-anything-else-you-can-write-in-javascript-5920>

2014 年，当我第一次开始用 HTML 和 CSS 构建 ui 时，我需要一些动态的东西，显然只有 JavaScript 可以完成这项工作！但是因为我需要在我的 ui 中构建一些动画，所以我学习了 jQuery 而不是 JavaScript Vanilla，并且很长一段时间我都认为我在使用 JavaScript，jQuery 就是 JavaScript！

所以那个帖子只会帮助像我一样困惑的人:)

JavaScript 是一种网络编程语言，这意味着如果你想在你的网页上建立一个 UX，JavaScript 是你可以使用的第一件东西或者最少的东西。

另一方面，jQuery 是一个 JavaScript 库，这意味着它是在 JavaScript 之上创建的，您应该只在需要的时候使用它。

如果您使用一些 JavaScript 框架，您可能不需要 jQuery，因为这些框架将处理几乎所有与 DOM 元素的交互。

当我使用像 Laravel 这样的 PHP 框架时，我经常使用 jQuery，任何与 UX 相关的东西我都只使用 jQuery，因为它快速易用，现在我使用 MeteorJS，我不再使用 jQuery，因为对于第一个 MeteorJS 来说，它处理事件和用户界面的渲染，甚至你可以使用 jQuery 来获取你的字段值，例如！如果您不想使用，普通的 JavaScript 会干净快速地完成这项工作。

## 总而言之

如果你像我一样是 web 开发的新手，最好先学习如何使用语言(JavaScript)而不是库(jQuery)。
