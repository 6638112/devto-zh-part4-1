# #VetsWhoCode 让生活更美好

> 原文:[https://dev.to/carlakroll7/vetswhocode-makes-be-better-4587](https://dev.to/carlakroll7/vetswhocode-makes-be-better-4587)

几年前，我在听播客。
那天的嘉宾是杰罗姆·哈达威，他在讨论#VetsWhoCode。一个非营利组织，希望帮助退伍军人学会成为前端开发人员，并有一个社区来讨论和成长。我是一名当过机械师的空军老兵，对代码一无所知。
当我离开军队时，我知道我想用计算机做些什么，但不确定那会是什么。当我发现#VetsWhoCode 时，我是一个小镇上有抱负的开发人员，心想，哇，这听起来是一个学习的好机会！我查了一下组织，当天就申请了。

课程很棒！我笑了，我学习了，我生气了，我兴奋了...然后又疯了。但最终，我发展了一些技能，帮助我在芝加哥成为一名成功的前端开发人员，并在 VWC 找到了一群理解我的人，我们真的一起工作，一起成长，比我想象的还要好。我两年前就结束了我的团队，但几乎没有一天我不去看看事情进展如何。我们中的许多人，包括我自己，仍然以某种方式为 VWC 做贡献。目前，我正在成为兽医学院的 CDO。

如果你是一名老兵，我会建议你申请成为这个社区的一员。工作量很大，但如果你投入时间和精力，这是值得的。如果你申请了，但第一次没有被录取，那就申请下一批。坚持有回报。
