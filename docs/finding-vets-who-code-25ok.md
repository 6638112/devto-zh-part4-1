# 寻找编码的兽医

> 原文：<https://dev.to/vetswhocode/finding-vets-who-code-25ok>

当我开始我的 Web 开发之旅时，就像许多其他人一样，从简单的谷歌搜索“如何成为一名 Web 开发人员”开始谷歌在大约半秒钟内立即返回了我的查询和数百万条结果。不久之后，我就知道我需要指导来驾驭各种语言、职业道路和资源。有军事背景让我去寻找和我有相同价值观和职业道德的人，这不可避免地让我去找编码的兽医。我提交了申请，完成了前期工作。就在我被这个项目录取后不久，但并不全是阳光和彩虹。

我做了这项工作，但事情就是不顺利。任何其他组织都会在他们背后给我下药，直到我通过并送我上路。但是兽医学院负责编码的领导让我坐下来，和我进行了一次坦率的谈话，那种真诚的、令人不快的谈话只会来自你身边的人，比如你多年的朋友或家人。基本上，我被告知，我正在做的工作没有削减它，我需要改变一些东西，改变得很快。

那次谈话一结束，我就又回到了工作中，学习、看视频、听业内人士的意见，最重要的是，我完成了这个项目，并获得了我的第一份开发工作，成为一名中级前端 Web 开发人员。

如果没有编码专家花时间不仅教我如何编码，还教我如何学习，我可能会一直停留在平庸的境界。
