# 每日挑战帖子#20 -数量检查

> 原文：<https://dev.to/thepracticaldev/daily-challenge-post-20-number-check-4782>

创建一个函数来检查一个数字的三个不同属性。

> 这个数是质数吗？这个数字是偶数吗？
> 数字是 10 的倍数吗？
> 
> 每个都应该返回 true 或 false，它们应该以数组的形式给出。
