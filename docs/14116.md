**排版**
CSS 有大量的文本属性，我记得当我开始学习它们的时候，我很难记住所有的功能。这是所有属性及其用途的列表。

**基于字体的属性:**
*font-family:允许您为您的文本选择字体
font-size:允许您设置字体的大小
font-style:允许您选择字体的样式(正常、倾斜、倾斜)
font-weight:允许您选择不同的字体粗细(正常、加粗、加粗、变浅)或带有数值(100、200、300… 900)
font-variant:允许您将段落设置为*

**基于文本的属性:**
*颜色:允许您选择字体的颜色，您可以通过命名颜色(蓝色、红色、橙色……)或使用十六进制代码(#0000ff、#ff0000、#ffa500)来完成..)
方向:允许您将文本的方向更改为从右向左，或从左向右(rtl，ltr)
字母间距:允许您选择文本中字母之间的距离(2px，4px，6px，-2px，-4px…)
行高:允许您选择文本中各行之间的距离。您可以选择使用将与字体大小相乘的数字、以像素为单位的固定高度或当前字体大小的百分比
文本对齐:允许您选择文本的对齐方式(左对齐、居中、右对齐、两端对齐)文本装饰:允许您选择文本的装饰方式(无、下划线、上划线、换行)
文本缩进:允许您选择具有固定值(10px、20px、50px …)或宽度百分比的文本中每一首行的缩进该属性有多个值。h-shadow v-shadow blur-radius color(2px 3px 2px # 000)
text-transform:允许您更改文本的形式(大写、小写、大写)
unicode-bidi:与“direction”属性一起使用，允许您设置或返回是否应覆盖文本以支持同一文档中的多种语言(normal、bidi-override)
vertical-align:允许您设置文本的垂直对齐方式(基线、sub、超级、顶部、文本顶部、中间、中间
空白:允许您选择如何处理文本中的空白(正常、nowrap、前置、前置行、前置换行)
单词间距:允许您选择单词之间的空格长度(10px、20px、50px …)*
