#### Shawn 你好，欢迎来到 React Advanced 的访谈！请分享你的故事。你是如何对 web 开发产生热情的？

我过去从事金融工作，基本上是一个“Excel 猴子”:用 Excel 编写金融模型，计算数字，试图根据数字做出决策。最终，手写公式变得太多了，所以我学习了 VBA。然后，我的电子表格变得太大，Excel 开始崩溃，所以我学习 Python 和 Haskell 来处理数字。

所有这些都是非正式的:在工作中学习之类的东西。最终，我意识到我最享受这份工作的编码部分；然而，我也是瓶颈——如果人们需要做一些分析，他们冲我大喊，我就按下按钮。我最终认为，如果我只学会了制作用户界面，我就可以脱离这个等式，开始编写人们会购买和使用的实际软件产品。这是我实现我想做网站开发的七年旅程。

#### 能否请你描述一下你以前在 Netlify 从事开发经验的工作经历？

我之前只做过一份开发工作，在纽约的一家大型对冲基金维护设计系统。很好，但不算伟大。2018 年，我开始更加认真地关注 React 社区，在 meetups 和 Twitter/Reddit 上变得活跃。我成为了/r/reactjs 的版主，并在八月被接受参加我的第一次会议演讲。我都是在空闲时间做的。最终，Netlify 注意到了我，他们正在寻找这种社区参与和反应的专业知识。

#### [](#how-did-you-get-involved-with-eggheadio-do-you-think-mentoring-and-teaching-is-your-ultimate%C2%A0calling)你是怎么和 egghead.io 扯上关系的？你认为指导和教学是你的终极使命吗？

我被 Joel 邀请了，他是 egghead.io 的创始人之一。我只是拿了一个我正在做的项目，把它变成了一门关于故事书、反应和打字的课程，并且做得非常好。我不确定教书是我的终极使命，因为它需要很大的耐心，但我喜欢做一点，egghead 是一个非常棒的地方。

#### [](#how-would-you-describe-your-involvement-in-open-source-how-many-projects-have-you-contributed-to%C2%A0)你会如何描述你参与开源的经历？你参与了多少项目？

开源很重要，因为它让我们可以免费学习，也大大降低了开发成本。我的第一个重大贡献是 React，我在 React 文档中记录了这个过程。我不知道我参与了多少项目。

现在对我来说重要的是我走得更深而不是更广。我还寻找没有其他人参与的项目(所以如果我不参与，什么都不会发生或者它会死掉)，而不是不需要我的项目(比如 React)。

#### React Typescript cheat sheet 背后是什么？为什么你觉得必须写它？为什么是 Typescript 而不是 JavaScript？

我觉得有必要做这个，因为我在学习工作用的打字稿，而且我觉得官方文档不能很好地满足我的需求。所以我就把我收集到的技巧列成了一张小抄，因为我发现自己一直在查找资料。最终，其他人开始投稿，现在它已经发展成一整套的备忘单。

我认为标语非常恰当:TypeScript 是可伸缩的 JavaScript。对 TS 的普遍批评是，它需要一个构建步骤，将来可能会被正式的 JS 类型所取代。对我来说，这些成本很小，而且收益远远大于成本。

如果 Airbnb 使用 TypeScript 的话，38%的生产错误是可以避免的。那些认为可以通过更多的测试来解决这个问题的人严重低估了 Airbnb 的测试文化，也低估了类型如何补充测试。2019 年，举证责任不再由 TypeScript 倡导者承担。

#### [](#are-you-studying-machine-learning-why-did-you-decide-to-study-the%C2%A0subject)你是学机器学习的吗？你为什么决定研究这个课题？

我确实在学习一些 ML 课程。我认为我的影响力因杠杆作用而增强。有许多形式的杠杆作用，但特别是软件，通过自动化和可伸缩性提供杠杆作用。这是非常有吸引力的利用。

我不打算成为一名专业的 ML 工程师，但我认为它在我一生中的预期重要性值得现在进行一些研究。特别是对计算机视觉([不合理有效](http://karpathy.github.io/2015/05/21/rnn-effectiveness/))和[生成对抗网络](https://thispersondoesnotexist.com/)感兴趣。此外，我正在通过 GATech OMSCS 进行，这将有助于我获得第二职业的一些正式资格。(完全不值钱除了移民官僚去查一个箱子。)

#### 如果你可以用三种方式来组织这个世界——没有匮乏、没有问题或没有规则——你会用哪种方式？

不稀缺。没有问题的生活会很无聊，没有规则的生活会很混乱。至少在没有匮乏的情况下，我们的问题将仅仅是“第一世界的问题”。但是没有孩子会挨饿。
