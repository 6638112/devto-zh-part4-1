嘿可爱的 **DEV** 社区！

我写这篇短文是为了向你展示我只用 HTML 和 CSS 建立的一个小网站😁耶！旧时光！

我一直想为骄傲月做一个项目，但我总是不能及时想到一些事情。直到这个月，我开始看到许多以骄傲为主题的品牌使用“错误的颜色”😱

所以我对这个课题做了一些研究。我说的“研究”是指我读了维基百科上关于[彩虹旗帜](https://en.wikipedia.org/wiki/Rainbow_flag)的文章，并意识到这并不那么容易。

通常人们认为 LGBTQ+ Pride 国旗需要有 7 种颜色:“彩虹的颜色”，但事实并非如此。当这种情况发生时，他们最终使用了合作主义运动的旗帜，而不是广为接受的(但不是唯一被认可的)骄傲旗。

我发现什么是正确的并没有最终的真理，但是在我自己思考了这个网站是否有意义之后，我最终决定它只是为了好玩，并开始了它。几天后，它就准备好了。

🥁 *鼓声* 🥁
