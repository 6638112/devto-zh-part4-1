### [](#5-run-simulator)5 运行模拟器

在这种情况下，我们将从`dCar`目录
运行模拟器

```
$ python donkey_gym/examples/reinforcement_learning/ddqn.py --sim=/Applications/donkey_sim.app/Contents/MacOS/donkey_sim 
```
