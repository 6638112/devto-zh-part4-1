# 我的第一次面试经历

> 原文：<https://dev.to/shamimahossai13/my-first-interview-experience-53i6>

差不多几个月后，我终于鼓起勇气去申请了，尽管我还远远不适合这个行业。我看到一篇关于一家初创公司的帖子，他们是唯一一家愿意雇佣拥有基本问题解决技能和基本编码知识的人的公司。以下是我的主要错误:

1.  我对自己抱有不切实际的高期望
2.  我变得紧张，忘记了我的基础知识，例如，不能区分 HTML 标签和属性
3.  我对实习生提出了不切实际的薪资期望
4.  我表现得不够自信，毫无准备，尽管在准备的前一天晚上我几乎没有睡觉

吸取的教训:

1.  不管你知道多少，练习控制你的神经

2.  在去面试之前，尽可能多地研究行业

3.  对薪水有现实的期望

4.  学会解决关于你缺乏的问题，并想办法让他们相信你能克服它们。

采访者非常乐于助人和理解，但我可以看到他们接近尾声时的肢体语言中的失望。我想这个机会最终并不属于我，但我很高兴有这样的经历，并希望很快找到我梦想中的工作。
