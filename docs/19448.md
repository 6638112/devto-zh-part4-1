# 2019 年谷歌代码之夏的前两周

> 原文：<https://dev.to/ravgeetdhillon/first-two-weeks-at-google-summer-of-code-2019-240f>

两周前，我还不确定这个项目要用的技术。我对这个项目中使用的一些工具完全不熟悉。但我支持自己，并能够完成这些事情。

到目前为止一切都很好。我学到了一些新东西，它们将成为这个项目的重要组成部分。

1.  液体
    *   液体构成了这个网站的基础。所有的条件和其他逻辑都是在 Liquid 的强大帮助下实现的。
2.  管道
    *   因为网站需要持续的集成和部署，Gitlab CI 是一个完美的工具。建设高效的管道将是这个网站的一项重要任务。

登录页面是该网站的中心，将提供各种其他资源的路径。我正在做一些新的部分，可能会删除/修改一些现有的部分。我找人画一些艺术品/插图，我需要在这个网站上。如果你能帮助这件事，请提出问题，我们将有一个健康的对话。一个[维基](https://wiki.gnome.org/Projects/GTK/WebsiteRedesign)也已经制作完成。关于这个项目的所有重要信息都在这里。我已经把 GTK 网站的原始项目放到了我的[工作区](https://gitlab.gnome.org/ravgeetdhillon/gtk-web)。该网站目前使用 Gitlab CI 部署，可以在[这里](https://ravgeetdhillon.pages.gitlab.gnome.org/gtk-web/)浏览。

如果您有任何建议或发现问题，请在此处[报告。](https://gitlab.gnome.org/ravgeetdhillon/gtk-web/issues)
