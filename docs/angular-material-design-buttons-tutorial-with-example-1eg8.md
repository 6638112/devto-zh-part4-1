# 带示例的角形材料设计按钮教程

> 原文：<https://dev.to/singhdigamber/angular-material-design-buttons-tutorial-with-example-1eg8>

贴[角材设计按钮教程举例](https://www.positronx.io/angular-material-design-buttons-tutorial-with-example/)最早出现在 [positronX.io](https://www.positronx.io) 上。

今天我们将学习有棱角的材质设计按钮，按钮指的是用户在网络和移动应用程序上采取的一个动作。我们通常在 web 和移动组件上放置按钮，比如弹出窗口、表单、卡片和工具栏。

Angular material design 官网提到的按钮有 7 种。主要按钮类型是由 Angular material design 提供的凸起按钮类型，当我们谈到高点击率(CTR)时，没有比材料设计中的凸起按钮更好的选项了。
