# 在你开始数据科学职业生涯的时候？需要方向吗？

> 原文：<https://dev.to/bettyes/at-the-start-of-your-data-science-career-need-directions-1hbp>

一些人，特别是来自学术界的人，向我询问如何开始数据科学的职业生涯。显然，有很多关于这个话题的在线资源，然而，这里是我对此的看法。如果这有帮助或者您有什么要补充的，请随时发送反馈。

首先，我想探索一下怎样才能成为一名优秀的数据科学家？当然，关于那个的博客多的是【1-3】。

**在我看来，必备技能包括:**良好的问题解决技能(找到解决方案，即使它并不完美)、项目管理技能(能够从头到尾看到一个项目)、与业务/非技术人员的出色沟通，以及能够在团队中很好地工作。后者可能是在硕士期间获得的，最有可能是在博士期间获得的。然而，有些人天生就有这些技能，而且可能在除了学位以外的其他职位上证明了这一点。如果你想提高你的一些软技能，看看这个[书籍推荐列表](https://medium.com/@jbyrnephd/the-importance-of-soft-skills-in-data-science-book-recommendations-3cbce1b11c67) [4]

关于基本的技术技能(再次是我个人的推荐):一门你熟悉的**编码语言**(一开始只关注一门。最常用的领域:Python 或 R)、扎实的**数学或统计知识**、关于要在**数据科学**中应用的技术的一般想法，以及一些在面试中展示的*示例项目*(如果你没有真实的业务项目，可以是另一个与业务环境有假想联系的数据科学项目)。作为开始，可以在 Kaggle 上找到一些很好的数据集，或者简单地使用谷歌的数据集搜索。但是还有很多其他公开的数据，比如这个调查伦敦持刀犯罪上升的视频。

## 编码知识:

我强烈建议定期练习你的编码技能(每周至少两天)。通过解决短任务来练习，例如在 [Codewars](http://www.codewars.com) 、 [DataCamp](https://www.datacamp.com/) 、[exercisesm](https://exercism.io/)等。特别有趣的是他们的指导语言，在那里你可以从有经验的程序员那里得到关于你的编码解决方案的反馈。

在 [Udacity](https://eu.udacity.com/) 或 [Coursera](https://www.coursera.org/) 上也有很多——通常是免费的——课程可以帮助你入门，例如:

[Python 简介——从 Python 3 开始](https://eu.udacity.com/course/introduction-to-python--ud1110)(免费)

[脸书 R - by 的数据分析](https://eu.udacity.com/course/data-analysis-with-r--ud651)(免费)

如果你陷入了困境，那么“stack overflow”是寻找答案的“绝对必须”,或者更好的是，通过张贴问题或试图帮助回答一些问题来贡献自己。编码挑战很有可能成为你数据科学工作面试的一部分。所以确保你有所准备。看看其他一些可能有帮助的文献，或者看看这个[有用的博客](https://towardsdatascience.com/notes-and-technical-questions-from-interviewing-as-a-data-scientist-in-2018-20e7e3ee4ab3)【6】提供一些问题和建议。

## 统计知识:

刷新你的统计知识。我的一个朋友推荐了一本书[‘Think Stats’](https://greenteapress.com/wp/think-stats-2e/)【7】，这本书很好地介绍了使用 python 编程的概率和统计。如果你选择 R 作为你的语言，你可以从[‘R 的统计学入门’](http://www.academia.dk/BiologiskAntropologi/Epidemiologi/PDF/Introductory_Statistics_with_R__2nd_ed.pdf)【8】开始。但是还有很多。同样，你也可以在 Udacity、Coursera 等网站上找到有用的课程。

## 数据科学知识:

为了提高您的数据科学知识，请查看 [Kaggle](https://www.kaggle.com/) 。他们不仅有很多免费的数据集可以玩，有可能参加一些数据科学挑战，而且他们还有很好的[入门教程](https://www.kaggle.com/getting-started/45536)。另一个免费的基于 R 的课程[‘统计学习’](https://lagunita.stanford.edu/courses/HumanitiesSciences/StatLearning/Winter2016/about)(主要关注回归和分类)【9】，我强烈推荐，附带免费书籍[‘统计学习介绍(R 的应用)’](http://faculty.marshall.usc.edu/gareth-james/ISL/)【10】。Packt publishing 提供了一系列关于数据科学的书籍，主要是关于 Python 和 R 的

最后，如果你正在搜索一些免费的编程书籍:[https://github . com/ebook foundation/free-programming-books/blob/master/free-programming-books . MD # r](https://github.com/EbookFoundation/free-programming-books/blob/master/free-programming-books.md#r)
