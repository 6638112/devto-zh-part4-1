# 编程课不及格如何吓住了我

> 原文：<https://dev.to/bbarbour/how-flunking-a-programming-class-deterred-me-1k63>

让我们回到 2004 年。我是一名高三学生。我参加了一个编程入门课程，并迫不及待地想了解它是如何工作的。你知道，前年我参加了一个网页设计课程，学习了 HTML 和 CSS 的基础知识。

我拿了那门课的 a，并且喜欢上了这门课。我想我对软件开发的热爱就是从那时开始的，但是这种热爱很快就失去了。

你看，我的编程入门课程是关于 C++编程语言的。

C++很难。

我班上有些孩子在这方面很出色，看着他们写代码让我内心感到恐惧。我落在了后面，最终不及格。在我生命中的那个时候，我还在学习如何学习。我还没有掌握我的注意力缺失症和心理健康问题，比如抑郁症。过不了几年，作为一个成年人，我会准备好再次挑战自己。

回过头来看，我应该向老师要时间来更深入地讲解这些科目，并帮助我确保我理解了。我知道她会的，她是个很棒的教练。

我还不够成熟。对我来说，我不是天生就学会的，这意味着它不适合我。我感到羞愧，我感到自卑，这种感觉会伴随我十多年。

编程的失败并没有阻止我从事计算机职业。我继续学习，专注于我认为更简单的东西，如硬件、网络和维修。这也没有阻止我把制作网站作为一种爱好。我会使用 phpBB 和 Wordpress 之类的软件，围绕我最喜欢的爱好、游戏或节目创建论坛。

但是，我总是止步于 PHP 和 Javascript。我撞墙了，它在编程。我寻找插件来解决问题，有时如果没有插件就放弃。我再也没有想过，如果我投入足够的时间和精力，我可以写自己的作品。

我的脚踝和手腕上有镣铐，我不能全速奔跑。

直到 2018 年 11 月，我才熬过这种恐惧。

是什么让我突破它？

一份更清晰的精神健康法案帮助很大。还有，我觉得我的好奇心最终战胜了恐惧。在这方面，我问了自己一个重要的问题。

“我有什么损失？”

没什么。最坏的情况是，我会对计算机有更好的理解。这将有助于我作为电脑技术人员的职业生涯，排除故障和诊断问题。

于是，我坐下来，开始了 Codecademy 上的 Web 开发之路。

从那天起，我就没有停止过，也许永远也不会。

最后一点，我确实计划将来学习 C++，只是为了报复它灌输给我的自卑感。不过现在，我很高兴尽可能多地学习 Javascript。
