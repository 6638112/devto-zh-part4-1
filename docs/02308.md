# 分享你最尴尬的壳牌管道

> 原文：<https://dev.to/sinewalker/share-your-most-embarrassing-shell-pipeline-356e>

有时候，公开你的代码气味可以宣泄情绪，并激励你去修复它。

这里可能是我目前在 Bash 点文件中最尴尬的 shell 管道:

```
 \grep -Rn "alias ${1}=" ${DOTFILES}/source \
                | awk -F: '{print "⍺:\t" $3 "\t--> " $1 ":" $2}' \
                    | sed "s/${1}=//g" 
```
