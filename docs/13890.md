### 从 Google Play 图书中查找图书

### 随时投稿上 **[GitHub](https://github.com/liyasthomas/books)**

我在网上随便冲浪，看到一篇关于 [Google Books API](https://developers.google.com/books) 的文章。我读了这些文档，决定创建一个最小、轻量、漂亮的 web 应用程序，使用 Google Books API 浏览&查找书籍。

这就是我创作[书的原因📚](https://liyasthomas.github.io/books)。

[![UI](img/aba20550ee899252f0d9f0dcfd2bc2b7.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--eeHskxbm--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://raw.githubusercontent.com/liyasthomas/books/master/icons/screely.png)

[Books](https://liyasthomas.github.io/colorbook) 是一个开源项目，利用了 [Google Books API](https://developers.google.com/books) 。

它的一些特点是:

*   按书名、作者、类别、流派等搜索书籍。

*   简约优雅的用户界面。

*   PWA 支持。

*   黑暗模式。离不开它！

[![Library](img/6b990674ac69d9ab8553101c6bc57c1c.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--L6PoNQns--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://raw.githubusercontent.com/liyasthomas/books/master/icons/screely2.png)

奖金:🌗我还增加了黑暗模式，这样你的眼睛在晚上就不会痛了！
