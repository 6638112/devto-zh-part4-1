# 害怕成功？(我的故事)

> 原文：<https://dev.to/shad11981/fear-of-success-my-story-415j>

请允许我自我介绍。我叫拉沙德。从我记事起，我就热爱科技。我总是对事物的工作原理和方式着迷。我也爱学习。我喜欢发现科技世界的新发展，也想成为一些新发现的一部分。我目前拥有网络安全学士学位，正在努力获得 Comptia Security+认证。既然我们已经介绍完了，让我们来谈谈我为什么要写这篇文章...我从 2014 年开始获得学士学位。你问我为什么没有认证？因为它们有点贵，而且我觉得我害怕成功...读起来很疯狂，承认起来更疯狂。我得到这个启示已经有一段时间了，我想看看我是不是疯了，或者是不是有其他人和我一样。我一直是个中等偏上的学生。我理解得很快，可以毫无问题地记住信息。如果我自己这么说的话，我有很好的记忆力。但是由于一些奇怪的原因，我不能沉浸在它的世界里。我在 IT 领域工作，是我现在的第三家公司的桌面分析师。在每家公司，我都学习系统，我的 IT 知识也在增长，但由于某种原因，我无法像我希望的那样离开 IT 服务台，闯入网络。我意识到我一直在破坏自己。我没有令人心碎的故事，也没有令人心碎的原因，事实上，我几乎拥有你想要的一切，能够追求我选择的职业道路。我有一个充满爱和支持的家庭。我有一份工作，每天工作 4 个 10 小时，其中 2 个周末离家很远，工作很少，连续休息 3 天。我也是一个父亲和继父，但孩子们不想和老人呆在家里，所以我连续三天几乎都是一个人。有了这样的时间和机会，你会认为我能够在几个月内轻松地准备好 Sec+考试....但是没有，我还在踩水...我经历了被高度激励的爆发，也在书中找各种借口不去做我知道我需要做的事情。在我花一个小时开车去上班的两天中的一天，我意识到我是我没有在 IT 领域进一步发展的原因。我发现自己在想象我的生活将会怎样，并且总是说“一旦我拿到第一个证书，一切都将改变。”我百分百相信。可是为什么我走不出自己的路？很难吗？不完全是。我喜欢科技，所以我喜欢学习它。我听的几乎每一个播客都以某种方式、形式或形式与科技相关。我做了愿景板。精确到分钟的学习时间表。每次有事情发生。为什么我害怕成功？是因为我相信别人比我更值得拥有吗？不，我相信每个人都有很多机会。因为我害怕成功会改变我。好了，我说了。最后。我真的很喜欢我是谁，我周围的人是谁，但我目睹了这么多的人在世界上上升，我不再知道他们是谁了。我不相信我会那样做，我宁愿认为我不会。但是在我脑海深处的某个地方，一个微小的声音一直在说“如果你做了呢？”。我不太害怕，但不得不应对成功的恐惧和压力实际上比我口头向任何人承认的都更让我害怕。我甚至不介意失败，因为我从中学习，我会继续成长，但成功...这让我害怕。既然我已经意识到我害怕成功，我现在就在这里公开宣布，我将在今年克服这种恐惧。我将参加并通过安全+认证考试，我将学习 Python 编程和 Linux，以便能够使用它们并创造新的东西。我将把这个平台作为我的责任场所，我希望能够激励和激励其他人也实现他们的目标。我可能不会像在 IT 领域追求目标那样专注于技术上的事情。我希望你加入我，并随时与我互动。我愿意学习和教任何愿意倾听的人。感谢你花时间阅读这个大脑垃圾。信不信由你，它对我帮助很大。我很欣赏你。拉沙德完毕。
