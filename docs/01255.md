# 光标设置为输入的特征

我想分享一些关于使用函数 focus()时输入光标的不同行为的经验。

*   如果跨属性设置输入元素的值，则当调用函数 focus()时，光标将设置在 start 中。

*   如果输入元素的值是跨值的属性设置的，则当调用函数 focus()时，光标将设置在输入的末尾。
