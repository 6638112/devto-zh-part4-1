我很高兴采访了 Pauline Narvas，内容是关于学习编程、教女孩编程以及在一个英国小城市做技术人员。Pauline 在 Twitter 上吸引了我的注意，因为她热爱编码，热衷于向女性开放科技世界。谢谢你的时间，宝琳！

## [](#i-know-from-twitter-that-you-are-involved-with-code-firstgirls-can-you-tell-me-how-you-got-involved-with-that-and-what-teaching-code-is-like)我从推特上知道你和[的代号有关:女生](https://www.codefirstgirls.org.uk)。你能告诉我你是如何参与其中的，教学代码是什么样的吗？

嘿！没关系，谢谢你邀请我😀我喜欢分享这个故事。我第一次接触 [Code First: Girls](https://www.codefirstgirls.org.uk) 要追溯到 2016 年，当时我在读生物医学大学二年级。那时候，我感觉停滞不前，没有动力，总体来说，没有任何目标。

当我收到来自 [Code First: Girls](https://www.codefirstgirls.org.uk) 关于谢菲尔德大学即将开设的一门课程的电子邮件时，我记得我兴奋地坐着申请并参与其中。我总是独自一人在房间里把编程作为一种业余爱好，当我和其他人一起编程时，是在我的 GCSE 计算课上，在一个满是男孩的房间里。这听起来令人耳目一新，令人兴奋，所以我报名成为一名学生。虽然我知道前端开发的基础，但我仍然带着对 Bootstrap 等框架以及如何使用 Git 协同工作的新理解完成了课程。

[![Pauline Narvas talks learning to code without a Computer Science degree](../Images/2b8b33d5dd015473157072ef0cfd36fe.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--CRFA8R39--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://www.nocsdegree.com/content/images/2019/07/Pauline-teaching.jpg)

参加完这个课程后，我自愿成为大使和讲师，在我的大学主持未来的课程。三年过去了，我仍然在教这门课！目前，我是利兹的[代码优先:女孩夏季强化课程](https://codefirstgirls.arlo.co/w/courses/4-leeds-summer-intensive)的首席讲师。教别人如何编码确实是一种美妙的体验，当你教的东西被理解和应用时，没有什么比看到灯泡时刻更好的了。它还帮助我进一步巩固了对前端开发的理解。

## 我在你的博客上看到你很小就开始做网站了，大概 8 岁？最初是什么让你对编程感兴趣的？

啊哈，是的！我过去甚至在那时也开了一个博客——嘘，请不要试图找到我以前的帖子，哈哈。当时正在玩的一款网游引起了我的兴趣。虽然我喜欢玩它，但我还是想改变一些小细节。鉴于这个游戏是在线的，我认为最好的第一步是学习如何在网上建立一个网站。

当我开始建立我的第一个网站时，我的注意力从游戏开发转向了网站建设。一路走来，我的博客(或多个博客——有一段时间我也有一个凡妮莎·哈金斯和高中音乐博客——一点也不感到羞耻)诞生了，用来记录我的学习，也在我的各个网站上托管一些“内容”。

## 你没有计算机科学学位，你是如何学习编码的？你参加过任何特定的在线课程或训练营吗？

当我打算开发一个在线游戏时，我第一次开始从各种尝试和错误中学习如何编码。迄今为止，我大部分时间都是自学的，从未真正接受过任何正式的培训。然而，我参加了前端开发的 [Code First: Girls](https://www.codefirstgirls.org.uk) 课程，在那里我了解了 Git。我还先做了另一个代码:python 和构建 web 应用的女生课程！

## [](#what-advice-do-you-have-for-someone-who-doesnt-have-a-computer-science-degree-and-wants-to-learn-coding)对于没有计算机专业学位，想学编码的人，你有什么建议？

我的主要建议是不要纠结于官方资格！只要你有学习的意愿、耐心和热情，你就会适应。在这个快节奏的行业里，不可能什么都知道，所以永远不要太纠结于试图赶上一切。学习编码是日积月累的努力的集合！我写了一篇关于我第一次学习编程的主要技巧和建议(以及参考资料)的博文，你可以[在这里](https://pawlean.com/2019/01/14/learning-to-code/)阅读更多内容。

## [](#whats-it-like-being-a-coder-in-a-relatively-small-city-in-the-uk-do-you-find-a-closer-sense-of-community-since-its-not-silicon-valley-where-everyone-is-a-developer)在英国一个比较小的城市做编码员是什么感觉？你有没有发现一种更紧密的社区意识，因为这里不是人人都是开发者的硅谷？

多么有趣的问题！来自硅谷或任何有大型开发者社区的地方的人们可能会认为谢菲尔德[/](https://www.google.com/maps/place/Sheffield/@53.3817818,-3.7412202,7z/data=!4m5!3m4!1s0x48790aa9fae8be15:0x3e2827f5af06b078!8m2!3d53.381129!4d-1.470085)[利兹](https://www.google.com/maps/place/Leeds/@53.8339351,-6.0194647,6z/data=!4m5!3m4!1s0x48793e4ada64bd99:0x51adbafd0213dca9!8m2!3d53.8007554!4d-1.5490774)没有这样的社区。但事实远非如此——谢菲尔德、利兹以及更广阔的英国的社区是我曾经参与过的最好的社区。

我在我参加的社区活动中感觉到一种归属感，我总是有机会与我以前从未见过的新朋友联系！这是一个由非常棒的人组成的紧密团结的团队，他们并不总是开发人员——技术领域的角色多样性是巨大的——我从未意识到它有多么巨大。这就是为什么我热爱谢菲尔德和利兹的社区。😀

## 我知道你精通多种语言，如 PHP、Python、R 等。你有最喜欢一起工作的人吗？

我更像是一个 UI 开发人员——所以前端的一切都会让我感兴趣！这完全是一个不受欢迎的意见，但我会永远爱我一些 CSS。

## [](#whats-it-like-being-on-the-digital-engineering-bt-graduate-scheme)上数字工程 BT 研究生方案是什么感觉？

参加英国电信研究生项目是一次非常美妙的经历。我绝对喜欢我所在的团队，我有一个难以置信的机会向数字工程领域的各种人学习，从开发人员、产品负责人到架构师或平台工程师。

在过去(几乎)一年里，我在这里学到了很多东西，我渴望继续学习更多的技术、工作方式等等。这真的是一次独特的经历，我总是很幸运能成为其中的一部分！我写了很多关于我的研究生计划的经历，你可以在这里继续。
