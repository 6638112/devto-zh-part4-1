## [](#browser)浏览器

我的主浏览器是[勇者](https://brave.com/)。这是一个伟大的，快速和安全的铬替代品。我还使用以下插件:

*   [uBlock Origin](https://chrome.google.com/webstore/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm) -我发现的最好的广告拦截器。
*   [口袋(保存到口袋)](https://chrome.google.com/webstore/detail/save-to-pocket/niloccemoadcdkdjlinkgdfekeahmflj) -以后再看。
*   [LastPass](https://chrome.google.com/webstore/detail/lastpass-free-password-ma/hdokiejnpimakedhajhdlcegeplioahd) -密码管理器。
*   Devtool 扩展: [Vue.js](https://chrome.google.com/webstore/detail/vuejs-devtools/nhdogjmejiglipccpnnnanhbledajbpd) 和 [React](https://chrome.google.com/webstore/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi)

我以前有一个叫 Wappalyzer 的...但是我发现这个扩展很奇怪，所以我把它删除了！

## [](#terminal-amp-command-line-apps)终端&命令行应用程序

我主要在 VSCode 中使用我的终端。不然我用 iTerm(不是大粉丝！).我的期末主题是 af-magic。

[![terminal screenshot](../Images/8ef97ed37a2f6ef2b89475b18ce8857a.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--fyOFilvF--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/f2mniwwuxag9amjjuqyv.png)

这是我使用的插件/应用列表:

*   [Zsh](https://github.com/robbyrussell/oh-my-zsh/wiki/Installing-ZSH) 跟[哦我的 Zsh](https://github.com/robbyrussell/oh-my-zsh) 。我的 zsh 是一个管理 Zsh 配置的框架。
*   Lighthouse (CLI) -渐进式 Web 应用的审计、性能指标和最佳实践。
*   [Hub](https://github.com/github/hub)-Git 命令的包装器 *(Git+Hub=GitHub)*
*   vtop -活动监视器的替代产品。
*   [Tree](https://formulae.brew.sh/formula/tree) -生成一棵树(类似于 Tree Windows 命令)。
*   一个超级方便的节点版本管理工具。

我也时不时用 Vim。

## [](#desktop-apps)桌面应用程序

谈到我的码头，我有一个非常简约的方法。如果我一天不使用一个应用程序，至少一次，我会把它从 dock 上移除。

[![app screenshot](../Images/1e92843fd6a76398943621c96ad90665.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--LYLzMiQM--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/k27m5rday57jbambrxyr.png)

我也使用这些应用程序:

*   [ProtonVPN](https://protonvpn.com) 我来买单！当你在一些亚洲国家旅行时，获得一个 VPN 是一种强制。
*   [Giphy Capture](https://giphy.com/apps/giphycapture) 是我需要与同事分享信息时的屏幕记录。我发现它很容易使用。
*   [OBS](https://obsproject.com) -屏幕录制，但用于较长的视频。我不怎么用它...
*   当我需要 MySQL 的 GUI 时。
