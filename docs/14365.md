# 观察科技领域的伟大领袖

> 原文:[https://dev . to/Ethan arrowood/observing-great-leadership-in-tech-55hl](https://dev.to/ethanarrowood/observing-great-leadership-in-tech-55hl)

科技公司的领导者们正在完成令人惊叹的事情，比如李美玲、T2、安尼尔·达什和 T4·纳特·弗里德曼。

李美玲是 Dot Labs 的领导者，这是一家为网站开发者提供指导、培训、咨询和学徒服务的公司。她是 Node.js 开发者关系的核心部分，也是开发者社区的重要组成部分。

Anil Dash 是 Glitch T1 的领导者，这是一个简单的工具，用于创建友好的、有创意的网络应用。他最近发布了 Glitch 的 [2018/2019 成功](https://twitter.com/anildash/status/1148603066092052482)，值得一读。Anil 始终如一地分享他关于包容性和道德技术的想法、知识和行动。

自从被微软收购后，Nat Friedman 一直领导 GitHub。他发布了许多新功能，并且在 GitHub 开发者社区中非常活跃。我有幸和他聊到了他的成功和他的一些领导价值观。在这个[电话](https://twitter.com/ArrowoodTech/status/1137499234679349251)中，我了解到了势头的价值，以及他如何将这归功于 GitHub 最近的成功。

我们的技术社区中还有哪些我们应该关注的领导者？

我特别寻找来自代表不足的群体的领导者，以及通过包容和有影响力的行动改善社区的领导者。

如果你还没有，我建议跟随我上面列出的领导者，这样你也可以向他们学习！
