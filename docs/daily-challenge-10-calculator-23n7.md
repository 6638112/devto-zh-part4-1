# 每日挑战#10 -计算器

> 原文：<https://dev.to/thepracticaldev/daily-challenge-10-calculator-23n7>

我们又回到了另一个代码挑战，[这个挑战来自 Codewars 的用户 obrok](https://www.codewars.com/kata/5235c913397cbf2508000048) :

> 创建一个简单的计算器，给出一串运算符(+ - *和/)和由空格分隔的数字，返回该表达式的值
> 
> 示例:

```
Calculator().evaluate("2 / 2 + 3 * 4 - 6") # => 7 
```

> 记住操作的顺序！乘法和除法有更高的优先级，应该从左到右执行。加法和减法的优先级较低，也应该从左向右执行。
