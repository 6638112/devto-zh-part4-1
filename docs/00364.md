# [](#aws-migration-hub)AWS 迁移中心

跟踪服务，用于跟踪迁移活动。

# [](#application-discovery-services)应用发现服务

它有助于在迁移时找到依赖关系。

# [](#database-migration-services)数据库迁移服务

它有助于将内部数据库迁移到 AWS。

# [](#server-migration-service)服务器迁移服务

它有助于将虚拟或物理服务迁移到 AWS 服务。
