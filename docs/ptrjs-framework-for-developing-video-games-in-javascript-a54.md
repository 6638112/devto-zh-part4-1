# 用 JavaScript 开发视频游戏的框架

> 原文：<https://dev.to/leptr/ptrjs-framework-for-developing-video-games-in-javascript-a54>

你好世界！我已经为这个项目工作了几个月，我很自豪地向你展示:ptr js——用 JavaScript 开发视频游戏的框架。具有讽刺意味的是，这一切都始于我不想使用任何框架，所以我决定使用普通的旧 JavaScript 来创建使用 HTML canvas 元素的视频游戏。但是当我看到画一个简单的圆需要多少行代码时，我震惊了。所以我想“为什么不把它全部放在一个函数里，这样我以后就可以用它了？”一个接一个，我已经为我开发视频游戏时需要的所有东西创建了一个函数，甚至更多。
几个月过去了，现在我们在这里！PtrJS 正式发布，你可以使用了！你只需要去 [GitHub repo](https://github.com/leptr/ptrjs) 并阅读文档。很好用，我发誓！
你可以在 GitHub 上的 [PtrJS repo](https://github.com/leptr/ptrjs) 找到源代码和文档！
我希望你会像我喜欢开发它一样喜欢使用这个框架。
干杯！
