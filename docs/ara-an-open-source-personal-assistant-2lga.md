## 什么是 Ara

Ara 是今年 1 月的一个想法，我想要一个我会喜欢的兼职项目，所以在一些朋友(Chris 和 Eric)的帮助下，我开始开发一个 Android 应用程序，后来成为 Ara。我想要一个伟大的开源语音助手应用程序，可以与谷歌和亚马逊这样的公司竞争。经过思考，我们确立了 3 个主要目标:

*   制作一个可以在本地运行的语音识别系统
*   完全的数据隐私
*   提供打开应用程序的有用信息

为了实现这些目标，我使用了 100%的开源软件，所以我可以保证没有跟踪，我使用的 SDK 有大量的文档，并让世界在他们自己的项目中使用我的代码。
我这样做也是为了证明这里描述的一个想法的概念:

## 需要做什么？

Ara 还没有完成，在测试版发布之前它需要 4 件主要的事情

*   扩展的语音功能
*   声音输出
*   Skills SDK (ara 曾经能够通过键盘检测几个命令，但是为了开发这个功能而放弃了)
*   IOT 支持

我们还在开发一些长期功能:

*   公共(托管在 are 服务器上)和私有(托管在公司服务器上)帐户系统，便于数据同步
