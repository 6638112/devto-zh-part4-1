编程语言是一种正式的计算机语言，用于向机器，尤其是计算机传达指令。编程语言可以用来创建程序或算法来控制机器的行为。

Java :
Java 是一种基于类和面向对象的编程语言。Java 最初是由*【詹姆斯·戈塞林】*在 1995 年开发的。
Java 用于开发手机应用、网络应用、桌面应用、游戏等等。
