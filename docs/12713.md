# 为什么我把笔记移到特雷罗

> 原文：<https://dev.to/fischgeek/why-i-moved-my-notes-to-trello-po9>

但是，Trello 是一个项目管理应用？没错，虽然它主要用于这个目的，但我发现它的灵活性开始对我来说成为一个笔记仓库有意义了。我会尽量把一个很长的背影故事说得越短越好，但是我也很早就纠结于这个话题。

我试过这个游戏中所有的主要(和次要)玩家。在我职业生涯的大部分时间里，我一直使用 OneNote。虽然这可能是我唯一一个持续使用的笔记应用程序，但它在某些地方有所欠缺。

我不太清楚是怎么回事，但使用 OneNote 的某些方面就是不令人满意。感觉很笨重。前几天，我的一位同事一针见血地描述道:

我觉得当微软做出这样的产品时，它就像一个学校的期末项目。它完成了工作，但还不够。

另一个失误是缺乏与其他第三方的集成。而我就是不够聪明，不会用他们的 API。我尝试过无数次，但都失败了。

我使用 Google Keep 也有一段时间了(和 OneNote 一起使用),我真的很喜欢这种体验。它有很多我一直在寻找的功能:使用起来很有趣，可以在手机、PC (chrome 应用)和浏览器上使用。它有能力有清单，格式化我的链接真的很酷，我可以分享/协作笔记，但它缺乏一个集成的 API。我的笔记卡在那里，只有在那里。离线对我来说没什么大不了的，但决定性因素是笔记本身缺乏文本格式。

我试过几次 Evernote。它有更好的集成功能，但这些都是有成本的，我不愿意在我已经捉襟见肘的预算中再增加一个月的账单来购买我不喜欢的东西。多年来，我一直在“你好，互联网”和“T2 大脑皮层”上听 CGP Grey 抱怨这个问题，这可能会让我产生偏见。不过说到底，它很像 OneNote。可能太开放了，我不知道？不喜欢。继续前进。

我开始尝试使用设备自带的本地应用。我的 iPhone 上的笔记(我是一个有 iPhone 的 Windows 用户 btw)。虽然 Notes 可以在手机和 iCloud.com 上使用，但它也缺乏集成，同步速度很慢。我甚至用了 Windows 上的记事本。我设置了一个系统，可以捕获我所有打开的记事本，并将它们保存到 Dropbox 的一个目录中。我可以把它们加载到我的手机上，但那也很慢，而且显然没有格式化。

我试过无数其他的免费笔记应用程序，但最终都因为这样或那样的原因被抛弃了。

最后，有一天当我在 Trello 做一些 PM 时，我突然意识到我们有一个标签为“收件箱”的专栏(大致模仿 GTD)。我单独查看了这个专栏的内容:每张卡片都是一个注释，每个注释还可以有描述、评论和清单！，到期日！，合作者！，还有标签！然后，我开始将我所有的笔记输入到我的个人 Trello 板上。来自 OneNote，它是这样的

*   每本书现在都是一块木板
*   每个部分现在都是一个列表
*   每一页现在都是一张卡片

我真的很喜欢这个。我不停地思考着所有的可能性。我可以通过 OneNote 的类似结构和其他应用程序缺乏的所有中间功能获得与 Google Keep 相同的有趣体验。API 超级好用！我可以在我的板上自定义背景。但是，最后最好的部分是，事实上，特雷罗决定，巴特勒电源的完整版本现在是免费的。那真是个好消息！把巴特勒想象成 IFTTT 为特雷罗制定的规则。

我使用 Butler 的一个简单例子是，我有一个名为“Pinned”的标签，当标签“Pinned”应用于任何卡时，它会将该卡移动到它所在的列表的顶部。当标签被移除时，它会将卡向下碰撞到任何其他现有的“固定”卡的下方。

他们改进了他们的提醒系统，所以有更多的选项来提醒卡到期日。

我还想提一下自定义字段的功能，它允许我以任何我想要的方式构造注释元数据。

我可以继续说下去，但我认为是时候强制 TL 了；用 Trello 做我的笔记效果很好！你应该试试！
