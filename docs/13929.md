### [](#a-look-back-in-history)回顾历史

从 1908 年到 1927 年，人类的存在经历了大约 20 年的时间才彻底改变。历史上的那个时刻就是我们用 t 型车取代马车的时刻。

交通、可达性、独立性、贸易以及我们日常生活中的一切都受到了这一进步的影响，汽车的创新继续渗透到我们现在的生活中。

### [](#the-now-the-present-and-our-future)现在，现在，和我们的未来

我们正在经历另一个伟大的历史时刻，正在参与我们这一代人最伟大的篇章之一——书写现代网络。

在未来 10-15 年，我们的每一种体验都将以某种形式消费现代网络。这意味着我们所做的一切，从与我们所爱的人交流到购物，都将受到向前发展的网络的影响。

我们现在正在看到这种变化，大多数基于网络的应用程序都被编写为渐进式网络应用程序，单页面应用程序成为常态，移动平台上的创新以我们从未想象过的方式让我们感到惊讶。

### [](#the-current-state-of-modern-web-advancement)现代网络发展的现状

当前推进现代网络的创新首先发生在不同技术社区的筒仓中，阻碍了我们通过更多协作驱动的努力所能实现的进步速度。

我们越快理解模式和相似之处，并一起创建通用的解决方案，我们的行业发展就越快。

对现代网络的发展和进步至关重要的另一件事是教育，帮助普通技术人员了解新的范例和技术，如服务人员、单页应用程序和先进的网络应用技术。这些新标准的消耗越快，我们的技术进步就越好，我们获得新技术的速度就越快。

采用的障碍也是开发人员在使用这些现代 web 技术时可能遇到的棘手问题。帮助开发人员解决早期采用的痛点并利用这些技术进行构建的知识库是推动我们行业向前发展的关键。一旦采用迈出大步，变化是迅速的，创新是繁荣的。

### [](#advancing-the-modern-web)推进现代网络

最终，致力于现代 web 平台和技术的技术专家只有一个目标，那就是通过使用可靠的实践(如反应式编程和现代框架)构建渐进式 web 应用程序，使工程师能够开发一次，并在所有平台上拥有可行的应用程序。

要了解如何最好地教育人们，以及需要什么工具来使不断变化的网络对所有寻求采用单页应用和渐进式网络应用技术的公司变得触手可及，还有许多工作要做。

### [](#this-dot-in-the-modern-web-ecosystem)现代网络生态系统中的这个点

我们在 Dot 的使命是通过广播行业的进步来帮助进一步加速新技术在现代网络生态系统中的应用。

我们致力于促进和加强与现代网络相关主题的对话，我们希望通过为开发应用程序的人们提供他们需要的信息，使他们更好、更快地开发应用程序，从而形成对网络平台更强的主人翁意识，并更好地理解标准机构和浏览器做出的决策。

我们希望我们为现代网络创建的平台将为每个人提供一个加入对话和参与现代网络创新的机制。
