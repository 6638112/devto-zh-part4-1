Java 简介

Java 是 Sun 微系统公司开发的一种编程语言，于 1995 年发布。Java 编程语言是应用最广泛的面向对象、基于类的通用编程语言。它是一种快速、安全且非常可靠的编程语言。Java 是许多公司构建软件应用程序(如 web 应用程序、游戏软件和 android 应用程序等)的首选编程语言。

Java 编程语言的演变

如果一个人想学习 Java，他必须知道 Java 编程语言产生的原因。开发 Java 编程语言的主要原因是:

```
To adapt easily to the changing environments and uses.
To bring improvements in the art of programming. 
```

随着你对 Java 的进一步了解，你会明白 Java 是由这两个因素同等比例地驱动的。

Java 与 C++相关，c++是使用 C 编程语言开发的。Java 受到了这两种语言的影响，因为它的语法来自于 C，而它的面向对象的特性受到了 C++的影响。

正如我们所看到的，每种语言发明的目的都是为了以一种比它之前的语言更好的方式提供问题的解决方案，Java 也不例外。

开发 Java 的其他主要原因是:

```
To create a platform independent language which can run on any environments irrespective of any platform.
It is developed with the concept of “Write once, run anywhere” wherein it enables the programmers to easily execute the programs on different platforms without making any modifications to the source code. 
```

在我们进一步讨论如何从头开始学习 Java 之前，我们最好花几分钟时间解决几个重要的问题。

学 Java 容易吗？

Java 和其他编程语言有相同的学习曲线。Java 胜过大多数编程语言的一些原因是:

```
It is platform independent.
It is a class-based object-oriented language which reduces the complexity of the coding. 
```

这些因素有助于程序员容易理解编程概念。即使没有任何基础编程语言，但有强烈的愿望去学习和理解这种奇妙的编程语言，也可以用简单的方法学习 Java。

Java 还有需求吗？

是的，Java 是最成熟的语言，因为大多数软件应用程序都是使用 Java 编程语言开发的。Java 以其独特的特性和持续的更新领先于大多数编程语言。
