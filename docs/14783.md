使用[https://vue-native.io/](https://vue-native.io/)

安装`npm install -g react-native-cli`

安装`npm install -g expo-cli`

安装`npm install -g vue-native-cli`

```
npm install -g vue-native-cli
vue-native init <projectName> // Initializes crna project
vue-native init <projectName> --no-crna // Initializes react-native project
cd <project-name> 
```
