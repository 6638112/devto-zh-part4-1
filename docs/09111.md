# 现在流行开源

> 原文:[https://dev.to/adityadroid/facelyt-now-open-sourced-pb](https://dev.to/adityadroid/facelyt-now-open-sourced-pb)

最近开源了我的一个旧项目“facely ly”。这是一款基于 webview 的 facebook 客户端应用，具有以下关键特性:

1.  主题支持
2.  聊天头支持
3.  夜间模式
4.  数据保护模式
5.  热点链接
6.  长按图片或帖子进行分享
7.  所有这些都包含在大约 800kb 的应用程序中

该应用程序在 play store 上有 100，000 多名用户。几年前它被谷歌撤下，因为据他们说，显然“大写的 F”与脸书的标志很相似，脸书对此很不高兴。

不过这里有 Github 的链接& APK 的链接，给那些有兴趣看应用程序/源代码的人。接受任何建议。:)
