# 网页设计和网页开发的区别是什么

> 原文：<https://dev.to/blazingcoders/what-is-the-difference-between-web-design-web-development-1710>

在这篇文章中，我们将讨论网页设计和网页开发的区别。

网页设计和网页开发，这是网站创建过程中两个不同的角色。网页设计包括创建网站的外观和工作方式，而网页开发包括开发一个网站，让它成为一个功能强大的网站。

网页设计
设计一个网站包括获取客户希望在网站上展示的简要信息，创建模型，并为其设计出最佳结构。

网站设计者对网站进行了大量的思考，使其具有创造性，并以创新的方式安排信息，这意味着他们在正确的地方看到了正确的内容。

设计过程的一部分被称为用户体验或 UX，你可能已经看到它被称为，但设计师显然也对网站的视觉外观和感觉负责，这被称为用户界面，或 UI。

一个网站的外观非常重要，因为你必须确保设计布局符合公司的品牌，并且信息易于阅读和消费。

有许多不同的软件(Adobe Photoshop、Sketch、Pixlr 等)可供网站设计师用来设计网站。

Web 开发
如前所述，Web 开发包括获取模型并通过代码赋予其生命，这样它就像一个活的网站，你可以放在互联网上供人们使用。

网站的基础是使用 HTML 创建的，它用代码组织网站的结构，然后使用 CSS 来设计 HTML 的样式，使其与设计师创建的设计的颜色和形状相匹配。

有许多不同的编程语言，并不是每个 web 开发人员都知道它们。他们通常会有自己的一些专长，但 HTML 和 CSS 是你开始的基础。开发人员不仅要负责确保网站正常运行并与设计相匹配，还要确保代码写得干净利落，这样用户就能很快加载。通常，开发人员会使用文本编辑器来编写和编辑他们的代码。

底线是，设计师需要更多地了解设计原则和创造力，以提出独特的布局，满足客户的需求，而开发人员则更注重基于逻辑的思维以及对代码和技术如何工作的理解。
