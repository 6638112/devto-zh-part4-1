# PHP 5 大框架:Laravel vs Yii vs Zend vs phal con vs Symfony，他们好的和不好的一面

> 原文:[https://dev . to/mkdev/top-5-PHP-frameworks-laravel-vs-yii-vs-Zend-vs-phal con-vs-symfony-they-and-bad-sides-26 a5](https://dev.to/mkdev/top-5-php-frameworks-laravel-vs-yii-vs-zend-vs-phalcon-vs-symfony-their-good-and-bad-sides-26a5)

рнр框架有各种各样的功能和生态系统，适合无数的任务。有了它们，你可以在短时间内创建更加复杂、安全和用户友好的应用程序和网站。

### [](#why-do-developers-actually-use-php-frameworks)为什么开发者实际上使用 PHP 框架？

开发人员选择它们是因为它们加速了开发过程，开箱即用且可伸缩。

要找到一个适合自己，并且能按预期工作的框架，并不是一件容易的事情。下面你可以找到前 5 个最流行的框架。

### [](#laravel)Laravel

#### [](#good-sides)好两面

*   最佳国际奥委会(控制反转)
*   便捷的迁移系统
*   嵌入式模块测试
*   模板引擎叶片
*   灵活路由
*   构建 REST API 的许多方法
*   Laravel 正在迅速发展
*   任何主题的大量文件
*   Laravel 在全世界都很受欢迎
*   具有调用堆栈功能的内置调试控制台
*   RBAC
*   ACL 插件

#### [](#bad-sides)坏边

*   如果你想访问所有的特性，你需要使用 Laravel facades，ide 不能识别某些类中的方法和属性，并显示警告信息
*   学习曲线比 Yii2 的更陡
*   不生成嵌入式接口

*学习曲线*:你需要精通 OOP 和数据库。

Laravel 开发人员的平均工资:$ 89.774/年或以上

### [](#phalcon)费尔康

#### [](#good-sides)好两面

*   高性能
*   许多功能
*   适合高负载环境
*   位于 RAM 中
*   不需要很多文件操作
*   不会消耗很多资源
*   您可以使用您喜欢的数据库库及其元素
*   数据库是通过 ORM 处理的，这带来了更高的性能
*   所有的过程都很快，因为框架可以直接访问 PHP 的内部结构

#### [](#bad-sides)坏边

*   它的源代码是用 C 写的
*   还不是很受欢迎
*   它是一个扩展(这意味着你可能无法在共享主机上运行应用程序)
*   有许多人仍然对费尔康一无所知

*学习曲线*:平滑。你需要精通 OOP，理解设计模式，并在一些项目中有实践经验。

Phalcon 开发人员的平均工资:$ 50.000/年或以上

### symphony

#### [](#good-sides)好两面

*   与 Yii 非常相似。Symfony 文档坚持认为它不是一个 MVC 框架
*   对共知的本机支持允许编写功能和验收测试
*   具有 YAML 组件，这对于任何框架来说都是一个巨大的优势
*   Drupal 和 PhpBB 等项目是使用一些 Symfony2 组件构建的
*   大型开发人员社区
*   大量现成的模块集，称为捆绑包
*   详细清晰的文件
*   相当高的核心性能
*   松耦合

#### [](#bad-sides)坏边

*   Symfony 使用令人恐惧的 ORM (Propel 和 Doctrine)并且是资源密集型的
*   陡峭的学习曲线
*   太多不同种类的实体
*   包含注释语法

*学习曲线*:陡峭。你需要精通 OOP，理解设计模式，并有一些其他框架的实践经验。

Symfony 开发人员的平均工资:$ 85.000/年或以上

### [](#zend-framework)Zend 框架

#### [](#good-sides)好两面

*   类继承
*   面向对象
*   为许多任务提供现成的解决方案
*   你可以把你喜欢的任何东西和任何东西结合起来
*   国际化工具
*   开发者社区的支持
*   高质量的文件

#### [](#bad-sides)坏边

*   不太适合快速发展
*   比其他一些框架慢(但对 90%的网站来说已经足够快了，数据库总是瓶颈。)
*   需要大量的时间来掌握
*   资源密集型

*学习曲线*:陡峭。你需要精通 OOP，理解设计模式，并有一些其他框架的实践经验。

Zend 开发人员的平均工资:$ 87000/年或以上

### [](#yii2-framework)Yii2 框架

#### [](#good-sides)好两面

*   基于 Web 的代码生成器
*   直观的 MVC 架构。你可能喜欢从头开始学习
*   利用常见的问题解决模式，使代码不那么混乱
*   使维护使用公共架构和方法的代码变得更加容易
*   伟大的开发者社区，随时准备帮助解决框架、一般性问题和新特性
*   节省花费在表单验证和安全检查等日常任务上的时间
*   易于配置以获得更好的性能
*   非资源密集型
*   不同的缓存选项
*   第三方库和类可以很容易地集成
*   高质量的安全工具
*   行为、助手、扩展基本功能的能力等等。
*   适合任何规模和复杂程度的项目

#### [](#bad-sides)坏边

*   不是真正灵活的路由
*   前端和后端库纠缠在一起

*学习曲线*:平滑。你需要精通 OOP 和数据库，这就足够了。

Yii2 开发人员的平均工资:$ 75.000/年或更高
