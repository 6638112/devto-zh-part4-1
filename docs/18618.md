# 今天是星期五:让我们一起来听音乐吧！

> 原文:[https://dev.to/max/it-s-friday-let-s-dev-in-music-2gc9](https://dev.to/max/it-s-friday-let-s-dev-in-music-2gc9)

作为一名开发人员，我真的很喜欢边听音乐边编码(这是我们工作的好处之一)。我认为这有助于我集中注意力和减轻压力。

在工作中，当我环顾四周，只有一两个人没有戴耳机...😅

[![Pingu](../Images/4b35ec14774b786eee6e4a4460078840.png)T2】](https://i.giphy.com/media/ku5EcFe4PNGWA/giphy.gif)

每周五我都会和你分享一首我特别喜欢的歌。我知道大家不会喜欢我要分享的歌曲，但这不是主要目的。这些帖子的目的是交流这种在我们的工作中专注或放松的好方法(编码是有压力的，它因每个开发人员而异)。

为什么？我想我们都可以交流一下我们在工作中听的歌曲和音乐种类(以及为什么)。对我来说，它带来快乐和幸福(尤其是星期五)。让我们放松一下，享受一下生活吧！

今天，是这首歌“爱。我”出自艺人“司机”。它来自专辑《森林》,真的很不错。

[https://www.youtube.com/embed/q0AOk4CxYhw](https://www.youtube.com/embed/q0AOk4CxYhw)

请告诉我你是否喜欢这首歌，帖子的类型，分享你正在听的任何一首歌，以及音乐对你工作的重要性。
