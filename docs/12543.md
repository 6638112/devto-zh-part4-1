## [](#water-css)水 CSS

[![watercss](../Images/061e19f5f9a6f6088b5ec61b8fb90a3c.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Izw0Sale--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/5qshwdhgh2o4vsc3jd98.png)

Water CSS 允许自动将通用布局样式应用于网页。

这就像有一根魔杖，用它来美化我们的网络应用程序。watercss 大放异彩的典型场景是当我们有一个简单的演示项目，但我们没有(或不想)投入时间来设计它。

来自作者:

> Water.css 是一个不需要任何类的 css 框架。你只要把它放在你的头牌里，然后忘掉它，它会默默地让一切变得更美好。

还有两种主题可供选择，可以轻松地在亮暗模式之间切换:

```
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/kognise/water.css@latest/dist/light.min.css">

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/kognise/water.css@latest/dist/dark.min.css"> 
```

## [](#draggable)可拖动

[![draggablelogo](../Images/11e908172917812e2ff2a91c832bcc83.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--GjmzVJTT--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/n76hds6i67one7fbe5xf.png)

Draggable 是一个 JS 库，提供了一组拖放动作。对我来说，它的一些功能看起来有点极端，但我喜欢[排序列表](https://shopify.github.io/draggable/examples/simple-list.html)，允许以不同的顺序排列列表的项目，以及[交换](https://shopify.github.io/draggable/examples/flexbox.html)功能，在带有卡片的仪表板中很有用，我们可以交换它们的原始位置。

[![dragexample](../Images/ce2c33c9899c62a7729d02257b524ca4.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Q6mUCIQt--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/azwhhe3nzil7pgswjlf4.png)

[可拖的例子](https://shopify.github.io/draggable/examples/)

## [](#one-html-page-challenge)一个 HTML 页面挑战

[![logo](../Images/7080a14f28aff6d72c7cfabe65b16d92.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--h3vKKhj4--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/0r1raltr49d9xgd6qs0q.png)

这不是一个 JS 库，但是我非常喜欢这个想法，所以我想和大家分享。它包括在一个 HTML 页面上开发的疯狂而亲切的想法的集合，没有任何第三方框架的帮助。

> 目标是在一个 html 文件中创建你想要的任何东西。练习你的技能，不需要图书馆的帮助，不需要文件分离，也不需要现代框架的帮助。有了这样的限制，你还能有多大的创造力？

看到用非常基本的资源可以实现什么是令人惊讶的，当然这对于任何水平的开发人员来说都是一个很好的培训场所。

你会在那里发表你的单页作品吗？如果有，请在评论里写上网址，我很好奇能看到新的 spa！！

想获得一些灵感，看看现有的[作品](https://metroxe.github.io/one-html-page-challenge/?ref=devawesome.io)

## [](#animate-on-scroll-library-aos)上卷库动画(AOS)

[![aos-logo](../Images/035bf8628968819f6534e849c19078bb.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--bN8vrj9L--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/w8h106l2wm5nh2h2lkzr.png)

你是在做一个公司的登录页面，还是仅仅想在你的 web 应用程序中添加一些吸引人的动画？

AOS 是一个有趣的库，允许在滚动页面时轻松地动画元素。有大量的内置动画，但也可以为特定场景添加自定义动画。

## [](#autonumeric-js)自动编号 JS

[![autnumericlogo](../Images/715090749351a097a5ed60e2795467fc.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--BWpX6cmI--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/lix3dvuoi4a2abcahx8d.png)

Autonumeric 是一个可配置的 javascript 库，用于格式化数字和货币。该库带有预定义的设置，但它还提供了一个[网络配置器](http://autonumeric.org/configurator)，您可以根据自己的具体需求进行定制。
