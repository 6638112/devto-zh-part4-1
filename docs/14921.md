# 我的游戏开发经历

> 原文:[https://dev . to/Elena dotnet/my-game-development-experience-1le 0](https://dev.to/elenadotnet/my-game-development-experience-1le0)

《精灵宝可梦:蓝宝石》从一开始就是一个很棒的体验。我发现了很多生物、地图和敌人...而从那以后，我开始问自己:我能修改游戏本身吗？我能自己创造吗？

gamedev 列车整装待发！下一站:ROM 黑客和 RPG 制造商！

从这些第一步开始，我尝试了很多技术来创建新的视频游戏，如果你想知道更多，请跟随我一起经历这个不可思议的冒险！

### [](#pygame-making-gamessss)pygame，制作游戏 sss🐍

当我独自学习编程时，Python 被认为是“最值得学习的编程语言”，容易学习，容易看到结果...所以 Pygame 是一个不错的选择...真是一团糟！

你甚至可以看到这个小实验的视频:

[https://www.youtube.com/embed/oFxy94jwvVA](https://www.youtube.com/embed/oFxy94jwvVA)

### [](#game-maker-studio-old-but-gold)游戏制作者工作室，老而金！⚰️

接下来呢？游戏制作者 1，安息吧！用合适的游戏引擎开始创作游戏是一件很酷的事情。它包括一个拖放系统，结果很快出现。

但是...免费版的资源(精灵，关卡，脚本)非常有限，所以它没有在我的电脑上持续。

[https://www.youtube.com/embed/ZrSKGMZKB5I](https://www.youtube.com/embed/ZrSKGMZKB5I)

### [](#unity-3d-the-king)团结 3D，王者？👑

下一步是 Unity3D，从 3D 项目开始学习基础知识...C#是我在正规课程中学到的第一门编程语言，现在我是一名. NET 开发人员。大团圆结局！所以把这种语言用于应用程序和游戏是很酷的，可以强化知识！

[https://www.youtube.com/embed/Mrl4JodEEv0](https://www.youtube.com/embed/Mrl4JodEEv0)

### [](#godot-engine-the-titan)戈多引擎，泰坦🤖

我第一次听说这个游戏引擎是在 YouTube 的一个视频中，谈到这个开源工具即将取代 Unity3D 成为市场上的主要引擎。我对此非常怀疑，看着一些显示出来的功能，却看不到任何好处...我忘记了戈多。

最相关的细节是它是开放源码软件，但没有别的。

直到有一天一切都变了。那天我想做一个 2D 游戏，简单有趣的。而且 Unity3D 在制作 2D 游戏时感觉并不好，所以我决定尝试一些新的东西。我不打算回 Game Maker 工作室，现在去哪里？

戈多发动机是解决方案。

漂亮，开放和不断增长的社区，酷 IDE，脚本编辑器和游戏引擎共享同一个窗口，GDscript(基于 Python)...这与我过去对这个问题的看法大相径庭！

当时，一个新的稳定版本已经发布:Godot 3.1！有很多新的功能和变化需要发现，这个项目在开发中专注于良好的用户体验，照顾开发者和我们的需求。

[https://www.youtube.com/embed/N93Eqc4dIZg](https://www.youtube.com/embed/N93Eqc4dIZg)

## [](#the-future)未来🔮

也许一个新的、开源的、强大的游戏引擎出现了，我会再次坠入爱河，也许我会和戈多在一起很多年...谁知道呢？

无论发生什么，我都会确保继续学习，同时享受快乐...我希望你也是。

再见，感谢阅读！
