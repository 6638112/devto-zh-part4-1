# 可信应用与不可信应用

> 原文：<https://dev.to/wagslane/trustworthy-vs-trustless-apps-4l9b>

莱恩·瓦格纳

在关于脸书新天秤座区块链的[听证会](https://www.youtube.com/watch?v=h4aPp3wGufY)之后，我们都明白*值得信任的*和*不值得信任的*应用之间的区别，这比以往任何时候都更重要。

值得信赖的应用程序是其开发人员为社区所了解和信任的应用程序。开发人员的声誉和业务危在旦夕，因此他们有动力确保他们的应用程序没有错误和恶意代码。

一个不可信的应用程序是一个代码为[开源](https://en.wikipedia.org/wiki/Open_source)的应用程序，这样社区就可以亲眼看到代码几乎没有 bug，没有恶意代码。不可信应用程序由开发人员加密签名，这样用户就知道他们下载的应用程序对应于开源代码。不可信的应用程序也不会移动用户数据，除非它是由只有用户持有的密钥加密的。

脸书充其量是一款值得信赖的应用。他们说会保证我们信息的安全，我们唯一的选择就是相信他们。当他们不可避免地无法保护用户数据时，国会严厉批评他们，但却未能理解这个问题。用户数据应该是不可信的。

## 哪个好？

理想情况下，应用程序是可信的**和不可信的**。它是开源的，并适当地管理用户数据，这可以由用户和社区来验证。这也意味着开发人员不是匿名的，并且在出现问题的情况下是可用的。

然而，如果你必须在值得信任和不值得信任的应用程序之间做出选择，不值得信任的应用程序每次都更好。“*不要相信，核实*”是涉及到你的私人数据时最好的座右铭。

因此，比特币比银行更好，开源安全软件永远比专有解决方案更安全。注意安全，永远做好自己的研究！
