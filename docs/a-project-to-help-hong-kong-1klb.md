# 一个帮助香港的项目

> 原文：<https://dev.to/hkchat/a-project-to-help-hong-kong-1klb>

嗨，我最近雇了一个 fiverr 来帮我创建一个项目来帮助香港，因为从香港到主要通信平台的连接被 ddos 攻击干扰，抗议者无法通过网络联系，但开发人员只完成了一半，他没有解决 bug，有人能帮忙吗？

[https://github.com/HKCHAT/HKCHAT](https://github.com/HKCHAT/HKCHAT)

这是一个开放的项目，我希望把它做成一个 snapd 包，这样每个可以访问网络的人都可以建立一个服务器来帮助社区。
