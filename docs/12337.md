### [](#help-others-accessing-the-tor-network-using-the-snowflake-extension)使用“雪花”分机帮助他人接入 Tor 网络

Tor 项目已经发布了一个扩展，它允许你使用[你的浏览器作为一个叫做“雪花”](https://chrome.google.com/webstore/detail/snowflake/mafpmfcccpbjnhfhjnllmmalhifmlcie)的 Tor 节点。这样你就可以和那些只有严格审查的国家的人分享你的(希望)审查较少的互联网接入。你不需要担心别人做的事情，因为你的浏览器不会成为一个出口节点。它仅成为一个常规节点。该扩展可以很容易地打开和关闭，并显示一些统计数据。

这个扩展不是一个自由访问互联网的工具，它是为网络的贡献者服务的。要想不受审查地自由访问互联网，你需要使用 [Tor 浏览器](https://www.torproject.org/download)。

### [](#make-your-website-faster-by-prefetching-content-while-the-browser-is-idle)通过在浏览器空闲时预取内容，让您的网站运行得更快

[QuickLink](https://github.com/GoogleChromeLabs/quicklink) 是由谷歌的 Chrome 实验室发布和维护的一个库。它允许您在浏览器空闲时预取内容。如果用户没有使用慢速互联网连接或数据保护模式，viewport 中的链接是预先获取的。该库非常小，文件大小不到 1 KB。关于选项和所需安装步骤的更多信息，请参见开头链接的 GitHub repo。

### 在你搜索的地方做关键词研究

前阵子开始学习更多关于 SEO 的知识。这有时很棘手，因为您无法立即验证您的结果。但是可靠的信息肯定会有所帮助。昨天，我到处都发现了[关键词](https://keywordseverywhere.com/)。这是一个火狐& Chrome 扩展，在搜索时显示关键词数量，并在你的搜索结果旁边的框中显示替代选项。非常方便，如果你打算写一篇文章，想知道是否有一个替代词或短语，你最终会发现更多的关键字和想法。

### [](#make-your-website-screenshots-look-like-a-browser-for-free-using-browserframe)使用“BrowserFrame”免费让你的网站截图看起来像浏览器

BrowserFrame.com 是一款免费的在线工具，可以将你的截图或网址打包到浏览器框架中。通过这种方式，你可以在网站或 pdf 文档中更好地解释你的 SaaS 或网站。

### [](#tell-your-story-in-form-of-a-comic-using-storyteller)用“说书人”以漫画的形式讲述你的故事

传达你的副业想法可能具有挑战性。没有人喜欢充斥着大量文本的网站。视觉元素是正确的选择。大多数人选择讲解视频来解释更复杂的主题，但这一浪潮似乎放缓了。

一个有趣的选择可能是[“说书人”](https://storyteller.launchaco.com/)工具。它允许你根据上传的图片创作漫画，并添加语音气泡来讲述你的故事。该项目处于预启动阶段，但是如果您计划很快启动一个新项目，那么它可能是值得关注的。

### [](#earn-money-with-captchas)用验证码赚钱

每天我们都为谷歌选择红绿灯、摩托车等等。对网站管理员的好处是验证访问计算机是由一个真正的人控制的。对谷歌来说，更大的好处是:为人工智能和机器学习标记他们的照片。简而言之:我们在为谷歌做无偿工作。

为什么不使用用户在你的网站上完成的工作来为付费客户标记数据呢？这就是 [hCaptcha](https://www.hcaptcha.com/) 正在采取的方法。你可以加入 hcaptcha 作为 Recaptcha 服务的替代品，通过破解验证码来赚钱。机器学习公司可以更容易地让他们的产品贴上标签，你作为网站管理员也有了另一个收入来源。
