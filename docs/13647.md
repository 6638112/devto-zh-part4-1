# 你看过的最好的演讲。

> 原文:[https://dev.to/svemaraju/best-talks-you-ve-watched-2eom](https://dev.to/svemaraju/best-talks-you-ve-watched-2eom)

想知道我们是否可以分享我们最喜欢的关于编程、Devops、安全或软件世界的话题。

我就分享几个我比较喜欢的，大多和 Python 有关~

*   Brandon Rhodes 告诉我们 Python 字典的内部工作原理。最初是 2010 年的演讲，后来是 2017 年的演讲。它极大地帮助了我理解为什么它首先被称为散列表。😂更有趣的是，为什么 Python 字典过去是无序的。

[https://www.youtube.com/embed/oMyy4Sm0uBs](https://www.youtube.com/embed/oMyy4Sm0uBs)
[https://www.youtube.com/embed/66P5FMkWoVU](https://www.youtube.com/embed/66P5FMkWoVU)T4】

*   Ned Batchelder 的精彩演讲展示了在他的工作中用来解决现实生活事件的调试技巧。像布兰登·罗兹一样，内德·巴切尔德拥有高超的演讲技巧，这是一次非常有趣的经历。

[https://www.youtube.com/embed/bAcfPzxB3dk](https://www.youtube.com/embed/bAcfPzxB3dk)

*   艾莉森·卡普特关于虫子的演讲。这实际上在我目前的工作场所引发了一场非常有趣的讨论，引发了一场关于我们过去共同面对的讨厌的 bug 的对话。我发现关于特性标志和服务器端门控的部分非常有见地。这是我以前知道的事情，在实际解决像她描述的这种重大事件的背景下，我从来没有真正想过。

[https://www.youtube.com/embed/HuuYwUxM-ZY](https://www.youtube.com/embed/HuuYwUxM-ZY)
