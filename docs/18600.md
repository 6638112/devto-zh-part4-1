## [](#get-doodling)[拿涂鸦](https://glitch.com/~doodle-place)

黄灵东的“[涂鸦空间](https://glitch.com/~doodle-place)”是一个虚拟世界，居住着用户提交的、通过计算制作的动画涂鸦。任何人都可以画一些东西，然后在被添加到其他人的涂鸦填充的线框世界之前，这些东西会自动动画。

当然，这是互联网，所以用户提交的涂鸦经常是阴茎。尽管如此，这还是很有趣😅

[https://glitch.com/embed/#!/embed/doodle-place?previewSize=100&path=index.html](https://glitch.com/embed/#!/embed/doodle-place?previewSize=100&path=index.html)

## [](#make-a-zine)[制作杂志](https://glitch.com/~zine-machine)

我们喜欢 Glitch 的杂志——我们甚至制作了一些自己的。所以我承认对[罗文·梅莱伍德](https://glitch.com/@rowan-m)的《T2》杂志《T3》情有独钟。

这是一个制作杂志的模板，你可以在线阅读或者打印、剪切和折叠来制作你自己的实体版本。你知道，去读 IRL。这种巫术是通过特定于打印的 CSS 网格布局实现的。但是通过重新组合模板，你就在制作自己的杂志的道路上前进了一大步。

## [](#celebrate-emoji-makers)[庆祝表情符号制作者](https://glitch.com/~ft-emoji-league)

如果你的工作场所和我的一样，你的群聊就是一大堆自定义表情符号。嗯，看起来《金融时报》也很像这样，至少从 Bren Brightwell 的应用程序“T2”来看是这样的。这是 Slack workspace 自定义表情符号的排名表，列出了创建最多的人的名字。你可以重新组合，交换一些变量，看看谁在你的办公室里有更多的*时间。*

[https://glitch.com/embed/#!/embed/ft-emoji-league?previewSize=100&path=index.html](https://glitch.com/embed/#!/embed/ft-emoji-league?previewSize=100&path=index.html)

本周综述中的其他应用包括一个完全由表情符号组成的在线多人世界，一个显示页面预览的 favicon，以及一个最小的 i18n 库。[看看他们](https://glitch.com/@glitch/glitch-this-week-june-12-2019)！
