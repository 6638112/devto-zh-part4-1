# 为什么我们应该拥抱深度假货的发展

> 原文:[https://dev . to/emurtzle/why-we-should-enchange-the-development-of-deep-fakes-2g KH](https://dev.to/emurtzle/why-we-should-embrace-the-development-of-deep-fakes-2gkh)

我们应该拥抱深度假货的发展，因为不这样做会糟糕得多。在不久的将来，深度造假者将能够近乎完美地复制一个人，以至于普通人将无法区分真人和深度造假者。虽然大型动画工作室为高预算电影创造了令人信服的 CGI 人类，但深度假货为小预算用户打开了这种力量，从而为滥用打开了大门。潜在的滥用可能以各种不同的形式出现，从宣传和假新闻到修改或制作犯罪镜头，以及与某人一起制作或编辑色情内容。为了打击这种情况，我们需要让开发商继续制造深度假货，因为如果我们把他们推到地下，我们就没有足够的知识来打击他们。

在接下来的几篇博文中，我将探索深层假货及其应用的不同方面，好的或坏的，敬请关注。
