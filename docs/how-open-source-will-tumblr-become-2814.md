# Tumblr 会变得多开源？

> 原文：<https://dev.to/ben/how-open-source-will-tumblr-become-2814>

如果你没看新闻的话，[威瑞森把 Tumblr 卖给了 automatic](https://www.theverge.com/2019/8/12/20802639/tumblr-verizon-sold-wordpress-blogging-yahoo-adult-content)。威瑞森在收购雅虎后拥有了 Tumblr。Automattic 是 Wordpress 背后的公司。

价格在 300 万英镑左右，比雅虎的 11 亿英镑有了显著的下降。几年前刚付的。我认为 Tumblr 如此便宜的部分原因是，Tumblr 的持续维护成本可能足够高，以至于它本身就是一项负债，需要有人来承担。这就是为什么城市有时会以 1 美元的价格将一个旧仓库卖给愿意修理它并承担费用的人。

知道了所有这些，我很好奇是否有人对 Tumblr 的未来有什么想法，因为它现在被巨大的开源支持者所拥有。

马特·莫楞威格的博客文章含糊地提到了计划:

> 在我们平台的底层技术中，我认为有一些很好的机会来标准化开源 WordPress 技术栈，但 Tumblr 上的前端用户体验将按照自己的方式发展。

Tumblr 给我的印象是，它是开源出版和社交网络未来最有趣的平台之一。

显然，我们这个社区的人对这种东西很感兴趣。
