# Vue Js:我最宝贵的资源

> 原文：<https://dev.to/corvindesigns/vue-js-my-most-valuable-resources-260p>

学习 Vue 对我来说是一次不可思议的冒险。我珍惜每一个机会来学习这个奇妙的 javascript 框架，尤其是接受新的项目来测试我的技能和扩展我的知识。

每当我看到其他开发人员在推特上谈论从 Vue 开始，我不禁为他们感到兴奋，并希望分享对我有很大帮助的资源。我决定在这里分享我的列表，以帮助其他那些刚开始使用 Vue 或者渴望了解更多的开发者。

## [Vue JS 2 -完整指南](https://www.udemy.com/vuejs-2-the-complete-guide/?couponCode=ACAD_W)

首先，如果没有最好的导师之一马克西米利安·施瓦茨米勒的帮助，我在 Vue 将一事无成。他的 Udemy Vue 课程对于任何开始 Vue 之旅的人来说都是一流的。课程中的大多数视频都很短，充满了信息和热情，这有助于激励更多人。小测验和项目非常有趣，真的有助于巩固你刚刚学到的东西。如果你对 Vue 感兴趣，这门课是必须要上的。

## [【Vue JS 2】忍者教程](https://www.youtube.com/watch?v=5LYrN_cAJoA&list=PL4cUxeGkcC9gQcYgjhBoeQH7wiAyZNrYa)

另一位令人敬畏的教练 Shaun Pelling 在他的 YouTube 频道上提供了一个精彩的 Vue 介绍系列。关于 Shaun 的指导，我最欣赏的是他非常简洁，并且很好地分解了概念，不仅清楚地解释了需要什么样的编码，还清楚地解释了为什么需要编码。Shaun 在 Udemy 上也有一门 Vue 课程，其中加入了 Firebase: [用 Vue JS 2 & Firebase](https://www.udemy.com/build-web-apps-with-vuejs-firebase/?couponCode=NINJAYT18) 构建网络应用

## [【full stack view】](https://www.newline.co/vue/)

由于 Vue CLI 3 的发布和即将到来的 Vue 3，我对在这里推荐一本书持观望态度，但这里有大量的内容提供了坚实的 Vue 基础。从那里，你可以去官方文档了解最近的 Vue 更新，没有任何问题。pdf 版本确实提供免费更新，因此可能值得关注。精装本可以在 **[亚马逊](https://www.amazon.com/Fullstack-Vue-Complete-Guide-Vue-js/dp/1987595297/ref=sr_1_1?keywords=fullstack+vue&qid=1568184958&s=gateway&sr=8-1)** 购买。

## [【vista master】](https://www.vuemastery.com)

虽然我不确定这是它所宣传的学习 Vue 的“终极”资源，但这里有很多东西可以让你开始使用 Vue 并建立一个坚实的基础。Vue Mastery 提供基于层级的订阅选项，以及免费访问的现场培训、精彩的会议讲座和播客。

## [弗拉维奥·科普斯的 Vue 手册](https://flaviocopes.com/page/ebooks/)

只要注册接收他的信息丰富的时事通讯，你就可以获得包括 Vue 手册在内的八本电子书。我还没有读完整本书，但是我所读的非常有帮助，写得很好。Flavio 为开发社区做出了如此巨大的贡献，这本书也不例外。支持他和他的工作。在 [Twitter](https://twitter.com/flaviocopes) 上关注他。看他的教程文章。他是一个伟大的开发者，提供了大量有用的信息。

## [Vue.js 官方文档](https://vuejs.org)

最后，直奔源头是不会错的。作为一个很少看说明书的人，我可以告诉你，在你的发展道路上不这样做只会阻碍你。文档整理得非常好，我觉得阅读起来很愉快。你还可以找到工作列表的链接，Vue dev 社区是一个很棒的社区。

我也确实想劝一件事就此了结。在深入研究 Vue 或任何其他框架之前，如果您对一些核心 JavaScript 概念有了很好的理解，这将使您的旅程变得更加容易。在学习 JavaScript 框架之前，另一位伟大的讲师 Brad Traversy 在他的视频 **[中做了出色的工作...](https://www.youtube.com/watch?v=Cm3U-NgJb9I)** 。

Vue Js 是一个漂亮的框架，使用起来非常有趣。我希望你们能和我一样从中找到快乐。编码快乐！
