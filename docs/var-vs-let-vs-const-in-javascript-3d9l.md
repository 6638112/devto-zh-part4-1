# Javascript 中的 Var vs Let vs Const

> 原文：<https://dev.to/speedysense/var-vs-let-vs-const-in-javascript-3d9l>

在本文中，我们解释了 JavaScript 中的 var vs let vs const。此外，我们还将了解 JavaScript 中的 var、let 和 const 关键字之间的区别。所有这三个关键字都用来在 JavaScript 中创建变量。首先，你必须了解 var 关键字，才能掌握 let 和 const 关键字的好处。所以让我们一个一个的开始。

Var vs let
首先，Var 和 Let 关键字的主要区别，var 是 follow 函数作用域，而 Let 是 follow 块作用域。var 可以在全局作用域或函数作用域中可用。let 只能在封闭块{}中访问，如在 for 循环或 if 条件中。

Let vs Const
const 关键字除了一点不同之外，几乎和 Let 关键字一样。不能给常量变量重新赋值。如果您尝试重新赋值，它将抛出语法错误(值已经声明)。
