首先，最好了解一下 JVM、JRE、JDK、IDE:

**1。JVM :**

Java 虚拟机是 Java 的核心。它是一个虚拟机，使计算机能够运行 Java 程序以及用其他语言编写的程序，这些程序也被编译成 Java 字节码。

**2。JRE :**

这是最常见的 Java 环境安装在个人电脑的笔记本电脑和台式机从因素。包含 JVM 特性的手机和智能手机最有可能包含运行应用程序的 JVM。

**3。JDK :**

简单地说，JDK 是 Java 程序的一个实现。JDK 包括一个私有的 JVM 和一些其他的资源来完成一个 Java 应用的开发。

**4。这里是 T1**

集成开发环境(IDE)是一种软件应用程序，为计算机程序员提供软件开发的综合设施。像 IntelliJ，Netbeans，Eclipse，...。
