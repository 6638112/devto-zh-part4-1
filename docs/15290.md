# 一些关于建议的建议

> 原文:[https://dev.to/alexantra/some-advice-on-advice-3o0g](https://dev.to/alexantra/some-advice-on-advice-3o0g)

作为一个自我完善的团体，我们不断地从任何我们能找到的地方寻求建议。

但并不是所有的建议都是好建议，以下是我要错过的建议:

# [](#avoid-broad-sweeping-declarations-to-avoid-x-over-y)避免宽泛的笼统声明，避免 X 超过 y。

你可能还没意识到，我们并没有在打仗。你不是发誓忠诚的卡布利特家的仆人。没有人会号召你拿起武器在街上与 PHP 战斗，这是不可能的。所以你不需要开始在你的手臂上刻 NodeJS 并谴责任何你认识的使用 PHP 的人。

你也不会因为只学了一点点就变得擅长某件事。你可以走捷径获得一项成就，但你不能走捷径获得知识和理解。

现在互联网上充斥着这类文章，见鬼，只要在 Dev.to 上搜索“web components”就行了。我的经验是:如果人们在互联网上讨论它，那么它仍然值得学习。唯一不值得学习的是那些没人再谈论的东西，还记得汇编语言吗？

一个好的开发者理解好的实践。为了理解好的实践，你需要知道什么是坏的实践。你需要这种整体观点，我甚至建议你先学习“坏”语言，以帮助你理解为什么“好”语言更优越。

大多数时候，当你肩负着学习的使命时，它不仅仅是为了大便和傻笑。如果你希望在这个领域找到一份工作，那么你不能不学某些东西。仅仅因为 BLAH 语言现在是 GLEH 的最佳实践，并不意味着许多招聘公司不使用 GLEH。公司和他们的技术栈通常都很老，并且倾向于坚持他们五年前做出的决定。了解 2019 年的语言不会帮助你找到一份支持 2015 年系统的工作。

我在数据领域经常看到这种情况。想盲目搜索数据的人会看到一篇又一篇的文章，说‘不要学 SQL，学 Python、NoSQL 或 R 吧’，但实际上这个建议是最具误导性的。首先，大多数公司只有一个简单的基于 sql 的数据仓库，除了 SQL 之外的任何知识都不会得到支持。更不用说 python 只是传递 SQL 命令，所以还是要懂 SQL 才能知道如何在 Python 中提取数据。

# [](#take-more-advice-from-the-faceless-and-nameless)多向不知名的无名之辈取经。

互联网是一个游戏，看看 LinkedIn 就明白我的意思了。你要么努力打造自己的品牌，要么根本不在乎。

品牌建设者会尽一切努力在你的眼中使自己合法化。问题是，这非常容易，因为他们没有责任。我可以在我的 Dev.to、CV、LinkedIn 上写我懂 JavaScript。没有测试证明我知道它。品牌建设者知道如何玩这个游戏。大量的追随者，高质量的发帖，持续的活动，品牌交易，这些看起来都不错，但毫无意义。至少超过了 30%,这是一场竞赛...到底部。

液体错误:内部

我曾经遇到过“数据中你应该关注的十个人”之一。除了两个人，他们都给我发了一条自动推特消息，期待与我交流。在我的时间线被包含至少 6 个标签和 4 个命名标签的大量转发和交叉推广淹没两周后，我取消了所有的关注。

最好的建议来自那些根本不在乎自己网络品牌的人。他们只是回答问题或给出自己的意见，然后继续前进。

这就是为什么最好的建议来自评论，而不是 Youtube——永远不要看 Youtube 的评论。Dev.to / Twitter / Stack 上的评论是由他们所处的环境来调节的。有人说了一些愚蠢的话，你可以立即看到周围的话语。有人说一些好的，同样的，上升的投票和评论告诉你这是一件好事。当一个品牌建立者发布彻头彻尾的废话时，他们没有节制，因为说“嗯，这是错误的”的评论被埋在所有机器人和品牌建立者希望舔一舔那甜蜜的肛门之下。

# [](#a-great-person-may-not-be-able-to-give-great-advice)伟大的人不一定能给出伟大的建议

教书很辛苦。人类容易忘记事情的倾向经常导致人们忘记他们在学习过程中遇到的困难和障碍。

有多少次你寻求帮助，他们说的第一句话是“哦，这很容易”，然后你茫然地盯着他们十分钟，因为你不明白，你害怕承认不知道一些“容易”的事情？

当人们说“哦，这很容易”时，他们的意思是“对我来说，以我的经验和知识...这很容易。”这是一个鸡巴举动，但不是故意的恶意。人们很难记住事情有多难。

人们可能也不知道他们是如何走到现在的。有些人通过运气和时机获得成功，有些人获得成功是因为他们本身就是前 10%的人。在这两种情况下，他们都不能给你切实的建议。

你也是一个不一样的人，不同的学习风格，不同的性格。只接受一个人的建议可能不适合你。在所有情况下，大众都可以提供你的建议。
