# 树莓派？

> 原文:[https://dev.to/aliyalewis/raspberry-pi-124d](https://dev.to/aliyalewis/raspberry-pi-124d)

树莓派到底是什么？也有蓝莓和苹果派吗？

我听过它在学校被谈论，在一个电视节目中被引用并扮演了关键角色(机器人先生具体来说是的，你绝对应该看！)，看了 Youtube 视频和现场演示，但我仍然不确定这是什么。那么，什么是树莓派呢？根据 Raspberry Pi 基金会:

这是一台低成本、信用卡大小的电脑，可以插入电脑显示器或电视，使用标准键盘和鼠标。从浏览互联网到制作电子表格或玩游戏，它能够做你期望台式电脑做的一切事情。”

[![](../Images/cdf05a94557db80487df3516d1b70807.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--wFobXQuV--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/da080bne7z2qluz029fn.jpg)

好吧，现在我对它有了更好的理解，为什么我要一个？事实证明，你可以用覆盆子酱做几乎无穷无尽的事情，但我只列举几个:

*   制作一个数码相框。

*   做一个机器人咖啡机。

*   建立一个老学校街机游戏。

*   流网飞和亚马逊视频(你可以看机器人先生顺便说一句)。

*   练习/提高你的编码技能。

“建立一个旧的学校街机游戏”听起来很有趣，但是我从哪里开始呢？我输入了“如何用树莓派开发街机游戏？”在谷歌搜索栏和几乎太多的信息弹出！有无数的 Youtube 视频/教程、博客帖子、网站等。那会给我指出正确的方向。我还看到树莓 Pi 工具包的价格从 27 美元到 90 美元不等(附件可以额外购买)。

[![](../Images/28e9821f2516227999ae9e95a5877ecc.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--rurxi2Zf--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/zvmcdtt3vpep24gdyki7.jpg)

在接下来的几周里，我将和学校的一个伙伴一起做一个项目，用树莓做一个东西！你们有人用它造过东西吗？怎么样了？有什么建议/窍门/意见吗？下面让我知道！
