> Nest(或 NestJS)是一个用于构建高效、可伸缩的 Node.js 服务器端应用程序的框架。它使用渐进式 JavaScript，由 TypeScript 构建并完全支持 TypeScript(仍然允许开发人员用纯 JavaScript 编写代码)，并结合了 OOP(面向对象编程)、FP(函数式编程)和 FRP(函数式反应编程)的元素。

当你看到基础知识和 NestJs 是如何组织的。你会很快看到与棱角分明的建筑的相似之处。多亏了那个和打字稿。我们用 Express 和 Node 编写后端项目有一种标准的风格(Nest 构建在#Express & #Node.js 之上)。
每一次从前到后的交互和切换都是那么的自然。编码运动和思维模式中的纯诗。
