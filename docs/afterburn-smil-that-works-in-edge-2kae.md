# 加力-在边缘工作的 SMIL

> 原文：<https://dev.to/divwidget/afterburn-smil-that-works-in-edge-2kae>

在 divWidget 中创建，将点移动和渐变动画添加到一个简单的星形多边形中。

这个动画可以在包括 Edge 在内的所有现代浏览器中运行。

在可视化编辑器中打开这个模板:[http://bit.ly/2FoJWTR](http://bit.ly/2FoJWTR)
