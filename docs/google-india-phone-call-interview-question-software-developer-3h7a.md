# 谷歌印度—电话面试问题(软件开发人员)

> 原文：<https://dev.to/dg92/google-india-phone-call-interview-question-software-developer-3h7a>

在我回答你的问题之前，这次面试是为了一个软件开发人员的职位，主要围绕 javascript 工作。

注意:我还在面试过程中，所以不确定我是否能参加面对面的面试🤞，祝我好运。

电话面试持续 25 分钟，其中 2-3 分钟用于解释该角色的职责，之后她问了大约 12 个问题，这些问题混合了我的简历、基本数据结构、算法和分析。

*注意:我围绕 javascript 概念、Nodejs、React、React-native、GraphQl、系统架构—@[http://overflowjs.com](http://overflowjs.com)
撰写文章，随时检查和改进您的技能或评论，涵盖您选择的任何主题。*

那么，让我们开始吧👇

1.  说说你自己？
2.  你研究过哪些技术？
3.  为什么我在过去的三年里换得如此频繁？
4.  你从零开始做过什么系统设计吗？
5.  快速分拣的最坏情况以及最坏情况何时发生？
6.  合并排序的最坏情况以及何时会发生？
7.  快速排序和合并排序哪个更快，什么时候，为什么？
8.  什么是哈希表，它与字典有什么不同？
9.  什么是平衡的二叉查找树？为什么它们被称为平衡的？
10.  在 javascript 中，解析是通过值或者引用来完成的，为什么呢？
11.  你将如何解决问题 2 的 24 次方？
12.  有问题要问我吗？

注意:如果你想让我写下这些问题的答案，请在下面评论👇

最后，她告诉我等一周，因为她会将答案传达给团队，并会带着反馈回来。

希望上面的问题能帮助那些想了解谷歌电话面试问题的人。
