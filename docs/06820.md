# 分享技术链接-一个分享技术资源的平台

> 原文:[https://dev . to/bharatdwarkani/share-tech-links-a-platform-to-share-tech-resources-2b 6](https://dev.to/bharatdwarkani/share-tech-links-a-platform-to-share-tech-resources-2ob6)

**分享技术链接** —一个向社区分享文章、博客、教程和软件技术资源链接的平台。

我最近开发了一个基于网络的应用程序，完全由我来构思、设计和后端。这个概念不同于其他博客/问答网站。这里只有链接可以用简短的描述来分享。

**URL**—[https://share tech links . com](https://sharetechlinks.com)

**使用了以下技术**

*   ASP.NET 核心 MVC
*   MySQL -后端
*   材料设计
*   托管在 Linux 服务器上

**特性**

*   标签支持
*   根据适当的主题对帖子进行分类
*   动态-每个页面运行使用后端没有静态内容
*   任何人都可以分享-在版主批准后将对公众可见。
*   游戏化-每个帖子的贡献者的信用
*   谷歌登录
*   SEO 优化
*   速度优化
*   渐进式网络应用
*   移动友好
*   书签功能
*   喜欢/不喜欢邮报
*   用户配置文件
*   用于排序的过滤器-最近、受欢迎程度、最多查看次数

在这个网站中，技术资源被很好地分类成跨越几种软件技术的主题和标签。

提供了各种技术的主题

*   一般
*   GitHub —项目链接
*   软件和工具
*   ASP.NET 核心
*   有角的
*   机器人
*   java 描述语言
*   数据库
*   Xamarin
*   布鲁兹
*   Web 开发
*   用户界面/UX 设计和更多未来

**创建帖子**

*   任何人都可以提交帖子，因为提交帖子需要使用 Google 帐户登录。
*   你可以分享一个博客文章的链接，与指定类别相关的文章。
*   创建的帖子将被发送给管理员审批。你将获得每个批准的职位 5 分
*   如果你没有找到合适的类别，你可以在笔记中。

*   应用程序通知
*   关注用户/类别

*   安卓应用
