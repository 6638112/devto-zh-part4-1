# 用儿童读物在 Twitch 上学习 Python！

> 原文:[https://dev . to/jimbobbennett/learning-python-on-twitch-with-a-kids-book-4130](https://dev.to/jimbobbennett/learning-python-on-twitch-with-a-kids-book-4130)

**在[twitch.tv/jimbobbennett](https://www.twitch.tv/jimbobbennett)T3 跟着我转圈**

我认为通过 Twitch 学习 Python 会很有趣，这样其他人可以看到我是如何学习的，并且可以跟着学习。有很多伟大的流，假设一些知识，以充分利用他们，但这个流回到 101 水平，这是伟大的总新手。

我关注的书是:

[初学者编码:使用 Python](https://amzn.to/2JfRBoZ)

[![](../Images/2cfce4d1b4f7e5a483aebe7f5e5ece21.png) ](https://www.amazon.co.uk/gp/product/1409599345/ref=as_li_ss_il?ie=UTF8&psc=1&linkCode=li3&tag=expecti-21&linkId=e7332562f8fb28b9188745a35472f86f&language=en_GB) [ ![Coding for Beginners: Using Python book cover](../Images/9f2b6e4f87f03d205ebabb0a6d5a9964.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--YUL7w9_G--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://ir-uk.amazon-adsystem.com/e/ir%3Ft%3Dexpecti-21%26language%3Den_GB%26l%3Dli3%26o%3D2%26a%3D1409599345)

这是来自于 Usbourne 出版社，他们出版的儿童书籍充满了有趣的插图和笑话，内容非常通俗易懂。我小时候很喜欢它们，我六岁的女儿现在也很喜欢它们——尤其是它们在太空书里面的样子。

本周早些时候，我做了我的第一个流，不幸的是，我不知道勾选复选框，以保存在 Twitch 上或发布到 YouTube 上——作为 Twitch 新手的诅咒！
