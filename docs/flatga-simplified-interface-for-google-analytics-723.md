# 谷歌分析的简化界面

> 原文：<https://dev.to/kambanthemaker/flatga-simplified-interface-for-google-analytics-723>

我想向你介绍我的产品， [FlatGA](https://flatga.io) ，谷歌分析的简化界面。我曾经构建了许多小产品，并查看 Google Analytics 来了解性能。我总是想要没有 GA 给你的信息过载的高水平度量。有一天，我决定利用谷歌分析 API，用 VueJS 前端和 NodeJS 后端构建 FlatGA。

我们今天在 ProductHunt 上发布了该产品，[产品搜索页面](https://www.producthunt.com/posts/flatga)，请在那里分享您的反馈。

## 一些重要的特征

1.  能够在单页界面中查看核心指标
2.  你可以下载排名靠前的推荐页面，以及你表现最好的页面
3.  除了您的公共电子邮件/访问令牌之外，不会保存任何其他信息。每条信息都是从谷歌分析中实时获取的！
