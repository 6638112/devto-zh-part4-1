# 今天是星期天，你在编码吗？

> 原文:[https://dev.to/andrewbrown/its-sunday-are-you-coding-3c7p](https://dev.to/andrewbrown/its-sunday-are-you-coding-3c7p)

星期天，我接到一个电话。时间表已经更改，新的演示已排定。大多数人在周末被工作召唤时都很生气。当我接到电话时，感觉就像蝙蝠信号，我马上行动起来。

我每天都编码。这对我很有效，因为我是个工作狂。我会不高兴，不编码。

我的问题是，你通常在星期天编码吗？
