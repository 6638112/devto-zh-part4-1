# 如何开始播客

> 原文：<https://dev.to/supremerumham/how-to-start-a-podcast-416o>

播客是接触新观众的好机会，但是开始播客的过程可能会很复杂。在这篇文章中，我将带你经历开始播客的过程，并为你提供开始播客的资源。

为你的播客创建一个电子邮件帐户
播客是基于 RSS 源的。任何人都可以查看 RSS 源。为您的播客创建 RSS 源时，您的电子邮件会被添加到 RSS 源中。这意味着任何想购买播客相关产品或播客垃圾邮件的人都可以很容易地找到你的电子邮件。你的播客越受欢迎，你就会收到越多的垃圾邮件。为您的播客创建一个电子邮件帐户可以防止垃圾邮件塞满您当前的收件箱。你可以使用 [Gmail](http://gmail.com) 或者 [ProtonMail](https://protonmail.com) 。

**练习**
下载主播 app 或使用语音备忘录，开始谈论任何事情。然后，听你录的。倾听时，你在寻找几种不同的东西:

你说话的音量:你说话的声音是太轻还是太大？你的语速:你是不是说得太快了？
清晰度:是否有任何故障或声音问题？
背景噪音:背景中有什么东西会干扰你的音频吗？
选择风格
一旦你练够了，下一步就是选择风格。有许多不同风格的播客。独白、采访、圆桌讨论等等。如果你决定采访别人，最好是尽快开始接触别人。

**获取设备**
要录制你的播客，你需要一个麦克风、耳机和录音软件。向我推荐的麦克风是 Audio-Technica ATR2100-USB。麦克风也需要一个 pop 滤波器。任何适合麦克风的 pop 滤波器都可以。

耳机在录制时使用，以聆听您的播客在录制时的声音。你可以使用任何一副耳机。

对于自己录制，你可以使用 [Audacity](https://www.audacityteam.org/) 来录制和编辑你的播客。Audacity 可用于在录制音频期间或之后提高或降低音频的音量。这款软件是免费的，网上有很多关于如何使用它的教程。你可以在这里看一个基础教程。

如果你决定为你的播客采访其他人，你可以使用 [Zencastr](https://zencastr.com) 。您可以免费录制长达 8 小时的音频。

**挑选一个名字和封面艺术**
接下来，你应该挑选一个名字。你的播客的名字应该让找到它的人对你的播客有一个基本的概念。没有必要在你的标题中出现“播客”这个词，因为这是多余的。举个例子，我的播客叫做*建立独立企业*。任何偶然进入我的播客的人马上就知道我的播客是一个商业播客。

选择名字后，你需要为你的播客设计封面。你的封面应该是明亮的颜色，如红色、蓝色或绿色。让你的封面成为亮色将有助于你的播客脱颖而出。当有人在浏览播客列表时，你明亮的封面更容易吸引他们的眼球，他们可能会因此而看你的播客。你可以使用 [Canva](https://www.canva.com/) 或 [Figma](https://www.figma.com/) 来创作封面艺术。

我做了一集关于我的播客的命名和封面艺术的决定，你可以在这里查看。

**录制播客预告片**
为了让您的播客被 iTunes、Google 和 Spotify 接受，您的 RSS 源不能为空。通过录制播客预告片，你可以避免在发布节目时出现问题。预告片是给你的听众一个更详细解释你的播客的机会。

**上传你的音频给主播**
此时你已经有了一个名字、封面、播客预告片，也许还有一集真正的剧集。是时候在 Anchor 中创建你的播客了。我有一个视频引导观众在主播上创建播客。你可以在这里查看[。](https://www.youtube.com/watch?v=q0ZX-ysqRiI)

将你的播客添加到 iTunes
一旦你创建了一个主播账户，你就应该有你的 RSS 源了。这里你有两个选择，你可以允许主播上传你的播客到每一个播客应用程序，或者你可以自己做。区别在于控制。

如果你在上传播客时犯了一个错误，比如节目笔记中的语法错误或选择了错误的时间，如果你自己上传播客，你可以自己解决它。如果你允许主播上传你的播客，你必须请求他们帮助修复这些错误。我有另一个视频向观众展示如何在 iTunes 中添加播客。

将您的播客提交到 iTunes 后，您需要等待至少 3 或 4 天才能被添加到 iTunes。然后，再过 3 到 4 天，你的播客就会被添加到其他播客应用程序中，如 covery 或 Castro。

-

在等待之后，你终于可以在 iTunes 上拥有一个播客，你可以和你的朋友和家人一起开始。加入一些相关的社区也是一个好主意。有几个播客社区会给你不同播客主题的建议。一个很棒的例子是在 Telegram 上打开播客。新公司正在进入这个领域并做出改变。播客和播客行业的未来非常光明。你应该参加今天的运动。
