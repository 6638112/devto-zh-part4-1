# 为什么工作时不应该戴耳机

> 原文：<https://dev.to/t4rzsan/please-don-t-use-headphones-when-working-261g>

有很多很好的理由让你在编码的时候戴上耳机听音乐。它们阻挡噪音，帮助你集中注意力。也有许多令人信服的理由说明你不应该这样做。

> 声明:以下是我自己的经验，可能不科学正确。

以下是你应该摘下耳机的三个原因:

*   对噪音更敏感:如果你一直戴着降噪耳机，最终无噪音环境将成为你的新常态。没有耳机你会很难集中注意力。
*   **错过社交**:在你的工作场所，你的同事会在周围谈论、分享知识和开玩笑。如果你戴着耳机，你会错过，你会在你和你的同龄人之间制造一个无形的障碍，让他们更难让你参与进来。
*   **听力损失**:这是最重要的一点。我知道我听起来像你的父亲，但使用耳机会大大增加听力损失和耳鸣的风险，这两种疾病都无法治愈。当我开始为似乎很久以前的事情编码时，我会戴上耳机来屏蔽噪音，集中注意力。可悲的是，我现在耳鸣。在日常生活中，这并不困扰我，但当我累了或压力过大时，这种情况会变得更糟，然后变得非常烦人。我知道当你年轻的时候你认为你是不可战胜的，但是请小心。

如果你在一个开放的办公室工作，噪音很大，戴上耳机可能会有很大帮助。但是拜托，拜托拜托，听听这个脾气暴躁的老头，尽量少做 [![🙃](img/7a51b6ec15b8e6b48dced9dda75961f1.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--A7JEkocc--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://s.w.oimg/core/emoji/12.0.0-1/72x72/1f643.png)
