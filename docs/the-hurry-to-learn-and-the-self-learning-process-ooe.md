# 学习的紧迫感和自学过程

> 原文：<https://dev.to/pachicodes/the-hurry-to-learn-and-the-self-learning-process-ooe>

我不久前开始学习编码，是在去年 10 月的某个时候。那意味着我已经学习了 8 个月。现在，我必须承认我经常感到绝望。
我想成为一名开发人员并开始在该领域工作的梦想感觉很遥远，只是一个梦想。
目前，我对 HTML 和 CSS 非常了解，我正处于用 JavaScript 真正编程的第一步。没有什么比我在我最喜欢的电视节目 lol 中看到的所有代码更酷了(机器人先生< 3)
但是，大学生通常需要 4 年才能毕业...4 年！
为什么在 7 个王国里，我会相信 8 个月的自学就能为市场做好准备？
