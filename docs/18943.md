# 自学开发人员资源

> 原文:[https://dev . to/Angela RAE 63/自学-开发者-资源-2pjk](https://dev.to/angelarae63/self-taught-developer-resources-2pjk)

只是想把我在《生命密码》找到的一段甜蜜的播客传给大家。[妮可·阿香博](https://twitter.com/lavie_encode)是一名自学成才的中级网络开发人员、博客作者和播客主持人，她致力于支持自学成才的程序员。我已经下载了她关于[的系列文章“剖析网络开发者路线图”](https://www.lavieencode.net/podcast/030-dissecting-the-web-developer-roadmap-basics/)到目前为止，它对我理解关键概念非常有帮助。所以，如果你和我一样，在网络发展的高速公路上感觉像车灯前的小鹿，妮可就是你的女孩。在这个系列中，她解开了术语的谜团，比如 Git、SSH 和语义版本，并在第二和第三集继续解释前端和后端。我不得不说，知道有像妮可这样的人在我身边，我感到很欣慰。
