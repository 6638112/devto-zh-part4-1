你好，戴夫！

几个月前，我发布了我的[三件好事](https://three-good-things.glitch.me)应用程序，我非常高兴，人们似乎发现它很有用。那个月晚些时候，我部署了一个[忧虑树应用](https://worry.netlify.com)的 m1，我经常用它来摆脱思维循环。

我发现我真的很喜欢制作这种工具，并认为是时候建立一个简单的网站来将它们汇编在一起了。

于是在我最骄傲的时刻，我购买了 [weareanxio.us](https://weareanxio.us) (懂了？焦虑？ *anxio.us* ？我很高兴我的搭档认为我很有趣。)昨天，[我部署到 Netlify](http://we-are-anxio.netlify.com) 。

我既兴奋又自豪——这是我第一次从 Github 成功部署😂虽然设计比我最初的草图有所迭代，但看到我的愿景在世界上实现真的很酷。

除了《三件好事》和《烦恼树》,还有 m1 瓶“冷静下来”,以及一些我正在做的其他事情的预览。在底部，还有一个精神健康和健康的资源列表。

感谢 DEV.to 一路上的鼓励和帮助，我希望你能在这里找到有用的东西！
