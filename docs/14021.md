## [](#what-are-the-main-challenges)主要挑战有哪些？

相同元素的不同样式。我尝试将背景作为内嵌样式添加到元素中，这样我就可以使背景动态化，因为我必须传递一个 url 链接到该背景规则，之后我创建了一个名为`.card`的类，并将其放入该元素中，我还添加了一些其他规则，如背景大小、背景覆盖和背景位置，以保持图像的纵横比，而不管容器的宽度和高度。当我试图用 Chrome Devtools 检查问题时，就会发生这种情况。
