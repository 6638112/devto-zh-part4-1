## [](#1andrew-ian-and-paul-gower-runescape)1。安德鲁、伊恩和保罗·高尔-江湖

[![RuneScape](../Images/4f70f81a416ad147071a6041aefb1803.png "RuneScape")T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--EBioJmWG--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://massivelyop.com/wp-content/uploads/2015/05/rsos-696x229.jpg)

由高尔三兄弟创建并于 2001 年出版的《runescape》是一款 MMORP 或更广为人知的大型多人在线角色扮演游戏，如果你玩过任何角色扮演游戏，就很容易理解你是一个出生在名为 Runescape 的世界中的人，你有不同的技能、级别和任务可以做，一切基本上都是用鼠标控制的！这样一个简单的游戏机制和它的 afk 能力使得这个游戏非常受欢迎，并且几乎在任何时候都有超过 70，000 的活跃用户在线！尽管高尔兄弟已经离开了 jagex 公司，但他们都非常富有，总价值超过 3.05 亿美元...

然而，我们在这里谈论是什么使他们的游戏如此具有革命性..让我们从它是用什么写的开始...

“我们用一种叫做 RuneScript 的脚本语言编写，这是我们的游戏引擎团队专门为制作 RuneScape 内容而开发的。然后，基于 Java 的游戏引擎读取这些 RuneScript 代码，并使其正常工作。随着我们向 RuneScript 添加新功能以允许它做新事情，rune script 也在不断变化和扩展。”- Mod John Jagex 开发团队

没错，runescape 是用自己的语言编写的，类似于 javascript 和 wiki html，它使用 java 引擎将代码翻译成 html

“嗯，在语法方面，它类似于任何 C 风格的语言，除了变量名以它们的类型为前缀(例如，%varname 是一个整数)。我们还缺乏完整的阵列支持。除此之外，它几乎可以做任何你在 C 或 Java 中能做的事情。”- Mod Chris E Jagex 游戏引擎 QA

## [](#3-alex-seropianbungie)3。亚历克斯·塞罗皮安/邦吉

从小到大，我认识的每个人都玩过《光晕:战斗进化》，直到我写这篇博客之前，我从来没有想过嘿，我想知道《光晕》是用什么代码写的，《光晕》当然是有史以来最好的第一人称射击游戏系列之一，至少是 bungie 制作的原版系列，新的《光晕》是否真的那么好还有待商榷，但我们在这里再次不是要谈论我对视频游戏的精英主义观点...但我们在这里谈论的是游戏背后的代码和背后的人...Bungie 是由 Alex Seropian 建立的，他们第一个真正成功的游戏是神话..我不太了解神话，但它给了他们足够的牵引力，被微软从 take two 收购，这导致了制作 halo，实际上这是我开始了解到 Runescape 有自己的语言实际上是一个异数，Bungie 就像大多数游戏开发公司使用 C++和 Java 创建 Halo，并使用相同的引擎作为 Doom，Call of duty， 哪个是雷神之锤 3 引擎，如果你们还记得那个游戏的话，它基本上是所有这些游戏的精简版，它们是作为大多数现代第一人称射击游戏的测试游戏制作的，这种认识让我得出了我的最终结论，即不是所有的游戏在开发方面都是原创的，但它们都是建立在已经存在的引擎之上的代码，然后从那里开始制作会更容易一些 当你不需要从头开始创造所有的东西，比如动作和角色模型时，你想要的游戏 ..尽管大多数游戏开发者最终都制作了自己的模型，但他们尽量避免创建游戏引擎...无论如何，回到光环，光环将是 2000 年代早期至中期最畅销的游戏之一，并最终会在前面提到的使命召唤中发布..另一款基于相同引擎的游戏也是用 C++编写的，不过该引擎可以处理 C、C++和 Python..python 游戏在任天堂系统上更常见，那里的图形负担不是很重，虽然我还是不确定为什么这对语言本身很重要，但这是我发现的。因此，当我沿着这条关于游戏开发者和他们的代码的道路前行时，我发现了我的下一个大目标...游戏引擎！！即将向你走来
