## [](#microsoft-365)微软 365

重要的事情先来！微软 365 是什么？

在选择云协作平台(Office 365 就是一个例子和领先平台)时，公司的主要关注点之一是信息的安全性。Office 365 面临的最大挑战之一与以下事实有关:将信息存储在云中比存储在组织内部更不安全。

微软推出了一个许可模式，其中包括一个令人印象深刻的安全功能集，名为 [Microsoft 365](https://www.microsoft.com/en-us/microsoft-365/) 。这是一个捆绑包，包括:

*   Windows 10
*   办公室 365
*   企业移动性+安全性

这个包有三种不同的风格:

*   [微软 365 业务](https://www.microsoft.com/en-us/microsoft-365/business)
*   [微软 365 企业版](https://www.microsoft.com/en-us/microsoft-365/enterprise)
*   [微软 365 教育](https://www.microsoft.com/en-us/education/buy-license/microsoft365)

在我看来，微软 365(Office 365 在其中占很大一部分)有两个紧密相连的主要分支:

*   合作
*   安全性

微软 365 的主要目的是**让企业以安全的方式进行协作**。

如果你想了解更多关于微软 365 的安全特性，点击[这里](https://www.microsoft.com/en-us/trustcenter/security/office365-security)。

下面是您应该迁移到云并开始使用 Microsoft 365 的主要原因的总结。总结主要集中在 Office 365 功能上。

## [](#collaboration-with-sharepoint-and-onedrive)与 SharePoint 和 OneDrive 的协作

除了电子邮件，使用 Office 365，信息将存储在 [SharePoint Online](https://products.office.com/en-us/sharepoint/sharepoint-online-collaboration-software) 或 [OneDrive](https://products.office.com/en-us/onedrive-for-business/online-cloud-storage) 中:

*   **SharePoint Online:**Office 365 协作的核心，也是您组织中共享信息的存放位置
*   **OneDrive:** 基于与 SharePoint 相同的技术，它是一个私人空间，允许组织中的每个员工存储他们的个人文档。每个员工获得 1 TB 到 5 TB 的容量[,具体取决于您的许可模式](https://docs.microsoft.com/en-us/office365/servicedescriptions/onedrive-for-business-service-description)

### Office Web Apps(SharePoint 和 OneDrive)

使用 Office 365，每个 Office 应用程序都有相应的 web 应用程序，允许您直接从浏览器协作处理文档，而不需要相应的 Office 客户端应用程序。以下是 Office 365 中可用的 Office Web 应用程序:

*   Word Online
*   Excel Online
*   PowerPoint Online
*   Visio Online
*   OneNote Online

每个 web 应用程序都允许您直接从浏览器打开和编辑文档。这种体验类似于 Office 客户端应用程序，在编辑体验方面只有一些限制(例如，您可以打开但不能编辑启用宏的 Excel 文件)。

### [](#realtime-collaboration-and-coauthoring-sharepoint-and-onedrive)实时协作和共同创作(SharePoint 和 OneDrive)

您是否厌倦了审阅文档通过电子邮件来回发送同一文档的连续副本，却不知道文档的最新版本是什么？Mee 也是，这就是合作创作的巨大帮助！

当文档存储在 SharePoint 或 OneDrive 中时，通过共同创作，您可以与组织内外的用户共享文档，并让他们实时协作处理文档，而无需保存同一文档的多个版本，也无需跟踪每个参与者所做的更改。所有投稿都合并到同一个文档中，您可以看到还有谁在编辑该文档，以及每个投稿人对该文档做了哪些更改。

[![](../Images/a9c4eb532e9b012aa3c0ebbc5589a3b5.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--M-8D2KpH--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i0.wp.com/blogit.create.pt/wp-content/uploads/2019/07/Word_Co-Authoring.png%3Fw%3D696%26ssl%3D1) 

<figcaption>实时协作(在 SharePoint 或 OneDrive 中共同创作 Word 文档)</figcaption>

这样，您就可以对同一份文档进行完全实时的协作，而不是通过电子邮件来回发送多份副本，从而大大减少邮箱的大小，提高您的工作效率。

SharePoint 和 OneDrive 利用安全共享，允许您选择谁可以访问和编辑文档。

要了解更多关于实时协作和共同创作的信息，请点击[此处](https://support.office.com/en-gb/article/document-collaboration-and-co-authoring-ee1509b4-1f6e-401e-b04a-782d26f564a4)。

### [](#stop-sending-attachments-by-email)停止通过电子邮件发送附件

使用 SharePoint 和 OneDrive 进行协作的另一个优势是，您无需在电子邮件中附加文档副本，只需共享 SharePoint 或 OneDrive 中存储的文档链接即可。

对于用户来说，附件看起来与普通附件完全一样，只是在文档图标附近有一个小的“云”符号。

[![](../Images/ba2e527c5587c81757eeadf57cc6a8b2.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--keDnTBV5--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i2.wp.com/blogit.create.pt/wp-content/uploads/2019/07/Outlook_Attachment_Share_Link_2.png%3Fw%3D696%26ssl%3D1)

要了解更多关于云附件的信息，请点击[此处](https://support.office.com/en-us/article/attach-files-in-outlook-on-the-web-48b8dca1-7a76-43ce-97d1-e1cf73893f55)。

### [](#office-365-groups)办公室 365 组

Office 365 群组是 Office 365 中团队协作的首选方法。创建后，Office 365 组为其成员提供以下内容:

*   安全组( [Azure AD](https://azure.microsoft.com/en-us/services/active-directory/) )
*   共享群邮箱( [Exchange](https://products.office.com/en-us/exchange/) )
*   共享团队网站( [SharePoint](https://products.office.com/en-us/sharepoint/) )
*   共享笔记本( [OneNote](https://products.office.com/en-us/onenote/) )
*   [微软团队](https://products.office.com/en-us/microsoft-teams/)中的团队(如果是通过团队创建的，或者如果后来微软团队中的团队与现有的 Office 365 组相关联)
*   团队规划空间([微软计划员](https://products.office.com/en-us/business/task-management-software))

Office 365 群组可以通过多种方式在 Office 365 中创建(其中包括:SharePoint、Outlook 和 Microsoft 团队)，其主要目的是促进组织内部的团队进行协作。例如，创建的共享组邮箱不是通讯组列表，而是 Exchange Online 中的共享邮箱，它允许组中的新成员在添加到组之前即时访问发送到共享邮箱的所有电子邮件！

要了解更多关于 Office 365 群组的信息，请点击[此处](https://support.office.com/en-gb/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2)。

## [](#mobility)机动性

Office 365 最强的优势之一就是移动性。您将能够从任何设备和任何位置访问您的文档、电子邮件和业务流程。

下面是一些你可以直接从手机或平板电脑上做的事情的例子:

*   使用 Exchange Online 和 Outlook Mobile 应用程序访问您的电子邮件和联系人
*   使用 Office mobile 应用程序在 SharePoint 工作组网站中打开共享文档并进行协作(有一个 Word、Excel、PowerPoint 等的移动版本)
*   使用 SharePoint 通讯网站与您的团队共享和访问重要新闻
*   使用 Power BI 访问您组织的仪表盘
*   使用 PowerApps 访问重要的业务流程(例如:批准您团队的费用)
*   使用 Microsoft Teams 移动应用程序直接从您的手机参加会议
*   使用移动 SharePoint 应用程序访问相关企业信息
