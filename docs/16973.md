# 自动化互联网，一个值得思考的有趣话题

> 原文:[https://dev . to/dechamp/the-automated-internet-a-fun-topic-to-think-2e7i](https://dev.to/dechamp/the-automated-internet-a-fun-topic-to-think-about-2e7i)

如果你停下来想想互联网有多少是由机器人驱动的，从你的人工智能到你简单的自动回复，想想没有一次人类互动会发生什么是很有趣的。

假设我们都注销了一天。没有一个人会与互联网互动...我们可以看到正在发生的一切的实时数据，那会是什么样子？

我没有统计数据，我确信我可以谷歌一下，但是我只是思考和想象，这个世界是什么样子的。

机器人互相交谈，既有垃圾邮件也有有用的信息。通过 api 端点、表单、扫描机器人等将数据从一个数据库转移到另一个数据库。

Bug 被创建和修复，完全没有人工干预。系统被病毒感染和清除，而一些系统对抗这些病毒并自我净化。

Instagrams、twitter、facebook 和其他网站仍然在发帖，就好像没有人离开他们的账户一样。

有多少会停止工作？多少会更好？:p

这一切都让我着迷。

你想过吗？对此有冷静的见解吗？更多关于会发生什么的想法？
