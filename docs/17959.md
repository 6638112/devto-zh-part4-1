# 我讨厌你的代码挑战

> 原文：<https://dev.to/__tsak__/i-hate-your-code-challenge-igk>

我最近申请了一家成熟的初创公司的工程管理职位。最初的面试很棒。我们在规定的时间内走了一个小时，感觉这是一次友好而有意义的谈话。最后，面试官问我，我是愿意管理一个现有团队，还是愿意组建一个新团队。我选择了第二个选项，并期待着这一过程的下一步。

它以开发测试的形式出现在 HackerRank 上。之前玩过 Codewars 之后，我注册了一个账户，然后开始练习他们的测试。与 Codewars 不同，HackerRank 更正式一些，面向有计算机科学背景的人(或者感觉上是这样)。大多数算法挑战和面试准备测试都伴随着额外的运行时复杂性测试。简单地编写冒泡排序来排序数组是行不通的。

总而言之，这是一次有趣的经历，也是打发一点空闲时间的好方法。所以，在花了大约五个小时准备面试后，我决定去做真正的测试(中间好好睡一觉)。

哦，我的天，我是不是在享受一种享受。我有整整两个小时来解决四个挑战:

1.  计算二维数组中的距离 <sup>1</sup> ( [我的解决方案](https://gist.github.com/tsak/42b26d15a81a4f9984c0973c4498d1dd))
2.  [对于第一个数组中的每个元素，计算小于或等于第二个数组中的元素](https://www.geeksforgeeks.org/element-1st-array-count-elements-less-equal-2nd-array/) ( [我的解决方案](https://gist.github.com/tsak/452ab2efffbd3b035a8439b6197ca80f))
3.  [数组中唯一对的数量](https://www.geeksforgeeks.org/number-of-unique-pairs-in-an-array/) ( [我的解决方案](https://gist.github.com/tsak/7d6780c7ef7e3e257e0099dac8dad35a))
4.  [最小唯一数组和](https://stackoverflow.com/questions/38384537/minimum-unique-array-sum) ( [我的解决方案](https://gist.github.com/tsak/6a48fdc07783384204d9165755a112ef))

这是以可视键盘的形式出现的，人们必须根据手指在按键间移动的距离来计算输入安全密码所需的时间

我计划在大约 30 分钟内解决每个任务，考虑到每个任务的复杂性，这是一个疯狂的时间限制。

最后，我设法解决了所有的挑战，但不得不忍受两个挑战中针对运行时复杂性(大输入超时)的失败测试。

事后看来，我可能希望(也需要)每个挑战至少两个小时。

这让我开始思考:究竟为什么有人要对任何人进行这样的测试？这告诉我关于候选人的什么？这些测试与真正的代码有什么关系呢？如果他们真的通过了这些测试，他们会写些什么呢？除非你是唐纳德·克努特(Donald Knuth ),或者在解决全球规模的谷歌问题。

去年，我有幸为两个团队聘请了开发人员。两个队最终都有大约五个人。我怀疑，在我帮助招聘的十个人中，只有一个人能在给定的时间内通过上述测试。尽管他们中的每一个人都完全有能力编写能够投入生产的代码，审查其他人的贡献，指导更多的初级开发人员，解决团队中的复杂挑战，在组合任意数量堆栈的系统中找到 bug，编写有意义的提交消息等等。

所有开发人员的招聘流程都是一样的。在第一次面试后，我会询问他们想要分享的任何代码。一些他们引以为豪或者已经努力了很长时间的东西。我还让人们接受了一项开发测试，要么在家里做，要么在办公室做，但规模要简单得多。考虑一个数组中字符的频率(在[和](https://www.geeksforgeeks.org/print-characters-frequencies-order-occurrence/)这几行上)。虽然我们不是运行预定义的测试，而是对候选人的提交进行代码审查，并根据代码质量、优雅性、测试和提交的解决方案的总体印象来判断。相当多没有正式通过测试的人还是被雇佣了，仅仅是因为很明显他们有正确的方法或者误解了任务，但是不管怎样都表现出了优秀的技能。

不管测试如何，没有什么比得上让某人成为你团队的一员，和你一起工作一段时间来了解他们的情况。不是每个人都面试得很好，不是每个人都在一个任意的时间限制内表现得很好，人们有时会有糟糕的日子。

正如一位黑客新闻评论员所说:

> 有代表性的带回家的工作样本和对话式的问题解决。我的建议是避开那些使用没有灵魂的千篇一律的程序的雇主，这种程序让人们感觉像是一件商品。

我完全同意。
