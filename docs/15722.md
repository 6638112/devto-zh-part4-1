# 每日挑战# 5-10 分钟步行

> 原文：<https://dev.to/thepracticaldev/daily-challenge-5-ten-minute-walk-1188>

嘿，该出发了。

今天的挑战由 CodeWars 上的用户[*@ JK hobic*修改而来:](https://www.codewars.com/kata/take-a-ten-minute-walk)

> 你生活在一个所有道路都呈完美网格状的城市里。你有过早或过晚赴约的不幸习惯，所以你决定创建一个步行生成应用程序。
> 
> 任何时候你按下按钮，它都会给你发送一个单字母字符串数组，代表行走的方向(例如['n '，' s '，' w '，' e'])。你总是只朝一个方向走一个街区。穿过一个街区需要一分钟。
> 
> 创建一个程序，该程序将返回一个表示行走的单字母字符串数组。该程序应该接受用户决定步行的时间量的输入，并应该将用户带回他们的起始位置。

我希望我们镇上的街道也是这样的，这样开车就容易多了。

祝你好运，编码快乐！
