# 开始吧:开始开发网站需要哪些工具？

幸运的是，你真的不需要任何工具来开始 web 开发。你可以打开记事本或任何其他文本编辑器，开始编写代码，这没问题，但对于初学者，我建议使用可以帮助你记忆代码的工具，而不是每 5 秒钟就去查阅一次教程或文档。

为此，我建议使用微软开发的 Visual Studio 代码。它支持代码的自动完成，并为您编写的代码提供建议。

我建议不要使用任何支持所见即所得的工具。可视化编辑器对你不好，对你未来的开发没有帮助，因为它 90%的时候会生成非常糟糕的代码，对你真正学习代码没有帮助。

但是如果我可以使用可视化编辑器，我为什么要学习编码呢？

我说过可视化编辑器生成的代码**真的很糟糕**。随着人工智能和机器学习的发展，这种情况可能会在几年内发生变化，但即便如此，编辑人员也从未真正很好地处理逻辑实现。

此外，编写代码并看到输出正常工作也更有趣。:)

# 还需要其他工具吗？

你实际上只需要一个浏览器。我建议使用 Chrome 或 Firefox。在编辑风格、响应开发、调试 JavaScript 或分析页面方面，Chrome 拥有卓越的开发工具，但它会监视你(因为它是谷歌)。

如果你是一个隐私拥护者，我建议你使用 Firefox。它也有很好的样式开发工具，一个非常好的布局助手和其他不错的工具。只是谷歌 Chrome 背后的调试器和分析工具。

在设置并运行了你喜欢的工具之后，你就可以开始写你的第一个小网站了。
