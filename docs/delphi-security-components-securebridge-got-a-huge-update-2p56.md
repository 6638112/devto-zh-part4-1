# Delphi 安全组件 SecureBridge 得到了一个巨大的更新

> 原文：<https://dev.to/devartsoftware/delphi-security-components-securebridge-got-a-huge-update-2p56>

Devart 最近推出了期待已久的 SecureBridge 更新，这是一款强大的安全解决方案。这些新功能旨在加强对 TCP 流量的保护。

[![SecureBridge Got a Huge Update](img/7989d29313bf0d6933c33520205b5fc1.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--XSki_97n--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://www.devart.com/sbridimg/sbridge_ide.png)

Devart 是为开发人员和数据库管理员提供专业数据库管理软件和 Delphi 数据访问组件的公认供应商，它发布了支持 macOS 64 位、Lazarus 2.0.4 和 SignalR 协议 2.2 的 **[SecureBridge 9.1](https://www.devart.com/sbridge/)** 。其他值得注意的更新包括新的椭圆 Diffie-Hellman 密钥交换算法和改进的 TLS / SSL 协议的证书服务器验证。

新增和改进功能的完整列表包括:

支持 macOS 64 位；
·支持 Lazarus 2 . 0 . 4；
增加了对 SignalR 协议版本 2.2 的支持；
增加了支持 SignalR 客户端的 TScHubConnection 组件；
增加了对 TLS/SSL/SSH 协议 x25519 算法的支持；
增加了对证书撤销列表(CRL)的支持；
·改进了 TLS/SSL 协议中的证书服务器验证；
增加了通过 HTTP/HTTPS 发送数据的分块传输编码；TScHttpWebRequest。添加 BeforeSendData 事件。

**SecureBridge** 提供可用作 SSH、SFTP、SSL、FTPS、HTTP/HTTPS、WebSocket 和 SignalR 协议的客户端和服务器的组件，以保护不可信网络上的数据流。它还与数据访问组件兼容，以防止数据拦截和窃取。要了解当前版本的更多信息，请访问**[https://blog . devart . com/secure bridge-supports-MAC OS-64 bit . html](https://blog.devart.com/securebridge-supports-macos-64bit.html)**。
