# 改变技术壁垒比例的新方法——通过开源打破壁垒

> 原文:[https://dev . to/this dot media/a-new-approach-to-changing-the-ratio-through-open-source-3k 51](https://dev.to/thisdotmedia/a-new-approach-to-changing-the-ratio-in-tech-breaking-barriers-through-open-source-3k51)

Dot 启动了开源学徒计划，希望创造一个更具包容性的技术社区。

这个 Dot，一个著名的女性经营的发展咨询公司，宣布推出他们的开源学徒计划，该计划为公司提供了一条投资改变技术比例的途径。

虽然新兵训练营帮助代表性不足的少数族裔进入了科技行业，平均 43%的女性和 25%的黑人/拉丁美洲人([课程报告](https://theundercoverrecruiter.com/bootcamp-grads-diversity/))，但只有一半的毕业生能够在合理的时间内找到工作([骰子](https://insights.dice.com/2018/03/14/bootcamp-grads-trouble-finding-work/))。

随着多样性比率继续失衡，支持谷歌和微软等公司开源开发的资金继续呈指数级增长。它具有商业意义；谷歌的开源框架 Angular 平均每月超过 150 万用户，脸书的开源图书馆 React 支持每月超过 2000 万次下载。

通过 Dot 的开源学徒计划，公司可以将支持这些技术的用户所需的大量工作引导到等待机会获得第一份技术工作的女性技术人员中。

该计划提供了无数的解决方案，包括多样化的渠道，候选人审查以防止招聘经理的面试偏见，以及为新开发人员的发展创造包容性环境所需的辅导。

“这是公司投资开发资金的一个好方法。告诉我们您的目标，我们将通过我们的学徒与您合作完成工作。我们希望企业不仅将此视为商业解决方案，还将其视为改变多元化比例的一种方式。这个项目最棒的地方在于，你可以在和我们的学徒一起工作后雇佣他们。”Dot 实验室的负责人李美玲说。

这个 Dot 与 Airbnb、Preact、Vue、NgRx、NativeScript、Angular、VMWare、Treebo Hotels、Applitools 等多个开源项目建立了成功的合作伙伴关系并做出了贡献，而且这个列表每周都在继续增长。

NgRx 的维护者和谷歌 Angular 团队的成员 Brandon Roberts 说:“和 URM 一起工作来获得一份提交的 PR 不可能比这更好了。他们反应非常迅速，乐于接受反馈，并在整个过程中快速处理要求的更改。”

“学徒计划不仅仅针对开源项目。事实上，如果贵公司愿意在咨询方面投资，我们很乐意与您合作。”伊娃·豪这样评价这个点。
