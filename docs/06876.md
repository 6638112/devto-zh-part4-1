我一直在我的移动家庭屏幕上安装 dev.to 应用程序，在我的桌面上安装它也非常令人兴奋。

你怎么想呢?

顺便说一下，随着 Chrome 的最新更新(我的版本是 76.0.3809.100(官方版本)(64 位))，如果 PWA 符合 Chrome 的可安装要求，Chrome 将在 omnibox 中显示一个图标(圆圈中的+号)。

如果你看不到它，你可以点击 Chrome 中的三点菜单，然后点击“安装渐进式网络应用程序……”它会提示你将它安装在桌面上，就像添加到主屏幕一样🎉
