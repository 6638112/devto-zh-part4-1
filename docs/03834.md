### [](#c-80-support)C# 8.0 支持

。NET Core 3.0 还将完全支持 C# 8.0，这为该语言带来了许多进步。新特性包括只读数字、更多用于数据和功能分离的模式表达式、使用变量声明、异步流等等。关于 C# 8.0 迄今为止的新特性的完整列表，请参见微软的文档。

### [](#improvements-for-the-iot-crowd)面向物联网人群的改进
