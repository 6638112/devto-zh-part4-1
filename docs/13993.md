# 作为开发人员的前 6 个月(注定是初学者)

> 原文:[https://dev . to/gf guys/first-6 months-as-a-developer-destined-to-初学者-4jig](https://dev.to/gfguys/first-6-months-as-a-developer-destined-to-beginners-4jig)

大家好，我的名字是卢卡斯！我今年 20 岁，是一名移动开发者培训生。

你可能在问自己:“这是什么帖子？”...所以我告诉你！我从事移动开发已经有一段时间了，我想和你分享一下它是怎样的，从开始到现在我经历了什么样的困难，以及如何不断激励自己不断成长。所以喝杯咖啡/茶，让我告诉你几件事。

和许多人一样，我也不知道一开始应该学习哪种编程语言。

我开始像大多数人一样:上大学，学习基础知识(HTML、CSS、SQL、C、Java)。但即便如此，我还不确定我会选择哪一方(手机还是网络)。事实是你没有也可能不会选择站在哪一边。

在学习了编程的基础知识之后，你将能够理解每一种语言的几乎所有内容，不管你的应用程序是用于 IOS、Android、Web、桌面还是脚本。

# [](#tips)**提示**

我不是专家，也不是高级开发人员，但是如果我能给你一些建议，我会说:

*   **保持专注**，无论如何！你还记得那个 MMORPG 吗？你以前不分昼夜地玩，你是你邻居中最好的玩家。这里必须应用相同的焦点规则！；

*   **尝试不同的语言**学习 it 概念(知识多=多才多艺的人)；

*   不要羞于提问。如今，沟通几乎是一切事情的关键；

*   **失败**是不可跳过的重要步骤。相信我，你会在尝试创造新功能、实现你从未听说过的新东西时失败很多次...但是，如果你真的喜欢它，你不会让它拖累你。你可以不断地质疑，寻找，给时间，学习如何编程不会从无到有；

*   每天练习每一个新概念，并为你已经完成的事情感到自豪。

*   最后但同样重要的是，**花时间**去吸收你正在学习的一切...享受一次户外散步，玩一些游戏，和你的朋友一起闲逛，培养一个爱好。

好的，好的...这些都是有用的建议，但是让我告诉你我的经验:

# [](#first-month)**正月**

我感到非常高兴，因为我终于可以和更有经验的开发人员一起学习，同时也大大提高了我的开发技能。我和一个人结对编程，他给我看了完整的代码，教了我一些新东西。

最棒的是，你几乎每天都会学到新东西。

# [](#second-to-third-month)**二至三月**

我学到了很多新的东西和概念，我的编程逻辑变得更好了，并且能够承担一些比我得到的第一个任务要困难得多的任务。

# [](#fifth-month-ahead)**正月初五**

和往常一样，我学到了很多新东西，能够与其他开发人员讨论更复杂的概念，开始了新的个人项目，并且对我在公司内部每天必须做的事情感到更加自信。

# [](#last-thoughts)**最后的想法**

我认为，毕竟，编程和你试图学习的任何其他东西没有太大的不同...例如，如果你想成为一名电子竞技职业选手，你必须花费至少 1 万个小时才能被认为是职业选手。做这样的类比可能会对你有所帮助，就像它在这段短暂的时间里对我有所帮助一样。一旦你下定决心，一切皆有可能。

每天我都有问题(我现在仍然有)，谷歌是我解决问题的主要工具，就像我的团队伙伴一样。

事实上，当有人在你身边帮助你，告诉你要实现的最佳实践，解释架构和流程时，你会学得更快，但是最重要的部分，最困难的工作必须来自你自己。这就是为什么我认为我学得很快，而且我每天都在和大家一起学习的主要原因。

# [](#conclusion)**结论**

就是这样！希望我能在这篇文章中帮助到一些人，请让我知道当你加入开发世界的时候，你的体验如何，我很想知道！

最后但同样重要的是，如果有任何关于英语的错误，我很抱歉。我的英语将永远是一项进步的工作。

如果你还在这里...**不要放弃！**
