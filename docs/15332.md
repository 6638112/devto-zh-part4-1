## 可以用它做什么？

它收集了 GitHub 上存储库的星级数，并显示了按照存储库自上次以来获得的星级数排序的存储库。

您可以找到迅速流行的存储库。

[![](img/c38aa58258958aa19e57ed4271903c1c.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--YeQdEIJm--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/997aczq5p8l8y52dtnsx.png)

## 我为什么要做？

为了收集信息，我阅读工程师写的关于软件的帖子。

他们在帖子中介绍了有用的库。

当我阅读帖子和搜索图书馆时，大多数都在 GitHub 上对公众开放。

直到那时我才知道这个储存库，但是这个储存库已经在 10K 上空获得了恒星。

在我经历了几次同样的事情之后，我开始想我应该去寻找那些流行起来或者迅速流行起来的令人敬畏的软件仓库。

这是我做它的原因。

感谢你阅读这篇文章。
