# 仙丹中的“<>”符叫什么？

> 原文：<https://dev.to/michael/what-is-the-operator-called-in-elixir-30di>

我正在阅读长生不老药指南，最近遇到了各种各样的操作者。其中之一是用于连接两个字符串的`<>`操作符。不过，它没有提到运营商的名字。这是我第一次在任何语言中看到它。

```
"Hello" <> " World"
"Hello World" 
```
