# VWC 最有价值球员，埃迪·普里斯拉克

> 原文:[https://dev.to/vetswhocode/vwc-mvp-eddie-prislac-1gfa](https://dev.to/vetswhocode/vwc-mvp-eddie-prislac-1gfa)

我想做的一件新事情是给在我们单位表现出色的退伍军人更多前瞻性的荣誉，并认可他们的工作和帮助退伍军人构建世界级 JavaScript 开发人员的承诺。其中一位老兵是埃迪·普里斯拉克。

Eddie 是一名海军陆战队员，他在十年前艰难地过渡到了 web 开发，这意味着他在进入这个行业的过程中犯了所有的错误。他目前是 Railroad19 的高级软件工程师，与 NBC 签约，大约两年前开始志愿加入我们的导师团队。虽然他总是对部队有所帮助，但他觉得自己可以做得更多，所以今年，我们让他领导编写代码的兽医的前期工作部分。

作为最初创建流程的人，前期工作是 VWC 最困难的部分。你必须有勇气对你的退伍军人伙伴说最艰难的话，不，以确保我们带入频道的人将尊重投入培训他们的时间
，我们的合作伙伴捐赠给我们的资源，导师捐赠的时间，并确保闲置的频道充满努力工作、渴望的部队。他是看门人，是确保您的捐款和资源得到合理使用的第一道防线，是#VetsWhoCode 的 Heimdall。他是你获得采访机会必须经过的人，代价是完成我们在 Github 上公开发布的[前期工作](https://github.com/Vets-Who-Code/prework)。我们这样做是为了看看你是否具备成为一名开发者的条件。你有足够的纪律性去做这项工作吗？勇于提问？坦然面对失败？但更重要的是，**愿意**去做工作？软件工程是一项很难掌握的技能，在一个充满代码学校的世界里，他们会因为你的联邦保险的退伍军人法案而接受你，这让一些退伍军人感到震惊，一个远程的、免费教授退伍军人的在线非营利组织比大多数营利性项目的入学门槛更高，但我们这样做是为了确保我们的军队能够为这个职业领域做好准备，这是他自豪地坚持的标准。

他的另一部分职责是处理那些不善于处理拒绝的退伍军人。残酷的现实是，在我们生活的世界里，谎言传播的速度是真相传播速度的两倍，哗众取宠。我向他解释了这一点，在最近经历了他的第一次“愤怒的退伍军人”的情况后，他仍然接受了任务并继续工作，他碰巧是另一名海军陆战队队员(想超级快地扰乱一名海军陆战队队员吗？做一名海军陆战队员，但不要表现得像个海军陆战队员)。他巧妙地处理了这种情况，这也是他成为 VWC MVP 的众多原因之一。在 twitter [@semperfried76](https://twitter.com/semperfried76) 上关注他，并感谢您对#VetsWhoCode 的持续服务。
