# 如何在工作中从坐着过渡到站着

> 原文:[https://dev . to/Molly/how-to-transition-从坐着到站着工作-o93](https://dev.to/molly/how-to-transition-from-sitting-to-standing-at-work-o93)

*免责声明:尽管我发现**工作时站立的好处**如果你有任何可能对身体有害的疾病，你应该咨询你的医生。*

几年前，我从工作时整天坐着过渡到整天站着。这不是一夜之间发生的，我慢慢适应了，但我很高兴我做到了。我白天会更清醒，我的身体整体感觉比以前更强壮、更好。经常有人问我是如何从整天坐着变成站着的，所以我决定写一篇关于这个的博客。

# [](#why-i-stand)我为什么站着

我知道你可能希望我一口气说出为什么站立对每个人都有好处的一大堆理由。然而，现实是，我们不知道。关于站立是否对你更健康的研究还没有定论。我将分享我所看到和感受到的益处，而不是比较研究。

对我来说，站着工作让我的身体感觉好多了。我的头和脖子以这样一种方式对齐，我再也不会像以前那样感到背部或颈部疼痛。另外，我的屁股不会因为整天坐在椅子上而酸痛。除了疼痛减少，总体上我感觉更强壮，更有耐力。因为站立比坐着需要更多的肌肉，我发现我的核心力量和背部现在更强壮了。当我过去坐着工作时，如果我在周末经常站着，我会感到筋疲力尽，因为我不习惯。现在，在周末，如果我一整天都站着，就不会那么累了，我可以毫不费力地坚持下去！

另一个我从未真正预料到的好处是，它有助于改善我的消化能力。我过去整天坐着会感到浮肿和便秘。现在我站了一整天，一切似乎都变得更好了，这让我感到很开心。

除了身体感觉更好，我精神上也很享受。我不喜欢长时间坐着不动，而且容易坐立不安。当我站着的时候，我觉得需要少坐立不安，当我坐立不安或四处走动的时候，当我站着的时候会更自然。当我编码时，我也倾向于听很多音乐，我通常会很喜欢。当我坐着的时候，我最多能做的就是点点头。既然我站起来了，我就可以尽情地低调地跳舞了！💃

# [](#standing-supplies)常备物资

如果你决定尝试在一天的部分或全部工作时间里站着，你首先需要一些用品。

1)坐式/立式办公桌——当你刚开始工作时，拥有一张坐式/立式办公桌是你目前最好的选择。我和我的同事都用的桌子是宜家的这些[可调节电动桌子。](https://www.ikea.com/us/en/catalog/products/S29282221/)

[![](../Images/607bb93f675b26ddd2d5536fd10eef91.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--aU3AU0if--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/g4kuyfyfiz9hezc3wf7x.png)

它们有各种不同的尺寸和配置来满足您的需求。如果这些看起来有点太贵，你也可以通过堆放家具或书籍来搭建一个临时站立的地方。
[![](../Images/0fb698690d12a5779170413e5665b410.png)T3】](https://res.cloudinary.com/practicaldev/image/fetch/s--f2sNEqkX--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/bkosa4ui54t6tpl5vl45.png)

2)站立垫——白天你可能需要一个小垫子来站立。就像办公椅有更好的减震效果一样，你的脚有了小垫子也会更舒服。有许多选择，从[基本的平垫子](https://www.amazon.com/Original-Anti-Fatigue-Phthalate-Ergonomically-Engineered/dp/B01B868D1M)到那些有[凸起和凹槽](https://www.amazon.com/Ergohead-Standing-Desk-Mat-Ergonomically/dp/B071RGMZMB)可供摆弄的垫子。这两种我都用过，不能说我真的更喜欢哪一种。

3)好鞋子——我不想打断你，但是如果你想站一整天，你需要一双好鞋子。我确保我上班穿的任何东西，比如凉鞋、平底鞋或靴子，都有足够的支撑力。另一个选择是在你的办公室放一双网球鞋或类似 Crocs 的东西。当你站着的时候，穿上它们。我有一双黑色 Crocs ,当我决定穿低支撑鞋的时候，我会一直穿着。

4)(可选)压缩袜子/袖子——这是我需要的最后一件可以站一整天的衣服。它们只是为我的小腿提供了一点点额外的支撑，防止我的小腿被夹板夹住。以下是我使用的不同类型:

# [](#making-the-switch)制作开关

好了，你有了所有的供应品，现在你实际上如何进行转换呢？首先，你要慢慢开始。这就是为什么一开始有一个可调节的坐/立桌是一个巨大的优势。你永远不需要移动，你可以调节它，一天坐一会儿，一天站一会儿。从每天站一个小时开始。一周左右，增加到两个小时。一旦你适应了，就去三楼，以此类推。关键是要听从自己的身体，不要把自己逼得太紧。

当我第一次出门时，在我买压缩袜之前，我发现任何超过几个小时的事情都会给我带来胫骨夹板。我没有推它，而是减少了站立时间，直到我找到了压缩袜的解决方案。倾听你的身体。开始的时候，你可能会发现你回家的时候有点累，因为你在白天花了更多的时间走路。这是完全正常的，也是适应整天站着的一部分。最终，你的身体和站立的肌肉会变得更强壮，整天站着和你整天坐着没有什么不同。

如果你决定整天站着，有一个小缺点，那就是你永远也不能回到整天坐着的状态。说真的，当我现在旅行时，我可以坐半天，然后我需要变得有创造力来站一天。案例及要点:

总的来说，站立极大地提高了我工作内外的生活质量。如果你想知道你是否能体验到我的一些好处，那么我建议你试一试！
