# 我有我的团队使用的 HTML 模板，我可以在上面构建一个所见即所得吗？如何将内容注入到 HTML 模板的主体中

> 原文：<https://dev.to/alexantra/i-have-html-templates-my-team-uses-can-i-build-a-wysiwyg-over-the-top-how-do-i-inject-content-into-the-body-of-a-html-template-387j>

不完全是 webdev，但我已经构建了一个 HTML/CSS 模板，我的团队将它放入我们创建的报告仪表板中。

因为我认为 HTML 非常简单，所以我创建了一个文档，上面写着“如果你想更改标题，更改第 46 行的 H1”或者如果你想添加额外的列表，做这个或者额外的段落，做那个。

然而，我的用户正在努力让他们的头脑明白他们需要在 HTML 模板中改变什么来定制它以满足他们的需求。

我想知道是否有一种方法可以构建一个 WYSIWYG，让我的用户可以编写他们需要显示的内容，然后将这些内容注入我的模板主体，允许他们将最终代码复制到他们的仪表板中。
