大家好，欢迎来到我称之为**“本周 JavaScript”**的新一期节目。在这篇文章中，我想介绍一个来自我们亲爱的 Node.js + JavaScript 社区的人，向他们提出一些有趣的问题，并为新来者提供一些建议。

希望展示这些有巨大影响力的人开始就像其他人一样，如果他们成功了，你也可以！

本周，我们邀请到了 Natalia Tepluhina(在 Twitter 上也叫 T0 @ N _ Tepluhina T1)，她是 Vue.js 核心团队的一员，是 GitLab 的高级前端开发人员，也是所有了不起的人。

我们来看看她说了些什么，好吗？

## 1。告诉我一些关于你自己的情况(爱好、教育等)

我是 Vue.js 核心团队成员，GitLab 高级前端工程师。

我有计算机科学硕士学位

此外，我还是一名会议发言人，也是与 Vue.js 相关的不同主题文章的作者。由于这些活动，我获得了网络技术领域的谷歌开发专家的头衔。

## 2。你几岁开始编程的？那是什么语言？

我当时 13 岁，这是最基本的。

## 3。你从事 JavaScript 工作多久了？

大约 4 年

## 4。是什么让你开始的？

我想把我的职业从 DBA 变成前端工程师

## 5。如果可以重新定义语言，你会改变什么？

类型强制>。<

## 6。你认为 JavaScript 最好的特性是什么？

灵活性。

*(Fernando 的注释:完全同意，JS 的相同关键特性再次被强调为最佳特性之一)。*

## 7。你会给刚开始学 JS/Node 的人什么建议？

建造一些东西。阅读教程和听课程是很好的，但是如果不重新构建/失败/构建，你就无法学习这门语言。

## 8。你有什么特别想推荐的学习资源吗？

我认为免费代码营是一个好的开始

## 9。有没有你想推广的项目/网站/东西？

Vue.js <3
这绝对是“三巨头”最好的学习框架

## 10。最喜欢的超级英雄？

琴·葛蕾又名凤凰
