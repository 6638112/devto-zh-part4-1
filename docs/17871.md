# 启动商业用途的 Rails 应用程序有什么清单吗？

> 原文：<https://dev.to/galtdea/any-checklist-for-launching-a-rails-app-for-commercial-use-2mfh>

我即将推出一款用于商业用途的 rails 应用。最初，我将使用它来管理我自己的一些事件。它将有大约 100 个用户，但我有点紧张，因为将处理一些财务信息。我在 heroku 上部署了一些应用程序，但没有商业化的。

有资源吗？启动前喜欢待办事项列表。举个例子，我的一个朋友刚刚告诉我，一个应用程序如何处理用户/客户信息是有规定的。接待顾客的最低要求是什么？所需的数据库、服务器等。Heroku 有不同的包，但是很难找到需要的。
