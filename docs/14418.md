[`test-each`](https://github.com/ehmicky/test-each) 是一个 Node.js 库，使用不同的输入重复测试([数据驱动测试](https://en.wikipedia.org/wiki/Data-driven_testing)):

*   独立测试运行程序:与您当前的设置兼容
*   为任何 JavaScript 类型(不仅仅是 JSON)生成描述性的、唯一的测试标题
*   在每个可能的输入组合上循环(笛卡尔乘积)
*   可以使用随机函数(模糊测试)
*   快照测试友好
