如果你想成为世界上最差的程序员，我强烈推荐阅读那些在一些程序员的生活中被证明有效的步骤。

#### 一次性学习多种语言/框架

[![](img/c1a0e564fa9e27b3ade04ee1b3821dd0.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--qXTIGLj6--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://miro.medium.com/max/700/1%2ALXkSRnqxsh4pBmmD9ZeHFw.png)

如果你想成为程序员，同时学习 8 种语言和 5 种框架。

#### 为面包屑创造复杂的工作

[![](img/a345236898c49bf8060f750d0e769c37.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--Fb9mVt4y--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://miro.medium.com/max/700/1%2AvqTPNJGxh5kBDoNQ5TmVbA.png)

作为世界上最差的程序员，最好的部分就是做一个复杂的系统，并以 25 美元的价格出售。

#### 讨论最佳语言/框架

[![](img/6c4f652bc7c051b6d9828c903b883f4d.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--jjE6TPJw--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://miro.medium.com/max/700/1%2Ah72bSS8UZ4ona8scW6Cxgw.png)

与同学讨论指出 React 比 angular 更好会给你带来很大的感觉，即使 React 能解决所有问题。

#### 不使用版本控制

[![](img/ace13ca5b7ff729bb4a044c399b093ba.png)](https://res.cloudinary.com/practicaldev/image/fetch/s---fr3TFTZ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://miro.medium.com/max/700/1%2AzWhHZUDWuxJwfEoumC1-OQ.png)

你可以使用 Google Drive 来修改你的代码。最好的做法！

#### 你在一个团队吗？单独工作！

[![](img/7149acbb2fc3799571db6f6d3ec8c394.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--pWCwYC61--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://miro.medium.com/max/700/1%2AL30BHpyUmdluxpzanwujiw.png)

独自工作，你不必依赖另一个人。要知道，“想做成事情，自己动手”。

#### 不主动出击

[![](img/b52a3bb70fd44a870c90a363067f7210.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--1TtxIJs5--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://miro.medium.com/max/700/1%2AoPqw8eUjVCdKFw7HhlprHg.png)

主动做什么？什么都不会发生，一切都会好的！

#### 不尊重资深开发者

高级开发人员不知道你在说什么，你在 facebook 上看到的那个帖子是一个知道如何在 react 中创建一个组件的 17 岁男孩写的，他知道自己在说什么。

#### 没有纪律

[![](img/5143f7cb24d33b62e65b6e8814a83d52.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--oATb4k1R--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://miro.medium.com/max/700/1%2Ajs4IF-Nq6rA66wnMFKLELg.png)

我们成为程序员是为了随时做你想做的事情，这是我们的世界，习惯吧！

#### 忘记你的精神和身体健康

[![](img/0e6a12a4532a266b509532e8a716f5a4.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--gEpa_IVM--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://miro.medium.com/max/700/1%2API0jVieukS6KIuN4m8xqWA.png)

你只需要呆在电脑前，不用担心你的健康。离开电脑几个小时去锻炼可能会让你花钱。时间就是金钱，写下来！

#### 远离你爱的人

[![](img/23ea256ff7577f2a58ef13a8561c2c1f.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--kwYSxk_9--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://miro.medium.com/max/700/1%2AxvMGwXmXiGTbNrPexBzJJg.png)

学习，做个人项目，做明天要做的项目的交付。没有人的时间，但没有问题，因为他们永远在你身边。
