**但究竟什么是颤音呢？**
颤振是谷歌开发的跨平台应用程序开发和 web 应用程序的 SDK，该工具具有以下原理:

性能:开发的应用程序具有本机应用程序性能；
丰富的本机 UI，具有每个平台的自然用户体验；
通过使用小部件和热重装来快速开发(即在编辑器中保存代码并可以测试更改时更新应用程序的启动)。

float 的第一版于 2017 年 5 月发布，目前(2019 年)为 1.7 版，具有 Web 开发功能。

### *浮子分为两个主要层:*

引擎，以 C/C++编写，负责执行结构。它由二维渲染和文本库“”skia 和开发中使用的语言 Dart 组成；

Framework，使用[dart](https://dart.dev)语言，是我们创建应用程序小部件实现的地方。
负责视觉元件，包括 Android 和 iOS 专用的元件，以及回应配置的彩现。

### *但 Dart 到底为什么？*

Dart 是 Google 在 2011 年推出的一种编程语言，旨在取代 JavaScript 作为浏览器中的主要内置语言。

它的一些特点包括以对象为导向，声明性的，大部分由 Google 维护，并且被归类。此外，选择这种语言浮动最重要和最关键的是，它可以及时编译。

*   Dart 是以快速、可预知的本机代码编写的，几乎所有浮点都可以写入 Dart。这不仅使颤振快速，几乎所有的东西(包括所有的小部件)都可以定制。

*   Dart 还可以使用 JIT(即时装载)进行编译，以实现超快的开发周期和改变状态的工作流(包括流行的浮点状态热重加载子秒)。

如果您熟悉 TypeScript，甚至熟悉 Java 和/或. net，则可以很容易地处理 Dart，因为它是一种类似于上面所说的语言，而且我们还使用 Classes、Async/await。

在开发过程中，对数据进行分类可以帮到你很大的忙，因为你真的知道你要创建的每种方法的输入和输出数据。
