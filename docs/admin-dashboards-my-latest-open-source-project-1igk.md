# Admin-Dashboards.com——走向开源

> 原文：<https://dev.to/sm0ke/admin-dashboards-my-latest-open-source-project-1igk>

各位编码员好，

几天前，我有点惊讶地看到[Admin-Dashboards.com](https://admin-dashboards.com/)域名是免费的，于是我想出了让它成为[开源](https://en.wikipedia.org/wiki/Open_source)服务的主意。

## 动机

如果你是一名自由职业者或一家小公司，并且你的投资组合中有一个**管理仪表板**产品，在没有足够预算的情况下，试图推广和销售它会很有挑战性。在谷歌搜索关键词“**管理仪表板**”的第一页，到处都是定位良好的公司，如 ColorLib、WrapBootstrap、Creative-Tim 以及其他同级别的公司。

## 工作原理

*   分叉 [admin-dashboard 项目](https://github.com/admin-dashboards/dashboards) -发布在 Github 上
*   为你的管理面板添加一个描述性的帖子，比如这个[示例](https://github.com/admin-dashboards/dashboards/blob/master/content/posts/2019-21-06---argon-dashboard.md)
*   审核后，你的应用将在[Admin-Dashboards.com](https://admin-dashboards.com/)上可见
*   查看最多的仪表板将在下获得一个回购，以直接更新仪表板信息。

`Note` -该应用内置于[盖茨比](https://www.gatsbyjs.org/)
