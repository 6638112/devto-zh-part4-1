# 使用

1.  克隆存储库

    ```
     git clone https://github.com/PJijin/Personal-Website.git 
    ```

2.  运行此命令安装依赖项

    ```
     npm install 
    ```

3.  打开 src/config.js，然后输入您的社交媒体帐户用户名。

4.  从 src/components/About/about.js 自定义“关于我们”页面

5.  奔跑

    ```
    npm run start 
    ```
