*   React Native 基于最受欢迎的技术栈 JavaScript，所以教程和社区支持对 JavaScript 来说是惊人的，而且市场上已经有很多 JavaScript 开发人员，那么为什么你应该花时间学习一种非常未知的语言 Dart 来开发 flutter 上的应用程序呢？？

### [](#1-just-like-others-flutter-is-not-just-for-android-and-ios)**1。就像其他人一样，Flutter 不只是针对 Android 和 iOS**

* * *

*   在 1.0 版本之后，flutter 发展得非常快。在 1.0 版本中，它只是像其他人一样用于针对 Android 和 iOS。

*   但是随着 Flutter 1.5 的发布，flutter team 让所有人惊叹不已。他们正式宣布，从现在开始，flutter 不仅适用于 Android 和 iOS，还将适用于 Web、桌面和嵌入式设备。阿雅:他们现在正在试映，这意味着他们很快就会推出它。

*   因此，如果你学习 flutter，你将成为一个人的军队，因为从一个框架，你将能够针对每一个主要平台。你只需要掌握一个框架。想象一下颤动给你的力量。🔥

*   如果你想为 ***Web*** 开发，flutter 可以帮你。

*   如果你想开发本地 Android 和 iOS 应用程序 ，flutter 可以帮助你。

*   如果你想开发适用于 ***Windows、macOS 或 Linux*** 的应用程序，flutter 正适合你。

*   最后，如果你想为智能手表和智能家居产品等*嵌入式设备开发应用程序，这里也有 flutter 可供你使用。*

 *### [](#2-fuchsia-os-the-most-powerful-beast-in-the-game)**2。Fuchsia OS:游戏中最强大的野兽。**

*   学习 flutter 的第二个也是最令人惊讶的原因是，在未来，Fuchsia 中的整个应用程序开发将在 flutter 中完成。

### [](#5-features-of-flutter)**5。颤振的特征**

* * *

*   最后，让我们讨论一下颤振的特征。颤振有四个主要特征。
*   有状态热重装
*   漂亮的应用程序
*   多产的
*   高性能

让我们详细讨论每个话题。

##### [](#stateful-hot-reload)有状态热重装

*   Flutter 为您提供有状态热重装。如果你有原生应用程序开发的经验，那么你必须知道编译整个应用程序需要多少时间。

*   但是 flutter 为您提供了有状态热重载的便利。因此，只需几秒钟，您就可以查看您的更改。就像在 web 开发中你刷新网页来查看你的更改一样，同样的概念也适用于此。

*   为了提供有状态的热重载，flutter 在调试模式下使用 Dart 虚拟机构建来快速注入新代码。

##### [](#beautiful-apps)漂亮的应用

*   你可以从你选择的设计语言中得到任何你想要的外观。Flutter 支持材质设计和基于 Cupertino 的小部件。所以用 flutter 构建的应用程序自然是美丽的，你不需要太多的努力来使它们变得美丽。

##### [](#productive)多产

*   Flutter 有丰富的内置功能，可以让你的开发更快更好。

*   它有一组丰富的小部件，可以让您的生活更轻松，开发更快速。只要尝试一次，你就会爱上 flutter 在盒子里给你的疯狂力量。

##### [](#high-performance)高性能

*   对于渲染窗口小部件，Flutter 使用自己的高性能 2D 渲染引擎 Skia。在 Skia 的帮助下，flutter 能够提供超过 60fps 的性能。所以你的动画看起来更流畅，你的用户界面自然也更快。
