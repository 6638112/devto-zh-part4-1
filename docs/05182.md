科特林的历史

首次出现于 2011 年，当前稳定版本为 1.3.41。

2012 年 2 月，开源。

Kotlin v1.0 于 2016 年 2 月 15 日发布。

在 Google I/O 2017 上，Google 宣布了对 Kotlin onAndroid 的一流
支持。

Kotlin v1.2 于 2017 年 11 月 28 日发布。在 JVM 和 Javascript 之间共享代码。

2019 年 5 月 7 日，谷歌宣布 Kotlin 成为 Android 应用开发者的第一个
首选语言。

科特林是什么？

“Kotlin 是一种针对 JVM、Android

甚至 JavaScript。"

1.  现代静态类型

2.  与 java 100%互操作

3.  适用于 java 的任何地方

4.  目标是本机、Javascript、JVM 和 Android

5.  面向对象，也是一种函数式语言

6.  开放源码

7.  由 Jetbrains 开发(是的，是那些构建我们可爱的 Android 工作室的极客)。
