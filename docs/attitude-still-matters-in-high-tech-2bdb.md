# 在高科技领域，态度仍然很重要！

> 原文：<https://dev.to/mabumatar/attitude-still-matters-in-high-tech-2bdb>

在软件革命的早期，程序员就是国王！如果你知道如何制作复杂的 2 或 3 代语言代码，你基本上，嗯，统治！企业不得不屈从于你的要求，改变策略，模仿你的杰作。

展望移动的、高度分布式的、永远在线的、基于云的软件时代，突然之间你优雅的不可触及的代码不再酷了！框架、代码生成器、模型驱动的工程和特定领域的语言已经真正地将编程技术转变成了一种纯粹的机械活动。诚然，我们仍然需要那些优化的算法和搜索技术，但是这些项目需要多少程序员呢？我认为不多。

今天，软件定义一切世界要求软件工程师构建系统，而不仅仅是编程。工程本质上是一种实用的跨学科工艺，需要协作、论证、标准化和妥协。今天的软件系统是由具有不同技能和文化的不同团队构建的。团队由开发人员、测试人员、发布经理、项目经理、产品经理、架构师等组成。团队成员必须协作，互相学习，争论，妥协，并最终交付解决方案。

所以，如果你仍然被封闭在你自己的立方体里，从你的代码行镜头看世界，你最好检查一下你的视力。因为你猜怎么着？在高科技领域，态度仍然很重要！
