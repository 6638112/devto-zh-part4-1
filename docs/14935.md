# [](#introduction)简介

如果你不能让自己买一本书，订阅一份时事通讯，在你的“以后看”中添加一吨 youtube 视频，或者甚至在你生命中的每一秒都想知道你正在做的事情是否更糟，这篇短文是给你的。

所以你在这个介绍中认出了你自己...好吧。

-好消息:你似乎真的有学习和成长的动力。

-坏消息:这不是通过压倒你的头脑与所有的东西，你会达到你的目标。更糟糕的是，你可能压力太大，太沮丧，以至于你会毁掉自己，被迫放弃。当然，编程是一条长期的道路。所以这个想法就是要热爱经历它，否则你迟早会放弃。

反正这是我的信念。在这种情况下，我经常问自己:“直到生命的尽头，你还能感受到你现在的感受吗？”。或者更粗暴:“你想反复感受这些觉受直到你死去吗？他们会让你快乐吗？”如果答案是:“上帝啊，如果我每天都有这种感觉……”，你应该重新考虑你的工作流程。

就我个人而言，如果后面的目标对我来说足够有价值，我可以承受一段时间的巨大压力。顺便说一下，这种情况很少发生，因为选择，而是需要。我倾向于避免它，因为它很痛苦(当然),也因为我最重要的目标很少是短期目标。

我认为这是我们大多数人的情况。我们可能想在夏天拥有完美的身材，但我们更愿意在生活中保持这种完美的身材。事实上，这个目标看起来雄心勃勃，但这通常是我们真正想要，忽视它是一个坏主意。

所以问问你自己什么对你真正重要，如果你最终有一个长期目标，考虑在整个过程中管理压力。

# [](#conclusion)结论

跑题够多了，

我想提醒你的简单想法是:

1)你不可能学到外面所有的东西。因为你不能，所以你必须选择对你最重要的事情。

是啊！！！什么对你最重要！这就是为什么我们需要“保持简单”，因为我们做的每一件事，每一种无用的思考都是我们和目标之间的另一个障碍，甚至是我们和让我们快乐的事物之间的另一个障碍。我们只是失去了时间和精力(即我们为了学习、创造或完成其他任务而消耗的宝贵资源)去考虑这些我们目标之上的多余层面。

对自己诚实。意识到你在做什么。你的工作会更有效率，更快乐，所有这些都会持续很长一段时间。
