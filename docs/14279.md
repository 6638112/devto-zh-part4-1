# Java 编程语言的 3 个组成部分

> 原文:[https://dev . to/yedukondalu 319/3-Java 编程语言的组件-59di](https://dev.to/yedukondalu319/3-components-of-java-programming-language-59di)

计算机编程是一个持续的、变化最快的领域。编程的目的之一是用一种典型的方式找到一个问题的解决方案。程序员总是需要保持相关性并应对新的挑战，除了拥有非常好的编程知识之外，非常重要的是，他们还需要了解他们编写代码的环境的内部结构。

在这个博客中，我们将讨论 Java 编程语言的三个重要组成部分；任何 Java 开发人员都必须了解这 3 个组件。我们还建议您阅读我们的博客“学习 Java 编程的 7 个基础知识”,从今天开始编写代码，其中讨论了 Java 编程的基本概念。

Java 程序员用人类可读的语言编写程序，称为源代码。CPU 或中央处理单元或芯片不理解用任何编程语言编写的源代码。这些芯片或计算机只理解一件事:机器代码或机器语言。这些机器码运行在 CPU 层，因此，不同型号的 CPU 会有不同的机器码。

我们不需要担心机器码，因为编程对我们来说就是源代码。为了帮助机器理解源代码，需要将源代码转换成机器语言，即可执行代码。

所有这些功能都发生在 Java 编程语言的以下 3 个组件中:

Java 虚拟机。(JVM)
Java 运行时环境。(JRE)
Java 开发工具包。(JDK)
让我们在博客的第一部分讨论这 3 个组件，然后在博客的第二部分，我们也将浏览 Java 的各种应用。

JVM 是什么？

Java 虚拟机(JVM)执行编译器生成的字节码。
