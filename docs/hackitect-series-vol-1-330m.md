## 以黑客之旅为使命

我决定写这篇文章，作为 Hackitect world 即将发布的系列文章的前传。这个开始会有点软。

我决定写这篇文章，作为即将到来的黑客世界系列文章的前传。首先，我将开始一点非传统的。这一章的主要目的是激励有才能的年轻人投身于信息技术和网络安全。你可以去很多方向。您可能是风险经理、SOC 专家、审计员、顾问、渗透测试员，甚至是安全架构师。并且不去限制同时担任更多的角色，比如伦理黑客和架构师。你认为这不可能吗？

## 作为概念的黑客

我从来不喜欢他们对我拳脚相加，或者试图把我放在我的公司名单上。也不要让它发生。每个人都必须找到正确的方式来使用和定义自己的天赋。工作开始和结束的时间，当同事角色早已不在的时候。Hackitect 实际上是一个介于道德黑客、SSDLC(安全应用程序开发)和架构师之间的角色。当然，它可以是多种多样的，但重要的是，这个角色不仅限于我是 x 方面的专家。Hackitect 的道路主要意味着发现和不断学习新事物的愿望，而不是限制技术或工具。与此同时，Hackitect 必须保留这些知识，以理解技术是如何工作的，并且能够深入到非常详细的细节，然后再回到高层次的观点。漫长而艰难的旅程。有时候，这似乎是不可实现的。一般来说，对于建筑师来说，如果不动手，他们会完全脱离现实。在一个不切实际的设计中，以充满盒子的图片形式交付解决方案的建筑师不是建筑师，而是伪艺术家。再说一遍，动手就是存档。

## 从哪里开始

先看一下信息安全管理体系和 demming 循环。理想情况下，你需要开始 Udemy 课程(他们有很好的关于 pentesting 的课程)并探索 OWASP (SAMM，ASVS，编码指南)之类的标准。好的教程也可以是 KALI Linux 中的工具学习。如何开始渗透测试的一个很好的来源是安装并启动 WebGoat。如果你只想进行渗透测试，最好阅读 OSSTM(手册)和 OWASP 测试指南。对于架构理解，你需要知道什么是架构，然后学习像 TOGAF 或者 SABSA([https://sabsa.org/](https://sabsa.org/))这样的框架。如果你想变得非常现代，在敏捷的环境下工作，那就去 devsecops 工作室(【http://www.devsecops.org/】T2)吧。如果你对云安全感兴趣，你不能错过 AWS 良好实现的框架和云安全联盟白皮书。还有许多其他的想法需要研究——法医学、身份、密码学、网络、移动应用和无限的其他。您需要理解，应用程序和基础设施是一个硬币的两面。别忘了，当你学习的时候。探索支持安全性的所有方面(服务管理、开发、配置管理、CI/CD 和许多其他有趣的神奇宝贝)。如果你够勇敢，就去查查零信任架构。

*好吧，这很混乱，所以让我们简单点:*

*   疯狂编码，并尝试在 Github 上创建你的 on repo(强者可以开源他们的代码动物)
*   没有研究，这是行不通的。
*   你疯狂地阅读，然后再阅读和学习。
*   测试工具并评估它们。
*   计划并让事情变得非常简单。
*   当你不明白的时候不要哭。当你没有发现什么的时候哭吧。
*   高度的道德必须是第一位的。
*   了解系统如何攻击攻击者，以及系统如何构建(分析和综合)。
*   如果是培训生，找个导师。
*   如果你是比较有经验的专业人士，找个教练。
*   谦虚一点，选择从事网络安全工作并不是一条轻松的路。
*   确定您的黑客领域，并遵循小步骤。
*   更高的目标，如获得 CISSP，CEH，GWAPT，CSSP，CSLP，CRISC，SABSA，TOGAF，OSCP，CCSP 和其他人是优秀的。用耐心和决心武装自己是很重要的。因为 IT 安全不是一份工作，而是你使命的一部分。

## 明白自己永远不会知道一切

在你的黑客之旅中，你发现的越多，你就越发现你不能理解所有的事情。你将永远无法把握 SIEM、安全开发、云、MDM、微软& Linux 生态系统、开源解决方案、道德黑客等等。有了这个，你需要高兴并学会生活。我在某处读到过一句很好的引语。为了成为大师，你必须先成为初学者。是一千次。不要害怕失败，因为你只会学习。就我个人而言，我可以说每 3 年，我都觉得自己忘记了一切，重新开始。这是对技术快速发展的征税。

## 潜进潜出

学习一个高层次的抽象概念并向它之外的人解释是一件困难的事情。学习如何作为一个商人工作，并学习如何把它推销给你公司的业务部门。高水平的设计能力需要多年的实践。重要的是也忘记了工程本身，偶尔弄脏自己的手去工作，以不错过平日与现实的接触。不要让高层次观点的刻板印象掩盖了你作为一个儿童工程师的想法。工程和研究是建筑工作的一部分。

## 提防骗子

每个人都会答应你可能和不可能。这是真的，没有人能给你比我能给你的更多。不要相信承诺。当愿景至少有点可及时，它们是伟大的。如果有很多幻觉，穿上你的背包，飞到另一个美丽的地方。没有自己的创新和行动，自己的工作也不会更好。如果你有导师，并不意味着他就是你的老师。他指引你，给你指出正确的方向。唯一能让你前进的人只有你自己。没有人会给你基于无人机计算的区块链的人工智能。实事求是地用自己的作品去建设，把未来建设成自己喜欢的样子。许多经理甚至更多的专家说的比做的多。学会识别适合你工作生活的伴侣。

## 没有人是行不通的，想想规模化学习吧

IT 安全很难找到好的大团队。我们在非常有趣的项目上工作，这让我思考。在技术上，我们已经解决了尖端技术。我靠得太近，无法触摸到真正的未来产品，但是我够不到它们。我意识到，我在团队中是孤独的，没有人是我的领域。这是主要的一点。非常需要能把想法变成现实的人才！没有团队成员，就不可能完成 hackitect 之旅。相互学习是成功的关键因素。一个人，真糟糕。

## 慢慢来

还不快点！！计划有效你的黄金宝贵时间。思考学习，享受学习。如果你太匆忙，你会与现实失去联系，开始迷失。有效的时间规划以及对你的黑客之路的适当好处是关键因素。如果你有时间，你就有时间留给自己和批判性思维。如果你得到信息，评估信息是否真实和相关。在这些日子里非常重要。尽量简化。不要尝试，让事情变得简单。正如我的一个朋友所说:“我是个简单的人，我喜欢简单的解决方案！”。

我计划让这个话题更加个人化。然而，我决定为下一次留下一些子弹。
