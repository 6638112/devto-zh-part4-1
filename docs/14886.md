# 投票:你对(会议)谈话不给予反馈的原因

> 原文：<https://dev.to/arnoutboks/poll-your-reasons-for-not-giving-feedback-on-conference-talks-1ee8>

大多数会议和一些聚会都提供了一种方式来提供对每个单独谈话的反馈，要么通过反馈平台，如 [joind.in](https://joind.in/) ，要么通过定制的反馈表格。这种反馈对于演讲者(改进他们的演讲)和会议组织者(帮助选择演讲和评估)都是至关重要的。

我注意到，一般来说，只有大约 5%的与会者对会议讨论留下了反馈。我想找出为什么大多数人不这样做(也许还有如何解决这个问题)。请通过**在下面的推文**中回答调查和/或留下您改进反馈流程的想法的回复来帮忙。
