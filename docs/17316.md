# 应用程序 GUI 组件的一致性

> 原文:[https://dev . to/the fern/consistency-in-applications-GUI-components-49be](https://dev.to/thefern/consistency-in-applications-gui-components-49be)

我是那些不喜欢更新应用程序的生物之一，除非它们停止工作。没有设置自动更新。不幸的是，android 上的一些应用程序有强制更新，如果这是一个安全更新，我并不反对，但在 imo 中，每个版本都不应该强制进行小更新。我不喜欢自动更新的原因是，它们会狠狠地打你的脸，更新伴随着变化或更多的广告，因为缺乏更好的词，不得不重新学习整个 gui。变化范围从最小到最大，再到在几乎无法使用后完全找到另一个应用程序。

以下是我列出的已经改变的、我经常使用的或者曾经经常使用的应用程序。

## [](#google-keep)Google Keep

如果你没有听说过 google keep，那你就错过了。我每天都用它来写下想法，无论是什么，组织，购物清单，提醒，我至少有十几个标签，实际上有一个叫做 dev.to，我在那里保存文章的想法。我可以做清单，也可以拍照，用我的笔记笔在上面写字。在某一点上，他们更新了应用程序，并改变了我身上的颜色，是的，我知道旧的 apk 在互联网上的某个地方，但它也有其他很酷的更新。在评论上，人们很生气。另一个大的变化是引入了浮动操作按钮，或者简称为 fab 来添加新注释。肌肉记忆是一个婊子，原谅我的法语，但当你使用一个应用程序多年来改变一个主要 UI 组件的位置是一个坏主意。在后来的更新中，他们删除了 fab，但仍然保留了可怕的颜色。

Keep 没有提供选项来打开更新后的应用程序的传统颜色，甚至选择自己的颜色，这应该不会太难实现。最后，我的眼睛适应了新的柔和的颜色，这个应用程序在我的日常使用中根深蒂固，所以我必须适应。

## [](#tumblr)Tumblr

Tumblr 是我最喜欢的应用程序之一，直到一次更新基本上扼杀了我的应用程序。我在我的 feed 上看到了一堆我没有关注的东西，持续滚动的广告有点太多了。

amirite，肯定有比滚动广告更好的赚钱方式吧？比如以固定价格销售应用程序？

## [](#pinterest)Pinterest

我甚至不打算告诉你我花了多少时间收集别针、木工、微控制器、编程、食谱，这个清单还在继续，看看这里，它以我从未想过的方式激发了我的创造力。然后一个更新发生了，**为什么神？**我注意到的第一件事是一个白色的圆圈(变形按钮),我不知道它是干什么用的，所以我按了一下，画面就变了。究竟是什么？我看的帖子在哪里？

我注意到的第二件事是更多的广告，你看到这个可怕的趋势了吗？我并不反对广告，但请保持优雅，不要让我看到每 5 针就有一针。如果你想要一个优雅广告的好例子，看看[火星游戏](https://play.google.com/store/apps/details?id=com.pomelogames.MarsGame&hl=en_US)。广告在适当的时间间隔播放，而且是可选的。

我注意到的第三件事是，如果我进行搜索，我看到的是与搜索无关的广告。不用说，这些天我很少使用 pinterest 应用程序，除了看看我的旧大头针。

## [](#spotify)Spotify

我成为 spotify 付费用户已经很久了，事实上，spotify 和 dropbox 是我唯一持有的订阅。我不是订阅的忠实粉丝，但是嘿，有时候你不能在这个世界上生存。CD 或必须在 iTunes 中管理 mp3 的日子已经一去不复返了。

这个改变实际上是一个好的改变，将底部的主工具栏从 5 个项目变成了 3 个。我不赞成的其他小变化是隐藏的重复按钮，它过去可以在主屏幕上访问，现在你必须去顶部的三个按钮，然后弹出一个新菜单。这个变化让我觉得 spotify 不鼓励歌曲重复，对我来说是一个奇怪的用户界面变化。
