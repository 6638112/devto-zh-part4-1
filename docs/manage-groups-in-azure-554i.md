# 管理 Azure 中的群组

> 原文：<https://dev.to/cheahengsoon/manage-groups-in-azure-554i>

**管理群组成员资格**

尝试在您的 Azure AD 租户中管理用户的组成员身份。有关如何操作的说明，请参考位于[https://docs . Microsoft . com/en-us/Azure/Active-Directory/Active-Directory-groups-members-Azure-portal](https://docs.microsoft.com/en-us/azure/active-directory/active-directory-groups-members-azure-portal)的“使用 Azure Active Directory 添加或删除组成员”。

**创建群组并添加成员**

尝试在 Azure AD 中创建一个组并添加成员。有关如何操作的说明，请参考“使用 Azure Active Directory 创建基本组并添加成员”，网址为[https://docs . Microsoft . com/en-us/Azure/Active-Directory/Active-Directory-groups-Create-Azure-portal](https://docs.microsoft.com/en-us/azure/active-directory/active-directory-groups-create-azure-portal)。使用组来执行管理任务，例如同时向几个用户或设备分配许可证或权限。

**管理个人资料信息**

尝试在 Azure AD 中添加或更改用户的个人资料信息。有关如何操作的说明，请参考位于[https://docs . Microsoft . com/en-us/Azure/Active-Directory/Active-Directory-users-profile-Azure-portal](https://docs.microsoft.com/en-us/azure/active-directory/active-directory-users-profile-azure-portal)的“使用 Azure Active Directory 添加或更新用户的个人资料信息”。

✔️如果有时间，请尝试其他用户和组管理任务。如果你想通过使用 Azure PowerShell 来尝试其中的一些任务，请参考位于[https://docs . Microsoft . com/en-us/PowerShell/module/AzureAD/？view = azure ads-2.0](https://docs.microsoft.com/en-us/powershell/module/Azuread/?view=azureadps-2.0)。
