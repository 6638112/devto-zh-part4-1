### [](#more-about-me)更多关于我

我上大学是为了音乐教育和表演，我的主要乐器是中提琴。我已经演奏了不到 20 年，音乐是一件令人惊奇的事情。我从未完成我的学位，从一份工作跳到另一份工作，从未热爱过我所做的事情。

这就是编码的用武之地。我一直对它很感兴趣，去年在决定从事前端 Web 开发之前，我曾考虑过几种不同的职业，以下是我的想法:

*   **前端 Web 开发**
*   **网络安全**
*   **AWS 解决方案架构师**
*   **...前端 Web 开发**

我想，我希望从事前端工作的部分原因是，我有能力去创造和想象我所创造的东西。

### [](#what-im-working-on)我在做什么

我目前正在我的作品集网站上工作——它一直在扩展，我不确定我是否真的觉得它已经“完成”了。除了我自己的网站，我还在为当地企业做一些网站，包括一家啤酒厂、一家环保公司和一家食品公司。

我也在学习普通的 JavaScript，最终会转向 React，但是，在添加任何库之前，我想对 JS 有一个扎实而透彻的理解。我正在通过[树屋](https://teamtreehouse.com/)、[自由代码营](https://www.freecodecamp.org/)以及我自己对所学内容的实践来学习。

我经常将学习编码语言等同于学习音乐，并结合类似的实践技巧——这是我非常感激学到的东西。
