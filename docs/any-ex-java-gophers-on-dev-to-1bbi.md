# dev.to 上有没有前 Java 地鼠？

> 原文：<https://dev.to/lilianaziolek/any-ex-java-gophers-on-dev-to-1bbi>

如果你是一个曾经喜欢 Java 的有经验的 Java 开发者(那个很重要！)并过渡到 Go——我很希望你能分享一下你的想法/经历。

*   你换工作的理由是什么？
*   事情就是不“合”的时候，最痛苦的时刻是什么？
*   有哪些“啊哈！”突然事情变得简单了。
*   有哪些东西是从 Java 翻译过来的，但可能不是以一种显而易见的方式？
*   有哪些“香蕉皮”/陷阱等着一个前 Java 开发人员掉进去了？
*   哪些 Java 习惯很难学，但却很重要？
*   有没有一些领域/用例你仍然会选择 Java 而不是 Go？
*   有没有一些领域/用例是你永远无法回到 Java 而不是 go 的？
*   在旅程开始时，你会对自己说些什么？
