# 自托管平台与托管平台

> 原文：<https://dev.to/lauraharvsey/self-hosted-vs-managed-platforms-401b>

互联网每天接待超过 10 亿的用户。无论是将这项先进技术用于工作、学校还是纯粹的娱乐，所有这些信息都可以在网站上找到。网站是实体商店的虚拟复制品，这些商店过去每天都吸引游客。除了访问这些网站之外，许多人都有理由创建自己的网站。无论你是想建立一个网站来促进赚钱还是写一个个人博客，在开始之前都有一些重要的步骤需要考虑。

如果你打算经营一个网站，你可能了解从头开始建立一个网站所需要的复杂的编码技巧。虽然几行基本的代码就可以构建一个网站的结构，但是需要更多的时间来学习用户已经习惯的高级功能。幸运的是，有许多选项可以帮助您更快地发布内容。有了这些选项，您可以跳过从头开始编写 web 平台代码的复杂任务。在选择之前，理解两个基本选项之间的区别很重要:自托管和托管解决方案。

以下是在决策过程中应该考虑的两者之间的主要区别。

### 我应该使用自托管还是托管 Web 平台？

在你建立一个网站之前，你必须首先选择一个自托管或托管的网络平台。在做出这个决定之前，你不能在网上发布任何品牌内容。简而言之，如果您选择托管选项，您将会牺牲可定制性来换取简单性和易用性。

托管平台为人们开始在线发布内容提供了一种简单快捷的方式，而不必麻烦服务器或管理升级。托管平台还提供了性能和可靠性的提升，因为一切都是集中管理的。[澳大利亚托管审查网站 Aussie Hosting 的研究显示](https://bestwebhostingaustralia.org)托管平台的运行效率提高了 30%以上，安全性更高，停机时间更少。

另一方面，自托管平台提供了卓越的定制功能，可以操纵风格和功能来满足您的个人需求。虽然托管平台对可以改变的内容有限制，但自托管平台基本上有无限的可能性。起初，自托管平台由于其整体的个性化似乎是显而易见的答案。然而，这些选项需要更多的工作和准备。

在决定一个选项之前，这里有几个关于你的需求的问题需要考虑。

*   我需要立即发布我的站点吗，或者我需要一些时间来准备？
*   我的网站需要什么特殊的功能吗？
*   我需要在代码级别访问我的网站吗？
*   我是否需要全天候的客户支持来帮助支持我的网站？
*   我愿意每月花多少钱来保持我的网站正常运行？

在你彻底回答了这些问题之后，你可以开始考虑托管平台和自托管平台之间的主要区别。

### 托管平台和自托管平台的主要区别

尽管名字听起来很相似，托管平台和自托管平台还是有一些主要的区别。自托管平台在注重细节的网页设计师中很受欢迎，他们精通编码并愿意在构建网站时花费额外的时间。另一方面，托管平台在企业主和其他需要网站功能但不想尽快发布内容的个人中很受欢迎。选择托管选项的人不担心控制他们网站的每一个细节。事实上，如果有这个选项，大多数人甚至不知道如何进行代码级编辑。

### 托管平台

出版物、博客和企业主都喜欢托管平台的方便和简单。有了这些提供商，你可以在几秒钟内运行一个博客或电子商务网站。你不必担心安全或更新。您还可以选择新的布局，而不必离开网站。然而，与自托管选项相比，您在操作站点时会受到严重限制。有了这个选择，你就放弃了一些自由来换取简单和速度。

#### 托管平台的优势

*   易于设置
*   乐于助人的支持人员
*   不断更新
*   托管服务器
*   与其他博主联系

#### 托管平台的缺点

*   有限的插件和主题
*   在代码级别没有访问权限
*   无法上传插件和主题

对于许多网站所有者和博主来说，有限的插件和主题不是问题。对于不懂编码的人来说更是如此。支持团队将能够处理您遇到的任何重大问题。

### 自托管平台

任何需要特定设计或强大功能的网站所有者都必须选择自托管平台。有了这个选项，你将能够访问成千上万个不同的主题和插件。将这些上传到你的网站上就像点击一个按钮一样简单。如果需要，您甚至可以创建自己的功能。对于需要定制站点的开发人员来说，自托管平台是显而易见的选择。

#### 自托管平台的优点

*   庞大的插件和主题库
*   代码级别的访问
*   定制一切的能力

#### 自托管平台的缺点

*   管理服务器
*   安装和升级平台
*   与插件冲突
*   仅通过论坛提供支持

自托管平台为托管选项提供了更高级别的定制。通过一些额外的努力和知识，你可以调整网站的每个方面来满足你的特定需求和愿望。对于那些需要独特功能和设计的网站所有者来说，这是一个很好的选择，而这些功能和设计在托管平台上是不可能的。尽管自托管平台涉及到一些额外的工作，但它并不像一些人所说的那样可怕。对于那些想走这条路的人来说，[数字海洋是一个很好的选择](https://dev.to/stevenanthonyrevo/first-time-experience-with-digital-ocean-from-shared-hosting-environment)。

如果你是一个新手，想建立你的第一个网站，托管网站是更好的选择。它们提供了许多通过一键安装和用户指南简化的功能。如果你走的是自助路线，请做好更多工作的准备。
