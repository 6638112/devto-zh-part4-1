请欢迎 Klara 成为 JS Kongress 2020 计划委员会(PC)的成员。[![👏](img/63fb1e489156ceee3f67db2ecd2d5ee9.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--UbHLI3df--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://s.w.oimg/core/emoji/12.0.0-1/72x72/1f44f.png)[![👏](img/63fb1e489156ceee3f67db2ecd2d5ee9.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--UbHLI3df--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://s.w.oimg/core/emoji/12.0.0-1/72x72/1f44f.png)[![👏](img/63fb1e489156ceee3f67db2ecd2d5ee9.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--UbHLI3df--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://s.w.oimg/core/emoji/12.0.0-1/72x72/1f44f.png)据她说，会议就是在与他人的交流中获得见解、想法&灵感。这就是为什么她的意见在日程安排过程中非常有价值。

首先，我们问了她几个问题:

**你为什么期待#jskongress？**

JS kong RESS 2019[编者注:去年 3 月]实际上是我作为开发者参加的第一次会议。你把标准定得太高了。车队倾注了如此多的爱，我很高兴也很自豪能在 2020 年成为其中的一员。

你如何看待或联想到 2020 年的格言“扩展 JS——挑战极限:大规模平台、数据驱动架构和现代 API”？

会很大的。*咳*但是说实话，我对编程还相当陌生。参加会议并不总是为了讨论或成为最博学的专家。能够获得关于主题的见解、想法和灵感同样重要。

你对这个项目的愿景是什么？

当我今年早些时候参加 JS Kongress 时，我非常激动，因为这个项目是如此的多样化和精心选择。它不仅提供了关于特定的、更高级的主题的讲座，还提供了初学者友好的、更基础的讲座。这是一个很好的方法，我希望我们作为一个委员会将继续跟进，为每个参加的人找到一个好的组合。

你还记得你第一次接触 JS 的时候吗？原因是什么，或者是谁？

是的，就像昨天一样。我通过 Twitter 标签#100DaysOfCode 学习编程。在学习了 HTML 和基本 CSS 之后，我的许多同龄人都在为 JavaScript 而奋斗，我也一样。但我很快就爱上了它&它提供的所有可能性——交互式网络、生成艺术、前端、后端……

你对所有演讲者和所有想提交演讲想法的人有什么激励吗？

动手吧！[![😊](img/c93db309c22972f11e79494e2dd33eba.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--0FvlDT7J--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://s.w.oimg/core/emoji/12.0.0-1/72x72/1f60a.png)

如果你能接触到观众中的一个人，那就是赢了。如果你对一个话题充满热情，发现了一些东西或者从事一些有趣的事情，请分享。这不是关于发表完美的演讲。它是关于激励人们&分享知识。

你在慕尼黑生活和工作。你有什么秘密建议给每个参加 JS Kongress 的人吗，他们需要看什么，去哪里吃饭，或者散步的美丽景点(或其他)？

只要迈出门去，散散步。慕尼黑是一座如此美丽的城市。如果你有机会，一定要去探索周围的乡村——那里美得令人窒息。我总是心存敬畏。PS:就像好朋友推荐的，试试一个[Schmalznudel @ caféFrischhut](https://www.likealocalguide.com/munich/cafe-frischhut)。【T2![😉](img/d01d77fce0ff5b3851e7fc16aa3e21dc.png)
