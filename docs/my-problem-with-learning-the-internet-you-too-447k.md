# 我学习互联网的问题。你也是吗？

> 原文：<https://dev.to/edijaloedoc/my-problem-with-learning-the-internet-you-too-447k>

我有一个问题...

当我坐下来，回顾概念，接触库/框架，做笔记，等等...我迷路了。

我似乎不能选择一件事并坚持下去。我一直试图尽可能多的学习，但我从来没有完全掌握和理解基础。甚至跟随文档可能会导致一个小项目或一个有补充材料链接的教程，或者更好的，一个视频！我迷路了！

这次 web 开发之旅不同于我曾经尝试过的任何事情。在此之前，当在工作中承担新任务或学习特定概念时，生活是简单的。做事只有两种方式:正确的方法和错误的方法。这使得它更容易付诸实践。

有了这些看似无穷无尽的方法，在概念的基础上用你自己的技能去创造，我迷失在实用性和实现中。再加上大量的资源，让我感觉自己置身于数字海洋之中。

正因为如此，我开始接触许多主题和项目，但从未完全完成或理解我正在研究的主题。这就好像我是一个厨师，试图做一道没有所有配料或安全标准的菜。我可以做一些看起来可以吃的东西，但我可能会割破手指，把整盘菜都扔掉。

基础:HTML、CSS、JS
近期库:React
近期框架:Angular
教程:Odin 项目、FreeCodeCamp
个人项目:作品集、社交媒体 App

还有更多！

我可以发泄更多，但我真的只是想分享我的痛苦，因为我继续在 web 开发的世界中找到我的立足点。
