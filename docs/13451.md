# PHP 框架指南& 2019 年十大最佳 PHP 框架

> 原文：<https://dev.to/yedukondalu319/php-frameworks-guide-top-10-best-php-framework-2019-a5j>

PHP 被世界上 83%的网站使用，因此是世界上广泛使用的网页设计和开发语言之一。Rasmond Lerdoff 并没有把它作为一种编程语言来开发。最初只是为了管理他的个人网站。但后来，经过数月的测试和改进工作，正式版于 1997 年发布。PHP 早先被缩写为个人主页，但现在它被更好地称为超文本预处理器。

最新版本的 PHP 7 旨在通过重构 Zend 引擎来优化 PHP 性能，并利用更紧凑的数据结构和改进的缓存局部性，同时保持近乎完全的语言兼容性。使用框架是因为它的开发速度更快，更容易维护，问题更少，安全性更好。
为什么要用顶级的 PHP 框架？

使用 PHP 框架有一些很好的理由:

```
Organization of code and file is extremely easy
Numerous libraries and tools are available that can help in form validation, database abstraction, input/output filtering, session, and cookie handling, and managing email, calendar, and pagination etc.
Most of the frameworks follow MVC model architecture
Frameworks help out with less code which ultimately speeds up development
Frameworks come with various input and output filtering functions that add an additional security layer to protect the website against certain attacks.
Frameworks suit well to teamwork
With active community support, frameworks come with a solution for any query very quickly 
```

2019 年最佳 PHP 框架排行榜

1.  拉勒韦尔

自 2015 年以来，它一直是最受青睐和最广泛使用的 PHP 框架。Laravel 简化了授权逻辑并控制对资源的访问。它支持带有许多预装特性的面向对象的库。Artisan 之类的工具也包括在内，它们帮助执行乏味的编程任务，否则这些任务将由大多数程序员手动执行。它还支持 MVC 架构，如 Symphony，确保逻辑和表示的清晰性。

1.  CakePHP

CakePHP 最好的一点是它 100%的向后兼容性，这使得它可以立即升级。尽管它不适应新的 PHP 习惯，但它已经以稳定版本的形式显示了一些一致性。使用胖模型、瘦控制器和自动路由方法意味着大部分工作是自动化的。

1.  Zend 框架

代码的可重用性是 Zend framework 最大的优势。吸收面向对象方法的所有优点，只使用你需要的东西，去掉其他东西，一个充满工具、组件和库的包，自由选择自己的模型，一个拥有精心制作的文档的大型社区作为其丰富性和可用性的补充。
