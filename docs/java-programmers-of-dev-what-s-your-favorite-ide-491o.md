# DEV 的 Java 程序员，你最喜欢的 IDE 是什么？

> 原文：<https://dev.to/prahladyeri/java-programmers-of-dev-what-s-your-favorite-ide-491o>

我已经很久没有接触 Java 了，但是在那个时候，Eclipse 是我的最爱。Netbeans 也不错，但是我更喜欢 Eclipse 支持所有不同类型的项目，并且有多种语言的插件，甚至 Python (PyDEV)和 PHP (PDT)！

但自 2012 年以来发生了很多变化，我转向了 Python 和简单的代码编辑器，如 Atom 和 Notepad++，IDE 世界也朝着完全不同的方向发展。

另一件事是谷歌在 2016 年左右停止支持 Eclipse，并将重点从 Eclipse 转移到 Intellij 公司的 Android Studio。我对 android 编程不太感兴趣，但如果你选择这个方向，这可能是你唯一有限的选择。

但最重要的问题是，如今开发者更喜欢什么？你更喜欢 Intellij 还是 Eclipse？
