# 什么时候开始寻找下一个角色？

> 原文：<https://dev.to/chris_bertrand/when-to-start-looking-for-your-next-role-3b8n>

什么时候开始找下一份工作比较合适？

你已经在同一个地方呆了一/五/十年了。你进步了吗？你学到新东西了吗？你是否觉得你现在的雇主在经济上低估了你，或者你的努力没有得到认可？你该怎么办？

有你能申请的内部空缺吗？你向你的老板提出过任何担忧吗？你觉得这样做舒服吗？

你一直在关注市场吗？有什么吸引你的吗？你有可以联系的首选招聘人员吗？

你担心继续生活吗？你喜欢和你一起工作的人吗？通勤体面吗？原地不动有很多因素。

最终决定权在你。

你怎么想呢?
