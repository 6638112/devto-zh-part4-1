# 招聘人员为什么要隐藏公司名称？🤷‍♂️

> 原文：<https://dev.to/steveblue/why-do-recruiters-hide-the-name-of-the-company-15ep>

呃，另一个招聘人员刚刚在 LinkedIn 上给我发消息说某个未披露的公司有一些机会。这大概是今年第一百次了。

我感到很幸运，有这么多招聘人员给我打电话，然而这些角色几乎都不是我感兴趣的职位。最重要的是，招聘人员没有提到公司的名字。他们都这样做。

[![](img/2f4f394200bef921d473f3f63691ad1f.png)](https://i.giphy.com/media/NlCtOwpyUp15e/giphy.gif)

今天我鼓起勇气问他们中的一个人为什么。招聘人员回答道:“不幸的是，加上我客户的名字会让我的工作变得毫无意义，所以它不在那里。”

如果你把公司的名字写在最初的公报上，认为你的工作毫无意义是完全没有道理的。这乍一看似乎是明智的，但它完全不是。

新工作的面试不应该像一部悬疑电影。

[![](img/d1f75deef735efd554f5d9662a281041.png)](https://i.giphy.com/media/pC8uNEr5Y5DzO/giphy.gif)

招聘人员的工作是建立关系。招聘主要是牵线搭桥。招聘人员与公司有关系。你不知道。不说出公司的名字就像去相亲一样。如果你知道这家公司的名字，你可以对他们做一些调查。让我们不要浪费我们两个人的时间。

这并不像你真的有机会看到公司的名字就撒腿就跑。除非你认识某个在公司工作的人，而那个人会在内部推荐你。这种情况发生的可能性微乎其微。招聘人员在推销你的技能方面比你更有经验。一旦招聘人员联系了你，直接联系公司只是迷失在他们的官僚作风中是徒劳的。沿着这条路走下去也可能需要更长的时间。

事实上，它可以变得更糟，但这是另一个帖子的主题。

[![](img/f28033311bacc76847abe55f1abb13a7.png)](https://i.giphy.com/media/nknFriks5Yu6A/giphy.gif)

好的招聘人员很难找到。如果你碰巧和一个对你很好的人一起工作，保持一个开放的沟通渠道。你会得到回报的。

你怎么想呢?
