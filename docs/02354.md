# 如何以非“专业开发者”的方式使用 Python？

> 原文:[https://dev . to/davidmm 1707/how-do-you-use-python-in-a-not-professional-developer-way-29bg](https://dev.to/davidmm1707/how-do-you-use-python-in-a-not-professional-developer-way-29bg)

我喜欢 Python。事实上，几年来它一直是我最喜欢的语言(也许在尝试过围棋之后我会改变主意...).

但是我最喜欢 Python 的一个特性是，你可以将它用于你的非专业开发人员的日常工作中。

我们知道我们可以用它来创建 web 后端、REST APIs、部署等等，但是我喜欢 Python 的一点是，与其他语言不同，它可以被非程序员轻松地用来帮助他们完成任务。

例如，我读到过秘书通过学习 Python 来自动化花费他们几个小时甚至几天的任务，操作文件，使用基本的正则表达式从文件和/或电子邮件中提取电话，以及诸如此类的事情。

我开始创建一些操作文本文件的小项目，通过搜索文本文件中的文本，修改它们，如果找到某些条件就重命名文件，等等，我喜欢这样。

所以我的问题是:你如何在日常生活中使用 Python？尤其是你不是职业程序员的话。
