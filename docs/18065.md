在本教程中，我们将通过例子来理解 **Angular 9|8 ngIf，ngIfElse，ngIfThen** 结构指令。Angular 9|8 提供了广泛的基本指令，帮助您根据应用于模板的条件渲染模板。

Angular 8 中的**结构指令作用于布尔值模式，如果值为真，则渲染模板，如果值为假或空，则不渲染模板。在本教程中，我将通过例子向你展示 **Angular 8 结构指令**是如何工作的。让我们更仔细地看看结构指令属性。**

| 财产 | 详述 |
| --- | --- |
| **ngIf** | 它的只写属性作用于布尔表达式，以识别作为呈现模板的条件。 |
| **ngIfThen** | 它是只写属性，当条件返回 true 时，它会呈现模板。 |
| **ngIfElse** |  |

它是只写属性，当条件返回 false 时，它会呈现模板。
