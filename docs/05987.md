大家好，

在我目前的团队中，我们用 React 构建前端，但我一直对 Angular JS 很好奇。

这就是为什么我决定用 Angular 来写我的个人博客！

该应用程序被部署到 AWS S3，并通过 AWS Cloudfront 分发。
DNS 由路由 53 管理。
基础设施是通过 Cloudformation 建模的，令我惊讶的是，53 号公路是最难建模的。我花了整整一个下午才让它工作起来(meehh)。
对于我的下一个爱好项目，我将尝试 [AWS cdk](https://aws.amazon.com/cdk/) 而不是 Cloudformation。

这是[博客](https://napicella.com/)的链接。

我和 Angular 玩得很开心，结果对于两天的工作来说还不错。让我知道你的想法！
