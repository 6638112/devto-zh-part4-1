# 向降价链接添加标题

> 原文:[https://dev . to/dzhavat/adding-a-title-to-a-markdown-link-1565](https://dev.to/dzhavat/adding-a-title-to-a-markdown-link-1565)

这是我最近学到的一些关于降价链接的小知识。

您可以通过在 URL 后面用引号括起来为链接添加标题。

```
[Link to something awesome](https://dev.to/ "Check out Dev.to") 
```
