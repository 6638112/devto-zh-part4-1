# API 经济学:市场将为更智能的 API 支付更少的费用

> 原文：<https://dev.to/alibaba50144995/apinomics-marketplace-to-pay-less-for-smarter-apis-62d>

刚刚推出了一个新的 API 平台，希望得到您的意见。这对社区是有用的。Apinomics 是一个为更智能的 API 支付更少费用的市场。

我们真切地感受到 API 的商业模式不公平，有时与我们使用 API 的方式无关的痛苦。此外，浏览所有文档并意识到调用哪些 API 端点来满足需求，然后解析数据以使用所需的数据点，这通常是一件痛苦的事情。我们通常用基于 API 调用的商业模型来支付所有的数据。然而，大多数情况下，我们只使用一些对最终 API 用户并不总是有用的数据。

因为这种挫败感，我们创造了养蜂学。您可以根据自己的需要定制自己喜欢的 API，并且只根据它为您的业务或项目带来的价值进行支付。

更详细地说，它提供了以下值:

1.  降低您的 API 成本

为您使用的东西付费。让定价适应您从 API 创造价值的方式。

1.  自定义 API

只选择你真正需要的，并把你最喜欢的 API 整合在一起。

1.  长期集成 API

如果发生变化，我们会自动调整 API 或找到替代 API。你们的结合永远不会中断。

我们在[www.apinomics.co](http://www.apinomics.co)上提供了一些 API 示例，该平台仍然只接受邀请，但你可以从网站上请求。据我们计算，平均来说，我们可以降低 40%的成本。我们的目标是让任何商业 API 变得更便宜、更智能。我希望得到您对此的反馈:)
