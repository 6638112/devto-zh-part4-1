# Embedly 现在增加了对 DECS 代码片段的支持。

> 原文:[https://dev . to/decs _ dapp/embedy-now-adds-support-for-decs-code-snippets-5822](https://dev.to/decs_dapp/embedly-now-adds-support-for-decs-code-snippets-5822)

Embedly 提供了一套工具、API 和库来帮助您将媒体提供商的内容嵌入到您的网站和应用程序中。更丰富的内容对访问你的网站或使用你的应用的人来说意味着更吸引人的体验。从今天开始，Embedly 刚刚增加了对随处嵌入 DECS 代码片段的支持。

[![](../Images/84abbcda363863fc39e693e17af1c882.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--F9XmdOjL--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1600/1%2ASuEn2KxJ5Xp9z5ZF-hCKWA.png)

你现在可以在任何博客或网页上直接分享你的 DECS 代码片段。所有主要的博客平台都为嵌入提供商的内容提供无缝支持，包括 WordPress、Blogger、Tumblr、Medium 和许多其他的。

要将丰富的代码片段添加到博客文章中，请将 DECS 代码片段的 URL 复制并粘贴到 Medium story 编辑器中。Medium 对嵌入式内容的无缝支持将这个链接转换成您在下面看到的嵌入式丰富片段:-)
