# 编程背后的思维模式

> 原文：<https://dev.to/zex/mindset-behind-programming-4e7p>

对 0/1 世界来说，我们是外星人。🤓大多数时候，我们需要与它交流，才能完成现实世界中的事情。

这不仅仅是多一门编程语言，也是在你的工具箱中多一个方便的工具来帮助你“搞定它！”瞬间。所以这个工具可以是一种编程语言，一个有趣的框架，一套协议，一个版本控制系统，等等。

不要为了说话而说话，要为了交流而说话。编码就是要和机器交流，来完成你的工作，实现你的目标。良好的沟通也是如此。

尽可能多地编码来增强你的编码肌肉。

尽可能多地阅读好的代码，了解它有多优秀。

多学习一些基础知识，比如操作系统、网络、算法等等，从来都不是一个坏主意。

你不需要掌握你的工具来完成事情，你通过完成越来越多的事情来掌握它。但是你需要掌握它才能又好又快地把事情做好。

了解你的工具，不要只是复制粘贴英雄。人不是生来就知道一切的。这需要时间，但是值得。

保持高效率，感觉不行就暂停。把自己逼得太远是毁掉 0/1 世界的好方法，他们会通过花费更多的时间、精力甚至金钱来报复你。你能从停顿中得到的将是一个更好的主意。

不要给自己设限，明确你的目标和期望。你想要什么？你希望它是什么样子的？

快乐黑客，快乐编码！8D
