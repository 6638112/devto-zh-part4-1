# 代码故事 E6: Jonathan Perichon 谈简单的 API

> 原文：<https://dev.to/codestory/code-story-e6-jonathan-perichon-on-simple-apis-1ich>

[https://www.youtube.com/embed/URn5vTYwZTk](https://www.youtube.com/embed/URn5vTYwZTk)

通过在洛杉矶的一次偶然实习，乔纳森·佩里松从法国来到了美国。这样做，他爱上了文化和机遇。在为一家要求对用户进行背景调查的初创公司工作后，他亲眼目睹了这一过程是多么陈旧和缓慢。大约 6 个月后，他和他的联合创始人开始构建一个 API 驱动的解决方案，为候选人背景调查的提交、报告和工作流程提供便利…等等。

在上面的剪辑中，Jonathan Perichon 讨论了他为什么喜欢简单的 API。如果你喜欢你听到的，看看下面完整的一集。
