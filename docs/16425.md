# 你用什么词/短语来包含所有人？

> 原文:[https://dev . to/Darren Vong/what-word-phrase-do-you-use-include-every one-485 a](https://dev.to/darrenvong/what-word-phrase-do-you-use-to-include-everyone-485a)

我们说和写的语言似乎天生就有偏见。例如，指一群男人，有很多词可供选择:小伙子、绅士、小伙子...等等。女性的情况类似。

如果你想指代所有人，而不用“所有人”呢？这听起来像是一个反问句，但是十有八九，我发现人们用来这样做的替代词是“伙计”。对我来说，“伙计们”是指一群男人的另一个词，所以当你用这个词时，你真的包括所有人吗？

说一些欧洲语言的人可能会说，按照惯例，使用阳性复数词(如代词)是为了指代混合群体中的每个人。然而，我不敢苟同，并声称这是男性偏见和过时的习俗，似乎不再适用于我们生活的世界。

为了确保每个人都被真正包括在内，在一次活动或一次会议中，你对由混合性别组成的人群有什么替代选择？

我最喜欢的一些英语单词是:you ' all，乡亲们，peeps。你的呢？
