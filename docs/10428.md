# 每日挑战#28 -杀死怪物！

> 原文:[https://dev . to/the practical dev/daily-challenge-28-kill-the-monster-25 D2](https://dev.to/thepracticaldev/daily-challenge-28-kill-the-monster-25d2)

我们今天对你来说是一个真正的挑战——你正在和一个怪物战斗，并且足够强壮，几下就能杀死它。但是你每打 3 拳，怪物就打你一拳。你的健康是$ h；怪物数量是$m，怪物可以给你的伤害是$dm。

> 写一个函数，将计算:你收到了多少点击，你收到了多少损害和你的剩余健康。如果你的生命值< = 0，你死了，函数应该返回“英雄死了”。
> 
> 例子
> 
> `killMonsters(100, 3, 33); // => "hits: 0, damage: 0, health: 100"`
> T1】
> 
> *注*:所有数字严格为正数。你的函数应该总是返回一个字符串。
