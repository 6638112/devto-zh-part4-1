# globster.xyz - glob 测试仪

> 原文:[https://dev.to/cdomajno/globster-xyz-glob-tester-37im](https://dev.to/cdomajno/globster-xyz-glob-tester-37im)

你有没有想过你的全球模式是否正确？

globster.xyz 是一个测试和可视化 glob 模式匹配的工具。
在上面的输入框中输入您的 glob 模式，所有匹配的文件
将在树形目录结构中以蓝色突出显示。当你的 glob 不工作时，用它
来调试，或者从
提供的例子中进行实验和学习。它使用引擎盖下的 minimatch(3 . 0 . 4 版)。
