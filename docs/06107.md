PHP 代表“超文本预处理器”，递归首字母缩写。毫无疑问，它是最流行和最广泛使用的服务器端脚本语言之一，用于构建静态和动态网站。

PHP 极其强大，是一个高度安全的平台。PHP 在全球范围内的名气和需求已经聚集了来自任何知名的 [PHP 开发公司](https://www.hiddenbrains.co.uk/php-web-development.html)的开发人才库，或者你可以从中选择的自由职业者。

### [](#what-are-the-benefits-of-using-php)使用 PHP 有什么好处

开源 - PHP 是一个开源框架，可以免费下载，让你不再需要付费或购买额外的软件。它使开发更快、更容易、更简单。PHP 开发人员的社区是巨大的，不断共享升级、代码库等。

成本效益–使用 PHP 开发网站的成本很低，有许多集成的解决方案不需要像其他语言一样的初始成本。

**可扩展性**–脸书，雅虎！和 Wikipedia 是 PHP 如何帮助他们扩展业务的一些活生生的例子。这建立了 PHP 的灵活性和可伸缩性。

**速度&多功能性**–在 PHP 上开发的代码可以在所有重要的平台上运行，并与 Windows、Linux、macOS 和 UNIX 等平台无缝协作。它还以无与伦比的速度支持主要的 web 服务器，如 Apache、Microsoft Internet Information Server、个人 web 服务器，以及 Netscape、iPlanet、Caudium 和 Web 浏览器等服务器。

## [](#how-to-analyze-the-true-potential-of-a-php-developer)如何分析一个 PHP 开发者的真实潜力？

了解 PHP 很好，但还不够。一个专业的开发人员也了解所有的框架，比如 Laravel、Codeigniter、Symfony、CakePHP、Yii、Zend、Phalcon 和 Joomla。

--> PHP 专家工作知识
- >面向对象编程知识
- >应具有 SQL/MySQL 数据库和 it 云环境工作经验
->HTML 和 CSS 以及第三方 API 知识。
- >了解 JavaScript 和控制系统
- >应该熟悉 Drupal、Joomla、WordPress 等 CMS
