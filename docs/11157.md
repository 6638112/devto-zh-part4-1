昨天我们已经学习了如何渲染纹理。这是一个很好的能力，可以在场景完全渲染后产生一些很好的效果，但是我们可以利用屏幕外渲染来做其他事情。

交互式 3D 中一个重要的事情是点击检测。虽然它可能是用 javascript 完成的，但它涉及一些复杂的数学。相反，我们可以:

*   为每个对象指定唯一的纯色
*   将场景渲染到纹理
*   读取光标下的像素颜色
*   将颜色与对象匹配
