# 你考虑过你的软件对环境的影响吗？

> 原文：<https://dev.to/tbutterwith/do-you-consider-the-environmental-impact-of-your-software-keg>

当编写或部署软件时，您是否考虑了它对环境的影响？在选择托管等服务时，这一因素会考虑到设计过程中吗？

最近，随着云计算环境开始公布它们的排放统计数据，我越来越多地思考这个问题。例如[谷歌 100%碳中和](https://cloud.google.com/sustainability/)，AWS 50%碳中和[承诺 100%碳中和](https://aws.amazon.com/about-aws/sustainability/)，微软[加倍重视可持续发展](https://blogs.microsoft.com/on-the-issues/2019/04/15/were-increasing-our-carbon-fee-as-we-double-down-on-sustainability/)
