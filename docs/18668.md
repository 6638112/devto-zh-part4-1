# React 本机 WebView

> 原文：<https://dev.to/skptricks/react-native-webview-4e86>

来源: [React 原生 WebView](https://www.skptricks.com/2019/06/react-native-webview.html)

React Native WebView 是一个用于加载 web 内容或网页的组件。WebView 组件从核心 react-native 库导入。现在，它从内置的核心 react-native 中被替换出来，放在 react-native-webview 库中。WebView 是使用自定义 HTTP URL 的 react native to Open-Embed 在线网站网页中 android 和 iOS 应用程序中使用的通用组件。开发者可以根据自己的需求修改 WebView 设置。

注意:React Native WebView 建议导入 react-native-webview 或 react-native-community。
