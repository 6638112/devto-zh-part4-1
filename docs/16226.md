# 有用的 Visual Studio 代码扩展

> 原文：<https://dev.to/alexbaban/useful-visual-studio-code-extensions-4h6d>

我认为最有用的 Visual Studio 代码扩展列表。

* * *

## **1。Ritwick Dey 的直播服务器**

([https://marketplace.visualstudio.com/items?itemName=ritwickdey。直播服务器](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)

这是一个带有实时浏览器重载的 HTTP 服务器，你可以在状态栏上点击一下来启动或停止服务器。

* * *

## **2。毛华超休息客户端**

([https://marketplace.visualstudio.com/items?itemName=humao.rest-client](https://marketplace.visualstudio.com/items?itemName=humao.rest-client)

使用这个扩展，您可以以纯文本格式创建常规的 HTTP 请求并运行它们，然后直接在 Visual Studio 代码中查看响应。您也可以将多个 HTTP 命令放在一个文件中，并保存它们以备将来使用。具有对“基本身份验证”的身份验证支持。

* * *

## **3。亚伦·邦德更好的评论**

([https://marketplace.visualstudio.com/items?itemName = aaron-bond . better-comments](https://marketplace.visualstudio.com/items?itemName=aaron-bond.better-comments)

这个扩展为代码注释增加了“视觉区别”。您可以用某些字符“标记”(分类)您的评论，使它们更突出，这样当您寻找特定类型时，您可以轻松找到它。

* * *

## **4。科恩拉德公司的支架对着色机 2**

([https://marketplace.visualstudio.com/items?itemName = Coen raads . bracket-pair-colorizer-2](https://marketplace.visualstudio.com/items?itemName=CoenraadS.bracket-pair-colorizer-2)

添加额外的颜色(蓝色、黄色、紫色等。)来匹配代码中的大括号，并为您提供了更多的线索来帮助您更快地理解代码编辑器中正在发生的事情。

* * *

## **5。Josh Johnson 的 XML 工具**

([https://marketplace.visualstudio.com/items?itemName=DotJoshJohnson.xml](https://marketplace.visualstudio.com/items?itemName=DotJoshJohnson.xml)

当你使用 XML 时(现在不太受欢迎)，这个扩展增加了一个格式化特性，并用“XML 树视图”增强了浏览器。你也可以缩小(把它变成一行)你的 XML。

* * *

## **6。vscode-icons 团队的 vscode-icons**

([https://marketplace.visualstudio.com/items?itemName = vs code-icons-team . vs code-icons](https://marketplace.visualstudio.com/items?itemName=vscode-icons-team.vscode-icons)

向具有不同扩展名的文件添加图标图像。每个文件扩展名都有不同的图标。这使得区分文件结构中的文件类型更加容易。

* * *

## **7。米奇·丹尼的编码解码**

([https://marketplace.visualstudio.com/items?itemName=mitchdenny.ecdc](https://marketplace.visualstudio.com/items?itemName=mitchdenny.ecdc)

允许您快速将一个或多个文本选择与各种格式相互转换。

* * *

## **8。米克斯塔德的 DotENV**

([https://marketplace.visualstudio.com/items?itemName=mikestead.dotenv](https://marketplace.visualstudio.com/items?itemName=mikestead.dotenv)

编辑*时增加语法高亮显示。env* 文件。

* * *

## **9。马尔科·斯塔齐的流浪档案支持**

([https://marketplace.visualstudio.com/items?itemName=marcostazi。VS-code-flower file](https://marketplace.visualstudio.com/items?itemName=marcostazi.VS-code-vagrantfile)

它为*流浪文件*文件提供语法高亮支持。

* * *

## **10。王一一增强了降价预览功能**

([https://marketplace.visualstudio.com/items?itemName = shd 101 wyy . markdown-preview-enhanced](https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced)

在当前编辑器窗格的一侧显示呈现的 HTML 标记。

* * *

## **11。GitLens — Git 增压**Git kraken

([https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens)

显示谁修改了每行代码。此外，它还显示了代码更改的时间和相关的提交消息。

* * *
