# 马斯洛的需求层次理论

> 原文:[https://dev.to/vorakl/maslow-s-hierarchy-of-needs-2kec](https://dev.to/vorakl/maslow-s-hierarchy-of-needs-2kec)

马斯洛的需求层次理论是一种理论，通过几个层次阶段展示了人类心理发展的道路。每个阶段都有一套共同的动机和需求。重要的观察是，某个动机属于一个适当的阶段，为了发生，所有先前的阶段必须得到很好的满足。马斯洛理论的主要目标是展示一个人达到心理发展最高水平所需满足的所有先决条件——**自我实现**——只有当所有基本和精神需求都得到满足时才能实现。

[自我实现](https://en.wikipedia.org/wiki/Self-actualization)是一种被非常明确地感知到的需求，被描述为完成一个人所能做的一切，成为一个人最大的成就，实现一个人的全部潜力的愿望。马斯洛用这个术语来描述一种*欲望*，而不是一种驱动力。在他后来的理论中，马斯洛还注意到，最充分的实现可以在把自己交给超越自己的东西中找到。

对我个人来说，这个理论在理解什么是多年来推动我学习、移民并最终意识到我要去哪里的隐性力量方面发挥了令人大开眼界的作用。
