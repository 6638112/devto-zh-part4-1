# 检查 Node.js 应用程序中是否安装了依赖项

> 原文:[https://dev . to/cadoangelus/check-a-dependency-is-installed-in-your-node-js-app-29ee](https://dev.to/cadoangelus/check-whether-a-dependency-is-installed-in-your-node-js-app-29ee)
