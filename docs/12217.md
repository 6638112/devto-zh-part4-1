# 为我的第一次科技会议做准备

> 原文:[https://dev . to/sarahscode/preining-for-my ' s first-tech-conference-59m 6](https://dev.to/sarahscode/preparing-for-my-first-tech-conference-59m6)

今天我将参加在纽约举行的 Codeland 会议。这是我第一次参加科技会议，我很紧张，但也很兴奋。

Codeland 是由 CodeNewbie 组织的，这是一个播客和超级支持社区，旨在帮助那些初学编码的人(但超级欢迎所有层次的人)。我从播客和 twitter 上听说了这次会议(事实上，我一度考虑提交一份 CFP)，但没有注册，部分原因是我以前从未参加过会议，不知道我是否会喜欢它，部分原因是我已经进入全职找工作模式一段时间了，不知道我是否会开始一份新工作，这份工作可能不会热衷于我在那里这么早离开一天。

几周前我终于做出了注册的决定。新工作的事情看起来不会在会议之前发生(如果发生的可能性很小，如果我的公司不让我离开，我愿意吃掉这笔钱)。至于不知道我是否喜欢会议...从我的公寓坐地铁 30 分钟(或更少)就能到达会议地点。如果我不喜欢，我可以离开。我很少有机会参加这样的会议...所以我想我应该抓住机会，现在就尝试一下。

一旦我决定注册，下一步就是买票。Codeland 提供“仅讲座”和“讲座和选择研讨会”门票。我真的对讲座非常感兴趣，但我和一位家庭成员讨论了一下，决定既然有些研讨会看起来很有趣，我应该多花 20 美元买研讨会的门票。我想我会在离活动更近的时候决定我想做什么。

#### [](#what-to-learn)学什么？

活动前几周，我收到了一封电子邮件，邀请我选择一个研讨会(和赛道)。我不知道我必须提前选择(我不知道这是不是一个正常的会议，我只是不知道，因为我是一个会议新手，还是这个会议特有的)，我也不知道研讨会有多快就满员了，所以我在早上收到电子邮件时花了一些时间查看研讨会的描述和要求，并将其缩小到一个物联网研讨会和一个可访问性研讨会，但因为我不是一个物联网的大人物(除了使用物联网东西之外...失陪一下，我去请谷歌打开我的风扇...在这种热浪中非常有用)并且对带回家的玩具非常感兴趣，我最终报名参加了由 Luisa Morales 主持的“考虑无障碍建筑”研讨会。可访问性在 web 开发中是非常重要的，我并没有投入太多的精力，所以我真的希望我能从这个研讨会中学到很多，并把我学到的东西应用到我未来的工作中。

虽然真的，我不可能错了任何车间，他们看起来都很有趣。

这个研讨会(与我正在看的物联网研讨会不同)只在两个分支中的一个分支中提供，所以我正式将自己放在了分支 B 中。分支 A 中有一些我很遗憾错过的讲座，但分支 B 中有一些我很期待听到的好讲座。我对 Jo-Wayne Josephs 的演讲“一个移民的科技之旅”特别感兴趣，但我认为所有的演讲听起来都很有趣，我肯定我会喜欢其中的大部分或全部。

#### [](#advance-planning-my-meals)提前规划我的伙食

长话短说...去一些地方对我来说很难，因为我不能吃东西。我严格遵守犹太教规，这意味着不仅我能吃的食物有限制，(就我而言)我所有的食物都必须来自犹太教规认证的厨房。当我注册参加活动时，有一个选择犹太食品的选项，但在最近有人没有意识到我不会吃未经认证的厨房的食物(对于没有在遵守犹太教规的人身边长大的人来说，这是一个可以理解的错误)之后，我想确保会有符合我家人标准的食物。

我在官方会议账户上发了推文询问食物，他们鼓励我发邮件到一个特定的电子邮件地址询问。我一直没有那样做，最后上周晚些时候，我给他们发了一封电子邮件。答案是，是的，他们会有犹太食品，但由于 NYU(会场)目前没有他们的犹太厨房，这将是来自外部供应商，他们会告诉我谁当他们知道。几天后，我收到一封电子邮件，告诉我食物将从哪里来(弗雷斯科)，由于我对这家公司很熟悉(他们在我当地的 7-11 便利店和斯塔滕岛扬基棒球队比赛中出售三明治)，我知道他们的犹太食品认证和食物对我来说是完全可以接受的。

我也不确定他们是否会做三明治以外的东西，我想我可能一整天都不想吃三明治，所以我决定给自己打包一袋麦片当早餐和/或点心。我还打包了水果作为零食，因为吃水果从来都不是一个坏主意。

这提醒了我...不知道他们白天有没有零食？

#### [](#what-does-my-day-look-like)我的一天是什么样子的？

报名参加在纽约举行的会议的好处之一是，我不必担心航班或酒店...我走出公寓，跳上地铁(而且只有一列火车！)，而我在。但是仍然有一些事情需要弄清楚...

1.  **什么时候出现**-8 点开始登记和早餐。现在我知道肯定有适合我的食物，我想去那里吃早餐(我也希望有像样的咖啡...如果没有，我可能不得不在早餐中间跳出来吃点东西)，但我不认为我需要在它开始的时候就在那里。现在，我的目标是在 8:15 到 8:30 之间到达，这将有希望让我有足够的时间去登记和吃早餐，但没有太多的额外时间，我觉得超级尴尬。如果事情变得太尴尬，我总是可以站在角落里打电话，假装自己超级忙。
2.  研讨会/讲座 -一天从欢迎仪式和两个讲座开始，然后是第一次休息。休息之后，我将进入我的易访问性工作室，希望不会太不舒服(我不知道是否会和其他相关人员互动，我对此不太擅长)。研讨会结束后是午休时间。如果早晨对我来说太难了，那可能就是我结束一天的时候了。或者我可能会考虑在做决定前出去散散步，呼吸一下新鲜空气。但是如果我感觉还好，那就是午餐时间了，希望我能找到一些人一起吃饭。午饭后是讲座，我会坐在那里，拿着我的笔记本和一支笔(自我提醒:带上笔记本和笔)，记下我正在学习的所有有趣的东西。如果在那之后我仍然感觉良好，我会在听闭幕主题演讲之前在展厅逛逛(并希望带一些免费的纪念品回家)，这是一个我个人感兴趣的话题。
3.  理想情况下，我想去参加庆功宴。我不是一个喜欢派对的人，我不知道在这种情况下“afterparty”意味着什么，但我认为这是体验的一部分，所以我想尝试一下。但是如果这一天感觉非常疲惫，我不认为我有足够的精力度过，我会跳过庆功宴，直接回家。

#### [](#what-do-i-look-like)我长什么样？

挑选服装对我来说是件大事。总的来说，我喜欢看起来相对不错(尽管我的成功程度不一)，但对于半特殊的场合，比如我上学的第一天或我的第一次科技会议，我喜欢买一套新衣服和/或挑选一些特别的东西。上周我去买了两套衣服，一套是为了参加社交活动，一套是为了参加会议，但最终只为社交活动买了些东西，所以我又回到了起点。

天气预报说 81 度，有分散的雷雨，这意味着我需要穿适合下雨的鞋子，因为我不想穿雨靴，我可能会穿运动鞋。这意味着我不能指望我的鞋子让我的服装看起来更好，那必须来自服装本身。

在仔细检查了我能找到的衣服之后，我选定了我上一份工作第一天穿的那套衣服——这是一套舒适的衣服，有我需要的口袋，看起来还不错。在我把它摆好之后，我想起了关于这套服装的其他事情——我上一份工作的第一天是一次无意的米老鼠[迪士尼乐园](https://disneybound.co/)活动，所以我试图找到我最好的米老鼠配饰(主要是我的米老鼠耳环...除非我想带着我的马拉松赛跑耳朵到处走，但我不想。我想到了另一个迪士尼乐园的想法，想到了一些我不确定是否可行的东西，所以我决定把两套衣服都准备好，然后早上再做决定。

我很高兴我想到让我的服装成为迪士尼主题，因为我喜欢从我最喜欢的角色中寻找服装灵感，即使没有人知道这是我想要的，它也总能让我微微一笑。我认为穿上让我感觉良好的服装会让我感觉更舒服。

#### 我的会议目标是什么？

我这次会议的主要目标是度过这一天。最重要的是，我知道我能做到。如果我学到一些新东西，酷。如果我遇到一些牛逼的人，就更爽了。如果我得到一些有趣的新礼物，太棒了。如果我一整天都没有感到完全不自在，不觉得我不应该离开...任务完成。

#### [](#whats-next)接下来是什么？

这次会议对我来说是一次考验。我真的很想更多地参与技术社区，我也想参加会议，但拥挤的房间和新的情况对我来说非常困难。我想享受这次会议，但我也想看看它是否能激励我参加其他会议。
