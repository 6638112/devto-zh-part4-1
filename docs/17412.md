# 隐私问题还是阅读理解问题？

> 原文:[https://dev . to/kappa lucky/privacy-problems-or-reading-comprehension-issues-g0o](https://dev.to/kappalucky/privacy-problems-or-reading-comprehension-issues-g0o)

我不断收到关于“谷歌在监视我们”和“如何删除你的账户”的文章在我的各种订阅中寻找更私密的东西...

我一直觉得，人们根本不会去阅读和指责一家公司不“清楚”，或者干脆不知道这家公司到底卖什么。

以谷歌为例，我已经被锁定多年；我一直意识到他们本质上是一家广告公司，并使用它从我这里学到的数据来销售我可能喜欢的产品，以及在他们的平台上个性化我所做的一切。在我眼里是双赢。

作为一名开发者和/或企业主，你对软件(尤其是自由软件)用户不知道数据是如何或为什么被收集而引起的隐私问题有什么感觉？

你觉得某种形式的教育会有益于教导其他人，这是一个分析他们个人数据的程序，而不是来自公司的实际个人？
