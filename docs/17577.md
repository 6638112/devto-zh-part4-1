# 构建一个内置 CMS 的静态网站

> 原文：<https://dev.to/triptych/hylia-build-a-static-site-with-built-in-cms-5h9d>

[Hylia](https://twitter.com/andybelldesign/status/1141633780408606720) 是你通过 [11ty](https://www.11ty.io/) 和 [Netlify](https://www.netlify.com/) 上的主机生成的一个新的“静态站点”。您可以为自己的特殊主题创建博客、个人网站或粉丝网站。它是由[安迪·贝尔](https://twitter.com/andybelldesign)创建来帮助你脱离媒体、Wordpress 和其他网站的

查看上面的推文，[访问网站](https://hylia.website/)。
