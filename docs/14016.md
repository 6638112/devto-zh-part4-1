# 对我的简单介绍

> 原文：<https://dev.to/dexterstpierre/a-brief-introduction-to-me-34ap>

你好！我只是想给你简单介绍一下我是谁，我做什么，以及我开始写博客的目的。

我叫德克斯特·圣皮埃尔。我是一名在明尼苏达州圣保罗的网站开发人员。我目前做一些自由职业者，同时也是一家建筑公司的承包商，从事各种内部工具的工作，这些工具用于跟踪工作订单、收入、工作时间、设备和各种其他东西。我使用 Javascript 构建全栈 web 应用程序，最近对 JAM (JavaScript、API、标记)栈非常感兴趣，希望了解更多信息，并为使用它的各种公司构建更多网站。作为一名体力劳动者(我在训练营学会了编程)，我很高兴能够将 web 应用程序和优秀网站的巨大可能性带入建筑、景观和房地产领域，这些行业在很大程度上仍然依靠手工流程运行

我开始写博客有几个不同的原因。我想记住我学过的东西。我最近意识到，在过去的几年里，我学了太多的东西，以至于学了不久就忘记了很大一部分，我也不记得我是如何学到很多我知道的东西的。所以我想有一个记录，记录我什么时候学的东西，怎么学的东西，供自己以后参考。我还想把互联网添加到集体信息库中。我数不清有多少次，我在某件事情上纠结了几个小时，好不容易才在谷歌上找到现成的答案。我想帮助我的后来人，让他们能够更容易地解决我也曾苦苦挣扎的问题。

我以前从未写过博客，所以我在这里的帖子可能相当不规则，我可能一周写三次，然后三个月不写了。但是当我挣扎着经历一些事情或者学习一些新的令人兴奋的事情时，我会写下来并与世界分享。感谢你加入我的旅程。
