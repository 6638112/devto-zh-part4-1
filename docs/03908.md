# 用鼓舞人心的情绪板提升 UX 设计

> 原文:[https://dev . to/oodlestudio/enhancing-UX-design-with-inspiring-mood-boards-2k no](https://dev.to/oodlesstudio/enhancing-ux-design-with-inspiring-mood-boards-2kno)

伟大的设计反映了现实世界中的灵感和美学。对于一家 UX 设计公司来说，利用自然和人造的设计参考来创建数字设计尤为重要。情绪板是一种这样的 UX 研究技术，通过收集各种密切相关的设计思想和概念来塑造数字景观。这种做法使 UX 的设计师能够明确他们的想法和设计动机，创造性地构建直观的数字产品。

这篇博文探讨了情绪板在 UX 设计过程中的重要性和使用案例。

情绪板及其在设计世界中的意义

设计师用情绪板作为画布，以图像、字体、字体等形式表达某种设计思想。这个想法是通过捕捉灵感和与品牌身份产生共鸣的元素来可视化最终设计。一个理想的情绪板包括图形、用户界面特征、图标和其他描述品牌的项目。

保罗·兰德是 IBM、UPS 和其他全球企业的标志设计师，他用下面的话解释了策划设计的艺术

“艺术家是虚构或真实事物的收藏家。他积攒东西的热情就像一个小男孩往口袋里塞东西一样。废料堆和博物馆被同样的好奇心所包围。他拍快照，做笔记，在桌布或报纸上，在信封或火柴盒的背面记录印象。为什么一件事而不是另一件事是神秘的一部分，但他是杂食的。”

――保罗·兰德:设计师的艺术

情绪板可以用物理形式和数字形式来表示。情绪板上的显示项目可能包括草图、织物、调色板、纹理、形状、字体和其他一些设计元素。

情绪板用户体验设计服务

策划情绪板的主要目的是分享网络界面、产品设计或品牌个性的共同愿景。它通过展示最终设计的大致轮廓来简化 UX 设计流程。

以下是情绪板如何为有效的数字设计搭建舞台-

1)建立愿景

情绪板是展示 UX 研究服务提供商吸收的研究材料的有形催化剂。它使整个设计团队能够建立并分享最终设计的共同愿景和目标。虽然情绪板不像线框或原型那样精确，但它们建立了品牌设计的情感和感官形象。

2)与利益相关者合作

让利益相关者接受 UX 的研究可能是一项艰巨的任务。然而，情绪板的时间和成本效益属性使其成为利益相关者容易接受的 UX 研究技术。此外，UI UX 设计服务的提供商可能会与决策者合作，在投资最终设计之前标记他们的需求。

如今，Pinterest 和 Tumblr 等各种在线平台充当了动态情绪板，以策划和分享设计灵感。然而，专业的网页设计服务强烈推荐物理情绪板来创建产品设计的牢固印象。

不同的情绪寄宿方式

1)使用照片和视觉材料

虽然可用于情绪板的图片数量并不缺乏，但一致的品牌调性很重要。UX 设计师必须策划高分辨率图形，视觉元素，真实或矢量图像，以及密切代表产品设计的快照。数字情绪板也可以保存短视频或 gif 来明确地谈论这个想法。

情绪板用户界面用户体验设计服务

2)乱涂短语和引语

文字强调了图像的某种意义。用户研究顾问可以使用手头产品的目标受众常用的上下文短语或俚语。情绪板也可以包括标语、品牌价值、定位声明、营销口号和其他重要的品牌引用。

品牌识别情绪板

3)包括颜色、阴影和字体

明确的调色板和排版是反映品牌身份的最有效的设计元素。用户体验设计服务的提供商应该仔细研究产品的可用性和影响，收集相似的色调和字体。情绪板被证明是一个测试各种颜色和字体的简单工具。它使设计师、开发人员和利益相关者能够用一连串的音调和声音来可视化最终的设计。
