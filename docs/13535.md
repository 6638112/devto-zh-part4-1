# 教你的客户:如何做得更好？

> 原文:[https://dev . to/arximughal/teaching-your-clients-how-do-it-better-5a di](https://dev.to/arximughal/teaching-your-clients-how-to-do-it-better-5adi)

上周，我的一个客户让我帮他为他们的产品建立一个视频库。我推荐他使用 YouTube 进行视频托管，并使用 WordPress 发布和嵌入视频，并与他们的客户分享(他不想使用 YouTube，因为有广告和其他东西)。

客户坚持将视频放在与网站相同的服务器上，这是一个共享的托管服务器。我试图告诉客户，从一个共享的主机服务器上传视频是很困难的。但是他坚持。

最后，我花了额外的时间来确保 WordPress 允许他上传大于 800MBs 的文件，并轻松处理它们。但最终，视频太慢，无法播放，客户也不喜欢。

经过与客户的简短通话和大量解释后，客户最终选择在 YouTube 上托管视频，这又成了我的额外工作。

你如何处理这种情况？你如何告诉你的客户什么更适合他们，什么不适合他们？
