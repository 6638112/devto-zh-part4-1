让我们从一个概念开始，设计是一个复杂的，有时是混乱的，但仍然有趣和令人兴奋的行业。

当你刚刚掌握这项技能时，有太多的事情需要研究、做、考虑——更不用说技术在不断发展，新的程序出现，新的趋势开始流行。所有这些都会给新来者带来巨大的压力。

平面设计中有不同种类的对象。标识、标志或字体既用于材料的艺术展示，也用于创造一个视觉交流环境。

他们必须同时执行一项或几项功能

*   吸引目标受众的注意力；
*   介绍公司，使其在竞争对手中脱颖而出；
*   告知和澄清，说明承载意义的部分；
*   刺激销售。

它们是用来识别和建立一个公司、其产品或服务的最重要的元素之一。设计师的失误可能会非常昂贵，尤其是当涉及到媒体或互联网上的主动广告时。主要的部分是考虑所有的细节，这些细节组成了更大的画面。创作各种各样的平面设计作品时，既要考虑到文化传统，同时又要能够运用现代营销技术。此外，重要的是要遵守基本的组成原则，如权宜之计，凝聚力，和谐和动态。
