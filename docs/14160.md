# 开发人员的身体和精神状况

> 原文:[https://dev . to/bo vermyer/developers-1 ojl](https://dev.to/bovermyer/physical-and-mental-conditioning-for-developers-1ojl)

我们倾向于过多地关注发展工作技能。我们忽略了表演中非常重要的一部分:我们自己！

身体和精神健康都决定了你工作表现的上限。虽然你可能擅长编写设备驱动软件，但你执行这种特殊技能的能力取决于你的大脑和身体是否允许你这么做。如果你身心俱疲，你就更有可能犯错误或者花更长的时间去完成一项特定的任务。

为此，你应该在日常生活中包括身体和精神训练。两者都不容易，但都是必要的。饮食也非常重要，但这超出了本文的范围。

关于这个主题有很多很好的读物。我将列出一些要点供您进一步研究。

> 注意:我不是医生，也不是心理学家。以下是基于我自己的经历和研究。在尝试剧烈运动前，请先咨询医生。

## [](#physical-training)体能训练

### [](#goals)目标

*   保持灵活性
*   保持良好的姿势
*   发展推动力量
*   发展拉力
*   保持运动耐力

### [](#points-to-iterate-on)点迭代

以下是实现上述目标的具体方法。

*   **力量训练:**练习提、拉重物。你的身体算重的东西。
*   耐力训练:练习以足够快的速度走动，让你的心率在比你习惯的时间更长的时间内上升。
*   **柔韧性训练:**只要有可能和可行，就练习拉伸。瑜伽对此有好处。高空瑜伽更好。
*   **规律运动:**每小时练习起床走动至少五分钟。

## [](#mental-training)精神训练

### [](#goals)目标

*   选择并保持积极的态度
*   有效管理期望
*   使用积极的自我对话
*   保持注意力集中
*   设定现实的目标
*   有效管理焦虑
*   有效管理情绪

### [](#points-to-iterate-on)点迭代

以下是将有助于发展上述技能的领域。

*   目标设定:练习设定不同复杂程度和时间限制的目标，记录你的结果，然后根据结果改变未来的目标标准。
*   自言自语:每当你发现自己对自己有负面想法时，练习将这种想法转化为积极的想法。
*   呼吸:练习意识到你的呼吸并控制它。
*   想象:练习想象你正在做的事情的积极的未来状态。
*   惯例:练习设定和维持惯例，尤其是在早上和晚上。
*   正念:只要有可能，练习对自己和周围环境的正念意识。避免一心多用。
