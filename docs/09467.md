# 每日挑战#34 -奇怪的字符串案件

> 原文：<https://dev.to/devteam/daily-challenge-34-weird-string-case-54bo>

这个[挑战](https://www.codewars.com/kata/52b757663a95b11b3d00062d)的目标是编写一个函数，它接受一个字符串并返回相同的字符串，所有偶数索引字符大写，所有奇数索引字符小写。

这种索引应该是从零开始的，零位置的索引被认为是偶数。输入字符串将只包含字母字符和空格。只有当有多个单词时，才应该有空格。

**例:**
to _ 怪异 _ 案件('字符串')；# = >返回'字符串'
到 _ 怪异 _ 情况('怪异字符串情况')# = >返回'怪异字符串情况'
